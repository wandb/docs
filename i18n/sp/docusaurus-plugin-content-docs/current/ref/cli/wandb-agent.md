# wandb agent

**Usage**

`wandb agent [OPTIONS] SWEEP_ID`

**Summary**

Run the W&B agent

**Options**

| **Option** | **Description** |
| :--- | :--- |
| -p, --project | The project of the sweep. |
| -e, --entity | The entity scope for the project. |
| --count | The max number of runs for this agent. |
| --help | Show this message and exit. |
