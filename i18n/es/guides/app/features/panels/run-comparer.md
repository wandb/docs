# Comparador de Runs

Utiliza el Comparador de Runs para ver qué métricas son diferentes a través de tus runs. Añade una nueva visualización y haz clic en **Comparador de Runs** para probarlo.

![](https://paper-attachments.dropbox.com/s_2BA455B46A7EB5D90BB456BA993340F060AB348F16A4BF63AB4BB2199F3052A2_1574210574429_demo+-+run+comparer+button.png)

Usa la opción **solo diferencias** para ocultar las filas donde los valores son iguales a través de los runs.
