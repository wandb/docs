<h3 id="number-notEqual"><code>number-notEqual</code></h3>

두 값의 불일치 여부를 결정합니다.

| 인수 |  |
| :--- | :--- |
| `lhs` | 비교할 첫 번째 값입니다. |
| `rhs` | 비교할 두 번째 값입니다. |
