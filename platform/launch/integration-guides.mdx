W&B Launch can submit jobs to a range of compute targets. Choose the guide that matches the environment where you want your agent to run jobs.

<CardGroup cols={2}>
  <Card title="Docker" icon="docker" href="/platform/launch/setup-launch-docker">
    Run jobs on a local machine or a single VM. Best for development, small workloads, and getting familiar with Launch.
  </Card>
  <Card title="Kubernetes" icon="dharmachakra" href="/platform/launch/setup-launch-kubernetes">
    Run jobs on a Kubernetes cluster (including EKS and GKE) using Helm charts, Kaniko image building, and Kubernetes job specs.
  </Card>
  <Card title="Amazon SageMaker" icon="aws" href="/platform/launch/setup-launch-sagemaker">
    Submit training jobs to Amazon SageMaker with ECR, S3, and IAM. Best for AWS-based ML pipelines.
  </Card>
  <Card title="Google Vertex AI" icon="google" href="/platform/launch/setup-vertex">
    Submit training jobs to Google Cloud Vertex AI using CustomJob specs and Artifact Registry.
  </Card>
</CardGroup>
