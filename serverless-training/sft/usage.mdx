Use Serverless supervised fine-tuning (SFT) through [OpenPipe's ART framework](https://art.openpipe.ai/getting-started/about) and the Serverless Training API.

To start using Serverless SFT, satisfy the [prerequisites](/serverless-training/prerequisites) for W&B tools, and then see the [ART Serverless SFT documentation](https://art.openpipe.ai/fundamentals/sft-training).

To learn about Serverless SFT's API endpoints, see the [Serverless Training API reference](/serverless-training/api-reference).
