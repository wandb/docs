<Card title="How does someone without an account see run results?" href="/support/models/articles/how-does-someone-without-an-account-see-" arrow="true" horizontal>
  Use anonymous='allow' in wandb.init to create temporary accounts that can view results for 7 days.
</Card>
<Card title="What are features that are not available to anonymous users?" href="/support/models/articles/what-are-features-that-are-not-available" arrow="true" horizontal>
  Anonymous accounts cannot persist data beyond 7 days or log artifacts. Claim runs to keep them.
</Card>
