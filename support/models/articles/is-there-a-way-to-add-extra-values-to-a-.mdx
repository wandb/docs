---
title: "Is there a way to add extra values to a sweep, or do I need to start a new one?"
keywords: ["Sweeps"]
---

Once a W&B Sweep starts, you can't change the sweep configuration. However, you can navigate to any table view, select runs using the checkboxes, and then choose the **Create sweep** menu option to generate a new sweep configuration based on previous runs.
