---
title: "How do I cite Weights & Biases in a paper?"
keywords: ["Academic"]
---

If you use Weights & Biases in academic work, cite it as a software tool. W&B does not have a required BibTeX entry, so use the software-citation style your venue prefers.

The following BibTeX entry is a starting point that you can adapt to match the version of `wandb` you used and your publisher's style guide:

```bibtex
@misc{wandb,
  title        = {Weights \& Biases},
  author       = {Weights \& Biases},
  howpublished = {\url{https://wandb.ai}},
  note         = {Software for experiment tracking, model management, and evaluation}
}
```

Recommendations:

- Report the version of `wandb` you used (for example, the value of `wandb.__version__`) so that readers can reproduce your setup.
- If your journal or conference requires an author list for software, list "Weights & Biases" as the author.
- If you also used [W&B Weave](/weave/), cite it separately using the same pattern with the Weave URL.
