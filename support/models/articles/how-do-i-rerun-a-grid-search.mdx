---
title: "How do I rerun a grid search?"
keywords: ["Sweeps", "Hyperparameter", "Runs"]
---

If a grid search completes but some W&B Runs crashed, delete those runs to rerun them. Then select the **Resume** button on the [Sweep list in W&B](/models/sweeps/visualize-sweep-results). Start new W&B Sweep agents using the new sweep ID.

The sweep doesn't rerun parameter combinations that already completed.
