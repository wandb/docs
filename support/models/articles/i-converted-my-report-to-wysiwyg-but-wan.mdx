---
title: "I converted my report to WYSIWYG but want to revert back to Markdown"
keywords: ["Reports", "Wysiwyg"]
---

If you converted a report from Markdown to WYSIWYG and want to undo the conversion, the available options depend on how the conversion happened and whether your editing session is still open:

- If the report conversion occurred through the message at the top, click **Revert** to restore the prior state. Any changes made after conversion are lost.
- If a single Markdown block was converted, use `cmd+z` to undo.
- If options to revert are unavailable because of a closed session, consider discarding the draft or editing from the last saved version. If neither works, contact W&B Support.
