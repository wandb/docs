title: "How can I resume a sweep using Python code?"
description: "Resume an existing W&B sweep by passing its sweep ID to the wandb.agent() function."
keywords: ["Sweeps", "Python"]
---

To resume a sweep, pass the `sweep_id` to the `wandb.agent()` function. 

```python
import wandb

sweep_id = "your_sweep_id"

def train():
    # Training code here
    pass

wandb.agent(sweep_id=sweep_id, function=train)
```
