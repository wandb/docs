keywords: ["Administrator"]

This page helps administrators of W&B enterprise deployments diagnose common issues when W&B doesn't recognize an enterprise license or its features are unavailable after you set the license. Use the following checks to identify the most likely cause before you contact support.

## License not recognized

If W&B doesn't recognize the license at all, the problem is usually with the key itself or where you configured it. Check the following:

- Verify the license key is correctly formatted with no extra spaces or characters.
- Ensure the license hasn't expired.
- Check that the license is set in the correct configuration location.

## Features not available after setting the license

If the license is accepted but expected features remain unavailable, the issue is typically with service state or license scope. Check the following:

- Restart your W&B services after you set the license.
- Verify the license includes the specific features you're trying to access.
- Check system logs for any license validation errors.

## License expiration warnings

To avoid an unexpected lapse in licensed functionality, stay ahead of expirations:

- Monitor the **System Settings** page for expiration notifications.
- Set up alerts for license expiration in your monitoring system.

## Get support

For help with enterprise licenses, contact your account team, [Sales](mailto:sales@wandb.ai), or [Support](mailto:support@wandb.ai).

For more information, see [Enterprise licenses](/platform/hosting/enterprise-licenses).
