The Files tab shows a maximum of 10,000 files. To download all files, use the [public API](/models/ref/python/public-api/api):

```python
import wandb

api = wandb.Api()
run = api.run('<entity>/<project>/<run_id>')
run.file('<file>').download()

for f in run.files():
    if <condition>:
        f.download()
```
