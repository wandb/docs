---
title: "How do I find an artifact from the best run in a sweep?"
description: "Retrieve artifacts from the best performing run in a W&B sweep using the Public API."
keywords: ["Artifacts"]
---

To retrieve artifacts from the best performing run in a sweep, use the following code:

```python
api = wandb.Api()
sweep = api.sweep("entity/project/sweep_id")
runs = sorted(sweep.runs, key=lambda run: run.summary.get("val_acc", 0), reverse=True)
best_run = runs[0]
for artifact in best_run.logged_artifacts():
    artifact_path = artifact.download()
    print(artifact_path)
```
