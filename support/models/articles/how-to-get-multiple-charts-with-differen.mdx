---
title: "How to get multiple charts with different selected runs?"
keywords: ["Reports"]
---

W&B Reports lets you display charts for different sets of runs side by side in the same report by using separate panel grids. This is useful when you want to compare results across distinct run selections without switching contexts.

To set this up, follow these steps:

1. Create multiple panel grids.
2. Apply filters to select the run sets for each panel grid. Each panel grid keeps its own run selection, so different charts can reflect different runs.
3. Generate the charts within the panel grids.

After you complete these steps, your report contains multiple panel grids, each of which shows charts for its own filtered set of runs.
