---
title: "How do I set up W&B alerts and notifications?"
keywords: ["Alerts", "Experiments", "Runs"]
---

You can set up alerts in and notifications using the W&B **Settings** page. W&B supports automated alerts in User Settings (run finished, run crashed) and programmatic alerts from training code. For `run.alert()` setup, Slack connection, and examples, see [Send an alert](/models/runs/alert). For event-driven Slack notifications on artifacts or metrics, see [Automations](/models/automations/).
