title: "wandb artifact"
description: "Commands for managing W&B artifacts, including uploading, downloading, listing, and caching."
---

Upload, download, and manage W&B artifacts.

## Usage

```bash
wandb artifact COMMAND [ARGS]...
```

## Commands

| Command | Description |
| :--- | :--- |
| [cache](/models/ref/cli/wandb-artifact/wandb-artifact-cache) | Manage the local artifact cache. |
| [get](/models/ref/cli/wandb-artifact/wandb-artifact-get) | Download an artifact from W&B by its path. |
| [ls](/models/ref/cli/wandb-artifact/wandb-artifact-ls) | List all artifacts in a W&B project. |
| [put](/models/ref/cli/wandb-artifact/wandb-artifact-put) | Upload an artifact to W&B. |
