## Usage

```bash
artifact [OPTIONS] COMMAND [ARGS]...
```

## Description

Upload, download, and manage W&B artifacts.

## Subcommands

| Command | Description |
|---------|-------------|
| `wandb artifact cache` | Manage the local artifact cache. |
| `wandb artifact get` | Download an artifact from W&B by its path. |
| `wandb artifact ls` | List all artifacts in a W&B project. |
| `wandb artifact put` | Upload an artifact to W&B. |
