## Usage

```bash
artifact [OPTIONS] COMMAND [ARGS]...
```

## Description

Upload, download, and manage W&B artifacts.

## Subcommands

| Command | Description |
|---------|-------------|
| [`wandb artifact cache`](/models/ref/cli/wandb-artifact/wandb-artifact-cache) | Manage the local artifact cache. |
| [`wandb artifact get`](/models/ref/cli/wandb-artifact/wandb-artifact-get) | Download an artifact from W&B by its path. |
| [`wandb artifact ls`](/models/ref/cli/wandb-artifact/wandb-artifact-ls) | List all artifacts in a W&B project. |
| [`wandb artifact put`](/models/ref/cli/wandb-artifact/wandb-artifact-put) | Upload an artifact to W&B. |
