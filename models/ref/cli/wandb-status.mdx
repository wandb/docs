Display the current W&B configuration settings.

Print all active W&B settings as formatted JSON, including the base URL, API key, project, entity, and other resolved values.

For example, to show current settings:

```bash
$ wandb status
```

## Usage

```bash
wandb status [OPTIONS]
```

## Options

| Option | Description |
| :--- | :--- |
| `--settings` | Display the current settings. (default: True) |
