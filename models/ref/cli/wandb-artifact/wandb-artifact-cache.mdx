## Usage

```bash
Usage: cache [OPTIONS] COMMAND [ARGS]...
```

## Description

Manage the local artifact cache.

    Cache downloaded artifact files locally to avoid redundant downloads.

    Use subcommands to inspect and reclaim disk space used by the cache.
    
## Subcommands

| Command | Description |
|---------|-------------|
| [`wandb artifact cache cleanup`](wandb-artifact-cache-cleanup) | Reduce the local artifact cache size. |
