Initialize or update W&B configuration for the current directory.

Set a project and entity, create local W&B settings, and prepare the directory for experiment tracking.

For example, set up W&B for the current directory with guided prompts for team and project selection:

```bash
$ wandb init
```

To set the default project to "foobar" and the default entity to "team-awesome" without prompts:

```bash
$ wandb init --project foobar --entity team-awesome
```

To set the W&B mode to offline:

```bash
$ wandb init --mode offline
```

To reset existing W&B configuration for the current directory:

```bash
$ wandb init --reset
```

## Usage

```bash
wandb init [OPTIONS]
```

## Options

| Option | Description |
| :--- | :--- |
| `--project`, `-p` | Set the project to upload runs to. |
| `--entity`, `-e` | Set the entity to scope the project to. |
| `--reset` | Reset existing W&B configuration for the directory. (default: False) |
| `--mode`, `-m` | Set the W&B mode. One of 'online', 'offline', or 'disabled'. |
