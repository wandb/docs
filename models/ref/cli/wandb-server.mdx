Commands for operating a local W&B server

## Usage

```shell
wandb server [OPTIONS] COMMAND [ARGS]...
```


## Options

### `help`

Show this message and exit.                

| Flag | Default | Type | 
|------|---------|------|
| `--help` | False | BOOL |

## CLI Help

```
Usage: server [OPTIONS] COMMAND [ARGS]...

  Commands for operating a local W&B server

Options:
  --help  Show this message and exit.

Commands:
  start  Start a local W&B server in a Docker container.
  stop   Stop a local W&B server
```
