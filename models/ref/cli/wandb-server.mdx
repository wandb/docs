title: "wandb server"
---

Commands for operating a local W&B server

## Usage

```bash
wandb server COMMAND [ARGS]...
```

## Commands

| Command | Description |
| :--- | :--- |
| [start](/models/ref/cli/wandb-server/wandb-server-start) | Start a local W&B Server instance in a Docker container. |
| [stop](/models/ref/cli/wandb-server/wandb-server-stop) | Stop a running local W&B server. |
