Start a detached wandb-core service.

## Usage

```bash
wandb beta core start [OPTIONS]
```

## Options

| Option | Description |
| :--- | :--- |
| `--idle-timeout` | Shut down wandb-core after this much idle time with no connected clients. Uses Go duration syntax, for example 30s, 10m, or 0 to disable idle shutdown. (default: 10m) |
