## Usage

```bash
leet [PATH] | COMMAND [ARGS]...
```

## Description

W&B LEET: the Lightweight Experiment Exploration Tool.

    A terminal UI for viewing your W&B runs locally.
## Subcommands

| Command | Description |
|---------|-------------|
| [`wandb beta leet config`](wandb-beta-leet-config) | Edit LEET configuration. |
| [`wandb beta leet run`](wandb-beta-leet-run) | Launch the LEET TUI. |


## Examples

wandb beta leet                 View latest run
        wandb beta leet ./wandb         View runs in directory
