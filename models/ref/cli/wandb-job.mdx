## Usage

```bash
job [OPTIONS] COMMAND [ARGS]...
```

## Description

Commands for managing and viewing W&B jobs

## Subcommands

| Command | Description |
|---------|-------------|
| [`wandb job create`](wandb-job-create) | Create a job from a source, without a wandb run. |
| [`wandb job describe`](wandb-job-describe) | Describe a launch job. Provide the launch job in the form of: entity/project/job-name:alias-or-version |
| [`wandb job list`](wandb-job-list) | List jobs in a project |
