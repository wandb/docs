## Usage

```bash
Usage: online [OPTIONS]
```

## Description

Re-enable cloud syncing for W&B runs.

    Clear the offline mode setting so subsequent runs in this directory
    sync data to the W&B cloud.

    Undo a previous call to `wandb offline`.
