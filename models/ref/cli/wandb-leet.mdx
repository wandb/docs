## Usage

```bash
leet [PATH] | COMMAND [ARGS]...
```

## Description

W&B LEET: the Lightweight Experiment Exploration Tool.

A terminal UI for viewing your W&B runs locally.



## Subcommands

| Command | Description |
|---------|-------------|
| `wandb leet config` | Edit LEET configuration. |
| `wandb leet run` | Launch the LEET TUI. |
| `wandb leet symon` | Launch the standalone system monitor. |
