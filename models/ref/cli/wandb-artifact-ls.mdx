---
title: wandb artifact-ls
---


List all artifacts in a W&B project.

    Display the latest version of each artifact collection in a
    project, showing type, last updated time, size, and name.

    The PATH is an entity/project string.

    Examples:

    List all artifacts in a project

    ```bash
    wandb artifact ls my-team/my-project
    ```

    List only artifacts of a specific type

    ```bash
    wandb artifact ls --type model my-team/my-project
    ```
    

## Usage

```shell
wandb artifact ls [OPTIONS] PATH
```


## Options

### `type`

Filter artifacts by type.                

| Flag | Default | Type | 
|------|---------|------|
| `-t, --type` | None | STRING |

### `help`

Show this message and exit.                

| Flag | Default | Type | 
|------|---------|------|
| `--help` | False | BOOL |
