## Usage

```bash
stop [OPTIONS]
```

## Description

Stop a running local W&B server.

    Stops the Docker container named `wandb-local` that was started
    by `wandb server start`. Requires Docker to be installed.

    For example, to stop the local W&B server:

```bash
wandb server stop
```
