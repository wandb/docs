Pull files from W&B

## Usage

```shell
wandb pull [OPTIONS] RUN
```


## Options

### `project`

The project you want to download.                

| Flag | Default | Type | 
|------|---------|------|
| `-p, --project` | None | STRING |

### `entity`

The entity to scope the listing to.                

| Flag | Default | Type | 
|------|---------|------|
| `-e, --entity` | models | STRING |

### `help`

Show this message and exit.                

| Flag | Default | Type | 
|------|---------|------|
| `--help` | False | BOOL |
