## Usage

```bash
beta [OPTIONS] COMMAND [ARGS]...
```

## Description

Beta versions of wandb CLI commands.

    These commands may change or even completely break in any release of wandb.

## Subcommands

| Command | Description |
|---------|-------------|
| `wandb beta core` | Manage a shared local wandb-core service for multi-process workloads. |
| `wandb beta leet` | W&B LEET: the Lightweight Experiment Exploration Tool. |
| `wandb beta sync` | Upload .wandb files specified by PATHS. |
