Run the W&B local sweep controller

## Usage

```shell
wandb controller [OPTIONS] SWEEP_ID
```


## Options

### `verbose`

Display verbose output                

| Flag | Default | Type | 
|------|---------|------|
| `--verbose` | False | BOOL |

### `help`

Show this message and exit.                

| Flag | Default | Type | 
|------|---------|------|
| `--help` | False | BOOL |
