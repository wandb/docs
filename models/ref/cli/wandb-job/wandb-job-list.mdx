---
title: wandb job-list
---


List jobs in a project

## Usage

```shell
wandb job list [OPTIONS]
```


## Options

| Flag | Type | Description |
|------|------|-------------|
| `-p, --project` | STRING | The project you want to list jobs from. **Default:** None |
| `-e, --entity` | STRING | The entity the jobs belong to **Default:** models |
| `--help` | BOOL | Show this message and exit. **Default:** False |
