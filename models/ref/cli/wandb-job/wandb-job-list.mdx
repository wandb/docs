## Usage

```bash
list [OPTIONS]
```

## Description

List jobs in a project.





## Options

| Flag | Type | Description |
|------|------|-------------|
| `--project, -p` | STR | The project you want to list jobs from. **Default**: None |
| `--entity, -e` | STR | The entity the jobs belong to. **Default**: models |
