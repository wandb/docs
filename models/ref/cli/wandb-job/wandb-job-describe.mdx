## Usage

```bash
describe [OPTIONS] JOB
```

## Description

Describe a launch job. Provide the launch job in the form
    of: entity/project/job-name:alias-or-version.



## Arguments

| Name | Default | Type |
|------|---------|------|
| `job` | STR | True |
