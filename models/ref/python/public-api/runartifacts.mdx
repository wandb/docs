## <kbd>class</kbd> `RunArtifacts`
An iterable collection of artifacts associated with a specific run. 


### <kbd>property</kbd> RunArtifacts.cursor

An opaque cursor that marks the start of the next page to fetch. 

This value may be saved and passed as `start=` to a later paginated query to resume iteration from where this paginator left off. 



**Returns:**
 - `str | None`: The cursor property value.
