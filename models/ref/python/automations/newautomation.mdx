**Args:**
 
 - `name` (str | None): The name of this automation.
 - `description` (str | None): An optional description of this automation.
 - `enabled` (bool | None): Whether this automation is enabled.  Only enabled automations will trigger.
 - `event` (Optional[Annotated]): The event that will trigger this automation.
