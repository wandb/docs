When you create an evaluation table, you specify which columns in your data correspond to the inputs, outputs, and scores. The following code snippet shows the high-level structure of an eval table object:

```python
import wandb

wandb.EvalTable(
    input_columns=["column1", "column2"],
    output_columns=["output_column1", "output_column2"],
    score_columns=["score_column1", "score_column2"],
)
```

Pass the names of the columns in your data to `input_columns`, `output_columns`, and `score_columns` to specify which columns correspond to the inputs, outputs, and scores in the eval table. You can pass multiple column names to each of these parameters.

```python
import pandas as pd
import wandb

df = pd.DataFrame(
    [
        {
            "image_id": "img_001",
            "true_label": "cat",
            "predicted_label": "cat",
            "confidence": 0.97,
            "correct": True,
        },
        {
            "image_id": "img_002",
            "true_label": "dog",
            "predicted_label": "cat",
            "confidence": 0.72,
            "correct": False,
        },
        {
            "image_id": "img_003",
            "true_label": "car",
            "predicted_label": "car",
            "confidence": 0.89,
            "correct": True,
        },
    ]
)

df["correct"] = df["correct"].astype(object)  # preserves native Python bools

with wandb.init(project="classifier-eval-table-demo") as run:
    eval_table = wandb.EvalTable(
        dataframe=df,
        input_columns=["image_id", "true_label"],
        output_columns=["predicted_label"],
        score_columns=["correct", "confidence"],
    )
    run.log({"validation_predictions_eval": eval_table})
```

</Info>

W&B displays the input columns (`input_columns`) in the **Inputs** section of the eval table, the output columns (`output_columns`) in the **Outputs** section, and the score columns (`score_columns`) in the **Scores** section. 

For example, consider the code example from the [Create an evaluation table](/models/evaltables/create-an-evaluation-table) page. In that example, the eval table is created from a pandas DataFrame with the following columns: `image_id`, `true_label`, `predicted_label`, `confidence`, and `correct`. The column names are passed as to `EvalTable` as follows:

```python
import wandb

with wandb.init(project="classifier-eval-table-demo") as run:
    eval_table = wandb.EvalTable(
        dataframe=df,
        input_columns=["image_id", "true_label"],
        output_columns=["predicted_label"],
        score_columns=["correct", "confidence"],
    )
    run.log({"validation_predictions_eval": eval_table})
```

## Filter data

Filter which data is displayed in the eval table by clicking the **Filter** button above the table.

From left to right, select the field you want to filter, the operator, and the value.

<Frame>
    <img src="/images/evaltables/filter.png" alt="Evaluation table filter" width="100%" />
</Frame>

Click on the `X` to remove a filter. You can add multiple filters to the eval table.

## Add or remove columns

Add or remove columns from the eval table by clicking the **Columns** button above the table. The available options depend on the columns you logged to the eval table.

To add a column, click on the column name to select it. To remove a column, click on the column name to deselect it.
