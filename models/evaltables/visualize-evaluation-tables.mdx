title: View evaluation tables
---

1. Navigate to your project's workspace.
2. Select the run that contains the evaluation table you want to view.
3. Select the **Charts** tab.

## Compare runs

1. Navigate to your project's workspace.
2. Scroll down to the **Evaluation Tables** panel.
3. Click the **+** button to add a run to compare.

W&B sets the run furthest to the left as the initial reference run. Hover your mouse over the run you want to set as the reference and select **Set as reference** from the dropdown menu.
