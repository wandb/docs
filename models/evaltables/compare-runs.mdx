---
title: Compare runs with eval tables
---

Use eval tables to compare multiple runs side-by-side. W&B shows the inputs, outputs, and scores for each run, as well as the differences in scores between the runs. 

To compare multiple runs:

1. Navigate to your project's workspace.
2. Scroll down to the **Evaluation Tables** panel.
3. Click the **+** button to add a run.

W&B groups columsns that are shared across the runs you select to compare. For example, if you select two runs that both have the same `input_columns`, W&B groups those columns together in the **Inputs** section of the eval table. The same is true for `output_columns` and `score_columns`.

## Reference run

When you compare multiple runs, W&B sets one of the runs as the reference run. The reference run is used to calculate the differences in scores between the runs. By default, W&B sets the run furthest to the left as the reference run. You can change the reference run by hovering over a run and selecting **Set as reference** from the dropdown menu.


## Aggregate scores

When you compare two or more runs, W&B automatically calculates an aggregate score between them. Based on value's type, W&B calculates the aggregate score as follows:

| Type of score | Aggregate score |
| --- | --- |
| Boolean | true_count and true_fraction |
| Numeric | mean |
| String / categorical | not averaged, so they don’t produce a scalar aggregate |
| None values | skipped before calculating the aggregate |

## Score deltas

When you compare two or more runs, W&B automatically calculates the delta between values passed to `score_columns` for each run you select to compare. The delta is calculated as the difference between the value in the reference run and the value in the other run.

For example, if `summer-butterfly-9` is the reference run, and `gentle-flower-8` is the other run, the delta for the `confidence` score is calculated as follows:

| `summer-butterfly-9` | `gentle-flower-8` | delta |
| --- | --- | --- |
| 0.43 | 0.62 | +0.19 |
| 0.92 | 0.86 | -0.06 |
| 0.97 | 0.85 | -0.12 |
