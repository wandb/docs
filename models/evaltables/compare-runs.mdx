---
title: Compare runs with evaluation tables
---

Use evaluation tables to compare multiple runs side-by-side. W&B shows the inputs, outputs, and scores for each run, as well as the differences in scores between the runs. 

To compare multiple runs:

1. Navigate to your project's workspace.
2. Scroll down to the **Evaluation Tables** panel.
3. Click the **+** button to add a run to compare.

Values passed to `input_columns`, `output_columns`, and `score_columns` (`wandb.EvalTable(input_columns=, output_columns=, score_columns=`)) are grouped together across runs you select.

## Reference run

When you compare multiple runs, W&B sets one of the runs as the reference run. The reference run is used to calculate the differences in scores between the runs. By default, W&B sets the run furthest to the left as the reference run. You can change the reference run by hovering over a run and selecting **Set as reference** from the dropdown menu.
