---
title: Artifact コレクションの削除
---

```python
"""
W&B からartifact collectionを削除する。
"""
import wandb

# W&B API を初期化する
api = wandb.Api()

# 名前とタイプでartifact collectionを削除する
# 名前の形式: <entity>/<project>/<run_path>
collection = api.artifact_collection(
    name="<entity>/<project>/<run_path>",
    type_name="<artifact_type>"
)

collection.delete()
```
