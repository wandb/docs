<div id="re-run-an-evaluation-job">
  ## 評価ジョブを再実行する
</div>

状況に応じて、評価ジョブを再実行したり、その設定を表示したりする方法がいくつかあります。

* 直前の評価ジョブをもう一度実行するには、[モデルを評価する](#evaluate-your-model) の手順に従います。宛先のプロジェクトを選択すると、前回選択したモデル アーティファクトの詳細とベンチマークが自動的に反映されます。必要に応じて調整を行い、評価ジョブを起動します。
* プロジェクトの **Runs** タブまたは run セレクターから評価ジョブを再実行するには、run 名にカーソルを合わせて再生アイコンをクリックします。ジョブ設定ドロワーが表示され、設定があらかじめ入力されています。必要に応じて設定を調整し、**Launch** をクリックします。
* 別のプロジェクトから評価ジョブを再実行するには、その設定をインポートします:
  1. [モデルを評価する](#evaluate-your-model) の手順に従います。宛先のプロジェクトを選択した後、**Import configuration** をクリックします。
  2. インポート元の評価ジョブを含むプロジェクトを選択し、次に評価ジョブの run を選択します。ジョブ設定ドロワーが表示され、設定があらかじめ入力されています。
  3. 必要に応じて設定を調整します。
  4. **Launch** をクリックします。
