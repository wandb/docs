<div id="review-evaluation-results">
  ## 評価結果を確認
</div>

宛先プロジェクトの Workspace にある W&amp;B Models で、評価ジョブの結果を確認します。

1. ページ上部の円形の矢印アイコンをクリックして最近の run モーダルを開きます。ここでは、評価ジョブがプロジェクト内の他の Runs と一緒に表示されます。評価ジョブに leaderboard がある場合は、**Leaderboard** をクリックして leaderboard を全画面で開くか、run 名をクリックしてプロジェクト内で単一 run 表示として開きます。
2. 評価ジョブのトレースは、Workspace の **Evaluations** セクション、または **Weave** サイドバーパネルの **Traces** タブで表示します。
3. **Overview** タブをクリックすると、評価ジョブの設定や summary メトリクスなどの詳細情報を表示できます。
4. **Logs** タブをクリックすると、評価ジョブのデバッグログを表示、検索、またはダウンロードできます。
5. **Files** タブをクリックすると、コード、ログ、設定、その他の出力ファイルを含む評価ジョブのファイルを参照、表示、またはダウンロードできます。

<div id="customize-a-leaderboard">
  ## リーダーボードをカスタマイズする
</div>

リーダーボードには、指定したプロジェクトに送信されたすべての評価ジョブの結果が表示されます。各行は、各評価ジョブの各ベンチマークに対応しています。各列には、評価ジョブの trace、入力値、出力値などの詳細が表示されます。リーダーボードの詳細については、[Weave の Leaderboards](/ja/weave/guides/core-types/leaderboards) を参照してください。

<Tip>リーダーボードの結果にフィードバックを送信するには、**フィードバック**列の絵文字アイコンまたはチャットアイコンをクリックします。</Tip>

* デフォルトでは、すべての評価ジョブが表示されます。左側の run selector を使用して、評価ジョブをフィルターまたは検索します。
* デフォルトでは、評価ジョブはグループ化されていません。1 つ以上の列でグループ化するには、**Group** アイコンをクリックします。グループの表示／非表示を切り替えたり、グループを展開してその runs を表示したりできます。
* デフォルトでは、すべての operation が表示されます。1 つの operation のみを表示するには、**All ops** をクリックして operation を選択します。
* 列で並べ替えるには、列見出しをクリックします。列の表示をカスタマイズするには、**Columns** をクリックします。
  * デフォルトでは、ヘッダーは 1 階層で整理されています。ヘッダーの深さを増やすと、関連するヘッダーをまとめて整理できます。
  * 個々の列を選択または選択解除して表示／非表示にしたり、クリック 1 回ですべての列を表示／非表示にしたりできます。
  * 列を固定すると、固定されていない列より前に表示できます。

<div id="export-a-leaderboard">
  ## リーダーボードをエクスポートする
</div>

リーダーボードをエクスポートするには、次の手順に従います。

1. **Columns** ボタンの近くにあるダウンロード アイコンをクリックします。
2. エクスポート サイズを最適化するため、デフォルトではトレースルートのみがエクスポートされます。完全なトレースをエクスポートするには、**Trace roots only** をオフにします。
3. エクスポート サイズを最適化するため、デフォルトではフィードバックとコストはエクスポートされません。これらをエクスポートに含めるには、**フィードバック** または **Costs** を切り替えます。
4. デフォルトでは、エクスポートは JSONL 形式です。形式をカスタマイズするには、**Export to file** をクリックして形式を選択します。
5. ブラウザーでリーダーボードをエクスポートするには、**Export** をクリックします。
6. リーダーボードをプログラムでエクスポートするには、**Python** または **cURL** を選択し、**Copy** をクリックしてスクリプトまたはコマンドを実行します。
