---
title: Runs をピン留めする（簡易版）
---

最大 6 件の run をピン留めして、Workspace の上部に固定表示できます。ベースライン run がある場合は、ベースラインが暗黙的にピン留めされるため、ピン留めできる run は最大 5 件です。ピン留めした run は、ほかの run に並べ替えやフィルターを適用しても表示されたままになります。ピン留めした run は、円形のピンアイコンとともに run selector の上部に表示され、視覚的な区切り線でほかの run と分けて表示されます。

run をピン留めするには、次の手順に従います。

1. Workspace にアクセスします。
2. run selector または Runs table で、ピン留めする run を検索します。
3. `...` アクションメニューをクリックし、**Pin run** を選択します。
