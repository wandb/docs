---
title: Runs のピン留め（簡易版）
---

最大 6 個まで run をピン留めして、ワークスペースの上部に表示しておくことで、簡単にアクセスできるようにできます。ベースライン run がある場合は、そのベースラインが暗黙的にピン留めされるため、ピン留めできる run は最大 5 個です。ピン留めされた run は、他の run に適用されたソートやフィルターに関係なく表示され続けます。ピン留めされた run は、円形のピンアイコン付きで run セレクターの上部に表示され、他の run とは視覚的な区切りによって分けられます。

run をピン留めするには:

1. ワークスペースに移動します。
2. run セレクターまたは runs テーブルで、ピン留めしたい run を探します。
3. アクション `...` メニューをクリックして、**Pin run** を選択します。
