---
title: Runs のピン留め（簡易版）
---

最大 6 個まで run をピン留めして、Workspace の上部に表示しておくことで、すぐにアクセスできるようにできます。ベースライン run がある場合は、そのベースラインが暗黙的にピン留めされるため、ピン留めできる run は最大 5 個です。ピン留めされた run は、他の run に適用された並べ替えやフィルターに関係なく表示されたままになります。ピン留めされた run は、円形のピンアイコン付きで run selector の上部に表示され、他の run とは視覚的な区切り線で分けられます。

run をピン留めするには:

1. Workspace にアクセスします。
2. run selector または Runs table で、ピン留めしたい run を探します。
3. `...` のアクションメニューをクリックし、**Pin run** を選択します。
