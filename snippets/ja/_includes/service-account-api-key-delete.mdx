---
title: Service Account API キーの削除
---

組織またはチームのサービスアカウントが所有している API キーを削除するには、次の手順に従います。

1. [Organization settings](https://wandb.ai/account-settings/) に移動し、**API Keys** をクリックします。
2. API キーを探します。リストには、組織およびチームのサービスアカウントが所有するすべての API キーが含まれます。キー名または ID で検索またはフィルタリングでき、任意の列でソートできます。
3. 削除ボタンをクリックします。
