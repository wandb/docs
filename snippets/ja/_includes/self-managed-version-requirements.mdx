---
title: Self Managed のバージョン要件
---

| ソフトウェア | 最低バージョン                              |
| ------------ | -------------------------------------------- |
| Kubernetes   | v1.32 以上（[Supported Kubernetes versions](https://kubernetes.io/releases/patch-releases/)）  |
| Helm         | v3.x                                         |
| MySQL        | v8.0.x が必須で、v8.0.32 以上が必要です。v8.0.44 以上を推奨します。<br/>Aurora MySQL 3.x リリースは v3.05.2 以上である必要があります  |
| Redis        | v7.x                                         |
