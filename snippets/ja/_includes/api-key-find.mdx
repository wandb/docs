---
title: API キーの検索
---

<Note>
  **Dedicated Cloud** と **Self-Managed** の場合:

  * 組織管理者は、組織内のすべてのユーザーおよびサービスアカウントの API キーを検索または一覧表示できます。
  * チーム管理者は、自分が管理するチーム内のサービスアカウントの API キーを検索または一覧表示できます。
  * 管理者ではないユーザーは、自分の API キーを検索または一覧表示できます。
</Note>

API キーを見つけるには、詳細に応じて **Personal API key** または **Service Account API key** タブを選択します。

<Tabs>
  <Tab title="Personal API key">
    自分のユーザー ID に紐づく個人用 API キーを見つけるには:

    1. W&amp;B にログインし、ユーザープロフィールアイコンをクリックしてから **User Settings** をクリックします。
    2. **API Keys** セクションまでスクロールします。
  </Tab>

  <Tab title="Service account API key">
    組織またはチームのサービスアカウントが所有する API キーを見つけるには:

    1. 組織設定の **API Keys** タブにアクセスします。
    2. 一覧から API キーを検索します。オーナー、キー名、キー ID で一覧を検索またはフィルターできます。
  </Tab>
</Tabs>

API キーの表には、識別用としてキー ID (各 API キーの先頭部分) が表示されます。完全な secret API キーは、作成時に一度だけ表示されます。既存のキーを使用する必要があっても、完全な secret が保存されていない場合は、新しい API キーを作成する必要があります。
