---
title: APIキーを検索
---

<Note>
  **専用クラウド** と **セルフマネージド** では：

  * 組織管理者は、組織内のすべてのユーザーおよびサービスアカウントのAPIキーを検索または一覧表示できます。
  * チーム管理者は、自身が管理するチーム内のサービスアカウントのAPIキーを検索または一覧表示できます。
  * 管理者ではないユーザーは、自身のAPIキーを検索または一覧表示できます。
</Note>

APIキーを検索するには、該当する **個人用APIキー** または **サービスアカウントAPIキー** タブを選択して詳細を確認します。

<Tabs>
  <Tab title="個人用APIキー">
    自分のユーザーIDに紐づく個人用APIキーを検索するには：

    1. W&amp;B にログインし、ユーザープロフィールアイコンをクリックしてから、**ユーザー設定** をクリックします。
    2. **APIキー** セクションまでスクロールします。
  </Tab>

  <Tab title="サービスアカウントAPIキー">
    組織またはチームのサービスアカウントに紐づくAPIキーを検索するには：

    1. 組織設定の **APIキー** タブにアクセスします。
    2. 一覧からAPIキーを検索します。所有者、キー名、またはキーIDで一覧を検索またはフィルターできます。
  </Tab>
</Tabs>

APIキーの表には、識別用にキーID (各APIキーの先頭部分) が表示されます。完全なシークレットAPIキーが表示されるのは、作成時の一度だけです。既存のキーを使用する必要があっても、完全なシークレットを保存していない場合は、新しいAPIキーを作成する必要があります。
