<div id="configure-wb-to-use-your-bucket">
  ### W&amp;B でバケットを使用するように設定する
</div>

バケットのプロビジョニングが完了したら、W&amp;B がそのバケットを使用するように設定します：

* **Self-Managed デプロイ**: Helm の values の設定については [Operator 設定ガイド](/ja/platform/hosting/self-managed/operator#object-storage-bucket) を参照してください
* **Dedicated Cloud**: [System Console](/ja/platform/hosting/iam/sso#system-console) から設定します
* **Multi-tenant Cloud**: チームを作成または編集する際に設定します
