<div id="configure-wb-to-use-your-bucket">
  ### W&amp;B が使用するバケットを設定する
</div>

バケットがプロビジョニングされたら、W&amp;B がそのバケットを使用するように設定します。

* **セルフマネージドのデプロイ**: Helm values の設定については、[Operator configuration guide](/ja/platform/hosting/self-managed/operator#object-storage-bucket)を参照してください
* **専用クラウド**: [System Console](/ja/platform/hosting/iam/sso#system-console)から設定します
* **Multi-tenant Cloud**: チームの作成時または編集時に設定します
