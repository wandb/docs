<div id="configure-wb-to-use-your-bucket">
  ### W&amp;B で버킷を使用するように設定する
</div>

버킷のプロビジョニングが完了したら、W&amp;B がその버킷を使用するように設定します:

* **Self-Managed デプロイ**: Helm values の設定については [Operator configuration guide](/ja/platform/hosting/self-managed/operator#object-storage-bucket) を参照してください
* **専用クラウド**: [System Console](/ja/platform/hosting/iam/sso#system-console) から設定します
* **Multi-tenant Cloud**: チームを作成または編集するときに設定します
