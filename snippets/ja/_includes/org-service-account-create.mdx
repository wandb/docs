---
title: Org サービスアカウントの作成
---

新しい組織スコープのサービスアカウントと API キーを作成するには、次の手順を実行します。

1. W&amp;B にログインし、ユーザープロフィールアイコンをクリックしてから、以下を実行します。
    - **Dedicated Cloud** または **Self-Managed** の場合: **Organization Dashboard** をクリックし、次に **Service Accounts** をクリックします。
    - **Multi-tenant Cloud** の場合: **Service Accounts** をクリックします。
2. **Create service account** をクリックします。
3. 名前を入力し、デフォルトのチームを選択します。
4. **Create** をクリックします。
5. 作成したばかりのサービスアカウントを見つけます。
6. アクションメニュー（`...`）をクリックし、**Create API key** をクリックします。
7. API キーの名前を入力し、**Create** をクリックします。
8. API キーをコピーして安全な場所に保管します。
9. **Done** をクリックします。
