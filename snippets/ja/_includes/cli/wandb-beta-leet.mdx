LEET は、W&amp;B run の統計、メトリクス、およびシステムの正常性を表示するためのローカル端末 UI です。LEET は、SSH や `tmux` などのツールを使用してリモートシステムや HPC 環境で作業するユーザー向けに設計された、キーボード操作中心のインターフェースです。

![LEET 端末 UI で run を表示しているスクリーンショット](/images/leet/leet-tui.png)

主な機能は次のとおりです。

* **キーボード優先の navigation**: すべての機能にキーボードショートカットでアクセスできます
* **ライブ更新**: run の進行に合わせて `.wandb` ファイルをリアルタイムで監視します
* **メトリクスのフィルタリング**: glob パターンまたは部分文字列一致を使用してメトリクスをフィルターします
* **スマートズーム**: マウスホイールのズームで最新のデータポイントを表示したままにします
* **カスタマイズ可能なグリッド**: メトリクス表示とシステム表示の行数と列数を設定できます
* **人が読みやすい形式**: システムメトリクスはわかりやすい単位で表示されます (例: &quot;GPU Utilization (%)&quot;、&quot;RAM Used (GB)&quot;) 

LEET は 3 つのパネルからなる Layout を提供します。

* **左サイドバー**: 環境変数、設定、summary 統計を表示する Run Overview
* **中央パネル**: ログされたメトリクスを点字スタイルの折れ線チャートで表示するメトリクスグリッド
* **右サイドバー**: GPU utilization、RAM、CPU、そのほかのシステム正常性指標を表示する System Metrics
