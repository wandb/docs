LEET は、W&amp;B run の統計情報、メトリクス、およびシステムヘルスを表示するためのローカルターミナル UI です。LEET はキーボード操作を前提としたインターフェースで、SSH や `tmux` のようなツールを使ってリモートシステムや HPC 環境で作業するユーザー向けに設計されています。

![Screenshot demonstrating viewing a run in the LEET terminal UI](/images/leet/leet-tui.png)

主な機能は次のとおりです。

* **キーボード優先のナビゲーション**: すべての機能にキーボードショートカットでアクセス可能
* **ライブ更新**: run の進行に合わせて `.wandb` ファイルをリアルタイムに監視
* **メトリクスのフィルタリング**: グロブパターンや部分一致を使ってメトリクスをフィルタリング
* **スマートズーム**: マウスホイールでズームしても最新のデータポイントが常に表示されるように維持
* **カスタマイズ可能なグリッド**: メトリクスおよびシステム表示用の行数と列数を設定可能
* **人間が読みやすいフォーマット**: システムメトリクスをわかりやすい単位（例: 「GPU Utilization (%)」「RAM Used (GB)」）で表示

LEET は 3 つのペインからなるレイアウトを提供します。

* **左サイドバー**: 環境変数、設定、および要約統計情報を表示する Run Overview
* **中央ペイン**: ログされたメトリクスを点字風のラインチャートで表示するメトリクスグリッド
* **右サイドバー**: GPU 使用率、RAM、CPU などのシステムヘルス指標を表示する System Metrics
