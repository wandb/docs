LEET は、W&amp;B run の統計、メトリクス、システムの健全性を表示するためのローカルのターミナル UI です。LEET は、SSH や `tmux` などのツールを使ってリモートシステムや HPC 環境で作業するユーザー向けに設計された、キーボード操作中心のインターフェースです。



主な機能は次のとおりです。

* **キーボードファーストのナビゲーション**: すべての機能にキーボードショートカットでアクセス可能
* **ライブ更新**: run の進行に合わせて `.wandb` ファイルをリアルタイムで監視
* **メトリクスのフィルタリング**: glob パターンまたは部分文字列一致を使用してメトリクスをフィルター
* **スマートズーム**: マウスホイールでズームしても最新のデータポイントを表示範囲内に維持
* **カスタマイズ可能なグリッド**: メトリクス表示とシステム表示の行数と列数を設定
* **読みやすい形式**: システムメトリクスをわかりやすい単位で表示 (例: &quot;GPU 使用率 (%)&quot;、&quot;使用 RAM (GB)&quot;) 

LEET は 3 パネルの Layout を提供します。

* **左サイドバー**: 環境変数、設定、summary 統計情報を表示する run の概要
* **中央パネル**: ログされたメトリクスを点字スタイルの折れ線 chart で表示するメトリクスグリッド
* **右サイドバー**: GPU 使用率、RAM、CPU、その他のシステム健全性指標を表示するシステムメトリクス
