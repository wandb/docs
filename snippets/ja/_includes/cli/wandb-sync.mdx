---
title: Wandb Sync
---

W&amp;B の run データをクラウドに同期します。

PATH が指定されている場合は、指定したパスで見つかった run を同期します。パスが指定されていない場合は、まず `./wandb` を検索し、その後 `wandb/` サブディレクトリを検索します。

特定の run を同期するには、次のように実行します:

wandb sync ./wandb/run-20250813&#95;124246-n67z9ude

または、同等のコマンドとして次のように実行します:

wandb sync ./wandb/run-20250813&#95;124246-n67z9ude/run-n67z9ude.wandb
