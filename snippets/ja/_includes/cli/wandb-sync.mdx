---
title: Wandb Sync
---

W&B の run データをクラウドに同期します。

PATH が指定されている場合は、そのパスで見つかった run を同期します。パスが指定されていない場合は、まず `./wandb` を検索し、次に `wandb/` サブディレクトリを検索します。

特定の run を同期するには:

wandb sync ./wandb/run-20250813_124246-n67z9ude

同等のコマンドは次のとおりです:

wandb sync ./wandb/run-20250813_124246-n67z9ude/run-n67z9ude.wandb
