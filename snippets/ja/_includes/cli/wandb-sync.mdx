---
title: Wandb Sync
---

W&amp;Bのrunデータをクラウドに同期します。

PATH が指定されている場合は、そのパスで見つかった run を同期します。パスが指定されていない場合は、まず `./wandb` を探し、次に `wandb/` サブディレクトリを探します。

特定の run を同期するには、次を実行します。

wandb sync ./wandb/run-20250813&#95;124246-n67z9ude

または同様に、次のように指定することもできます。

wandb sync ./wandb/run-20250813&#95;124246-n67z9ude/run-n67z9ude.wandb
