---
title: Wandb Sync
---

W&B の run データをクラウドと同期します。

PATH が指定された場合は、そのパスで見つかった run を同期します。パスが指定されていない場合は、まず `./wandb` を検索し、その後に `wandb/` サブディレクトリを検索します。

特定の run を同期するには、次のように実行します:

wandb sync ./wandb/run-20250813_124246-n67z9ude

または同等のコマンドとして、次のように実行します:

wandb sync ./wandb/run-20250813_124246-n67z9ude/run-n67z9ude.wandb
