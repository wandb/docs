---
title: Wandb Verify
---

ローカルの W&B インスタンスをチェックおよび検証します。W&B は次の点をチェックします：

ホストが `api.wandb.ai` ではないことを確認します（ホストチェック）。

指定された API キーを使用して、ユーザーが正しくログインしているかを検証します（ログインチェック）。

リクエストが HTTPS 経由で行われていることを確認します（セキュアリクエスト）。

オブジェクトストアの CORS（Cross-Origin Resource Sharing）設定を検証します（CORS 設定）。

メトリクスを記録し、ファイルを保存およびダウンロードして、run が正しく記録されアクセス可能であるかを確認します（run チェック）。

Artifacts を保存およびダウンロードして、 Artifacts の保存および取得システムが期待どおりに動作しているかを検証します（artifact チェック）。

署名付き URL アップロードを処理できることを確認するためにファイルをアップロードし、GraphQL エンドポイントをテストします（GraphQL PUT チェック）。

プロキシ経由で大きなペイロードを送信できるかどうかを確認します（大きなペイロードチェック）。

インストールされている W&B パッケージのバージョンが最新であり、サーバーと互換性があることを検証します（W&B バージョンチェック）。

sweep を作成して実行し、sweep 機能が正しく動作していることを確認します（Sweeps チェック）。
