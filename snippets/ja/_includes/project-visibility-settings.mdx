---
title: プロジェクトの可視性設定
---

1. W&amp;B App で、プロジェクト内の任意のページから、左側のナビゲーションにある**Overview**をクリックします。
2. 右上の **Edit** をクリックします。
3. **Project visibility** の新しい値を選択します。

   * **Team** (デフォルト) : あなたのチームだけがこのプロジェクトを表示および編集できます。
   * **Restricted**: 招待されたメンバーだけがこのプロジェクトにアクセスでき、公開アクセスは無効になります。
   * **Open**: 誰でも run を送信したりレポートを作成したりできますが、編集できるのはあなたのチームだけです。教室での利用、公開ベンチマーク競技、またはそのほかの一時的な用途にのみ適しています。
   * **Public**: 誰でもこのプロジェクトを表示できますが、編集できるのはあなたのチームだけです。

     <Note>
       W&amp;B の管理者が **Public** の可視性を無効にしている場合、これを選択することはできません。代わりに、閲覧専用の[W&amp;B Report](/ja/models/reports/collaborate-on-reports#share-a-report)を共有するか、W&amp;B 組織の管理者に問い合わせてください。
     </Note>
4. **Save** をクリックします。

プロジェクトのプライバシー設定をより厳しくした場合、プロジェクトへのアクセス権を戻すには、個々のユーザーを再度招待する必要があることがあります。
