<div id="general">
  ### 一般
</div>

| **キー**        | **説明**           |
| :------------ | :--------------- |
| `h`, `?`      | このヘルプ画面の表示を切り替える |
| `q`, `ctrl+c` | 終了する             |
| `alt+r`       | run データを再読み込みする  |

<div id="panels">
  ### パネル
</div>

| **Key** | **説明**                            |
| :------ | :-------------------------------- |
| `[`     | run の概要を表示する左サイドバーの表示/非表示を切り替える   |
| `]`     | システムメトリクスを表示する右サイドバーの表示/非表示を切り替える |
| `l`     | run コンソールのログペインの表示/非表示を切り替える      |
| `s`     | システムメトリクスペインの表示/非表示を切り替える         |

<div id="navigation">
  ### ナビゲーション
</div>

| **キー**                | **説明**            |
| :-------------------- | :---------------- |
| `N`, `pgup`           | グラフのページ間を移動       |
| `n`, `pgdown`         | グラフのページ間を移動       |
| `alt+N`, `alt+pgup`   | システムメトリクスのページ間を移動 |
| `alt+n`, `alt+pgdown` | システムメトリクスのページ間を移動 |

<div id="charts">
  ### チャート
</div>

| **キー**   | **説明**                                      |
| :------- | :------------------------------------------ |
| `/`      | パターンでメトリクスをフィルターします (glob または部分文字列一致をサポート)  |
| `ctrl+l` | 現在のフィルターをクリア                                |
| `\`      | システムメトリクスのフィルタリングを切り替え                      |

<div id="run-overview">
  ### run の概要
</div>

| **キー**   | **説明**                                                   |
| :------- | :------------------------------------------------------- |
| `o`      | 概要項目をフィルター (環境/設定/summary セクションでは `@e`、`@c`、`@s` を使用可能)  |
| `ctrl+k` | 概要フィルターをクリア                                              |

<div id="configuration">
  ### 設定
</div>

| **キー** | **説明**                    |
| :----- | :------------------------ |
| `c`    | メトリクスグリッドの列数を設定           |
| `r`    | メトリクスグリッドの行数を設定           |
| `C`    | システムグリッドの列数を設定 (Shift+c)  |
| `R`    | システムグリッドの行数を設定 (Shift+r)  |

<div id="run-overview-navigation-when-sidebar-open">
  ### run の概要のナビゲーション (サイドバーが開いている場合)
</div>

| **キー**             | **説明**                         |
| :----------------- | :----------------------------- |
| `up`, `down`       | セクション内の項目を移動                   |
| `tab`, `shift+tab` | セクション間を切り替え                    |
| `left`, `right`    | セクション内のページを移動                  |
| `@e`, `@c`, `@s`   | 特定のセクションをフィルターする (環境/設定/summary)  |

<div id="mouse">
  ### マウス
</div>

| **操作**               | **説明**              |
| :------------------- | :------------------ |
| `mouse wheel`        | フォーカスされているグラフを拡大/縮小 |
| `shift+mouse select` | テキストを選択             |
