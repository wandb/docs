---
title: Leet のキーボードショートカット
---

<div id="general">
  ### 一般
</div>

| **キー** | **説明** |
| :--- | :--- |
| `h`, `?` | ヘルプ画面の表示／非表示を切り替える |
| `q`, `ctrl+c` | 終了する |
| `alt+r` | run データを再読み込みする |

<div id="panels">
  ### パネル
</div>

| **キー** | **説明** |
| :--- | :--- |
| `[` | run 概要の左サイドバーを表示／非表示に切り替え |
| `]` | システムメトリクスの右サイドバーを表示／非表示に切り替え |

<div id="navigation">
  ### ナビゲーション
</div>

| **キー** | **説明** |
| :--- | :--- |
| `N`, `pgup` | チャートページを切り替えます |
| `n`, `pgdown` | チャートページを切り替えます |
| `alt+N`, `alt+pgup` | システムメトリクスのページを切り替えます |
| `alt+n`, `alt+pgdown` | システムメトリクスのページを切り替えます |

<div id="charts">
  ### チャート
</div>

| **キー** | **説明** |
| :--- | :--- |
| `/` | パターンでメトリクスをフィルタリング（glob または部分一致に対応） |
| `ctrl+l` | 現在のフィルターをクリア |

<div id="run-overview">
  ### Run 概要
</div>

| **Key** | **説明** |
| :--- | :--- |
| `o` | 概要の項目をフィルター（environment/config/summary セクション向けに `@e`、`@c`、`@s` をサポート） |
| `ctrl+k` | 概要フィルターをクリア |

<div id="configuration">
  ### 設定
</div>

| **キー** | **説明** |
| :--- | :--- |
| `c` | メトリクスグリッドの列数を設定 |
| `r` | メトリクスグリッドの行数を設定 |
| `C` | システムグリッドの列数を設定 (Shift+c) |
| `R` | システムグリッドの行数を設定 (Shift+r) |

<div id="run-overview-navigation-when-sidebar-open">
  ### Run 概要ナビゲーション（サイドバー表示時）
</div>

| **Key** | **Description** |
| :--- | :--- |
| `up`, `down` | セクション内の項目を移動 |
| `tab`, `shift+tab` | セクション間を切り替え |
| `left`, `right` | セクション内のページを切り替え |
| `@e`, `@c`, `@s` | 特定のセクションを絞り込み（environment/config/summary） |

<div id="mouse">
  ### マウス
</div>

| **操作** | **説明** |
| :--- | :--- |
| `mouse wheel` | フォーカス中のチャートを拡大／縮小 |
| `shift+mouse select` | テキストを選択 |
