---
title: Leet のキーボードショートカット
---

<div id="general">
  ### 一般
</div>

| **キー** | **説明** |
| :--- | :--- |
| `h`, `?` | このヘルプ画面の表示を切り替えます |
| `q`, `ctrl+c` | 終了します |
| `alt+r` | run データを再読み込みします |

<div id="panels">
  ### パネル
</div>

| **キー** | **説明** |
| :--- | :--- |
| `[` | run 概要の左サイドバーの表示／非表示を切り替え |
| `]` | システム メトリクスの右サイドバーの表示／非表示を切り替え |

<div id="navigation">
  ### ナビゲーション
</div>

| **キー** | **説明** |
| :--- | :--- |
| `N`, `pgup` | チャート ページを切り替える |
| `n`, `pgdown` | チャート ページを切り替える |
| `alt+N`, `alt+pgup` | システム メトリクス ページを切り替える |
| `alt+n`, `alt+pgdown` | システム メトリクス ページを切り替える |

<div id="charts">
  ### チャート
</div>

| **キー** | **説明** |
| :--- | :--- |
| `/` | パターンでメトリクスを絞り込み（glob または部分一致に対応） |
| `ctrl+l` | アクティブなフィルターをクリア |

<div id="run-overview">
  ### Run 概要
</div>

| **キー** | **説明** |
| :--- | :--- |
| `o` | 概要項目をフィルター（environment/config/summary セクション向けの `@e`、`@c`、`@s` に対応） |
| `ctrl+k` | 概要フィルターをクリア |

<div id="configuration">
  ### 設定
</div>

| **キー** | **説明** |
| :--- | :--- |
| `c` | メトリクス グリッドの列数を設定 |
| `r` | メトリクス グリッドの行数を設定 |
| `C` | システム グリッドの列数を設定 (Shift+c) |
| `R` | システム グリッドの行数を設定 (Shift+r) |

<div id="run-overview-navigation-when-sidebar-open">
  ### Run Overview ナビゲーション（サイドバーを開いているとき）
</div>

| **キー** | **説明** |
| :--- | :--- |
| `up`, `down` | セクション内の項目を移動 |
| `tab`, `shift+tab` | セクション間を切り替え |
| `left`, `right` | セクション内のページを移動 |
| `@e`, `@c`, `@s` | 特定のセクション（environment / config / summary）を絞り込み |

<div id="mouse">
  ### マウス
</div>

| **アクション** | **説明** |
| :--- | :--- |
| `mouse wheel` | フォーカスしているチャートをズームイン／ズームアウト |
| `shift+mouse select` | テキストを選択 |
