<div id="workspace">
  ### Workspace
</div>

| キー                               | 操作                                 |
| -------------------------------- | ---------------------------------- |
| `h`, `?`                         | ヘルプを切り替えます。                        |
| `q`, `ctrl+c`                    | 終了します。                             |
| `alt+r`                          | LEET を再起動します。                      |
| `esc`                            | Runs リストにフォーカスを移します。               |
| `enter`                          | ハイライトされている run を単一 run ビューで開きます。   |
| `1`                              | メトリクスグリッドを切り替えます。                  |
| `[`                              | Runs サイドバーを切り替えます。                 |
| `2`                              | システムメトリクスペインを切り替えます。               |
| `]`                              | run 概要サイドバーを切り替えます。                |
| `3`                              | メディアペインを切り替えます。                    |
| `4`                              | コンソールログペインを切り替えます。                 |
| `f`                              | 名またはメタデータで Runs をフィルターします。         |
| `ctrl+f`                         | Runs フィルターをクリアします。                 |
| `space`                          | run を選択または選択解除します。                 |
| `p`                              | run をピン留めまたはピン留め解除します。             |
| `/`                              | メトリクスをフィルターします。                    |
| `\`                              | システムメトリクスをフィルターします。                |
| `ctrl+/`, `ctrl+l`               | メトリクスフィルターをクリアします。                 |
| `ctrl+\`                         | システムメトリクスフィルターをクリアします。             |
| `o`                              | run 概要の項目をフィルターします。                |
| `ctrl+o`                         | run 概要フィルターをクリアします。                |
| `y`                              | フォーカス中のチャートのモードを順に切り替えます。          |
| `c`                              | フォーカス中のグリッドの列を設定します。               |
| `r`                              | フォーカス中のグリッドの行を設定します。               |
| `tab`, `shift+tab`               | 表示されているペイン間でフォーカスを切り替えます。          |
| `w`, `a`, `s`, `d` or arrow keys | フォーカス中のペイン内を移動します。                 |
| `N`, `pgup`                      | 前のページに移動します。                       |
| `n`, `pgdown`                    | 次のページに移動します。                       |
| `home`                           | 最初の項目、最初のページ、または最初のメディアフレームに移動します。 |
| `end`                            | 最後の項目、最後のページ、または最新のメディアフレームに移動します。 |

<div id="symon">
  ### SYMON
</div>

| キー                         | 操作                        |
| -------------------------- | ------------------------- |
| `h`, `?`                   | ヘルプの表示を切り替えます。            |
| `q`, `ctrl+c`              | 終了します。                    |
| `alt+r`                    | 再起動します。                   |
| `y`                        | フォーカス中のチャートのモードを順に切り替えます。 |
| `\`                        | システムメトリクスをフィルタリングします。     |
| `ctrl+\`                   | システムメトリクスのフィルターをクリアします。   |
| `c`, `C`                   | グリッドの列数を設定します。            |
| `r`, `R`                   | グリッドの行数を設定します。            |
| `w`, `a`, `s`, `d` または矢印キー | チャートのフォーカスを移動します。         |
| `N`, `pgup`                | 前のチャートページに移動します。          |
| `n`, `pgdown`              | 次のチャートページに移動します。          |
| `home`                     | 最初のチャートページに移動します。         |
| `end`                      | 最後のチャートページに移動します。         |

<div id="mouse">
  ### マウス
</div>

| 入力                      | 操作                           |
| ----------------------- | ---------------------------- |
| クリック                    | チャートにフォーカスするか、メディアタイルを選択します。 |
| ホイール                    | フォーカスしたチャートをズームします。          |
| 右クリックしてドラッグ             | チャート上の最も近いポイントを確認します。        |
| `alt` を押しながら右クリックしてドラッグ | 表示中のすべてのチャートを同期して確認します。      |
| `shift` を押しながらドラッグ      | ターミナルのテキストを選択します。            |
