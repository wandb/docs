---
title: セルフマネージド環境のインストール検証
---

インストールを検証するには、W&amp;B は [W&amp;B CLI](/ja/models/ref/cli/) の使用を推奨します。verify コマンドは、すべてのコンポーネントと設定を確認するための複数のテストを実行します。

<Note>
  この step では、最初の管理者ユーザーアカウントがブラウザーで作成されていることを想定しています。
</Note>

次の step に従ってインストールを検証します。

1. W&amp;B CLI をインストールします：

```bash
pip install wandb
```

2. W&amp;B にログインします：

```bash
wandb login --host=https://YOUR_DNS_DOMAIN
```

例えば：

```bash
wandb login --host=https://wandb.company-name.com
```

3. インストールを確認する：

```bash
wandb verify
```

インストールが正常に完了し、W&amp;B のデプロイが問題なく動作している場合、次のような出力が表示されます。
