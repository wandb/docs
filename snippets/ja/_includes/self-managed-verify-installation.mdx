---
title: Self Managed インストールを確認する
---

インストールを確認するには、W&amp;B では [W&amp;B CLI](/ja/models/ref/cli/) の使用を推奨しています。`wandb verify` コマンドは、コンポーネントと設定が想定どおりに動作することを確認するテストを実行します。

<Note>
  この手順では、最初の管理者ユーザーアカウントをブラウザで作成することを前提としています。
</Note>

インストールを確認するには:

1. W&amp;B CLI をインストールします:

   ```bash
   pip install wandb
   ```

2. W&amp;B にログインします:

   ```bash
   wandb login --host=https://YOUR_DNS_DOMAIN
   ```

   例えば:

   ```bash
   wandb login --host=https://wandb.company-name.com
   ```

3. インストールを確認する:

   ```bash
   wandb verify
   ```

コマンドの実行後、インストールが成功すると、次の出力が表示されます:
