---
title: Self Managed インストールを確認する
---

インストールを確認するには、W&amp;B では [W&amp;B CLI](/ja/models/ref/cli/) の使用を推奨しています。`verify` command は、すべてのコンポーネントと設定を検証する複数のテストを実行します。

<Note>
  この step では、最初の管理者ユーザーアカウントがブラウザで作成済みであることを前提としています。
</Note>

インストールを確認するには、次の step に従います。

1. W&amp;B CLI をインストールします:

```bash
pip install wandb
```

2. W&amp;B にログインします:

```bash
wandb login --host=https://YOUR_DNS_DOMAIN
```

例えば、

```bash
wandb login --host=https://wandb.company-name.com
```

3. インストールを確認する:

```bash
wandb verify
```
