---
title: 自己管理環境でのインストールの検証
---

インストールを検証するには、W&amp;B では [W&amp;B CLI](/ja/models/ref/cli/) の使用を推奨します。`verify` コマンドは、すべてのコンポーネントと設定が正しく動作しているか確認するために、いくつかのテストを実行します。

<Note>
  この手順では、最初の管理者ユーザーアカウントがブラウザーで作成されていることを前提とします。
</Note>

次の手順に従ってインストールを検証します。

1. W&amp;B CLI をインストールします。

```bash
pip install wandb
```

2. W&amp;B にログインします：

```bash
wandb login --host=https://YOUR_DNS_DOMAIN
```

例えば、

```bash
wandb login --host=https://wandb.company-name.com
```

3. インストールを確認する：

```bash
wandb verify
```

インストールが正常に完了し、W&amp;B デプロイ環境が問題なく動作している場合は、次のような出力が表示されます。
