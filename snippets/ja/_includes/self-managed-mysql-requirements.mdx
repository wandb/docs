---
title: 自己管理型 MySQL の要件
---

W&amp;B には外部 MySQL データベースが必要です。

本番環境では、W&amp;B はマネージドデータベースサービスの利用を強く推奨します。

* [AWS RDS Aurora MySQL](https://aws.amazon.com/rds/aurora/)
* [Google Cloud SQL for MySQL](https://cloud.google.com/sql/mysql)
* [Azure Database for MySQL](https://azure.microsoft.com/en-us/products/mysql/)

マネージドデータベースサービスは、自動バックアップ、監視、高可用性、パッチ適用などを提供し、運用上のオーバーヘッドを削減します。

MySQL の完全な要件 (推奨されるサイズと設定パラメーターを含む) については、[reference architecture](/ja/platform/hosting/self-managed/ref-arch/#mysql) を参照してください。データベース作成用の SQL については、[bare-metal guide](/ja/platform/hosting/self-managed/bare-metal/#mysql-database) を参照してください。デプロイ環境のデータベース設定に関する質問がある場合は、[support](mailto:support@wandb.com) もしくは担当の AISE までお問い合わせください。
