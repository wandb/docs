---
title: セルフマネージド MySQL 要件
---

W&amp;B では外部 MySQL データベースが必要です。

本番では、W&amp;B はマネージドデータベースサービスの使用を強く推奨します。

* [AWS RDS Aurora MySQL](https://aws.amazon.com/rds/aurora/)
* [Google Cloud SQL for MySQL](https://cloud.google.com/sql/mysql)
* [Azure Database for MySQL](https://azure.microsoft.com/en-us/products/mysql/)

マネージドデータベースサービスでは、自動バックアップ、モニタリング、高可用性、パッチ適用が提供され、運用上のオーバーヘッドを削減できます。

サイズ設定の推奨事項や設定パラメーターを含む MySQL の完全な要件については、[リファレンスアーキテクチャ](/ja/platform/hosting/self-managed/ref-arch/#mysql)を参照してください。データベース作成用 SQL については、[bare-metal guide](/ja/platform/hosting/self-managed/bare-metal/#mysql-database)を参照してください。デプロイのデータベース設定に関する質問がある場合は、[サポート](mailto:support@wandb.com) または担当の AISE にお問い合わせください。
