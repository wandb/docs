---
title: サービスアカウント APIキーを追加で作成
---

サービスアカウントが所有するAPIキーを作成するには、次の手順に従います。

1. チームまたは組織のSettingsで、**Service Accounts** タブにアクセスします。
2. 一覧からサービスアカウントを検索します。
3. アクションメニュー (`...`) をクリックし、**Create API key** をクリックします。
4. APIキーの名を指定し、**Create** をクリックします。
5. 表示されたAPIキーをすぐにコピーして、安全に保管します。
6. **Done** をクリックします。

異なる環境やワークフローに対応するために、1つのサービスアカウントに対して複数のAPIキーを作成できます。
