---
title: Service Account APIキーの追加作成（シングルテナント）
---

サービスアカウントが所有する APIキーを作成するには、次の手順を実行します。

1. チームまたは組織の設定で **Service Accounts** タブを開きます。
2. 一覧から該当するサービスアカウントを見つけます。
3. 작업 메뉴 (`...`) をクリックし、**Create API key** をクリックします。
4. APIキーの名前を入力し、**Create** をクリックします。
5. 表示された APIキーをすぐにコピーして、安全な場所に保管します。
6. **Done** をクリックします。

1 つのサービスアカウントに対して複数の APIキーを作成し、異なる環境やワークフローごとに使い分けることができます。
