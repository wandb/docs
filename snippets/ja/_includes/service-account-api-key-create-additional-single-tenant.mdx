---
title: サービスアカウント APIキーを追加作成する（Single Tenant）
---

サービスアカウントに紐づくAPIキーを作成するには、次の手順に従います。

1. チームまたは組織の設定にある **Service Accounts** タブにアクセスします。
2. 一覧からサービスアカウントを検索します。
3. アクションメニュー (`...`) をクリックし、**Create API key** をクリックします。
4. APIキーの名を入力し、**Create** をクリックします。
5. 表示されたAPIキーをすぐにコピーして、安全な場所に保管します。
6. **Done** をクリックします。

1 つのサービスアカウントに対して複数のAPIキーを作成できるため、異なる環境やワークフローに対応できます。
