---
title: 表をログする
---

```python
"""
W&Bに表をログする。
"""
import wandb

# 2列2行のデータを持つ表オブジェクトを作成する
my_table = wandb.Table(
    columns=["a", "b"],
    data=[["a1", "b1"], ["a2", "b2"]],
    log_mode="<log_mode>"
    )

# 新しいrunを開始する
with wandb.init(project="<project>") as run:
    # W&Bに表をログする
    run.log({"<table_name>": my_table})
```
