---
title: 表をログに記録する
---

```python
"""
W&B にテーブルをログします。
"""
import wandb

# 2列2行のデータを持つテーブルオブジェクトを作成する
my_table = wandb.Table(
    columns=["a", "b"],
    data=[["a1", "b1"], ["a2", "b2"]],
    log_mode="<log_mode>"
    )

# 新しい run を開始する
with wandb.init(project="<project>") as run:
    # テーブルを W&B にログする
    run.log({"<table_name>": my_table})
```
