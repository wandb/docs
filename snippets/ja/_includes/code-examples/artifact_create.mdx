title: Artifact の作成
---

```python
"""
W&B の Artifacts を作成してログに記録します。まず Artifacts
オブジェクトを作成します。次に、ファイル、ディレクトリ、または外部参照を Artifacts に追加します。
最後に、 Artifacts を W&B の run にログします。
"""
import wandb

# Artifacts オブジェクトを作成する
artifact = wandb.Artifact(name="<artifact_name>", type="<artifact_type>")

# Artifacts に1つ以上のファイルを追加する
artifact.add_file(local_path="data/file1.csv")

# Artifacts にディレクトリを再帰的に追加する
artifact.add_dir(local_path="data/directory")

# Artifacts に外部 URI 参照を追加する
artifact.add_reference(uri="s3://my-bucket/path/to/data.csv")

with wandb.init(project="<project>") as run:
    # トレーニングおよびログのコードをここに記述する

    # Artifacts をログに記録する例
    run.log_artifact(artifact)
```
