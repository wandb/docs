title: Artifact の更新
---

```python
"""
W&B の run 内で既存の W&B Artifacts の説明を更新します。

このコードは W&B の run を初期化し、名前とエイリアスで指定された Artifacts を取得して、
説明を更新し、変更を保存します。
"""
import wandb

with wandb.init(entity="<entity>", project="<project>") as run:
    # 名前とエイリアスで Artifacts を取得する
    artifact = run.use_artifact(artifact_or_name="<artifact>:<alias>")
    # Artifacts の説明を更新する
    artifact.description = "<description>"
    # 更新された Artifacts を保存する
    artifact.save()
```
