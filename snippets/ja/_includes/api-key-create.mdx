API キーを作成するには、詳細は **Personal API key** または **Service Account API key** タブを選択してください。

<Tabs>
  <Tab title="Personal API key">
    ユーザー ID に紐づく個人用 API キーを作成するには、次の手順に従います。

    1. W&amp;B にログインし、ユーザープロフィールアイコンをクリックしてから **User Settings** をクリックします。
    2. **Create new API key** をクリックします。
    3. API キーのわかりやすい名前を入力します。
    4. **Create** をクリックします。
    5. 表示された API キーをすぐにコピーし、安全な場所に保管します。
  </Tab>
