---
title: セルフマネージド MySQL EOL アップグレード
---

<Important>
  MySQL 8.0.x は 2026 年 4 月にサポート終了を迎えます。W&amp;B Self-Managed デプロイメントでは、セキュリティパッチと重大なバグ修正を受け取れる、サポート対象の MySQL バージョンを実行する必要があります。コミュニティ版 MySQL を実行している場合は、**MySQL 8.4.x** をインストールするか、アップグレードしてください。マネージドサービスを使用している場合は、ご利用のプロバイダがサポート対象かつパッチ適用済みとして文書化しているエンジンバージョンを実行してください (例: Amazon RDS for MySQL、Google Cloud SQL for MySQL、Azure Database for MySQL) 。W&amp;B は、MySQL 8.4.0 および現行の 8.4.x リリースでプラットフォームを検証しています。まだ MySQL 8.0.x を使用している場合は、[Upgrade MySQL to 8.4.x](/ja/platform/hosting/self-managed/operator#upgrade-mysql-to-84x) の手順に従ってアップグレードを計画してください。
</Important>
