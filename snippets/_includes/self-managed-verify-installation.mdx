To verify the installation, W&B recommends using the [W&B CLI](/models/ref/cli/). The `wandb verify` command runs tests that confirm components and configurations work as expected.

<Note>
This procedure assumes that you create the first admin user account in a browser.
</Note>

To verify the installation:

1. Install the W&B CLI:

   ```bash
   pip install wandb
   ```

2. Log in to W&B:

   ```bash
   wandb login --host=https://YOUR_DNS_DOMAIN
   ```

   For example:

   ```bash
   wandb login --host=https://wandb.company-name.com
   ```

3. Verify the installation:

   ```bash
   wandb verify
   ```
