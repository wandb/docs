To verify the installation, use the [W&B CLI](/models/ref/cli/). The `wandb verify` command runs several tests that confirm all components and configurations work as expected.

<Note>
This procedure assumes that you have already created the first admin user account in the browser.
</Note>

Follow these steps:

1. Install the W&B CLI:

   ```bash
   pip install wandb
   ```

2. Log in to W&B, replacing `[YOUR-DNS-DOMAIN]` with the DNS domain of your deployment:

   ```bash
   wandb login --host=https://[YOUR-DNS-DOMAIN]
   ```

   For example:

   ```bash
   wandb login --host=https://wandb.company-name.com
   ```

3. Verify the installation:

   ```bash
   wandb verify
   ```
