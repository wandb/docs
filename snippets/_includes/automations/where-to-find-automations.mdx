- **In the Automations hub**: Select **Automations** in the left navigation to see every automation you can access across all scopes and projects. You choose the scope (Organization, Team, Registry, Registry collection, or project) when you create the automation. See [Automations hub](/models/automations#automations-hub).
- **In a project**: Open the project, then select the **Automations** tab in the project sidebar.
- **In a registry**: Open the registry, then select the **Automations** tab.
