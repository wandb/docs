---
title: "Avertissement : affichage unique de la clé API"
---

<Warning>
  La clé API complète n’est affichée qu’une seule fois, au moment de sa création. Une fois la boîte de dialogue fermée, vous ne pouvez plus afficher la clé API complète. Seul l’ID de la clé (première partie de la clé) reste visible dans vos paramètres. Si vous perdez la clé API complète, vous devez créer une nouvelle clé API.
</Warning>
