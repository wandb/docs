---
title: Wandb Docker Run
---

Encapsule `docker run` et ajoute les variables d’environnement WANDB&#95;API&#95;KEY et WANDB&#95;DOCKER.

Définit également le runtime sur nvidia si l’exécutable nvidia-docker est présent sur le système et que `--runtime` n’a pas été défini.

Voir `docker run --help` pour plus de détails.
