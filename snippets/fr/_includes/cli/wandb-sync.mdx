---
title: Wandb Sync
---

Synchronisez les données de run W&amp;B dans le cloud.

Si PATH est fourni, synchronisez les runs trouvés à l’emplacement indiqué. Si aucun chemin n’est spécifié, recherchez d’abord `./wandb`, puis un sous-répertoire `wandb/`.

Pour synchroniser un run spécifique :

wandb sync ./wandb/run-20250813&#95;124246-n67z9ude

Ou, de façon équivalente :

wandb sync ./wandb/run-20250813&#95;124246-n67z9ude/run-n67z9ude.wandb
