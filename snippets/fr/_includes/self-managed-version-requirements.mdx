---
title: Exigences de version pour Self-Managed
---

| Logiciel   | Version minimale                                                                                                                                                                          |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Kubernetes | v1.32 ou plus récente ([Versions de Kubernetes prises en charge](https://kubernetes.io/releases/patch-releases/))                                                                         |
| Helm       | v3.x                                                                                                                                                                                      |
| MySQL      | v8.0.x est requis ; v8.0.32 ou plus récente, v8.0.44 ou plus récente est recommandée.<br />Pour les versions Aurora MySQL 3.x, vous devez utiliser la v3.05.2 ou une version plus récente |
| Redis      | v7.x                                                                                                                                                                                      |
