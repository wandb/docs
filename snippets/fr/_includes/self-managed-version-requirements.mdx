---
title: Exigences de version pour Self-Managed
---

| Logiciel   | Version minimale                                                                                                                                                                                 |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Kubernetes | v1.32 ou version ultérieure ([Versions de Kubernetes prises en charge](https://kubernetes.io/releases/patch-releases/))                                                                          |
| Helm       | v3.x                                                                                                                                                                                             |
| MySQL      | v8.0.x est requis ; v8.0.32 ou version ultérieure ; v8.0.44 ou version ultérieure est recommandée.<br />Pour les versions Aurora MySQL 3.x, vous devez utiliser la version v3.05.2 ou ultérieure |
| Redis      | v7.x                                                                                                                                                                                             |
