---
title: Prérequis de version pour le déploiement auto-géré
---

| Logiciel   | Version minimale                                                                                                                                            |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Kubernetes | v1.32 ou ultérieure ([Versions de Kubernetes prises en charge](https://kubernetes.io/releases/patch-releases/))                                             |
| Helm       | v3.x                                                                                                                                                        |
| MySQL      | v8.0.x est requis ; v8.0.32 ou ultérieure ; v8.0.44 ou ultérieure est recommandée.<br />Les versions Aurora MySQL 3.x doivent être en v3.05.2 ou ultérieure |
| Redis      | v7.x                                                                                                                                                        |
