---
title: Supprimer une clé API de compte de service
---

Pour supprimer une clé API appartenant à un compte de service d&#39;une organisation ou d&#39;une équipe :

1. Accédez à [Organization settings](https://wandb.ai/account-settings/), puis cliquez sur **API Keys**.
2. Repérez la clé API. La liste contient toutes les clés API appartenant à des comptes de service d&#39;organisation et d&#39;équipe. Vous pouvez effectuer une recherche ou appliquer un filtre par nom de clé ou ID, et trier selon n&#39;importe quelle colonne.
3. Cliquez sur le bouton Delete.
