---
title: Supprimer une clé API de compte de service
---

Pour supprimer une clé API appartenant à un compte de service d’organisation ou d’équipe :

1. Accédez à [Paramètres de l’organisation](https://wandb.ai/account-settings/), puis cliquez sur **API Keys**.
2. Trouvez la clé API. La liste comprend toutes les clés API appartenant à des comptes de service d’organisation et d’équipe. Vous pouvez effectuer une recherche ou filtrer par nom ou ID de clé, et trier par n’importe quelle colonne.
3. Cliquez sur le bouton de suppression.
