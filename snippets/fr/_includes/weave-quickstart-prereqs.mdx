<div id="prerequisites">
  ## Prérequis
</div>

* Un [compte W&amp;B](https://wandb.ai/signup)
* Python 3.8+ ou Node.js 18+
* Packages requis installés :
  * **Python**: `pip install weave openai`
  * **TypeScript**: `npm install weave openai`
* Une [clé API OpenAI](https://platform.openai.com/api-keys) définie en tant que variable d&#39;environnement
