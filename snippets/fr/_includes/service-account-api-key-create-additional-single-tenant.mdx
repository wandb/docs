---
title: Créer une clé API supplémentaire pour un compte de service en locataire unique
---

Pour créer une clé API appartenant à un compte de service :

1. Accédez à l’onglet **Service Accounts** dans les paramètres de votre équipe ou de votre organisation.
2. Trouvez le compte de service dans la liste.
3. Cliquez sur le menu d’actions (`...`), puis sur **Create API key**.
4. Saisissez un nom pour la clé API, puis cliquez sur **Create**.
5. Copiez immédiatement la clé API affichée et stockez-la en lieu sûr.
6. Cliquez sur **Done**.

Vous pouvez créer de multiples clés API pour un même compte de service afin de prendre en charge différents environnements ou flux de travail.
