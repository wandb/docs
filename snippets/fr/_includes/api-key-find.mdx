---
title: Trouver une clé API
---

<Note>
  Dans **Cloud dédié** et **Autogéré** :

  * Les administrateurs de l’organisation peuvent trouver ou lister les clés API de tous les utilisateurs et comptes de service de l’organisation.
  * Les administrateurs d’équipe peuvent trouver ou lister les clés API des comptes de service dans les Teams qu’ils administrent.
  * Les utilisateurs non administrateurs peuvent trouver ou lister leurs propres clés API.
</Note>

Pour trouver une clé API, sélectionnez l’onglet **Clé API personnelle** ou **Clé API de compte de service** pour afficher plus de détails.

<Tabs>
  <Tab title="Clé API personnelle">
    Pour trouver une clé API personnelle associée à votre ID utilisateur :

    1. Connectez-vous à W&amp;B, cliquez sur l’icône de votre profil utilisateur, puis sur **Paramètres utilisateur**.
    2. Faites défiler jusqu’à la section **Clés API**.
  </Tab>

  <Tab title="Clé API de compte de service">
    Pour trouver une clé API appartenant à un compte de service d’organisation ou d’équipe :

    1. Accédez à l’onglet **Clés API** dans les paramètres de votre organisation.
    2. Trouvez la clé API dans la liste. Vous pouvez rechercher ou filtrer la liste par propriétaire, nom de clé ou ID de clé.
  </Tab>
</Tabs>

Le tableau des clés API affiche l’ID de la clé (la première partie de chaque clé API) à des fins d’identification. La clé API secrète complète n’est affichée qu’une seule fois, au moment de sa création. Si vous devez utiliser une clé existante mais que vous n’avez pas enregistré le secret complet, vous devez créer une nouvelle clé API.
