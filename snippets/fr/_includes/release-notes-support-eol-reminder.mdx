---
title: Rappel de fin de support dans les notes de version
---

<Note>Une version de W&amp;B Server est prise en charge pendant 12 mois à compter de sa date de sortie initiale. Si vous utilisez [Autogéré](/fr/platform/hosting/hosting-options/self-managed/), vous devez effectuer la mise à niveau vers une version prise en charge dans les délais afin de continuer à bénéficier de l’assistance. Pour plus de détails, voir [les versions de W&amp;B Server prises en charge](https://docs.wandb.ai/release-notes/server-releases) et [les versions archivées de W&amp;B Server](https://docs.wandb.ai/release-notes/server-releases-archived).</Note>
