Pour créer une clé API, sélectionnez l’onglet **Clé API personnelle** ou **Clé API de compte de service** pour en voir les détails.

<Tabs>
  <Tab title="Clé API personnelle">
    Pour créer une clé API personnelle associée à votre ID utilisateur :

    1. Connectez-vous à W&amp;B, cliquez sur l’icône de votre profil, puis sur **Paramètres utilisateur**.
    2. Cliquez sur **Créer une nouvelle clé API**.
    3. Saisissez un nom descriptif pour votre clé API.
    4. Cliquez sur **Créer**.
    5. Copiez immédiatement la clé API affichée et conservez-la dans un emplacement sécurisé.
  </Tab>
