---
title: Exigences MySQL pour Autogéré
---

W&amp;B nécessite une base de données MySQL externe.

Pour la production, W&amp;B recommande fortement d&#39;utiliser des services de base de données gérés :

* [AWS RDS Aurora MySQL](https://aws.amazon.com/rds/aurora/)
* [Google Cloud SQL for MySQL](https://cloud.google.com/sql/mysql)
* [Azure Database for MySQL](https://azure.microsoft.com/en-us/products/mysql/)

Les services de base de données gérés fournissent des sauvegardes automatisées, la surveillance, une haute disponibilité, l&#39;application de correctifs et réduisent la charge opérationnelle.

Voir l&#39;[architecture de référence](/fr/platform/hosting/self-managed/ref-arch/#mysql) pour connaître l&#39;ensemble des exigences MySQL, y compris les recommandations de dimensionnement et les paramètres de configuration. Pour le SQL de création de la base de données, voir le [guide bare-metal](/fr/platform/hosting/self-managed/bare-metal/#mysql-database). Pour toute question sur la configuration de la base de données de votre déploiement, contactez l&#39;[assistance](mailto:support@wandb.com) ou votre AISE.
