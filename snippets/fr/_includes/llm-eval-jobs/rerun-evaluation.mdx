<div id="re-run-an-evaluation-job">
  ## Relancer un job d&#39;évaluation
</div>

Selon votre situation, il existe plusieurs façons de relancer un job d&#39;évaluation ou d&#39;en afficher la configuration.

* Pour relancer le dernier job d&#39;évaluation, suivez les étapes de [Évaluer votre modèle](#evaluate-your-model). Sélectionnez le projet de destination, puis les détails de l&#39;artefact du modèle et les benchmarks sélectionnés la dernière fois sont renseignés automatiquement. Facultativement, apportez des modifications, puis lancez le job d&#39;évaluation.
* Pour relancer un job d&#39;évaluation depuis l&#39;onglet **Runs** du projet ou le sélecteur de run, survolez le nom de l&#39;exécution et cliquez sur l&#39;icône de lecture. Le volet de configuration du job s&#39;affiche avec les paramètres préremplis. Facultativement, ajustez les paramètres, puis cliquez sur **Launch**.
* Pour relancer un job d&#39;évaluation depuis un autre projet, importez sa configuration :
  1. Suivez les étapes de [Évaluer votre modèle](#evaluate-your-model). Après avoir sélectionné le projet de destination, cliquez sur **Importer la configuration**.
  2. Sélectionnez le projet qui contient le job d&#39;évaluation à importer, puis sélectionnez le run du job d&#39;évaluation. Le volet de configuration du job s&#39;affiche avec les paramètres préremplis.
  3. Facultativement, ajustez la configuration.
  4. Cliquez sur **Launch**.
