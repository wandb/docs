<div id="export-an-evaluation-job-configuration">
  ## Exporter la configuration d’un job d’évaluation
</div>

Exportez la configuration d’un job d’évaluation depuis l’onglet **Files** du run.

1. Ouvrez le run dans la vue dédiée à un seul run.
2. Cliquez sur l’onglet **Files**.
3. Cliquez sur le bouton de téléchargement situé à côté de `config.yaml` pour le télécharger en local.
