---
title: Créer un compte de service limité à l’organisation
---

Pour créer un compte de service limité à l’organisation et une clé API :

1. Connectez-vous à W&amp;B.
2. Cliquez sur l’icône de votre profil, puis accédez à **Comptes de service** :
   * **Cloud dédié** ou **Autogéré** : cliquez sur **Tableau de bord de l’organisation**, puis sur **Comptes de service**.
   * **Cloud mutualisé** : cliquez sur **Comptes de service**.
3. Cliquez sur **Créer un compte de service**.
4. Saisissez un nom et sélectionnez une équipe par défaut.
5. Cliquez sur **Créer**.
6. Repérez le compte de service que vous venez de créer, puis cliquez sur le menu d’action (<Icon icon="ellipsis" iconType="solid" />), puis sur **Créer une clé API**.
7. Saisissez un nom pour la clé API, puis cliquez sur **Créer**.
8. Copiez la clé API et conservez-la en lieu sûr.
9. Cliquez sur **Terminé**.
