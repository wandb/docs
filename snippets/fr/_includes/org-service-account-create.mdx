---
title: Créer un compte de service limité à l’organisation
---

Pour créer un nouveau compte de service limité à l’organisation et une clé API :

1. Connectez-vous à W&amp;B, cliquez sur l’icône de votre profil, puis :
   * **Cloud dédié** ou **Autogéré** : cliquez sur **tableau de bord de l’organisation**, puis sur **Service Accounts**.
   * **Cloud mutualisé** : cliquez sur **Service Accounts**.
2. Cliquez sur **Create service account**.
3. Saisissez un nom et sélectionnez une équipe par défaut.
4. Cliquez sur **Create**.
5. Repérez le compte de service que vous venez de créer.
6. Cliquez sur le menu d’actions (`...`), puis sur **Create API key**.
7. Saisissez un nom pour la clé API, puis cliquez sur **Create**.
8. Copiez la clé API et conservez-la en lieu sûr.
9. Cliquez sur **Done**.
