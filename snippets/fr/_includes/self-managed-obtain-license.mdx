---
title: "Autogéré : obtenir une License"
---

Vous avez besoin d’une License W&amp;B pour déployer W&amp;B Autogéré.

1. Si vous n’avez pas encore de compte W&amp;B, créez-en un.
2. Si vous avez besoin d’une License d’essai Enterprise incluant l’assistance pour des fonctionnalités de sécurité importantes et d’autres capacités adaptées aux entreprises, [soumettez une requête](https://wandb.ai/site/for-enterprise/self-hosted-trial) ou contactez votre équipe W&amp;B.
3. Sinon, ouvrez le [Deploy Manager](https://deploy.wandb.ai/deploy) pour générer une License d’essai gratuite. L’URL vous redirige vers un formulaire **Obtenir une License pour W&amp;B Local**. Fournissez les informations suivantes :
   * Le titulaire de la License
   * Le type de déploiement
   * Un nom et une Description facultative pour l’instance
4. Cliquez sur **Générer la Clé de License**.

Une page s’affiche avec un Aperçu de votre déploiement, ainsi que la License associée à l’instance.
