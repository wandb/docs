---
title: "Autogéré : obtenir une licence"
---

Vous avez besoin d’une licence W&amp;B pour déployer W&amp;B Autogéré.

1. Si vous n’avez pas encore de compte W&amp;B, créez-en un.
2. Si vous avez besoin d’une licence d’essai Enterprise incluant l’assistance pour des fonctionnalités de sécurité importantes et d’autres capacités adaptées aux entreprises, [soumettez une demande](https://wandb.ai/site/for-enterprise/self-hosted-trial) ou contactez votre équipe W&amp;B.
3. Sinon, ouvrez le [Deploy Manager](https://deploy.wandb.ai/deploy) pour générer une licence d’essai gratuite. L’URL vous redirige vers un formulaire **Obtenir une licence pour W&amp;B Local**. Fournissez les informations suivantes :
   * Le titulaire de la licence
   * Le type de déploiement
   * Un nom et une description facultative pour l’instance
4. Cliquez sur **Générer la clé de licence**.

Une page s’affiche avec un aperçu de votre déploiement, ainsi que la licence associée à l’instance.
