---
title: Épingler des runs (condensé)
---

Épinglez jusqu’à 6 runs pour les garder facilement accessibles en haut de votre espace de travail. Si vous avez un run de référence, vous pouvez épingler jusqu’à 5 runs, car le run de référence est implicitement épinglé. Les runs épinglés restent visibles, quels que soient le tri ou le filtrage appliqués aux autres runs. Les runs épinglés apparaissent en haut du sélecteur de run, avec une icône d’épingle circulaire, séparés des autres runs par un séparateur visuel.

Pour épingler un run :

1. Accédez à votre espace de travail.
2. Dans le sélecteur de run ou le tableau des runs, trouvez le run que vous souhaitez épingler.
3. Cliquez sur le menu d’action `...`, puis sélectionnez **Épingler le run**.
