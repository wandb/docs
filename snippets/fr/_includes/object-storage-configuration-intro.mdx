<div id="configure-wb-to-use-your-bucket">
  ### Configurer W&amp;B pour utiliser votre bucket
</div>

Une fois votre bucket provisionné, configurez W&amp;B pour l’utiliser :

* **Déploiements autogérés** : Voir le [guide de configuration de l’opérateur](/fr/platform/hosting/self-managed/operator#object-storage-bucket) pour configurer les valeurs Helm
* **Cloud dédié** : Configurer via la [System Console](/fr/platform/hosting/iam/sso#system-console)
* **Cloud mutualisé** : Configurer lors de la création ou de la modification d’une équipe
