---
title: Créer un compte de service à portée d'équipe
---

Pour créer un compte de service à portée d&#39;équipe et une clé API :

1. Dans les paramètres de votre équipe, cliquez sur **Service Accounts**.
2. Cliquez sur **New Team Service Account**.
3. Saisissez un nom pour le compte de service.
4. Définissez **Authentication Method** sur **Generate API key** (par défaut). Si vous sélectionnez **Federated Identity**, le compte de service ne peut pas détenir de clés API.
5. Cliquez sur **Create**.
6. Repérez le compte de service que vous venez de créer, puis cliquez sur son menu **d&#39;action (<Icon icon="ellipsis" iconType="solid" />)** et cliquez sur **Create API key**.
7. Saisissez un nom pour la clé API, puis cliquez sur **Create**.
8. Copiez la clé API et conservez-la en lieu sûr.
9. Cliquez sur **Done**.
