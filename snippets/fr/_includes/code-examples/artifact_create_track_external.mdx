---
title: Créer et suivre un artefact externe
---

```python
"""
Suit un artefact externe en ajoutant une référence à un chemin de bucket de stockage cloud.
"""

import wandb

# Créer un objet artefact
artifact = wandb.Artifact(name="<artifact_name>", type="<artifact_type>")

# Ajouter une référence au chemin du bucket
artifact.add_reference(uri = "uri/to/your/bucket/path")

# Initialiser un run W&B
with wandb.init(project="<project>") as run:

  # Journaliser les métadonnées de l'artefact
  run.log_artifact(artifact)
```
