title: TTL d’artefact

```python
"""
Créer et journaliser un artefact avec une politique TTL dans W&B.
"""
import wandb
from datetime import timedelta

# Créer un artefact avec une politique TTL
artifact = wandb.Artifact(name="<artifact_name>", type="<artifact_type>")
artifact.add_file("<file_path>")
artifact.ttl = timedelta(days=30)  # Définir la politique TTL

with wandb.init(project="<project>", entity="<entity>") as run:
    # Journaliser l'artefact avec TTL
    run.log_artifact(artifact)
```
