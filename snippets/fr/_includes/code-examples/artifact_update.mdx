---
title: Mise à jour de l’artefact
---

```python
"""
Mettre à jour la description d'un artefact W&B existant dans un run W&B.

Ce code initialise un run W&B, récupère l'artefact spécifié par nom et alias,
met à jour sa description et enregistre les modifications.
"""
import wandb

with wandb.init(entity="<entity>", project="<project>") as run:
    # Récupérer l'artefact par nom et alias
    artifact = run.use_artifact(artifact_or_name="<artifact>:<alias>")
    # Mettre à jour la description de l'artefact
    artifact.description = "<description>"
    # Enregistrer l'artefact mis à jour
    artifact.save()
```
