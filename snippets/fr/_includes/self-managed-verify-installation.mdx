---
title: "Auto-hébergé : vérifier l’installation"
---

Pour vérifier l’installation, W&amp;B recommande d’utiliser le [W&amp;B CLI](/fr/models/ref/cli/). La commande verify exécute plusieurs tests pour vérifier l’ensemble des composants et de la configuration.

<Note>
  Cette étape suppose que le premier compte administrateur a été créé depuis le navigateur.
</Note>

Suivez ces étapes pour vérifier l’installation :

1. Installez le W&amp;B CLI :

```bash
pip install wandb
```

2. Connectez-vous à W&amp;B :

```bash
wandb login --host=https://YOUR_DNS_DOMAIN
```

Par exemple :

```bash
wandb login --host=https://wandb.company-name.com
```

3. Vérifiez l’installation :

```bash
wandb verify
```
