<div id="general">
  ### General
</div>

| **Clé**       | **Description**                      |
| :------------ | :----------------------------------- |
| `h`, `?`      | Afficher ou masquer cet écran d’aide |
| `q`, `ctrl+c` | Quitter                              |
| `alt+r`       | Recharger les données du run         |

<div id="panels">
  ### Panneaux
</div>

| **Clé** | **Description**                                              |
| :------ | :----------------------------------------------------------- |
| `[`     | Basculer la barre latérale gauche avec l’aperçu du run       |
| `]`     | Basculer la barre latérale droite avec les métriques système |
| `l`     | Basculer le volet des journaux de console du run             |
| `s`     | Basculer le volet des métriques système                      |

<div id="navigation">
  ### Navigation
</div>

| **Clé**               | **Description**                                  |
| :-------------------- | :----------------------------------------------- |
| `N`, `pgup`           | Passer d’une page de graphiques à l’autre        |
| `n`, `pgdown`         | Passer d’une page de graphiques à l’autre        |
| `alt+N`, `alt+pgup`   | Passer d’une page de métriques système à l’autre |
| `alt+n`, `alt+pgdown` | Passer d’une page de métriques système à l’autre |

<div id="charts">
  ### Graphiques
</div>

| **Clé**  | **Description**                                                                                            |
| :------- | :--------------------------------------------------------------------------------------------------------- |
| `/`      | Filtrer les métriques selon un motif (prend en charge les correspondances de type glob ou par sous-chaîne) |
| `ctrl+l` | Effacer le filtre actif                                                                                    |
| `\`      | Basculer le filtrage des métriques système                                                                 |

<div id="run-overview">
  ### Aperçu du run
</div>

| **Clé**  | **Description**                                                                                                           |
| :------- | :------------------------------------------------------------------------------------------------------------------------ |
| `o`      | Filtrer les éléments de l’aperçu (prend en charge `@e`, `@c`, `@s` pour les sections Environnement/configuration/summary) |
| `ctrl+k` | Effacer le filtre de l’aperçu                                                                                             |

<div id="configuration">
  ### Configuration
</div>

| **Clé** | **Description**                                     |
| :------ | :-------------------------------------------------- |
| `c`     | Définir les colonnes de la grille des métriques     |
| `r`     | Définir les lignes de la grille des métriques       |
| `C`     | Définir les colonnes de la grille système (Shift+c) |
| `R`     | Définir les lignes de la grille système (Shift+r)   |

<div id="run-overview-navigation-when-sidebar-open">
  ### Navigation de l’aperçu du run (lorsque la barre latérale est ouverte)
</div>

| **Clé**            | **Description**                                                      |
| :----------------- | :------------------------------------------------------------------- |
| `up`, `down`       | Parcourir les éléments de la section                                 |
| `tab`, `shift+tab` | Passer d’une section à l’autre                                       |
| `left`, `right`    | Parcourir les pages de la section                                    |
| `@e`, `@c`, `@s`   | Filtrer une section spécifique (Environnement/configuration/summary) |

<div id="mouse">
  ### Souris
</div>

| **Action**           | **Description**                           |
| :------------------- | :---------------------------------------- |
| `mouse wheel`        | Zoom avant/arrière sur le graphique actif |
| `shift+mouse select` | Sélectionner du texte                     |
