<div id="general">
  ### Général
</div>

| **Clé**       | **Description**              |
| :------------ | :--------------------------- |
| `h`, `?`      | Basculer cet écran d’aide    |
| `q`, `ctrl+c` | Quitter                      |
| `alt+r`       | Recharger les données du run |

<div id="panels">
  ### Panneaux
</div>

| **Clé** | **Description**                                                             |
| :------ | :-------------------------------------------------------------------------- |
| `[`     | Basculer l’affichage de la barre latérale gauche avec l’aperçu du run       |
| `]`     | Basculer l’affichage de la barre latérale droite avec les métriques système |
| `l`     | Basculer l’affichage du panneau des journaux de console du run              |
| `s`     | Basculer l’affichage du panneau des métriques système                       |

<div id="navigation">
  ### Navigation
</div>

| **Clé**               | **Description**                               |
| :-------------------- | :-------------------------------------------- |
| `N`, `pgup`           | Naviguer entre les pages de graphiques        |
| `n`, `pgdown`         | Naviguer entre les pages de graphiques        |
| `alt+N`, `alt+pgup`   | Naviguer entre les pages de métriques système |
| `alt+n`, `alt+pgdown` | Naviguer entre les pages de métriques système |

<div id="charts">
  ### Graphiques
</div>

| **Clé**  | **Description**                                                                                 |
| :------- | :---------------------------------------------------------------------------------------------- |
| `/`      | Filtrer les métriques par motif (prend en charge la correspondance par glob ou par sous-chaîne) |
| `ctrl+l` | Effacer le filtre actif                                                                         |
| `\`      | Basculer le filtrage des métriques système                                                      |

<div id="run-overview">
  ### Aperçu du run
</div>

| **Clé**  | **Description**                                                                                                            |
| :------- | :------------------------------------------------------------------------------------------------------------------------- |
| `o`      | Filtrer les éléments de l’aperçu (prend en charge `@e`, `@c`, `@s` pour les sections environnement/configuration/synthèse) |
| `ctrl+k` | Effacer le filtre de l’aperçu                                                                                              |

<div id="configuration">
  ### Configuration
</div>

| **Clé** | **Description**                                              |
| :------ | :----------------------------------------------------------- |
| `c`     | Définir le nombre de colonnes de la grille des métriques     |
| `r`     | Définir le nombre de lignes de la grille des métriques       |
| `C`     | Définir le nombre de colonnes de la grille système (Shift+c) |
| `R`     | Définir le nombre de lignes de la grille système (Shift+r)   |

<div id="run-overview-navigation-when-sidebar-open">
  ### Navigation de l’Aperçu du run (quand la barre latérale est ouverte)
</div>

| **Clé**            | **Description**                                                       |
| :----------------- | :-------------------------------------------------------------------- |
| `up`, `down`       | Naviguer dans les éléments de la section                              |
| `tab`, `shift+tab` | Passer d’une section à l’autre                                        |
| `left`, `right`    | Naviguer dans les pages de la section                                 |
| `@e`, `@c`, `@s`   | Filtrer une section spécifique (environnement/configuration/synthèse) |

<div id="mouse">
  ### Souris
</div>

| **Action**                    | **Description**                                 |
| :---------------------------- | :---------------------------------------------- |
| `molette de la souris`        | Zoom avant/arrière sur le graphique sélectionné |
| `shift+sélection à la souris` | Sélectionner du texte                           |
