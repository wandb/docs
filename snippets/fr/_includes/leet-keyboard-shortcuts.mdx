<div id="workspace">
  ### Espace de travail
</div>

| Clé                              | Action                                                                                 |
| -------------------------------- | -------------------------------------------------------------------------------------- |
| `h`, `?`                         | Basculer l’affichage de l’aide.                                                        |
| `q`, `ctrl+c`                    | Quitter.                                                                               |
| `alt+r`                          | Redémarrer LEET.                                                                       |
| `esc`                            | Placer le focus sur la liste des runs.                                                 |
| `enter`                          | Ouvrir le run mis en surbrillance dans la vue run unique.                              |
| `1`                              | Basculer la grille des métriques.                                                      |
| `[`                              | Basculer la barre latérale des runs.                                                   |
| `2`                              | Basculer le volet des métriques système.                                               |
| `]`                              | Basculer la barre latérale d’aperçu du run.                                            |
| `3`                              | Basculer le volet multimédia.                                                          |
| `4`                              | Basculer le volet des journaux de la console.                                          |
| `f`                              | Filtrer les runs par nom ou métadonnées.                                               |
| `ctrl+f`                         | Effacer le filtre des runs.                                                            |
| `space`                          | Sélectionner ou désélectionner le run.                                                 |
| `p`                              | Épingler ou désépingler le run.                                                        |
| `/`                              | Filtrer les métriques.                                                                 |
| `\`                              | Filtrer les métriques système.                                                         |
| `ctrl+/`, `ctrl+l`               | Effacer le filtre des métriques.                                                       |
| `ctrl+\`                         | Effacer le filtre des métriques système.                                               |
| `o`                              | Filtrer les éléments de l’aperçu du run.                                               |
| `ctrl+o`                         | Effacer le filtre d’aperçu.                                                            |
| `y`                              | Faire défiler les modes du graphique actif.                                            |
| `c`                              | Définir les colonnes de la grille active.                                              |
| `r`                              | Définir les lignes de la grille active.                                                |
| `tab`, `shift+tab`               | Parcourir les volets visibles en changeant le focus.                                   |
| `w`, `a`, `s`, `d` or arrow keys | Se déplacer dans le volet actif.                                                       |
| `N`, `pgup`                      | Page précédente.                                                                       |
| `n`, `pgdown`                    | Page suivante.                                                                         |
| `home`                           | Aller au premier élément, à la première page ou à la première image multimédia.        |
| `end`                            | Aller au dernier élément, à la dernière page ou à l’image multimédia la plus récente. |

<div id="symon">
  ### SYMON
</div>

| Clé                                        | Action                                            |
| ------------------------------------------ | ------------------------------------------------- |
| `h`, `?`                                   | Afficher ou masquer l’aide.                       |
| `q`, `ctrl+c`                              | Quitter.                                          |
| `alt+r`                                    | Redémarrer.                                       |
| `y`                                        | Faire défiler les modes du graphique actif. |
| `\`                                        | Filtrer les métriques système.                    |
| `ctrl+\`                                   | Effacer le filtre des métriques système.          |
| `c`, `C`                                   | Définir les colonnes de la grille.                |
| `r`, `R`                                   | Définir les lignes de la grille.                  |
| `w`, `a`, `s`, `d` ou les touches fléchées | Déplacer la sélection du graphique.               |
| `N`, `pgup`                                | Page de graphiques précédente.                    |
| `n`, `pgdown`                              | Page de graphiques suivante.                      |
| `home`                                     | Première page de graphiques.                      |
| `end`                                      | Dernière page de graphiques.                      |

<div id="mouse">
  ### Souris
</div>

| Entrée                       | Action                                                                |
| ---------------------------- | --------------------------------------------------------------------- |
| Clic                         | Donner le focus à un graphique ou sélectionner une vignette de média. |
| Molette                      | Zoomer sur le graphique ayant le focus.                               |
| Clic droit + glisser         | Inspecter le point le plus proche sur un graphique.                   |
| `alt` + clic droit + glisser | Inspecter tous les graphiques visibles de façon synchronisée.         |
| `shift` + glisser            | Sélectionner le texte du terminal.                                    |
