---
title: Créer des clés API supplémentaires pour un compte de service
---

Pour créer une clé API associée à un compte de service :

1. Accédez à l’onglet **Service Accounts** dans les paramètres de votre équipe ou de votre organisation.
2. Trouvez le compte de service dans la liste.
3. Cliquez sur le menu d’actions (`...`), puis sur **Créer une clé API**.
4. Saisissez un nom pour la clé API, puis cliquez sur **Créer**.
5. Copiez immédiatement la clé API affichée et conservez-la en lieu sûr.
6. Cliquez sur **Done**.

Vous pouvez créer de multiples clés API pour un même compte de service afin de prendre en charge différents environnements ou flux de travail.
