title: Création simplifiée d’une clé API
---

<Note>
  Pour aller plus vite, créez une clé API en accédant directement aux [Paramètres utilisateur](https://wandb.ai/settings). Copiez immédiatement la clé API nouvellement créée et enregistrez-la dans un endroit sécurisé, par exemple dans un gestionnaire de mots de passe.
</Note>
