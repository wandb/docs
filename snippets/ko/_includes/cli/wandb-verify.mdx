title: Wandb Verify
---

W&amp;B의 로컬 인스턴스를 점검하고 확인합니다. W&amp;B는 다음 항목을 확인합니다.

호스트가 `api.wandb.ai`가 아닌지 확인합니다(호스트 확인).

제공된 API 키를 사용해 사용자가 올바르게 로그인했는지 확인합니다(로그인 확인).

요청이 HTTPS를 통해 이루어지는지 확인합니다(보안 요청).

객체 저장소의 CORS(Cross-Origin Resource Sharing) 설정을 검증합니다(CORS 설정).

메트릭을 로깅하고 파일을 저장 및 다운로드하여 run이 올바르게 기록되고 접근 가능한지 확인합니다(run 확인).

artifact 저장 및 검색 시스템이 예상대로 작동하는지 확인하기 위해 artifact를 저장하고 다운로드합니다(artifact 확인).

파일을 업로드해 서명된 URL 업로드를 처리할 수 있는지 확인함으로써 GraphQL 엔드포인트를 테스트합니다(GraphQL PUT 확인).

프록시를 통해 큰 payload를 전송할 수 있는지 확인합니다(대용량 payload 확인).

설치된 W&amp;B 패키지 버전이 최신 상태이며 서버와 호환되는지 확인합니다(W&amp;B 버전 확인).

sweep 기능이 올바르게 작동하는지 확인하기 위해 sweep을 생성하고 실행합니다(sweeps 확인).
