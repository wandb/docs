---
title: Wandb Verify
---

로컬 W&B 인스턴스를 점검하고 검증합니다. W&B는 다음 항목을 확인합니다:

호스트가 `api.wandb.ai`가 아닌지 확인합니다(호스트 체크).

제공된 API 키를 사용해 사용자가 올바르게 로그인되어 있는지 검증합니다(로그인 체크).

요청이 HTTPS를 통해 전송되는지 확인합니다(보안 요청).

객체 스토어의 CORS(Cross-Origin Resource Sharing) 설정을 검증합니다(CORS 구성).

메트릭을 로깅하고 파일을 저장 및 다운로드하여 실행이 올바르게 기록되고 접근 가능한지 확인합니다(실행 체크).

Artifact를 저장하고 다운로드하여 Artifact 저장 및 조회 시스템이 의도한 대로 동작하는지 검증합니다(Artifact 체크).

파일을 업로드하여 GraphQL 엔드포인트를 테스트하고, 서명된 URL 업로드를 처리할 수 있는지 확인합니다(GraphQL PUT 체크).

프록시를 통해 대용량 페이로드를 전송할 수 있는지 확인합니다(대용량 페이로드 체크).

설치된 W&B 패키지 버전이 최신이며 서버와 호환되는지 검증합니다(W&B 버전 체크).

Sweep을 생성하고 실행하여 Sweep 기능이 정상적으로 동작하는지 확인합니다(Sweep 체크).
