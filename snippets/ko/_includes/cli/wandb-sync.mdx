---
title: Wandb 동기화
---

W&amp;B run 데이터를 클라우드에 동기화합니다.

PATH를 지정하면 해당 경로에서 찾은 run을 동기화합니다. 경로를 지정하지 않으면 먼저 `./wandb`를 검색한 다음, `wandb/` 하위 디렉터리를 검색합니다.

특정 run을 동기화하려면:

wandb sync ./wandb/run-20250813&#95;124246-n67z9ude

또는 다음과 같이 동일하게 수행할 수 있습니다:

wandb sync ./wandb/run-20250813&#95;124246-n67z9ude/run-n67z9ude.wandb
