LEET은 W&amp;B run의 통계, 메트릭, 시스템 상태를 확인할 수 있는 로컬 터미널 UI입니다. LEET은 SSH나 `tmux`와 같은 도구를 사용해 원격 시스템이나 HPC 환경에서 작업하는 사용자를 위해 설계된 키보드 중심 인터페이스입니다.

![Screenshot demonstrating viewing a run in the LEET terminal UI](/images/leet/leet-tui.png)

주요 기능은 다음과 같습니다:

* **키보드 중심 내비게이션**: 모든 기능을 키보드 단축키로 이용 가능
* **실시간 업데이트**: run이 진행되는 동안 `.wandb` 파일을 실시간으로 모니터링
* **메트릭 필터링**: glob 패턴 또는 부분 문자열 매칭으로 메트릭 필터링
* **스마트 줌**: 마우스 휠로 줌을 사용할 때 최신 데이터 포인트를 항상 화면에 유지
* **사용자 정의 가능한 그리드**: 메트릭 및 시스템 표시를 위한 행과 열 개수를 설정 가능
* **사람이 읽기 쉬운 형식**: 시스템 메트릭을 이해하기 쉬운 단위로 표시 (예: &quot;GPU Utilization (%)&quot;, &quot;RAM Used (GB)&quot;)

LEET은 세 패널로 구성된 레이아웃을 제공합니다:

* **왼쪽 사이드바**: 환경 변수, 설정, 요약 통계를 보여주는 Run Overview
* **중앙 패널**: 로그된 메트릭에 대한 점자(Braille) 스타일 선형 차트가 있는 메트릭 그리드
* **오른쪽 사이드바**: GPU 사용률, RAM, CPU 및 기타 시스템 상태 지표를 표시하는 System Metrics
