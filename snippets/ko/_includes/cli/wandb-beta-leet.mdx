LEET는 W&amp;B run의 통계, 메트릭, 시스템 상태를 확인할 수 있는 로컬 터미널 UI입니다. LEET는 SSH나 `tmux` 같은 도구를 사용해 원격 시스템이나 HPC 환경에서 작업하는 사용자를 위해 설계된 키보드 중심 인터페이스입니다.

![LEET 터미널 UI에서 run을 보는 모습을 보여주는 스크린샷](/images/leet/leet-tui.png)

주요 기능은 다음과 같습니다.

* **키보드 우선 내비게이션**: 모든 기능에 키보드 단축키로 접근할 수 있습니다
* **실시간 업데이트**: run이 진행되는 동안 `.wandb` 파일을 실시간으로 모니터링합니다
* **메트릭 필터링**: glob 패턴이나 부분 문자열 일치를 사용해 메트릭을 필터링합니다
* **스마트 확대/축소**: 마우스 휠로 확대/축소해도 최신 데이터 포인트가 계속 뷰에 표시됩니다
* **사용자 지정 가능한 그리드**: 메트릭 및 시스템 디스플레이의 행과 열 수를 설정할 수 있습니다
* **읽기 쉬운 형식**: 시스템 메트릭을 이해하기 쉬운 단위로 표시합니다(예: &quot;GPU 사용량 (%)&quot;, &quot;사용된 RAM (GB)&quot;)

LEET는 3개의 패널 레이아웃을 제공합니다.

* **왼쪽 사이드바**: 환경 변수, 설정, 요약 통계를 보여주는 run Overview
* **가운데 패널**: 로깅된 메트릭을 점자 스타일 선 차트로 보여주는 메트릭 그리드
* **오른쪽 사이드바**: GPU 사용량, RAM, CPU 및 기타 시스템 상태 지표를 표시하는 시스템 메트릭
