---
title: 셀프 관리형 설치 검증
---

설치를 검증하려면 [W&amp;B CLI](/ko/models/ref/cli/) 사용을 권장합니다. `verify` 명령은 모든 컴포넌트와 설정을 검증하는 여러 테스트를 실행합니다.

<Note>
  이 단계에서는 첫 번째 관리자 사용자 계정이 브라우저를 통해 생성되었다고 가정합니다.
</Note>

설치를 검증하려면 다음 단계를 따르세요:

1. W&amp;B CLI를 설치합니다:

```bash
pip install wandb
```

2. W&amp;B에 로그인합니다:

```bash
wandb login --host=https://YOUR_DNS_DOMAIN
```

예를 들어:

```bash
wandb login --host=https://wandb.company-name.com
```

3. 설치를 확인하세요:

```bash
wandb verify
```

설치가 성공적으로 완료되고 W&amp;B 배포가 완전히 정상적으로 동작하면 다음과 같은 출력이 표시됩니다:
