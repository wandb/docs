```python
"""
이전에 저장된 runs에 태그를 하나 이상 추가합니다.

run이 완료된 후 또는 run 프로세스 외부에서 작업할 때
Public API를 사용하여 저장된 데이터의 태그를 업데이트합니다.
이는 runs를 소급하여 정리하거나 분류해야 할 때 유용합니다.

run에 새 태그를 추가하려면 기존 태그와 새 태그를 포함하는
새 목록으로 `tags` 속성을 업데이트하세요.
run의 경로는 `entity/project/run_id`로 구성됩니다.
태그를 업데이트한 후 run의 `update()` 메서드를 호출하여 변경 사항을 저장해야 합니다.
"""
import wandb

entity = "<entity>"
project = "<project>"
run_id = "<run-id>"  # Replace with the ID of the run you want to update

# Path consists of entity/project/run_id
with wandb.Api().run(f"{entity}/{project}/{run_id}") as run:
  run.tags.append("<tag>")
  run.update()
```
