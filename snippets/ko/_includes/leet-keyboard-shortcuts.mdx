<div id="workspace">
  ### Workspace
</div>

| 키                           | 액션                                      |
| --------------------------- | --------------------------------------- |
| `h`, `?`                    | 도움말을 표시/숨깁니다.                           |
| `q`, `ctrl+c`               | 종료합니다.                                  |
| `alt+r`                     | LEET를 다시 시작합니다.                         |
| `esc`                       | Runs 목록으로 포커스를 이동합니다.                   |
| `enter`                     | 현재 강조 표시된 run을 단일 run 뷰에서 엽니다.          |
| `1`                         | 메트릭 그리드를 표시/숨깁니다.                       |
| `[`                         | Runs 사이드바를 표시/숨깁니다.                     |
| `2`                         | 시스템 메트릭 패널을 표시/숨깁니다.                    |
| `]`                         | run 개요 사이드바를 표시/숨깁니다.                   |
| `3`                         | 미디어 패널을 표시/숨깁니다.                        |
| `4`                         | 콘솔 로그 패널을 표시/숨깁니다.                      |
| `f`                         | Runs를 이름 또는 메타데이터로 필터링합니다.              |
| `ctrl+f`                    | Runs 필터를 지웁니다.                          |
| `space`                     | run을 선택하거나 선택 해제합니다.                    |
| `p`                         | run을 고정하거나 고정 해제합니다.                    |
| `/`                         | 메트릭을 필터링합니다.                            |
| `\`                         | 시스템 메트릭을 필터링합니다.                        |
| `ctrl+/`, `ctrl+l`          | 메트릭 필터를 지웁니다.                           |
| `ctrl+\`                    | 시스템 메트릭 필터를 지웁니다.                       |
| `o`                         | run 개요 항목을 필터링합니다.                      |
| `ctrl+o`                    | 개요 필터를 지웁니다.                            |
| `y`                         | 포커스된 차트 모드를 차례로 전환합니다.                   |
| `c`                         | 포커스된 그리드의 열을 설정합니다.                     |
| `r`                         | 포커스된 그리드의 행을 설정합니다.                     |
| `tab`, `shift+tab`          | 표시 중인 패널 사이에서 포커스를 순환합니다.               |
| `w`, `a`, `s`, `d` 또는 화살표 키 | 포커스된 패널 내에서 이동합니다.                      |
| `N`, `pgup`                 | 이전 페이지로 이동합니다.                          |
| `n`, `pgdown`               | 다음 페이지로 이동합니다.                          |
| `home`                      | 첫 번째 항목, 첫 페이지 또는 첫 번째 미디어 프레임으로 이동합니다. |
| `end`                       | 마지막 항목, 마지막 페이지 또는 최신 미디어 프레임으로 이동합니다.  |

<div id="mouse">
  ### 마우스
</div>

| 입력                   | 액션                          |
| -------------------- | --------------------------- |
| 클릭                   | 차트에 포커스를 두거나 미디어 타일을 선택합니다. |
| 휠                    | 포커스된 차트를 확대/축소합니다.          |
| 오른쪽 클릭 후 드래그         | 차트에서 가장 가까운 지점을 확인합니다.      |
| `alt` + 오른쪽 클릭 후 드래그 | 표시된 모든 차트를 동기화하여 함께 확인합니다.  |
| `shift` + 드래그        | 터미널 텍스트를 선택합니다.             |
