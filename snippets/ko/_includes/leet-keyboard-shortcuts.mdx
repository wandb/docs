---
title: Leet 키보드 단축키
---

<div id="general">
  ### 일반
</div>

| **Key** | **Description** |
| :--- | :--- |
| `h`, `?` | 이 도움말 화면 표시/숨기기 |
| `q`, `ctrl+c` | 종료하기 |
| `alt+r` | 실행 데이터 새로 고침 |

<div id="panels">
  ### 패널
</div>

| **키** | **설명** |
| :--- | :--- |
| `[` | 실행 개요가 표시되는 왼쪽 사이드바 표시/숨기기 |
| `]` | 시스템 메트릭이 표시되는 오른쪽 사이드바 표시/숨기기 |

<div id="navigation">
  ### 탐색
</div>

| **키** | **설명** |
| :--- | :--- |
| `N`, `pgup` | 차트 페이지 간 이동 |
| `n`, `pgdown` | 차트 페이지 간 이동 |
| `alt+N`, `alt+pgup` | 시스템 메트릭 페이지 간 이동 |
| `alt+n`, `alt+pgdown` | 시스템 메트릭 페이지 간 이동 |

<div id="charts">
  ### 차트
</div>

| **단축키** | **설명** |
| :--- | :--- |
| `/` | 패턴으로 메트릭을 필터링합니다. (glob 또는 부분 문자열 일치를 지원합니다.) |
| `ctrl+l` | 현재 적용된 필터를 지웁니다. |

<div id="run-overview">
  ### 실행 개요
</div>

| **Key** | **Description** |
| :--- | :--- |
| `o` | 개요 항목 필터링 (`@e`, `@c`, `@s`를 사용해 environment/config/summary 섹션 필터링 지원) |
| `ctrl+k` | 개요 필터 초기화 |

<div id="configuration">
  ### 구성
</div>

| **Key** | **설명** |
| :--- | :--- |
| `c` | 메트릭 그리드 열 설정 |
| `r` | 메트릭 그리드 행 설정 |
| `C` | 시스템 그리드 열 설정 (Shift+c) |
| `R` | 시스템 그리드 행 설정 (Shift+r) |

<div id="run-overview-navigation-when-sidebar-open">
  ### 실행 개요 탐색 (사이드바가 열려 있을 때)
</div>

| **Key** | **Description** |
| :--- | :--- |
| `up`, `down` | 섹션 내 항목 이동 |
| `tab`, `shift+tab` | 섹션 간 전환 |
| `left`, `right` | 섹션 내 페이지 이동 |
| `@e`, `@c`, `@s` | 특정 섹션만 보기 (environment/config/summary) |

<div id="mouse">
  ### 마우스
</div>

| **동작** | **설명** |
| :--- | :--- |
| `mouse wheel` | 포커스된 차트 확대/축소 |
| `shift+mouse select` | 텍스트 선택 |
