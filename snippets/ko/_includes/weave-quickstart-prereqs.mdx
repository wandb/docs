<div id="prerequisites">
  ## 사전 준비 사항
</div>

- [W&B 계정](https://wandb.ai/signup)
- Python 3.8+ 또는 Node.js 18+
- 필수 패키지 설치:
  - **Python**: `pip install weave openai`
  - **TypeScript**: `npm install weave openai`
- 환경 변수로 설정한 [OpenAI API 키](https://platform.openai.com/api-keys)
