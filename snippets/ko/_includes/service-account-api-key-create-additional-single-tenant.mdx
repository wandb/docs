---
title: Service Account Api Key Create Additional Single Tenant
---

서비스 계정 소유의 API 키를 생성하려면:

1. 팀 또는 조직 설정의 **Service Accounts** 탭으로 이동하세요.
2. 목록에서 서비스 계정을 찾으세요.
3. 작업 메뉴(`...`)를 클릭한 후 **Create API key**를 클릭하세요.
4. API 키의 이름을 입력한 후 **Create**를 클릭하세요.
5. 표시된 API 키를 즉시 복사하여 안전한 곳에 보관하세요.
6. **Done**을 클릭하세요.

하나의 서비스 계정에 대해 여러 API 키를 생성하여 서로 다른 환경이나 워크플로를 지원할 수 있습니다.
