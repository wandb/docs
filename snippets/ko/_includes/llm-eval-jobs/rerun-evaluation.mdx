<div id="re-run-an-evaluation-job">
  ## 평가 잡 다시 실행하기
</div>

상황에 따라 평가 잡을 다시 실행하거나 해당 구성을 확인하는 방법은 여러 가지가 있습니다.

- 마지막 평가 잡을 다시 실행하려면 [모델 평가하기](#evaluate-your-model)의 단계를 따르세요. 대상 프로젝트를 선택하면, 지난번에 선택했던 모델 아티팩트 상세 정보와 벤치마크 항목이 자동으로 채워집니다. 필요하다면 값을 수정한 뒤 평가 잡을 실행하세요.
- 프로젝트의 **Runs** 탭 또는 run 선택기에서 평가 잡을 다시 실행하려면, Run 이름 위에 마우스를 올리고 재생 아이콘을 클릭하세요. 잡 구성 드로어가 미리 채워진 설정과 함께 열립니다. 필요하다면 설정을 수정한 뒤 **Launch**를 클릭하세요.
- 다른 프로젝트에서 평가 잡을 다시 실행하려면, 해당 구성을 가져오세요:
    1. [모델 평가하기](#evaluate-your-model)의 단계를 따르세요. 대상 프로젝트를 선택한 후 **Import configuration**을 클릭합니다.
    1. 가져올 평가 잡이 포함된 프로젝트를 선택한 다음, 해당 평가 잡 실행을 선택합니다. 잡 구성 드로어가 미리 채워진 설정과 함께 열립니다.
    1. 필요하다면 구성을 수정합니다.
    1. **Launch**를 클릭합니다.
