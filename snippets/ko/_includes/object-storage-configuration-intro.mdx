<div id="configure-wb-to-use-your-bucket">
  ### W&amp;B에서 버킷 사용 설정
</div>

버킷 준비가 완료되면 W&amp;B가 해당 버킷을 사용하도록 설정합니다:

* **Self-Managed 배포**: Helm values 설정에 대해서는 [Operator configuration guide](/ko/platform/hosting/self-managed/operator#object-storage-bucket)를 참조하세요.
* **Dedicated Cloud**: [System Console](/ko/platform/hosting/iam/sso#system-console)에서 설정하세요.
* **Multi-tenant Cloud**: 팀을 생성하거나 수정할 때 설정하세요.
