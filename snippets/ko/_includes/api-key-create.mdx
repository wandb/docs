API 키를 생성하려면 자세한 내용을 보려면 **개인 API 키** 또는 **서비스 계정 API 키** 탭을 선택하세요.

<Tabs>
  <Tab title="개인 API 키">
    사용자 ID에 소유권이 있는 개인 API 키를 생성하려면:

    1. W&amp;B에 로그인한 후 사용자 프로필 아이콘을 클릭하고 **User Settings**를 클릭합니다.
    2. **Create new API key**를 클릭합니다.
    3. API 키를 식별할 수 있는 설명적인 이름을 입력합니다.
    4. **Create**를 클릭합니다.
    5. 표시된 API 키를 즉시 복사하여 안전한 위치에 보관합니다.
  </Tab>

</Tabs>
