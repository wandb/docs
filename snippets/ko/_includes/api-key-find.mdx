---
title: API Key 찾기
---

<Note>
**Dedicated Cloud** 및 **Self-Managed** 환경에서:
- 조직 관리자는 조직의 모든 사용자 및 서비스 계정에 대한 API 키를 조회하거나 목록을 확인할 수 있습니다.
- 팀 관리자는 자신이 관리하는 팀의 서비스 계정에 대한 API 키를 조회하거나 목록을 확인할 수 있습니다.
- 비관리자 사용자는 자신의 API 키를 조회하거나 목록을 확인할 수 있습니다.
</Note>

API 키를 찾으려면 **Personal API key** 또는 **Service Account API key** 탭을 선택해 자세한 내용을 확인합니다.

<Tabs>
<Tab title="Personal API key">
사용자 ID가 소유한 개인 API 키를 찾으려면:

1. W&B에 로그인한 뒤 사용자 프로필 아이콘을 클릭하고 **User Settings**를 클릭합니다.
2. **API Keys** 섹션까지 스크롤합니다.

</Tab>
<Tab title="Service account API key">
조직 또는 팀 서비스 계정이 소유한 API 키를 찾으려면:

1. 조직 설정에서 **API Keys** 탭으로 이동합니다.
2. 목록에서 API 키를 찾습니다. 소유자, 키 이름 또는 키 ID로 목록을 검색하거나 필터링할 수 있습니다.

</Tab>
</Tabs>

API 키 테이블에는 식별을 위해 키 ID(각 API 키의 첫 번째 부분)만 표시됩니다. 전체 비밀 API 키는 생성 시 한 번만 표시됩니다. 기존 키를 사용해야 하지만 전체 비밀 키를 따로 저장해 두지 않았다면 새 API 키를 생성해야 합니다.
