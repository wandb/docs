---
title: API 키 찾기
---

<Note>
  **Dedicated Cloud** 및 **Self-Managed**에서는 다음과 같습니다.

  * 조직 관리자는 조직의 모든 사용자와 서비스 계정에 대한 API 키를 찾거나 목록으로 확인할 수 있습니다.
  * 팀 관리자는 자신이 관리하는 Teams 내 서비스 계정의 API 키를 찾거나 목록으로 확인할 수 있습니다.
  * 관리자가 아닌 사용자는 자신의 API 키를 찾거나 목록으로 확인할 수 있습니다.
</Note>

API 키를 찾으려면 아래의 **개인 API 키** 또는 **서비스 계정 API 키** 탭을 선택해 자세한 내용을 확인하세요.

<Tabs>
  <Tab title="개인 API 키">
    사용자 ID에 속한 개인 API 키를 찾으려면 다음 단계를 따르세요.

    1. W&amp;B에 로그인한 다음 사용자 프로필 아이콘을 클릭하고 **User Settings**를 클릭합니다.
    2. **API 키** 섹션까지 스크롤합니다.
  </Tab>

  <Tab title="서비스 계정 API 키">
    조직 또는 팀 서비스 계정에 속한 API 키를 찾으려면 다음 단계를 따르세요.

    1. 조직 설정의 **API 키** 탭으로 이동합니다.
    2. 목록에서 API 키를 찾습니다. 소유자, 키 이름 또는 키 ID를 기준으로 목록을 검색하거나 필터링할 수 있습니다.
  </Tab>
</Tabs>

API 키 표에는 식별용으로 키 ID(각 API 키의 첫 번째 부분)가 표시됩니다. 전체 비밀 API 키는 생성 시 한 번만 표시됩니다. 기존 키를 사용해야 하지만 전체 비밀 키를 저장해 두지 않았다면 새 API 키를 만들어야 합니다.
