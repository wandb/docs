<Note>자세한 내용은 [Supported Server releases](https://docs.wandb.ai/release-notes/server-releases) 및 [Archived Server releases](https://docs.wandb.ai/release-notes/server-releases-archived)를 참조하세요. W&amp;B Server 릴리스는 최초 출시일로부터 12개월 동안 지원됩니다. 참고로, [Self-Managed](/ko/platform/hosting/hosting-options/self-managed/)를 사용하는 고객은 지원을 유지하기 위해 적시에 지원되는 릴리스로 업그레이드할 책임이 있습니다.</Note>
