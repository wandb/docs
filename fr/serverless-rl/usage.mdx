Utilisez Serverless RL via le [framework ART d&#39;OpenPipe](https://art.openpipe.ai/getting-started/about) et l&#39;[API Serverless RL](/fr/serverless-rl/api-reference).

Pour commencer à utiliser Serverless RL, assurez-vous de remplir les [prérequis](/fr/serverless-rl/prerequisites) pour utiliser les outils W&amp;B, puis suivez le [démarrage rapide](https://art.openpipe.ai/getting-started/quick-start) d&#39;ART.

* Pour obtenir des exemples de code et des flux de travail, consultez le [notebook Google Colab](https://colab.research.google.com/github/openpipe/art-notebooks/blob/main/examples/2048/2048.ipynb).
* Pour en savoir plus sur les points de terminaison de l&#39;API de Serverless RL, consultez la [référence de l&#39;API Serverless RL](/fr/serverless-rl/api-reference).
