<Note>
  L&#39;API Serverless RL fournit des points de terminaison pour gérer les tâches d&#39;entraînement, y compris l&#39;apprentissage par renforcement (RL) serverless et le fine-tuning supervisé (SFT). L&#39;API est compatible avec l&#39;API OpenAI pour les complétions de chat.
</Note>

Utilisez cette référence pour consulter les exigences d&#39;authentification, l&#39;URL de base et les points de terminaison disponibles pour les complétions de chat, les modèles, les tâches d&#39;entraînement et les contrôles de santé.&quot;

<div id="authentication">
  ## Authentification
</div>

Toutes les requêtes API doivent être authentifiées à l’aide de votre clé API W&amp;B. Créez une clé API à l’adresse [wandb.ai/settings](https://wandb.ai/settings).

Incluez votre clé API dans l’en-tête `Authorization` :

```http
Authorization: Bearer YOUR_API_KEY
```

<div id="base-url">
  ## URL de base
</div>

```text
https://api.training.wandb.ai/v1
```

<div id="available-endpoints">
  ## Points de terminaison disponibles
</div>

Les sections suivantes décrivent les points de terminaison disponibles, regroupés par type de ressource.

<div id="chat-completions">
  ### Complétions de chat
</div>

* **[POST /v1/chat/completions](https://docs.wandb.ai/serverless-rl/api-reference/chat-completions/create-chat-completion-1)** - Créer une complétion de chat
* **[POST /v1/chat/completions/](https://docs.wandb.ai/serverless-rl/api-reference/chat-completions/create-chat-completion)** - Créer une complétion de chat

<div id="models">
  ### Modèles
</div>

* **[POST /v1/preview/models](https://docs.wandb.ai/serverless-rl/api-reference/models/create-model)** - Créer un modèle
* **[DELETE /v1/preview/models/{model_id}](https://docs.wandb.ai/serverless-rl/api-reference/models/delete-model)** - Supprimer un modèle
* **[DELETE /v1/preview/models/{model_id}/checkpoints](https://docs.wandb.ai/serverless-rl/api-reference/models/delete-model-checkpoints)** - Supprimer les points de contrôle du modèle
* **[GET /v1/preview/models/{model_id}/checkpoints](https://docs.wandb.ai/serverless-rl/api-reference/models/list-model-checkpoints)** - Lister les points de contrôle du modèle
* **[POST /v1/preview/models/{model_id}/log](https://docs.wandb.ai/serverless-rl/api-reference/models/log)** - Journal

<div id="training-jobs">
  ### Tâches d&#39;entraînement
</div>

* **[POST /v1/preview/sft-training-jobs](https://docs.wandb.ai/serverless-rl/api-reference/training-jobs/create-sft-training-job)** - Créer une tâche d&#39;entraînement SFT
* **[POST /v1/preview/training-jobs](https://docs.wandb.ai/serverless-rl/api-reference/training-jobs/create-rl-training-job)** - Créer une tâche d&#39;entraînement RL
* **[GET /v1/preview/training-jobs/{training_job_id}](https://docs.wandb.ai/serverless-rl/api-reference/training-jobs/get-training-job)** - Obtenir une tâche d&#39;entraînement
* **[GET /v1/preview/training-jobs/{training_job_id}/events](https://docs.wandb.ai/serverless-rl/api-reference/training-jobs/get-training-job-events)** - Obtenir les événements de la tâche d&#39;entraînement
