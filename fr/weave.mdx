W&amp;B Weave est une plateforme d&#39;observabilité et d&#39;évaluation qui vous aide à suivre, évaluer et améliorer votre application LLM. Avec Weave, vous pouvez :

* [Observer et déboguer](/fr/weave/quickstart) votre application LLM
* [Évaluer](/fr/weave/tutorial-eval) les réponses de votre application à l&#39;aide de juges LLM et d&#39;évaluateurs personnalisés

<div id="install-weave">
  ## Installer Weave
</div>

W&amp;B Weave fournit des bibliothèques Python et TypeScript. Pour installer la bibliothèque Weave, exécutez la commande suivante :

<Tabs>
  <Tab title="Python">
    ```bash
    pip install weave
    ```
  </Tab>

  <Tab title="TypeScript">
    ```bash
    pnpm install weave
    ```
  </Tab>
</Tabs>

Pour commencer à utiliser la bibliothèque Weave, créez un [compte Weights &amp; Biases (W&amp;B)](https://wandb.ai) ainsi qu&#39;une [clé API dans Paramètres utilisateur](https://wandb.ai/settings). La clé API vous permet de vous authentifier auprès de votre compte W&amp;B et de commencer à y envoyer des données.
