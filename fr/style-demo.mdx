<div id="notes">
  ## Notes
</div>

* Tous les styles s’adaptent automatiquement au mode sombre
* Les couleurs sont définies sous forme de variables CSS pour faciliter la maintenance
* Les gradients utilisent systématiquement les couleurs de la marque
* Les effets de survol fournissent un retour visuel interactif
