---
description: "Exportez les runs, les métriques, les Artifacts et les Reports depuis une instance W&B Cloud dédié à l'aide de l'API du SDK Python."
title: Exporter des données depuis Cloud dédié
---

Si vous souhaitez exporter toutes les données gérées dans votre instance Cloud dédié, vous pouvez utiliser l&#39;API du SDK W&amp;B pour extraire les runs, les métriques, les Artifacts, et bien plus encore avec l&#39;[Import and Export API](/fr/models/ref/python/public-api/). Le tableau suivant présente quelques-uns des principaux cas d&#39;utilisation de l&#39;export.

| Objectif                           | Documentation                                                                                                                                                                                                 |
| ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Exporter les métadonnées du projet | [Projects API](/fr/models/ref/python/public-api/projects)                                                                                                                                                        |
| Exporter les runs d&#39;un projet  | [Runs API](/fr/models/ref/python/public-api/runs)                                                                                                                                                                |
| Exporter les Reports               | [Report and Workspace API](/fr/models/reports/clone-and-export-reports/)                                                                                                                                         |
| Exporter des Artifacts             | [Explore artifact graphs](/fr/models/artifacts/explore-and-traverse-an-artifact-graph/), [Download and use artifacts](/fr/models/artifacts/download-and-use-an-artifact/#download-and-use-an-artifact-stored-on-wb) |

Si vous gérez des Artifacts stockés dans le Cloud dédié avec [Secure Storage Connector](/fr/platform/app/settings-page/teams/#secure-storage-connector), vous n&#39;aurez peut-être pas besoin d&#39;exporter les Artifacts à l&#39;aide de l&#39;API du SDK W&amp;B.

<Note>
  L&#39;utilisation de l&#39;API du SDK W&amp;B pour exporter l&#39;ensemble de vos données peut être lente si vous avez un grand nombre de runs, d&#39;Artifacts, etc. W&amp;B recommande d&#39;exécuter le processus d&#39;export par lots de taille appropriée afin de ne pas surcharger votre instance Cloud dédié.
</Note>
