---
description: "Exportez les runs, les métriques, les artefacts et les rapports d'une instance W&B Cloud dédié à l'aide de l'API du SDK Python."
title: Exporter des données depuis Cloud dédié
---

Si vous souhaitez exporter toutes les données gérées dans votre instance Cloud dédié, vous pouvez utiliser l&#39;API du SDK Python de W&amp;B pour extraire les runs, les métriques, les artefacts, etc., à l&#39;aide de l&#39;[API d&#39;importation et d&#39;exportation](/fr/models/ref/python/public-api/). Le tableau suivant présente quelques-uns des principaux cas d&#39;utilisation de l&#39;exportation.

| Objectif                                 | Documentation                                                                                                                                                                                                                         |
| ---------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Exporter les métadonnées d&#39;un projet | [API Projects](/fr/models/ref/python/public-api/projects)                                                                                                                                                                                |
| Exporter les runs d&#39;un projet        | [API Runs](/fr/models/ref/python/public-api/runs)                                                                                                                                                                                        |
| Exporter les rapports                    | [API Report and Workspace](/fr/models/reports/clone-and-export-reports/)                                                                                                                                                                 |
| Exporter les artefacts                   | [Explorer les graphes d&#39;artefacts](/fr/models/artifacts/explore-and-traverse-an-artifact-graph/), [Télécharger et utiliser des artefacts](/fr/models/artifacts/download-and-use-an-artifact/#download-and-use-an-artifact-stored-on-wb) |

Si vous gérez des artefacts stockés dans le Cloud dédié avec [Secure Storage Connector](/fr/platform/app/settings-page/teams/#secure-storage-connector), il se peut que vous n&#39;ayez pas besoin d&#39;exporter les artefacts à l&#39;aide de l&#39;API du SDK Python de W&amp;B.

<Note>
  L&#39;utilisation de l&#39;API du SDK Python de W&amp;B pour exporter l&#39;ensemble de vos données peut être lente si vous avez un grand nombre de runs, d&#39;artefacts, etc. W&amp;B recommande d&#39;exécuter le processus d&#39;exportation par lots de taille appropriée afin de ne pas surcharger votre instance Cloud dédié.
</Note>
