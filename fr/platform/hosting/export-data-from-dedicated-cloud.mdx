---
description: "Exporter des runs, des métriques, des artefacts et des rapports depuis une instance de Cloud dédié de W&B en utilisant l'API du SDK W&B."
title: Exporter des données du Cloud dédié
---

Si vous souhaitez exporter toutes les données gérées dans votre instance de Cloud dédié, vous pouvez utiliser l&#39;API du SDK W&amp;B pour extraire les runs, les métriques, les artefacts, et plus encore à l&#39;aide de l&#39;[API d&#39;importation et d&#39;exportation](/fr/models/ref/python/public-api/). Le tableau suivant couvre certains des principaux cas d&#39;usage d&#39;exportation.

| Objectif                           | Documentation                                                                                                                                                                                                                         |
| ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Exporter les métadonnées du projet | [API Projects](/fr/models/ref/python/public-api/projects)                                                                                                                                                                                |
| Exporter les runs dans un projet   | [API Runs](/fr/models/ref/python/public-api/runs)                                                                                                                                                                                        |
| Exporter les rapports              | [API Report and Workspace](/fr/models/reports/clone-and-export-reports/)                                                                                                                                                                 |
| Exporter les artefacts             | [Explorer les graphes d&#39;artefacts](/fr/models/artifacts/explore-and-traverse-an-artifact-graph/), [Télécharger et utiliser des artefacts](/fr/models/artifacts/download-and-use-an-artifact/#download-and-use-an-artifact-stored-on-wb) |

Si vous gérez des artefacts stockés dans le Cloud dédié avec [Connecteur de stockage sécurisé](/fr/platform/app/settings-page/teams/#secure-storage-connector), vous n&#39;aurez peut-être pas besoin d&#39;exporter les artefacts en utilisant l&#39;API du SDK W&amp;B.

<Note>
  Utiliser l&#39;API du SDK W&amp;B pour exporter toutes vos données peut être lent si vous avez un grand nombre de runs, d&#39;artefacts, etc. W&amp;B recommande d&#39;exécuter le processus d&#39;exportation par lots de taille appropriée afin de ne pas surcharger votre instance de Cloud dédié.
</Note>
