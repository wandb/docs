---
title: Comment puis-je résoudre une erreur « permission denied » dans Launch ?
---

Si vous rencontrez le message d&#39;erreur `Launch Error: Permission denied`, cela indique que vous ne disposez pas des autorisations suffisantes pour enregistrer des runs dans le projet souhaité. Les causes possibles sont les suivantes :

1. Vous n&#39;êtes pas connecté sur cette machine. Exécutez [`wandb login`](/fr/models/ref/cli/wandb-login) dans la ligne de commande.
2. L&#39;entité spécifiée n&#39;existe pas. L&#39;entité doit être votre nom d&#39;utilisateur ou le nom d&#39;une équipe existante. Créez une équipe si nécessaire depuis la [page de facturation](https://app.wandb.ai/billing).
3. Vous ne disposez pas des autorisations nécessaires sur le projet. Demandez au créateur du projet de définir le paramètre de confidentialité sur **Open** pour autoriser l&#39;enregistrement de runs dans le projet.
