---
title: Comment puis-je résoudre une erreur « permission denied » dans Launch ?
---

Si vous rencontrez le message d’erreur `Launch Error: Permission denied`, cela indique que vous ne disposez pas des autorisations nécessaires pour enregistrer dans le projet souhaité. Les causes possibles sont les suivantes :

1. Vous n’êtes pas connecté sur cette machine. Exécutez [`wandb login`](/fr/models/ref/cli/wandb-login) dans la ligne de commande.
2. L’entité spécifiée n’existe pas. L’entité doit être votre nom d’utilisateur ou le nom d’une équipe existante. Créez une équipe si nécessaire depuis la page [Abonnements](https://app.wandb.ai/billing).
3. Vous n’avez pas les autorisations nécessaires sur le projet. Demandez au créateur du projet de modifier le paramètre de confidentialité sur **Open** pour autoriser l’enregistrement de runs dans le projet.
