---
title: Existe-t-il de bonnes pratiques pour utiliser Launch efficacement ?
keywords: ["conseils Launch", "compte de service Launch", "bonnes pratiques pour la file d'attente", "configuration de l'agent"]
---

Suivez ces bonnes pratiques pour éviter les erreurs de configuration courantes et conserver des flux de travail Launch faciles à maintenir :

* Créez la file d&#39;attente avant de démarrer l&#39;agent afin de simplifier la configuration. Sinon, l&#39;agent renverra des erreurs tant que vous n&#39;aurez pas ajouté de file d&#39;attente.
* Créez un compte de service W&amp;B pour démarrer l&#39;agent afin qu&#39;il ne soit pas associé au compte d&#39;un utilisateur en particulier.
* Utilisez `wandb.Run.config` pour gérer les hyperparamètres, ce qui vous permet d&#39;écraser des valeurs lors de la réexécution d&#39;un job. Pour plus d&#39;informations sur l&#39;utilisation de `argparse`, voir [Définir la configuration avec argparse](/fr/models/track/config/#set-the-configuration-with-argparse).
