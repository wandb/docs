---
title: Comment puis-je contrôler qui peut envoyer dans une file d'attente ?
keywords: ["autorisations de file d’attente", "restreindre l’accès à la file d’attente", "file d’attente d’équipe", "envoyer dans une file d’attente"]
---

Les files d’attente sont propres à une équipe Users. Définissez l’entité propriétaire lorsque vous créez la file d’attente. Pour restreindre l’accès, modifiez la composition de l’équipe.
