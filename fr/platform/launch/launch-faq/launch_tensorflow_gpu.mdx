---
title: Comment faire fonctionner W&B Launch avec TensorFlow sur GPU
---

Pour exécuter des jobs TensorFlow sur des GPU avec W&amp;B Launch, spécifiez une image de base personnalisée pour la construction du conteneur. Cela permet au conteneur résultant d’accéder au GPU. Ajoutez un tag d’image sous la clé `builder.accelerator.base_image` dans la configuration des ressources. Par exemple :

```json
{
    "gpus": "all",
    "builder": {
        "accelerator": {
            "base_image": "tensorflow/tensorflow:latest-gpu"
        }
    }
}
```

<Note>
  Dans les versions antérieures à W&amp;B 0.15.6, utilisez `cuda` au lieu de `accelerator` comme clé parente de `base_image`.
</Note>
