---
title: "Je ne veux pas que W&B génère un conteneur pour moi : puis-je quand même utiliser Launch ?"
---

Pour lancer une image Docker déjà créée, remplacez les éléments entre `<>` par vos valeurs et exécutez :

```bash
wandb launch -d <docker-image-uri> -q <queue-name> -E <entrypoint>
```

Cette commande crée un job et lance un run.

Pour créer un job à partir d’une image, exécutez :

```bash
wandb job create image <image-name> -p <project> -e <entity>
```
