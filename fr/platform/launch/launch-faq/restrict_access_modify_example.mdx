---
title: Comment les administrateurs peuvent-ils restreindre les utilisateurs disposant d’un accès en modification ?
---

Contrôlez l’accès à certains champs de la file d’attente pour les utilisateurs qui ne sont pas administrateurs d’équipe à l’aide des [modèles de configuration de file d’attente](/fr/platform/launch/setup-queue-advanced/). Les administrateurs d’équipe définissent les champs que les utilisateurs non administrateurs peuvent voir et fixent les limites de modification. Seuls les administrateurs d’équipe peuvent créer ou modifier des files d’attente.
