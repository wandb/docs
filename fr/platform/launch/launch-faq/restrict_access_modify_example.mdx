---
title: Comment les administrateurs peuvent-ils restreindre l’accès en modification à certains utilisateurs ?
---

Contrôlez l’accès à certains champs de file d’attente pour les utilisateurs qui ne sont pas administrateurs d’équipe grâce aux [modèles de configuration de file d&#39;attente](/fr/platform/launch/setup-queue-advanced/). Les administrateurs d’équipe définissent quels champs les utilisateurs non administrateurs peuvent voir et fixent les limites de modification. Seuls les administrateurs d’équipe peuvent créer ou modifier des files d’attente.
