Classe de l&#39;agent Launch qui interroge les files d&#39;attente de runs spécifiées et lance des runs pour wandb launch.

```python
LaunchAgent(
    api: Api,
    config: Dict[str, Any]
)
```

| Arguments | Description                                                       |
| :-------- | :---------------------------------------------------------------- |
| `api`     | Objet API à utiliser pour effectuer des requêtes vers le backend. |
| `config`  | Dictionnaire de configuration de l’agent.                         |

| Attributs                | Description                                                                                            |
| :----------------------- | :----------------------------------------------------------------------------------------------------- |
| `num_running_jobs`       | Renvoie le nombre de jobs, sans compter les ordonnanceurs.                                             |
| `num_running_schedulers` | Renvoie uniquement le nombre d’ordonnanceurs.                                                          |
| `thread_ids`             | Renvoie une liste des clés correspondant aux identifiants des threads en cours d’exécution de l’agent. |
