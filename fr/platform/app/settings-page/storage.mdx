title: Gérer le stockage
---

Si vous approchez de votre limite de stockage ou que vous la dépassez, plusieurs solutions s’offrent à vous pour gérer vos données. La solution la plus adaptée dépend de votre type de compte et de la configuration actuelle de votre projet.&quot;

<div id="manage-storage-consumption">
  ## Gérer la consommation de stockage
</div>

W&amp;B propose différentes méthodes pour optimiser votre consommation de stockage :

* Utilisez des [artefacts de référence](/fr/models/artifacts/track-external-files/) pour suivre les fichiers enregistrés en dehors du système W&amp;B, au lieu de les téléverser vers le stockage W&amp;B.
* Utilisez un [bucket de stockage cloud externe](/fr/platform/app/settings-page/teams/) pour le stockage. *(Enterprise uniquement)*

<div id="delete-data">
  ## Supprimer des données
</div>

Vous pouvez également choisir de supprimer des données pour rester en dessous de votre limite de stockage. Il existe plusieurs façons de procéder :

* Supprimez des données de manière interactive dans l&#39;interface utilisateur de l&#39;application.
* [Définissez une politique TTL](/fr/models/artifacts/ttl/) sur Artifacts afin qu&#39;ils soient supprimés automatiquement.
