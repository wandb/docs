[Cline](https://cline.bot) est un assistant de codage basé sur l&#39;IA. Ce tutoriel explique comment configurer Cline pour utiliser les modèles fournis par W&amp;B Inference.

L&#39;agent Cline est disponible sous forme d&#39;outil en ligne de commande ou d&#39;intégration avec de nombreux IDE. Cette page montre comment le configurer avec la CLI Cline et comme extension Visual Studio Code, mais la configuration dans d&#39;autres IDE devrait être similaire.

<div id="prerequisites">
  ## Prérequis
</div>

Vous aurez besoin de votre [clé API W&amp;B](../prerequisites#set-up-your-wb-account-and-project).

<div id="set-up-cline-in-the-command-line">
  ## Configurer Cline en ligne de commande
</div>

Installez le [Cline CLI](https://docs.cline.bot/cline-cli/installation).

```
npm install -g cline
```

Voir les [instructions d’installation de Cline](https://docs.cline.bot/cline-cli/installation) en cas de problème.

Ces instructions ont été testées avec la version `2.5.1` de Cline CLI. Vous pouvez exécuter `cline version` pour vérifier la version installée.

Exécutez la commande suivante en remplaçant `<your_api_key>` par votre clé API W&amp;B.

```
cline auth -k <your_api_key> -p openai -b https://api.inference.wandb.ai/v1 -m moonshotai/Kimi-K2.5
```

Cela configurera Cline pour utiliser l’endpoint compatible OpenAI de W&amp;B Inference et le modèle Kimi K2.5. Pour connaître la disponibilité des modèles et la tarification, voir [notre catalogue de modèles](https://wandb.ai/inference).

Exécutez un test simple pour vérifier que tout fonctionne :

```
cline "What is 2 + 2?"
```

Si Cline renvoie une réponse, votre installation et votre authentification sont terminées.

Pour plus d’informations sur l’utilisation de la CLI Cline, voir le [Démarrage rapide de Cline](https://docs.cline.bot/cline-cli/installation#quick-start).


<div id="set-up-cline-in-visual-studio-code">
  ## Configurer Cline dans Visual Studio Code
</div>

Vous pouvez également installer Cline en tant qu’extension pour Visual Studio Code. Recherchez **Cline** dans le Marketplace des extensions VS Code, ou installez-le depuis le [Marketplace Visual Studio Code](https://marketplace.visualstudio.com/items?itemName=saoudrizwan.claude-dev).

Cliquez sur le bouton `Install`.

<Frame>![Installation de l’extension VS Code](/images/inference/inference-tutorial-cline-install.png)</Frame>

Cliquez sur l’icône Cline nouvellement ajoutée dans votre barre d’activité pour ouvrir la barre latérale Cline. Si l’icône Cline ne s’affiche pas dans la barre d’activité, cliquez sur le menu déroulant pour voir plus d’options.

<img src="/images/inference/inference-tutorial-cline-activity-bar.png" alt="Installation de l’extension VS Code" style={{width: "60px"}} />

Sélectionnez &quot;Bring my own API key&quot;, puis cliquez sur le bouton **Continue**.

<img src="/images/inference/inference-tutorial-cline-byok.png" alt="Sélectionner Bring Your Own Key" style={{width: "300px"}} />

Renseignez les valeurs suivantes en remplaçant `<your_api_key>` par votre clé API W&amp;B :

| Paramètre                 | Valeur                            |
| ------------------------- | --------------------------------- |
| Fournisseur d’API         | OpenAI Compatible                 |
| URL de base               | https://api.inference.wandb.ai/v1 |
| Clé API OpenAI Compatible | `<your_api_key>`                  |
| ID du modèle              | moonshotai/Kimi-K2.5              |

Configurez la tarification en développant Model Configuration. Pour connaître la disponibilité et la tarification des modèles, consultez [notre catalogue de modèles](https://wandb.ai/inference).

Cliquez sur le bouton `Continue`.

Essayez d’envoyer un prompt comme &quot;Écrivez un programme Python pour calculer les 10 premiers nombres de Fibonacci&quot;.

<img src="/images/inference/inference-tutorial-cline-act.png" alt="Soumettre votre tâche de codage à l’agent" style={{width: "300px"}} />

Pour en savoir plus sur l’utilisation de Cline, consultez la [documentation Cline](https://docs.cline.bot).
