<div id="using-model-ids">
  ## Utilisation des ID de modèle
</div>

Lorsque vous utilisez l’API, spécifiez le modèle à l’aide de son `Model ID` figurant dans le tableau ci-dessus. Par exemple :

```python
response = client.chat.completions.create(
    model="meta-llama/Llama-3.1-8B-Instruct",
    messages=[...]
)
```
