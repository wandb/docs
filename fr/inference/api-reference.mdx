Découvrez comment utiliser l&#39;API W&amp;B Inférence pour accéder par programmation aux modèles de fondation.

<div id="base-url">
  ## URL de base
</div>

Accédez au service d’inférence à :

```plaintext
https://api.inference.wandb.ai/v1
```

<Note>
  **Important**

  Pour utiliser ce point de terminaison, vous devez disposer de :

  * Un compte W&amp;B avec des crédits d’Inférence
  * Une clé API W&amp;B valide

  Si vous appartenez à plus d’une équipe ou si vous souhaitez associer votre utilisation à un projet, vous aurez également besoin des identifiants d’équipe et de projet. Dans les exemples de code, ils apparaissent sous la forme `<your-team>/<your-project>`. À défaut, votre entité par défaut et le nom de projet `inference` seront utilisés.
</Note>

<div id="available-methods">
  ## Méthodes disponibles
</div>

L’API W&amp;B Inférence fournit des points de terminaison compatibles avec OpenAI pour interagir avec des modèles de fondation :

* **[Chat Completions](/fr/inference/api-reference/chat-completions)** - Créer des complétions de chat à l’aide de différents modèles de fondation
* **[List Models](/fr/inference/api-reference/list-models)** - Obtenir tous les modèles disponibles et leurs identifiants

<div id="authentication">
  ## Authentification
</div>

Toutes les requêtes API nécessitent une authentification à l’aide de votre clé API W&amp;B. Créez une clé API sur [wandb.ai/settings](https://wandb.ai/settings).

Incluez votre clé API dans les en-têtes de requête :

* Pour le SDK OpenAI : définissez le paramètre `api_key`
* Pour les appels API directs : utilisez `Authorization: Bearer <your-api-key>`
