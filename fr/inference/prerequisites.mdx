<div id="set-up-your-wb-account-and-project">
  ## Configurez votre compte et votre projet W&amp;B
</div>

Vous avez besoin des éléments suivants pour accéder à W&amp;B Inference :

1. **Un compte W&amp;B**
   Inscrivez-vous sur [W&amp;B](https://app.wandb.ai/login?signup=true)

2. **Une clé API W&amp;B**

   <ApiKeyCreate />

3. **Un projet W&amp;B**
   Créez un projet dans votre compte W&amp;B pour suivre l’utilisation

<div id="set-up-your-environment-python">
  ## Configurer votre environnement (Python)
</div>

Pour utiliser l&#39;API d&#39;Inférence avec Python, vous devez également :

1. Satisfaire aux exigences générales ci-dessus

2. Installer les bibliothèques requises :

   ```bash
   pip install openai weave
   ```

<Note>
  **Note**

  La bibliothèque `weave` est facultative, mais recommandée. Elle vous permet de tracer vos applications LLM. Pour en savoir plus, consultez le [Démarrage rapide de Weave](/fr/models/quickstart/).

  Voir les [exemples d&#39;utilisation](/fr/inference/examples/) pour obtenir des exemples de code avec W&amp;B Inference et Weave.
</Note>
