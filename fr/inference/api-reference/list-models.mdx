Obtenez la liste de tous les modèles disponibles et de leurs ID. Utilisez-la pour sélectionner des modèles dynamiquement ou vérifier ceux qui sont disponibles.

<div id="request-examples">
  ## Exemples de requêtes
</div>

<Tabs>
  <Tab title="Python" />

  <Tab title="Bash">
    ```bash
    curl https://api.inference.wandb.ai/v1/models \
      -H "Content-Type: application/json" \
      -H "Authorization: Bearer <your-api-key>" \
      -H "OpenAI-Project: <your-team>/<your-project>"
    ```
  </Tab>
</Tabs>
