Découvrez comment utiliser le service W&amp;B Inférence via l’interface web. Avant d’utiliser l’interface, assurez-vous de satisfaire aux [prérequis](/fr/inference/prerequisites/).

<div id="access-the-inference-service">
  ## Accéder au service Inférence
</div>

Vous pouvez accéder au service Inférence à partir de trois endroits :

<div id="direct-link">
  ### Lien direct
</div>

Accédez à [https://wandb.ai/inference](https://wandb.ai/inference).

<div id="from-the-inference-tab">
  ### Depuis l’onglet Inférence
</div>

1. Accédez à votre compte W&amp;B à l’adresse [https://wandb.ai/](https://wandb.ai/)
2. Sélectionnez **Inférence** dans la barre latérale gauche
3. La page affiche les modèles disponibles ainsi que des informations sur chaque modèle

<Frame>
  <img src="/images/inference/inference-playground-single.png" alt="Utilisation d’un modèle d’inférence dans le Playground" />
</Frame>

### Depuis l’onglet Playground

1. Sélectionnez **Playground** dans la barre latérale gauche. L’interface de chat du Playground s’affiche
2. Survolez **W&amp;B Inférence** dans la liste déroulante des LLM. Une liste déroulante affichant les modèles disponibles apparaît à droite
3. Dans la liste déroulante des modèles, vous pouvez :
   * Cliquer sur le nom de n’importe quel modèle pour [l’essayer dans le Playground](#try-a-model-in-the-playground)
   * [Comparer plusieurs modèles](#compare-multiple-models)

<Frame>
  <img src="/images/inference/inference-playground.png" alt="La liste déroulante des modèles d’Inférence dans le Playground" />
</Frame>

<div id="try-a-model-in-the-playground">
  ## Essayer un modèle dans le Playground
</div>

Après avoir [sélectionné un modèle](#access-the-inference-service), vous pouvez l’essayer dans le Playground. Les actions disponibles sont les suivantes :

* [Personnaliser les réglages et paramètres du modèle](/fr/weave/guides/tools/playground#customize-playground-settings)
* [Ajouter, réessayer, modifier et supprimer des messages](/fr/weave/guides/tools/playground#message-controls)
* [Enregistrer et réutiliser un modèle avec des paramètres personnalisés](/fr/weave/guides/tools/playground#saved-models)
* [Comparer plusieurs modèles](#compare-multiple-models)

<div id="compare-multiple-models">
  ## Comparer plusieurs modèles
</div>

Vous pouvez comparer côte à côte plusieurs modèles d’Inférence dans le Playground. Accédez à la vue Compare à deux endroits :

<div id="from-the-inference-tab">
  ### Depuis l’onglet Inférence
</div>

1. Sélectionnez **Inférence** dans la barre latérale gauche. La page des modèles disponibles s’affiche.
2. Cliquez n’importe où sur la carte de modèle (sauf sur son nom) pour le sélectionner. La bordure de la carte devient bleue.
3. Répétez l’opération pour chaque modèle que vous souhaitez comparer.
4. Cliquez sur **Comparer N modèles dans le Playground** sur l’une des cartes sélectionnées. `N` indique le nombre de modèles sélectionnés.
5. La vue de comparaison s’ouvre.

Vous pouvez maintenant comparer des modèles et utiliser toutes les fonctionnalités de [Essayer un modèle dans le Playground](#try-a-model-in-the-playground).

<Frame>
  <img src="/images/inference/inference-playground-compare.png" alt="Sélectionnez plusieurs modèles à comparer dans le Playground" />
</Frame>

<div id="from-the-playground-tab">
  ### Depuis l&#39;onglet Playground
</div>

1. Sélectionnez **Playground** dans la barre latérale gauche. L&#39;interface de chat du Playground s&#39;affiche
2. Survolez **W&amp;B Inférence** dans la liste déroulante des LLM. La liste déroulante des modèles s&#39;affiche à droite
3. Sélectionnez **Comparer** dans la liste déroulante. L&#39;onglet **Inférence** s&#39;affiche
4. Cliquez n&#39;importe où sur une carte de modèle (sauf sur le nom du modèle) pour la sélectionner. La bordure de la carte devient bleue
5. Répétez l&#39;opération pour chaque modèle que vous souhaitez comparer
6. Cliquez sur **Comparer N modèles dans le Playground** sur l&#39;une des cartes sélectionnées. La vue de comparaison s&#39;ouvre

Vous pouvez maintenant comparer des modèles et utiliser toutes les fonctionnalités de [Essayer un modèle dans le Playground](#try-a-model-in-the-playground).

<div id="view-billing-and-usage-information">
  ## Consulter les informations de facturation et d’utilisation
</div>

Les administrateurs de l’organisation peuvent suivre le solde de crédits, l’historique d’utilisation et les factures à venir depuis l’interface W&amp;B :

1. Accédez à la page **Billing** de W&amp;B dans l’interface
2. Trouvez la carte de facturation d’Inférence dans le coin inférieur droit
3. À partir de là, vous pouvez :
   * Cliquer sur **Voir l’utilisation** pour voir l’évolution de votre utilisation au fil du temps
   * Consulter les frais d’inférence à venir (pour les plans payants)

<Tip>
  Consultez la [page de tarification d’Inférence](https://wandb.ai/site/pricing/inference) pour voir les détails de tarification par modèle.
</Tip>
