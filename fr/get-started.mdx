<div id="product-comparison">
  ## Comparaison des produits
</div>

Bienvenue sur Weights &amp; Biases ! Avant de commencer à utiliser nos produits, il est important d’identifier ceux qui correspondent à votre cas d’utilisation.

| Produit                                   | Idéal pour                                   | Fonctionnalités clés                                                                                           |
| ----------------------------------------- | -------------------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| **[W&amp;B Models](#w%26b-models)**       | Entraîner des modèles de ML à partir de zéro | suivi des expériences, optimisation des hyperparamètres, registre de modèles, visualisations                   |
| **[W&amp;B Weave](#w%26b-weave)**         | Créer des applications LLM                   | Tracing, gestion des prompts, évaluation, suivi des coûts pour les applications d’IA en production             |
| **[W&amp;B Inference](#w%26b-inference)** | Utiliser des modèles préentraînés            | modèles open source hébergés, accès à l’API, playground de modèles pour les tests                              |
| **[W&amp;B Training](#w%26b-training)**   | Fine-tuning de modèles                       | Créer et déployer des LoRAs et des adaptations personnalisées de modèles avec l’apprentissage par renforcement |
