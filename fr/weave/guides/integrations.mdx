W&amp;B Weave fournit des intégrations de journalisation pour les fournisseurs de LLM et les frameworks d&#39;orchestration les plus répandus. Ces intégrations vous permettent de tracer en toute transparence les appels effectués via différentes bibliothèques, afin de mieux surveiller et analyser vos applications d&#39;IA.

Si vous utilisez dans votre application des bibliothèques de fournisseurs de LLM (comme OpenAI, Anthropic, Cohere ou Mistral), vous souhaitez que ces appels d&#39;API apparaissent dans W&amp;B Weave sous forme d&#39;Appels tracés : entrées, sorties, latence, utilisation des jetons et coût. Sans cela, vous devriez encapsuler chaque `client.chat.completions.create()` (ou équivalent) dans `@weave.op` ou recourir à une instrumentation manuelle, ce qui est fastidieux et rend les oublis faciles.

Weave intercepte automatiquement (applique un patch à) les bibliothèques clientes LLM prises en charge. Le code de votre application reste inchangé : vous utilisez le SDK du fournisseur comme d&#39;habitude, et chaque requête est enregistrée comme un Appel Weave. Vous bénéficiez d&#39;un traçage complet avec une configuration minimale.
