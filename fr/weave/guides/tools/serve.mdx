---
title: "Servir"
description: "Exposez vos ops et modèles Weave sous forme de points de terminaison d’API pour le service de prédictions et exécuter l’inférence en production."
---

À partir d’une référence Weave vers n’importe quel modèle Weave, vous pouvez run :

```
weave serve <ref>
```

pour run un serveur FastAPI pour ce modèle. Accédez à [http://0.0.0.0:9996/docs](http://0.0.0.0:9996/docs) pour interroger le modèle de manière interactive.


<div id="install-fastapi">
  ## Installer FastAPI
</div>

```bash
pip install fastapi uvicorn
```


<div id="serve-model">
  ## Servir le modèle
</div>

Dans un terminal, exécutez :

```bash
weave serve <your model ref>
```

Obtenez la référence de votre modèle en accédant au modèle et en la copiant dans l’interface utilisateur. Elle doit ressembler à ceci :
`weave://your_entity/project-name/YourModel:<hash>`

Pour l’utiliser, accédez au lien vers l’interface Swagger UI, cliquez sur le point de terminaison `predict`, puis sur &quot;Try it out!&quot;.
