---
title: "Serve"
description: "Exposez vos opérations et modèles Weave sous forme de points de terminaison API pour servir des prédictions et exécuter l’inférence en Production."
---

À partir d’une référence Weave vers n’importe quel modèle Weave, vous pouvez exécuter :

```
weave serve <ref>
```

pour exécuter un serveur FastAPI pour ce modèle. Accédez à [http://0.0.0.0:9996/docs](http://0.0.0.0:9996/docs) pour interroger le modèle de manière interactive.


<div id="install-fastapi">
  ## Installer FastAPI
</div>

```bash
pip install fastapi uvicorn
```


<div id="serve-model">
  ## Serve modèle
</div>

Dans un terminal, exécutez :

```bash
weave serve <your model ref>
```

Obtenez la référence de votre modèle en accédant au modèle et en la copiant depuis l&#39;interface utilisateur. Elle doit ressembler à ceci :
`weave://your_entity/project-name/YourModel:<hash>`

Pour l&#39;utiliser, accédez au lien vers l&#39;interface Swagger UI, cliquez sur le point de terminaison `predict`, puis sur &quot;Try it out!&quot;.
