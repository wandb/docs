<div id="deploy-to-google-cloud">
  ## Déployer sur Google Cloud
</div>

<Note>
  `weave deploy` nécessite que `gcloud` soit installé et configuré sur votre machine. `weave deploy gcp` utilisera la configuration existante si elle n’est pas spécifiée directement via des arguments de ligne de commande.
</Note>

Pour déployer un modèle Weave sur Google Cloud, exécutez :

```
weave deploy gcp <ref>
```

Une fois le déploiement effectué, le terminal renvoie une URL selon le schéma suivant :

```
Service URL: <PATH_TO_MODEL>`. Visit `<PATH_TO_MODEL>/docs` to interact with your model!
```
