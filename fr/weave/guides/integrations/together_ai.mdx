Together AI est une plateforme de création et d’affinage de modèles d’IA générative, centrée sur les LLM open source, qui permet aux clients d’affiner et d’héberger leurs propres modèles.

<Info>
  La prise en charge complète du package Python `together` par Weave est actuellement en cours de développement.
</Info>

Bien que la prise en charge complète du package Python `together` par Weave soit actuellement en cours de développement, Together prend en charge la compatibilité du SDK OpenAI ([documentation](https://docs.together.ai/docs/openai-api-compatibility)), que Weave détecte et intègre automatiquement.

Pour utiliser l’API Together, remplacez simplement la clé API par votre clé [Together API](https://docs.together.ai/docs/quickstart), définissez `base_url` sur `https://api.together.xyz/v1`, puis choisissez comme modèle l’un de leurs [modèles de chat](https://docs.together.ai/docs/inference-models#chat-models).

```python lines {5, 10-13}
import os
import openai
import weave

weave.init('together-weave')

system_content = "You are a travel agent. Be descriptive and helpful."
user_content = "Tell me about San Francisco"

client = openai.OpenAI(
    api_key=os.environ.get("TOGETHER_API_KEY"),
    base_url="https://api.together.xyz/v1",
)
chat_completion = client.chat.completions.create(
    model="mistralai/Mixtral-8x7B-Instruct-v0.1",
    messages=[
        {"role": "system", "content": system_content},
        {"role": "user", "content": user_content},
    ],
    temperature=0.7,
    max_tokens=1024,
)
response = chat_completion.choices[0].message.content
print("Together response:\n", response)
```

Bien qu’il s’agisse d’un exemple simple pour démarrer, consultez notre guide [OpenAI](/fr/weave/guides/integrations/openai#track-your-own-ops) pour plus de détails sur la manière d’intégrer Weave à vos propres fonctions dans des cas d’usage plus complexes.
