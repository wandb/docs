title: "OpenRouter"
description: "Utilisez l'interface unifiée d'OpenRouter pour de nombreux LLM avec une intégration Weave automatique"
---

Openrouter.ai est une interface unifiée pour de nombreux LLM, compatible à la fois avec des modèles de base comme OpenAI GPT-4, Anthropic Claude et Google Gemini, mais aussi avec des modèles open source comme LLama-3, Mixtral et [bien d&#39;autres](https://openrouter.ai/models). Certains modèles sont même proposés gratuitement.

OpenRouter propose une API REST et une compatibilité du SDK OpenAI ([docs](https://docs.together.ai/docs/openai-api-compatibility)) que Weave détecte et intègre automatiquement (voir le [guide de démarrage rapide](https://openrouter.ai/docs/quick-start) d&#39;OpenRouter pour plus de détails).

Pour faire passer votre code du SDK OpenAI à OpenRouter, remplacez la clé API par votre clé [API OpenRouter](https://openrouter.ai/docs/api-keys), `base_url` par `https://openrouter.ai/api/v1`, et le modèle par l&#39;un de ses nombreux [modèles de chat](https://openrouter.ai/docs/models). Lorsque vous appelez `weave.init()`, fournissez un nom de projet pour vos traces. Si vous n&#39;en indiquez pas, votre entité par défaut est utilisée. Pour trouver ou mettre à jour votre entité par défaut, reportez-vous à [Paramètres utilisateur](https://docs.wandb.ai/platform/app/settings-page/user-settings/#default-team) dans la documentation de W&amp;B Models.

```python lines {5,10-12}
import os
import openai
import weave

weave.init('openrouter-weave')

system_content = "You are a travel agent. Be descriptive and helpful."
user_content = "Tell me about San Francisco"

client = openai.OpenAI(
    api_key=os.environ.get("OPENROUTER_API_KEY"),
    base_url="https://openrouter.ai/api/v1",
)
chat_completion = client.chat.completions.create(
    extra_headers={
    "HTTP-Referer": $YOUR_SITE_URL, # Facultatif, pour inclure votre application dans les classements d'openrouter.ai.
    "X-Title": $YOUR_APP_NAME, # Facultatif. S'affiche dans les classements d'openrouter.ai.
    },
    model="meta-llama/llama-3.1-8b-instruct:free",
    messages=[
        {"role": "system", "content": system_content},
        {"role": "user", "content": user_content},
    ],
    temperature=0.7,
    max_tokens=1024,
)
response = chat_completion.choices[0].message.content
print("Model response:\n", response)
```

Bien qu’il s’agisse d’un exemple simple pour démarrer, consultez notre guide [OpenAI](/fr/weave/guides/integrations/openai#track-your-own-ops) pour plus de détails sur la manière d’intégrer Weave à vos propres fonctions pour des cas d’usage plus complexes.
