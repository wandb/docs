W&amp;B Weave peut tracer des applications à agents sur plusieurs tours de conversation sans avoir à instrumenter manuellement chaque tour de conversation. Pour les SDK d’agents et les harnesses pris en charge, appelez `weave.init()` et Weave applique automatiquement un patch au framework. Weave capture les conversations, les tours de conversation, les appels LLM et les appels d’outil, puis les affiche dans la vue **Agents**. Cette page renvoie vers des guides d’intégration pour trois catégories : les harness d’agents, les SDK d’agents et les agents personnalisés (y compris ceux qui émettent des spans OpenTelemetry).
