---

L&#39;outil de *graphiques de trace* dans W&amp;B Weave vous permet d&#39;explorer, de visualiser et de déboguer les métriques au niveau des traces (par ex. latence, coût, jetons) à l&#39;aide de graphiques interactifs. Vous pouvez utiliser les graphiques de trace par défaut ou créer les vôtres via le générateur de graphique de traces personnalisé.

<Frame>
  ![Graphiques de trace en action](/weave/guides/tracking/imgs/plots-example.png)
</Frame>

<div id="get-started">
  ## Prise en main
</div>

1. Accédez à la page **Traces** de votre projet.
2. (Facultatif) Sélectionnez **Filtre** pour filtrer les traces sélectionnées (par exemple par date et heure ou par opération).
3. Dans le coin supérieur droit de la vue **Traces**, cliquez sur l’icône **Show Metrics** pour ouvrir le volet latéral des graphiques de trace.

   ![L’icône Show Metrics](/weave/guides/tracking/imgs/plots-show-metrics-icon.png)

   Vous pouvez ensuite :

   * Afficher les [graphiques de trace par défaut](#default-trace-plots).
   * Créer un [graphique de traces personnalisé](#create-a-custom-trace-plot).
4. Les graphiques se mettent à jour dynamiquement en fonction de vos filtres de traces et de vos sélections.

<div id="default-trace-plots">
  ## Graphiques de trace par défaut
</div>

Lorsque vous ouvrez pour la première fois le panneau des graphiques de trace, Weave génère automatiquement quelques graphiques de trace basés sur les données de trace disponibles dans votre projet :

* Graphique à barres (coût ou latence regroupés par intervalles de temps)
* Graphique linéaire (latence au fil du temps)
* Nuage de points (par ex. jetons de prompt vs. jetons de complétion)

Chaque graphique de trace est interactif :

* Survolez pour afficher des infobulles
* Faites glisser pour zoomer
* Double-cliquez pour réinitialiser
* Cliquez sur des points dans les nuages de points pour ouvrir une trace spécifique

<div id="create-a-custom-trace-plot">
  ## Créer un graphique de traces personnalisé
</div>

Vous pouvez également créer un graphique de traces personnalisé. Pour créer un graphique de traces personnalisé, procédez comme suit :

1. Dans le volet latéral des graphiques de trace, cliquez sur **➕ Add Chart**.

   ![Bouton ➕ Add Chart.](/weave/guides/tracking/imgs/plots-add-chart.png)

2. Dans la fenêtre contextuelle, sélectionnez l’un des types de graphiques de trace disponibles :

   ![Types de graphiques de trace personnalisés disponibles](/weave/guides/tracking/imgs/plots-custom-chart-types.png)

3. Pour le type de graphique de trace sélectionné, configurez votre graphique de trace. Pour plus d’informations sur les options de configuration selon le type de graphique de trace, voir [Paramètres des graphiques de trace par type de graphique](#trace-plot-settings-by-plot-type).

4. Cliquez sur **Save chart** pour enregistrer votre graphique.

<div id="trace-plot-settings-by-plot-type">
  ### Paramètres des graphiques de trace par type de graphique
</div>

Lors de l’ajout ou de la modification d’un graphique de traces personnalisé, les options de configuration disponibles varient légèrement selon le type de graphique de trace sélectionné. Le tableau détaille les options configurables.

| Paramètre    | Nuage de points  | Graphique linéaire  | graphique à barres  |
| ------------ | ---------------- | ------------------- | ------------------- |
| Axe Y        | ✅ Requis         | ✅ Requis            | ✅ Requis            |
| Axe des x    | ✅ Sélectionnable | Fixe (`started at`) | Fixe (`started at`) |
| Regroupement | ✅ Facultatif     | ✅ Facultatif        | ✅ Facultatif        |
| Binning      | ❌ Non utilisé    | ✅ Utilisé           | ✅ Utilisé           |
| Agrégation   | ❌ Non utilisée   | ✅ Utilisée          | ✅ Utilisée          |
