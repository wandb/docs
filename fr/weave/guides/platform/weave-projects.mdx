Les projets Weave permettent d’organiser les ressources associées, notamment les traces, les évaluations, les modèles, les jeux de données et d’autres ressources Weave. Ils aident votre équipe à mieux s’organiser, simplifient la collaboration et le partage, et vous aident à suivre l’évolution de vos applications LLM au fil du temps.

<div id="project-paths">
  ## Chemins de projet
</div>

[Teams](/fr/platform/hosting/iam/org_team_struct#team) (également appelées Entities) possèdent et gèrent les projets de votre organisation W&amp;B. Le nom de votre équipe et le nom de votre projet constituent le chemin d’accès à votre projet dans l’interface utilisateur W&amp;B. Vous utilisez également ce chemin dans votre code pour associer vos ressources Weave au projet.

Par exemple, si le nom de votre équipe est `my-team` et le nom de votre projet est `my-project`, le chemin d’accès à votre projet est :

```
my-team/my-project
```

Utilisez ce chemin lorsque vous initialisez Weave dans votre code pour associer les données de votre application au projet :

```
weave.init('my-team/my-project')
```

<div id="create-and-find-projects">
  ## Créer et trouver des projets
</div>

Vous pouvez créer un projet Weave dans l’interface utilisateur W&amp;B ou en définissant un nouveau chemin de projet dans votre code.

<Tabs>
  <Tab title="W&B UI">
    Pour créer un nouveau projet dans l’interface utilisateur W&amp;B :

    1. Accédez à [wandb.ai](https://wandb.ai).
    2. Sélectionnez **New project**. La boîte de dialogue modale **Create a new project** s’ouvre.
    3. Dans la boîte de dialogue modale, configurez les champs suivants :
       * **Team** : l’équipe existante à associer au projet.
       * **Name** : un nom pour votre projet.
       * **Project visibility** : les personnes autorisées à voir et à modifier le projet.
       * **Description** : (facultatif) une courte description du projet pour vous aider à en suivre et à en organiser l’objectif.
    4. Après avoir configuré les champs nécessaires, sélectionnez **Create project**. Le projet est alors créé.

    Pour retrouver votre projet nouvellement créé, sélectionnez le **logo Weights &amp; Biases** dans le coin supérieur gauche pour ouvrir le menu de **navigation globale**. Ce menu répertorie vos projets les plus récents et les équipes auxquelles vous appartenez, ce qui vous permet de changer rapidement de contexte. Sélectionnez votre projet pour ouvrir l’espace de travail du projet.

    Vous pouvez également ouvrir votre **Profil** depuis le menu situé sous votre nom dans le coin supérieur droit. L’onglet **Projects** répertorie tous vos projets.

    Lorsque vous ouvrez l’espace de travail du nouveau projet, l’interface propose des Démarrages rapides pour W&amp;B Models et W&amp;B Weave. La barre d’outils du projet contient également des outils pour chaque produit. Après avoir associé des données Weave à votre projet dans votre code, la barre latérale n’affiche plus que les outils Weave et le tableau de bord Weave devient disponible.
  </Tab>

  <Tab title="Weave SDK">
    Pour créer un projet Weave directement depuis votre code, fournissez un nouveau chemin de projet comme argument à `weave.init()` :

    <CodeGroup>
      ```python
      import weave

      weave.init('my-team/my-llm-app')
      ```

      ```typescript
      import * as weave from 'weave';

      await weave.init('my-team/my-llm-app');
      ```
    </CodeGroup>

    Lorsque vous exécutez le code, Weave crée le nouveau projet dans votre compte W&amp;B et téléverse toutes les données applicables du run vers le projet. Weave configure également le projet comme un projet Weave (et non un projet W&amp;B Models), et la barre latérale de l’interface utilisateur n’affiche que les outils Weave.
  </Tab>
</Tabs>

<div id="view-projects-in-the-ui">
  ## Voir les projets dans l’interface
</div>

Pour accéder à un projet dans l’interface Weave :

1. Accédez à [wandb.ai](https://wandb.ai).
2. Cliquez sur le **logo Weights &amp; Biases** dans le **coin supérieur gauche** pour ouvrir le **menu de navigation globale**, puis sélectionnez votre projet dans la barre latérale **Projects**. Cela ouvre la page d’aperçu du projet, ou la page **Traces** si vous avez déjà enregistré des traces dans ce projet.
3. Depuis la page d’aperçu, vous pouvez :
   * Consulter les statistiques de votre projet, comme le nombre de traces enregistrées dans le projet et l’espace de stockage qu’il utilise.
   * Accéder aux traces, aux évaluations et à d’autres ressources à l’aide de la barre latérale.
   * Accéder aux traces, évaluations, modèles et jeux de données récemment enregistrés.

<div id="delete-a-project">
  ## Supprimer un projet
</div>

Pour supprimer un projet de votre compte W&amp;B :

1. Accédez à [wandb.ai](https://wandb.ai).
2. Sélectionnez votre projet dans la barre latérale **Projects**. Cela ouvre la page d’aperçu du projet, ou la page **Traces** si vous avez déjà enregistré des traces pour ce projet.
3. Sélectionnez le bouton **Project** (<Icon icon="circle-info" size={16} />) dans la barre latérale pour ouvrir la page d’aperçu du projet si elle n’est pas déjà affichée.
4. Sélectionnez le menu à trois points (<Icon icon="ellipsis" size={16} />) et choisissez **Supprimer le projet** dans la liste. Cela ouvre la boîte de dialogue modale **Supprimer le projet**.
5. Dans la fenêtre modale, saisissez le nom du projet dans le champ de confirmation, puis cliquez sur **Supprimer**. Le projet est définitivement supprimé de votre compte W&amp;B.
