<div id="base-url">
  ## URL de base
</div>

Pour [W&amp;B Cloud mutualisé](/fr/platform/hosting/hosting-options/multi_tenant_cloud) et [W&amp;B Cloud dédié](/fr/platform/hosting/hosting-options/dedicated-cloud), utilisez l’URL de base suivante :

```
https://trace.wandb.ai
```

Pour les [instances W&amp;B autogérées](/fr/weave/guides/platform/weave-self-managed), utilisez :

```
https://<your-subdomain>.wandb.io/traces
```

<div id="authentication">
  ## Authentification
</div>

Incluez votre clé API W&amp;B dans les en-têtes de la requête :

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" https://trace.wandb.ai/...
```
