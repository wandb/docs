## Propriétés

<div id="agentdescription">
  ### agentDescription
</div>

• `Facultatif` **agentDescription**: `string`

Description de l’agent compréhensible par un humain. Elle est propagée
comme valeur `agentDescription` par défaut à chaque `Turn` créé via
`startTurn()`, sauf si le tour de conversation définit sa propre valeur ;
elle est émise dans le span `invoke_agent` de chaque tour de conversation
sous la forme de `gen_ai.agent.description`.

<div id="agentid">
  ### agentId
</div>

• `Facultatif` **agentId**: `string`

Identifiant stable de l’agent. Il est propagé comme `agentId` par défaut à chaque
`Turn` créé via `startTurn()`, sauf si le tour de conversation définit le sien ; il est émis
dans le span `invoke_agent` de chaque tour de conversation sous la forme de `gen_ai.agent.id`.

<div id="agentversion">
  ### agentVersion
</div>

• `Facultatif` **agentVersion**: `string`

Chaîne de version de l’agent. Utilisée comme valeur `agentVersion` par défaut pour
chaque tour de conversation créé via `startTurn()`, sauf si le tour de conversation définit sa propre valeur ;
elle est émise dans le span `invoke_agent` de chaque tour de conversation sous la forme de `gen_ai.agent.version`.

<div id="attributes">
  ### attributes
</div>

• `facultatif` **attributes**: `Attributes`

Attributs personnalisés ajoutés à chaque span émis par la conversation.

Toute clé définie ici qui entre en conflit avec l’attribut `gen_ai.*` / `weave.*` propre à un span n’est pas prise en charge ; la valeur du span prévaut.
