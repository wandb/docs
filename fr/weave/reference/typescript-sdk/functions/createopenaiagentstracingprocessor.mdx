  # createOpenAIAgentsTracingProcessor

▸ **createOpenAIAgentsTracingProcessor**(): `TracingProcessor`

Créer un processeur de Tracing Weave pour OpenAI Agents.

<div id="returns">
  #### Renvoie
</div>

`TracingProcessor`

Une instance de `TracingProcessor` qui peut être enregistrée auprès d’OpenAI Agents

`Exemple`

```typescript
import { addTraceProcessor } from '@openai/agents';
import { createOpenAIAgentsTracingProcessor } from 'weave';

const processor = createOpenAIAgentsTracingProcessor();
addTraceProcessor(processor);
```
