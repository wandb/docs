<div id="flushotel">
  # flushOTel
</div>

▸ **flushOTel**(): `Promise`&lt;`void`&gt;

Force le vidage de tous les spans GenAI mis en tampon par le processeur de spans actif.

Se résout immédiatement si `weave.init()` n&#39;a pas été appelé. Sinon,
délègue à `BasicTracerProvider.forceFlush()`, qui attend que le
processeur de spans sous-jacent vide sa file d&#39;attente et termine son cycle d&#39;export OTLP.

Appelez cette fonction avant l&#39;arrêt du processus lorsque vous utilisez `'simple'` ou tout autre processeur susceptible
d&#39;avoir des opérations en cours, ou dans des tests qui doivent observer les spans exportés
de manière synchrone.
