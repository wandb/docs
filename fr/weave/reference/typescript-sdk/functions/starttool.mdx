<div id="starttool">
  # startTool
</div>

▸ **startTool**(`opts`): [`Tool`](../classes/tool)

Démarre un span d&#39;outil. Résolution du parent (correspond au principe de conception « plat par
défaut, hiérarchique si vous imbriquez ») :

* Si un LLM est actif, l&#39;outil y est imbriqué.
* Sinon, l&#39;outil est au même niveau, sous le tour de conversation actuel.

Lève une exception si aucun tour de conversation ni aucun LLM n&#39;est actif.
