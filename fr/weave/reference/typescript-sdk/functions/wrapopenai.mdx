<div id="wrapopenai">
  # wrapOpenAI
</div>

▸ **wrapOpenAI**&lt;`T`&gt;(`openai`): `T`

Encapsule l’API OpenAI pour activer le tracing des fonctions lors des appels à OpenAI.

<div id="type-parameters">
  #### Paramètres de type
</div>

| Nom | Type                |
| :-- | :------------------ |
| `T` | extends `OpenAIAPI` |

<div id="parameters">
  #### Paramètres
</div>

| Nom      | Type |
| :------- | :--- |
| `openai` | `T`  |

<div id="returns">
  #### Renvoie
</div>

`T`

`Example`

```ts
const openai = wrapOpenAI(new OpenAI());
const result = await openai.chat.completions.create({
  model: 'gpt-3.5-turbo',
  messages: [{ role: 'user', content: 'Hello, world!' }]
});
```
