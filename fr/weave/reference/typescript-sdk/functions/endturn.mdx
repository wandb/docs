<div id="endturn">
  # endTurn
</div>

▸ **endTurn**(`opts?`): `void`

Met fin au tour de conversation en cours. Sans effet si aucun tour de conversation n’est actif.

<div id="parameters">
  #### Paramètres
</div>

| Nom     | Type             |
| :------ | :--------------- |
| `opts?` | `SpanEndOptions` |

<div id="returns">
  #### Renvoie
</div>

`void`

`Exemple`

```ts twoslash
// @noErrors
weave.endTurn();
```

`Exemple`

```ts twoslash
// @noErrors
weave.endTurn({endTime: new Date('2026-05-29T10:00:01.700Z')});
```

`Exemple`

```ts twoslash
// @noErrors
weave.endTurn({error: new Error('agent loop diverged')});
```
