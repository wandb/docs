<div id="startturn">
  # startTurn
</div>

▸ **startTurn**(`opts?`): [`Turn`](../classes/turn)

Démarre un nouveau tour de conversation. Si une Conversation est active, le tour hérite de son
`conversationId` ; sinon, le tour n’a pas d’identifiant de conversation.

<div id="parameters">
  #### Paramètres
</div>

| Nom    | Type                                 |
| :----- | :----------------------------------- |
| `opts` | [`TurnInit`](../interfaces/turninit) |

<div id="returns">
  #### Renvoie
</div>

[`Turn`](../classes/turn)

`Exemple`

```ts twoslash
// @noErrors
weave.startTurn({agentName: 'research-bot'});
```

`Exemple`

```ts twoslash
// @noErrors
weave.startTurn({
  agentName: 'research-bot',
  startTime: new Date('2026-05-29T10:00:00.000Z'),
});
```
