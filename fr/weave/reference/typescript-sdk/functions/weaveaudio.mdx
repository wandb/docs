<div id="weaveaudio">
  # weaveAudio
</div>

▸ **weaveAudio**(`options`): [`WeaveAudio`](../interfaces/weaveaudio)

Crée un nouvel objet WeaveAudio

<div id="parameters">
  #### Paramètres
</div>

| Nom       | Type              | Description                                                                                                                                                                                    |
| :-------- | :---------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `options` | `WeaveAudioInput` | Les options de ce type de média - `data` : les données audio brutes sous forme de Buffer - `audioType` : (facultatif) le type de fichier audio ; actuellement, seul `'wav'` est pris en charge |

<div id="returns">
  #### Renvoie
</div>

[`WeaveAudio`](../interfaces/weaveaudio)

`Exemple`

```ts
const audioBuffer = fs.readFileSync('path/to/audio.wav');
const weaveAudio = weaveAudio({ data: audioBuffer });
```
