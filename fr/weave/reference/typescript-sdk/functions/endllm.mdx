<div id="endllm">
  # endLLM
</div>

▸ **endLLM**(`opts?`): `void`

Met fin au LLM actuel. N’a aucun effet si aucun LLM n’est actif.

<div id="parameters">
  #### Paramètres
</div>

| Nom     | Type             |
| :------ | :--------------- |
| `opts?` | `SpanEndOptions` |

<div id="returns">
  #### Renvoie
</div>

`void`

`Exemple`

```ts twoslash
// @noErrors
weave.endLLM();
```

`Exemple`

```ts twoslash
// @noErrors
weave.endLLM({endTime: new Date('2026-05-29T10:00:00.800Z')});
```

`Exemple`

```ts twoslash
// @noErrors
weave.endLLM({error: new Error('llm call failed')});
```
