  # wrapClaudeAgentSdk
</div>

▸ **wrapClaudeAgentSdk**&lt;`T`&gt;(`sdk`): `T`

Instrumentez manuellement le module `@anthropic-ai/claude-agent-sdk`.

Utilisez cette fonction lorsque l&#39;instrumentation automatique ne s&#39;applique pas — par ex. avec un bundler
dont le chargement des modules ne peut pas être observé par les hooks CJS/ESM, ou avec un chemin d&#39;import qu&#39;ils
ne couvrent pas. Nécessite `@anthropic-ai/claude-agent-sdk` &gt;= 0.3.178. Renvoie une
vue du module dont l&#39;export `query` est tracé ; utilisez l&#39;objet renvoyé
plutôt que l&#39;import d&#39;origine (le `query` du SDK est un export avec accesseur en lecture seule,
donc la liaison d&#39;origine ne peut pas être patchée sur place) :
