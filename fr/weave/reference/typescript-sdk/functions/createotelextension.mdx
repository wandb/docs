<div id="createotelextension">
  # createOtelExtension
</div>

▸ **createOtelExtension**(`opts?`): (`pi`: `PiExtensionApi`) =&gt; `void`

Crée une extension pour l&#39;agent de code pi qui génère des spans OTEL sur
l&#39;ensemble du cycle de vie de l&#39;agent, conformément aux conventions sémantiques GenAI.

Lorsque `weave.init(...)` a été appelé, les spans sont automatiquement exportés
vers le serveur de traces Weave à l&#39;adresse `/agents/otel/v1/traces`. Sinon, transmettez un
`tracer` personnalisé dans `opts`.

| Nom    | Type                   |
| :----- | :--------------------- |
| `opts` | `OtelExtensionOptions` |
