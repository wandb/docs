<div id="withattributes">
  # withAttributes
</div>

▸ **withAttributes**&lt;`T`&gt;(`attrs`, `fn`): `Promise`&lt;`T`&gt; | `T`

Ajoute des attributs au contexte d’exécution actuel afin que tous les appels créés
dans `fn` en héritent automatiquement. Les attributs sont écrits dans l’enregistrement d’appel
sur le serveur de trace et sont visibles dans l’interface Weave et le filtrage, ce qui les rend
idéaux pour étiqueter des runs avec des ID de requête, des tenants, des expériences, etc.

Exemple :

```ts
await withAttributes({requestId: 'abc'}, async () => {
  await myOp();
});
```
