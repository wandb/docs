  # OpDecorator
</div>

Ƭ **OpDecorator**&lt;`T`&gt;: (`value`: `T`, `context`: `ClassMethodDecoratorContext`) =&gt; `T` | `void` &amp; (`target`: `Object`, `propertyKey`: `string` | `symbol`, `descriptor`: `TypedPropertyDescriptor`&lt;`T`&gt;) =&gt; `TypedPropertyDescriptor`&lt;`T`&gt; | `void`

Type d’assistance pour les décorateurs
Il s’agit d’une fonction décorateur qui peut être utilisée à la fois avec les décorateurs hérités et les décorateurs Stage 3.

Pour les décorateurs Stage 3 :
target: La fonction décorée (T)
context: MethodDecoratorContext

Pour les décorateurs hérités :
target: Le prototype (méthodes d’instance) ou le constructeur (méthodes statiques)
propertyKey: Le nom de la méthode
descriptor: Le descripteur de propriété contenant la méthode
