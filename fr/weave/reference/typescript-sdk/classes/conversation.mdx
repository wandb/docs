<div id="startturn">
  ### startTurn
</div>

▸ **startTurn**(`opts?`): [`Turn`](./turn)

Démarre un nouveau `Turn` dans cette `Conversation`. Le `Turn` hérite du
`conversationId` de la conversation ; `agentName`, `agentId`, `agentDescription`,
`agentVersion` et `model` utilisent par défaut les valeurs de la conversation lorsqu&#39;ils ne sont pas
fournis dans `opts`.

<div id="parameters">
  #### Paramètres
</div>

| Nom    | Type                                 |
| :----- | :----------------------------------- |
| `opts` | [`TurnInit`](../interfaces/turninit) |

<div id="returns">
  #### Renvoie
</div>

[`Turn`](./turn)

`Exemple`

```ts twoslash
// @noErrors
const turn = conversation.startTurn();
```

`Exemple`

```ts twoslash
// @noErrors
const turn = conversation.startTurn({
  model: 'gpt-4o',
  agentName: 'research-bot',
  agentId: 'research-bot-prod',
  agentDescription: 'Looks up facts on Wikipedia.',
  agentVersion: '1.4.2',
  userMessage: 'What is the weather in Tokyo?',
  systemInstructions: ['You are a helpful weather bot.'],
  startTime: new Date('2026-05-29T10:00:00.000Z'),
});
```
