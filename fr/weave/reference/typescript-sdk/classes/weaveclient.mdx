<div id="addscore">
  ### addScore
</div>

▸ **addScore**(`predictCallId`, `scorerCallId`, `runnableRefUri`, `scorerOutput`): `Promise`&lt;`chaîne de caractères`&gt;

Ajoute un résultat du scorer (par exemple, la sortie du scorer) à un appel.
Utilisé dans l’Évaluation impérative pour associer les résultats du scorer aux appels de prédiction.

<div id="parameters">
  #### Paramètres
</div>

| Nom              | Type     | Description                                         |
| :--------------- | :------- | :-------------------------------------------------- |
| `predictCallId`  | `chaîne de caractères` | ID de l&#39;Appel predict auquel associer le retour |
| `scorerCallId`   | `chaîne de caractères` | ID de l&#39;Appel scorer qui a généré le retour     |
| `runnableRefUri` | `chaîne de caractères` | URI du scorer (Op ou réf. d&#39;objet)              |
| `scorerOutput`   | `any`    | Sortie du scorer                                    |
