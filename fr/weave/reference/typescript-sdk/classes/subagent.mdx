## Méthodes

<div id="addevent">
  ### addEvent
</div>

<Warning>
  **Obsolète.** Enregistrez plutôt ces données via [setAttributes](./subagent#setattributes).
  OpenTelemetry abandonne progressivement l’API d’événements de span (`Span.addEvent`). Ces
  méthode fonctionne toujours et les données d’événements de span existantes restent valides.
  Voir [https://opentelemetry.io/blog/2026/deprecating-span-events/](https://opentelemetry.io/blog/2026/deprecating-span-events/)

  `Exemple`

  ```ts twoslash
  // @noErrors
  span.addEvent('context_compacted', {removedMessages: 12});
  ```
</Warning>

▸ **addEvent**(`name`, `attributes?`, `startTime?`): `this`

Ajoute un événement nommé au span. Utile pour marquer des moments qui ne correspondent pas à des spans, comme la compaction du contexte, la détection de boucles d&#39;outil ou le déclenchement de guardrails. Émet un avertissement et n&#39;a aucun effet après `end()`. Reproduit le comportement de `OTel Span.addEvent`.

</div>

| Nom           | Type         |
| :------------ | :----------- |
| `name`        | `string`     |
| `attributes?` | `Attributes` |
| `startTime?`  | `TimeInput`  |

<div id="end">
  ### end
</div>

▸ **end**(`opts?`): `void`

Lisez les valeurs actuelles des champs (pour refléter les mutations effectuées via `record()`
depuis `start`) et fermez le span. Cette opération est idempotente. Passez `error` pour marquer le span comme ayant échoué;
passez `endTime` pour antidater la fermeture.

<div id="parameters">
  #### Paramètres
</div>

| Nom     | Type             |
| :------ | :--------------- |
| `opts?` | `SpanEndOptions` |

<div id="setattributes">
  ### setAttributes
</div>

▸ **setAttributes**(`attributes`): `this`

Définit plusieurs attributs du span en une seule fois. Affiche un avertissement et n’a aucun effet après
`end()`. Reprend le comportement de OTel `Span.setAttributes` (et de
`set_attributes` du SDK Python).

<div id="parameters">
  #### Paramètres
</div>

| Nom          | Type         |
| :----------- | :----------- |
| `attributes` | `Attributes` |

<div id="returns">
  #### Renvoie
</div>

`this`

`Exemple`

```ts twoslash
// @noErrors
span.setAttributes({'weave.tag': 'prod', 'gen_ai.response.id': id});
```
