Requête visant à mettre à jour l&#39;état d&#39;annotation d&#39;un élément de la file d&#39;attente pour l&#39;annotateur actuel.

Transitions d&#39;état valides :

* (absence) -&gt; &#39;in&#95;progress&#39; : Marquer l&#39;élément comme étant en cours (uniquement lorsqu&#39;aucun enregistrement n&#39;existe)
* (absence) -&gt; &#39;completed&#39; ou &#39;skipped&#39; : Terminer ou ignorer directement l&#39;élément
* &#39;in&#95;progress&#39; ou &#39;unstarted&#39; -&gt; &#39;completed&#39; ou &#39;skipped&#39; : Terminer ou ignorer un élément déjà démarré
