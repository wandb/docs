  # Les évaluateurs comme garde-fous
</div>

Les évaluateurs Weave sont des classes spéciales dotées d’une méthode `score` qui permet d’évaluer les performances d’un appel. Ils peuvent aller de règles très simples à des LLM complexes servant de juges.

Dans ce notebook, nous allons explorer comment utiliser les évaluateurs comme garde-fous pour empêcher votre LLM de générer du contenu nuisible ou inapproprié.
