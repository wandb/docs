W&amp;B Weave trace les agents sur plusieurs tours de conversation créés avec des SDK et des harnesses populaires, sans avoir à instrumenter manuellement chaque tour de conversation. Installez un plugin pour le harness de votre agent, ou appelez `weave.init()` dans le code qui utilise un SDK d’agent pris en charge, et Weave patche automatiquement le framework. Les sessions, les tours de conversation, les appels LLM et les appels d’outil s’affichent dans la vue **Agents** de votre projet.

Choisissez l’une des intégrations prises en charge ci-dessous.
