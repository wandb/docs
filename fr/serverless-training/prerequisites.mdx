Effectuez ces étapes avant d’utiliser les fonctionnalités de Serverless Training via le framework OpenPipe ART ou l’API.

<Tip>
  Avant de commencer, consultez les [informations sur l’utilisation et les limites](/fr/serverless-training/usage-limits) afin de comprendre les coûts et les restrictions.
</Tip>

<div id="sign-up-and-create-an-api-key">
  ## S’inscrire et créer une clé API
</div>

Si vous n’avez pas encore de compte W&amp;B, inscrivez-vous avant de continuer. Pour authentifier votre machine auprès de W&amp;B, vous devez d’abord générer une clé API.

<ApiKeyCreate />

<div id="create-a-project-in-wb">
  ## Créer un projet dans W&amp;B
</div>

Créez un projet dans votre compte W&amp;B pour suivre l&#39;utilisation, consigner les métriques d&#39;entraînement et sauvegarder les modèles entraînés. Pour plus d&#39;informations, voir le [guide Projects](https://docs.wandb.ai/models/track/project-page).

<div id="next-steps">
  ## Étapes suivantes
</div>

Une fois les prérequis remplis, essayez les ressources suivantes :

* Consultez la [référence de l’API](/fr/serverless-training/api-reference) pour en savoir plus sur les points de terminaison disponibles.
* Essayez le [démarrage rapide d’ART](https://art.openpipe.ai/getting-started/quick-start).
