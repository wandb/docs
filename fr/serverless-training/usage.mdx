Utilisez Serverless RL via [le framework ART d&#39;OpenPipe](https://art.openpipe.ai/getting-started/about) et la [Serverless Training API](/fr/serverless-training/api-reference).

Pour commencer à utiliser Serverless RL, vous devez remplir les [prérequis](/fr/serverless-training/prerequisites) pour utiliser les outils W&amp;B, puis suivre le [démarrage rapide](https://art.openpipe.ai/getting-started/quick-start) d’ART.

Pour obtenir des exemples de code et de flux de travail, voir le [notebook Google Colab](https://colab.research.google.com/github/openpipe/art-notebooks/blob/main/examples/2048/2048.ipynb).
