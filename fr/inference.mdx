W&amp;B Inference vous donne accès à des modèles fondamentaux open source de premier plan via W&amp;B Weave et une API compatible avec OpenAI. Vous pouvez :

* Créer des applications d&#39;IA et des agents sans vous inscrire chez un fournisseur d&#39;hébergement ni héberger vous-même un modèle
* Essayer les [modèles pris en charge](/fr/inference/models) dans le [W&amp;B Weave Playground](/fr/weave/guides/tools/playground)

Avec Weave, vous pouvez tracer, évaluer, surveiller et améliorer vos applications basées sur W&amp;B Inference.

<div id="quickstart">
  ## Démarrage rapide
</div>

Voici un exemple simple en Python :

```python
import openai

client = openai.OpenAI(
    # L'URL de base personnalisée pointe vers W&B Inference
    base_url='https://api.inference.wandb.ai/v1',

    # Créez une clé API sur https://wandb.ai/settings
    api_key="<your-api-key>",

    # Facultatif : équipe et projet pour le suivi de l'utilisation
    project="<your-team>/<your-project>",
)

response = client.chat.completions.create(
    model="meta-llama/Llama-3.1-8B-Instruct",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Tell me a joke."}
    ],
)

print(response.choices[0].message.content)
```
