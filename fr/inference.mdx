W&amp;B Inference vous donne accès à des modèles fondamentaux open source de premier plan via W&amp;B Weave et une API compatible avec OpenAI. Vous pouvez :

* Créer des applications d&#39;IA et des agents sans vous inscrire chez un fournisseur d&#39;hébergement ni héberger vous-même un modèle
* Essayer les [modèles pris en charge](/fr/inference/models) dans le [W&amp;B Weave Playground](/fr/weave/guides/tools/playground)

Avec Weave, vous pouvez tracer, évaluer, surveiller et améliorer vos applications basées sur W&amp;B Inference.
