description: Mettre à jour un artifact existant pendant qu'un run est actif ou en utilisant uniquement l'API publique.
title: Mettre à jour un artifact
---

Définissez les valeurs souhaitées pour mettre à jour la `description`, les `metadata` et l&#39;`alias` d&#39;un artifact. Mettez à jour un run précédemment consigné dans W&amp;B avec l&#39;API publique W&amp;B ([`wandb.Api`](/fr/models/ref/python/public-api/api)). Utilisez `wandb.Run.save()` pour mettre à jour un artifact lorsqu&#39;il est initialisé pour la première fois et qu&#39;il est toujours actif.

<Note>
  **Quand utiliser wandb.Artifact.save() ou wandb.Run.log&#95;artifact()**

  * Utilisez `Artifact.save()` pour mettre à jour un artifact existant sans démarrer un nouveau run.
  * Utilisez `wandb.Run.log_artifact()` pour créer un nouvel artifact et l&#39;associer à un run spécifique.
</Note>

Utilisez l&#39;API publique W&amp;B ([`wandb.Api`](/fr/models/ref/python/public-api/api)) pour mettre à jour un artifact. Utilisez la classe wandb.Artifact ([`wandb.Artifact`](/fr/models/ref/python/experiments/artifact)) pendant qu&#39;un run est actif.

<Warning>Vous ne pouvez pas mettre à jour l&#39;alias d&#39;un artifact lié à un modèle dans le Model Registry.</Warning>

<Tabs>
  <Tab title="Pendant un run">
    L&#39;exemple de code suivant montre comment mettre à jour la description d&#39;un artifact à l&#39;aide de l&#39;API [`wandb.Artifact`](/fr/models/ref/python/experiments/artifact) :

    ```python
    import wandb

    with wandb.init(project="<example>") as run:
        artifact = run.use_artifact("<artifact-name>:<alias>")
        artifact.description = "<description>"
        artifact.save()
    ```
  </Tab>

  <Tab title="API publique W&B">
    L&#39;exemple suivant met à jour un artifact avec [`wandb.Api`](/fr/models/ref/python/public-api/api) :

    Pour plus d&#39;informations, voir l&#39;[API Artifact](/fr/models/ref/python/experiments/artifact) de Weights and Biases.
  </Tab>

  <Tab title="Avec des collections">
    Vous pouvez également mettre à jour une collection d&#39;Artifacts de la même manière qu&#39;un artifact individuel :

    ```python
    import wandb
    with wandb.init(project="<example>") as run:
        api = wandb.Api()
        artifact = api.artifact_collection(type="<type-name>", collection="<collection-name>")
        artifact.name = "<new-collection-name>"
        artifact.description = "<This is where you'd describe the purpose of your collection.>"
        artifact.save()
    ```

    Pour plus d&#39;informations, voir la référence [Artifacts Collection](/fr/models/ref/python/public-api/api).
  </Tab>
</Tabs>
