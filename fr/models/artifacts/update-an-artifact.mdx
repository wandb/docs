---
description: Mettre à jour un artefact existant dans un run W&B ou en dehors.
title: Mettre à jour un artefact
---

Renseignez les valeurs souhaitées pour mettre à jour la `description`, les `metadata` et l’`alias` d’un artefact. Appelez la méthode `save()` pour mettre à jour l’artefact sur les serveurs W&amp;B. Vous pouvez mettre à jour un artefact pendant un run W&amp;B ou en dehors d’un run.

<Note>
  **Quand utiliser Artifact.save() ou wandb.Run.log&#95;artifact()**

  * Utilisez `Artifact.save()` pour mettre à jour un artefact existant sans créer de nouveau run.
  * Utilisez `wandb.Run.log_artifact()` pour créer un nouvel artefact et l’associer à un run spécifique.
</Note>

Utilisez l’API publique W&amp;B ([`wandb.Api`](/fr/models/ref/python/public-api/api)) pour mettre à jour un artefact en dehors d’un run. Utilisez l’API Artifact ([`wandb.Artifact`](/fr/models/ref/python/experiments/artifact)) pour mettre à jour un artefact pendant un run.

<Warning>
  Vous ne pouvez pas mettre à jour l’alias d’un artefact lié à un modèle dans le Model Registry.
</Warning>

<Tabs>
  <Tab title="Pendant un run">
    L’exemple de code suivant montre comment mettre à jour la description d’un artefact à l’aide de l’API [`wandb.Artifact`](/fr/models/ref/python/experiments/artifact) :

    ```python
    import wandb

    with wandb.init(project="<example>") as run:
        artifact = run.use_artifact("<artifact-name>:<alias>")
        artifact.description = "<description>"
        artifact.save()
    ```
  </Tab>

  <Tab title="En dehors d’un run">
    L’exemple de code suivant montre comment mettre à jour la description d’un artefact à l’aide de l’API `wandb.Api` :

    ```python
    import wandb

    api = wandb.Api()

    artifact = api.artifact("entity/project/artifact:alias")

    # Mettre à jour la description
    artifact.description = "My new description"

    # Mettre à jour de façon sélective les clés des métadonnées
    artifact.metadata["oldKey"] = "new value"

    # Remplacer entièrement les métadonnées
    artifact.metadata = {"newKey": "new value"}

    # Ajouter un alias
    artifact.aliases.append("best")

    # Supprimer un alias
    artifact.aliases.remove("latest")

    # Remplacer complètement les alias
    artifact.aliases = ["replaced"]

    # Enregistrer toutes les modifications de l’artefact
    artifact.save()
    ```

    Pour plus d&#39;informations, voir l’[API Artifact](/fr/models/ref/python/experiments/artifact) de W&amp;B.
  </Tab>

  <Tab title="Avec des collections">
    Vous pouvez également mettre à jour une collection d’artefacts de la même manière qu’un artefact individuel :

    ```python
    import wandb
    with wandb.init(project="<example>") as run:
        api = wandb.Api()
        artifact = api.artifact_collection(type="<type-name>", collection="<collection-name>")
        artifact.name = "<new-collection-name>"
        artifact.description = "<This is where you'd describe the purpose of your collection.>"
        artifact.save()
    ```

    Pour plus d&#39;informations, voir la référence [Artifacts Collection](/fr/models/ref/python/public-api/api).
  </Tab>
</Tabs>
