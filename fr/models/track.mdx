<div id="how-it-works">
  ## Fonctionnement
</div>

Suivez une expérience de machine learning avec quelques lignes de code :

1. Créez un [W&amp;B Run](/fr/models/runs/).
2. Enregistrez un dictionnaire d&#39;hyperparamètres, comme le taux d&#39;apprentissage ou le type de modèle, dans votre configuration ([`wandb.Run.config`](/fr/models/track/config/)).
3. Consignez des métriques ([`wandb.Run.log()`](/fr/models/track/log/)) au fil de l&#39;entraînement, par exemple l&#39;accuracy et la perte.
4. Enregistrez les sorties d&#39;un run, comme les poids du modèle ou un tableau de prédictions.

Le code suivant illustre un flux de travail courant de suivi des expériences avec W&amp;B. Remplacez `my-team` par votre entité d’équipe W&amp;B et `my-project-name` par le nom de votre projet.

```python
import wandb

entity = "<my-team>"  # Définissez l'entité sur le nom de votre équipe W&B
project = "my-project-name"  # Définissez le projet sur le nom de votre projet W&B

# À la sortie de ce bloc, le système attend que les données enregistrées finissent de se téléverser.
# Si une exception est levée, le run est marqué comme échoué.
with wandb.init(entity=entity, project="my-project-name") as run:
  # Enregistrez les entrées du modèle et les hyperparamètres.
  run.config.learning_rate = 0.01

  # Exécutez le code de votre expérience.
  for epoch in range(num_epochs):
```
