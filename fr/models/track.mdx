Suivez des expériences de machine learning avec seulement quelques lignes de code. Vous pouvez ensuite consulter les résultats dans un [tableau de bord interactif](/fr/models/track/workspaces/) ou exporter vos données vers Python pour y accéder de façon programmatique à l’aide de notre [API publique](/fr/models/ref/python/public-api/).

Utilisez les intégrations W&amp;B avec des frameworks populaires tels que [Keras](/fr/models/integrations/keras). Voir [les intégrations W&amp;B](/fr/models/integrations) pour la liste complète des intégrations et savoir comment ajouter W&amp;B à votre code.

<div id="how-it-works">
  ## Fonctionnement
</div>

Suivez une expérience de machine learning avec quelques lignes de code :

1. Créez un [W&amp;B Run](/fr/models/runs/).
2. Enregistrez un dictionnaire d&#39;hyperparamètres, comme le taux d&#39;apprentissage ou le type de modèle, dans votre configuration ([`wandb.Run.config`](/fr/models/track/config/)).
3. Journalisez des métriques ([`wandb.Run.log()`](/fr/models/track/log/)) au fil de l&#39;entraînement, par exemple l&#39;accuracy et la perte.
4. Enregistrez les sorties d&#39;un run, comme les poids du modèle ou un tableau de prédictions.

  ## Pour commencer

Selon votre cas d’utilisation, explorez les ressources suivantes pour démarrer avec W&amp;B Experiments :

* Lisez le [Démarrage rapide W&amp;B](/fr/models/quickstart/) pour découvrir, étape par étape, les commandes du W&amp;B Python SDK que vous pouvez utiliser pour créer, suivre et utiliser un artifact de jeu de données.
* Explorez ce chapitre pour apprendre à :
  * Créer une expérience
  * Configurer des expériences
  * Journaliser les données d’expériences
  * Consulter les résultats des expériences
* Explorez la [bibliothèque Python de W&amp;B](/fr/models/ref/python/) dans le [Guide de référence de l’API W&amp;B](/fr/models/ref/python/).
