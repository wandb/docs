W&amp;B prend en charge des raccourcis clavier pour vous aider à naviguer et à interagir plus efficacement avec les expériences, les espaces de travail et les données. Ce guide de référence présente les raccourcis clavier de la W&amp;B App et de l&#39;interface de terminal W&amp;B LEET (Lightweight Experiment Exploration Tool).

<Tabs>
  <Tab title="App">
    Ces raccourcis clavier fonctionnent dans la W&amp;B App.

    ## Gestion de l&#39;espace de travail

    | Shortcut                                             | Description                                                                                                    |
    | ---------------------------------------------------- | -------------------------------------------------------------------------------------------------------------- |
    | **Cmd+Z** (macOS) / **Ctrl+Z** (Windows/Linux)       | Annulez une modification apportée dans l&#39;interface, par exemple à l&#39;espace de travail ou à un panneau. |
    | **Cmd+Shift+Z** (macOS) / **Ctrl+Y** (Windows/Linux) | Rétablissez une modification précédemment annulée dans l&#39;espace de travail.                                |

    ## Navigation

    | Shortcut                                       | Description                                                                                                                                                                                                               |
    | ---------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
    | **Tab**                                        | Naviguez entre les éléments interactifs.                                                                                                                                                                                  |
    | **Cmd+J** (macOS) / **Ctrl+J** (Windows/Linux) | Basculez entre les onglets Workspaces et Runs dans la barre latérale du projet.                                                                                                                                           |
    | **Cmd+.** (macOS) / **Ctrl+.** (Windows/Linux) | Réduisez ou restaurez la barre latérale du sélecteur de Runs pour libérer de l&#39;espace à l&#39;écran.                                                                                                                  |
    | **Cmd+K** (macOS) / **Ctrl+K** (Windows/Linux) | Ouvrez la boîte de dialogue de recherche rapide pour rechercher parmi les projets, les runs et d&#39;autres ressources.                                                                                                   |
    | **Esc**                                        | Dans toute la W&amp;B App, quittez les vues de panneau en plein écran, fermez les panneaux de paramètres, fermez la boîte de dialogue de recherche rapide, fermez un éditeur ou fermez d&#39;autres fenêtres superposées. |

    ## Navigation dans les panneaux

    | Shortcut                                                                                                 | Description                                                                                   |
    | -------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------- |
    | **Left Arrow / Right Arrow**                                                                             | Parcourez les panneaux d&#39;une section en mode plein écran.                                 |
    | **Esc**                                                                                                  | Quittez la vue de panneau en plein écran et revenez à l&#39;espace de travail.                |
    | **Cmd+Left Arrow / Cmd+Right Arrow** (macOS)<br />**Ctrl+Left Arrow / Ctrl+Right Arrow** (Windows/Linux) | Lorsqu&#39;un panneau multimédia est affiché en plein écran, déplacez le curseur d&#39;étape. |

    ## Reports

    | Shortcut               | Description                                                                   |
    | ---------------------- | ----------------------------------------------------------------------------- |
    | **Delete / Backspace** | Supprimez la grille de panneaux sélectionnée du rapport.                      |
    | **Enter**              | Insérez un bloc Markdown après avoir saisi &quot;/mark&quot; dans un rapport. |
    | **Esc**                | Quittez l&#39;éditeur de rapport.                                             |
    | **Tab**                | Naviguez entre les éléments interactifs dans un rapport.                      |

    ## Notes

    * La plupart des raccourcis clavier utilisent **Cmd** sur macOS et **Ctrl** sur Windows/Linux.
    * La W&amp;B App implémente une gestion personnalisée de certains raccourcis par défaut du navigateur.
    * Certains raccourcis dépendent du contexte et ne fonctionnent que dans des zones spécifiques de l&#39;application.
  </Tab>
