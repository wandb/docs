W&amp;B propose des raccourcis clavier pour vous aider à naviguer plus efficacement et à interagir avec les expériences, les espaces de travail et les données. Ce guide de référence présente les raccourcis clavier de l&#39;application W&amp;B et de l&#39;interface de terminal W&amp;B LEET (Lightweight Experiment Exploration Tool).

<Tabs>
  <Tab title="App">
    Ces raccourcis clavier fonctionnent dans l&#39;application W&amp;B.

    ## Gestion du Workspace

    | Shortcut                                             | Description                                                                                                       |
    | ---------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------- |
    | **Cmd+Z** (macOS) / **Ctrl+Z** (Windows/Linux)       | Annuler une modification effectuée dans l&#39;interface utilisateur, par exemple dans le Workspace ou un panneau. |
    | **Cmd+Shift+Z** (macOS) / **Ctrl+Y** (Windows/Linux) | Rétablir une modification précédemment annulée dans le Workspace.                                                 |

    ## Navigation

    | Shortcut                                       | Description                                                                                                                                                                                                                         |
    | ---------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
    | **Tab**                                        | Naviguer entre les éléments interactifs.                                                                                                                                                                                            |
    | **Cmd+J** (macOS) / **Ctrl+J** (Windows/Linux) | Basculer entre les onglets Workspaces et Runs dans la barre latérale du projet.                                                                                                                                                     |
    | **Cmd+.** (macOS) / **Ctrl+.** (Windows/Linux) | Réduire ou restaurer la barre latérale du sélecteur de Runs pour libérer de l&#39;espace à l&#39;écran.                                                                                                                             |
    | **Cmd+K** (macOS) / **Ctrl+K** (Windows/Linux) | Ouvrir la boîte de dialogue de recherche rapide pour rechercher parmi les projets, les runs et d&#39;autres ressources.                                                                                                             |
    | **Esc**                                        | Dans toute l&#39;application W&amp;B, quitter les vues de panneau en plein écran, fermer les volets de paramètres, fermer la boîte de dialogue de recherche rapide, fermer un éditeur ou masquer d&#39;autres fenêtres superposées. |

    ## Navigation dans les panneaux

    | Shortcut                                                                                                 | Description                                                                                          |
    | -------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------- |
    | **Left Arrow / Right Arrow**                                                                             | Parcourir les panneaux d&#39;une section en mode plein écran.                                        |
    | **Esc**                                                                                                  | Quitter la vue du panneau en plein écran et revenir au Workspace.                                    |
    | **Cmd+Left Arrow / Cmd+Right Arrow** (macOS)<br />**Ctrl+Left Arrow / Ctrl+Right Arrow** (Windows/Linux) | Lors de l&#39;affichage d&#39;un panneau multimédia en plein écran, déplacer le curseur d&#39;étape. |

    ## Panneaux multimédias

    | Shortcut                                                  | Description                                                                                                                                 |
    | --------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
    | **Cmd + +/-** (macOS)<br />**Ctrl + +/-** (Windows/Linux) | Lors de l&#39;affichage d&#39;une image en plein écran, zoomer ou dézoomer.<br />Une fois zoomé, cliquer-glisser pour déplacer l&#39;image. |
    | **Cmd + 0** (macOS)<br />**Ctrl + 0** (Windows/Linux)     | Lors de l&#39;affichage d&#39;une image en plein écran, réinitialiser le zoom à 100 %.                                                      |
    | **Shift + L**                                             | Lors de l&#39;affichage d&#39;une image en plein écran, ajuster le zoom à la fenêtre.                                                       |

    ## Reports

    | Shortcut               | Description                                                                   |
    | ---------------------- | ----------------------------------------------------------------------------- |
    | **Delete / Backspace** | Supprimer la grille de panneaux sélectionnée du rapport.                      |
    | **Enter**              | Insérer un bloc Markdown après avoir saisi &quot;/mark&quot; dans un rapport. |
    | **Esc**                | Quitter l&#39;éditeur de rapport.                                             |
    | **Tab**                | Naviguer entre les éléments interactifs dans un rapport.                      |

    ## Notes

    * La plupart des raccourcis clavier utilisent **Cmd** sur macOS et **Ctrl** sur Windows/Linux.
    * L&#39;application W&amp;B implémente une gestion personnalisée pour certains raccourcis par défaut du navigateur.
    * Certains raccourcis dépendent du contexte et ne fonctionnent que dans des zones spécifiques de l&#39;application.
  </Tab>
