Cette page explique comment utiliser les nuages de points dans W&amp;B.

<div id="use-case">
  ## Cas d’utilisation
</div>

Utilisez des nuages de points pour comparer plusieurs runs et visualiser les performances d’une expérience :

* Tracez des lignes indiquant les valeurs minimales, maximales et moyennes.
* Personnalisez les infobulles des métadonnées.
* Contrôlez la couleur des points.
* Ajustez les plages des axes.
* Utilisez une échelle logarithmique pour les axes.

<div id="create-a-scatter-plot">
  ## Créer un nuage de points
</div>

Pour créer un nuage de points dans l’interface W&amp;B :

1. Accédez à l’onglet **Workspaces**.
2. Dans le panneau **Graphiques**, cliquez sur le menu d’actions `...`.
3. Dans le menu contextuel, sélectionnez **Ajouter des panneaux**.
4. Dans le menu **Ajouter des panneaux**, sélectionnez **Nuage de points**.
5. Définissez les axes `x` et `y` pour représenter les données que vous souhaitez afficher. Si vous le souhaitez, définissez des plages minimale et maximale pour vos axes ou ajoutez un axe `z`.
6. Cliquez sur **Appliquer** pour créer le nuage de points.
7. Affichez le nouveau nuage de points dans le panneau **Graphiques**.
