Les graphiques en coordonnées parallèles résument en un coup d’œil la relation entre un grand nombre d’hyperparamètres et de métriques du modèle.

<Frame>
  <img src="/images/app_ui/parallel_coordinates.gif" alt="Graphique en coordonnées parallèles" />
</Frame>

Un panneau de coordonnées parallèles comporte les composants suivants :

* **Axes** : Hyperparamètres de [`wandb.Run.config`](/fr/models/tables/evaluate-models) et métriques de [`wandb.Run.log()`](/fr/models/tables/evaluate-models).
* **Lignes** : Chaque ligne représente un run. Survolez une ligne pour afficher une info-bulle avec des détails sur le run. Toutes les lignes correspondant aux filtres actuels s’affichent. Si vous désactivez l’icône en forme d’œil pour un run, sa ligne est atténuée.

<div id="create-a-parallel-coordinates-panel">
  ## Créer un panneau de coordonnées parallèles
</div>

Pour ajouter un panneau de coordonnées parallèles à votre Workspace :

1. Accédez à la page d’accueil de votre Workspace.
2. Cliquez sur **Add Panels**.
3. Sélectionnez **Coordonnées parallèles**.

<div id="panel-settings">
  ## Paramètres du panneau
</div>

Pour configurer le panneau, cliquez sur le bouton Edit situé dans son coin supérieur droit.

Vous pouvez ajuster les paramètres suivants :

* **Infobulle** : au survol, une légende s’affiche avec des informations sur chaque run.
* **Titres** : modifiez les titres des axes pour les rendre plus lisibles.
* **Gradient** : personnalisez le gradient avec la plage de couleurs de votre choix.
* **Échelle logarithmique** : chaque axe peut être affiché indépendamment sur une échelle logarithmique.
* **Inverser l’axe** : inversez le sens de l’axe — c’est utile lorsque vous avez à la fois des colonnes accuracy et loss.

Pour un exemple interactif, voir [un panneau de coordonnées parallèles en direct](https://app.wandb.ai/example-team/sweep-demo/reports/Zoom-in-on-Parallel-Coordinates-Charts--Vmlldzo5MTQ4Nw).
