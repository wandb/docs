Les graphiques en coordonnées parallèles résument d’un coup d’œil la relation entre un grand nombre d’hyperparamètres et les métriques du modèle.

<Frame>
  <img src="/images/app_ui/parallel_coordinates.gif" alt="Graphique en coordonnées parallèles" />
</Frame>

* **Axes** : différents hyperparamètres issus de [`wandb.Run.config`](/fr/models/tables/evaluate-models) et métriques issues de [`wandb.Run.log()`](/fr/models/tables/evaluate-models).
* **Lignes** : chaque ligne représente un run. Survolez une ligne pour afficher une infobulle contenant des détails sur le run. Toutes les lignes correspondant aux filtres actuels sont affichées, mais si vous désactivez l’œil, elles apparaissent grisées.

<div id="create-a-parallel-coordinates-panel">
  ## Créer un panneau de coordonnées parallèles
</div>

1. Accédez à la page d’accueil de votre espace de travail
2. Cliquez sur **Add Panels**
3. Sélectionnez **coordonnées parallèles**

<div id="panel-settings">
  ## Paramètres du panneau
</div>

Pour configurer le panneau, cliquez sur le bouton de modification dans le coin supérieur droit du panneau.

* **Info-bulle** : Au survol, une légende s’affiche avec des informations sur chaque run
* **Titres** : Modifiez les titres des axes pour les rendre plus lisibles
* **Gradient** : Personnalisez le gradient avec la plage de couleurs de votre choix
* **Échelle logarithmique** : Chaque axe peut être configuré indépendamment pour s’afficher sur une échelle logarithmique
* **Inverser l’axe** : Inversez le sens de l’axe — utile lorsque vous avez à la fois la précision et la perte comme colonnes
