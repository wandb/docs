title: Graphiques en barres
---

Un graphique en barres présente des données catégorielles à l’aide de barres rectangulaires, affichées verticalement ou horizontalement. Les graphiques en barres s’affichent par défaut avec `wandb.Run.log()` lorsque toutes les valeurs enregistrées sont de longueur 1.

<Frame>
  <img src="/images/app_ui/bar_plot.png" alt="Affichage de box plots et de graphiques en barres horizontaux dans W&B" />
</Frame>

Personnalisez-les dans les paramètres du graphique pour limiter le nombre maximal de runs affichés, regrouper les runs selon n’importe quelle configuration et renommer les libellés.

<Frame>
  <img src="/images/app_ui/bar_plot_custom.png" alt="Graphique en barres personnalisé" />
</Frame>

<div id="customize-bar-plots">
  ## Personnaliser les graphiques en barres
</div>

Vous pouvez également créer des graphiques **Box** ou **Violin** afin de regrouper de nombreuses statistiques de synthèse dans un seul type de graphique**.**

1. Regroupez les runs via le tableau des runs.
2. Cliquez sur &#39;Add panel&#39; dans le Workspace.
3. Ajoutez un &#39;Bar Chart&#39; standard et sélectionnez la métrique à représenter.
4. Sous l&#39;onglet &#39;Grouping&#39;, sélectionnez &#39;box plot&#39; ou &#39;Violin&#39;, etc., pour utiliser l&#39;un de ces styles.
