---
title: Supprimer un registre
description: "Supprimez un W&B Registry en tant qu’administrateur d’équipe ou administrateur du registre à l’aide du SDK Python ou de l’interface W&B App."
---

Cette page explique comment un administrateur d’équipe ou un administrateur du registre peut supprimer un registre.

* Un administrateur d’équipe peut supprimer n’importe quel registre de l’organisation.
* Un administrateur du registre peut supprimer un registre qu’il a créé.

La suppression d’un registre supprime également les collections qui appartiennent à ce registre, mais ne supprime pas les Artifacts liés au registre. Un artifact de ce type reste dans le projet d’origine où il a été enregistré.

<Tabs>
  <Tab title="Python SDK">
    Utilisez la méthode `delete()` de l’API `wandb` pour supprimer un registre par programmation. L’exemple suivant montre comment :

    1. Récupérer le registre que vous souhaitez supprimer avec `api.registry()`.
    2. Appeler la méthode `delete()` sur l’objet registre renvoyé afin de supprimer le registre.
  </Tab>

  <Tab title="W&B App">
    1. Accédez à W&amp;B Registry à l’adresse [https://wandb.ai/registry/](https://wandb.ai/registry/).
    2. Sélectionnez le registre que vous souhaitez supprimer.
    3. Cliquez sur l’icône d’engrenage dans le coin supérieur droit pour afficher les paramètres du registre.
    4. Pour supprimer le registre, cliquez sur l’icône de corbeille dans le coin supérieur droit de la page des paramètres.
    5. Confirmez la suppression du registre en saisissant son nom dans la fenêtre modale qui s’affiche, puis cliquez sur **Delete**.
  </Tab>
</Tabs>
