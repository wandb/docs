<div id="export-reports">
  ## Exporter des rapports
</div>

Exportez un rapport au format PDF ou LaTeX. Dans votre rapport, sélectionnez le menu **action (<Icon icon="ellipsis-vertical" iconType="solid" />)** pour ouvrir le menu déroulant. Choisissez **Download and**, puis sélectionnez le format de sortie PDF ou LaTeX.

<div id="cloning-reports">
  ## Cloner des rapports
</div>

<Tabs>
  <Tab title="W&B App">
    Dans votre rapport, sélectionnez le **menu d’actions (<Icon icon="ellipsis-vertical" iconType="solid" />)** pour afficher le menu déroulant. Cliquez sur le bouton **Cloner ce rapport**. Dans la fenêtre modale, choisissez une destination pour le rapport cloné. Cliquez sur **Cloner le rapport**.

    <Frame>
      <img src="/images/reports/clone_reports.gif" alt="Clonage de rapports" />
    </Frame>

    Clonez un rapport pour réutiliser le modèle et le format d’un projet. Les projets clonés sont visibles par votre équipe si vous clonez un projet dans le compte de l’équipe. Les projets clonés dans le compte d’un utilisateur individuel ne sont visibles que par cet utilisateur.
  </Tab>

  <Tab title="Report and Workspace API">
    Chargez un rapport à partir d’une URL pour l’utiliser comme modèle.

    Modifiez le contenu de `new_report.blocks`.

    ```python
    pg = wr.PanelGrid(
        runsets=[
            wr.Runset(ENTITY, PROJECT, "First Run Set"),
            wr.Runset(ENTITY, PROJECT, "Elephants Only!", query="elephant"),
        ],
        panels=[
            wr.LinePlot(x="Step", y=["val_acc"], smoothing_factor=0.8),
            wr.BarPlot(metrics=["acc"]),
            wr.MediaBrowser(media_keys="img", num_columns=1),
            wr.RunComparer(diff_only="split", layout={"w": 24, "h": 9}),
        ],
    )
    new_report.blocks = (
        report.blocks[:1] + [wr.H1("Panel Grid Example"), pg] + report.blocks[1:]
    )
    new_report.save()
    ```
  </Tab>
</Tabs>
