<div id="export-reports">
  ## Exporter des rapports
</div>

Exportez un rapport au format PDF ou LaTeX. Dans votre rapport, sélectionnez l’icône à trois points pour ouvrir le menu déroulant. Choisissez **Télécharger**, puis sélectionnez le format de sortie PDF ou LaTeX.

<div id="cloning-reports">
  ## Cloner des rapports
</div>

<Tabs>
  <Tab title="W&B App">
    Dans votre rapport, sélectionnez l’icône à trois points pour ouvrir le menu déroulant. Choisissez le bouton **Clone this report**. Sélectionnez une destination pour le rapport cloné dans la fenêtre modale. Choisissez **Clone report**.

    <Frame>
      <img src="/images/reports/clone_reports.gif" alt="Cloner des rapports" />
    </Frame>

    Clonez un rapport pour réutiliser le modèle et le format d’un projet. Un rapport cloné est visible par votre équipe si vous le clonez dans le compte de l’équipe. Un rapport cloné dans le compte d’une personne n’est visible que par cet utilisateur.
  </Tab>

  <Tab title="API Report and Workspace">
    Chargez un rapport à partir d’une URL pour l’utiliser comme modèle.

    ```python
    report = wr.Report(
        project=PROJECT, title="Quickstart Report", description="That was easy!"
    )  # Créer
    report.save()  # Enregistrer
    new_report = wr.Report.from_url(report.url)  # Charger
    ```

    Modifiez le contenu de `new_report.blocks`.

    ```python
    pg = wr.PanelGrid(
        runsets=[
            wr.Runset(ENTITY, PROJECT, "First Run Set"),
            wr.Runset(ENTITY, PROJECT, "Elephants Only!", query="elephant"),
        ],
        panels=[
            wr.LinePlot(x="Step", y=["val_acc"], smoothing_factor=0.8),
            wr.BarPlot(metrics=["acc"]),
            wr.MediaBrowser(media_keys="img", num_columns=1),
            wr.RunComparer(diff_only="split", layout={"w": 24, "h": 9}),
        ],
    )
    new_report.blocks = (
        report.blocks[:1] + [wr.H1("Panel Grid Example"), pg] + report.blocks[1:]
    )
    new_report.save()
    ```
  </Tab>
</Tabs>
