</Note>

Cette page décrit comment exporter un rapport W&amp;B vers un format de fichier portable. Elle explique également comment cloner un rapport existant afin de réutiliser sa structure comme point de départ pour de nouveaux travaux.

<div id="export-a-report">
  ## Exporter un rapport
</div>

Exportez un rapport au format PDF ou LaTeX pour partager son contenu en dehors de W&amp;B ou pour archiver une version statique de votre analyse. Dans votre rapport, sélectionnez le **menu d’actions (<Icon icon="ellipsis-vertical" iconType="solid" />)**. Choisissez **Download** et sélectionnez le format de sortie PDF ou LaTeX.

<div id="clone-a-report">
  ## Cloner un rapport
</div>

Clonez un rapport pour réutiliser le modèle et la mise en forme d’un projet existant comme base d’un nouveau rapport. Vous pouvez cloner des rapports dans l’interface de l’application W&amp;B ou par programmation avec la Report and Workspace API.

<Tabs>
  <Tab title="W&B App">
    Dans votre rapport, sélectionnez le **menu d’actions (<Icon icon="ellipsis-vertical" iconType="solid" />)**. Cliquez sur **Cloner ce rapport**. Dans la fenêtre modale, choisissez une destination pour le rapport cloné. Cliquez sur **Cloner le rapport**.

    <Frame>
      <img src="/images/reports/clone_reports.gif" alt="Clonage de rapports" />
    </Frame>

    Lorsque vous clonez un rapport, vous en définissez la destination. Si vous le clonez dans une équipe, tous les membres de l’équipe peuvent le voir. Si vous le clonez dans votre compte personnel, vous seul pouvez le voir par défaut.
  </Tab>

  <Tab title="Report and Workspace API">
    Utilisez la Report and Workspace API pour charger un rapport existant à partir de son URL et le réutiliser comme modèle pour un nouveau rapport.

    Chargez un rapport à partir d’une URL pour l’utiliser comme modèle. Remplacez `PROJECT` par le nom de votre projet W&amp;B.

    ```python
    report = wr.Report(
        project=PROJECT, title="Quickstart Report", description="That was easy!"
    )  # Créer
    report.save()  # Enregistrer
    new_report = wr.Report.from_url(report.url)  # Charger
    ```

    Après avoir chargé le rapport, modifiez le contenu de `new_report.blocks` pour personnaliser le rapport cloné, puis enregistrez-le. Remplacez `ENTITY` par le nom de votre entité W&amp;B.

    ```python
    pg = wr.PanelGrid(
        runsets=[
            wr.Runset(ENTITY, PROJECT, "First Run Set"),
            wr.Runset(ENTITY, PROJECT, "Elephants Only!", query="elephant"),
        ],
        panels=[
            wr.LinePlot(x="Step", y=["val_acc"], smoothing_factor=0.8),
            wr.BarPlot(metrics=["acc"]),
            wr.MediaBrowser(media_keys="img", num_columns=1),
            wr.RunComparer(diff_only="split", layout={"w": 24, "h": 9}),
        ],
    )
    new_report.blocks = (
        report.blocks[:1] + [wr.H1("Panel Grid Example"), pg] + report.blocks[1:]
    )
    new_report.save()
    ```
  </Tab>
</Tabs>
