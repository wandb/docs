<Note>
  La W&amp;B Report and Workspace API est en préversion publique.
</Note>

Sélectionnez un onglet ci-dessous pour découvrir comment créer un rapport dans la W&amp;B App ou par programmation avec la W&amp;B Report and Workspace API.

Consultez ce [Google Colab](https://colab.research.google.com/github/wandb/examples/blob/master/colabs/intro/Report_API_Quickstart.ipynb) pour voir un exemple de création de rapport par programmation.

<Tabs>
  <Tab title="W&B App">
    1. Accédez à l’espace de travail de votre projet dans la W&amp;B App.

    2. Cliquez sur **Créer un rapport** dans le coin supérieur droit de votre espace de travail.

       <Frame>
         <img src="/images/reports/create_a_report_button.png" alt="Bouton de création de rapport" />
       </Frame>

    3. Une fenêtre modale apparaît. Sélectionnez les graphiques avec lesquels vous souhaitez commencer. Vous pourrez ajouter ou supprimer des graphiques plus tard depuis l’interface du rapport.

       <Frame>
         <img src="/images/reports/create_a_report_modal.png" alt="Fenêtre modale de création de rapport" />
       </Frame>

    4. Sélectionnez l’option **Filter run sets** pour empêcher l’ajout de nouveaux runs à votre rapport. Vous pouvez activer ou désactiver cette option. Une fois que vous avez cliqué sur **Créer un rapport**, un brouillon de rapport sera disponible dans l’onglet du rapport pour poursuivre votre travail.
  </Tab>

  <Tab title="Onglet Report">
    1. Accédez à l’espace de travail de votre projet dans la W&amp;B App.
    2. Sélectionnez l’onglet **Reports** (icône de presse-papiers) dans votre projet.
    3. Sélectionnez le bouton **Create Report** sur la page du rapport.

       <Frame>
         <img src="/images/reports/create_report_button.png" alt="Bouton de création de rapport" />
       </Frame>
  </Tab>

  <Tab title="Report and Workspace API">
    Créez un rapport par programmation :

    1. Installez le SDK W&amp;B (`wandb`) et la Report and Workspace API (`wandb-workspaces`) :
       ```bash
       pip install wandb wandb-workspaces
       ```

    2. Ensuite, importez le module workspaces
       ```python
       import wandb
       import wandb_workspaces.reports.v2 as wr
       ```

    3. Créez un rapport avec `wandb_workspaces.reports.v2.Report`. Créez une instance de rapport à l’aide de l’API publique de la classe Report ([`wandb.apis.reports`](/fr/models/ref/python/public-api/api#reports)). Spécifiez un nom pour le projet.
       ```python
       report = wr.Report(project="report_standard")
       ```

    4. Enregistrez le rapport. Les Reports ne sont pas téléversés vers le serveur W&amp;B tant que vous n’appelez pas la méthode `save()` :
       ```python
       report.save()
       ```

    Pour savoir comment modifier un rapport de manière interactive dans l’UI de l’App ou par programmation, voir [Modifier un rapport](/fr/models/reports/edit-a-report/).
  </Tab>
</Tabs>
