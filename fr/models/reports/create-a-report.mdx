<Note>
  L&#39;API Report and Workspace de W&amp;B est en préversion publique.
</Note>

Sélectionnez un onglet pour découvrir comment créer un rapport dans la W&amp;B App ou par programmation avec l&#39;API Report and Workspace.

Pour voir un exemple de création programmatique d&#39;un rapport, consultez ce [Google Colab](https://colab.research.google.com/github/wandb/examples/blob/master/colabs/intro/Report_API_Quickstart.ipynb).

<Tabs>
  <Tab title="W&B App">
    1. Accédez à l&#39;espace de travail de votre projet dans la W&amp;B App.

    2. Dans l&#39;angle supérieur droit de votre espace de travail, cliquez sur **Create report**.

       <Frame>
         <img src="/images/reports/create_a_report_button.png" alt="Bouton Create report" />
       </Frame>

    3. Une fenêtre modale s&#39;affiche. Sélectionnez les graphiques avec lesquels vous souhaitez commencer. Vous pourrez ajouter ou supprimer des graphiques plus tard depuis l&#39;interface du rapport.

       <Frame>
         <img src="/images/reports/create_a_report_modal.png" alt="Fenêtre modale Create report" />
       </Frame>

    4. Sélectionnez l&#39;option **Filter run sets** pour empêcher l&#39;ajout de nouveaux runs à votre rapport. Vous pouvez activer ou désactiver cette option. Un brouillon de rapport est enregistré automatiquement. Accédez-y dans l&#39;onglet **Reports**.
  </Tab>

  <Tab title="Onglet Reports">
    1. Accédez à l&#39;espace de travail de votre projet dans la W&amp;B App.
    2. Sélectionnez l&#39;onglet **Reports** dans votre projet.
    3. Cliquez sur **Create report**.

       <Frame>
         <img src="/images/reports/create_report_button.png" alt="Bouton Create report" />
       </Frame>
  </Tab>

  <Tab title="API Report and Workspace">
    Créez un rapport par programmation :

    1. Installez le SDK W&amp;B (`wandb`) et l&#39;API Report and Workspace (`wandb-workspaces`) :
       ```bash
       pip install wandb wandb-workspaces
       ```

    2. Importez le SDK Python W&amp;B et l&#39;API Report and Workspace.
       ```python
       import wandb
       import wandb_workspaces.reports.v2 as wr
       ```

    3. Créez un rapport avec `wandb_workspaces.reports.v2.Report`. Créez une instance de rapport à l&#39;aide de l&#39;API publique de la classe Report ([`wandb.apis.reports`](/fr/models/ref/python/public-api/api#reports)). Spécifiez un nom de projet.
       ```python
       report = wr.Report(project="report_standard")
       ```

    4. Enregistrez le rapport. Les rapports ne sont pas téléversés vers W&amp;B tant que vous n&#39;appelez pas la méthode `.save()` :
       ```python
       report.save()
       ```

    Pour plus d&#39;informations, voir [Modifier un rapport](/fr/models/reports/edit-a-report).
  </Tab>
</Tabs>
