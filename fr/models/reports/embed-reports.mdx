Intégrez un rapport W&amp;B dans une page ou un outil externe afin que votre équipe puisse consulter les résultats d’expérience en direct aux côtés d’autres documents. Cette page explique comment générer un code d’intégration à partir d’un rapport et l’utiliser dans HTML, Confluence, Notion ou Gradio.

<div id="html-iframe-element">
  ## Élément HTML `iframe`
</div>

Utilisez un élément HTML `iframe` pour intégrer un rapport dans n’importe quelle page web qui accepte le HTML personnalisé. W&amp;B fournit un code d’intégration prédéfini que vous pouvez copier directement depuis le rapport.

Sélectionnez le bouton **Share** en haut à droite d’un rapport pour ouvrir une boîte de dialogue, puis sélectionnez **Copy embed code**. L’extrait copié est un élément HTML `iframe` que vous pouvez coller dans n’importe quelle page qui accepte le HTML personnalisé.

<Note>
  Seuls les rapports *publics* sont visibles lorsqu’ils sont intégrés.
</Note>

<div id="gradio">
  ## Gradio
</div>

Pour intégrer un rapport W&amp;B dans une application Gradio, y compris une application hébergée sur Hugging Face Spaces, utilisez l’élément `gr.HTML` de Gradio pour afficher une `iframe` pointant vers l’URL du rapport.

```python
import gradio as gr


def wandb_report(url):
    iframe = f'<iframe src={url} style="border:none;height:1024px;width:100%">'
    return gr.HTML(iframe)


with gr.Blocks() as demo:
    report = wandb_report(
        "https://wandb.ai/_scott/pytorch-sweeps-demo/reports/loss-22-10-07-16-00-17---VmlldzoyNzU2NzAx"
    )
demo.launch()
```
