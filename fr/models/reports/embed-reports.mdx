<div id="html-iframe-element">
  ## Élément HTML iframe
</div>

Sélectionnez le bouton **Share** dans le coin supérieur droit d’un rapport. Une fenêtre modale apparaît. Dans cette fenêtre modale, sélectionnez **Copy embed code**. Le code copié sera rendu dans un élément HTML Inline Frame (IFrame). Collez le code copié dans l’élément HTML iframe de votre choix.

<Note>
  Seuls les rapports **publics** peuvent être consultés lorsqu’ils sont intégrés.
</Note>

<div id="gradio">
  ## Gradio
</div>

Vous pouvez utiliser l’élément `gr.HTML` pour intégrer des rapports W&amp;B dans des applications Gradio et les utiliser dans Hugging Face Spaces.

```python
import gradio as gr


def wandb_report(url):
    iframe = f'<iframe src={url} style="border:none;height:1024px;width:100%">'
    return gr.HTML(iframe)


with gr.Blocks() as demo:
    report = wandb_report(
        "https://wandb.ai/_scott/pytorch-sweeps-demo/reports/loss-22-10-07-16-00-17---VmlldzoyNzU2NzAx"
    )
demo.launch()
```
