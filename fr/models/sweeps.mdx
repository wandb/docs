Utilisez W&amp;B sweep pour automatiser la recherche d&#39;hyperparamètres et visualiser un suivi des expériences riche en éléments interactifs. Choisissez parmi des méthodes de recherche populaires, comme la recherche bayésienne, la recherche sur grille et la recherche aléatoire, pour explorer l&#39;espace des hyperparamètres. Faites évoluer et parallélisez le sweep sur une ou plusieurs machines.

<div id="how-it-works">
  ## Fonctionnement
</div>

Créez un sweep à l’aide de deux commandes [W&amp;B CLI](/fr/models/ref/cli/) :

1. Initialisez un sweep.

```bash
  wandb sweep --project <project-name> <path-to-config file>
```

2. Lancez l’agent de sweep.

```bash
  wandb agent <sweep-ID>
```

<Note>
  L’extrait de code précédent, ainsi que le notebook Colab lié depuis cette page, montrent comment initialiser et créer un sweep avec la CLI W&amp;B. Voir le [guide pas à pas de sweep](/fr/models/sweeps/walkthrough/) pour utiliser le SDK Python afin de configurer, d’initialiser et d’exécuter un sweep.
</Note>
