Choisissez parmi des méthodes de recherche populaires, comme la recherche bayésienne, la recherche sur grille et la recherche aléatoire, pour explorer l&#39;espace des hyperparamètres. <div id="how-it-works">
  ## Fonctionnement
</div>

Créez un balayage à l’aide de deux commandes [W&amp;B CLI](/fr/models/ref/cli/) :

1. Initialisez un balayage.

```bash
  wandb sweep --project <project-name> <path-to-config file>
```

2. Démarrez l’agent de balayage.

```bash
  wandb agent <sweep-ID>
```

<Note>
  L’extrait de code précédent, ainsi que le notebook Colab lié depuis cette page, montrent comment initialiser et créer un sweep avec la CLI W&amp;B. Voir le [guide pas à pas de Sweeps](/fr/models/sweeps/walkthrough/) pour utiliser le SDK Python afin de configurer, d’initialiser et d’exécuter un sweep.
</Note>
