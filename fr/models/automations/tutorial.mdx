Cette page renvoie vers deux tutoriels complets qui vous montrent comment créer une automatisation dans W&amp;B. Chaque tutoriel présente un scénario différent, afin que vous puissiez suivre celui qui correspond à ce que vous voulez créer. Sélectionnez un tutoriel pour obtenir des instructions détaillées.

* **[Tutoriel d&#39;automatisation de projet](/fr/models/automations/project-automation-tutorial)**: Recevoir une alerte lorsqu&#39;un run échoue (notification Slack).
* **[Tutoriel d&#39;automatisation du registre](/fr/models/automations/registry-automation-tutorial)**: Déclencher un webhook lorsqu&#39;un alias (par exemple, `production`) est ajouté à un artefact.
