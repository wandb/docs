Gérez les automatisations W&amp;B de manière programmatique avec l’API Python `wandb` lorsque vous automatisez des flux de travail par script plutôt que d’utiliser le W&amp;B App.

<Note>
  La **création** et la **mise à jour** programmatiques des automatisations via `wandb.Api().create_automation()` et les fonctions utilitaires associées peuvent échouer sur certaines versions du client Python `wandb` en raison d’une vérification de fonctionnalité côté client (par exemple, pour les automatisations basées sur l’état d’exécution). En attendant la publication d’un correctif du SDK, utilisez le [W&amp;B App](/fr/models/automations/create-automations) pour créer et modifier des automatisations. Les méthodes `list`, `get` et `delete` ne sont pas affectées. Suivi interne : WB-34263.
</Note>
