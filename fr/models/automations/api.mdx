<Note>
  La **création** et la **mise à jour** programmatiques des automatisations via `wandb.Api().create_automation()` et les fonctions utilitaires associées peuvent échouer avec les versions récentes du client Python `wandb` en raison d’une vérification de fonctionnalité effectuée côté client (par exemple, pour les automatisations liées à l’état des runs). Utilisez la [W&amp;B App](/fr/models/automations/create-automations) pour créer et modifier des automatisations jusqu’à la publication d’un SDK corrigé. Les opérations de liste, de récupération et de suppression peuvent encore fonctionner avec votre version ; si vous rencontrez des erreurs, passez par l’App. Suivi interne : WB-34263.
</Note>
