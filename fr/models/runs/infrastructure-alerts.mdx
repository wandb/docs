---
title: Visualiser les alertes d’infrastructure CoreWeave
description: "Consultez les alertes d’infrastructure CoreWeave, comme les pannes de GPU et les dépassements thermiques, sur les graphiques de vos runs d’expériences W&B."
---

Observez les alertes d’infrastructure, comme les pannes de GPU, les dépassements thermiques, entre autres, pendant les expériences d’apprentissage automatique que vous journalisez dans W&amp;B. Pendant un [run W&amp;B](/fr/models/runs), [CoreWeave Mission Control](https://coreweave.com/mission-control) surveille votre infrastructure de calcul.

<Note>
  Cette fonctionnalité est en préversion et n’est disponible que lors de l’entraînement sur un cluster CoreWeave. Contactez votre représentant W&amp;B pour obtenir l’accès.
</Note>

Si une erreur se produit, CoreWeave transmet ces informations à W&amp;B. W&amp;B ajoute les informations d’infrastructure sur les graphiques de votre run dans le Workspace de votre projet. CoreWeave tente de résoudre automatiquement certains problèmes, et W&amp;B affiche ces informations sur la page du run.
