title: Personnaliser les couleurs des runs
---

W&amp;B attribue automatiquement une couleur à chaque run que vous créez dans votre projet. Vous pouvez modifier la couleur par défaut d&#39;un run pour le distinguer visuellement des autres runs dans le tableau et les graphiques. Réinitialisez l&#39;espace de travail de votre projet pour restaurer les couleurs par défaut de tous les runs du tableau.

Les couleurs des runs sont définies localement. Sur la page du projet, les couleurs personnalisées s&#39;appliquent uniquement à votre propre espace de travail. Dans Reports, les couleurs personnalisées des runs s&#39;appliquent uniquement au niveau de la section. Vous pouvez visualiser le même run dans différentes sections, chacune pouvant utiliser des couleurs personnalisées différentes.

<div id="edit-default-run-colors">
  ## Modifier les couleurs par défaut des runs
</div>

1. Cliquez sur l&#39;onglet **Runs** dans la barre latérale du projet.
2. Cliquez sur la pastille colorée à côté du nom du run dans la colonne **Nom**.
3. Sélectionnez une couleur dans la palette ou le sélecteur de couleurs, ou saisissez un code hexadécimal.

<Image src="/images/runs/run-color-palette.png" alt="Modifier la couleur par défaut d’un run dans l’espace de travail du projet" />

<div id="randomize-run-colors">
  ## Randomiser les couleurs des runs
</div>

Pour randomiser les couleurs de tous les runs du tableau :

1. Cliquez sur l’onglet **Runs** dans la barre latérale du projet.
2. Survolez l’en-tête de la colonne **Nom**, cliquez sur les trois points horizontaux (**...**), puis sélectionnez **Randomiser les couleurs des runs** dans le menu déroulant.

<Note>
  L’option permettant de randomiser les couleurs des runs n’est disponible qu’après avoir modifié le tableau des runs d’une manière ou d’une autre, par exemple en triant, en filtrant, en effectuant une recherche ou en regroupant.
</Note>

<div id="reset-run-colors">
  ## Réinitialiser les couleurs des runs
</div>

{/* L’option permettant d’attribuer aléatoirement des couleurs aux runs n’est disponible que s’il y a au moins deux runs dans le tableau ou le sélecteur et que vous avez apporté une modification à la vue (tri, filtrage, recherche ou regroupement). */}

Pour restaurer les couleurs par défaut de tous les runs du tableau :

1. Cliquez sur l’onglet **Runs** dans la barre latérale du projet.
2. Survolez l’en-tête de la colonne **Nom**, cliquez sur les trois points horizontaux (**...**), puis sélectionnez **Réinitialiser les couleurs** dans le menu déroulant.
