---
title: Supprimer des runs
description: Découvrez comment supprimer des runs d’un projet W&B à l’aide de la W&B App.
---

Supprimez un ou plusieurs runs d’un projet à l’aide de la W&amp;B App.

1. Accédez au projet qui contient les runs que vous souhaitez supprimer.
2. Sélectionnez l’onglet **Runs**.
3. Cochez la case à côté des runs que vous souhaitez supprimer.
4. Cliquez sur le bouton **Supprimer** (icône de corbeille) au-dessus du tableau.
5. Dans le panneau qui s’affiche, cliquez sur **Supprimer**.

<Note>
  Un ID de run ne peut pas être réutilisé, même après la suppression du run. Toute tentative de réutilisation échouera avec une erreur.
</Note>

<Note>
  Pour les projets qui contiennent un grand nombre de runs, vous pouvez utiliser soit la barre de recherche pour filtrer les runs que vous souhaitez supprimer à l’aide de Regex, soit le bouton **Filtre** pour filtrer les runs en fonction de leur statut, de leurs tags ou d’autres propriétés.
</Note>

  ### Organigramme de suppression des runs

Le diagramme suivant illustre l’ensemble du processus de suppression d’un run, y compris la gestion des artifacts associés et des liens du registre :

```mermaid
graph TB
    Start([L'utilisateur initie<br/>la suppression du run]) --> RunSelect[Sélectionner les Runs<br/>à supprimer]
    
    RunSelect --> DeletePrompt{Supprimer les Artifacts<br/>associés ?}
    
    DeletePrompt -->|Non| DeleteRunOnly[Supprimer le run uniquement<br/><br/>- Métadonnées du run supprimées<br/>- Artifacts toujours disponibles<br/>- Accès aux artifacts conservé]
    
    DeletePrompt -->|Oui| CheckArtifacts[Vérifier les<br/>Artifacts associés]
    
    CheckArtifacts --> HasRegistry{Artifacts liés au<br/>registre de modèles ?}
    
    HasRegistry -->|Oui| RegistryWarning[⚠️ Avertissement<br/><br/>Les modèles du registre seront supprimés<br/>Les alias Production seront affectés]
    HasRegistry -->|Non| DirectDelete
    
    RegistryWarning --> ConfirmRegistry{Confirmer la suppression<br/>du modèle du registre ?}
    
    ConfirmRegistry -->|Non| DeleteRunOnly
    ConfirmRegistry -->|Oui| DirectDelete[Supprimer le run + Artifacts<br/><br/>- Métadonnées du run supprimées<br/>- Artifacts supprimés définitivement<br/>- Liens du registre supprimés<br/>- Impossible à récupérer]
    
    DeleteRunOnly --> PartialEnd([Run supprimé<br/>Artifacts conservés])
    DirectDelete --> FullEnd([Run + Artifacts<br/>supprimés définitivement])

```

<Warning>
  Lorsque vous supprimez un run et choisissez de supprimer les artifacts associés, ceux-ci sont définitivement supprimés et ne peuvent plus être récupérés, même si le run est restauré ultérieurement. Cela inclut les artifacts liés au registre.
</Warning>
