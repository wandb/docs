---
title: Arrêter des runs
description: Arrêtez des runs par programmation à l’aide du W&B Python SDK ou manuellement dans la W&B App.
---

Arrêtez un run par programmation avec le W&amp;B Python SDK ou de manière interactive dans la W&amp;B App.

<Tabs>
  <Tab title="Python SDK">
    1. Accédez au terminal ou à l’éditeur de code où vous avez initialisé le run.
    2. Appuyez sur `Ctrl+D` pour arrêter le run.
  </Tab>

  <Tab title="W&B App">
    1. Accédez au projet dans lequel votre run enregistre ses données.
    2. Sélectionnez le run que vous souhaitez arrêter dans le sélecteur de run.
    3. Choisissez l’onglet **Aperçu**.
    4. Sélectionnez le bouton d’arrêt à côté du champ **State**.

    À côté du champ **State**, l’état du run passe de `running` à `Killed`.
  </Tab>
</Tabs>

Voir [Champs d’état](/fr/models/runs/run-states) pour la liste complète des états possibles d’un run.
