<div id="1-log-a-table">
  ## 1. Journaliser un tableau
</div>

Journalisez un tableau avec W&amp;B. Vous pouvez soit créer un nouveau tableau, soit fournir un dataframe Pandas.

<Tabs>
  <Tab title="Créer un tableau">
    Pour créer et journaliser un nouveau tableau, utilisez :

    * [`wandb.init()`](/fr/models/ref/python/functions/init) : Créer un [run](/fr/models/runs/) pour suivre les résultats.
    * [`wandb.Table()`](/fr/models/ref/python/data-types/table) : Créer un nouvel objet tableau.
      * `columns` : Définir les noms des colonnes.
      * `data` : Définir le contenu de chaque ligne.
    * [`wandb.Run.log()`](/fr/models/ref/python/experiments/run.md/#method-runlog) : Journaliser le tableau pour l’enregistrer dans W&amp;B.

    Voici un exemple :

    ```python
    import wandb

    with wandb.init(project="table-test") as run:
        # Créez et journalisez un nouveau tableau.
        my_table = wandb.Table(columns=["a", "b"], data=[["a1", "b1"], ["a2", "b2"]])
        run.log({"Table Name": my_table})
    ```
  </Tab>

  <Tab title="Dataframe Pandas">
    Passez un dataframe Pandas à `wandb.Table()` pour créer un nouveau tableau.

    ```python
    import wandb
    import pandas as pd

    df = pd.read_csv("my_data.csv")

    with wandb.init(project="df-table") as run:
        # Créez un nouveau tableau à partir du DataFrame
        # et journalisez-le dans W&B.
      my_table = wandb.Table(dataframe=df)
      run.log({"Table Name": my_table})
    ```

    Pour plus d’informations sur les types de données pris en charge, voir [`wandb.Table`](/fr/models/ref/python/data-types/table) dans le guide de référence de l’API W&amp;B.
  </Tab>
</Tabs>

<div id="2-visualize-tables-in-your-project-workspace">
  ## 2. Visualisez les tableaux dans l’espace de travail du projet
</div>

Consultez le tableau obtenu dans votre espace de travail.

1. Accédez à votre projet dans W&amp;B App.
2. Sélectionnez le nom de votre run dans l’espace de travail du projet. Un panneau distinct est ajouté pour chaque clé de tableau.
