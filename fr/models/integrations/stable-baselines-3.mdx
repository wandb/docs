[Stable Baselines 3](https://github.com/DLR-RM/stable-baselines3) (SB3) est un ensemble d&#39;implémentations fiables d&#39;algorithmes d&#39;apprentissage par renforcement en PyTorch. L&#39;intégration SB3 de W&amp;B :

* Enregistre des métriques telles que les pertes et les retours par épisode.
* Téléverse des vidéos montrant les agents en train de jouer.
* Enregistre le modèle entraîné.
* Journalise les hyperparamètres du modèle.
* Journalise les histogrammes de gradients du modèle.

Consultez un [exemple de run d&#39;entraînement SB3](https://wandb.ai/wandb/sb3/runs/1jyr6z10).

<div id="log-your-sb3-experiments">
  ## Journalisez vos expériences avec SB3
</div>

```python
from wandb.integration.sb3 import WandbCallback

model.learn(..., callback=WandbCallback())
```

<div id="wandbcallback-arguments">
  ## Arguments de WandbCallback
</div>

| Argument             | Utilisation                                                                                                              |
| :------------------- | :----------------------------------------------------------------------------------------------------------------------- |
| `verbose`            | Niveau de verbosité de la sortie de sb3                                                                                  |
| `model_save_path`    | Chemin du dossier où le modèle sera enregistré. La valeur par défaut est `None`, donc le modèle n&#39;est pas enregistré |
| `model_save_freq`    | Fréquence d&#39;enregistrement du modèle                                                                                 |
| `gradient_save_freq` | Fréquence d&#39;enregistrement du gradient. La valeur par défaut est 0, donc les gradients ne sont pas enregistrés       |

<div id="basic-example">
  ## Exemple de base
</div>

L’intégration W&amp;B SB3 utilise les journaux de sortie de TensorBoard pour journaliser vos métriques

```python
import gym
from stable_baselines3 import PPO
from stable_baselines3.common.monitor import Monitor
from stable_baselines3.common.vec_env import DummyVecEnv, VecVideoRecorder
import wandb
from wandb.integration.sb3 import WandbCallback


config = {
    "policy_type": "MlpPolicy",
    "total_timesteps": 25000,
    "env_name": "CartPole-v1",
}
run = wandb.init(
    project="sb3",
    config=config,
    sync_tensorboard=True,  # téléversement automatique des métriques TensorBoard de SB3
    monitor_gym=True,  # téléversement automatique des vidéos des agents en train de jouer
    save_code=True,  # facultatif
)


def make_env():
    env = gym.make(config["env_name"])
    env = Monitor(env)  # enregistre des statistiques telles que les rendements
    return env


env = DummyVecEnv([make_env])
env = VecVideoRecorder(
    env,
    f"videos/{run.id}",
    record_video_trigger=lambda x: x % 2000 == 0,
    video_length=200,
)
model = PPO(config["policy_type"], env, verbose=1, tensorboard_log=f"runs/{run.id}")
model.learn(
    total_timesteps=config["total_timesteps"],
    callback=WandbCallback(
        gradient_save_freq=100,
        model_save_path=f"models/{run.id}",
        verbose=2,
    ),
)
run.finish()
```
