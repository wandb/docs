[Stable Baselines 3](https://github.com/DLR-RM/stable-baselines3) (SB3) est un ensemble d&#39;implémentations fiables d&#39;algorithmes d&#39;apprentissage par renforcement en PyTorch. L&#39;intégration SB3 de W&amp;B :

* Enregistre des métriques telles que les pertes et les retours d&#39;épisode.
* Téléverse des vidéos d&#39;agents en train de jouer.
* Enregistre le modèle entraîné.
* Journalise les hyperparamètres du modèle.
* Journalise les histogrammes de gradients du modèle.

Consultez un [exemple de run d&#39;entraînement SB3](https://wandb.ai/wandb/sb3/runs/1jyr6z10).

<div id="log-your-sb3-experiments">
  ## Journalisez vos expériences SB3
</div>

```python
from wandb.integration.sb3 import WandbCallback

model.learn(..., callback=WandbCallback())
```

<div id="wandbcallback-arguments">
  ## Arguments de WandbCallback
</div>

| Argument             | Utilisation                                                                                                                   |
| :------------------- | :---------------------------------------------------------------------------------------------------------------------------- |
| `verbose`            | Niveau de verbosité de la sortie de sb3                                                                                       |
| `model_save_path`    | Chemin vers le dossier où le modèle sera enregistré. La valeur par défaut est `None`, donc le modèle n&#39;est pas enregistré |
| `model_save_freq`    | Fréquence d&#39;enregistrement du modèle                                                                                      |
| `gradient_save_freq` | Fréquence d&#39;enregistrement des gradients. La valeur par défaut est 0, donc les gradients ne sont pas enregistrés          |
