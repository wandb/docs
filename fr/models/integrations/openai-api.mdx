<div id="install-openai-python-api-library">
  ## Installer la bibliothèque de l’API Python OpenAI
</div>

L’intégration autolog de W&amp;B fonctionne avec OpenAI version 0.28.1 et les versions antérieures.

Pour installer la version 0.28.1 de l’API Python OpenAI, exécutez :

```python
pip install openai==0.28.1
```

<div id="use-the-openai-python-api">
  ## Utilisez l’API Python OpenAI
</div>

<div id="1-import-autolog-and-initialise-it">
  ### 1. Importez autolog et initialisez-le
</div>

Commencez par importer `autolog` de `wandb.integration.openai`, puis initialisez-le.

```python
import os
import openai
from wandb.integration.openai import autolog

autolog({"project": "gpt5"})
```

Vous pouvez également passer à `autolog` un dictionnaire contenant les arguments qu’accepte `wandb.init()`. Cela inclut le nom du projet, le nom de l’équipe, l’entité, etc. Pour en savoir plus sur [`wandb.init()`](/fr/models/ref/python/functions/init), voir le Guide de référence de l’API.

<div id="2-call-the-openai-api">
  ### 2. Appelez l&#39;API OpenAI
</div>

Chaque appel que vous effectuez à l&#39;API OpenAI est désormais automatiquement consigné dans W&amp;B.

```python
os.environ["OPENAI_API_KEY"] = "XXX"

chat_request_kwargs = dict(
    model="gpt-3.5-turbo",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Who won the world series in 2020?"},
        {"role": "assistant", "content": "The Los Angeles Dodgers"},
        {"role": "user", "content": "Where was it played?"},
    ],
)
response = openai.ChatCompletion.create(**chat_request_kwargs)
```

<div id="3-view-your-openai-api-inputs-and-responses">
  ### 3. Consultez vos entrées et réponses de l’API OpenAI
</div>

Cliquez sur le lien W&amp;B [run](/fr/models/runs/) généré par `autolog` à l’**étape 1**. Vous êtes alors redirigé vers l’espace de travail du projet dans W&amp;B App.

Sélectionnez un run que vous avez créé pour afficher le tableau des traces, la chronologie des traces et l’architecture du modèle du LLM OpenAI utilisé.

<div id="turn-off-autolog">
  ## Désactiver l’autolog
</div>

W&amp;B recommande d’appeler `disable()` pour arrêter tous les processus W&amp;B une fois que vous avez fini d’utiliser l’API OpenAI.

```python
autolog.disable()
```

Vos entrées et les complétions générées seront désormais consignées dans W&amp;B, prêtes à être analysées ou partagées avec vos collègues.
