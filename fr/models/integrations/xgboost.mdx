  ## Référence de `WandbCallback`
</div>

<div id="functionality">
  ### Fonctionnalité
</div>

Passer `WandbCallback` à un modèle XGBoost permet de :

* journaliser la configuration du modèle booster dans W&amp;B
* journaliser les métriques d&#39;évaluation collectées par XGBoost, comme rmse, accuracy, etc., dans W&amp;B
* journaliser les métriques d&#39;entraînement collectées par XGBoost (si vous fournissez des données à `eval_set`)
* journaliser le meilleur score et la meilleure itération
* enregistrer et téléverser votre modèle entraîné dans W&amp;B Artifacts (lorsque `log_model = True`)
* journaliser le graphique d&#39;importance des variables lorsque `log_feature_importance=True` (par défaut).
* Capturer la meilleure métrique d&#39;évaluation dans `wandb.Run.summary` lorsque `define_metric=True` (par défaut).

<div id="arguments">
  ### Arguments
</div>

* `log_model`: (boolean) si True, enregistrez et envoyez le modèle vers W&amp;B Artifacts

* `log_feature_importance`: (boolean) si True, journalisez un graphique à barres de l&#39;importance des variables

* `importance_type`: (str) l&#39;une des valeurs suivantes : `{weight, gain, cover, total_gain, total_cover}` pour un modèle arborescent. `weight` pour un modèle linéaire.

* `define_metric`: (boolean) si True (par défaut), enregistrez les performances du modèle à la meilleure étape de l&#39;entraînement, plutôt qu&#39;à la dernière, dans votre `run.summary`.

Vous pouvez consulter le [code source de WandbCallback](https://github.com/wandb/wandb/blob/main/wandb/integration/xgboost/xgboost.py).

Pour plus d&#39;exemples, consultez le [dépôt d&#39;exemples sur GitHub](https://github.com/wandb/examples/tree/master/examples/boosting-algorithms).

<div id="tune-your-hyperparameters-with-sweeps">
  ## Ajustez vos hyperparamètres avec Sweeps
</div>

Pour obtenir les meilleures performances de vos modèles, vous devez ajuster des hyperparamètres, comme la profondeur de l’arbre et le taux d’apprentissage. W&amp;B [Sweeps](/fr/models/sweeps/) est une boîte à outils puissante pour configurer, orchestrer et analyser des campagnes de tests d’hyperparamètres à grande échelle.
