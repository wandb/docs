Cette bibliothèque s&#39;appuie sur la bibliothèque Transformers de Hugging Face. Simple Transformers vous permet d&#39;entraîner et d&#39;évaluer rapidement des modèles Transformer. Trois lignes de code suffisent pour initialiser un modèle, entraîner le modèle et évaluer un modèle. Elle prend en charge la classification de séquences, la classification de tokens (NER), les questions-réponses, le Fine-tuning de modèles de langage, l&#39;entraînement de modèles de langage, la génération de langage, le modèle T5, les tâches Seq2Seq, la classification multimodale et l&#39;IA conversationnelle.

Pour utiliser W&amp;B pour visualiser l&#39;entraînement du modèle, définissez un nom de projet pour W&amp;B dans l&#39;attribut `wandb_project` du dictionnaire `args`. Cela consigne toutes les valeurs d&#39;hyperparamètres, les pertes d&#39;entraînement et les métriques d&#39;évaluation dans le projet indiqué.

```python
model = ClassificationModel('roberta', 'roberta-base', args={'wandb_project': 'project-name'})
```

Tout argument supplémentaire à transmettre à `wandb.init` peut être passé via `wandb_kwargs`.


<div id="structure">
  ## Structure
</div>

La bibliothèque est conçue avec une classe distincte pour chaque tâche de NLP. Les classes qui offrent des fonctionnalités similaires sont regroupées.

* `simpletransformers.classification` - Comprend tous les modèles de Classification.
  * `ClassificationModel`
  * `MultiLabelClassificationModel`
* `simpletransformers.ner` - Comprend tous les modèles de reconnaissance d&#39;entités nommées.
  * `NERModel`
* `simpletransformers.question_answering` - Comprend tous les modèles de Questions-réponses.
  * `QuestionAnsweringModel`

Voici quelques exemples de base

<div id="multilabel-classification">
  ## Classification multi-étiquettes
</div>

```text
  model = MultiLabelClassificationModel("distilbert","distilbert-base-uncased",num_labels=6,
    args={"reprocess_input_data": True, "overwrite_output_dir": True, "num_train_epochs":epochs,'learning_rate':learning_rate,
                'wandb_project': "simpletransformers"},
  )
   # Entraîner le modèle
  model.train_model(train_df)

  # Évaluer le modèle
  result, model_outputs, wrong_predictions = model.eval_model(eval_df)
```


<div id="question-answering">
  ## Questions-réponses
</div>

```text
  train_args = {
    'learning_rate': wandb.config.learning_rate,
    'num_train_epochs': 2,
    'max_seq_length': 128,
    'doc_stride': 64,
    'overwrite_output_dir': True,
    'reprocess_input_data': False,
    'train_batch_size': 2,
    'fp16': False,
    'wandb_project': "simpletransformers"
}

model = QuestionAnsweringModel('distilbert', 'distilbert-base-cased', args=train_args)
model.train_model(train_data)
```
