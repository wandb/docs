<div id="introduction">
  ## Introduction
</div>

Le fine-tuning de modèles GPT-3.5 ou GPT-4 sur Microsoft Azure avec W&amp;B vous permet de suivre, d’analyser et d’améliorer les performances des modèles en capturant automatiquement les métriques et en facilitant une évaluation systématique grâce aux outils de suivi des expériences et d’évaluation de W&amp;B.

  <img src="/images/integrations/aoai_ft_plot.png" alt="Métriques de fine-tuning d’Azure OpenAI" />
</Frame>

<div id="prerequisites">
  ## Prérequis
</div>

* Configurez le service Azure OpenAI en suivant la [documentation officielle d’Azure](https://wandb.me/aoai-wb-int).
* Configurez un compte W&amp;B à l’aide d’une clé API.

<div id="workflow-overview">
  ## Aperçu du flux de travail
</div>

<div id="1-fine-tuning-setup">
  ### 1. Configuration du fine-tuning
</div>

* Préparez les données d&#39;entraînement conformément aux exigences d&#39;Azure OpenAI.
* Configurez le job de fine-tuning dans Azure OpenAI.
* W&amp;B suit automatiquement le processus de fine-tuning et enregistre les métriques et les hyperparamètres.

<div id="2-experiment-tracking">
  ### 2. Suivi des expériences
</div>

Lors du fine-tuning, W&amp;B capture :

* Les métriques d&#39;entraînement et de validation
* Les hyperparamètres du modèle
* L’utilisation des ressources
* Les Artifacts d&#39;entraînement

<div id="3-model-evaluation">
  ### 3. Évaluation du modèle
</div>

Après le Fine-tuning, utilisez [W&amp;B Weave](https://weave-docs.wandb.ai) pour :

* Évaluer les résultats du modèle à l’aide de jeux de données de référence
* Comparer les performances entre différentes itérations de Fine-tuning
* Analyser le comportement du modèle sur des cas de test spécifiques
* Prendre des décisions fondées sur les données pour choisir le modèle
