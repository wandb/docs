W&amp;B s’intègre à [Databricks](https://www.databricks.com/) en adaptant l’expérience des notebooks Jupyter de W&amp;B à l’environnement Databricks.

<div id="configure-databricks">
  ## Configurer Databricks
</div>

1. Installez wandb sur le cluster

   Accédez à la configuration de votre cluster, sélectionnez votre cluster, puis cliquez sur **Libraries**. Cliquez sur **Install New**, choisissez **PyPI**, puis ajoutez le package `wandb`.

2. Configurez l&#39;authentification

   Pour authentifier votre compte W&amp;B, vous pouvez ajouter un secret Databricks que vos notebooks pourront consulter.

<div id="examples">
  ## Exemples
</div>

<div id="simple-example">
  ### Exemple simple
</div>

```python
import os
import wandb

api_key = dbutils.secrets.get("wandb", "api_key")
wandb.login(key=api_key)

with wandb.init() as run:
    run.log({"foo": 1})
```
