[PaddleOCR](https://github.com/PaddlePaddle/PaddleOCR) vise à créer des outils d’OCR multilingues, performants, de pointe et pratiques, qui aident les utilisateurs à entraîner de meilleurs modèles et à les mettre en œuvre avec PaddlePaddle. PaddleOCR prend en charge un large éventail d’algorithmes de pointe liés à l’OCR et a développé des solutions industrielles. PaddleOCR propose désormais une intégration W&amp;B pour journaliser les métriques d’entraînement et d’évaluation, ainsi que les points de contrôle du modèle avec les métadonnées correspondantes.

<div id="sign-up-and-create-an-api-key">
  ## Inscrivez-vous et générez une clé API
</div>

Une clé API permet d’authentifier votre machine auprès de W&amp;B. Vous pouvez générer une clé API depuis votre profil utilisateur.

<ApiKeyCreateStreamlined />

1. Cliquez sur l’icône de votre profil dans le coin supérieur droit.
2. Sélectionnez **Paramètres utilisateur**, puis faites défiler jusqu’à la section **Clés API**.

<div id="add-wandb-to-your-configyml-file">
  ## Ajoutez wandb à votre fichier `config.yml`
</div>

PaddleOCR exige que les variables de configuration soient fournies dans un fichier yaml. L’ajout de l’extrait suivant à la fin du fichier de configuration yaml journalisera automatiquement toutes les métriques d’entraînement et de validation dans un tableau de bord W&amp;B, ainsi que les points de contrôle du modèle :

```python
Global:
    use_wandb: True
```

Vous pouvez également ajouter sous l’en-tête `wandb` du fichier yaml tous les arguments supplémentaires facultatifs que vous souhaitez passer à [`wandb.init()`](/fr/models/ref/python/functions/init) :

```
wandb:  
    project: CoolOCR  # (facultatif) il s'agit du nom du projet wandb 
    entity: my_team   # (facultatif) si vous utilisez une équipe wandb, vous pouvez indiquer le nom de l'équipe ici
    name: MyOCRModel  # (facultatif) il s'agit du nom du run wandb
```

<div id="pass-the-configyml-file-to-trainpy">
  ## Passez le fichier `config.yml` à `train.py`
</div>

Le fichier YAML est ensuite passé en argument au [script d&#39;entraînement](https://github.com/PaddlePaddle/PaddleOCR/blob/release/2.5/tools/train.py) disponible dans le dépôt PaddleOCR.

```bash
python tools/train.py -c config.yml
```

Une fois que vous avez exécuté votre fichier `train.py` avec W&amp;B activé, un lien est généré pour vous rediriger vers votre tableau de bord W&amp;B :
