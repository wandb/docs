[Prodigy](https://prodi.gy/) est un outil d&#39;annotation permettant de créer des données d&#39;entraînement et d&#39;évaluation pour les modèles d&#39;apprentissage automatique, l&#39;analyse des erreurs, l&#39;inspection et le nettoyage des données. Les [tableau W&amp;B](/fr/models/tables/tables-walkthrough/) vous permettent de journaliser, visualiser, analyser et partager des jeux de données (et plus encore !) dans W&amp;B.

L&#39;[intégration W&amp;B avec Prodigy](https://github.com/wandb/wandb/blob/master/wandb/integration/prodigy/prodigy.py) ajoute une fonctionnalité simple et facile à utiliser pour téléverser votre jeu de données annoté avec Prodigy directement dans W&amp;B afin de l&#39;utiliser avec Tables.

Exécutez quelques lignes de code, comme celles-ci :

```python
import wandb
from wandb.integration.prodigy import upload_dataset

with wandb.init(project="prodigy"):
    upload_dataset("news_headlines_ner")
```

<div id="quickstart">
  ## Démarrage rapide
</div>

Utilisez `wandb.integration.prodigy.upload_dataset` pour téléverser votre jeu de données Prodigy annoté directement depuis la base de données Prodigy locale vers W&amp;B dans notre format [Tableau](/fr/models/ref/python/data-types/table). Pour plus d’informations sur Prodigy, y compris l’installation et la configuration, veuillez vous référer à la [documentation Prodigy](https://prodi.gy/docs/).

W&amp;B essaiera automatiquement de convertir les images et les champs d’entités nommées en [`wandb.Image`](/fr/models/ref/python/data-types/image) et [`wandb.Html`](/fr/models/ref/python/data-types/html)respectivement. Des colonnes supplémentaires peuvent être ajoutées au tableau résultant pour inclure ces visualisations.
