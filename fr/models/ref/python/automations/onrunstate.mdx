L’état d’un run est modifié.

<div id="args">
  ## Arguments
</div>

* `event_type` :
* `scope` : Portée de l’événement : doit être un projet ou une entité d’équipe/d’organisation.
* `filter` : Condition(s) sur l’état du run à remplir pour que cet événement se déclenche.

<div id="examples">
  ## Exemples
</div>

Définissez un événement qui se déclenche pour tout run du projet &quot;my-project&quot; lorsque
son état passe à &quot;finished&quot; (c.-à-d. réussi) ou &quot;failed&quot; :

```python
from wandb import Api
from wandb.automations import OnRunState

api = Api()
project = api.project(name="my-project")

event = OnRunState(
    scope=project,
    filter=RunEvent.state.in_(["finished", "failed"]),
)
```
