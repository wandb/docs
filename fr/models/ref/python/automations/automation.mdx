## <kbd>class</kbd> `Automation`

Instance locale d’une automatisation W&amp;B enregistrée pouvant être modifiée.

### <kbd>méthode</kbd> `Automation.__init__`

```python
__init__(
    typename__: 'Literal['Trigger']' = 'Trigger',
    id: 'str',
    created_at: 'datetime',
    updated_at: 'datetime | None' = None,
    name: 'str',
    description: 'str | None',
    enabled: 'bool',
    scope: '_ArtifactSequenceScope | _ArtifactPortfolioScope | ProjectScope',
    event: 'SavedEvent',
    action: 'SavedLaunchJobAction | SavedNotificationAction | SavedWebhookAction | SavedNoOpAction'
) → None
```

**Arguments :**

* `typename__` (Literal[&#39;Trigger&#39;]):
* `id` (str):
* `created_at` (datetime): La date et l’heure de création de cette automatisation.
* `updated_at` (Optional[datetime]): La date et l’heure de la dernière mise à jour de cette automatisation, le cas échéant.
* `name` (str): Le nom de cette automatisation.
* `description` (Optional[str]): Une description facultative de cette automatisation.
* `enabled` (bool): Indique si cette automatisation est activée. Seules les automatisations activées se déclenchent.
* `scope` (Union[&#95;ArtifactSequenceScope, &#95;ArtifactPortfolioScope, ProjectScope]): La portée dans laquelle l’événement déclencheur doit se produire.
* `event` (SavedEvent): L’événement qui déclenchera cette automatisation.
* `action` (Union[SavedLaunchJobAction, SavedNotificationAction, SavedWebhookAction, SavedNoOpAction]): L’action exécutée lorsque cette automatisation est déclenchée.

**Retourne :**
Un objet `Automation`.
