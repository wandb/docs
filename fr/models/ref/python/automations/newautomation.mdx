* `enabled` : Indique si cette automatisation est activée. Seules les automatisations activées se déclencheront.
* `event` : L&#39;événement qui déclenchera cette automatisation.
* `action` : L&#39;action qui sera exécutée lorsque cette automatisation sera déclenchée.
