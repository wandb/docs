## <kbd>class</kbd> `OnLinkArtifact`

Un nouvel artifact est associé à une collection.

Exemples :
Définissez un événement qui se déclenche lorsqu’un artifact est associé à la
collection &quot;my-collection&quot; avec l’alias &quot;prod&quot; :

```python
from wandb import Api
from wandb.automations import OnLinkArtifact, ArtifactEvent

api = Api()
collection = api.artifact_collection(name="my-collection", type_name="model")

event = OnLinkArtifact(
    scope=collection,
    filter=ArtifactEvent.alias.eq("prod"),
)
```
