## <kbd>class</kbd> `OnRunMetric`

Une métrique de run satisfait une condition définie par l&#39;utilisateur.

Exemples :
Définir un événement qui se déclenche pour tout run dans le projet &quot;my-project&quot; lorsque
la moyenne des 5 dernières valeurs de la métrique &quot;my-metric&quot; dépasse 123,45 :

```python
from wandb import Api
from wandb.automations import OnRunMetric, RunEvent

api = Api()
project = api.project(name="my-project")

event = OnRunMetric(
    scope=project,
    filter=RunEvent.metric("my-metric").avg(5).gt(123.45),
)
```
