## <kbd>class</kbd> `Member`

### <kbd>méthode</kbd> `Member.__init__`

```python
__init__(service_api: 'ServiceApi', team: 'str', attrs: 'Mapping[str, Any]')
```

**Arguments :**

* `service_api` :  l’instance de l’API de service à utiliser pour interroger W&amp;B.
* `team` (str) :  le nom de l’équipe à laquelle appartient ce membre
