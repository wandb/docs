## <kbd>class</kbd> `Reports`

Reports est un itérateur `lazy` d’objets `BetaReport`.

### <kbd>méthode</kbd> `Reports.__init__`

```python
__init__(
    client: 'RetryingClient',
    project: 'Project',
    name: 'str | None' = None,
    entity: 'str | None' = None,
    per_page: 'int' = 50
)
```

**Arguments :**

* `client` (`wandb.apis.internal.Api`) :  L&#39;instance du client API à utiliser.
* `project` (`wandb.sdk.internal.Project`) :  Le projet dont il faut récupérer les rapports.
* `name` (str, optional) :  Le nom du rapport à utiliser pour le filtrage. Si `None`, récupère tous les rapports.
* `entity` (str, optional) :  Le nom de l&#39;entité du projet. Par défaut, utilise l&#39;entité du projet.
* `per_page` (int) :  Nombre de rapports à récupérer par page (la valeur par défaut est 50).
