## <kbd>class</kbd> `ProjectArtifactCollections`

Collections d&#39;artefacts dans un projet.

**Arguments :**

* `client`:  L&#39;instance cliente à utiliser pour interroger W&amp;B.
* `entity`:  L&#39;entité (utilisateur ou équipe) propriétaire du projet.
* `project`:  Le nom du projet à interroger pour les collections d&#39;artefacts.
* `filters`:  Mappage facultatif des filtres à appliquer à la requête.
* `order`:  Chaîne facultative permettant de spécifier l&#39;ordre des résultats. Si vous faites précéder order de +, l&#39;ordre est croissant (par défaut). Si vous faites précéder order de -, l&#39;ordre est décroissant.
* `per_page`:  Le nombre de collections d&#39;artefacts à récupérer par page. La valeur par défaut est 50.
