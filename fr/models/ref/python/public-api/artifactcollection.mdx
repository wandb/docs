## <kbd>class</kbd> `ArtifactCollection`

Une collection d&#39;artifacts représentant un groupe d&#39;artifacts associés.

**Arguments :**

* `client`:  Instance cliente à utiliser pour interroger W&amp;B.
* `entity`:  L&#39;entité (utilisateur ou équipe) propriétaire du projet.
* `project`:  Le nom du projet dans lequel rechercher des collections d&#39;artifacts.
* `name`:  Le nom de la collection d&#39;artifacts.
* `type`:  Le type de la collection d&#39;artifacts (par ex., &quot;jeu de données&quot;, &quot;modèle&quot;).
* `organization`:  Nom de l&#39;organisation, facultatif, le cas échéant.
* `attrs`:  Mappage facultatif d&#39;attributs pour initialiser la collection d&#39;artifacts. Si aucun n&#39;est fourni, l&#39;objet chargera ses attributs depuis W&amp;B lors de l&#39;initialisation.
