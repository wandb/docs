## <kbd>classe</kbd> wandb.apis.public.ArtifactCollection

Une collection d&#39;artifacts représentant un groupe d&#39;artifacts associés.

  ## Propriétés
</div>

### <kbd>propriété</kbd> aliases

Les alias de toutes les versions d&#39;artifact contenues dans cette collection.

### <kbd>propriété</kbd> created_at

La date de création de la collection d&#39;artifacts.

### <kbd>propriété</kbd> description

Description de la collection d&#39;artifacts.

### <kbd>propriété</kbd> entité

L’entité (utilisateur ou équipe) propriétaire du projet.

### <kbd>propriété</kbd> id

L’identifiant unique de la collection d’artifacts.

### <kbd>propriété</kbd> nom

Le nom de la collection d&#39;artifacts.

### <kbd>propriété</kbd> project

Le projet qui contient la collection d&#39;artifacts.

### <kbd>propriété</kbd> tags

Les tags de la collection d&#39;artifacts.

### <kbd>propriété</kbd> type

Renvoie le type de la collection d&#39;artifacts.

### <kbd>propriété</kbd> updated_at

La date de la dernière mise à jour de la collection d’artifacts.
