## <kbd>class</kbd> `ArtifactCollection`

Une collection d&#39;artefacts représentant un groupe d&#39;artefacts associés.

**Arguments :**

* `client`:  Instance client à utiliser pour exécuter des requêtes vers W&amp;B.
* `entity`:  L&#39;entité (utilisateur ou équipe) propriétaire du projet.
* `project`:  Le nom du projet dans lequel rechercher des collections d&#39;artefacts.
* `name`:  Le nom de la collection d&#39;artefacts.
* `type`:  Le type de la collection d&#39;artefacts (par ex. : &quot;jeu de données&quot;, &quot;modèle&quot;).
* `organization`:  Nom de l&#39;organisation, facultatif le cas échéant.
* `attrs`:  Mappage facultatif d&#39;attributs pour initialiser la collection d&#39;artefacts.  S&#39;il n&#39;est pas fourni, l&#39;objet chargera ses attributs depuis W&amp;B lors de l&#39;initialisation. 

### <kbd>propriété</kbd> ArtifactCollection.aliases
