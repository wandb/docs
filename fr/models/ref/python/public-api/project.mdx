### <kbd>méthode</kbd> `Project.collections`

```python
collections(
    filters: 'Mapping[str, Any] | None' = None,
    order: 'str | None' = None,
    per_page: 'int' = 50
) → public.ProjectArtifactCollections
```

Renvoie toutes les collections d’artifacts associées à ce projet.

**Arguments :**

* `filters`:  Dictionnaire facultatif de filtres à appliquer à la requête.
* `order`:  Chaîne facultative permettant de spécifier l’ordre des résultats. Si vous faites précéder `order` d’un +, l’ordre est croissant (par défaut). Si vous faites précéder `order` d’un -, l’ordre est décroissant.
* `per_page`:  Le nombre de collections d’artifacts à récupérer par page. La valeur par défaut est 50.
