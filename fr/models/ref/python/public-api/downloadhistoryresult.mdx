## <kbd>class</kbd> `DownloadHistoryResult`

Résultat du téléchargement des exports de l’historique d’un run.

**Attributs :**

* `paths` :  Chemins des fichiers d’historique téléchargés.
* `errors` :  Un dictionnaire qui associe les chemins de fichiers aux messages d’erreur des fichiers dont le téléchargement a échoué. `None` si tous les téléchargements ont réussi.
* `contains_live_data` :  Indique si le run contient des données en direct qui n’ont pas encore été exportées vers des fichiers parquet.
