## <kbd>class</kbd> `RunArtifacts`

Une collection itérable d&#39;Artifacts associée à un run donné. 

### <kbd>propriété</kbd> RunArtifacts.cursor

Un curseur opaque qui indique le début de la page suivante à récupérer.

Cette valeur peut être enregistrée et transmise via `start=` à une requête paginée ultérieure afin de reprendre l’itération là où cet objet de pagination s’est arrêté.

**Retourne :**

* `str | None` : La valeur de la propriété du curseur.
