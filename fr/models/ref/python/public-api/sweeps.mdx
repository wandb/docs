## <kbd>class</kbd> `Sweeps`

Un itérateur lazy sur une collection d’objets `Sweep`.

**Exemples :**

```python
from wandb.apis.public import Api

sweeps = Api().project(name="project_name", entity="entity").sweeps()

# Itérer sur les Sweeps et afficher les détails
for sweep in sweeps:
     print(f"Sweep name: {sweep.name}")
     print(f"Sweep ID: {sweep.id}")
     print(f"Sweep URL: {sweep.url}")
     print("----------")
```


### <kbd>méthode</kbd> `Sweeps.__init__`

```python
__init__(
    client: 'RetryingClient',
    entity: 'str',
    project: 'str',
    per_page: 'int' = 50
) → Sweeps
```

Une collection itérable d’objets `Sweep`.

**Arguments :**

* `client` :  Le client API utilisé pour interroger W&amp;B.
* `entity` :  L’entité à laquelle appartiennent les Sweeps.
* `project` :  Le projet qui contient les Sweeps.
* `per_page` :  Le nombre de Sweeps à récupérer par requête API.
