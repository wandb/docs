### <kbd>méthode</kbd> ArtifactType.collections()

```python
self,
filters: 'Mapping[str, Any] | None' = None,
order: 'str | None' = None,
per_page: 'int' = 50,
start: 'str | None' = None
```

Obtenir toutes les collections d&#39;artifacts associées à ce type d&#39;artifact.

<div id="arguments">
  ##### Arguments
</div>

* `filters`: Mappage facultatif des filtres à appliquer à la requête.
* `order`: Chaîne facultative permettant de spécifier l’ordre des résultats. S’il est préfixé par &#39;+&#39;, le tri est croissant (par défaut). S’il est préfixé par &#39;-&#39;, le tri est décroissant. Par défaut, l’ordre utilisé est l’ID de la collection, par ordre décroissant.
* `per_page`: Le nombre de collections d&#39;artifacts à récupérer par page. La valeur par défaut est 50.
* `start`: Curseur de pagination pour reprendre une requête antérieure, capturé à partir de l’attribut `.cursor` d’un paginateur précédent.
