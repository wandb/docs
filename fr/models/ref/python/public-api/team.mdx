## <kbd>classe</kbd> wandb.apis.public.Team

```python
(service_api: 'ServiceApi',
name: 'str',
attrs: 'Mapping[str, Any] | None' = None)
```

Classe représentant une équipe W&amp;B.

Cette classe fournit des méthodes pour gérer les équipes W&amp;B, notamment pour créer des équipes,
inviter des membres et gérer des comptes de service. Elle hérite de Attrs
pour gérer les attributs de l’équipe.

<div id="args">
  ## Arguments
</div>

* `service_api` : L’instance d’API de service à utiliser pour interroger W&amp;B.
* `name` : Le nom de la équipe
* `attrs` : Dictionnaire facultatif des attributs de la équipe

<div id="examples">
  ## Exemples
</div>

Recherchez une équipe et invitez un membre.

```python
import wandb

api = wandb.Api()
team = api.team("my-team")
team.invite("user@example.com")
```

Créez une équipe et ajoutez un compte de service.

```python
import wandb

api = wandb.Api()
team = api.create_team("my-team")
team.create_service_account("CI service account")
```
