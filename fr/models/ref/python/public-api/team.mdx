## <kbd>class</kbd> `Team`

Classe représentant une équipe W&amp;B.

Cette classe fournit des méthodes pour gérer les équipes W&amp;B, notamment pour créer des équipes, inviter des membres et gérer des comptes de service. Elle hérite de Attrs pour gérer les attributs de l’équipe.

### <kbd>méthode</kbd> `Team.__init__`

```python
__init__(
    client: 'RetryingClient',
    name: 'str',
    attrs: 'Mapping[str, Any] | None' = None
)
```

**Arguments :**

* `client` (`wandb.apis.public.Api`) :  L’instance d’API à utiliser
* `name` (str) :  Le nom de la équipe
* `attrs` (dict) :  Dictionnaire facultatif des attributs de la équipe

**Remarque :**

> La gestion des équipes requiert les autorisations appropriées.

### <kbd>méthode</kbd> `Team.invite`

```python
invite(username_or_email: 'str', admin: 'bool' = False) → bool
```

Invitez un utilisateur dans une équipe.

**Arguments :**

* `username_or_email`:  (str) Le nom d’utilisateur ou l’adresse e-mail de l’utilisateur que vous souhaitez inviter.
* `admin`:  (bool) Indique si cet utilisateur doit devenir administrateur de la équipe.  La valeur par défaut est `False`.

**Retourne :**
`True` en cas de succès, `False` si l’utilisateur avait déjà été invité ou n’existait pas.
