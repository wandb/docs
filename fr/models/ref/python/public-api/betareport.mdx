BetaReport est une classe associée aux rapports créés dans W&amp;B.

Elle fournit un accès aux attributs du rapport (nom, description, utilisateur, spec,
horodatages), ainsi qu&#39;à des méthodes permettant de récupérer les runs et les sections associés et d&#39;effectuer le rendu du rapport en HTML.

<div id="attributes">
  ## Attributs
</div>

* `id` (`string`) : ID unique du rapport.
* `display_name` (`string`) : Nom d’affichage lisible du rapport.
* `name` (`string`) : Nom du rapport. Utilisez `display_name` pour un nom plus convivial.
* `description` (`string`) : Description du rapport.
* `user` (`User`) : Dictionnaire contenant les informations de l’utilisateur (nom d’utilisateur, e-mail) ayant créé le rapport.
* `spec` (`dict`) : Spécification du rapport.
* `url` (`string`) : URL du rapport.
* `updated_at` (`string`) : Horodatage de la dernière mise à jour.
* `created_at` (`string`) : Horodatage de la création du rapport.
