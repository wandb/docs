## <kbd>class</kbd> `BetaReport`

BetaReport est une classe associée aux rapports créés dans W&amp;B.

Permet d&#39;accéder aux attributs du rapport (`name`, `description`, `user`, `spec`, horodatages) ainsi qu&#39;à des méthodes pour récupérer les runs et sections associés, et pour générer le rendu du rapport en HTML.

**Attributs :**

* `id` (string):  ID unique du rapport.
* `display_name` (string):  Display nom du rapport, lisible par l&#39;humain.
* `name` (string):  Nom du rapport. Utilisez `display_name` pour un nom plus convivial.
* `description` (string):  Description du rapport.
* `user` (User):  Dictionnaire contenant les informations de l&#39;utilisateur (nom d&#39;utilisateur, e-mail) ayant créé le rapport.
* `spec` (dict):  La spécification du rapport.
* `url` (string):  L&#39;URL du rapport.
* `updated_at` (string):  Horodatage de la dernière mise à jour.
* `created_at` (string):  Horodatage de création du rapport.
