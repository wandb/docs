## <kbd>classe</kbd> `BetaReport`

BetaReport est une classe associée aux rapports créés dans W&amp;B.

Elle fournit un accès aux attributs du rapport (nom, description, utilisateur, spec, horodatages), ainsi qu&#39;à des méthodes permettant de récupérer les runs et les sections associés et d&#39;effectuer le rendu du rapport en HTML.

**Attributs :**

* `id` (string) :  Identifiant unique du rapport.
* `display_name` (string) :  Nom d&#39;affichage du rapport, lisible par l&#39;utilisateur.
* `name` (string) :  Nom du rapport. Utilisez `display_name` pour un nom plus convivial.
* `description` (string) :  Description du rapport.
* `user` (User) :  Dictionnaire contenant les informations de l&#39;utilisateur (nom d&#39;utilisateur, e-mail) ayant créé le rapport.
* `spec` (dict) :  Spécification du rapport.
* `url` (string) :  URL du rapport.
* `updated_at` (string) :  Timestamp de la dernière mise à jour.
* `created_at` (string) :  Timestamp de création du rapport.
