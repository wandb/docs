## <kbd>class</kbd> `Object3D`

Classe W&amp;B pour les nuages de points 3D. 

### <kbd>méthode</kbd> `Object3D.__init__`

```python
__init__(
    data_or_path: 'np.ndarray | str | pathlib.Path | TextIO | dict',
    caption: 'str | None' = None,
    **kwargs: 'str | FileFormat3D | None'
) → None
```

Crée un objet W&amp;B Object3D.

**Arguments :**

* `data_or_path` :  Object3D peut être initialisé à partir d’un fichier ou d’un tableau NumPy.
* `caption` :  Légende associée à l’objet pour l’affichage.

**Exemples :**
La forme du tableau NumPy doit être l’une des suivantes

```text
[[x y z],       ...] nx3
[[x y z c],     ...] nx4 where c is a category with supported range [1, 14]
[[x y z r g b], ...] nx6 where is rgb is color
```
