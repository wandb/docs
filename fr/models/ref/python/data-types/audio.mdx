## <kbd>class</kbd> `Audio`

Classe W&amp;B pour les extraits audio. 

### <kbd>méthode</kbd> `Audio.__init__`

```python
__init__(
    data_or_path: 'str | pathlib.Path | list | np.ndarray',
    sample_rate: 'int | None' = None,
    caption: 'str | None' = None
)
```

Accepte un chemin vers un fichier audio ou un tableau NumPy contenant des données audio.

**Arguments :**

* `data_or_path`:  Chemin vers un fichier audio ou tableau NumPy contenant des données audio.
* `sample_rate`:  Taux d’échantillonnage, requis lors du passage d’un tableau NumPy brut de données audio.
* `caption`:  Légende à afficher avec l’audio.
