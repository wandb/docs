## <kbd>classe</kbd> wandb.Audio

```python
(data_or_path: 'str | pathlib.Path | list | np.ndarray',
sample_rate: 'int | None' = None,
caption: 'str | None' = None)
```

Classe W&amp;B pour les clips audio.

<div id="args">
  ## Arguments
</div>

* `data_or_path`: Chemin vers un fichier audio ou tableau NumPy de données audio.
* `sample_rate`: Taux d’échantillonnage, requis lors du passage d’un tableau NumPy brut de données audio.
* `caption`: Légende à afficher avec l’audio.
