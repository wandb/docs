## <kbd>classe</kbd> wandb.Html

```python
data: 'str | pathlib.Path | TextIO',
inject: 'bool' = True,
data_is_not_path: 'bool' = False
```

Classe W&amp;B permettant de journaliser du contenu HTML dans W&amp;B.

<div id="args">
  ## Arguments
</div>

* `data`: Une chaîne représentant le chemin d’un fichier avec l’extension &quot;.html&quot;, ou une chaîne ou un objet IO contenant du HTML littéral.
* `inject`: Ajoute une feuille de style à l’objet HTML. Si cette option est définie sur False, le HTML est transmis tel quel.
* `data_is_not_path`: Si cette option est définie sur False, les données sont traitées comme le chemin d’un fichier.
