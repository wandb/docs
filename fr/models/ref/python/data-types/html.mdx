## <kbd>class</kbd> `Html`

Classe W&amp;B permettant d’enregistrer du contenu HTML dans W&amp;B. 

### <kbd>méthode</kbd> `Html.__init__`

```python
__init__(
    data: 'str | pathlib.Path | TextIO',
    inject: 'bool' = True,
    data_is_not_path: 'bool' = False
) → None
```

Crée un objet HTML W&amp;B.

**Arguments :**
data:  Chaîne correspondant soit au chemin d’un fichier avec l’extension &quot;.html&quot;, soit à une chaîne ou un objet IO contenant du HTML littéral.

* `inject`:  Ajoute une feuille de style à l’objet HTML. Si ce paramètre est défini sur False, le HTML est transmis tel quel.
* `data_is_not_path`:  Si ce paramètre est défini sur False, les données sont traitées comme le chemin d’un fichier.

**Exemples :**
Vous pouvez l’initialiser en fournissant le chemin d’un fichier :

```python
with wandb.init() as run:
    run.log({"html": wandb.Html("./index.html")})
```

Vous pouvez également l’initialiser en fournissant du HTML littéral, sous forme de chaîne ou d’objet IO :

```python
with wandb.init() as run:
    run.log({"html": wandb.Html("<h1>Hello, world!</h1>")})
```
