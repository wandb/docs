## <kbd>fonction</kbd> wandb.plot.histogram

```python
table: 'wandb.Table',
value: 'str',
title: 'str' = '',
split_table: 'bool' = False
```

Construit un histogramme à partir d&#39;un tableau W&amp;B.

<div id="args">
  ## Arguments
</div>

* `table`: Le tableau W&amp;B contenant les données de l&#39;histogramme.
* `value`: Le libellé de l&#39;axe des classes (x-axis).
* `title`: Le titre de l&#39;histogramme.
* `split_table`: Indique si le tableau doit être placé dans une section distincte de l&#39;interface W&amp;B. Si `True`, le tableau sera affiché dans une section nommée &quot;Custom Chart Tables&quot;. La valeur par défaut est `False`.

<div id="returns">
  ## Retourne
</div>

`CustomChart`: Un objet de graphique personnalisé qui peut être enregistré dans W&amp;B. Pour enregistrer le graphique, transmettez-le à `wandb.log()`.

<div id="examples">
  ## Exemples
</div>

```python
import math
import random
import wandb

# Générer des données aléatoires
data = [[i, random.random() + math.sin(i / 10)] for i in range(100)]

# Créer un tableau W&B
table = wandb.Table(
    data=data,
    columns=["step", "height"],
)

# Créer un graphique histogramme
histogram = wandb.plot.histogram(
    table,
    value="height",
    title="My Histogram",
)

# Enregistrer le graphique histogramme dans W&B
with wandb.init(...) as run:
    run.log({"histogram-plot1": histogram})
```
