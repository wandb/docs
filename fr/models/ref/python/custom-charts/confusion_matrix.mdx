### <kbd>fonction</kbd> `confusion_matrix`

```python
confusion_matrix(
    probs: 'Sequence[Sequence[float]] | None' = None,
    y_true: 'Sequence[T] | None' = None,
    preds: 'Sequence[T] | None' = None,
    class_names: 'Sequence[str] | None' = None,
    title: 'str' = 'Confusion Matrix Curve',
    split_table: 'bool' = False
) → CustomChart
```

Construit une matrice de confusion à partir d&#39;une séquence de probabilités ou de prédictions.

**Arguments :**

* `probs` :  Une séquence de probabilités prédites pour chaque classe. La forme de la séquence doit être (N, K), où N est le nombre d&#39;échantillons et K est le nombre de classes. Si fourni, `preds` ne doit pas être fourni.
* `y_true` :  Une séquence d&#39;étiquettes réelles.
* `preds` :  Une séquence d&#39;étiquettes de classes prédites. Si fourni, `probs` ne doit pas être fourni.
* `class_names` :  Séquence de noms de classes. Si non fourni, les noms de classes seront définis comme &quot;Class&#95;1&quot;, &quot;Class&#95;2&quot;, etc.
* `title` :  Titre du graphique de la matrice de confusion.
* `split_table` :  Indique si le tableau doit être divisé en une section séparée dans l&#39;interface utilisateur W&amp;B. Si `True`, le tableau sera affiché dans une section nommée &quot;Custom Chart Tables&quot;. La valeur par défaut est `False`.

**Renvoie :**

* `CustomChart` :  Un objet graphique personnalisé pouvant être enregistré dans W&amp;B. Pour enregistrer le graphique, transmettez-le à `wandb.log()`.

**Exceptions levées :**

* `ValueError` :  Si `probs` et `preds` sont tous deux fournis, ou si le nombre de prédictions et d&#39;étiquettes réelles ne sont pas égaux. Si le nombre de classes prédites uniques dépasse le nombre de noms de classes, ou si le nombre d&#39;étiquettes réelles uniques dépasse le nombre de noms de classes.
* `wandb.Error` :  Si numpy n&#39;est pas installé.

**Exemples :**
Journalisation d&#39;une matrice de confusion avec des probabilités aléatoires pour la classification de la faune sauvage :

```python
import numpy as np
import wandb

# Définir les noms de classes pour la faune sauvage
wildlife_class_names = ["Lion", "Tiger", "Elephant", "Zebra"]

# Générer des étiquettes réelles aléatoires (0 à 3 pour 10 échantillons)
wildlife_y_true = np.random.randint(0, 4, size=10)

# Générer des probabilités aléatoires pour chaque classe (10 échantillons x 4 classes)
wildlife_probs = np.random.rand(10, 4)
wildlife_probs = np.exp(wildlife_probs) / np.sum(
    np.exp(wildlife_probs),
    axis=1,
    keepdims=True,
)

# Initialiser le run W&B et enregistrer la matrice de confusion
with wandb.init(project="wildlife_classification") as run:
    confusion_matrix = wandb.plot.confusion_matrix(
         probs=wildlife_probs,
         y_true=wildlife_y_true,
         class_names=wildlife_class_names,
         title="Wildlife Classification Confusion Matrix",
    )
    run.log({"wildlife_confusion_matrix": confusion_matrix})
```

Dans cet exemple, des probabilités aléatoires sont utilisées pour générer une matrice de confusion.

Journalisation d&#39;une matrice de confusion avec des prédictions de modèle simulées et une précision de 85 % :

```python
import numpy as np
import wandb

# Définir les noms de classes pour la faune sauvage
wildlife_class_names = ["Lion", "Tiger", "Elephant", "Zebra"]

# Simuler les étiquettes réelles pour 200 images d'animaux (distribution déséquilibrée)
wildlife_y_true = np.random.choice(
    [0, 1, 2, 3],
    size=200,
    p=[0.2, 0.3, 0.25, 0.25],
)

# Simuler les prédictions du modèle avec une précision de 85 %
wildlife_preds = [
    y_t
    if np.random.rand() < 0.85
    else np.random.choice([x for x in range(4) if x != y_t])
    for y_t in wildlife_y_true
]

# Initialiser le run W&B et enregistrer la matrice de confusion
with wandb.init(project="wildlife_classification") as run:
    confusion_matrix = wandb.plot.confusion_matrix(
         preds=wildlife_preds,
         y_true=wildlife_y_true,
         class_names=wildlife_class_names,
         title="Simulated Wildlife Classification Confusion Matrix",
    )
    run.log({"wildlife_confusion_matrix": confusion_matrix})
```

Dans cet exemple, les prédictions sont simulées avec une précision de 85 % pour générer une matrice de confusion.
