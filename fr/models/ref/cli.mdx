<div id="usage">
  ## Utilisation
</div>

```bash
wandb [OPTIONS] COMMAND [ARGS]...
```

<div id="options">
  ## Options
</div>

| Option      | Description                                             |
| :---------- | :------------------------------------------------------ |
| `--version` | Afficher la version, puis quitter. (par défaut : False) |
