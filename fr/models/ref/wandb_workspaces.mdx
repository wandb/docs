L’API W&amp;B Reports and Workspaces, accessible via `wandb_workspaces`, vous permet de créer des [Reports](/fr/models/reports/), qui peuvent être publiés sur le web pour partager des résultats, ainsi que de personnaliser un [Workspace](/fr/models/app/features/cascade-settings/) dans lequel ont été réalisés des travaux d’entraînement et de fine-tuning.

</Note>

  ## Installation et configuration
</div>

<div id="sign-up-and-create-an-api-key">
  ### Inscrivez-vous et créez une clé API
</div>

Pour authentifier votre machine auprès de W&amp;B, vous devez d’abord générer une clé API dans les [Paramètres utilisateur](https://wandb.ai/settings).

<div id="install-and-import-packages">
  ### Installer et importer des paquets
</div>

Installez la bibliothèque W&amp;B Rapport and Workspaces.

```python
pip install wandb-workspaces
```

<div id="create-a-report">
  ### Créer un rapport
</div>

Pour créer un rapport, indiquez l’entité de votre team et saisissez un nom pour votre rapport. Remplacez le texte délimité par vos valeurs :

Ensuite, ajoutez des blocs et des panneaux à votre rapport. Par exemple, le code suivant crée un rapport avec une table des matières, un en-tête et un paragraphe :

```python
report.blocks = [
    wr.TableOfContents(),
    wr.H1("Text and images example"),
    wr.P("Lorem ipsum dolor sit amet."),
]
report.save()
```
