L&#39;API W&amp;B Reports and Workspaces, accessible via `wandb_workspaces`, vous permet de créer des [rapports](/fr/models/reports/), qui peuvent être publiés sur le web pour partager vos résultats, ainsi que de personnaliser un [Workspace](/fr/models/app/features/cascade-settings/) dans lequel ont été menés les travaux d&#39;entraînement et de fine-tuning.

<div id="sign-up-and-create-an-api-key">

Pour authentifier votre machine auprès de W&amp;B, vous devez d’abord générer une clé API dans les [Paramètres utilisateur](https://wandb.ai/settings).

<div id="install-and-import-packages">
  ### Installer et importer des packages
</div>

Installez la bibliothèque W&amp;B Report and Workspaces.

```python
pip install wandb-workspaces
```

<div id="create-a-report">
  ### Créer un rapport
</div>

Pour créer un rapport, indiquez l’entité de votre équipe et donnez un nom à votre rapport. Remplacez le texte entre chevrons par vos valeurs :

```python
import wandb_workspaces.reports.v2 as wr 
# Créer
report = wr.Report(
    entity="<team_entity>",
    project="<project_name>",
    title='Quickstart Report',
    description="That was easy!"
)

# Enregistrer le rapport
report.save()
```

Ensuite, ajoutez des blocs et des panneaux à votre rapport. Par exemple, le code suivant crée un rapport avec une table des matières, un en-tête et un paragraphe :

```python
report.blocks = [
    wr.TableOfContents(),
    wr.H1("Text and images example"),
    wr.P("Lorem ipsum dolor sit amet."),
]
report.save()
```

<div id="create-a-workspace">
  ### Créer un Workspace
</div>

Le code suivant montre comment créer un Workspace avec une section contenant trois panneaux : un graphique en courbes, un graphique à barres et un graphique scalaire. Remplacez le texte entre chevrons par vos valeurs :

```python
# Comment importer
import wandb_workspaces.workspaces as ws

# Créer un Workspace
ws.Workspace(
     entity="<team_entity>", # entité propriétaire du Workspace
     project="<project_name>", # projet associé à ce Workspace
     sections=[
         ws.Section(
             name="<Validation Metrics>",
             panels=[
                 wr.LinePlot(x="Step", y=["<val_loss>"]),
                 wr.BarPlot(metrics=["<val_accuracy>"]),
                 wr.ScalarChart(metric="<f1_score>", groupby_aggfunc="<mean>"),
             ],
             is_open=True,
         ),
     ],
)
workspace.save()
```
