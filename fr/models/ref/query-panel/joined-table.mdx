### <a id="joinedtable-file" />`joinedtable-file`

Renvoie le *fichier* d’un *joined-table*

| Argument      | Description       |
| :------------ | :---------------- |
| `joinedTable` | Le *joined-table* |

<div id="return-value">
  #### Valeur de retour
</div>

La *fichier* d’une *joined-table*

### <a id="joinedtable-rows" />`joinedtable-rows`

Renvoie les lignes d’une *joined-table*

| Argument      | Description                                                                                                         |
| :------------ | :------------------------------------------------------------------------------------------------------------------ |
| `joinedTable` | La *joined-table*                                                                                                   |
| `leftOuter`   | Indique s’il faut inclure les lignes du tableau de gauche qui n’ont pas de correspondance dans le tableau de droite |
| `rightOuter`  | Indique s’il faut inclure les lignes du tableau de droite qui n’ont pas de correspondance dans le tableau de gauche |

<div id="return-value">
  #### Valeur de retour
</div>

Les lignes du *joined-table*
