### <a id="joinedtable-file" />`joinedtable-file`

Renvoie le *fichier* d’un *tableau fusionné*

| Argument      | Description           |
| :------------ | :-------------------- |
| `joinedTable` | Le *tableau fusionné* |

<div id="return-value">
  #### Valeur de retour
</div>

Le *fichier* d’un *tableau fusionné*

### <a id="joinedtable-rows" />`joinedtable-rows`

Renvoie les lignes d’une *tableau fusionné*

| Argument      | Description                                                                                                               |
| :------------ | :------------------------------------------------------------------------------------------------------------------------ |
| `joinedTable` | La *tableau fusionné*                                                                                                         |
| `leftOuter`   | Indique s’il faut inclure les lignes du tableau de gauche qui n’ont pas de ligne correspondante dans le tableau de droite |
| `rightOuter`  | Indique s’il faut inclure les lignes du tableau de droite qui n’ont pas de ligne correspondante dans le tableau de gauche |

<div id="return-value">
  #### Valeur de retour
</div>

Les lignes du *tableau fusionné*
