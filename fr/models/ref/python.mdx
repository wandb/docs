Le W&amp;B Python SDK, accessible via `wandb`, vous permet d’entraîner et d’effectuer le fine-tuning de modèles, ainsi que de gérer des modèles de l’expérimentation à la Production.

> Après avoir effectué vos opérations d’entraînement et de fine-tuning avec ce SDK, vous pouvez utiliser [l’API publique](/fr/models/ref/python/public-api) pour interroger et analyser les données qui ont été enregistrées, ainsi que [l’API Reports and Workspaces](/fr/models/ref/wandb_workspaces) pour générer un [rapport](/fr/models/reports/) publiable sur le Web et résumant votre travail.

Pour authentifier votre machine auprès de W&amp;B, vous devez d’abord générer une clé API dans [Paramètres utilisateur](https://wandb.ai/settings).

<div id="install-and-import-packages">
  ### Installer et importer des packages
</div>

Installez la bibliothèque W&amp;B.

```
pip install wandb
```
