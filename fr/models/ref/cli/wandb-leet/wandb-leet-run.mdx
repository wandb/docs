```bash

```

<div id="description">
  ## Description
</div>

Lancez le TUI LEET.

LEET est une interface utilisateur de terminal permettant d’afficher un W&amp;B Run spécifié par un PATH facultatif.

PATH peut inclure un fichier .wandb, un répertoire d’exécution contenant un fichier .wandb,
ou une URL de W&amp;B Run.
Si PATH n’est pas fourni, la commande recherchera le run le plus récent.
