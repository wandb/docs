<div id="usage">
  ## Utilisation
</div>

```bash
purge-cache [OPTIONS]
```

<div id="description">
  ## Description
</div>

Supprime du cache local de W&amp;B les journaux, l’historique des runs et les
artifacts.

<div id="options">
  ## Options
</div>

| Option    | Type            | Description                                                                                                                                                                      |
| --------- | --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `--age`   | STR             | Supprime les éléments datant de plus de la période spécifiée (par ex., &#39;10s&#39;, &#39;5m&#39;, &#39;8h&#39;, &#39;7d&#39;, &#39;6M&#39;, &#39;1y&#39;). **Par défaut** : 0d |
| `--force` | Indicateur BOOL | Ne demande pas de confirmation avant de supprimer des fichiers. **Par défaut** : False                                                                                           |
