Supprime du cache local de W&amp;B les journaux en cache, l’historique du run et les artefacts.


<div id="usage">
  ## Utilisation
</div>

```bash
wandb purge-cache [OPTIONS]
```


<div id="options">
  ## Options
</div>

| Option    | Description                                                                                                                                                                     |
| :-------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `--age`   | Supprime les éléments datant de plus que la période spécifiée (par ex. : &#39;10s&#39;, &#39;5m&#39;, &#39;8h&#39;, &#39;7d&#39;, &#39;6M&#39;, &#39;1y&#39;) (par défaut : 0d) |
| `--force` | Ne demande pas de confirmation lors de la suppression de fichiers. (par défaut : False)                                                                                         |
