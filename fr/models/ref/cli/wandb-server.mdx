<div id="subcommands">

</div>

| Commande             | Description                                                              |
| -------------------- | ------------------------------------------------------------------------ |
| `wandb server start` | Démarre une instance locale du serveur W&amp;B dans un conteneur Docker. |
| `wandb server stop`  | Arrête un serveur W&amp;B local en cours d’exécution.                    |
