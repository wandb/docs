<div id="usage">
  ## Utilisation
</div>

```bash
wandb docker [DOCKER_RUN_ARGS] [DOCKER_IMAGE] [OPTIONS]
```


<div id="arguments">
  ## Arguments
</div>

| Argument          | Description                   | Requis |
| :---------------- | :---------------------------- | :----- |
| `DOCKER_RUN_ARGS` | Aucune description disponible | Non    |
| `DOCKER_IMAGE`    | Aucune description disponible | Non    |

<div id="options">
  ## Options
</div>

| Option      | Description                                                                                                   |
| :---------- | :------------------------------------------------------------------------------------------------------------ |
| `--nvidia`  | Utilise le runtime nvidia ; la valeur par défaut est nvidia si nvidia-docker est présent (par défaut : False) |
| `--digest`  | Affiche le digest de l&#39;image puis quitte (par défaut : False)                                             |
| `--jupyter` | Exécute jupyter lab dans le conteneur (par défaut : False)                                                    |
| `--dir`     | Répertoire du conteneur dans lequel monter le code (par défaut : /app)                                        |
| `--no-dir`  | Ne monte pas le répertoire courant (par défaut : False)                                                       |
| `--shell`   | Shell avec lequel démarrer le conteneur (par défaut : /bin/bash)                                              |
| `--port`    | Port hôte auquel lier jupyter (par défaut : 8888)                                                             |
| `--cmd`     | La commande à exécuter dans le conteneur                                                                      |
| `--no-tty`  | Exécute la commande sans TTY (par défaut : False)                                                             |
