Démarrez un conteneur W&amp;B local (obsolète, voir wandb server --help)


<div id="usage">
  ## Utilisation
</div>

```bash
wandb local [OPTIONS]
```


<div id="options">
  ## Options
</div>

| Option         | Description                                                             |
| :------------- | :---------------------------------------------------------------------- |
| `--port`, `-p` | Le port hôte auquel associer W&amp;B local (par défaut : 8080)          |
| `--env`, `-e`  | Variables d’environnement à transmettre à wandb/local (par défaut : []) |
| `--daemon`     | Exécuter ou non en mode daemon (par défaut : True)                      |
| `--upgrade`    | Mettre à jour vers la version la plus récente (par défaut : False)      |
