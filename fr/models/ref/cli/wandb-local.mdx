Démarrer un conteneur W&amp;B local (obsolète, voir wandb server --help)


<div id="usage">
  ## Utilisation
</div>

```bash
wandb local [OPTIONS]
```


<div id="options">
  ## Options
</div>

| Option         | Description                                                            |
| :------------- | :--------------------------------------------------------------------- |
| `--port`, `-p` | Le port hôte auquel associer W&amp;B local (par défaut : 8080)         |
| `--env`, `-e`  | Variables d&#39;environnement à passer à wandb/local (par défaut : []) |
| `--daemon`     | Exécuter ou non en mode démon (par défaut : True)                      |
| `--upgrade`    | Mettre à niveau vers la version la plus récente (par défaut : False)   |
