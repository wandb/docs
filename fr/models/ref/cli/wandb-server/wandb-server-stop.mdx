<div id="description">
  ## Description
</div>

Arrêter un serveur W&amp;B local en cours d’exécution.

Arrête le conteneur Docker nommé `wandb-local` lancé
par `wandb server start`. Docker doit être installé.

Par exemple, pour arrêter le serveur W&amp;B local :

```bash
wandb server stop
```
