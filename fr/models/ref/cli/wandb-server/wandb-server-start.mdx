<div id="usage">
  ## Utilisation
</div>

```bash
start [OPTIONS]
```

<div id="description">
  ## Description
</div>

Démarrez une instance locale de W&amp;B Server dans un conteneur Docker.

Téléchargez et exécutez l’image Docker wandb/local. Associez le
port hôte spécifié au port 8080 du conteneur et montez un volume Docker
persistant nommé &quot;wandb&quot; pour stocker les données.

Configurez le paramètre &quot;base&#95;url&quot; pour qu’il pointe vers le serveur local afin
que les appels clients W&amp;B suivants l’utilisent. Invitez à vous connecter si aucune
clé API n’est trouvée.

Docker doit être installé et en cours d’exécution sur la machine hôte.

Cette commande démarre uniquement l’interface utilisateur de W&amp;B Models. Elle est fournie uniquement
pour les tests locaux. En production, l’opérateur Kubernetes W&amp;B est la méthode recommandée
pour déployer W&amp;B Serv[https://docs.wandb.ai/platform/hosting/self-managed/operator](https://docs.wandb.ai/platform/hosting/self-managed/operator)ged/operator.

Par exemple, pour démarrer un serveur W&amp;B local sur le port par défaut (8080) :

```bash
wandb server start
```

Pour démarrer le serveur sur le port 9090 :

```bash
wandb server start -p 9090
```

Pour démarrer le serveur au premier plan :

```bash
wandb server start --no-daemon
```

<div id="options">
  ## Options
</div>

| Indicateur   | Type      | Description                                                                                                     |
| ------------ | --------- | --------------------------------------------------------------------------------------------------------------- |
| `--port, -p` | STR       | Le port hôte auquel lier le serveur W&amp;B. **Par défaut** : 8080                                              |
| `--env, -e`  | STR       | Variables d’environnement à transmettre à l’image Docker wandb/local. **Par défaut** : []                       |
| `--daemon`   | BOOL Flag | Exécute le serveur en arrière-plan. Utilisez --no-daemon pour l’exécuter au premier plan. **Par défaut** : True |
