Démarrez un serveur W&amp;B en local


<div id="usage">
  ## Utilisation
</div>

```bash
wandb server start [OPTIONS]
```


<div id="options">
  ## Options
</div>

| Option         | Description                                                                 |
| :------------- | :-------------------------------------------------------------------------- |
| `--port`, `-p` | Port hôte sur lequel le serveur W&amp;B est lié (par défaut : 8080)         |
| `--env`, `-e`  | Variables d&#39;environnement à transmettre à wandb/local (par défaut : []) |
| `--daemon`     | Exécuter ou non en mode démon (par défaut : True)                           |
