Démarrer un serveur W&amp;B local


<div id="usage">
  ## Utilisation
</div>

```bash
wandb server start [OPTIONS]
```


<div id="options">
  ## Options
</div>

| Option         | Description                                                                 |
| :------------- | :-------------------------------------------------------------------------- |
| `--port`, `-p` | Le port hôte auquel associer serveur W&amp;B (par défaut : 8080)             |
| `--env`, `-e`  | Variables d&#39;environnement à transmettre à wandb/local (par défaut : []) |
| `--daemon`     | Activer ou non le mode démon (par défaut : True)                            |
