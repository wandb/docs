Télécharger des fichiers depuis Weights &amp; Biases


<div id="usage">
  ## Utilisation
</div>

```bash
wandb pull RUN [OPTIONS]
```


<div id="arguments">
  ## Arguments
</div>

| Argument | Description                   | Requis |
| :------- | :---------------------------- | :----- |
| `RUN`    | Aucune description disponible | oui    |

<div id="options">
  ## Options
</div>

| Option            | Description                                                   |
| :---------------- | :------------------------------------------------------------ |
| `--project`, `-p` | Le projet à télécharger.                                      |
| `--entity`, `-e`  | L’entité sur laquelle limiter la liste. (par défaut : models) |
