Lister les jobs d’un projet


<div id="usage">
  ## Utilisation
</div>

```bash
wandb job list [OPTIONS]
```


<div id="options">
  ## Options
</div>

| Option            | Description                                                      |
| :---------------- | :--------------------------------------------------------------- |
| `--project`, `-p` | Le projet dont vous voulez lister les jobs.                      |
| `--entity`, `-e`  | L’entité à laquelle appartiennent les jobs (par défaut : models) |
