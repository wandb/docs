<div id="options">
  ## Options
</div>

| Option                 | Description                                                       |
| :--------------------- | :---------------------------------------------------------------- |
| `--cloud`              | Se connecter au cloud plutôt qu&#39;en local (par défaut : False) |
| `--host`, `--base-url` | Se connecter à une instance spécifique de W&amp;B                 |
| `--relogin`            | Forcer une nouvelle connexion si vous êtes déjà connecté.         |
| `--verify`             | Vérifier les identifiants de connexion (par défaut : False)       |
