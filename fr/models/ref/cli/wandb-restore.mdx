Restaure le code, la configuration et l’état Docker d’un run. Récupère le code depuis le commit le plus récent si le code n’a pas été enregistré avec `wandb.save()` ou `wandb.init(save_code=True)`.


<div id="usage">
  ## Utilisation
</div>

```bash
wandb restore RUN [OPTIONS]
```


<div id="arguments">
  ## Arguments
</div>

| Argument | Description                   | Requis |
| :------- | :---------------------------- | :----- |
| `RUN`    | Aucune description disponible | Oui    |

<div id="options">
  ## Options
</div>

| Option            | Description                                                                             |
| :---------------- | :-------------------------------------------------------------------------------------- |
| `--no-git`        | Ne restaure pas l’état git (par défaut : False)                                         |
| `--branch`        | Indique s’il faut créer une branche ou utiliser un checkout détaché (par défaut : True) |
| `--project`, `-p` | Le projet dans lequel vous souhaitez téléverser.                                        |
| `--entity`, `-e`  | L’entité à laquelle limiter la liste.                                                   |
