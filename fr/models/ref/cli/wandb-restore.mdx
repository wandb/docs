Restaure le code, la configuration et l’état de Docker d’un run. Récupère le code à partir du dernier commit si le code n’a pas été enregistré avec `wandb.save()` ou `wandb.init(save_code=True)`.


<div id="usage">
  ## Utilisation
</div>

```bash
wandb restore RUN [OPTIONS]
```


<div id="arguments">
  ## Arguments
</div>

| Argument | Description                   | Requis |
| :------- | :---------------------------- | :----- |
| `RUN`    | Aucune description disponible | Oui    |

<div id="options">
  ## Options
</div>

| Option            | Description                                                                        |
| :---------------- | :--------------------------------------------------------------------------------- |
| `--no-git`        | Ne restaure pas l’état de Git (par défaut : False)                                 |
| `--branch`        | Indique s’il faut créer une branche ou passer en HEAD détachée (par défaut : True) |
| `--project`, `-p` | Le projet vers lequel vous souhaitez téléverser.                                   |
| `--entity`, `-e`  | L’entité à laquelle limiter la portée de la liste.                                 |
