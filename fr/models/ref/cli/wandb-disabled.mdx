Désactivez W&amp;B.


<div id="usage">
  ## Utilisation
</div>

```bash
wandb disabled [OPTIONS]
```


<div id="options">
  ## Options
</div>

| Option      | Description                                       |
| :---------- | :------------------------------------------------ |
| `--service` | Désactiver le service W&amp;B (par défaut : True) |
