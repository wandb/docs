<div id="description">
  ## Description
</div>

Désactive les fonctionnalités de W&amp;B.

Lorsque W&amp;B est en mode `disabled`, aucune donnée n’est journalisée ni synchronisée.

Utilisez `wandb enable` pour rétablir les fonctionnalités de W&amp;B.

Utilisez `wandb offline` pour arrêter la synchronisation avec le cloud tout en continuant à journaliser les données
localement.

Par exemple, pour désactiver W&amp;B afin que le script train.py s’exécute sans journaliser ni
synchroniser de données vers W&amp;B :

```bash
wandb disabled
```

Ensuite, entraînez le modèle sans journalisation ni synchronisation vers W&amp;B :

```bash
python train.py  # Ne journalise ni ne synchronise les données vers W&B
```

Restaurez les fonctionnalités de W&amp;B lorsque vous êtes prêt à reprendre la journalisation et la synchronisation :

```bash
wandb enabled
```
