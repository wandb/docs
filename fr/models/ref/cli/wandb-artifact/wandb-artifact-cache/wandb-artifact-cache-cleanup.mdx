Nettoyer les fichiers les moins utilisés du cache d’Artifacts


<div id="usage">
  ## Utilisation
</div>

```bash
wandb artifact cache cleanup TARGET_SIZE [OPTIONS]
```


<div id="arguments">
  ## Arguments
</div>

| Argument      | Description                   | Requis |
| :------------ | :---------------------------- | :----- |
| `TARGET_SIZE` | Aucune description disponible | Oui    |

<div id="options">
  ## Options
</div>

| Option          | Description                                             |
| :-------------- | :------------------------------------------------------ |
| `--remove-temp` | Supprimer les fichiers temporaires (par défaut : False) |
