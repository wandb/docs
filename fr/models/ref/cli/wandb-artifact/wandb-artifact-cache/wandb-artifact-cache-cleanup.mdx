Supprimez du cache des artefacts les fichiers les moins utilisés


<div id="usage">
  ## Utilisation
</div>

```bash
wandb artifact cache cleanup TARGET_SIZE [OPTIONS]
```

  ## Arguments
</div>

| :------------ | :---------------------------- | :----- |
| `TARGET_SIZE` | Aucune description disponible | oui    |
