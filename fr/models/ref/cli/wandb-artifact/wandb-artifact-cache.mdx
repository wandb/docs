<div id="description">
  ## Description
</div>

Gérez le cache local des artefacts.

Mettez en cache localement les fichiers d’artefacts téléchargés afin d’éviter les téléchargements redondants.

Utilisez les sous-commandes pour inspecter le cache et récupérer l’espace disque qu’il utilise.

<div id="subcommands">
  ## Sous-commandes
</div>

| Commande                       | Description                                    |
| ------------------------------ | ---------------------------------------------- |
| `wandb artifact cache cleanup` | Réduit la taille du cache local des artefacts. |
