Téléverser un artefact sur wandb


<div id="usage">
  ## Utilisation
</div>

```bash
wandb artifact put PATH [OPTIONS]
```


<div id="arguments">
  ## Arguments
</div>

| Argument | Description                         | Requis |
| :------- | :---------------------------------- | :----- |
| `PATH`   | Aucune description n’est disponible | Oui    |

<div id="options">
  ## Options
</div>

| Option                | Description                                                                                                |
| :-------------------- | :--------------------------------------------------------------------------------------------------------- |
| `--name`, `-n`        | Le nom de l&#39;artefact à envoyer : projet/nom&#95;artefact                                               |
| `--description`, `-d` | Une description de cet artefact                                                                            |
| `--type`, `-t`        | Le type de l&#39;artefact (par défaut : jeu de données)                                                    |
| `--alias`, `-a`       | Un alias à appliquer à cet artefact (par défaut : [&#39;latest&#39;])                                      |
| `--id`                | Le run vers lequel vous souhaitez téléverser.                                                              |
| `--resume`            | Reprendre le dernier run depuis votre répertoire actuel.                                                   |
| `--skip_cache`        | Ignore la mise en cache lors du téléversement des fichiers d&#39;artefact. (par défaut : False)            |
| `--policy`            | Définit la politique de stockage lors du téléversement des fichiers d&#39;artefact. (par défaut : mutable) |
