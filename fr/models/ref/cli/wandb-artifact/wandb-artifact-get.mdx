Télécharger un artefact depuis wandb


<div id="usage">
  ## Utilisation
</div>

```bash
wandb artifact get PATH [OPTIONS]
```


<div id="arguments">
  ## Arguments
</div>

| Argument | Description                   | Requis |
| :------- | :---------------------------- | :----- |
| `PATH`   | Aucune description disponible | Oui    |

<div id="options">
  ## Options
</div>

| Option   | Description                                                     |
| :------- | :-------------------------------------------------------------- |
| `--root` | Le répertoire dans lequel vous souhaitez télécharger l’artefact |
| `--type` | Le type d’artefact à télécharger                                |
