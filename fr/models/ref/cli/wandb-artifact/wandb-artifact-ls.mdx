<div id="usage">
  ## Utilisation
</div>

```bash
ls [OPTIONS] PATH
```

<div id="description">
  ## Description
</div>

Répertorie tous les artefacts d’un projet W&amp;B.

Affiche la dernière version de chaque collection d’artefacts d’un
projet. Indique le type, la date de dernière mise à jour, la taille et le nom.

Le PATH est constitué de l’entité et du projet (entité/projet) à partir
desquels vous voulez lister les artefacts.

Par exemple, pour lister tous les artefacts d’un projet appelé &quot;foobar&quot; dans
l’entité &quot;team-awesome&quot; :

```bash
wandb artifact ls team-awesome/foobar
```

Pour lister uniquement les artefacts de type &quot;model&quot; dans le même projet :

```bash
wandb artifact ls --type model team-awesome/foobar
```
