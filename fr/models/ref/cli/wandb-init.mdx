Configurer un répertoire avec Weights &amp; Biases


<div id="usage">
  ## Utilisation
</div>

```bash
wandb init [OPTIONS]
```


<div id="options">
  ## Options
</div>

| Option            | Description                                                               |
| :---------------- | :------------------------------------------------------------------------ |
| `--project`, `-p` | Le projet à utiliser.                                                     |
| `--entity`, `-e`  | L&#39;entité à laquelle associer le projet.                               |
| `--reset`         | Réinitialiser les paramètres (par défaut : False)                         |
| `--mode`, `-m`    | Peut être « online », « offline » ou « disabled ». Par défaut : `online`. |
