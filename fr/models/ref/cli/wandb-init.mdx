Configurer un répertoire pour Weights &amp; Biases


<div id="usage">
  ## Utilisation
</div>

```bash
wandb init [OPTIONS]
```


<div id="options">
  ## Options
</div>

| Option            | Description                                                                                                             |
| :---------------- | :---------------------------------------------------------------------------------------------------------------------- |
| `--project`, `-p` | Le projet à utiliser.                                                                                                   |
| `--entity`, `-e`  | L’entité à laquelle le projet est associé.                                                                              |
| `--reset`         | Réinitialise les paramètres (par défaut : False)                                                                        |
| `--mode`, `-m`    | Peut être &quot;online&quot;, &quot;offline&quot; ou &quot;disabled&quot;. La valeur par défaut est &quot;online&quot;. |
