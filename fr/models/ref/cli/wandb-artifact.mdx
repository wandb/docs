Commandes pour interagir avec les Artifacts

<div id="usage">
  ## Utilisation
</div>

```bash
wandb artifact COMMAND [ARGS]...
```

<div id="commands">
  ## Commandes
</div>

| Commande                                                     | Description                                                    |
| :----------------------------------------------------------- | :------------------------------------------------------------- |
| [cache](/fr/models/ref/cli/wandb-artifact/wandb-artifact-cache) | Commandes permettant d’interagir avec le cache d&#39;artifacts |
| [get](/fr/models/ref/cli/wandb-artifact/wandb-artifact-get)     | Télécharger un artifact depuis wandb                           |
| [ls](/fr/models/ref/cli/wandb-artifact/wandb-artifact-ls)       | Lister tous les artifacts dans un projet wandb                 |
| [put](/fr/models/ref/cli/wandb-artifact/wandb-artifact-put)     | Téléverser un artifact vers wandb                              |
