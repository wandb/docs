Exécutez l’agent W&amp;B


<div id="usage">
  ## Utilisation
</div>

```bash
wandb agent SWEEP_ID [OPTIONS]
```


<div id="arguments">
  ## Arguments
</div>

| Argument   | Description                         | Requis |
| :--------- | :---------------------------------- | :----- |
| `SWEEP_ID` | Aucune description n’est disponible | Oui    |

<div id="options">
  ## Options
</div>

| Option                    | Description                                                                                                                                                                                                                                                                                                           |
| :------------------------ | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `--project`, `-p`         | Le nom du projet auquel sont envoyés les runs W&amp;B créés à partir du balayage. Si le projet n&#39;est pas spécifié, le run est envoyé vers un projet nommé &#39;Uncategorized&#39;.                                                                                                                                |
| `--entity`, `-e`          | Le nom d’utilisateur ou de l’équipe vers lesquels vous souhaitez envoyer les runs W&amp;B créés par le balayage. Assurez-vous que l’entité que vous spécifiez existe déjà. Si vous ne spécifiez pas d’entité, le run sera envoyé vers votre entité par défaut, qui correspond généralement à votre nom d’utilisateur. |
| `--count`                 | Le nombre maximal de runs pour cet agent.                                                                                                                                                                                                                                                                             |
| `--forward-signals`, `-f` | Transmet aux runs enfants les signaux reçus par l’agent (par ex. SIGINT/SIGTERM) afin qu’ils puissent s’arrêter proprement. (par défaut : False)                                                                                                                                                                      |
