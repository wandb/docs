<div id="usage">
  ## Utilisation
</div>

```bash
launch-agent [OPTIONS]
```

<div id="description">
  ## Description
</div>

Exécutez un agent Launch de W&amp;B.

<div id="options">
  ## Options
</div>

| Option           | Type | Description                                                                                                                                                                                                                                                        |
| ---------------- | ---- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `--queue, -q`    | STR  | Nom d&#39;une file d&#39;attente à surveiller par l&#39;agent. Plusieurs options `-q` sont prises en charge. **Par défaut :** aucune                                                                                                                               |
| `--entity, -e`   | STR  | Entité à utiliser. Par défaut, utilise l&#39;utilisateur actuellement connecté. **Par défaut :** aucune                                                                                                                                                            |
| `--log-file, -l` | STR  | Destination des journaux internes de l&#39;agent. Utilisez `-` pour la sortie standard. Par défaut, tous les journaux des agents sont écrits dans `debug.log` dans votre sous-répertoire `wandb/` ou dans `WANDB_DIR` s&#39;il est défini. **Par défaut :** aucune |
| `--max-jobs, -j` | STR  | Nombre maximal de jobs Launch que cet agent peut exécuter en parallèle. La valeur par défaut est 1. Définissez `-1` pour supprimer toute limite supérieure. **Par défaut :** aucune                                                                                |
| `--config, -c`   | STR  | Chemin vers le fichier YAML de configuration de l&#39;agent à utiliser. **Par défaut :** aucune                                                                                                                                                                    |
| `--verbose, -v`  | INT  | Affiche une sortie détaillée. **Par défaut :** 0                                                                                                                                                                                                                   |
