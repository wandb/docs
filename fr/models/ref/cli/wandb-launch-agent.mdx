Exécutez un agent Launch de W&amp;B.


<div id="usage">
  ## Utilisation
</div>

```bash
wandb launch-agent [OPTIONS]
```


<div id="options">
  ## Options
</div>

| Option             | Description                                                                                                                                                                                                            |
| :----------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `--queue`, `-q`    | Le nom d’une file d’attente à surveiller par l’agent. Plusieurs indicateurs `-q` sont pris en charge.                                                                                                                  |
| `--entity`, `-e`   | L’entité à utiliser. Par défaut, il s’agit de l’utilisateur actuellement connecté.                                                                                                                                     |
| `--log-file`, `-l` | Destination des journaux internes de l’agent. Utilisez `-` pour stdout. Par défaut, tous les journaux des agents sont écrits dans `debug.log` dans votre sous-répertoire `wandb/` ou dans `WANDB_DIR` s’il est défini. |
| `--max-jobs`, `-j` | Le nombre maximal de launch jobs que cet agent peut exécuter en parallèle. La valeur par défaut est 1. Définissez `-1` pour ne pas avoir de limite supérieure.                                                         |
| `--config`, `-c`   | chemin vers le fichier YAML de configuration de l’agent à utiliser                                                                                                                                                     |
| `--verbose`, `-v`  | Affiche une sortie détaillée (par défaut : 0)                                                                                                                                                                          |
