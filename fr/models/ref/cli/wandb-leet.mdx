<div id="description">
  ## Description
</div>

W&amp;B LEET : l’outil léger d’exploration des expériences.

Une interface utilisateur dans le terminal pour consulter localement vos runs W&amp;B.



<div id="subcommands">
  ## Sous-commandes
</div>

| Commande            | Description                         |
| ------------------- | ----------------------------------- |
| `wandb leet config` | Modifie la configuration de LEET.   |
| `wandb leet run`    | Lance l’interface TUI de LEET.      |
| `wandb leet symon`  | Lance le moniteur système autonome. |
