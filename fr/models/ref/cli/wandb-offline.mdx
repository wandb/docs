Enregistrez localement les données enregistrées sur W&amp;B sans les téléverser dans le cloud.

Utilisez `wandb online` ou `wandb sync` pour téléverser les runs hors ligne.


<div id="usage">
  ## Utilisation
</div>

```bash
wandb offline
```
