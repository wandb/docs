Téléverser les fichiers .wandb spécifiés par PATHS.

Il s&#39;agit d&#39;une réimplémentation bêta de `wandb sync`. Elle n&#39;est pas encore complète sur le plan fonctionnel, son bon fonctionnement n&#39;est pas garanti et elle peut évoluer de manière non rétrocompatible dans n&#39;importe quelle version de wandb.

PATHS peut inclure des fichiers .wandb, des répertoires de run contenant des fichiers .wandb et des répertoires &quot;wandb&quot; contenant des répertoires de run.

Par exemple, pour synchroniser tous les runs d&#39;un répertoire :

```bash
wandb beta sync ./wandb
```

Pour synchroniser un run précis :

```bash
wandb beta sync ./wandb/run-20250813_124246-n67z9ude
```

Ou, de façon équivalente :

```bash
wandb beta sync ./wandb/run-20250813_124246-n67z9ude/run-n67z9ude.wandb
```

</div>
