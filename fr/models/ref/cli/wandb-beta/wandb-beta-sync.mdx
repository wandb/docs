<div id="usage">
  ## Utilisation
</div>

```bash
sync [OPTIONS] [PATHS]...
```

<div id="description">
  ## Description
</div>

Téléversez les fichiers .wandb spécifiés par PATHS.

Il s’agit d’une réimplémentation bêta de `wandb sync`.
Elle n’est pas complète sur le plan fonctionnel, son fonctionnement n’est pas garanti et elle peut évoluer
de manière non rétrocompatible dans n’importe quelle version de wandb.

PATHS peut inclure des fichiers .wandb, des répertoires de run contenant des fichiers .wandb,
ainsi que des répertoires &quot;wandb&quot; contenant des répertoires de run.

Par exemple, pour synchroniser tous les Runs du répertoire .wandb actuel :

```bash
wandb beta sync ./wandb
```

Pour synchroniser un run précis en spécifiant le répertoire du run :

```bash
wandb beta sync ./wandb/run-20250813_124246-n67z9ude
```

Ou, de façon équivalente :

```bash
wandb beta sync ./wandb/run-20250813_124246-n67z9ude/run-n67z9ude.wandb
```
