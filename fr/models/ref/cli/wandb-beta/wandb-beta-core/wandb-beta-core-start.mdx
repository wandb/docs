| Option           | Type | Description                                                                                                                                                                                                       |
| ---------------- | ---- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `--idle-timeout` | STR  | Arrête wandb-core après cette durée d&#39;inactivité sans client connecté. Utilise la syntaxe de durée de Go, par exemple 30s, 10m ou 0 pour désactiver l&#39;arrêt en cas d&#39;inactivité. **Par défaut** : 10m |
