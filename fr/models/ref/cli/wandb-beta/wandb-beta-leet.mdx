W&amp;B LEET : l’outil léger d’exploration des expériences.

Une interface de terminal pour consulter localement vos runs W&amp;B.

<div id="subcommands">

| ------------------------ | ------------------------------------ |

| `wandb beta leet run`    | Lancez l’interface TUI de LEET.      |
| `wandb beta leet symon`  | Lancez le moniteur système autonome. |
