</div>

```bash
run [OPTIONS] [PATH]
```

<div id="description">
  ## Description
</div>

Lancez l’interface terminal LEET.

LEET est une interface terminal permettant d’afficher un run W&amp;B indiqué par un PATH facultatif.

PATH peut inclure un fichier .wandb ou un répertoire contenant un fichier .wandb.
Si PATH n’est pas fourni, la commande recherche le run le plus récent.

| `path` | STR        | False |
