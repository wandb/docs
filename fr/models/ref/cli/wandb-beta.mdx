---
title: "wandb bêta"
---

Versions bêta des commandes de la CLI wandb.

Ces commandes peuvent changer, voire cesser complètement de fonctionner, d’une version de wandb à l’autre.

<div id="usage">
  ## Utilisation
</div>

```bash
wandb beta COMMAND [ARGS]...
```

<div id="commands">
  ## Commandes
</div>

| Commande                                           | Description                                               |
| :------------------------------------------------- | :-------------------------------------------------------- |
| [leet](/fr/models/ref/cli/wandb-beta/wandb-beta-leet) | W&amp;B LEET : l’outil léger d’exploration d’expériences. |
| [sync](/fr/models/ref/cli/wandb-beta/wandb-beta-sync) | Importe les fichiers .wandb spécifiés par PATHS.          |
