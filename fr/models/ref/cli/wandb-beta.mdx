---
title: "wandb bêta"
---

Versions bêta des commandes CLI de wandb.

Ces commandes peuvent être modifiées, voire cesser complètement de fonctionner dans n’importe quelle version de wandb.

<div id="usage">
  ## Utilisation
</div>

```bash
wandb beta COMMAND [ARGS]...
```

<div id="commands">
  ## Commandes
</div>

| Commande                                           | Description                                               |
| :------------------------------------------------- | :-------------------------------------------------------- |
| [leet](/fr/models/ref/cli/wandb-beta/wandb-beta-leet) | W&amp;B LEET : l’outil léger d’exploration d’expériences. |
| [sync](/fr/models/ref/cli/wandb-beta/wandb-beta-sync) | Téléverse les fichiers .wandb spécifiés dans PATHS.       |
