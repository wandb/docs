Créez un tableau d’évaluation à l’aide du SDK Python W&amp;B ou convertissez un [W&amp;B Table](/fr/models/tables) existant avec [ARIA](/fr/aria).

<div id="create-an-eval-table">
  ## Créer un tableau d’évaluation
</div>

Utilisez la classe `wandb.EvalTable` pour créer un tableau d’évaluation.

Par exemple, supposons que vous disposiez du dataframe pandas suivant :

```python
import pandas as pd
import wandb

df = pd.DataFrame(
    [
        {
            "image_id": "img_001",
            "true_label": "cat",
            "predicted_label": "cat",
            "confidence": 0.97,
            "correct": True,
        },
        {
            "image_id": "img_002",
            "true_label": "dog",
            "predicted_label": "cat",
            "confidence": 0.72,
            "correct": False,
        },
        {
            "image_id": "img_003",
            "true_label": "car",
            "predicted_label": "car",
            "confidence": 0.89,
            "correct": True,
        },
    ]
)
```

{/* ```python
  with wandb.init(project="classifier-table-demo") as run:

Créez une tableau d’évaluation en passant le DataFrame à `wandb.EvalTable`. Utilisez les arguments `input_columns`, `output_columns` et `score_columns` pour identifier le rôle de chaque colonne.

L’exemple suivant utilise :

* `image_id` comme entrée
* `predicted_label` comme sortie
* `correct` et `confidence` comme scores

```python
with wandb.init(project="classifier-eval-table-demo") as run:
    eval_table = wandb.EvalTable(
        dataframe=df,
        input_columns=["image_id"],
        output_columns=["predicted_label"],
        score_columns=["correct", "confidence"],
    )
    run.log({"validation_predictions_eval": eval_table})    
```

<div id="convert-a-wb-table-to-an-eval-table">
  ## Convertir une W&amp;B Table en tableau d’évaluation
</div>

Utilisez la bannière de conversion pour convertir une W&amp;B Table existante en tableau d’évaluation :

1. Accédez au projet qui contient la W&amp;B Table.
2. Sélectionnez le bouton **Try now** dans la bannière de conversion. Ce bouton ouvre [ARIA](/fr/aria) avec des informations sur la table à convertir.
3. Suivez les instructions dans la fenêtre de conversation d&#39;ARIA.

Si vous avez fermé la bannière ou qu&#39;elle n&#39;apparaît pas, ouvrez ARIA manuellement :

1. Sélectionnez le cercle bleu dans le coin supérieur droit de la page pour ouvrir ARIA.
2. Saisissez `/convert-eval-table` dans la conversation d&#39;ARIA.
3. Sélectionnez Send, représenté par la flèche vers le haut (<Icon icon="arrow-up" iconType="solid" />) dans l&#39;angle inférieur droit de la conversation.

La conversion d&#39;une W&amp;B Table ne modifie pas la table d&#39;origine.
