title: "Consulter les tableaux d’évaluation"
---

Lorsque vous créez un tableau d’évaluation, indiquez quelles colonnes contiennent les entrées, les sorties et les scores. W&amp;B regroupe ces colonnes dans les sections correspondantes du tableau d’évaluation.

L&#39;exemple suivant montre la structure de base d&#39;un objet EvalTable :

```python
import wandb

wandb.EvalTable(
    input_columns=["column1", "column2"],
    output_columns=["output_column1", "output_column2"],
    score_columns=["score_column1", "score_column2"],
)
```

Passez un ou plusieurs noms de colonne à chacun des arguments suivants :

* `input_columns` : données fournies au modèle, comme une image, un prompt ou une étiquette de référence.
* `output_columns` : valeurs produites par le modèle, comme une prédiction ou une réponse générée.
* `score_columns` : métriques ou autres valeurs utilisées pour évaluer la sortie.

W&amp;B affiche ces colonnes dans les sections Inputs, Outputs et Scores du tableau d’évaluation.

Par exemple, prenez l’exemple de code de [Create an Eval Table](/fr/models/evaltables/create-an-evaluation-table) :

```python
import wandb

with wandb.init(project="classifier-eval-table-demo") as run:
    eval_table = wandb.EvalTable(
        dataframe=df,
        input_columns=["image_id", "true_label"],
        output_columns=["predicted_label"],
        score_columns=["correct", "confidence"],
    )
    run.log({"validation_predictions_eval": eval_table})
```

Dans cet exemple :

* La section **Inputs** contient les colonnes `image_id` et `true_label`.
* La section **Outputs** contient la colonne `predicted_label`.
* La section **Scores** contient les colonnes `correct` et `confidence`.

<div id="filter-data">
  ## Filtrer les données
</div>

Utilisez des filtres pour n’afficher que les lignes correspondant à des conditions spécifiques.

Pour ajouter un filtre :

1. Sélectionnez le bouton **Filtre** au-dessus du tableau.
2. Dans le menu déroulant, sélectionnez la colonne à filtrer.
3. Sélectionnez un opérateur.
4. Spécifiez une valeur.

W&amp;B applique automatiquement le filtre au tableau et n’affiche que les lignes correspondant à la condition.

<Frame>
  <img src="/images/evaltables/filter.png" alt="Filtre du tableau d’évaluation" />
</Frame>

Vous pouvez appliquer plusieurs filtres. Pour supprimer un filtre, sélectionnez le `X` situé à côté.

<div id="show-or-hide-columns">
  ## Afficher ou masquer des colonnes
</div>

Utilisez le menu **Colonnes** pour définir quelles colonnes s’affichent dans le tableau d’évaluation. Les colonnes disponibles dépendent des données enregistrées dans le tableau.

Pour afficher ou masquer une colonne :

1. Sélectionnez **Colonnes** au-dessus du tableau.
2. Sélectionnez une colonne pour l’afficher, ou désélectionnez-la pour la masquer.
