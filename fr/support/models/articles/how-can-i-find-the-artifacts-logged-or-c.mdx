---
title: "Comment trouver les artefacts enregistrés ou consommés par un run ? Comment trouver les runs qui ont produit ou consommé un artefact ?"
keywords: ["Artifacts"]
---

W&amp;B suit les artefacts enregistrés par chaque run et ceux utilisés par chaque run afin de construire un graphe d&#39;artefacts. Ce graphe est un graphe biparti, orienté et acyclique dont les nœuds représentent des runs et des artefacts. Un exemple peut être consulté [ici](https://wandb.ai/shawn/detectron2-11/artifacts/dataset/furniture-small-val/06d5ddd4deeb2a6ebdd5/graph) (cliquez sur &quot;Explode&quot; pour développer le graphe).

Utilisez l&#39;API publique pour naviguer dans le graphe de manière programmatique, en partant soit d&#39;un artefact, soit d&#39;un run.

<Tabs>
  <Tab title="Depuis un artefact">
    ```python
    api = wandb.Api()

    artifact = api.artifact("project/artifact:alias")

    # Remonter le graphe depuis un artefact :
    producer_run = artifact.logged_by()
    # Descendre le graphe depuis un artefact :
    consumer_runs = artifact.used_by()

    # Descendre le graphe depuis un run :
    next_artifacts = consumer_runs[0].logged_artifacts()
    # Remonter le graphe depuis un run :
    previous_artifacts = producer_run.used_artifacts()
    ```
  </Tab>

  <Tab title="Depuis un run">
    ```python
    api = wandb.Api()

    run = api.run("entity/project/run_id")

    # Descendre le graphe depuis un run :
    produced_artifacts = run.logged_artifacts()
    # Remonter le graphe depuis un run :
    consumed_artifacts = run.used_artifacts()

    # Remonter le graphe depuis un artefact :
    earlier_run = consumed_artifacts[0].logged_by()
    # Descendre le graphe depuis un artefact :
    consumer_runs = produced_artifacts[0].used_by()
    ```
  </Tab>
</Tabs>
