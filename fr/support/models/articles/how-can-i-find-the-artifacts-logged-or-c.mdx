---
title: "Comment trouver les Artifacts enregistrés ou consommés par un run ? Comment trouver les runs qui ont produit ou consommé un artifact ?"
keywords: ["Artifacts"]
---

W&amp;B suit les Artifacts enregistrés par chaque run ainsi que ceux utilisés par chaque run afin de construire un graphe d’Artifacts. Ce graphe est un graphe biparti orienté acyclique, dont les nœuds représentent des runs et des Artifacts. Vous pouvez en voir un exemple [ici](https://wandb.ai/shawn/detectron2-11/artifacts/dataset/furniture-small-val/06d5ddd4deeb2a6ebdd5/graph) (cliquez sur &quot;Explode&quot; pour déployer le graphe).

Utilisez l’API publique pour parcourir le graphe par code, à partir d’un artifact ou d’un run.
