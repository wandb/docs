---
title: "Est-il possible de récupérer un artifact après avoir été supprimé avec un run ?"
keywords: ["Artifacts"]
---

Lors de la suppression d’un run, une invite vous demande si vous souhaitez supprimer les artifacts associés. Si vous choisissez cette option, les artifacts sont supprimés définitivement et il est impossible de les récupérer, même si le run lui-même est restauré par la suite.
