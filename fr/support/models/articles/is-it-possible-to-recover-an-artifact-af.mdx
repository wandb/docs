---
title: "Est-il possible de récupérer un artefact après avoir été supprimé avec un run ?"
keywords: ["Artifacts"]
---

Lorsque vous supprimez un run, une invite vous demande si vous souhaitez également supprimer les artefacts associés. Si vous choisissez cette option, les artefacts sont définitivement supprimés et ne peuvent pas être récupérés, même si le run lui-même est restauré par la suite.
