---
title: "Est-il possible de récupérer un artifact après avoir été supprimé avec un run ?"
keywords: ["Artifacts"]
---

Non. Lorsque vous supprimez un run, une invite vous demande si vous souhaitez supprimer les artifacts associés. Si vous choisissez cette option, W&amp;B supprime définitivement les artifacts. Vous ne pouvez pas les récupérer, même si vous restaurez le run plus tard.
