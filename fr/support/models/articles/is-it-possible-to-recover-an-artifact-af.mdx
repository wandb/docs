---
title: "Est-il possible de récupérer un artefact après sa suppression avec un run ?"
keywords: ["Artifacts"]
---

Lorsque vous supprimez un run, un message vous demande si vous souhaitez également supprimer les artefacts associés. Si vous choisissez cette option, les artefacts sont supprimés définitivement et il est impossible de les récupérer, même si le run lui-même est restauré par la suite.
