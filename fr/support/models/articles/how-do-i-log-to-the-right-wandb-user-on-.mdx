---
title: "Comment puis-je associer les runs au bon compte wandb sur une machine partagée ?"
keywords: ["Journaux"]
---

Lorsque plusieurs utilisateurs partagent une machine, les runs peuvent être associés au mauvais compte W&amp;B si vous ne définissez pas explicitement les identifiants. Pour vous assurer que les runs sont associés au bon compte W&amp;B, définissez la variable d&#39;environnement `WANDB_API_KEY` pour l&#39;authentification. Si cette variable est exportée dans l&#39;environnement, elle fournit les bons identifiants lors de la connexion. Vous pouvez également définir directement la variable d&#39;environnement dans le script.

Pour associer les runs W&amp;B suivants de cette session shell à votre compte, exportez votre clé API : `export WANDB_API_KEY=[API-KEY]`. Remplacez `[API-KEY]` par votre clé API, que vous pouvez créer sur [wandb.ai/settings](https://wandb.ai/settings).
