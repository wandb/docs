---
title: "Comment puis-je me connecter au bon compte wandb sur une machine partagée ?"
keywords: ["Logs"]
---

Lorsque vous utilisez une machine partagée, assurez-vous que les runs sont bien associés au bon compte wandb en définissant la variable d&#39;environnement `WANDB_API_KEY` pour l&#39;authentification. Si elle est définie dans l&#39;environnement, cette variable fournit les bons identifiants lors de la connexion. Vous pouvez également définir directement la variable d&#39;environnement dans le script.

Exécutez la commande `export WANDB_API_KEY=X`, en remplaçant X par votre clé API. Créez une clé API sur [wandb.ai/settings](https://wandb.ai/settings).
