---
title: "Où les artefacts sont-ils téléchargés et comment puis-je définir cet emplacement ?"
keywords: ["Artifacts", "Variables d’environnement"]
---

Par défaut, les artefacts sont téléchargés dans le dossier `artifacts/`. Pour modifier l’emplacement :

* Spécifiez-le dans [`wandb.Artifact().download`](/fr/models/ref/python/public-api/api) :

  ```python
  wandb.Artifact().download(root="<path_to_download>")
  ```

* Définissez la variable d’environnement `WANDB_ARTIFACT_DIR` ([variables d’environnement](/fr/models/track/environment-variables)) :

  ```python
  import os
  os.environ["WANDB_ARTIFACT_DIR"] = "<path_to_download>"
  ```
