---
title: "Où les Artifacts sont-ils téléchargés et comment puis-je contrôler leur emplacement ?"
keywords: ["Artifacts", "Variables d'environnement"]
---

Par défaut, les Artifacts sont téléchargés dans le dossier `artifacts/`. Pour modifier cet emplacement :

* Passez le chemin à [`wandb.Artifact().download`](/fr/models/ref/python/public-api/api) :

  ```python
  wandb.Artifact().download(root="<path_to_download>")
  ```

* Définissez la variable d&#39;environnement `WANDB_ARTIFACT_DIR` [Variables d&#39;environnement](/fr/models/track/environment-variables) :

  ```python
  import os
  os.environ["WANDB_ARTIFACT_DIR"] = "<path_to_download>"
  ```
