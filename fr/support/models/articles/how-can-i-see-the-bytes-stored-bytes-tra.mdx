---
title: "Comment voir les octets stockés, les octets suivis et les heures de suivi de mon organisation ?"
keywords: ["Administrateur"]
---

Consultez les octets stockés, les octets suivis et les heures de suivi de votre organisation dans les paramètres de votre organisation :

1. Accédez aux paramètres de votre organisation à l&#39;adresse `https://wandb.ai/account-settings/<organization-name>/settings`.
2. Sélectionnez l&#39;onglet **Facturation**.
3. Dans la section **Utilisation pour cette période de facturation**, cliquez sur le bouton **Voir l&#39;utilisation**.

Veillez à remplacer les valeurs entre `<>` par le nom de votre organisation.
