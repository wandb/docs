---
title: "Comment puis-je voir les octets stockés, les octets suivis et les heures suivies de mon organisation ?"
keywords: ["Administrateur"]
---

Voir les octets stockés, les octets suivis et les heures suivies de votre organisation dans les paramètres de l’organisation :

1. Accédez à la page des paramètres de votre organisation à l’adresse `https://wandb.ai/account-settings/<organization-name>/settings`.
2. Sélectionnez l’onglet **Billing**.
3. Dans la section **Usage this billing period**, sélectionnez le bouton **Voir l’utilisation**.

Veillez à remplacer les valeurs entre `<>` par le nom de votre organisation.
