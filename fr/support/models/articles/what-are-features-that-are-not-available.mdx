---
title: "Fonctionnalités non disponibles pour les utilisateurs anonymes"
keywords: ["Anonyme"]
---

Les comptes anonymes vous permettent d’essayer W&amp;B sans vous inscrire, mais plusieurs fonctionnalités sont restreintes. Prenez connaissance des limitations suivantes afin de décider si vous souhaitez créer un compte standard avant d’enregistrer des travaux importants.

* **Aucune donnée persistante** : les comptes anonymes conservent les Runs pendant 7 jours. Associez les données d&#39;un run anonyme à un compte standard en les enregistrant.

<Frame>
  <img src="/images/app_ui/anon_mode_no_data.png" alt="Interface du mode anonyme" />
</Frame>

* **Pas de journalisation d&#39;artifact** : lorsque vous journalisez un artifact dans un run anonyme, un avertissement apparaît sur la ligne de commande :
  ```bash
  wandb: WARNING Artifacts logged anonymously cannot be claimed and expire after 7 days.
  ```

* **Aucune page de profil ou de paramètres** : l&#39;interface utilisateur n&#39;inclut pas certaines pages, car elles ne sont utiles que pour les comptes standard.
