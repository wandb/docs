---
title: "Quelles fonctionnalités ne sont pas disponibles pour les utilisateurs anonymes ?"
keywords: ["Anonyme"]
---

* **Aucune donnée persistante** : les Runs sont conservés pendant 7 jours dans un compte anonyme. Associez les données d&#39;un run anonyme à un compte standard en les enregistrant.

<Frame>
  <img src="/images/app_ui/anon_mode_no_data.png" alt="Interface du mode anonyme" />
</Frame>

* **Pas de journalisation d&#39;artifact** : un avertissement apparaît sur la ligne de commande lorsque vous tentez de journaliser un artifact dans un run anonyme :
  ```bash
  wandb: WARNING Artifacts logged anonymously cannot be claimed and expire after 7 days.
  ```

* **Aucune page de profil ou de paramètres** : l&#39;interface utilisateur n&#39;inclut pas certaines pages, car elles ne sont utiles que pour les comptes standard.
