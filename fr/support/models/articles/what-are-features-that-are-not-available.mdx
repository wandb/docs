---
title: "Quelles sont les fonctionnalités qui ne sont pas disponibles pour les utilisateurs anonymes ?"
keywords: ["Anonyme"]
---

* **Pas de données persistantes** : les runs sont conservés pendant 7 jours dans un compte anonyme. Associez les données d’un run anonyme à un vrai compte en les enregistrant dans ce compte.

<Frame>
  <img src="/images/app_ui/anon_mode_no_data.png" alt="Interface du mode anonyme" />
</Frame>

* **Pas d’enregistrement d’artefacts** : un avertissement s’affiche sur la ligne de commande lorsque vous tentez d’enregistrer un artefact dans un run anonyme :
  ```bash
  wandb: WARNING Artifacts logged anonymously cannot be claimed and expire after 7 days.
  ```

* **Pas de pages de profil ni de paramètres** : l’interface utilisateur n’inclut pas certaines pages, car elles ne sont utiles que pour les vrais comptes.
