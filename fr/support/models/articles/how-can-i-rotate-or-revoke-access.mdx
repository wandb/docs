---
title: "Comment effectuer une rotation ou révoquer l’accès ?"
keywords: ["Administrateur", "Sécurité"]
---

Les clés personnelles et les clés de compte de service peuvent faire l’objet d’une rotation ou être révoquées. Créez une nouvelle clé API ou un nouvel utilisateur de compte de service, puis reconfigurez les scripts pour utiliser la nouvelle clé. Une fois la reconfiguration terminée, supprimez l’ancienne clé API de votre profil ou de Teams.
