---
title: "Comment puis-je renouveler ou révoquer un accès ?"
keywords: ["Administrateur", "Sécurité"]
---

Les clés de comptes personnels et de comptes de service peuvent être renouvelées ou révoquées. Créez une nouvelle clé API ou un nouvel utilisateur de compte de service, puis reconfigurez les scripts pour utiliser la nouvelle clé. Une fois la reconfiguration terminée, supprimez l&#39;ancienne clé API de votre profil ou de votre équipe.
