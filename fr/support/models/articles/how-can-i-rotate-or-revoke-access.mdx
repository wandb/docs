---
title: "Comment effectuer une rotation ou révoquer l’accès ?"
keywords: ["Administrateur", "Sécurité"]
---

Vous pouvez faire pivoter ou révoquer les clés personnelles et les clés de compte de service. Créez une nouvelle clé API ou un nouvel utilisateur de compte de service, puis reconfigurez vos scripts pour utiliser la nouvelle clé. Une fois que vos scripts utilisent la nouvelle clé, supprimez l’ancienne clé API de votre profil ou de votre équipe.
