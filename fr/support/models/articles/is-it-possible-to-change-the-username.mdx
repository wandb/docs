---
title: "Est-il possible de modifier le nom d’utilisateur ?"
keywords: ["Administrateur", "Gestion des utilisateurs"]
---

Il n’est pas possible de modifier le nom d’utilisateur après la création d’un compte. Créez plutôt un nouveau compte avec le nom d’utilisateur souhaité.
