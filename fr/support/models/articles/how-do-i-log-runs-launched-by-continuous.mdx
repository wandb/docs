---
title: "Comment puis-je enregistrer des runs lancés via l’intégration continue ou des outils internes ?"
keywords: ["Runs", "Logs"]
---

Pour lancer des tests automatisés ou des outils internes qui enregistrent dans W&amp;B, créez un **compte de service** sur la page Team settings. Cette action permet d’utiliser une clé API de service pour les jobs automatisés, y compris ceux exécutés via l’intégration continue. Pour attribuer les jobs du compte de service à un utilisateur spécifique, définissez les variables d’environnement `WANDB_USERNAME` ou `WANDB_USER_EMAIL`.
