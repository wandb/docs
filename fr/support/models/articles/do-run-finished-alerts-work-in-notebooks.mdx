---
title: "Les alertes « Run Finished » fonctionnent-elles dans les notebooks ?"
keywords: ["Alertes", "Notebooks"]
---

Non. Les alertes **Run Finished** (activées via l’option **Run Finished** dans les Paramètres utilisateur) ne fonctionnent qu’avec les scripts Python et restent désactivées dans les environnements Jupyter Notebook, afin d’éviter l’envoi d’une notification à chaque exécution de cellule.

Utilisez plutôt `run.alert()` dans les environnements notebook.
