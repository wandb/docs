---
title: "Puis-je utiliser wandb hors ligne ?"
keywords: ["Experiments", "Variables d’environnement"]
---

Si l’entraînement s’effectue sur une machine sans connexion Internet, suivez les étapes ci-dessous pour envoyer les résultats vers les serveurs :

1. Définissez la variable d’environnement `WANDB_MODE=offline` pour enregistrer les métriques localement sans connexion Internet.
2. Lorsque vous êtes prêt à les envoyer, exécutez `wandb init` dans votre répertoire pour définir le nom du projet.
3. Utilisez `wandb sync YOUR_RUN_DIRECTORY` pour transférer les métriques vers le service cloud et consulter les résultats dans l’application Web hébergée.

Pour confirmer que le run est hors ligne, vérifiez `run.settings._offline` ou `run.settings.mode` après avoir exécuté `wandb.init()`.
