---
title: "Quel effet `wandb.init()` a-t-il sur mon processus d’entraînement ?"
keywords: ["Variables d’environnement", "Experiments"]
featured: true
---

Lorsque `wandb.init()` s’exécute dans un script d’entraînement, un appel d’API crée un objet run sur les serveurs. Un nouveau processus est lancé pour transmettre et collecter les métriques, ce qui permet au processus principal de fonctionner normalement. Le script écrit dans des fichiers locaux tandis que le processus distinct transmet les données aux serveurs, y compris les métriques système. Pour désactiver la transmission en continu, exécutez `wandb off` depuis le répertoire d’entraînement ou définissez la variable d’environnement `WANDB_MODE` sur `offline`.
