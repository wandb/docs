---
title: "Que fait wandb.init() dans mon processus d'entraînement ?"
keywords: ["Variables d'environnement", "Experiments"]
featured: true
---

Lorsque `wandb.init()` s&#39;exécute dans un script d&#39;entraînement, un appel d&#39;API crée un objet run sur les serveurs. Un nouveau processus démarre pour transmettre et collecter les métriques, ce qui permet au processus principal de continuer à fonctionner normalement. Le script écrit dans des fichiers locaux tandis que le processus distinct transmet les données aux serveurs, y compris les métriques système. Pour désactiver cette transmission, exécutez `wandb off` depuis le répertoire d&#39;entraînement ou définissez la variable d&#39;environnement `WANDB_MODE` sur `offline`.
