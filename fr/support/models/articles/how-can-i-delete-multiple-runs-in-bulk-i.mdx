---
title: "Comment supprimer plusieurs runs en une seule fois plutôt qu’un par un ?"
keywords: ["Projects", "Runs"]
---

Utilisez l’[API publique](/fr/models/ref/python/public-api/api) pour supprimer plusieurs runs en une seule opération :

```python
import wandb

api = wandb.Api()
runs = api.runs('<entity>/<project>')
for run in runs:
    if <condition>:
        run.delete()
```
