---
title: "Existe-t-il un package Anaconda pour Weights and Biases ?"
keywords: ["Python"]
---

Il existe un package Anaconda que vous pouvez installer avec `pip` ou `conda`. Avec `conda`, obtenez le package depuis le canal [conda-forge](https://conda-forge.org).

<Tabs>
  <Tab title="pip">
    ```shell
    # Créer un environnement conda
    conda create -n wandb-env python=3.8 anaconda
    # Activer l'environnement
    conda activate wandb-env
    # Installer wandb avec pip
    pip install wandb
    ```
  </Tab>

  <Tab title="conda">
    ```shell
    conda activate myenv
    conda install wandb --channel conda-forge
    ```
  </Tab>
</Tabs>

En cas de problème d&#39;installation, reportez-vous à la [documentation Anaconda sur la gestion des packages](https://docs.conda.io/projects/conda/en/latest/user-guide/tasks/manage-pkgs.html).
