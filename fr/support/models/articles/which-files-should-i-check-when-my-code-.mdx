---
title: "Quels fichiers dois-je vérifier lorsque mon code plante ?"
keywords: ["Journaux"]
---

Pour le run concerné, consultez `debug.log` et `debug-internal.log` dans `wandb/run-[DATE]_[TIME]-[RUN-ID]/logs`, dans le répertoire où votre code s’exécute.
