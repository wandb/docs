---
title: "Quels fichiers dois-je vérifier lorsque mon code plante ?"
keywords: ["Journaux"]

Pour le run concerné, consultez `debug.log` et `debug-internal.log` dans `wandb/run-<date>_<time>-<run-id>/logs`, dans le répertoire où s’exécute votre code.
