---
title: "Comment puis-je utiliser le paramètre `resume` lors de la reprise d’un run dans W&B ?"
keywords: ["Reprise"]
---

Pour utiliser le paramètre `resume` dans W&amp;B, définissez l’argument `resume` dans `wandb.init()` en spécifiant `entity`, `project` et `id`. L’argument `resume` accepte les valeurs `"must"` ou `"allow"`.

```python
  run = wandb.init(entity="your-entity", project="your-project", id="your-run-id", resume="must")
```
