---
title: "Comment puis-je configurer les alertes et les notifications W&B ?"
keywords: ["Alertes", "Experiments", "Exécutions"]
---

Vous pouvez configurer les alertes et les notifications à l’aide de la page **Settings** de W&amp;B. W&amp;B prend en charge les alertes automatisées dans les Paramètres utilisateur (run finished, run crashed), ainsi que les alertes programmatiques à partir du code d’entraînement. Pour savoir comment configurer `run.alert()`, connecter Slack et consulter des exemples, voir [Envoyer une alerte](/fr/models/runs/alert). Pour les notifications Slack déclenchées par des événements sur les artifacts ou les métriques, voir [Automatisations](/fr/models/automations/).
