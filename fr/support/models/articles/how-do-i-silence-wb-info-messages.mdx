---
title: "Comment puis-je désactiver les messages d’information de W&B ?"
keywords: ["notebooks", "Variables d’environnement"]
---

Pour ne plus afficher les messages du journal dans votre notebook, comme celui-ci :

```
INFO SenderThread:11484 [sender.py:finish():979]
```

Définissez le niveau de journalisation sur `logging.ERROR` pour n’afficher que les erreurs et masquer les messages de niveau info.

```python
import logging

logger = logging.getLogger("wandb")
logger.setLevel(logging.ERROR)
```

Pour réduire considérablement la sortie du journal, définissez la variable d’Environnement `WANDB_QUIET` sur `True`. Pour désactiver complètement la sortie du journal, définissez la variable d’Environnement `WANDB_SILENT` sur `True`. Dans un notebook, définissez `WANDB_QUIET` ou `WANDB_SILENT` avant d’exécuter `wandb.login` :

<Tabs>
  <Tab title="Notebook">
    ```python
    %env WANDB_SILENT=True
    ```
  </Tab>

  <Tab title="Python">
    ```python
    import os

    os.environ["WANDB_SILENT"] = "True"
    ```
  </Tab>
</Tabs>
