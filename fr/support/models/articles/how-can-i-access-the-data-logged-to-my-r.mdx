---
title: "Comment accéder directement et par programmation aux données enregistrées dans mes runs ?"
keywords: ["Experiments"]
---

L’objet historique permet de suivre les métriques enregistrées avec `wandb.log`. Accédez à l’objet historique à l’aide de l’API :

```python
api = wandb.Api()
run = api.run("username/project/run_id")
print(run.history())
```
