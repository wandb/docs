title: "Comment puis-je obtenir le nom aléatoire du run dans mon script ?"
keywords: ["Experiments"]

Appelez la méthode `.save()` d&#39;un objet run pour enregistrer le run en cours. Récupérez ensuite le nom via l&#39;attribut `name` de l&#39;objet run.
