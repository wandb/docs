---
title: "Que se passe-t-il si la connexion Internet est interrompue pendant l'entraînement d'un modèle ?"
keywords: ["Variables d'environnement", "Panne"]
---

Si la bibliothèque ne peut pas se connecter à Internet, elle entre dans une boucle de tentatives et continue d&#39;essayer de transmettre les métriques jusqu&#39;à ce que le réseau soit rétabli. Le programme continue de s&#39;exécuter pendant ce temps.

Pour exécuter votre programme sur une machine sans accès à Internet, définissez `WANDB_MODE=offline`. Cette configuration stocke les métriques localement sur le disque dur. Ensuite, exécutez `wandb sync DIRECTORY` pour envoyer les données au serveur.
