---
title: "Comment récupérer ces ID de version et ETags dans W&amp;B ?"
keywords: ["Artifacts"]
---

Si une référence d’artefact est enregistrée dans W&amp;B et que la gestion des versions est activée sur les buckets, les ID de version s’affichent dans l’interface utilisateur d’Amazon S3. Pour récupérer ces ID de version et ETags dans W&amp;B, récupérez l’artefact, puis accédez aux entrées de manifeste correspondantes. Par exemple :

```python
artifact = run.use_artifact("my_table:latest")
for entry in artifact.manifest.entries.values():
    versionID = entry.extra.get("versionID")
    etag = entry.extra.get("etag")
```
