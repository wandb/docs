---
title: "Comment wandb transmet-il les journaux et écrit-il sur le disque ?"
keywords: ["Variables d'environnement"]
---

W&amp;B met les événements en file d&#39;attente en mémoire et les écrit sur le disque de manière asynchrone afin de gérer les pannes et de prendre en charge la configuration `WANDB_MODE=offline`, ce qui permet une synchronisation après la journalisation.

Dans le terminal, observez le chemin vers le répertoire local du run. Ce répertoire inclut un fichier `.wandb`, qui fait office de magasin de données. Pour la journalisation des images, W&amp;B stocke les images dans le sous-répertoire `media/images` avant de les téléverser vers le stockage dans le cloud.
