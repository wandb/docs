La formule de moyenne mobile exponentielle correspond à celle utilisée dans TensorBoard.

Pour plus de détails sur l’implémentation Python équivalente, se référer à cette [explication sur Stack OverFlow](https://stackoverflow.com/questions/42281844/what-is-the-mathematics-behind-the-smoothing-parameter-in-tensorboards-scalar/75421930#75421930). Le code source de l’algorithme de lissage de TensorBoard (à l’heure où nous écrivons ces lignes) se trouve [ici](https://github.com/tensorflow/tensorboard/blob/34877f15153e1a2087316b9952c931807a122aa7/tensorboard/components/vz_line_chart2/line-chart.ts#L699).
