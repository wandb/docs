---
title: "Comment puis-je enregistrer le commit Git associé à mon run ?"
keywords: ["Experiments"]
---

Lorsque `wandb.init` est appelé, le système collecte automatiquement les informations Git, notamment le lien du dépôt distant et le SHA du dernier commit. Ces informations apparaissent sur la [page du run](/fr/models/runs/#view-logged-runs). Pour les afficher, assurez-vous que le répertoire de travail courant au moment de l’exécution du script se trouve dans un dossier géré par Git.

Le commit Git et la commande utilisée pour exécuter l’expérience restent visibles pour l’utilisateur, mais sont masqués aux utilisateurs externes. Dans les projets publics, ces détails restent privés.
