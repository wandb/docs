---
title: "Comment enregistrer le commit Git associé à mon run ?"
keywords: ["Experiments"]
---

Lorsque vous appelez `wandb.init`, W&amp;B collecte automatiquement les informations Git, y compris le lien du dépôt distant et le SHA du commit le plus récent. Ces informations apparaissent sur la [page du run](/fr/models/runs/#view-logged-runs). Pour les afficher, assurez-vous que le répertoire de travail courant au moment d’exécuter le script se trouve dans un dossier géré par Git.

Vous pouvez voir le commit Git et la commande utilisée pour exécuter l’expérience, mais ils sont masqués pour les utilisateurs externes, même dans les projets publics.
