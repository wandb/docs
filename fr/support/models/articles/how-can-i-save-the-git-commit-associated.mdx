---
title: "Comment enregistrer le commit Git associé à mon run ?"
keywords: ["Experiments"]
---

Lors de l’appel à `wandb.init`, le système collecte automatiquement les informations Git, y compris le lien du dépôt distant et le SHA du commit le plus récent. Ces informations apparaissent sur la [page du run](/fr/models/runs/#view-logged-runs). Pour les afficher, assurez-vous que le répertoire de travail courant au moment d’exécuter le script se trouve dans un dossier géré par Git.

Le commit Git et la commande utilisée pour exécuter l’expérience restent visibles pour l’utilisateur, mais sont masqués pour les utilisateurs externes. Dans les projets publics, ces informations restent privées.
