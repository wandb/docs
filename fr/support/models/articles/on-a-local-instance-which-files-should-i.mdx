---
title: "Sur une instance locale, quels fichiers vérifier en cas de problème ?"
keywords: ["Administrateur"]
---

Lorsque vous résolvez des problèmes sur une instance locale W&amp;B, commencez par récupérer le **Debug Bundle**. Ce bundle contient les journaux et les informations système dont vous avez besoin pour diagnostiquer les problèmes.

Pour télécharger le bundle, un administrateur ouvre la page `/system-admin`, sélectionne l’icône W&amp;B dans le coin supérieur droit, puis choisit **Debug Bundle**.
