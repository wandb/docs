---
title: "Peut-on regrouper des runs par tags ?"
keywords: ["Runs"]
---

Un run peut avoir plusieurs tags, le regroupement par tags n&#39;est donc pas pris en charge. Ajoutez une valeur à l&#39;objet [`config`](/fr/models/track/config) pour ces runs, puis regroupez-les plutôt selon cette valeur de configuration. Vous pouvez [définir la configuration une fois votre run terminé](/fr/models/track/config#set-the-configuration-after-your-run-has-finished) via l&#39;API.
