---
title: "Comment puis-je passer d’un compte à l’autre sur la même machine ?"
keywords: ["Variables d’environnement"]
---

Pour gérer deux comptes W&amp;B depuis la même machine, stockez les deux clés API dans un fichier. Utilisez le code suivant dans vos dépôts pour passer d’une clé à l’autre en toute sécurité, afin d’éviter que des clés secrètes ne soient incluses dans le contrôle de version.

```python
if os.path.exists("~/keys.json"):
    os.environ["WANDB_API_KEY"] = json.loads("~/keys.json")["work_account"]
```
