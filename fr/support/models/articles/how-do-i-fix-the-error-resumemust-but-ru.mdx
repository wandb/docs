---
title: "Comment puis-je résoudre l'erreur `resume='must' but run (<run_id>) doesn't exist` ?"
keywords: ["Reprise", "Runs"]
---

Si vous rencontrez l&#39;erreur `resume='must' but run (<run_id>) doesn't exist`, cela signifie que le run que vous essayez de reprendre n&#39;existe pas dans le projet ou l&#39;entité. Assurez-vous d&#39;être connecté à la bonne instance et que le projet ainsi que l&#39;entité sont correctement définis :

```python
wandb.init(entity=<entity>, project=<project>, id=<run-id>, resume='must')
```

Exécutez [`wandb login --relogin`](/fr/models/ref/cli/wandb-login) pour vérifier que vous êtes bien authentifié.
