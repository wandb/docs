---
title: "Comment puis-je résoudre l'erreur `resume='must' but run ([RUN-ID]) doesn't exist` ?"
keywords: ["Reprise", "Runs"]
---

Si vous rencontrez l&#39;erreur `resume='must' but run ([RUN-ID]) doesn't exist`, cela signifie que le run que vous essayez de reprendre n&#39;existe pas dans le projet ou l&#39;entité. Assurez-vous d&#39;être connecté à la bonne instance et que le projet ainsi que l&#39;entité sont correctement définis. Remplacez `[ENTITY]`, `[PROJECT]` et `[RUN-ID]` par votre entité, votre projet et votre ID du run :

```python
wandb.init(entity=[ENTITY], project=[PROJECT], id=[RUN-ID], resume='must')
```

Pour vérifier que vous êtes bien authentifié, exécutez [`wandb login --relogin`](/fr/models/ref/cli/wandb-login).
