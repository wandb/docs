Pour reprendre un balayage, passez le `sweep_id` à la fonction `wandb.agent()`.

```python
import wandb

sweep_id = "your_sweep_id"

def train():
    # Code d'entraînement ici
    pass

wandb.agent(sweep_id=sweep_id, function=train)
```
