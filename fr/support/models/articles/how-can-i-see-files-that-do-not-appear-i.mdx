---
title: "Comment voir les fichiers qui n'apparaissent pas dans l'onglet Files ?"
keywords: ["Experiments"]
---

L&#39;onglet Files affiche jusqu&#39;à 10 000 fichiers. Pour télécharger tous les fichiers, utilisez l&#39;[API publique](/fr/models/ref/python/public-api/api) :

```python
import wandb

api = wandb.Api()
run = api.run('<entity>/<project>/<run_id>')
run.file('<file>').download()

for f in run.files():
    if <condition>:
        f.download()
```
