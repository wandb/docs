---
title: "InitStartError : erreur de communication avec le processus wandb"
keywords: ["Experiments", "Plantage de run"]
---

Cette erreur indique que la bibliothèque rencontre un problème lors du lancement du processus qui synchronise les données avec le serveur.

Les solutions de contournement suivantes permettent de résoudre le problème dans certains environnements :

<Tabs>
  <Tab title="Linux et OS X">
    ```python
    wandb.init(settings=wandb.Settings(start_method="fork"))
    ```
  </Tab>

  <Tab title="Google Colab">
    Pour les versions antérieures à `0.13.0`, utilisez :

    ```python
    wandb.init(settings=wandb.Settings(start_method="thread"))
    ```
  </Tab>
</Tabs>
