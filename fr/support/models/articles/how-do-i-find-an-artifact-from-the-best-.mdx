---
title: "Comment puis-je trouver un artifact provenant du meilleur run d’un balayage ?"
keywords: ["Artifacts"]
---

Pour récupérer les Artifacts du run le plus performant d’un balayage, utilisez le code suivant :

```python
api = wandb.Api()
sweep = api.sweep("entity/project/sweep_id")
runs = sorted(sweep.runs, key=lambda run: run.summary.get("val_acc", 0), reverse=True)
best_run = runs[0]
for artifact in best_run.logged_artifacts():
    artifact_path = artifact.download()
    print(artifact_path)
```
