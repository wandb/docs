---
title: "Comment puis-je modifier les couleurs de chaque run dans un même groupe ?"
keywords: ["Runs", "Workspaces"]
---

Il n’est pas possible de modifier les couleurs des runs individuels au sein d’un groupe. Tous les runs d’un même groupe partagent la même couleur.
