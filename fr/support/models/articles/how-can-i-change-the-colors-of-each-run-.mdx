---
title: "Comment changer les couleurs de chaque run d’un même groupe ?"
keywords: ["Runs", "Workspaces"]
---

Il n’est pas possible de modifier la couleur de runs individuels au sein d’un groupe. Tous les runs d’un même groupe partagent la même couleur.
