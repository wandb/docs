---
title: "Comment puis-je remplacer le nom du run par son ID ?"
keywords: ["Experiments"]
---

Pour remplacer le nom du run par l’ID du run, utilisez l’extrait de code suivant :

```python
import wandb

with wandb.init() as run:
   run.name = run.id
   run.save()
```
