---
title: "Comment corriger une erreur du type `AttributeError: module 'wandb' has no attribute ...` ?"
keywords: ["Échecs de run"]
---

Si vous rencontrez une erreur telle que `AttributeError: module 'wandb' has no attribute 'init'` ou `AttributeError: module 'wandb' has no attribute 'login'` lorsque vous importez `wandb` en Python, cela signifie que `wandb` n’est pas installé ou que l’installation est corrompue, mais qu’un répertoire `wandb` existe dans le répertoire de travail actuel. Pour corriger cette erreur, désinstallez `wandb`, supprimez le répertoire, puis réinstallez `wandb` :

```bash
pip uninstall wandb; rm -rI wandb; pip install wandb
```
