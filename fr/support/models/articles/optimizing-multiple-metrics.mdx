title: "Optimisation de métriques multiples"
keywords: ["Sweeps", "Métriques"]
---

Pour optimiser plusieurs métriques dans un seul run, utilisez la somme pondérée des métriques individuelles.

```python
with wandb.init() as run:
  # Journaliser les métriques individuelles
  metric_a = run.summary.get("metric_a", 0.5)
  metric_b = run.summary.get("metric_b", 0.7)
  # ... journaliser d'autres métriques si nécessaire
  metric_n = run.summary.get("metric_n", 0.9)

  # Combiner les métriques avec des poids
  # Ajuster les poids selon vos objectifs d'optimisation
  # Par exemple, pour donner plus d'importance à metric_a et metric_n :  
  metric_combined = 0.3 * metric_a + 0.2 * metric_b + ... + 1.5 * metric_n
  run.log({"metric_combined": metric_combined})
```

Journalisez la nouvelle métrique combinée et définissez-la comme objectif d’optimisation :

```yaml
metric:
  name: metric_combined
  goal: minimize
```
