---
title: "Comment puis-je enregistrer un artefact dans un run existant ?"
keywords: ["Artifacts"]
---

Il est parfois nécessaire d’indiquer qu’un artefact est la sortie d’un run précédemment enregistré. Dans ce cas, réinitialisez l’ancien run et enregistrez de nouveaux artefacts comme suit :

```python
with wandb.init(id="existing_run_id", resume="allow") as run:
    artifact = wandb.Artifact("artifact_name", "artifact_type")
    artifact.add_file("my_data/file.txt")
    run.log_artifact(artifact)
```
