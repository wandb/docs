---
title: "Est-ce que cela fonctionne uniquement pour Python ?"
keywords: ["Python"]
---

La bibliothèque prend en charge Python 2.7 et versions ultérieures, ainsi que Python 3.6 et versions ultérieures. Son architecture facilite l’intégration avec d’autres langages de programmation. Pour surveiller d’autres langages, contactez [contact@wandb.com](mailto:contact@wandb.com).
