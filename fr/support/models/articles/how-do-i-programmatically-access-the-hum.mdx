---
title: "Comment puis-je accéder par programmation au nom lisible de l’exécution ?"
keywords: ["Experiments"]
---

L’attribut `.name` d’un [`wandb.Run`](/fr/models/ref/python/experiments/run) est accessible comme suit :

```python
import wandb

with wandb.init() as run:
   run_name = run.name
   print(f"The human-readable run name is: {run_name}")
```
