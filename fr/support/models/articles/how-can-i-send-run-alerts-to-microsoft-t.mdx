---
title: "Comment envoyer des alertes de run à Microsoft Teams ?"
keywords: ["Alertes"]
---

W&amp;B n&#39;envoie pas directement d&#39;alertes à Microsoft Teams, mais vous pouvez les acheminer par e-mail afin que votre équipe voie les notifications de run dans un canal partagé. Pour recevoir des alertes W&amp;B dans Teams, suivez ces étapes :

1. **Configurez une adresse e-mail pour votre canal Teams.** Créez une adresse e-mail pour le canal Teams dans lequel vous souhaitez recevoir des alertes.
2. **Transférez les e-mails d&#39;alerte W&amp;B à l&#39;adresse e-mail du canal Teams.** Configurez W&amp;B pour envoyer des alertes par e-mail, puis transférez ces e-mails vers l&#39;adresse e-mail de votre canal Teams.

Une fois ces étapes terminées, les alertes de run W&amp;B arrivent dans votre canal Teams sous forme de messages transférés par e-mail.
