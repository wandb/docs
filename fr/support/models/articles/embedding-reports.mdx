Cette page décrit comment partager un rapport W&amp;B en l’intégrant dans une autre page, comme un article de blog, un wiki interne ou un site web externe. L’intégration permet aux lecteurs de consulter votre rapport dans son contexte sans quitter la page hôte.

En haut à droite de votre rapport, cliquez sur **Share**. En bas de la fenêtre pop-up, copiez le code d’intégration. Collez ce code dans le HTML de la page où vous souhaitez que le rapport apparaisse.
