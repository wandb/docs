---
title: "Puis-je simplement attribuer le run ID comme nom de l'exécution ?"
keywords: ["Experiments"]
---

Oui. Pour remplacer le nom de l&#39;exécution par le run ID, utilisez l&#39;extrait de code suivant :

```python
import wandb

with wandb.init() as run:
   run.name = run.id
   run.save()
```
