---
title: "Puis-je simplement définir le nom du run comme l’ID du run ?"
keywords: ["Experiments"]
---

Oui. Pour remplacer le nom du run par l’ID du run, utilisez l’extrait de code suivant :

```python
import wandb

with wandb.init() as run:
   run.name = run.id
   run.save()
```
