---
title: "Quelle est la différence entre les modes de wandb.init ?"
keywords: ["Experiments"]
---

Ces modes sont disponibles :

* `online` (par défaut) : Le client envoie les données au serveur wandb.
* `offline` : Le client stocke les données localement sur la machine au lieu de les envoyer au serveur wandb. Utilisez la commande [`wandb sync`](/fr/models/ref/cli/wandb-sync) pour synchroniser les données ultérieurement.
* `disabled` : Le client simule le fonctionnement en renvoyant des objets factices et empêche toute communication réseau. Toute journalisation est désactivée, mais toutes les méthodes d’API factices restent appelables. Ce mode est généralement utilisé pour les tests.
