---
title: "L’état de mon run est `crashed` dans l’interface, mais il tourne toujours sur ma machine. Que dois-je faire pour récupérer mes données ?"
keywords: ["Experiments", "Crashs de run"]
---

Vous avez probablement perdu la connexion à votre machine pendant l’entraînement. Récupérez vos données en exécutant [`wandb sync [PATH_TO_RUN]`](/fr/models/ref/cli/wandb-sync). Le chemin de votre run correspond à un dossier du répertoire `wandb` dont le nom correspond à l’ID du run en cours.
