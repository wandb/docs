---
title: "L'état de mon run est `crashed` dans l'UI alors qu'il s'exécute toujours sur ma machine. Que faire pour récupérer mes données ?"
keywords: ["Experiments", "Run Crashes"]
---

Cet état signifie généralement que vous avez perdu la connexion à votre machine pendant l&#39;entraînement. Pour récupérer vos données, exécutez [`wandb sync [PATH_TO_RUN]`](/fr/models/ref/cli/wandb-sync). Le chemin de votre run correspond au dossier de votre répertoire `wandb` dont le nom correspond à l&#39;ID du run en cours.
