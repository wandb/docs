---
title: "L'état de mon run est `crashed` dans l'UI alors qu'il s'exécute toujours sur ma machine. Que faire pour récupérer mes données ?"
keywords: ["Experiments", "Run Crashes"]
---

Vous avez probablement perdu la connexion à votre machine pendant l&#39;entraînement. Récupérez vos données en exécutant [`wandb sync [PATH_TO_RUN]`](/fr/models/ref/cli/wandb-sync). Le chemin de votre run correspond à un dossier de votre répertoire `wandb` dont le nom correspond au Run ID du run en cours.
