---
title: "Pourquoi la même métrique apparaît-elle plus d'une fois ?"
keywords: ["Experiments", "métriques"]
---

Lorsque vous enregistrez différents types de données sous la même clé, ils sont séparés dans la base de données. Cela entraîne plusieurs entrées portant le même nom de métrique dans la liste déroulante de l&#39;UI. Les types de données séparés sont `number`, `string`, `bool`, `other` (principalement des tableaux), ainsi que tout type de données `wandb` comme `Histogram` ou `Image`. Pour éviter ce problème, n&#39;envoyez qu&#39;un seul type par clé.

Les noms de métriques ne distinguent pas les majuscules des minuscules. Évitez d&#39;utiliser des noms qui ne diffèrent que par la casse, comme `"My-Metric"` et `"my-metric"`.
