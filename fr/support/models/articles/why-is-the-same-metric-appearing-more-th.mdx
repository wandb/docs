---
title: "Pourquoi la même métrique apparaît-elle plus d’une fois ?"
keywords: ["Experiments", "Métriques"]
---

Lorsque vous journalisez différents types de données sous une même clé, ils sont séparés dans la base de données. Cela entraîne plusieurs entrées portant le même nom de métrique dans le menu déroulant de l’interface utilisateur. Les types de données regroupés sont `number`, `string`, `bool`, `other` (principalement des tableaux), ainsi que tout type de données `wandb`, comme `Histogram` ou `Image`. N’envoyez qu’un seul type par clé pour éviter ce problème.

Les noms de métriques ne tiennent pas compte de la casse. Évitez d’utiliser des noms qui ne diffèrent que par la casse, comme `"My-Metric"` et `"my-metric"`.
