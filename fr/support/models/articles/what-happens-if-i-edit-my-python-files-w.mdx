---
title: "Que se passe-t-il si je modifie mes fichiers Python pendant l’exécution d’un sweep ?"
keywords: ["Sweeps"]
---

Pendant l’exécution d’un sweep :

* Si le script `train.py` utilisé par le sweep est modifié, le sweep continue d’utiliser le fichier `train.py` original
* Si des fichiers référencés par le script `train.py` sont modifiés, par exemple des fonctions utilitaires dans le script `helper.py`, le sweep se met à utiliser la version mise à jour de `helper.py`.
