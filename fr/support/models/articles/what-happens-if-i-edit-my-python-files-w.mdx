---
title: "Que se passe-t-il si je modifie mes fichiers Python pendant qu’un balayage est en cours d’exécution ?"
keywords: ["Sweeps"]
---

Pendant qu’un balayage est en cours d’exécution :

* Si le script `train.py` qu’utilise le balayage change, le balayage continue d’utiliser le `train.py` original
* Si les fichiers auxquels le script `train.py` fait référence changent, comme les fonctions d’assistance dans le script `helper.py`, le balayage commence à utiliser le `helper.py` mis à jour.
