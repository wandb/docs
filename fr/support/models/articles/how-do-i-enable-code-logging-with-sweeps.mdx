---
title: "Comment puis-je activer la journalisation du code avec Sweeps ?"
keywords: ["Sweeps"]
---

Pour activer la journalisation du code avec Sweeps, ajoutez `wandb.log_code()` après avoir initialisé le run W&amp;B. Cette action est nécessaire même lorsque la journalisation du code est activée dans les paramètres du profil W&amp;B. Pour une journalisation avancée du code, référez-vous à la [documentation de `wandb.log_code()` ici](/fr/models/ref/python/experiments/run#log_code).
