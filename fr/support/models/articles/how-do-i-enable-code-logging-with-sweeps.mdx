---
title: "Comment puis-je activer la journalisation du code avec Sweeps ?"
keywords: ["Sweeps"]
---

Pour activer la journalisation du code avec Sweeps, ajoutez `wandb.log_code()` après avoir initialisé le run W&amp;B. Cette étape est nécessaire même lorsque vous activez la journalisation du code dans les paramètres du profil W&amp;B. Pour la journalisation avancée du code, voir la [référence de `wandb.log_code()`](/fr/models/ref/python/experiments/run#log_code).
