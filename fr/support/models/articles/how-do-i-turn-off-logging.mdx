---
title: "Comment puis-je désactiver la journalisation ?"
keywords: ["Journaux"]
---

La commande `wandb offline` définit la variable d&#39;environnement `WANDB_MODE=offline`, ce qui empêche la synchronisation des données avec le serveur W&amp;B distant. Cette action s&#39;applique à tous les projets et empêche l&#39;envoi des données de journalisation aux serveurs W&amp;B.

Pour supprimer les messages d&#39;avertissement, utilisez le code suivant :

```python
import logging

logger = logging.getLogger("wandb")
logger.setLevel(logging.WARNING)
```
