---
title: "Comment puis-je désactiver la journalisation ?"
keywords: ["Journaux"]
---

La commande `wandb offline` définit la variable d’environnement `WANDB_MODE=offline`, ce qui empêche la synchronisation des données avec le serveur W&amp;B distant. Cette action s’applique à tous les projets et interrompt la journalisation des données vers les serveurs W&amp;B.

Pour supprimer les messages d’avertissement, utilisez le code suivant :

```python
import logging

logger = logging.getLogger("wandb")
logger.setLevel(logging.WARNING)
```
