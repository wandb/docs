---
title: "Comment une personne sans compte peut-elle voir les résultats d'un run ?"
keywords: ["Anonyme"]
---

Si quelqu&#39;un exécute le script avec `anonymous="allow"` :

1. **Créer automatiquement un compte temporaire** : W&amp;B vérifie si un compte est déjà connecté. Si ce n&#39;est pas le cas, W&amp;B crée un nouveau compte anonyme et enregistre la clé API pour cette session.
2. **Enregistrer rapidement les résultats** : les utilisateurs peuvent exécuter le script à plusieurs reprises et voir instantanément les résultats dans le tableau de bord W&amp;B. Ces runs anonymes non rattachés restent disponibles pendant 7 jours.
3. **Rattacher les données lorsqu&#39;elles deviennent utiles** : dès qu&#39;un utilisateur identifie des résultats utiles dans W&amp;B, il peut cliquer sur un bouton dans la bannière en haut de la page pour enregistrer les données de son run dans un vrai compte. Sans rattachement, les données du run sont supprimées après 7 jours.

<Warning>
  **Les liens vers des runs anonymes sont sensibles**. Ces liens permettent à n&#39;importe qui de voir et de rattacher des résultats d&#39;expérience pendant 7 jours. Ne partagez donc ces liens qu&#39;avec des personnes de confiance. Pour partager publiquement des résultats tout en masquant l&#39;identité de l&#39;auteur, contactez support@wandb.com pour obtenir de l&#39;assistance.
</Warning>

Lorsqu&#39;un utilisateur W&amp;B découvre puis exécute le script, ses résultats sont correctement enregistrés dans son compte, comme pour un run normal.
