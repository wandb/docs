---
title: "W&B utilise-t-il la bibliothèque `multiprocessing` ?"
keywords: ["Experiments"]
---

Oui, W&amp;B utilise la bibliothèque `multiprocessing`. Un message d’erreur comme celui-ci indique un problème potentiel :

```
An attempt has been made to start a new process before the current process 
has finished its bootstrapping phase.
```

Pour résoudre ce problème, ajoutez une protection du point d’entrée avec `if __name__ == "__main__":`. Cette protection est nécessaire lorsque vous lancez W&amp;B directement depuis le script.
