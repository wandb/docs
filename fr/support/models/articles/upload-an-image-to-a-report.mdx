---
title: "Importer une image dans un rapport"
keywords: ["Reports"]
---

Pour téléverser une image dans un rapport :

1. Sur une nouvelle ligne, appuyez sur `/`.
2. Faites défiler jusqu’à l’option **Image**.
3. Faites glisser-déposer une image dans le rapport.
