Le mode sombre est en version bêta et n&#39;est pas optimisé pour l&#39;accessibilité. Pour l&#39;activer :

1. Accédez aux [W&amp;B account settings](https://wandb.ai/settings).
2. Faites défiler jusqu&#39;à la section **Public preview features**.
3. Dans **UI Display**, sélectionnez **Dark mode** dans le menu déroulant.
