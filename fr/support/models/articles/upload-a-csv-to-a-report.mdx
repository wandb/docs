---
title: "Importer un CSV dans un rapport"
keywords: ["Reports"]
---

Pour importer un CSV dans un rapport, utilisez le format `wandb.Table`. Chargez le CSV dans votre script Python et enregistrez-le sous forme d’objet `wandb.Table`. Cette action affiche les données sous forme de tableau dans le rapport.
