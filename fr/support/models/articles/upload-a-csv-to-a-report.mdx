---
title: "Téléverser un CSV dans un rapport"
keywords: ["Reports"]
---

Pour téléverser un CSV dans un rapport, utilisez le format `wandb.Table`. Chargez le CSV dans votre script Python, puis journalisez-le sous forme d&#39;objet `wandb.Table`. Cette action affiche les données sous forme de tableau dans le rapport.
