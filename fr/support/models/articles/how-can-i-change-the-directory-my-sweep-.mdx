---
title: "Comment modifier le répertoire local dans lequel mon balayage écrit ses journaux ?"
keywords: ["Sweeps"]
---

Configurez la variable d’environnement `WANDB_DIR` pour définir le répertoire de journalisation des données de run W&amp;B. Par exemple :

```python
os.environ["WANDB_DIR"] = os.path.abspath("your/directory")
```
