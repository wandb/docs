---
title: "Comment modifier le répertoire local dans lequel mon balayage enregistre ses journaux ?"
keywords: ["Sweeps"]
---

Définissez le répertoire de journalisation des données de run W&amp;B en configurant la variable d&#39;environnement `WANDB_DIR`. Par exemple :

```python
os.environ["WANDB_DIR"] = os.path.abspath("your/directory")
```
