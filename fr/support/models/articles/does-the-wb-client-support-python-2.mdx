---
title: "Le client W&B prend-il en charge Python 2 ?"
keywords: ["Python"]
---

La bibliothèque cliente W&amp;B prenait en charge Python 2.7 et Python 3 jusqu’à la version 0.10 incluse. La prise en charge de Python 2.7 a pris fin avec la version 0.11 en raison de la fin de vie de Python 2. L’exécution de `pip install --upgrade wandb` sur un système Python 2.7 n’installe que les nouvelles versions de la série 0.10.x. La prise en charge de la série 0.10.x se limite aux correctifs de bugs critiques et aux patchs. La dernière version de la série 0.10.x qui prend en charge Python 2.7 est 0.10.33.
