---
title: "Le client W&B prend-il en charge Python 2 ?"
keywords: ["Python"]
---

La bibliothèque cliente W&amp;B prenait en charge Python 2.7 et Python 3 jusqu’à la version 0.10. La version 0.11 a supprimé la prise en charge de Python 2.7 en raison de la fin de vie de Python 2. Sur un système Python 2.7, `pip install --upgrade wandb` n’installe que les nouvelles versions de la série 0.10.x, qui inclut uniquement les correctifs de bug critiques et les patchs. La dernière version de la série 0.10.x à prendre en charge Python 2.7 est la 0.10.33.
