---
title: "Le client W&B prend-il en charge Python 2 ?"
keywords: ["Python"]
---

La bibliothèque cliente W&amp;B prenait en charge Python 2.7 et Python 3 jusqu’à la version 0.10. La prise en charge de Python 2.7 a été interrompue avec la version 0.11 en raison de la fin de vie de Python 2. L’exécution de `pip install --upgrade wandb` sur un système Python 2.7 n’installe que les nouvelles versions de la série 0.10.x. La prise en charge de la série 0.10.x inclut uniquement les correctifs de bug critiques et les patchs. La dernière version de la série 0.10.x à prendre en charge Python 2.7 est la 0.10.33.
