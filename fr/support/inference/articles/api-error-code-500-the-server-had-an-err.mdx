title: "Code d'erreur API 500 - Le serveur a rencontré une erreur lors du traitement de votre requête"
keywords: ["Erreurs du serveur"]
---

Une erreur 500 accompagnée du message &quot;Le serveur a rencontré une erreur lors du traitement de votre requête&quot; indique une erreur interne du serveur du côté de W&amp;B Inference.

<div id="why-this-happens">
  ## Pourquoi cela se produit
</div>

Les erreurs internes du serveur sont généralement temporaires et dues à des problèmes passagers côté serveur. Elles ne sont pas causées par votre requête ni par votre configuration.

<div id="what-you-can-do">
  ## Ce que vous pouvez faire
</div>

1. **Réessayez après une courte attente**
   * Attendez quelques secondes, puis réessayez votre requête
   * Utilisez une temporisation exponentielle pour les tentatives automatisées

2. **Contactez l’assistance si le problème persiste**
   * Si des erreurs 500 continuent de s’afficher après des tentatives multiples, contactez l’[assistance W&amp;B](mailto:support@wandb.com) avec les détails de votre requête
