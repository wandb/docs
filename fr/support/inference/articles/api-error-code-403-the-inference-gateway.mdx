---
title: "Code d’erreur API 403 - La passerelle d’inférence n’est pas activée pour votre organisation"
keywords: ["Authentification et accès", "Administrateur"]
---

Une erreur 403 accompagnée du message &quot;The inference gateway is not enabled for your organization&quot; signifie que la passerelle d’inférence n’est pas activée pour votre organisation, ce qui est nécessaire pour utiliser W&amp;B Inference.

<div id="why-this-happens">
  ## Pourquoi cela arrive
</div>

W&amp;B Inference nécessite que la passerelle d’inférence soit activée au niveau de l’organisation. Si votre organisation n’a pas activé cette fonctionnalité, toutes les requêtes d’API seront rejetées avec une erreur 403.

<div id="what-you-can-do">
  ## Ce que vous pouvez faire
</div>

1. **Contactez votre administrateur W&amp;B**
   * Demandez à l’administrateur W&amp;B de votre organisation d’activer la passerelle d’inférence

2. **Contactez l’assistance W&amp;B**
   * Si vous ne savez pas qui est votre administrateur ou si vous avez besoin d’aide pour activer la passerelle, contactez l’[assistance W&amp;B](mailto:support@wandb.com)
