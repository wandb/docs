---
title: "Code d’erreur API 403 : La passerelle d’inférence n’est pas activée pour votre organisation"
keywords: ["Authentification et accès", "Administrateur"]
---

Une erreur `403` accompagnée du message `The inference gateway is not enabled for your organization` signifie que la passerelle d’inférence n’est pas activée pour votre organisation, ce qui est nécessaire pour utiliser Serverless Inference. Cet article explique pourquoi cette erreur se produit et comment activer la passerelle.

<div id="why-this-happens">
  ## Pourquoi cela arrive
</div>

Serverless Inference nécessite que la passerelle d’inférence soit activée au niveau de l’organisation. Si votre organisation n’a pas activé cette fonctionnalité, la passerelle rejette toutes les requêtes d’API avec une erreur `403`.

<div id="what-you-can-do">
  ## Ce que vous pouvez faire
</div>

* **Contactez votre administrateur W&amp;B** : Demandez à l’administrateur W&amp;B de votre organisation d’activer la passerelle d’inférence.
* **Contactez l’assistance W&amp;B** : Si vous ne savez pas qui est votre administrateur ou si vous avez besoin d’aide pour activer la passerelle, contactez l’[assistance W&amp;B](mailto:support@wandb.com).
