---
title: "Code d'erreur API 503 - Le moteur est actuellement surchargé"
keywords: ["Erreurs du serveur"]
---

Une erreur 503 accompagnée du message &quot;Le moteur est actuellement surchargé, veuillez réessayer plus tard&quot; signifie que le serveur W&amp;B Inference subit actuellement un trafic important et ne peut pas traiter votre requête pour le moment.

<div id="why-this-happens">
  ## Pourquoi cela arrive
</div>

Pendant les périodes de forte demande, le moteur d’inférence peut être temporairement surchargé. Il s’agit d’une situation temporaire qui se résout généralement d’elle-même à mesure que le trafic diminue.

<div id="what-you-can-do">
  ## Ce que vous pouvez faire
</div>

1. **Réessayez après un court délai**
   * Attendez quelques secondes avant de réessayer
   * Utilisez un backoff exponentiel pour éviter d’accentuer la congestion

2. **Espacez les requêtes**
   * Si vous envoyez de nombreuses requêtes, pensez à les répartir dans le temps
   * Mettez en place une file d’attente des requêtes pour lisser les pics de trafic
