---
title: "Comment puis-je afficher du Markdown dans l’UI ?"
keywords: ["Rendu de l’UI"]
---

Entourez votre chaîne de `weave.Markdown(...)` avant de l’enregistrer, puis utilisez `weave.publish(...)` pour la stocker. Weave utilise le type de l’objet pour déterminer le rendu, et `weave.Markdown` est associé à un moteur de rendu UI connu. La valeur sera affichée dans l’UI sous la forme d’un objet Markdown mis en forme. Pour un exemple de code complet, voir [Affichage des appels](/fr/weave/guides/tracking/tracing#viewing-calls).
