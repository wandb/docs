---
title: "Comment puis-je afficher du Markdown dans l'interface utilisateur ?"
keywords: ["Rendu de l'UI"]
---

Encapsulez votre chaîne avec `weave.Markdown(...)` avant de l&#39;enregistrer, puis utilisez `weave.publish(...)` pour la stocker. Weave utilise le type de l&#39;objet pour déterminer le rendu, et `weave.Markdown` est associé à un moteur de rendu d&#39;interface utilisateur reconnu. La valeur s&#39;affichera dans l&#39;interface utilisateur sous la forme d&#39;un objet Markdown mis en forme. Pour un exemple de code complet, voir [Affichage des appels](/fr/weave/guides/tracking/tracing#viewing-calls).
