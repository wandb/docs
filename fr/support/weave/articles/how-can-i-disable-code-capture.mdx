---
title: "Comment puis-je désactiver la capture de code ?"
keywords: ["Capture de code"]
---

Pour désactiver la capture de code, utilisez l’une des méthodes suivantes :

* Lors de l’initialisation du client W&amp;B Weave, définissez `capture_code` sur `False` : `weave.init("entity/project", settings={"capture_code": False})`.
* Définissez la [variable d’environnement](/fr/weave/guides/core-types/env-vars) `WEAVE_CAPTURE_CODE=false`.
