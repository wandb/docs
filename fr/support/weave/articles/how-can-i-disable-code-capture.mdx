---
title: "Comment puis-je désactiver la capture de code ?"
keywords: ["Capture de code"]
---

Vous pouvez désactiver la capture de code pendant l’initialisation du client Weave : `weave.init("entity/project", settings={"capture_code": False})`.
Vous pouvez aussi utiliser la [variable d’environnement](/fr/weave/guides/core-types/env-vars) `WEAVE_CAPTURE_CODE=false`.
