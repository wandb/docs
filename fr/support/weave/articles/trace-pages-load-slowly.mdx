---
title: "Les pages de trace se chargent lentement"
keywords: ["Données de trace"]
---

Si les pages de trace se chargent lentement, réduisez le nombre de lignes affichées pour améliorer le temps de chargement. La valeur par défaut est `50`. Vous pouvez réduire le nombre de lignes soit via l’interface utilisateur, soit à l’aide de paramètres de requête.

<div id="adjust-via-the-ui-recommended">
  ### Ajuster via l’interface utilisateur (recommandé)
</div>

Utilisez le contrôle **Per page** en bas à droite de la page Traces pour ajuster le nombre de lignes affichées. Outre la valeur par défaut de `50`, vous pouvez aussi choisir `10`, `25` ou `100`.

<div id="use-query-parameters">
  ### Utiliser les paramètres de requête
</div>

Si vous préférez une approche manuelle, vous pouvez modifier le paramètre de requête `pageSize` dans l’URL de la requête pour lui donner une valeur inférieure à `100`.
