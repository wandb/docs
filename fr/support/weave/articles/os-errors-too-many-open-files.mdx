<div id="errno-24-too-many-open-files">
  ### `[Errno 24]: Trop de fichiers ouverts`
</div>

Cette erreur se produit lorsque le nombre de fichiers ouverts dépasse la limite définie par votre système d’exploitation. Dans Weave, cela peut arriver si vous travaillez avec de grands jeux de données d’images. Weave utilise `PIL` pour le traitement des images, ce qui garde les descripteurs de fichiers ouverts pendant toute la durée d’exécution du programme.

Pour résoudre ce problème, augmentez la limite système du nombre de fichiers ouverts à `65,536` à l’aide de `ulimit` :

```bash
ulimit -n 65536
```
