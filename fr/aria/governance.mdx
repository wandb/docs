<Info>
  Les options de contrôle de **Smart features** sont disponibles pour les comptes Enterprise et Pro.
</Info>

Les [administrateurs W&amp;B](/fr/platform/hosting/iam/access-management-intro#manage-users-and-teams-within-an-organization) peuvent gérer l’accès aux fonctionnalités basées sur les LLM et le partage de données afin d’améliorer les performances.

Pour modifier les paramètres de **Smart features** :

1. Accédez à W&amp;B App.
2. Dans le coin supérieur droit de la page, ouvrez le menu déroulant **Menu utilisateur**. Dans la section **Compte** de ce menu, sélectionnez **Paramètres**.
3. Dans l’onglet **Paramètres**, sélectionnez **Confidentialité**.
4. Faites défiler jusqu’à la section **Smart features**.
5. Basculez les options selon vos préférences. Les modifications sont enregistrées automatiquement.
