Le RL sans serveur est pris en charge via le [framework ART d&#39;OpenPipe](https://art.openpipe.ai/getting-started/about) et l&#39;[API W&amp;B Training](/fr/training/api-reference).

Pour commencer à utiliser le RL sans serveur, satisfaisez les [prérequis](/fr/training/prerequisites) pour utiliser les outils W&amp;B, puis suivez le [démarrage rapide](https://art.openpipe.ai/getting-started/quick-start) d&#39;ART.

* Pour des exemples de code et des flux de travail, voir le [notebook Google Colab](https://colab.research.google.com/github/openpipe/art-notebooks/blob/main/examples/2048/2048.ipynb).
* Pour en savoir plus sur les points de terminaison API du RL sans serveur, voir la [référence de l&#39;API W&amp;B Training](/fr/training/api-reference).
