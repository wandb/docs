&quot;Country, region, or territory not supported&quot;라는 메시지와 함께 403 오류가 발생하면, 지원되지 않는 위치에서 W&amp;B Inference에 액세스하고 있다는 뜻입니다.

<div id="why-this-happens">
  ## 이 문제가 발생하는 이유
</div>

W&amp;B Inference는 규정 준수 및 규제 요건으로 인해 지리적 제한이 있습니다. 이 서비스는 지원되는 지리적 위치에서만 사용할 수 있습니다.

<div id="what-you-can-do">
  ## 시도해 볼 수 있는 작업
</div>

1. **지리적 제한 확인**
   * 현재 지원되는 지역 목록은 [지리적 제한](/ko/inference/usage-limits/#geographic-restrictions)에서 확인하세요

2. **지원되는 지역에서 사용**
   * 지원되는 국가 또는 지역에서 서비스에 액세스하세요
   * 지원되는 지역에 있는 조직의 리소스를 사용하는 것도 고려해 보세요

3. **계정 담당 팀에 문의**
   * 엔터프라이즈 고객은 계정 담당자와 가능한 옵션을 논의할 수 있습니다
   * 일부 조직에는 특별한 계약이 적용될 수 있습니다

<div id="error-details">
  ## 오류 Details
</div>

이 오류가 나타나면:

```
{
  "error": {
    "code": 403,
    "message": "Country, region, or territory not supported"
  }
}
```

이는 API 요청 시점의 IP 주소 위치를 기준으로 확인됩니다.
