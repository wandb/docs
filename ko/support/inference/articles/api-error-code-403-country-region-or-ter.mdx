---
title: "API 오류 코드 403 - 지원되지 않는 국가, 지역 또는 영토"
keywords: ["인증 및 액세스"]
---

&quot;Country, region, or territory not supported&quot; 메시지와 함께 403 오류가 발생하면, 이는 지원되지 않는 위치에서 W&amp;B Inference에 액세스하려 한다는 의미입니다.

<div id="why-this-happens">
  ## 이런 일이 발생하는 이유
</div>

W&amp;B Inference에는 규정 준수 및 규제 요구 사항으로 인해 지리적 제한이 있습니다. 이 서비스는 지원되는 지리적 위치에서만 액세스할 수 있습니다.

<div id="what-you-can-do">
  ## 시도해 볼 수 있는 작업
</div>

1. **지리적 제한 확인**
   * 현재 지원되는 지역 목록은 [지리적 제한](/ko/inference/usage-limits/#geographic-restrictions)에서 확인하세요

2. **지원되는 지역에서 사용**
   * 지원되는 국가 또는 지역에서 서비스에 액세스하세요
   * 지원되는 지역에 있는 조직의 리소스를 사용하는 것도 고려해 보세요

3. **아카운트 담당자에게 문의**
   * 엔터프라이즈 고객은 담당 Account Executive와 이용 가능한 옵션을 논의할 수 있습니다
   * 일부 조직에는 특별한 계약이 적용될 수 있습니다

<div id="error-details">
  ## 오류 세부 정보
</div>

이 오류가 표시되면:

```
{
  "error": {
    "code": 403,
    "message": "Country, region, or territory not supported"
  }
}
```

이는 API Request 시점의 IP 주소 위치로 확인됩니다.
