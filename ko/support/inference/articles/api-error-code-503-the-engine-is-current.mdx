---
title: "API 오류 코드 503 - 엔진이 현재 과부하 상태입니다"
keywords: ["서버 오류"]
---

&quot;엔진이 현재 과부하 상태입니다. 나중에 다시 시도해 주세요&quot;라는 메시지와 함께 503 오류가 발생하면, W&amp;B Inference 서버에 트래픽이 많이 몰려 현재 Request를 처리할 수 없다는 의미입니다.

<div id="why-this-happens">
  ## 이 문제가 발생하는 이유
</div>

수요가 많은 기간에는 Inference 엔진에 일시적으로 과부하가 걸릴 수 있습니다. 이는 일시적인 상태이며, 일반적으로 트래픽이 줄어들면 저절로 해결됩니다.

<div id="what-you-can-do">
  ## 할 수 있는 작업
</div>

1. **잠시 후 다시 시도하세요**
   * Request를 다시 시도하기 전에 몇 초 정도 기다리세요
   * 혼잡을 가중하지 않도록 지수 백오프를 사용하세요

2. **Request를 분산해 보내세요**
   * 많은 Request를 보내는 경우, 시간 간격을 두고 보내는 것을 고려하세요
   * Request 큐잉을 구현하여 트래픽 급증을 완화하세요
