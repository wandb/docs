---

&quot;The inference gateway is not enabled for your organization&quot;라는 메시지와 함께 403 오류가 표시되면, 해당 조직에서 inference gateway가 활성화되어 있지 않다는 의미이며, W&amp;B Inference를 사용하려면 이 기능이 필요합니다.

<div id="why-this-happens">
  ## 이 문제가 발생하는 이유
</div>

W&amp;B Inference를 사용하려면 조직 수준에서 inference gateway가 활성화되어 있어야 합니다. 조직에서 이 기능을 활성화하지 않으면 모든 API 요청이 403 오류와 함께 거부됩니다.

<div id="what-you-can-do">
  ## 할 수 있는 일
</div>

1. **W&amp;B 관리자에게 문의**
   * 조직의 W&amp;B 관리자에게 inference gateway를 활성화해 달라고 요청하세요

2. **W&amp;B 지원팀에 문의**
   * 관리자가 누구인지 확실하지 않거나 gateway 활성화에 도움이 필요하면, [W&amp;B 지원팀](mailto:support@wandb.com)에 문의하세요
