---

&quot;The inference gateway is not enabled for your organization&quot;라는 메시지와 함께 403 오류가 발생하면, 해당 조직에 inference gateway가 활성화되어 있지 않은 것입니다. W&amp;B Inference를 사용하려면 inference gateway가 필요합니다.

<div id="why-this-happens">
  ## 왜 이런 일이 발생하나요
</div>

W&amp;B Inference를 사용하려면 조직 수준에서 inference gateway가 활성화되어 있어야 합니다. 조직에서 이 기능을 활성화하지 않으면 모든 API 요청이 403 오류와 함께 거부됩니다.

<div id="what-you-can-do">
  ## 수행 가능한 작업
</div>

1. **W&amp;B 관리자에게 문의하세요**
   * 조직의 W&amp;B 관리자에게 inference gateway 활성화를 요청하세요

2. **W&amp;B 지원팀에 문의하세요**
   * 관리자가 누구인지 모르거나 게이트웨이 활성화에 도움이 필요한 경우, [W&amp;B 지원팀](mailto:support@wandb.com)에 문의하세요
