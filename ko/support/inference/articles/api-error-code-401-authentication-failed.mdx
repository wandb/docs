&quot;Authentication failed&quot; 메시지와 함께 401 오류가 발생하는 경우, 인증 자격 증명이 올바르지 않거나 W&amp;B 프로젝트 entity 및/또는 이름이 올바르지 않음을 의미합니다.

<div id="why-this-happens">
  ## 이런 문제가 발생하는 이유
</div>

W&amp;B Inference API는 요청을 처리하기 위해 유효한 인증 자격 증명이 필요합니다. 이 오류는 다음과 같은 경우에 발생합니다.

* API 키가 유효하지 않거나, 만료되었거나, 취소된 경우
* W&amp;B 프로젝트 entity 이름이 올바르지 않은 경우
* W&amp;B 프로젝트 이름이 올바르지 않은 경우

<div id="what-you-can-do">
  ## 할 수 있는 작업
</div>

1. **API 키 확인**
   * 계정에 맞는 올바른 API 키를 사용하고 있는지 확인하세요
   * 필요한 경우 [W&amp;B settings](https://wandb.ai/settings)에서 API 키를 다시 생성하세요

2. **프로젝트 세부 정보 확인**
   * W&amp;B 프로젝트 entity(조직 또는 사용자 이름)가 올바른지 확인하세요
   * 프로젝트 이름이 기존 프로젝트와 일치하는지 확인하세요

3. **지원팀에 문의**
   * 자격 증명을 확인한 후에도 문제가 계속되면 [W&amp;B support](mailto:support@wandb.com)로 문의하세요
