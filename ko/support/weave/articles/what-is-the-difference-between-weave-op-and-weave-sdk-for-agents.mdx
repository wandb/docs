keywords: ["Agents", "Tracing", "weave.op"]

[`@weave.op`](/ko/weave/guides/tracking/tracing)은 개별 Python 함수를 트레이스하고 결과를 **Traces** 탭에 표시합니다.

Weave의 [에이전트 추적](/ko/weave/guides/tracking/trace-agents) (`weave.start_session`, `weave.start_turn`, `weave.start_llm`, `weave.start_tool`)은 전체 멀티턴 에이전트 대화를 모델링하여 결과를 **Agents** 탭에 표시합니다.

둘 다 동일한 `weave` API에 속하지만, 함수와 파이프라인을 트레이스하는지 아니면 멀티턴 에이전트 애플리케이션을 트레이스하는지에 따라 적절한 선택이 달라집니다.
