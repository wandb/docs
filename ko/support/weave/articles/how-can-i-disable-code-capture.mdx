Weave 클라이언트 초기화 시 코드 캡처를 비활성화할 수 있습니다: `weave.init("entity/project", settings={"capture_code": False})`.
[환경 변수](/ko/weave/guides/core-types/env-vars) `WEAVE_CAPTURE_CODE=false`를 사용해도 됩니다.
