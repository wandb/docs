---
title: "트레이스 데이터가 잘립니다"
keywords: ["Trace Data"]
---

때때로 큰 트레이스 데이터가 Weave UI에서 일부만 표시됩니다. 이 문제는 기본 트레이스 출력이 Weave가 직렬화하는 방법을 모르는 원시 맞춤형 Python 객체이기 때문에 발생합니다.

큰 트레이스 데이터가 잘리지 않게 하려면 모든 트레이스 데이터를 반환하는 문자열로 이루어진 딕셔너리를 정의하세요.

```python
import weave

class MyObj:
    def __init__(self, x: int):
        self.x = x

    def __repr__(self):
        return f"MyObj(x={self.x})"

    def to_dict(self):
        return {"x": self.x}

@weave.op()
def make_my_obj():
    x = "s" * 10_000
    return MyObj(x)
```
