---
title: "UI에서 Markdown을 어떻게 렌더링하나요?"
keywords: ["UI 렌더링"]
---

저장하기 전에 문자열을 `weave.Markdown(...)`으로 감싸고, 저장할 때는 `weave.publish(...)`를 사용하세요. Weave는 객체의 유형을 기준으로 렌더링 방식을 결정하며, `weave.Markdown`은 알려진 UI 렌더러에 매핑됩니다. 값은 UI에서 서식이 적용된 Markdown 객체로 표시됩니다. 전체 코드 샘플은 [호출 뷰](/ko/weave/guides/tracking/tracing#viewing-calls)를 참조하세요.
