---
title: "범례가 있는 플롯에 여러 선을 어떻게 그리나요?"
keywords: ["Experiments"]
---

`wandb.plot.line_series()`를 사용해 여러 선이 있는 맞춤형 차트를 만드세요. 선 차트를 보려면 [프로젝트 페이지](/ko/models/track/project-page)로 이동하세요. 범례를 추가하려면 `wandb.plot.line_series()`에 `keys` 인수를 포함하세요. 예를 들면 다음과 같습니다:

```python

with wandb.init(project="my_project") as run:

    run.log(
        {
            "my_plot": wandb.plot.line_series(
                xs=x_data, ys=y_data, keys=["metric_A", "metric_B"]
            )
        }
    )
```

멀티 라인 선형 플롯에 대한 자세한 내용은 **멀티 라인** 탭에서 [여기](/ko/models/track/log/plots#basic-charts)를 참고하세요.
