`wandb.plot.line_series()`를 사용해 여러 개의 선이 있는 사용자 지정 차트를 생성합니다. 선 차트를 보려면 [프로젝트 페이지](/ko/models/track/project-page)로 이동하세요. 범례를 추가하려면 `wandb.plot.line_series()`에 `keys` 인수를 지정하세요. 예시:

```python

with wandb.init(project="my_project") as run:

    run.log(
        {
            "my_plot": wandb.plot.line_series(
                xs=x_data, ys=y_data, keys=["metric_A", "metric_B"]
            )
        }
    )
```

다중 선 플롯에 관한 자세한 내용은 **Multi-line** 탭 아래의 [여기](/ko/models/track/log/plots#basic-charts)를 참고하세요.
