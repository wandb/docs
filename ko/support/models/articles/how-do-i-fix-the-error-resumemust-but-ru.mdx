---
title: "`resume='must' but run (<run_id>) doesn't exist` 오류는 어떻게 해결하나요?"
keywords: ["재개", "Runs"]
---

오류 `resume='must' but run (<run_id>) doesn't exist`가 발생하면 재개하려는 run이 프로젝트 또는 entity에 존재하지 않는다는 의미입니다. 올바른 인스턴스에 로그인되어 있고 프로젝트와 entity가 설정되었는지 확인하세요:

```python
wandb.init(entity=<entity>, project=<project>, id=<run-id>, resume='must')
```

인증되었는지 확인하려면 [`wandb login --relogin`](/ko/models/ref/cli/wandb-login)을 실행하세요.
