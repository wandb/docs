여러 run을 한 번에 삭제하려면 [public API](/ko/models/ref/python/public-api/api)를 사용하세요:

```python
import wandb

api = wandb.Api()
runs = api.runs('<entity>/<project>')
for run in runs:
    if <condition>:
        run.delete()
```
