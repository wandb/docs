[Public API](/ko/models/ref/python/public-api/api)를 사용하여 단일 오퍼레이션으로 여러 Runs를 삭제하세요:

```python
import wandb

api = wandb.Api()
runs = api.runs('<entity>/<project>')
for run in runs:
    if <condition>:
        run.delete()
```
