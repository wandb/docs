같은 머신에서 두 개의 W&amp;B 계정을 관리하려면 API 키 두 개를 모두 파일에 저장하세요. 저장소에서는 다음 코드를 사용해 키를 안전하게 전환하고, 비밀 키가 버전 관리에 커밋되지 않도록 하세요.

```python
if os.path.exists("~/keys.json"):
    os.environ["WANDB_API_KEY"] = json.loads("~/keys.json")["work_account"]
```
