같은 머신에서 두 개의 W&amp;B 계정을 관리하려면 두 API 키를 파일에 저장하세요. 비밀 키가 소스 제어에 커밋되지 않도록 하면서 키를 안전하게 전환하려면 리포지토리에서 다음 코드를 사용하세요.

```python
if os.path.exists("~/keys.json"):
    os.environ["WANDB_API_KEY"] = json.loads("~/keys.json")["work_account"]
```
