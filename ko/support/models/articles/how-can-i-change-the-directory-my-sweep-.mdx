환경 변수 `WANDB_DIR`를 설정하여 W&amp;B run 데이터의 로깅 디렉터리를 설정하세요. 예를 들면 다음과 같습니다:

```python
os.environ["WANDB_DIR"] = os.path.abspath("your/directory")
```
