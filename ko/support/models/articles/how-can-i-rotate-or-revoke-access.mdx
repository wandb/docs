---
title: "액세스 권한을 갱신하거나 철회하려면 어떻게 해야 하나요?"
keywords: ["관리자", "보안"]
---

개인 및 서비스 계정 키는 교체하거나 무효화할 수 있습니다. 새 API 키 또는 서비스 계정 사용자를 만든 다음, 스크립트가 새 키를 사용하도록 다시 설정하세요. 다시 설정한 후에는 프로필이나 팀에서 이전 API 키를 제거하세요.
