---
title: "'Failed to query for notebook' 오류는 어떻게 해결하나요?"
keywords: ["노트북", "환경 변수"]
---

오류 메시지 `"Failed to query for notebook name, you can set it manually with the WANDB_NOTEBOOK_NAME environment variable,"`가 나타나면 환경 변수를 설정해 해결할 수 있습니다. 이를 수행하는 방법은 여러 가지가 있습니다:

<Tabs>
  <Tab title="노트북">
    ```python
    %env "WANDB_NOTEBOOK_NAME" "notebook name here"
    ```
  </Tab>

  <Tab title="Python">
    ```python
    import os

    os.environ["WANDB_NOTEBOOK_NAME"] = "notebook name here"
    ```
  </Tab>
</Tabs>
