이 페이지에서는 Hydra와 W&amp;B를 함께 사용해 프로세스를 시작할 때 발생하는 응답 없음 문제를 해결하는 방법을 설명합니다. Hydra로 프로세스를 시작했을 때 응답하지 않는다면, 이는 Hydra와 W&amp;B 간의 멀티프로세싱 충돌 때문일 가능성이 높습니다.

이 문제를 해결하려면 W&amp;B의 멀티프로세싱 프로토콜을 `"thread"`로 변경하세요. 이렇게 하면 충돌을 방지해 프로세스를 정상적으로 시작할 수 있습니다. 다음 두 가지 방법 중 하나로 설정할 수 있습니다:

<div id="pass-a-settings-parameter-to-wandbinit">
  ## wandb.init에 settings 파라미터 전달
</div>

```python
wandb.init(settings=wandb.Settings(start_method="thread"))
```

<div id="set-a-global-environment-variable">
  ## 전역 환경 변수 설정
</div>

```bash
export WANDB_START_METHOD=thread
```
