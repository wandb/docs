title: "서로 다른 두 시간 척도로 메트릭을 기록할 수 있나요?"

---

예를 들어, 배치별로 트레이닝 정확도를 기록하고 에포크별로 검증 정확도를 기록하고 싶습니다.

예. 메트릭과 함께 `batch` 및 `epoch` 같은 인덱스를 기록하세요. 한 번은 `wandb.Run.log()({'train_accuracy': 0.9, 'batch': 200})`를 사용하고, 다른 번은 `wandb.Run.log()({'val_accuracy': 0.8, 'epoch': 4})`를 사용하세요. UI에서 각 chart마다 원하는 값을 x-axis로 설정하세요. 특정 인덱스의 기본 x-axis를 설정하려면 [Run.define&#95;metric()](/ko/models/ref/python/experiments/run#define_metric)을 사용하세요. 위 예시에는 다음 코드를 사용하세요:

```python
import wandb

with wandb.init() as run:
   run.define_metric("batch")
   run.define_metric("epoch")

   run.define_metric("train_accuracy", step_metric="batch")
   run.define_metric("val_accuracy", step_metric="epoch")
```
