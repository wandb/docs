예를 들어, 배치별로 트레이닝 정확도를, 에포크별로 검증 정확도를 log하고 싶습니다.

예. `batch`와 `epoch` 같은 인덱스를 메트릭과 함께 log하세요. 한 step에서는 `wandb.Run.log()({'train_accuracy': 0.9, 'batch': 200})`를 사용하고, 다른 step에서는 `wandb.Run.log()({'val_accuracy': 0.8, 'epoch': 4})`를 사용하세요. UI에서 각 chart의 x축으로 원하는 값을 설정하세요. 특정 인덱스에 대한 기본 x축을 설정하려면 [Run.define&#95;metric()](/ko/models/ref/python/experiments/run#define_metric)을 사용하세요. 위 예시에서는 다음 코드를 사용하세요:

```python
import wandb

with wandb.init() as run:
   run.define_metric("batch")
   run.define_metric("epoch")

   run.define_metric("train_accuracy", step_metric="batch")
   run.define_metric("val_accuracy", step_metric="epoch")
```
