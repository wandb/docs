---
title: "계정이 없는 사람은 run 결과를 어떻게 확인하나요?"
keywords: ["Anonymous"]
---

누군가 `anonymous="allow"`로 스크립트를 실행하면 다음과 같습니다.

1. **임시 계정 자동 생성**: W&amp;B는 로그인된 계정이 있는지 확인합니다. 계정이 없으면 새 익명 계정을 만들고, 해당 세션의 API 키를 저장합니다.
2. **결과를 빠르게 기록**: 사용자는 스크립트를 반복해서 실행하고 W&amp;B 대시보드에서 즉시 결과를 확인할 수 있습니다. 이렇게 소유권을 주장하지 않은 익명 run은 7일 동안 유지됩니다.
3. **유용한 경우 데이터 소유권 주장**: 사용자가 W&amp;B에서 가치 있는 결과를 찾으면, 페이지 상단 배너의 버튼을 클릭해 자신의 run 데이터를 실제 계정에 저장할 수 있습니다. 소유권을 주장하지 않으면 run 데이터는 7일 후 삭제됩니다.

<Warning>
  **익명 run 링크는 민감한 정보입니다**. 이 링크가 있으면 누구나 7일 동안 실험 결과를 보고 소유권을 주장할 수 있으므로, 신뢰할 수 있는 사람에게만 공유하세요. 작성자의 신원을 숨긴 채 결과를 공개적으로 공유하려면 support@wandb.com으로 문의해 도움을 받으세요.
</Warning>

W&amp;B 사용자가 이 스크립트를 찾아 실행하면, 일반적인 run과 마찬가지로 결과가 해당 계정에 올바르게 기록됩니다.
