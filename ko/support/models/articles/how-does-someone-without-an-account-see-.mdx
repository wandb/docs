---
title: "계정이 없는 사용자는 run 결과를 어떻게 볼 수 있나요?"
keywords: ["익명"]
---

누군가 `anonymous="allow"`로 스크립트를 실행하면 다음과 같습니다.

1. **임시 계정 자동 생성**: W&amp;B는 로그인된 계정이 있는지 확인합니다. 없으면 W&amp;B가 새 익명 계정을 생성하고 해당 세션의 API 키를 저장합니다.
2. **결과를 빠르게 기록**: 사용자는 스크립트를 반복해서 실행하고 W&amp;B 대시보드에서 결과를 즉시 확인할 수 있습니다. 이렇게 소유권이 아직 확인되지 않은 익명 run은 7일 동안 사용 가능합니다.
3. **유용한 경우 데이터 claim**: 사용자가 W&amp;B에서 가치 있는 결과를 확인하면 페이지 상단 배너의 버튼을 클릭해 자신의 run 데이터를 실제 계정에 저장할 수 있습니다. claim하지 않으면 run 데이터는 7일 후 삭제됩니다.

<Warning>
  **익명 run 링크는 민감한 정보입니다**. 이 링크가 있으면 누구나 7일 동안 실험 결과를 보고 claim할 수 있으므로, 신뢰할 수 있는 사람에게만 공유하세요. 작성자의 ID를 숨긴 채 결과를 공개적으로 공유하려면 지원을 위해 support@wandb.com으로 문의하세요.
</Warning>

W&amp;B 사용자가 스크립트를 찾아 실행하면 일반적인 run과 마찬가지로 결과가 해당 계정에 정상적으로 기록됩니다.
