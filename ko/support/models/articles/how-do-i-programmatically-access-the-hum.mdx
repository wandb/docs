[`wandb.Run`](/ko/models/ref/python/experiments/run)의 `.name` 속성은 다음과 같이 접근할 수 있습니다:

```python
import wandb

with wandb.init() as run:
   run_name = run.name
   print(f"사람이 읽을 수 있는 run 이름은: {run_name}")
```
