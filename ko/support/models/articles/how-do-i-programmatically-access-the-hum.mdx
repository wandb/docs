keywords: ["Experiments"]
---

[`wandb.Run`](/ko/models/ref/python/experiments/run)의 `.name` 속성에 액세스하려면 다음 코드를 사용하세요:

```python
import wandb

with wandb.init() as run:
   run_name = run.name
   print(f"The human-readable run name is: {run_name}")
```
