---
title: "익명 사용자에게 제공되지 않는 기능은 무엇인가요?"
keywords: ["익명"]
---

* **영구 데이터 없음**: Runs는 익명 계정에서 7일 동안 보관됩니다. 익명 run 데이터를 실제 계정에 저장해 소유권을 가져오세요.

<Frame>
  <img src="/images/app_ui/anon_mode_no_data.png" alt="익명 모드 인터페이스" />
</Frame>

* **artifact 로깅 없음**: 익명 run에 artifact를 로깅하려고 하면 명령줄에 경고가 표시됩니다.
  ```bash
  wandb: WARNING Artifacts logged anonymously cannot be claimed and expire after 7 days.
  ```

* **프로필 또는 설정 페이지 없음**: 이러한 페이지는 실제 계정에서만 유용하므로 UI에는 포함되지 않습니다.
