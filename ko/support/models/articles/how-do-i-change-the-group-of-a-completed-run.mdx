---
title: "완료된 run의 그룹을 어떻게 변경하나요?"
keywords: ["Runs"]
---

API를 사용해 완료된 run에 할당된 그룹을 변경할 수 있습니다. 이 기능은 웹 UI에는 표시되지 않습니다. 그룹을 업데이트하려면 다음 코드를 사용하세요:

```python
import wandb

api = wandb.Api()
run = api.run("<ENTITY>/<PROJECT>/<RUN_ID>")
run.group = "NEW-GROUP-NAME"
run.update()
```
