W&amp;B는 각 run이 로그한 아티팩트와 각 run이 사용한 아티팩트를 추적해 아티팩트 그래프를 구성합니다. 이 그래프는 run과 아티팩트를 나타내는 노드로 이루어진 방향성 있는 비순환 이분 그래프입니다. 예시는 [여기](https://wandb.ai/shawn/detectron2-11/artifacts/dataset/furniture-small-val/06d5ddd4deeb2a6ebdd5/graph)에서 확인할 수 있습니다(그래프를 확장하려면 &quot;Explode&quot;를 클릭하세요).

아티팩트 또는 run에서 시작해 Public API로 프로그래밍 방식으로 그래프를 탐색할 수 있습니다.

<Tabs>
  <Tab title="아티팩트에서">
    ```python
    api = wandb.Api()

    artifact = api.artifact("project/artifact:alias")

    # 아티팩트에서 그래프를 위쪽으로 따라갑니다:
    producer_run = artifact.logged_by()
    # 아티팩트에서 그래프를 아래쪽으로 따라갑니다:
    consumer_runs = artifact.used_by()

    # run에서 그래프를 아래쪽으로 따라갑니다:
    next_artifacts = consumer_runs[0].logged_artifacts()
    # run에서 그래프를 위쪽으로 따라갑니다:
    previous_artifacts = producer_run.used_artifacts()
    ```
  </Tab>

  <Tab title="run에서">
    ```python
    api = wandb.Api()

    run = api.run("entity/project/run_id")

    # run에서 그래프를 아래쪽으로 따라갑니다:
    produced_artifacts = run.logged_artifacts()
    # run에서 그래프를 위쪽으로 따라갑니다:
    consumed_artifacts = run.used_artifacts()

    # 아티팩트에서 그래프를 위쪽으로 따라갑니다:
    earlier_run = consumed_artifacts[0].logged_by()
    # 아티팩트에서 그래프를 아래쪽으로 따라갑니다:
    consumer_runs = produced_artifacts[0].used_by()
    ```
  </Tab>
</Tabs>
