---
title: "run 이름을 run ID로 설정할 수 있나요?"
keywords: ["Experiments"]
---

예.

run 이름을 run ID로 덮어쓰려면 다음 코드 스니펫을 사용하세요:

```python
import wandb

with wandb.init() as run:
   run.name = run.id
   run.save()
```
