`wandb.init`를 호출하면 W&amp;B는 원격 저장소 링크와 최신 커밋의 SHA를 포함한 Git 정보를 자동으로 수집합니다. 이 정보는 [run 페이지](/ko/models/runs/#view-logged-runs)에 표시됩니다. 이 정보를 보려면 스크립트를 실행할 때 현재 작업 디렉터리가 Git으로 관리되는 폴더 안에 있는지 확인하세요.

Git 커밋과 실험을 실행하는 데 사용한 명령어는 확인할 수 있지만, 공개 프로젝트에서도 외부 사용자에게는 숨겨집니다.
