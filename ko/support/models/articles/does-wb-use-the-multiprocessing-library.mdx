---
title: "W&B는 `multiprocessing` 라이브러리를 사용하나요?"
keywords: ["Experiments"]
---

예, W&amp;B는 `multiprocessing` 라이브러리를 사용합니다. 다음과 같은 오류 메시지가 표시되면 문제가 있을 수 있습니다.

```
An attempt has been made to start a new process before the current process 
has finished its bootstrapping phase.
```

이 문제를 해결하려면 `if __name__ == "__main__":`를 추가해 엔트리 포인트를 보호하세요. 이 보호 구문은 스크립트에서 W&amp;B를 직접 실행할 때 필요합니다.
