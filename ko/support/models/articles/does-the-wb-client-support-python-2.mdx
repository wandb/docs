---
title: "W&B 클라이언트는 Python 2를 지원하나요?"
keywords: ["Python"]
---

W&amp;B 클라이언트 라이브러리는 버전 0.10까지 Python 2.7과 Python 3를 모두 지원했습니다. 버전 0.11에서는 Python 2의 수명 종료로 인해 Python 2.7 지원이 제거되었습니다. Python 2.7 시스템에서 `pip install --upgrade wandb`를 실행하면 0.10.x 시리즈의 새 릴리스만 설치되며, 이 시리즈에는 중요한 버그 수정과 패치만 제공됩니다. Python 2.7을 지원하는 0.10.x 시리즈의 마지막 버전은 0.10.33입니다.
