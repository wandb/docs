트레이닝 설정에서 명령줄 인수를 전달하는 경우, 사용자 지정 CLI 명령과 함께 W&amp;B Sweeps를 사용할 수 있습니다.

아래 예시의 코드 스니펫은 사용자가 `train.py`라는 Python 스크립트를 실행하면서 스크립트가 파싱할 값을 전달하는 bash 터미널을 보여줍니다:

```bash
/usr/bin/env python train.py -b \
    your-training-config \
    --batchsize 8 \
    --lr 0.00001
```

사용자 지정 명령을 구현하려면 YAML 파일에서 `command` 키를 수정하세요. 이전 예시를 기준으로 하면, 설정은 다음과 같습니다:

```yaml
program:
  train.py
method: grid
parameters:
  batch_size:
    value: 8
  lr:
    value: 0.0001
command:
  - ${env}
  - python
  - ${program}
  - "-b"
  - your-training-config
  - ${args}
```

`${args}` 키는 스윕 설정에 있는 모든 파라미터로 확장되며, `argparse`에서 사용할 수 있도록 `--param1 value1 --param2 value2` 형식으로 지정됩니다.

`argparse` 외부의 추가 인수를 사용하려면 다음과 같이 구현하세요:

```python
parser = argparse.ArgumentParser()
args, unknown = parser.parse_known_args()
```

<Note>
  환경에 따라 `python`이 Python 2를 가리킬 수 있습니다. Python 3를 확실히 호출하려면 명령 설정에서 `python3`를 사용하세요.

  ```yaml
  program:
    script.py
  command:
    - ${env}
    - python3
    - ${program}
    - ${args}
  ```
</Note>
