그리드 서치가 완료되었지만 일부 W&amp;B Runs가 오류로 중단되어 다시 실행해야 하는 경우, 다시 실행할 특정 W&amp;B Runs를 삭제하세요. 그런 다음 [스윕 제어 페이지](/ko/models/sweeps/sweeps-ui)에서 **재개** 버튼을 선택하세요. 새 Sweep ID를 사용해 새로운 W&amp;B Sweep 에이전트를 시작하세요.

완료된 W&amp;B Run 파라미터 조합은 다시 실행되지 않습니다.
