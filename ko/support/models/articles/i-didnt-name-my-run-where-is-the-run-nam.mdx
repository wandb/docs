---
title: "run에 이름을 지정하지 않았는데, run 이름은 어디서 정해지나요?"
keywords: ["Experiments"]
---

run에 이름을 명시적으로 지정하지 않으면 W&amp;B가 프로젝트에서 해당 run을 식별할 수 있도록 무작위 이름을 할당합니다. 무작위 이름의 예시로는 `pleasant-flower-4` 및 `misunderstood-glade-2.가 있습니다.
