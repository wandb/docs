---
title: "run에 이름을 지정하지 않았는데, run 이름은 어디에서 오나요?"
keywords: ["Experiments"]
---

run에 이름을 명시적으로 지정하지 않으면 W&amp;B는 프로젝트에서 해당 run을 식별할 수 있도록 무작위 이름을 부여합니다. 무작위 이름의 예로는 `pleasant-flower-4`와 `misunderstood-glade-2. 등이 있습니다.
