이전 step의 로그를 덮어쓰려면 [forking](/ko/models/runs/forking)과 [rewind](/ko/models/runs/rewind)를 사용하세요.

<Note>
  Run 포크는 Multi-tenant Cloud와 Dedicated Cloud에서 공개 프리뷰로 제공됩니다. 현재 Self-Managed 배포에서는 사용하실 수 없습니다.

  공개 프리뷰 기능은 프로덕션 평가 용도로 사용할 수 있지만, 기능, API 및 동작은 현재 활발히 개발 중이며 정식 출시 전에 변경될 수 있습니다.
</Note>
