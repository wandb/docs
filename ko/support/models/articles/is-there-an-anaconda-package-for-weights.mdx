`pip` 또는 `conda`로 설치할 수 있는 Anaconda 패키지가 있습니다. `conda`의 경우 [conda-forge](https://conda-forge.org) 채널에서 패키지를 설치하세요.

<Tabs>
  <Tab title="pip">
    ```shell
    # conda 환경 생성
    conda create -n wandb-env python=3.8 anaconda
    # 환경 활성화
    conda activate wandb-env
    # pip를 사용해 wandb 설치
    pip install wandb
    ```
  </Tab>

  <Tab title="conda">
    ```shell
    conda activate myenv
    conda install wandb --channel conda-forge
    ```
  </Tab>
</Tabs>

설치 문제가 발생하면 패키지 관리에 대한 Anaconda [문서](https://docs.conda.io/projects/conda/en/latest/user-guide/tasks/manage-pkgs.html)를 참고하세요.
