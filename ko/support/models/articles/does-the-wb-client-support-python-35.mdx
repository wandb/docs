W&amp;B 클라이언트 라이브러리는 버전 0.11까지 Python 3.5를 지원했습니다. Python 3.5 지원은 지원 종료 시점에 맞춰 버전 0.12에서 중단되었습니다. 자세한 내용은 [버전 0.12 릴리스 노트](https://github.com/wandb/wandb/releases/tag/v0.12.0)를 참조하세요.
