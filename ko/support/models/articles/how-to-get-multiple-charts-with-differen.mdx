---
title: "선택한 Runs가 서로 다른 여러 차트를 표시하려면 어떻게 하나요?"
keywords: ["Reports"]
---

W&amp;B Reports에서는 별도의 패널 그리드를 사용해 동일한 report에서 서로 다른 run 세트의 차트를 나란히 표시할 수 있습니다. 이렇게 하면 컨텍스트를 전환하지 않고도 서로 다른 run 선택 간의 결과를 비교할 수 있어 유용합니다.

이렇게 설정하려면 다음 step을 따르세요:

1. 여러 패널 그리드를 만드세요.
2. 필터를 적용해 각 패널 그리드의 run 세트를 선택하세요. 각 패널 그리드는 자체 run 선택을 유지하므로 서로 다른 차트에 서로 다른 run이 반영될 수 있습니다.
3. 패널 그리드 내에서 차트를 생성하세요.

이 step을 완료하면 report에 여러 패널 그리드가 포함되며, 각 패널 그리드에는 자체적으로 필터링된 run 세트의 차트가 표시됩니다.
