W&amp;B Reports에서 다음 단계를 따르세요:

1. 여러 패널 그리드를 생성합니다.
2. 각 패널 그리드에서 원하는 run 집합을 선택할 수 있도록 필터를 적용합니다.
3. 패널 그리드 내에 원하는 차트를 생성합니다.
