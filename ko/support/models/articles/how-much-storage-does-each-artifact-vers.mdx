두 아티팩트 버전 간에 변경된 파일에만 저장소 비용이 발생합니다.

<Frame>
  <img src="/images/artifacts/artifacts-dedupe.PNG" alt="아티팩트 중복 제거" />
</Frame>

`animals`라는 이름의 이미지 아티팩트에 `cat.png`와 `dog.png`라는 두 개의 이미지 파일이 포함되어 있다고 가정해 보겠습니다.

```
images
|-- cat.png (2MB) # `v0`에서 추가됨
|-- dog.png (1MB) # `v0`에서 추가됨
```

이 아티팩트에는 버전 `v0`가 할당됩니다.

새 이미지 `rat.png`를 추가하면 다음 내용을 포함하는 새 아티팩트 버전 `v1`이 생성됩니다:

```
images
|-- cat.png (2MB) # `v0`에서 추가됨
|-- dog.png (1MB) # `v0`에서 추가됨
|-- rat.png (3MB) # `v1`에서 추가됨
```

버전 `v1`은 총 6MB를 추적하지만, 그중 3MB를 `v0`와 공유하므로 실제로는 3MB의 공간만 차지합니다. `v1`을 삭제하면 `rat.png`에 해당하는 3MB의 저장소가 회수됩니다. `v0`를 삭제하면 `cat.png`와 `dog.png`의 저장소 비용이 `v1`로 이전되어 `v1`의 저장소 크기가 6MB로 늘어납니다.
