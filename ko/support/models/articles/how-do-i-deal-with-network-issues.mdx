`wandb: Network error (ConnectionError), entering retry loop`와 같은 SSL 또는 네트워크 오류가 발생하면 다음 해결 방법을 사용하세요:

1. SSL 인증서를 업데이트하세요. Ubuntu 서버에서는 `update-ca-certificates`를 실행하세요. 보안 위험을 줄이면서 트레이닝 로그를 동기화하려면 유효한 SSL 인증서가 필수입니다.
2. 네트워크 연결이 불안정하면 [선택 환경 변수](/ko/models/track/environment-variables#optional-environment-variables) `WANDB_MODE`를 `offline`으로 설정해 오프라인 모드로 사용하고, 나중에 인터넷에 연결된 장치에서 파일을 동기화하세요.
3. 로컬에서 실행되어 클라우드 서버로 동기화할 필요가 없는 [W&amp;B Private Hosting](/ko/platform/hosting/) 사용을 고려하세요.

`SSL CERTIFICATE_VERIFY_FAILED` 오류의 경우, 회사 방화벽이 원인일 수 있습니다. 로컬 CA를 설정하고 다음을 실행하세요:

`export REQUESTS_CA_BUNDLE=/etc/ssl/certs/ca-certificates.crt`
