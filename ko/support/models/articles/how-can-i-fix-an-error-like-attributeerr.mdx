Python에서 `wandb`를 임포트할 때 `AttributeError: module 'wandb' has no attribute 'init'` 또는 `AttributeError: module 'wandb' has no attribute 'login'`와 같은 오류가 발생하면, `wandb`가 설치되지 않았거나 설치가 손상되었는데 현재 작업 디렉터리에 `wandb` 디렉터리가 있는 경우일 수 있습니다. 이 오류를 해결하려면 `wandb`를 제거하고 해당 디렉터리를 삭제한 다음 `wandb`를 다시 설치하세요:

```bash
pip uninstall wandb; rm -rI wandb; pip install wandb
```
