---
title: "코드가 충돌할 때 어떤 파일을 확인해야 하나요?"
keywords: ["로그"]
---

문제가 발생한 run의 경우, 코드를 실행한 디렉터리의 `wandb/run-<date>_<time>-<run-id>/logs`에서 `debug.log`와 `debug-internal.log`를 확인하세요.
