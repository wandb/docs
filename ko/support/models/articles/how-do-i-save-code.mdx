---
title: "코드를 저장하려면 어떻게 해야 하나요?‌"
keywords: ["Artifacts"]
---

각 run과 함께 코드를 저장하면 실험을 재현하고 어떤 버전의 코드가 특정 결과를 생성했는지 추적하는 데 도움이 됩니다. `wandb.init`에서 `save_code=True`를 사용하면 run을 시작하는 메인 스크립트나 노트북을 저장할 수 있습니다. run의 모든 코드를 저장하려면 W&amp;B Artifacts로 코드를 버전 관리하세요. 다음 예제는 이 과정을 보여줍니다.

```python
code_artifact = wandb.Artifact(type="code")
code_artifact.add_file("./train.py")
wandb.log_artifact(code_artifact)
```
