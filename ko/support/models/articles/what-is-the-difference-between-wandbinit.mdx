다음 모드를 사용할 수 있습니다:

* `online` (기본값): 클라이언트가 wandb 서버로 데이터를 전송합니다.
* `offline`: 클라이언트가 wandb 서버로 전송하는 대신 머신에 데이터를 로컬에 저장합니다. 나중에 데이터를 동기화하려면 [`wandb sync`](/ko/models/ref/cli/wandb-sync) 명령을 사용하세요.
* `disabled`: 클라이언트가 모의 객체를 반환해 동작을 시뮬레이션하고 모든 네트워크 통신을 차단합니다. 모든 로깅은 비활성화되지만, 모든 API 방법 스텁은 계속 호출할 수 있습니다. 이 모드는 일반적으로 테스트에 사용됩니다.
