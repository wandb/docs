트레이닝 중에 머신과의 연결이 끊어졌을 가능성이 높습니다. [`wandb sync [PATH_TO_RUN]`](/ko/models/ref/cli/wandb-sync)을 실행해 데이터를 복구하세요. run 경로는 `wandb` 디렉터리 안에서 진행 중인 run의 Run ID와 일치하는 폴더입니다.
