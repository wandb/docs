---
title: "완료 후 run에 부여된 그룹을 변경할 수 있나요?"
keywords: ["Runs"]
---

API를 사용하면 완료된 run에 부여된 그룹을 변경할 수 있습니다. 이 기능은 웹 UI에는 표시되지 않습니다. 다음 코드를 사용하여 그룹을 업데이트하세요:

```python
import wandb

api = wandb.Api()
run = api.run("<ENTITY>/<PROJECT>/<RUN_ID>")
run.group = "NEW-GROUP-NAME"
run.update()
```
