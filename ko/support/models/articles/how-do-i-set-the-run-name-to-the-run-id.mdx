---
title: "run name을 run ID로 설정하려면 어떻게 해야 하나요?"
keywords: ["Experiments"]
---

run name을 run ID로 덮어쓰려면 다음 코드 스니펫을 사용하세요:

```python
import wandb

with wandb.init() as run:
   run.name = run.id
   run.save()
```
