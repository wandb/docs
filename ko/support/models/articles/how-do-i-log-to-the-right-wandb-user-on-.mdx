---
title: "공유 머신에서 올바른 wandb 사용자로 어떻게 로그를 기록하나요?"
keywords: ["로그"]
---

여러 사용자가 한 머신을 공유하는 경우, 자격 증명을 명시적으로 설정하지 않으면 run이 잘못된 W&amp;B 계정에 로그를 기록할 수 있습니다. run이 올바른 W&amp;B 계정에 로그를 기록하도록 하려면 인증을 위해 `WANDB_API_KEY` 환경 변수를 설정하세요. 이 변수가 환경에 설정되어 있으면 로그인할 때 올바른 자격 증명이 제공됩니다. 또는 스크립트에서 환경 변수를 직접 설정할 수도 있습니다.

이 셸 세션에서 이후의 W&amp;B Runs를 계정에 연결하려면 API 키를 export하세요: `export WANDB_API_KEY=[API-KEY]`. `[API-KEY]`를 자신의 API 키로 바꾸세요. API 키는 [wandb.ai/settings](https://wandb.ai/settings)에서 생성할 수 있습니다.
