---
title: "Filestream 요청 속도 제한 초과 오류를 해결하려면 어떻게 해야 하나요?"
keywords: ["연결", "장애"]
---

W&amp;B에서 &quot;Filestream rate limit exceeded&quot; 오류를 해결하려면 다음 단계를 따르세요:

**로깅 최적화**:

* API 요청 수를 줄이기 위해 로깅 빈도를 낮추거나 로그를 일괄 전송합니다.
  * API 요청이 동시에 몰리지 않도록 실험 시작 시간을 분산합니다.

**장애 확인**:

* [W&amp;B status updates](https://status.wandb.com)를 확인하여 일시적인 서버 측 문제 때문은 아닌지 확인합니다.

**지원팀에 문의**:

* 요청 속도 제한 상향을 요청하려면 실험 설정의 세부 정보와 함께 W&amp;B 지원팀(support@wandb.com)에 문의합니다.
