W&amp;B에서 &quot;Filestream rate limit exceeded&quot; 오류를 해결하려면 다음 단계를 따르세요:

**로깅 최적화**:

* API 요청을 줄이기 위해 로깅 빈도를 낮추거나 로그를 일괄 처리하세요.
  * API 요청이 동시에 발생하지 않도록 실험 시작 시간을 분산하세요.

**장애 확인**:

* [W&amp;B 상태 업데이트](https://status.wandb.com)를 확인하여 일시적인 서버 측 문제로 인해 발생한 이슈가 아닌지 확인하세요.

**지원팀에 문의**:

* 요청 속도 제한 상향을 요청하려면 실험 설정의 세부 정보와 함께 W&amp;B 지원팀(support@wandb.com)에 문의하세요.
