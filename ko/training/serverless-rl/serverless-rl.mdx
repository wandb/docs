Serverless RL은 [OpenPipe의 ART 프레임워크](https://art.openpipe.ai/getting-started/about)와 [W&amp;B Training API](/ko/training/api-reference)를 통해 지원됩니다.

Serverless RL 사용을 시작하려면 먼저 W&amp;B 도구 사용을 위한 [사전 요구 사항](/ko/training/prerequisites)을 충족한 다음, ART [퀵스타트](https://art.openpipe.ai/getting-started/quick-start)를 따라 진행하세요.

* 코드 예제와 워크플로는 [Google Colab 노트북](https://colab.research.google.com/github/openpipe/art-notebooks/blob/main/examples/2048/2048.ipynb)을 참조하세요.
* Serverless RL의 API 엔드포인트에 대해 알아보려면 [W&amp;B Training API 레퍼런스](/ko/training/api-reference)를 참조하세요.
