<Note>
  W&amp;B Training API는 서버리스 강화학습(RL)과 지도 파인튜닝(SFT)을 포함한 학습 작업을 관리하고 상호작용하기 위한 엔드포인트를 제공합니다. 이 API는 채팅 완성(chat completion)을 위해 OpenAI와 호환됩니다.
</Note>

<div id="authentication">
  ## 인증
</div>

모든 API 요청에는 W&amp;B API 키 인증이 필요합니다. [wandb.ai/settings](https://wandb.ai/settings)에서 API 키를 생성하세요.

`Authorization` 헤더에 API 키를 포함하세요:

```
Authorization: Bearer YOUR_API_KEY
```

<div id="base-url">
  ## 기본 URL
</div>

```
https://api.training.wandb.ai/v1
```

<div id="available-endpoints">
  ## 사용 가능한 엔드포인트
</div>

<div id="chat-completions">
  ### chat-completions
</div>

* **[POST /v1/chat/completions](https://docs.wandb.ai/training/api-reference/chat-completions/create-chat-completion)** - Chat completion 생성
* **[POST /v1/chat/completions/](https://docs.wandb.ai/training/api-reference/chat-completions/create-chat-completion)** - Chat completion 생성

<div id="models">
  ### models
</div>

* **[POST /v1/preview/models](https://docs.wandb.ai/training/api-reference/models/create-model)** - 모델 생성
* **[DELETE /v1/preview/models/{model_id}](https://docs.wandb.ai/training/api-reference/models/delete-model)** - 모델 삭제
* **[DELETE /v1/preview/models/{model_id}/checkpoints](https://docs.wandb.ai/training/api-reference/models/delete-model-checkpoints)** - 모델 체크포인트 삭제
* **[GET /v1/preview/models/{model_id}/checkpoints](https://docs.wandb.ai/training/api-reference/models/list-model-checkpoints)** - 모델 체크포인트 목록 조회
* **[POST /v1/preview/models/{model_id}/log](https://docs.wandb.ai/training/api-reference/models/log)** - 로그 기록

<div id="training-jobs">
  ### training-jobs
</div>

* **[POST /v1/preview/sft-training-jobs](https://docs.wandb.ai/training/api-reference/training-jobs/create-sft-training-job)** - SFT Training Job 생성
* **[POST /v1/preview/training-jobs](https://docs.wandb.ai/training/api-reference/training-jobs/create-rl-training-job)** - RL Training Job 생성
* **[GET /v1/preview/training-jobs/{training_job_id}](https://docs.wandb.ai/training/api-reference/training-jobs/get-training-job)** - Training Job 조회
* **[GET /v1/preview/training-jobs/{training_job_id}/events](https://docs.wandb.ai/training/api-reference/training-jobs/get-training-job-events)** - Training Job 이벤트 조회

<div id="health">
  ### 상태
</div>

* **[GET /v1/health](https://docs.wandb.ai/training/api-reference/health/health-check)** - 상태 점검
* **[GET /v1/system-check](https://docs.wandb.ai/training/api-reference/health/system-check)** - 시스템 점검
