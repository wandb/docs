### <kbd>method</kbd> `add`

```python
add(
    feedback_type: 'str',
    payload: 'dict[str, Any] | None' = None,
    creator: 'str | None' = None,
    annotation_ref: 'str | None' = None,
    **kwargs: 'dict[str, Any]'
) → str
```

ref에 피드백을 추가합니다. 

feedback_type: 피드백 유형을 식별하는 문자열입니다. "wandb." 접두사는 예약되어 있습니다. creator: 피드백 생성자로 표시될 이름입니다.
