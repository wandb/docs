[weave](../) / ScoreLogger

ScoreLogger는 단일 예측의 점수 산정을 관리합니다.
EvaluationLogger.logPrediction()에서 반환되는 객체입니다.

`Example`

```ts
const pred = await ev.logPrediction(example, output);
await pred.logScore("accuracy", 0.95);
await pred.logScore("relevance", 0.8);
await pred.finish(); // prediction을 완료합니다
```
