  ### addScore
</div>

▸ **addScore**(`predictCallId`, `scorerCallId`, `runnableRefUri`, `scorerOutput`): `Promise`&lt;`string`&gt;

Call에 스코어러 결과(예: 스코어러 출력)를 추가합니다.
명령형 Evaluation에서 predict Call에 스코어러 결과를 첨부할 때 사용합니다.

| `scorerCallId`   | `string` | 피드백을 생성한 scorer Call ID   |
| `runnableRefUri` | `string` | scorer(Op 또는 객체 ref)의 URI |
