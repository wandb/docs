[weave](../) / ObjectRef

저장된 Weave 오브젝트에 대한 참조를 나타냅니다.

일반적으로 최종 사용자는 이 클래스와 직접 상호작용할 필요가 없습니다.

ObjectRef에는 Weave의 스토리지 시스템에서 저장된 오브젝트를 고유하게 식별하는 프로젝트 ID, 오브젝트 ID 및 digest가 포함되어 있습니다.

`Example`

```ts
const ref = new ObjectRef('my-project', 'abc123', 'def456');
const uri = ref.uri(); // weave:///my-project/object/abc123:def456
```

### fromUri

▸ **fromUri**(`uri`): [`ObjectRef`](./objectref)

Weave URI 문자열로부터 ObjectRef를 생성합니다.

#### Parameters

| Name | Type | Description |
| :------ | :------ | :------ |
| `uri` | `string` | 다음과 같은 형식의 Weave URI: weave:///entity/project/object/name:digest |

#### Returns

[`ObjectRef`](./objectref)

새로운 ObjectRef 인스턴스

`Throws`

URI 형식이 유효하지 않거나 오브젝트 참조가 아닌 경우 Error를 발생시킵니다.

`Example`

```ts
const ref = ObjectRef.fromUri('weave:///my-entity/my-project/object/my-dataset:abc123');
```
