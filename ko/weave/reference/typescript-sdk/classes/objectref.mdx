[weave](../) / ObjectRef

저장된 Weave 객체에 대한 참조를 나타냅니다.

일반적으로 최종 사용자는 이 클래스를 직접 사용할 필요가 없습니다.

ObjectRef에는 Weave의 스토리지 시스템에서 저장된 객체를 고유하게 식별하는
프로젝트 ID, 객체 ID 및 다이제스트가 포함됩니다.

`Example`

```ts
const ref = new ObjectRef('my-project', 'abc123', 'def456');
const uri = ref.uri(); // weave:///my-project/object/abc123:def456
```
