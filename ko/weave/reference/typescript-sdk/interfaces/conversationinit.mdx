<div id="agentdescription">
  ### agentDescription
</div>

• `Optional` **agentDescription**: `string`

사람이 읽을 수 있는 에이전트 설명입니다. 기본 `agentDescription`으로 전달되며,
각 `Turn`에서 별도로 설정하지 않는 한 `startTurn()`를 통해 생성되는 모든 `Turn`에 적용됩니다.
또한 각 `Turn`의 `invoke_agent` span에서 `gen_ai.agent.description`으로 기록됩니다.

<div id="agentid">
  ### agentId
</div>

• `Optional` **agentId**: `string`

고정된 에이전트 식별자입니다. Turn에서 자체 `agentId`를 설정하지 않으면
`startTurn()`을 통해 생성되는 모든 `Turn`에 기본 `agentId`로 전파되며,
각 Turn의 `invoke_agent` span에 `gen_ai.agent.id`로 기록됩니다.

<div id="attributes">
  ### attributes
</div>

• `선택` **attributes**: `Attributes`

대화에서 생성되는 모든 span에 적용되는 맞춤형 속성입니다.

여기서 span 자체의 `gen_ai.*` / `weave.*`
속성과 충돌하는 키는 지원되지 않으며, 이 경우 span의 값이 우선합니다.
