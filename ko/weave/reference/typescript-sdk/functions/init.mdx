<div id="init">
  # init
</div>

▸ **init**(`project`, `settings?`): `Promise`\<[`WeaveClient`](../classes/weaveclient)\>

Weave tracing이 동작하는 데 필요한 Weave client를 초기화합니다.

<div id="parameters">
  #### 파라미터
</div>

| Name | Type | Description |
| :------ | :------ | :------ |
| `project` | `string` | W&B 프로젝트 이름입니다(프로젝트 또는 엔터티/project 형식 모두 가능). W&B 팀(예: `team/project`)을 지정하지 않으면 기본 엔터티가 사용됩니다. 기본 엔터티를 확인하거나 업데이트하려면 https://docs.wandb.ai/guides/models/app/settings-page/user-settings/#default-team 의 User Settings 페이지를 참고하세요. |
| `settings?` | `SettingsInit` | (선택 사항) Weave 트레이싱 설정입니다. |

<div id="returns">
  #### 반환값
</div>

`Promise`\<[`WeaveClient`](../classes/weaveclient)\>

초기화가 완료된 Weave 클라이언트로 resolve되는 Promise입니다.

`Throws`

초기화에 실패하면 예외를 throw합니다.
