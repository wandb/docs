<div id="endllm">
  # endLLM
</div>

▸ **endLLM**(`opts?`): `void`

현재 LLM을 종료합니다. 활성화된 LLM이 없으면 아무 작업도 하지 않습니다.

<div id="parameters">
  #### 매개변수
</div>

| 이름      | 유형               |
| :------ | :--------------- |
| `opts?` | `SpanEndOptions` |

<div id="returns">
  #### 반환값
</div>

`void`

`예시`

```ts twoslash
// @noErrors
weave.endLLM();
```

`예시`

```ts twoslash
// @noErrors
weave.endLLM({endTime: new Date('2026-05-29T10:00:00.800Z')});
```

`예시`

```ts twoslash
// @noErrors
weave.endLLM({error: new Error('llm call failed')});
```
