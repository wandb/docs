<div id="startturn">
  # startTurn
</div>

▸ **startTurn**(`opts?`): [`Turn`](../classes/turn)

새 Turn을 시작합니다. 활성 상태인 Conversation이 있으면 Turn이 해당
`conversationId`를 상속받고, 그렇지 않으면 Turn에는 conversation ID가 없습니다.

<div id="parameters">
  #### 매개변수
</div>

| 이름     | 유형                                   |
| :----- | :----------------------------------- |
| `opts` | [`TurnInit`](../interfaces/turninit) |

<div id="returns">
  #### 반환값
</div>

[`Turn`](../classes/turn)

`예시`

```ts twoslash
// @noErrors
weave.startTurn();
```

`예시`

```ts twoslash
// @noErrors
weave.startTurn({
  agentId: 'research-bot-prod',
  agentName: 'research-bot',
  agentDescription: 'Looks up facts on Wikipedia.',
  agentVersion: '1.4.2',
  model: 'gpt-4o',
  userMessage: 'What is the weather in Tokyo?',
  systemInstructions: ['You are a helpful weather bot.'],
  startTime: new Date('2026-05-29T10:00:00.000Z'),
});
```
