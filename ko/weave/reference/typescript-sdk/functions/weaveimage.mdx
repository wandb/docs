  # weaveImage
</div>

▸ **weaveImage**(`options`): [`WeaveImage`](../interfaces/weaveimage)

새 WeaveImage 객체를 만듭니다.

<div id="parameters">
  #### 파라미터
</div>

| 이름        | 유형                | 설명                                                                                                      |
| :-------- | :---------------- | :------------------------------------------------------------------------------------------------------ |
| `options` | `WeaveImageInput` | 이 미디어 타입의 옵션입니다. - data: Buffer 형태의 원시 이미지 데이터 - imageType: (선택) 이미지 파일 유형으로, 현재는 &#39;png&#39;만 지원됩니다. |

<div id="returns">
  #### 반환값
</div>

[`WeaveImage`](../interfaces/weaveimage)

`예제`

```ts
const imageBuffer = fs.readFileSync('path/to/image.png');
const weaveImage = weaveImage({ data: imageBuffer });
```
