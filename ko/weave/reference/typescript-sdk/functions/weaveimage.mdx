# weaveImage

▸ **weaveImage**(`options`): [`WeaveImage`](../interfaces/weaveimage)

새로운 WeaveImage 오브젝트를 생성합니다.

#### Parameters

| 이름 | 타입 | 설명 |
| :------ | :------ | :------ |
| `options` | `WeaveImageInput` | 이 미디어 타입에 대한 옵션들 - 데이터: Buffer 형태의 원본 이미지 데이터 - imageType: (선택 사항) 이미지 파일의 타입으로, 현재는 'png'만 지원됩니다. |

#### Returns

[`WeaveImage`](../interfaces/weaveimage)

`Example`

```ts
const imageBuffer = fs.readFileSync('path/to/image.png');
const weaveImage = weaveImage({ data: imageBuffer });
```
