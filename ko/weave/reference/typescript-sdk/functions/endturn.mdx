<div id="endturn">
  # endTurn
</div>

▸ **endTurn**(`opts?`): `void`

현재 Turn을 종료합니다. 활성 Turn이 없으면 아무 작업도 수행하지 않습니다.

<div id="parameters">
  #### 매개변수
</div>

| 이름      | 유형               |
| :------ | :--------------- |
| `opts?` | `SpanEndOptions` |

<div id="returns">
  #### 반환값
</div>

`void`

`예시`

```ts twoslash
// @noErrors
weave.endTurn();
```

`예시`

```ts twoslash
// @noErrors
weave.endTurn({endTime: new Date('2026-05-29T10:00:01.700Z')});
```

`예시`

```ts twoslash
// @noErrors
weave.endTurn({error: new Error('agent loop diverged')});
```
