<div id="wrapopenai">
  # wrapOpenAI
</div>

▸ **wrapOpenAI**\<`T`\>(`openai`): `T`

OpenAI 호출을 추적할 수 있도록 OpenAI API를 래핑합니다.

<div id="type-parameters">
  #### 타입 매개변수
</div>

| 이름 | 타입 |
| :------ | :------ |
| `T` | extends `OpenAIAPI` |

<div id="parameters">
  #### 매개변수
</div>

| 이름 | 타입 |
| :------ | :------ |
| `openai` | `T` |

<div id="returns">
  #### 반환값
</div>

`T`

`예시`

```ts
const openai = wrapOpenAI(new OpenAI());
const result = await openai.chat.completions.create({
  model: 'gpt-3.5-turbo',
  messages: [{ role: 'user', content: 'Hello, world!' }]
});
```
