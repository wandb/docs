<div id="withattributes">
  # withAttributes
</div>

▸ **withAttributes**&lt;`T`&gt;(`attrs`, `fn`): `Promise`&lt;`T`&gt; | `T`

현재 실행 컨텍스트에 attributes를 연결하여, `fn` 내부에서 생성되는 모든 Call이
자동으로 해당 attributes를 상속받도록 합니다. Attributes는 트레이스 서버의 Call
record에 기록되고 Weave UI/필터링에 노출되므로, Request ID, 테넌트, 실험 등으로
Runs를 태깅하는 데 이상적입니다.

예시:

```ts
await withAttributes({requestId: 'abc'}, async () => {
  await myOp();
});
```
