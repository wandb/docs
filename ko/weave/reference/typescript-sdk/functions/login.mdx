<div id="login">
  # login
</div>

▸ **login**(`apiKey`, `host?`): `Promise`&lt;`void`&gt;

제공된 API 키를 사용하여 Weights &amp; Biases (W&amp;B)에 로그인합니다.
이 함수는 추후 사용을 위해 자격 증명을 netrc 파일에 저장하려고 시도하지만,
파일 시스템에 쓸 수 없더라도 실행을 계속합니다.

<div id="parameters">
  #### 파라미터
</div>

| Name     | 유형       | 설명                                                         |
| :------- | :------- | :--------------------------------------------------------- |
| `apiKey` | `string` | W&amp;B API 키입니다.                                          |
| `host?`  | `string` | (선택) 호스트 이름입니다. 일반적으로 맞춤형 W&amp;B Server를 사용하는 경우에만 필요합니다. |

<div id="returns">
  #### 반환값
</div>

`Promise`&lt;`void`&gt;

`Throws`

API 키가 지정되지 않았거나 Weave 트레이스 서버에 대한 연결을 확인할 수 없는 경우 예외를 발생시킵니다.
