<div id="startllm">
  # startLLM
</div>

▸ **startLLM**(`opts`): [`LLM`](../classes/llm)

현재 Turn의 하위 span으로 LLM span을 시작합니다. 활성화된 Turn이 없으면 예외를 발생시킵니다.

<div id="parameters">
  #### 매개변수
</div>

| 이름     | 유형                                 |
| :----- | :--------------------------------- |
| `opts` | [`LLMInit`](../interfaces/llminit) |

<div id="returns">
  #### 반환값
</div>

[`LLM`](../classes/llm)

`예시`

```ts twoslash
// @noErrors
weave.startLLM({model: 'gpt-4o-mini', providerName: 'openai'});
```

`예시`

```ts twoslash
// @noErrors
weave.startLLM({
  model: 'gpt-4o-mini',
  providerName: 'openai',
  startTime: new Date('2026-05-29T10:00:00.000Z'),
});
```
