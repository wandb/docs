<div id="base-url">
  ## 기본 URL
</div>

[W&amp;B Multi-tenant Cloud](/ko/platform/hosting/hosting-options/multi_tenant_cloud) 및 [W&amp;B Dedicated Cloud](/ko/platform/hosting/hosting-options/dedicated-cloud)의 경우 다음 기본 URL을 사용하세요:

```
https://trace.wandb.ai
```

[W&amp;B 자체 관리형 인스턴스](/ko/weave/guides/platform/weave-self-managed)에서는 다음을 사용하세요:

```
https://<your-subdomain>.wandb.io/traces
```

<div id="authentication">
  ## 인증
</div>

요청 헤더에 W&amp;B API 키를 포함합니다:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" https://trace.wandb.ai/...
```
