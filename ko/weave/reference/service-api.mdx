<div id="base-url">
  ## 기본 URL
</div>

```
https://trace.wandb.ai
```


<div id="authentication">
  ## 인증
</div>

요청 헤더에 W&amp;B API 키를 포함합니다:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" https://trace.wandb.ai/...
```

<div id="calls">
  ### 호출
</div>

- **[POST /call/end](https://docs.wandb.ai/weave/reference/service-api/calls/call-end)** - 호출 종료
- **[POST /call/read](https://docs.wandb.ai/weave/reference/service-api/calls/call-read)** - 호출 조회
- **[POST /call/start](https://docs.wandb.ai/weave/reference/service-api/calls/call-start)** - 호출 시작
- **[POST /call/update](https://docs.wandb.ai/weave/reference/service-api/calls/call-update)** - 호출 업데이트
- **[POST /call/upsert_batch](https://docs.wandb.ai/weave/reference/service-api/calls/call-start-batch)** - 호출 일괄 시작
- **[POST /calls/delete](https://docs.wandb.ai/weave/reference/service-api/calls/calls-delete)** - 호출 삭제
- **[POST /calls/query_stats](https://docs.wandb.ai/weave/reference/service-api/calls/calls-query-stats)** - 호출 통계 쿼리
- **[POST /calls/stream_query](https://docs.wandb.ai/weave/reference/service-api/calls/calls-query-stream)** - 호출 스트림 쿼리
