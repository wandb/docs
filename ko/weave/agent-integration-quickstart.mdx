W&amp;B Weave는 각 턴을 일일이 수동 계측하지 않아도, 널리 사용되는 SDK와 하니스로 구축된 멀티턴 에이전트를 트레이싱합니다. 에이전트 하니스용 플러그인을 설치하거나 지원되는 에이전트 SDK를 사용하는 코드에서 `weave.init()`를 호출하면, Weave가 프레임워크를 자동으로 패치합니다. 세션, 턴, LLM Call, 도구 Call은 프로젝트의 **Agents** 뷰에 표시됩니다.
