<Info>
  **사전 준비 사항**: 진행하기 전에 Node.js(버전 19 이상)를 설치하세요. <br />
  진행하기 전에 `docs.json`으로 업그레이드한 뒤, 기존 `mint.json` 파일을 삭제하세요.
</Info>

다음 단계에 따라 사용 중인 운영 체제에서 Mintlify를 설치하고 실행하세요:

**1단계**: Mintlify 설치:

<CodeGroup>
  ```bash npm
  npm i -g mintlify
  ```

  ```bash yarn
  yarn global add mintlify
  ```
</CodeGroup>

**2단계**: docs 디렉터리(`docs.json` 파일이 위치한 곳)로 이동한 다음, 다음 명령을 실행하세요:

```bash
mintlify dev
```

문서의 로컬 미리보기는 `http://localhost:3000`에서 확인할 수 있습니다.


<div id="custom-ports">
  ### 사용자 지정 포트
</div>

기본적으로 Mintlify는 포트 3000을 사용합니다. `--port` 플래그를 사용해 Mintlify가 실행되는 포트를 사용자 지정할 수 있습니다. 예를 들어 Mintlify를 포트 3333에서 실행하려면 다음 명령을 사용하세요:

```bash
mintlify dev --port 3333
```

이미 사용 중인 포트로 Mintlify를 실행하려고 하면, Mintlify는 다음으로 사용 가능한 포트를 사용합니다:

```md
Port 3000 is already in use. Trying 3001 instead.
```


<div id="mintlify-versions">
  ## Mintlify 버전
</div>

각 CLI 릴리스는 특정 Mintlify 버전에 대응됩니다. 로컬 웹사이트가 운영(프로덕션) 버전과 일치하지 않는 경우, 아래 명령으로 CLI를 업데이트하세요:

<CodeGroup>

```bash npm
npm i -g mintlify@latest
```

```bash yarn
yarn global upgrade mintlify
```

</CodeGroup>

<div id="validating-links">
  ## 링크 유효성 검사
</div>

CLI를 사용하면 문서에 포함된 참조 링크를 검증할 수 있습니다. 끊어진 링크를 찾으려면 다음 명령을 사용하세요:

```bash
mintlify broken-links
```

<div id="code-formatting">
  ## 코드 포맷팅
</div>

IDE에서 MDX를 인식하고 포맷팅할 수 있도록 확장 프로그램을 사용하는 것을 권장합니다. VS Code를 사용한다면 문법 하이라이팅을 위해 [MDX VSCode extension](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx), 코드 포맷팅을 위해 [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)를 사용하는 것을 고려하세요.

<div id="troubleshooting">
  ## 문제 해결
</div>

<AccordionGroup>
  <Accordion title='오류: darwin-arm64 런타임을 사용하여 "sharp" 모듈을 로드할 수 없습니다'>

    오래된 버전의 Node.js를 사용하고 있어서 발생했을 수 있습니다. 다음을 시도하세요:
    1. 현재 설치된 mintlify 버전 제거: `npm remove -g mintlify`
    2. Node v19 이상으로 업그레이드합니다.
    3. mintlify를 다시 설치합니다: `npm install -g mintlify`
  </Accordion>

  <Accordion title="문제: 알 수 없는 오류가 발생함">
  
    해결 방법: 시스템의 루트 디렉터리로 이동한 다음 \~/.mintlify 폴더를 삭제하세요. 이후 `mintlify dev`를 다시 실행합니다.
  </Accordion>
</AccordionGroup>

CLI 버전에서 무엇이 변경되었는지 궁금하다면 [CLI 변경 로그를 확인하세요.](https://www.npmjs.com/package/mintlify?activeTab=versions)
