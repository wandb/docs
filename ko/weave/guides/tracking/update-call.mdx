대부분의 Call 속성은 생성 후에는 변경할 수 없습니다. 지원되는 변경 작업은 다음과 같습니다.

* [표시 이름 설정](#set-display-name)
* [피드백 추가](#add-feedback)
* [Call 삭제](#delete-a-call)

UI에서 Call 세부 정보 페이지로 이동해 이러한 모든 변경 작업을 수행할 수 있습니다.
웹 앱에서 Call을 업데이트하려면 다음 단계를 따르세요.

1. [wandb.ai](https://wandb.ai/)로 이동한 다음 프로젝트를 선택합니다.
2. Weave 프로젝트 사이드바에서 **Traces**를 클릭합니다.
3. 테이블에서 보려는 Call을 찾습니다.
4. Call을 클릭해 세부 정보 페이지를 엽니다.
5. Call 세부 정보의 탭 바에서 `Feedback` 탭을 클릭합니다.

여기에서 Call의 표시 이름을 수정하고, 피드백을 추가하거나, Call을 삭제할 수 있습니다.

<div id="set-display-name">
  ### 표시 이름 설정
</div>

<Tabs>
  <Tab title="Python">
    Call의 표시 이름을 설정하려면 [`Call.set_display_name()`](/ko/weave/reference/python-sdk/trace/weave_client#method-set-display-name) 방법을 사용하세요.
  </Tab>

  <Tab title="TypeScript">
    Call의 표시 이름을 설정하려면 [`client.updateCall`](/ko/weave/reference/typescript-sdk/classes/weaveclient#updatecall)을 사용해 call ID로 직접 업데이트하세요.
  </Tab>

  <Tab title="HTTP API">
    Service API를 사용해 Call의 표시 이름을 설정하려면 [`/call/update`](https://docs.wandb.ai/weave/reference/service-api/calls/call-update) 엔드포인트에 요청을 보내세요.

    ```bash lines
    curl -L 'https://trace.wandb.ai/call/update' \
    -H 'Content-Type: application/json' \
    -H 'Accept: application/json' \
    -d '{
        "project_id": "string",
        "call_id": "string",
        "display_name": "string",
    }'
    ```
  </Tab>
</Tabs>

[실행 시점에 Call의 표시 이름을 설정](/ko/weave/guides/tracking/get-call-object)할 수도 있습니다.

<div id="delete-a-call">
  ### Call 삭제
</div>

<Tabs>
  <Tab title="Python">
    Python API를 사용해 Call을 삭제하려면 [`Call.delete`](/ko/weave/reference/python-sdk/trace/weave_client#method-delete)를 사용하세요.
  </Tab>

  <Tab title="TypeScript">
    ```plaintext lines
    이 기능은 아직 TypeScript SDK에서 사용할 수 없습니다.
    ```
  </Tab>

  <Tab title="HTTP API">
    Service API를 사용해 Call을 삭제하려면 [`/calls/delete`](https://docs.wandb.ai/weave/reference/service-api/calls/calls-delete) 엔드포인트로 요청을 보내세요.

    ```bash lines
    curl -L 'https://trace.wandb.ai/calls/delete' \
    -H 'Content-Type: application/json' \
    -H 'Accept: application/json' \
    -d '{
        "project_id": "string",
        "call_ids": [
            "string"
        ],
    }'
    ```
  </Tab>
</Tabs>

<div id="delete-multiple-calls">
  ### 여러 Call 삭제
</div>

<Tabs>
  <Tab title="Python">
    Python API를 사용해 여러 Call을 삭제하려면 `delete_calls()`에 Call ID 목록을 전달하세요.

    <Important>
      * 삭제할 수 있는 최대 Call 수는 `1000`개입니다.
      * Call을 삭제하면 해당 Call의 모든 하위 Call도 함께 삭제됩니다.
    </Important>
  </Tab>

  <Tab title="TypeScript">
    ```plaintext lines
    이 기능은 아직 TypeScript SDK에서 지원되지 않습니다.
    ```
  </Tab>
</Tabs>
