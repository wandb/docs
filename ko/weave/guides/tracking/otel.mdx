<div id="overview">
  ## Overview
</div>

WeaveлҠ” м „мҡ© м—”л“ңнҸ¬мқёнҠёлҘј нҶөн•ҙ OpenTelemetry нҳёнҷҳ нҠёл ҲмқҙмҠӨ лҚ°мқҙн„°лҘј мҲҳм§‘н•  мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙ м—”л“ңнҸ¬мқёнҠёлҘј мӮ¬мҡ©н•ҳл©ҙ OTLP(OpenTelemetry Protocol) нҳ•мӢқмқҳ нҠёл ҲмқҙмҠӨ лҚ°мқҙн„°лҘј Weave н”„лЎңм қнҠёлЎң м§Ғм ‘ м „мҶЎн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

<div id="endpoint-details">
  ## м—”л“ңнҸ¬мқёнҠё м„ёл¶Җ м •ліҙ
</div>

**Path**: `/otel/v1/traces`
**Method**: POST
**Content-Type**: `application/x-protobuf`
**Base URL**: OTel нҠёл ҲмқҙмҠӨ м—”л“ңнҸ¬мқёнҠёмқҳ кё°ліё URLмқҖ W&amp;B л°°нҸ¬ мң нҳ•м—җ л”°лқј лӢ¬лқјм§‘лӢҲлӢӨ.

* Multi-tenant Cloud:
  `https://trace.wandb.ai/otel/v1/traces`

* Dedicated Cloud л°Ҹ Self-Managed мқёмҠӨн„ҙмҠӨ:
  `https://<your-subdomain>.wandb.io/traces/otel/v1/traces`

`<your-subdomain>`мқ„ мЎ°м§Ғмқҳ кі мң н•ң W&amp;B лҸ„л©”мқёмңјлЎң л°”кҫём„ёмҡ”. мҳҲ: `acme.wandb.io`.

<div id="authentication-and-routing">
  ## мқёмҰқ л°Ҹ лқјмҡ°нҢ…
</div>

`wandb-api-key` н—ӨлҚ”м—җ W&amp;B API нӮӨлҘј м „лӢ¬н•ң нӣ„, `TracerProvider` нҒҙлһҳмҠӨм—җм„ң лӢӨмқҢ нӮӨлҘј OpenTelemetry лҰ¬мҶҢмҠӨ мҶҚм„ұмңјлЎң м§Җм •н•©лӢҲлӢӨ:

* `wandb.entity`: W&amp;B нҢҖ мқҙлҰ„ лҳҗлҠ” мӮ¬мҡ©мһҗ мқҙлҰ„мһ…лӢҲлӢӨ.
* `wandb.project`: нҠёл ҲмқҙмҠӨлҘј ліҙлӮј н”„лЎңм қнҠё мқҙлҰ„мһ…лӢҲлӢӨ.

лӢӨмқҢ мҳҲм ңлҠ” мқёмҰқкіј н”„лЎңм қнҠё лқјмҡ°нҢ…мқ„ кө¬м„ұн•ҳлҠ” л°©лІ•мқ„ ліҙм—¬мӨҚлӢҲлӢӨ:

3. Project Name: мһ¬лҜёмһҲлҠ” мқҙлҰ„мңјлЎң м§Җм–ҙ ліҙм„ёмҡ”!
4. `OPENAI_API_KEY`: [OpenAI dashboard](https://platform.openai.com/api-keys)м—җм„ң нҷ•мқён•  мҲҳ мһҲмҠөлӢҲлӢӨ.

<Warning>
  **м„ұлҠҘ к¶ҢмһҘ мӮ¬н•ӯ**: WeaveлЎң нҠёл ҲмқҙмҠӨлҘј м „мҶЎн•  л•ҢлҠ” н•ӯмғҒ `SimpleSpanProcessor` лҢҖмӢ  `BatchSpanProcessor`лҘј мӮ¬мҡ©н•ҳм„ёмҡ”. `SimpleSpanProcessor`лҠ” мҠӨнҢ¬мқ„ лҸҷкё°мӢқмңјлЎң лӮҙліҙлӮҙлҜҖлЎң лӢӨлҘё мӣҢнҒ¬лЎңл“ңмқҳ м„ұлҠҘм—җ мҳҒн–Ҙмқ„ мӨ„ мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙ мҳҲм ңл“Өм—җм„ңлҠ” `BatchSpanProcessor`лҘј мӮ¬мҡ©н•©лӢҲлӢӨ. мқҙ н”„лЎңм„ём„ңлҠ” мҠӨнҢ¬мқ„ л№„лҸҷкё°м ҒмңјлЎң нҡЁмңЁм ҒмңјлЎң мқјкҙ„ мІҳлҰ¬н•ҳлҜҖлЎң н”„лЎңлҚ•м…ҳ нҷҳкІҪм—җм„ң к¶ҢмһҘлҗ©лӢҲлӢӨ.
</Warning>

<Tabs>
  <Tab title="Python">
    лӢӨмқҢ мҪ”л“ңлҘј `openinference_example.py`мҷҖ к°ҷмқҖ Python нҢҢмқјм—җ л¶ҷм—¬ л„Јмңјм„ёмҡ”:

    ```python lines
    import os
    import openai
    from opentelemetry.exporter.otlp.proto.http.trace_exporter import OTLPSpanExporter
    from opentelemetry.sdk import trace as trace_sdk
    from opentelemetry.sdk.resources import Resource
    from opentelemetry.sdk.trace.export import ConsoleSpanExporter, BatchSpanProcessor
    from openinference.instrumentation.openai import OpenAIInstrumentor

    OPENAI_API_KEY = "YOUR_OPENAI_API_KEY"
    WANDB_BASE_URL = "https://trace.wandb.ai"
    ENTITY = "<your-team-name>"
    PROJECT = "<your-project-name>"

    OTEL_EXPORTER_OTLP_ENDPOINT = f"{WANDB_BASE_URL}/otel/v1/traces"

    # https://wandb.ai/settings м—җм„ң API нӮӨлҘј мғқм„ұн•ҳм„ёмҡ”.
    WANDB_API_KEY = os.environ["WANDB_API_KEY"]

    exporter = OTLPSpanExporter(
        endpoint=OTEL_EXPORTER_OTLP_ENDPOINT,
        headers={"wandb-api-key": WANDB_API_KEY},
    )

    tracer_provider = trace_sdk.TracerProvider(resource=Resource({
        "wandb.entity": ENTITY,
        "wandb.project": PROJECT,
    }))
    tracer_provider.add_span_processor(BatchSpanProcessor(exporter))

    # м„ нғқ мӮ¬н•ӯ: мҠӨнҢ¬мқ„ мҪҳмҶ”м—җ м¶ңл Ҙн•©лӢҲлӢӨ.
    tracer_provider.add_span_processor(BatchSpanProcessor(ConsoleSpanExporter()))

    OpenAIInstrumentor().instrument(tracer_provider=tracer_provider)

    def main():
        client = openai.OpenAI(api_key=OPENAI_API_KEY)
        response = client.chat.completions.create(
            model="gpt-3.5-turbo",
            messages=[{"role": "user", "content": "Describe OTel in a single sentence."}],
            max_tokens=20,
            stream=True,
            stream_options={"include_usage": True},
        )
        for chunk in response:
            if chunk.choices and (content := chunk.choices[0].delta.content):
                print(content, end="")

    if __name__ == "__main__":
        main()
    ```

  <Tab title="TypeScript">
    мқҙ мҳҲм ңмқҳ TypeScript кө¬нҳ„мқҖ Python кө¬нҳ„кіј л№„көҗн•  л•Ң лӢӨмқҢкіј к°ҷмқҖ н•өмӢ¬ м°Ёмқҙк°Җ мһҲмҠөлӢҲлӢӨ:

    * кі„мёЎмқ„ л“ұлЎқн•ҳкё° м „м—җ OpenAIлҘј лЁјм Җ мһ„нҸ¬нҠён•ҙм•ј н•©лӢҲлӢӨ(ESM лӘЁл“Ҳм—җм„ңлҠ” мқҙкІғмқҙ н•„мҡ”н•©лӢҲлӢӨ).
    * W&amp;Bмқҳ м—”л“ңнҸ¬мқёнҠёлҠ” protobufл§Ң н—Ҳмҡ©н•ҳлҜҖлЎң HTTP exporter лҢҖмӢ  `@opentelemetry/exporter-trace-otlp-proto`(protobuf нҳ•мӢқ)лҘј мӮ¬мҡ©н•©лӢҲлӢӨ.
    * `BatchSpanProcessor`лҠ” л№„лҸҷкё°м ҒмңјлЎң н”Ңлҹ¬мӢңлҗҳлҜҖлЎң, мҠӨнҢ¬мқҙ нҷ•мӢӨнһҲ н”Ңлҹ¬мӢңлҗҳлҸ„лЎқ мў…лЈҢ м „м—җ м§Җм—°мқ„ л‘җкі  `provider.shutdown()`мқ„ лӘ…мӢңм ҒмңјлЎң нҳём¶ңн•ҙм•ј н•©лӢҲлӢӨ.

    лӢӨмқҢ мҪ”л“ңлҘј `openinference_example.ts` к°ҷмқҖ TypeScript нҢҢмқјм—җ л¶ҷм—¬ л„Јмңјм„ёмҡ”:

    ```typescript lines {11,12}
    // мӨ‘мҡ”: кі„мёЎмқҙ нҢЁм№ҳн•  мҲҳ мһҲлҸ„лЎқ OpenAIлҘј лЁјм Җ мһ„нҸ¬нҠён•ҳм„ёмҡ”
    import OpenAI from "openai";
    import { NodeTracerProvider } from "@opentelemetry/sdk-trace-node";
    import { BatchSpanProcessor } from "@opentelemetry/sdk-trace-base";
    import { OTLPTraceExporter } from "@opentelemetry/exporter-trace-otlp-proto";
    import { resourceFromAttributes } from "@opentelemetry/resources";
    import { OpenAIInstrumentation, isPatched } from "@arizeai/openinference-instrumentation-openai";

    const OPENAI_API_KEY = process.env.OPENAI_API_KEY;
    const WANDB_BASE_URL = "https://trace.wandb.ai";
    const ENTITY = "<your-team-name>";
    const PROJECT = "<your-project-name>";

    const OTEL_EXPORTER_OTLP_ENDPOINT = `${WANDB_BASE_URL}/otel/v1/traces`;

    // https://wandb.ai/settings м—җм„ң API нӮӨлҘј мғқм„ұн•ҳм„ёмҡ”
    const WANDB_API_KEY = process.env.WANDB_API_KEY!;

    const exporter = new OTLPTraceExporter({
      url: OTEL_EXPORTER_OTLP_ENDPOINT,
      headers: { "wandb-api-key": WANDB_API_KEY },
    });

    const provider = new NodeTracerProvider({
      resource: resourceFromAttributes({
        "wandb.entity": ENTITY,
        "wandb.project": PROJECT,
      }),
      spanProcessors: [
        new BatchSpanProcessor(exporter)
      ],
    });

    provider.register();

    // tracer providerм—җ OpenAI кі„мёЎмқ„ л“ұлЎқн•©лӢҲлӢӨ
    const openAIInstrumentation = new OpenAIInstrumentation();
    openAIInstrumentation.setTracerProvider(provider);

    // ESMмқ„ мӮ¬мҡ©н•ҳлҜҖлЎң OpenAIлҘј мҲҳлҸҷмңјлЎң кі„мёЎн•©лӢҲлӢӨ
    openAIInstrumentation.manuallyInstrument(OpenAI);

    async function main() {
      console.log("OpenAIк°Җ нҢЁм№ҳлҗҳм—ҲлӮҳмҡ”?", isPatched());

      const client = new OpenAI({ apiKey: OPENAI_API_KEY });

      console.log("OpenAI API нҳём¶ң мӨ‘...");
      const response = await client.chat.completions.create({
        model: "gpt-3.5-turbo",
        messages: [{ role: "user", content: "Describe OTel in a single sentence." }],
        max_tokens: 50,
      });

      console.log("мқ‘лӢө:", response.choices[0]?.message?.content);
      console.log("мҠӨнҢ¬ н”Ңлҹ¬мӢң лҢҖкё° мӨ‘...");
    }

    (async () => {
      await main();

      // мҠӨнҢ¬мқ„ н”Ңлҹ¬мӢңн•  мӢңк°„мқ„ нҷ•ліҙн•©лӢҲлӢӨ
      console.log("мҠӨнҢ¬ н”Ңлҹ¬мӢңлҘј мң„н•ҙ 2мҙҲ лҢҖкё° мӨ‘...");
      await new Promise(resolve => setTimeout(resolve, 2000));

      await provider.shutdown(); // мў…лЈҢ м „ лҢҖкё° мӨ‘мқё лӘЁл“  мҠӨнҢ¬мқ„ н”Ңлҹ¬мӢңн•©лӢҲлӢӨ
      console.log("мў…лЈҢ мҷ„лЈҢ");
    })();
    ```

    ```python lines
    OpenAIInstrumentor().instrument(tracer_provider=tracer_provider)

    def main():
        client = openai.OpenAI(api_key=OPENAI_API_KEY)
        response = client.chat.completions.create(
            model="gpt-3.5-turbo",
            messages=[{"role": "user", "content": "Describe OTel in a single sentence."}],
            max_tokens=20,
            stream=True,
            stream_options={"include_usage": True},
        )
    ```

<div id="use-an-opentelemetry-collector">
  ## OpenTelemetry Collector мӮ¬мҡ©н•ҳкё°
</div>

мң„ мҳҲмӢңм—җм„ңлҠ” м• н”ҢлҰ¬мјҖмқҙм…ҳм—җм„ң WeaveлЎң нҠёл ҲмқҙмҠӨлҘј м§Ғм ‘ м „мҶЎн•©лӢҲлӢӨ. н”„лЎңлҚ•м…ҳ нҷҳкІҪм—җм„ңлҠ” м• н”ҢлҰ¬мјҖмқҙм…ҳкіј Weave мӮ¬мқҙмқҳ мӨ‘к°„ кі„мёөмңјлЎң [OpenTelemetry Collector](https://opentelemetry.io/docs/collector/)лҘј мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ. CollectorлҠ” м•ұм—җм„ң нҠёл ҲмқҙмҠӨлҘј мҲҳмӢ н•ң лӢӨмқҢ н•ҳлӮҳ мқҙмғҒмқҳ л°ұм—”л“ңлЎң м „лӢ¬н•©лӢҲлӢӨ.

<div id="set-up-a-collector">
  ### collector м„Өм •н•ҳкё°
</div>

лӢӨмқҢ мҳҲм ңм—җм„ңлҠ” лӢӨмқҢмқ„ мҲҳн–үн•ҳлҠ” л°©лІ•мқ„ ліҙм—¬мӨҚлӢҲлӢӨ:

* OTLP нҠёл ҲмқҙмҠӨлҘј мҲҳмӢ н•ҳкі , мқҙлҘј мқјкҙ„ мІҳлҰ¬н•ң л’Ө WeaveлЎң м „лӢ¬н•ҳлҠ” лЎңм»¬ м„ңлІ„(collector)лҘј л°°нҸ¬н•ҳлҠ” Docker м„Өм • нҢҢмқјмқ„ м„Өм •н•©лӢҲлӢӨ.
* DockerлҘј мӮ¬мҡ©н•ҙ лЎңм»¬м—җм„ң collectorлҘј мӢӨн–үн•©лӢҲлӢӨ.
* лҸ„м»Ө м»Ён…Ңмқҙл„Ҳм—җм„ң мӢӨн–ү мӨ‘мқё collectorлЎң нҠёл ҲмқҙмҠӨлҘј м „лӢ¬н•ҳлҠ” кё°ліё OpenAI callмқ„ м „мҶЎн•©лӢҲлӢӨ.

collectorлҘј мӮ¬мҡ©н•ҳл Өл©ҙ лЁјм Җ collectorк°Җ OTLP нҠёл ҲмқҙмҠӨлҘј мҲҳмӢ н•ҳкі  мқҙлҘј WeaveлЎң лӮҙліҙлӮҙлҸ„лЎқ кө¬м„ұн•ҳлҠ” `collector-config.yaml` нҢҢмқјмқ„ л§Ңл“ңм„ёмҡ”:

```yaml lines {23,26} collector-config.yaml title="collector-config.yaml" 
receivers:
  otlp:
    protocols:
      http:
        endpoint: 0.0.0.0:4318

exporters:
  otlphttp/weave:
    endpoint: ${env:WANDB_OTLP_ENDPOINT}
    headers:
      wandb-api-key: ${env:WANDB_API_KEY}
    sending_queue:
      batch:

processors:
  resource:
    attributes:
      - key: wandb.entity # лҰ¬мҶҢмҠӨ мҶҚм„ұ н•„л“ң
        value: ${env:DEFAULT_WANDB_ENTITY}  # мЈјмһ…н•  к°’
        action: insert # м•„м§Ғ м„Өм •лҗҳм§Җ м•ҠмқҖ кІҪмҡ°м—җл§Ң мӮҪмһ…
      - key: wandb.project
        value: ${env:DEFAULT_WANDB_PROJECT}
        action: insert 

service:
  pipelines:
    traces:
      receivers: [otlp]
      processors: [resource]
      exporters: [otlphttp/weave]
```

мқҙ м„Өм • нҢҢмқјмқҖ лӢӨмқҢмқ„ мҲҳн–үн•©лӢҲлӢӨ.

* нҸ¬нҠё `4318`(HTTP)м—җм„ң OTLP нҠёл ҲмқҙмҠӨлҘј мҲҳмӢ н•©лӢҲлӢӨ.
* `wandb-api-key` н—ӨлҚ”лҘј мӮ¬мҡ©н•ҙ Weaveмқҳ OTLP м—”л“ңнҸ¬мқёнҠёлЎң нҠёл ҲмқҙмҠӨлҘј лӮҙліҙлӮҙл©°, м—”л“ңнҸ¬мқёнҠё URLмқҖ `WANDB_OTLP_ENDPOINT`м—җм„ң, API нӮӨлҠ” `WANDB_API_KEY`м—җм„ң мқҪм–ҙмҳөлӢҲлӢӨ.
* `resource` н”„лЎңм„ём„ңлҘј мӮ¬мҡ©н•ҙ `wandb.entity`мҷҖ `wandb.project`лҘј лҰ¬мҶҢмҠӨ мҶҚм„ұмңјлЎң м„Өм •н•ҳкі , к°’мқҖ `DEFAULT_WANDB_ENTITY`мҷҖ `DEFAULT_WANDB_PROJECT`м—җм„ң мқҪм–ҙмҳөлӢҲлӢӨ. `insert` actionмқҖ м• н”ҢлҰ¬мјҖмқҙм…ҳ мҪ”л“ңм—җм„ң мқҙлҹ¬н•ң мҶҚм„ұмқ„ мқҙлҜё м„Өм •н•ҳм§Җ м•ҠмқҖ кІҪмҡ°м—җл§Ң мҶҚм„ұмқ„ мЈјмһ…н•©лӢҲлӢӨ.
* л„ӨнҠёмӣҢнҒ¬ мҳӨлІ„н—Өл“ңлҘј мӨ„мқҙкё° мң„н•ҙ exporterм—җ лӮҙмһҘлҗң `sending_queue`лҘј л°°м№ҳ мІҳлҰ¬мҷҖ н•Ёк»ҳ нҷңм„ұнҷ”н•©лӢҲлӢӨ.

collector м„Өм •мқ„ мҷ„лЈҢн•ң нӣ„, лӢӨмқҢ Docker commandм—җм„ң API л°Ҹ entity к°’мқ„ м—…лҚ°мқҙнҠён•ң л’Ө мӢӨн–үн•©лӢҲлӢӨ:

```bash lines {3,5}
docker run \
  -v ./config.yaml:/etc/otelcol-contrib/config.yaml \
  -e WANDB_API_KEY="<your-wandb-api-key>" \
  -e WANDB_OTLP_ENDPOINT="https://trace.wandb.ai/otel" \
  -e DEFAULT_WANDB_ENTITY="<your-team-name>" \
  -e DEFAULT_WANDB_PROJECT="YOUR_PROJECT" \
  -p 4318:4318 \
  otel/opentelemetry-collector-contrib:latest
```

лӢӨмқҢ мҳҲм ңлҠ” OTel нҠёл ҲмқҙмҠӨлҘј Weave мҠӨл Ҳл“ңлЎң м •лҰ¬н•ҳлҠ” л°©лІ•мқ„ ліҙм—¬мӨҚлӢҲлӢӨ. `wandb.thread_id`лҘј мӮ¬мҡ©н•ҙ кҙҖл Ё мҳӨнҚјл Ҳмқҙм…ҳмқ„ к·ёлЈ№нҷ”н•ҳкі , `wandb.is_turn`мқ„ мӮ¬мҡ©н•ҙ thread viewм—җм„ң н–үмңјлЎң н‘ңмӢңлҗҳлҠ” мғҒмң„ мҲҳмӨҖ мҳӨнҚјл Ҳмқҙм…ҳмқ„ н‘ңмӢңн•©лӢҲлӢӨ.
