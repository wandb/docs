title: "트레이스 플롯 사용하기"
description: "Weave에서 인터랙티브 차트로 트레이스 단위 메트릭을 시각화하고 디버깅하여, 기본 및 사용자 지정 플롯을 통해 지연 시간, 비용, 토큰 사용 패턴을 분석하고 LLM 애플리케이션 성능을 최적화합니다."
---

W&B Weave의 _trace plots_ 도구를 사용하면 인터랙티브 차트를 통해 트레이스 단위 메트릭(예: 지연 시간, 비용, 토큰)을 탐색하고 시각화하며 디버깅할 수 있습니다. 기본 트레이스 플롯을 사용하거나, 사용자 지정 트레이스 플롯 빌더를 통해 직접 플롯을 만들 수 있습니다.

<Frame>
![트레이스 플롯 예시](/weave/guides/tracking/imgs/plots-example.png)
</Frame>

<div id="get-started">
  ## 시작하기
</div>

1. 프로젝트의 **Traces** 페이지로 이동합니다.
2. (선택 사항) **Filter**를 선택해 특정 trace만 필터링합니다(예: 날짜/시간 또는 operation 기준).
3. **Traces** 뷰의 오른쪽 상단에서 **Show Metrics** 아이콘을 클릭해 trace 플롯 사이드 패널을 엽니다. 

    ![The  **Show Metrics** icon](/weave/guides/tracking/imgs/plots-show-metrics-icon.png)

    여기에서 다음을 수행할 수 있습니다:
   - [기본 trace 플롯](#default-trace-plots)을 확인합니다.
   - [사용자 정의 trace 플롯](#create-a-custom-trace-plot)을 생성합니다.
4. 차트는 trace 필터와 선택에 따라 동적으로 업데이트됩니다.

<div id="default-trace-plots">
  ## 기본 트레이스 플롯
</div>

처음 트레이스 플롯 패널을 열면 Weave가 사용 가능한 프로젝트 트레이스 데이터를 기반으로 몇 가지 기본 트레이스 플롯을 자동으로 생성합니다:

- 막대 차트(시간 구간별 비용 또는 지연 시간 그룹화)
- 선 차트(시간에 따른 지연 시간)
- 산점도(예: prompt tokens vs. completion tokens)

각 트레이스 플롯은 다음과 같이 상호작용할 수 있습니다:

- 마우스를 올려 툴팁 확인
- 드래그하여 확대/축소
- 더블 클릭하여 초기 상태로 재설정
- 산점도에서 포인트를 클릭해 특정 트레이스를 열기

<div id="create-a-custom-trace-plot">
  ## 사용자 정의 trace plot 생성
</div>

사용자 정의 trace plot도 생성할 수 있습니다. 사용자 정의 trace plot을 생성하려면 다음을 수행하세요:

1. trace plots 사이드 패널에서 **➕ Add Chart** 를 클릭합니다.

    ![**➕ Add Chart** 버튼.](/weave/guides/tracking/imgs/plots-add-chart.png)

2. 팝업 창에서 사용 가능한 trace plot 유형 중 하나를 선택합니다:

    ![사용 가능한 사용자 정의 trace plot 유형](/weave/guides/tracking/imgs/plots-custom-chart-types.png)
    
3. 선택한 trace plot 유형에 맞게 trace plot을 구성합니다. trace plot 유형별 구성 옵션에 대한 자세한 내용은 [플롯 유형별 trace plot 설정](#trace-plot-settings-by-plot-type)을 참고하세요.
4. 차트를 저장하려면 **Save chart** 를 클릭합니다.

<div id="trace-plot-settings-by-plot-type">
  ### 플롯 유형별 트레이스 플롯 설정
</div>

사용자 지정 트레이스 플롯을 추가하거나 편집할 때, 사용 가능한 구성 옵션은 선택한 트레이스 플롯 유형에 따라 조금씩 다릅니다. 아래 표는 각 유형에서 설정할 수 있는 옵션을 정리한 것입니다.

| 설정           | 산점도(Scatter Plot) | 꺾은선형 차트(Line Chart) | 막대 차트(Bar Chart)     |
| --------------- | -------------------- | ------------------------- | ------------------------ |
| Y축           | ✅ 필수              | ✅ 필수                  | ✅ 필수                  |
| X축           | ✅ 선택 가능          | 고정 (`started at`)      | 고정 (`started at`)      |
| Grouping    | ✅ 선택 사항          | ✅ 선택 사항              | ✅ 선택 사항              |
| Binning     | ❌ 사용 안 함         | ✅ 사용                   | ✅ 사용                   |
| Aggregation | ❌ 사용 안 함         | ✅ 사용                   | ✅ 사용                   |
