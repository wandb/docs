title: "트레이스 플롯 사용하기"
description: "Weave에서 대화형 차트를 사용해 트레이스 수준 메트릭을 시각화하고 디버깅하여, 기본 제공 및 사용자 지정 플롯으로 지연 시간, 비용, 토큰 사용 패턴을 탐색하고 LLM 애플리케이션 성능을 최적화합니다."
---

W&B Weave의 _trace plots_ 도구를 사용하면 대화형 차트를 통해 트레이스 수준 메트릭(예: 지연 시간, 비용, 토큰)을 탐색하고 시각화하며 디버깅할 수 있습니다. 기본 트레이스 플롯을 사용할 수도 있고, 사용자 지정 트레이스 플롯 빌더를 통해 직접 생성할 수도 있습니다.

![Trace plots 예시](/weave/guides/tracking/imgs/plots-example.png)

<div id="get-started">
  ## 시작하기
</div>

1. 프로젝트의 **Traces** 페이지로 이동합니다.
2. (선택 사항) **Filter**를 선택하여 trace를 필터링합니다(예: 날짜/시간 또는 operation 기준).
3. **Traces** 뷰의 오른쪽 상단에서 **Show Metrics** 아이콘을 클릭하여 trace 플롯 사이드 패널을 엽니다. 

    ![The  **Show Metrics** icon](/weave/guides/tracking/imgs/plots-show-metrics-icon.png)

    여기에서 다음을 할 수 있습니다:
   - [기본 trace 플롯](#default-trace-plots)을 확인합니다.
   - [사용자 지정 trace 플롯](#create-a-custom-trace-plot)을 생성합니다.
4. 차트는 trace 필터와 선택에 따라 동적으로 업데이트됩니다.

<div id="default-trace-plots">
  ## 기본 트레이스 플롯
</div>

처음 트레이스 플롯 패널을 열면, Weave가 사용 가능한 프로젝트 트레이스 데이터를 기반으로 몇 가지 트레이스 플롯을 자동 생성합니다:

- 막대 차트(시간 구간별로 그룹화된 비용 또는 지연 시간)
- 선 차트(시간에 따른 지연 시간)
- 산점도(예: prompt tokens vs. completion tokens)

각 트레이스 플롯은 상호작용할 수 있습니다:

- 마우스를 올려 툴팁 보기
- 드래그해서 확대/축소
- 더블 클릭해서 초기 상태로 되돌리기
- 산점도에서 점을 클릭해 특정 트레이스를 열기

<div id="create-a-custom-trace-plot">
  ## 사용자 지정 트레이스 플롯 만들기
</div>

사용자 지정 트레이스 플롯을 만들 수도 있습니다. 사용자 지정 트레이스 플롯을 만들려면 다음 단계를 따르세요.

1. 트레이스 플롯 사이드 패널에서 **➕ Add Chart** 를 클릭합니다.

    ![**➕ Add Chart** button.](/weave/guides/tracking/imgs/plots-add-chart.png)

2. 팝업에서 사용 가능한 트레이스 플롯 유형 중 하나를 선택합니다.

    ![Available custom trace plot types](/weave/guides/tracking/imgs/plots-custom-chart-types.png)
    
3. 선택한 트레이스 플롯 유형에 맞게 트레이스 플롯을 설정합니다. 트레이스 플롯 유형별 설정 옵션에 대한 자세한 내용은 [Trace plot settings by plot type](#trace-plot-settings-by-plot-type)을 참조하세요.
4. 차트를 저장하려면 **Save chart** 를 클릭합니다.

<div id="trace-plot-settings-by-plot-type">
  ### 플롯 유형별 트레이스 플롯 설정
</div>

커스텀 트레이스 플롯을 추가하거나 편집할 때, 사용 가능한 설정 옵션은 선택한 트레이스 플롯 유형에 따라 조금씩 달라집니다. 아래 표는 유형별로 설정 가능한 옵션을 정리한 것입니다.

| Setting         | 산점도 (Scatter Plot) | 선 차트 (Line Chart) | 막대 차트 (Bar Chart) |
| --------------- | --------------------- | --------------------- | ---------------------- |
| Y-axis      | ✅ 필수               | ✅ 필수               | ✅ 필수               |
| X-axis      | ✅ 선택 가능           | 고정됨 (`started at`) | 고정됨 (`started at`) |
| Grouping    | ✅ 선택 사항           | ✅ 선택 사항           | ✅ 선택 사항           |
| Binning     | ❌ 사용 안 함          | ✅ 사용함              | ✅ 사용함              |
| Aggregation | ❌ 사용 안 함          | ✅ 사용함              | ✅ 사용함              |
