title: "트레이스 플롯 사용하기"
description: "Weave에서 대화형 차트로 트레이스 수준 메트릭을 시각화하여 LLM 애플리케이션의 지연 시간, 비용, 토큰 사용 패턴을 탐색합니다."
---

W&amp;B Weave의 *트레이스 플롯* 도구를 사용하면 대화형 차트로 트레이스 수준 메트릭(예: 지연 시간, 비용, 토큰)을 탐색하고, 시각화하고, 디버깅할 수 있습니다. 기본 트레이스 플롯을 사용하거나 사용자 지정 트레이스 플롯 빌더로 직접 만들 수 있습니다.

<Frame>
  ![트레이스 플롯 예시](/weave/guides/tracking/imgs/plots-example.png)
</Frame>

<div id="get-started">
  ## 시작하기
</div>

1. 프로젝트의 **Traces** 페이지로 이동합니다.
2. (선택 사항) 선택한 트레이스를 필터링하려면(예: 날짜/시간 또는 오퍼레이션 기준) **Filter**를 선택합니다.
3. **Traces** 뷰의 오른쪽 상단에서 **Show Metrics** 아이콘을 클릭해 트레이스 플롯 사이드 패널을 엽니다.

   ![Show Metrics 아이콘](/weave/guides/tracking/imgs/plots-show-metrics-icon.png)

   여기에서 다음을 할 수 있습니다:

   * [기본 트레이스 플롯](#default-trace-plots)을 확인합니다.
   * [사용자 지정 트레이스 플롯](#create-a-custom-trace-plot)을 만듭니다.
4. 차트는 트레이스 필터와 선택에 따라 동적으로 업데이트됩니다.

<div id="default-trace-plots">
  ## 기본 트레이스 플롯
</div>

트레이스 플롯 패널을 처음 열면 Weave가 프로젝트에서 사용 가능한 트레이스 데이터를 기반으로 몇 가지 트레이스 플롯을 자동으로 생성합니다:

* 막대 차트(시간 구간별 비용 또는 지연 시간)
* 라인 차트(시간에 따른 지연 시간)
* 산점도(예: 프롬프트 토큰과 생성 토큰 비교)

각 트레이스 플롯은 대화형입니다:

* 마우스를 올려 툴팁 표시
* 드래그하여 확대
* 더블클릭하여 재설정
* 산점도의 포인트를 클릭하여 특정 트레이스 열기

<div id="create-a-custom-trace-plot">
  ## 사용자 지정 트레이스 플롯 만들기
</div>

사용자 지정 트레이스 플롯을 만들 수도 있습니다. 사용자 지정 트레이스 플롯을 만들려면 다음 단계를 따르세요.

1. 트레이스 플롯 사이드 패널에서 **➕ Add Chart**를 클릭합니다.

   ![➕ Add Chart 버튼.](/weave/guides/tracking/imgs/plots-add-chart.png)

2. 팝업에서 사용 가능한 트레이스 플롯 유형 중 하나를 선택합니다.

   ![사용 가능한 사용자 지정 트레이스 플롯 유형](/weave/guides/tracking/imgs/plots-custom-chart-types.png)

3. 선택한 트레이스 플롯 유형에 맞게 트레이스 플롯을 설정합니다. 트레이스 플롯 유형별 설정 옵션에 대한 자세한 내용은 [플롯 유형별 트레이스 플롯 설정](#trace-plot-settings-by-plot-type)을 참조하세요.

4. 차트를 저장하려면 **Save chart**를 클릭합니다.

<div id="trace-plot-settings-by-plot-type">
  ### 플롯 유형별 트레이스 플롯 설정
</div>

사용자 지정 트레이스 플롯을 추가하거나 편집할 때 사용 가능한 설정 옵션은 선택한 트레이스 플롯 유형에 따라 조금씩 다릅니다. 아래 표는 구성할 수 있는 옵션을 정리한 것입니다.

| 설정  | 산점도      | 라인 차트             | 막대 차트             |
| --- | -------- | ----------------- | ----------------- |
| Y축  | ✅ 필수     | ✅ 필수              | ✅ 필수              |
| X축  | ✅ 선택 가능  | 고정 (`started at`) | 고정 (`started at`) |
| 그룹화 | ✅ 선택 사항  | ✅ 선택 사항           | ✅ 선택 사항           |
| 비닝  | ❌ 사용 안 함 | ✅ 사용              | ✅ 사용              |
| 집계  | ❌ 사용 안 함 | ✅ 사용              | ✅ 사용              |
