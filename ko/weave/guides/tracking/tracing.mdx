<div id="ops">
  ## Calls
</div>

**Call**은 Op가 실행될 때 로그로 기록되는 실행 단위입니다. Op가 실행될 때마다 Weave는 다음 정보를 캡처하는 Call을 생성합니다:

* 입력 인수
* 출력 값
* 타이밍 및 지연 시간
* (중첩 호출의 경우) 부모-자식 관계
* 발생한 오류

Calls는 Weave UI에서 **Traces**로 표시되며, 디버깅, 분석, 평가에 필요한 데이터를 제공합니다. 전체 Call 객체 구조와 속성은 [Call schema reference](/ko/weave/guides/tracking/call-schema-reference)를 참조하세요.

Calls는 [OpenTelemetry](https://opentelemetry.io) 데이터 모델의 span과 유사합니다. Call은 다음이 가능합니다:

* Trace에 속함 (동일한 실행 컨텍스트 내의 calls 집합)
* 부모 및 자식 Calls를 가짐 (트리 구조 형성)
