  ## нҠёл ҲмқҙмҠӨ
</div>

к°ңл°ң мӨ‘мқҙл“  н”„лЎңлҚ•м…ҳ нҷҳкІҪмқҙл“ , LLM м• н”ҢлҰ¬мјҖмқҙм…ҳмқҳ нҠёл ҲмқҙмҠӨлҘј мӨ‘м•ҷ лҚ°мқҙн„°лІ мқҙмҠӨм—җ м ҖмһҘн•ҳлҠ” кІғмқҖ мӨ‘мҡ”н•©лӢҲлӢӨ. мқҙлҹ¬н•ң нҠёл ҲмқҙмҠӨлҠ” л””лІ„к№…м—җ нҷңмҡ©н•  мҲҳ мһҲкі , м• н”ҢлҰ¬мјҖмқҙм…ҳ к°ңм„ м—җ лҸ„мӣҖмқҙ лҗҳлҠ” лҚ°мқҙн„°м…ӢмңјлЎңлҸ„ мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

> **м°ёкі :** LiteLLMмқ„ мӮ¬мҡ©н•  л•ҢлҠ” `from litellm import completion` лҢҖмӢ  `import litellm`мңјлЎң лқјмқҙлёҢлҹ¬лҰ¬лҘј мһ„нҸ¬нҠён•ҳкі , completion н•ЁмҲҳлҠ” `litellm.completion`мңјлЎң нҳём¶ңн•ҙм•ј н•©лӢҲлӢӨ. мқҙл ҮкІҢ н•ҙм•ј лӘЁл“  н•ЁмҲҳмҷҖ нҢҢлқјлҜён„°к°Җ мҳ¬л°”лҘҙкІҢ м°ёмЎ°лҗ©лӢҲлӢӨ.

WeaveлҠ” LiteLLMмқҳ нҠёл ҲмқҙмҠӨлҘј мһҗлҸҷмңјлЎң мәЎмІҳн•©лӢҲлӢӨ. лқјмқҙлёҢлҹ¬лҰ¬лҠ” нҸүмҶҢмІҳлҹј мӮ¬мҡ©н•ҳл©ҙ лҗҳл©°, лЁјм Җ `weave.init()`лҘј нҳём¶ңн•ҙ мӢңмһ‘н•ҳм„ёмҡ”:

```python lines {4}
import litellm
import weave

weave.init("weave_litellm_integration")

openai_response = litellm.completion(
    model="gpt-3.5-turbo", 
    messages=[{"role": "user", "content": "Translate 'Hello, how are you?' to French"}],
    max_tokens=1024
)
print(openai_response.choices[0].message.content)

claude_response = litellm.completion(
    model="claude-3-5-sonnet-20240620", 
    messages=[{"role": "user", "content": "Translate 'Hello, how are you?' to French"}],
    max_tokens=1024
)
print(claude_response.choices[0].message.content)
```

мқҙм ң WeaveлҠ” LiteLLMмқ„ нҶөн•ҙ л°ңмғқн•ҳлҠ” лӘЁл“  LLM нҳём¶ңмқ„ м¶”м Ғн•ҳкі  лЎңк№…н•©лӢҲлӢӨ. Weave мӣ№ мқён„°нҺҳмқҙмҠӨм—җм„ң нҠёл ҲмқҙмҠӨлҘј нҷ•мқён•  мҲҳ мһҲмҠөлӢҲлӢӨ.


<div id="wrapping-with-your-own-ops">
  ## мһҗмІҙ opsлЎң к°җмӢёкё°
</div>

Weave opsлҠ” мӢӨн—ҳн•ҳлҠ” лҸҷм•Ҳ мҪ”л“ң лІ„м „мқ„ мһҗлҸҷмңјлЎң кҙҖлҰ¬н•ҳкі  мһ…л Ҙкіј м¶ңл Ҙмқ„ мәЎмІҳн•ҙ кІ°кіјлҘј мһ¬нҳ„ к°ҖлҠҘн•ҳкІҢ л§Ңл“ӯлӢҲлӢӨ. `@weave.op()`мңјлЎң лҚ°мҪ”л ҲмқҙнҢ…н•ң н•ЁмҲҳлҘј л§Ңл“Өкі  к·ё м•Ҳм—җм„ң LiteLLMмқҳ completion н•ЁмҲҳлҘј нҳём¶ңн•ҳкё°л§Ң н•ҳл©ҙ, Weaveк°Җ мһ…л Ҙкіј м¶ңл Ҙмқ„ м¶”м Ғн•ҙ мӨҚлӢҲлӢӨ. лӢӨмқҢмқҖ мҳҲмӢңмһ…лӢҲлӢӨ:

```python lines {4,6}
import litellm
import weave

weave.init("weave_litellm_integration")

@weave.op()
def translate(text: str, target_language: str, model: str) -> str:
    response = litellm.completion(
        model=model,
        messages=[{"role": "user", "content": f"Translate '{text}' to {target_language}"}],
        max_tokens=1024
    )
    return response.choices[0].message.content

print(translate("Hello, how are you?", "French", "gpt-3.5-turbo"))
print(translate("Hello, how are you?", "Spanish", "claude-3-5-sonnet-20240620"))
```


<div id="create-a-model-for-easier-experimentation">
  ## лҚ” мүҪкІҢ мӢӨн—ҳн•  мҲҳ мһҲлҸ„лЎқ `Model` л§Ңл“Өкё°
</div>

кө¬м„ұ мҡ”мҶҢк°Җ л§Һм•„м§Җл©ҙ мӢӨн—ҳмқ„ мІҙкі„м ҒмңјлЎң м •лҰ¬н•ҳкё°к°Җ м–ҙл өмҠөлӢҲлӢӨ. `Model` нҒҙлһҳмҠӨлҘј мӮ¬мҡ©н•ҳл©ҙ мӢңмҠӨн…ң н”„лЎ¬н”„нҠёлӮҳ мӮ¬мҡ© мӨ‘мқё лӘЁлҚёмІҳлҹј м•ұмқҳ мӢӨн—ҳ кҙҖл Ё м„ёл¶Җ м •ліҙлҘј мҲҳм§‘н•ҳкі  м •лҰ¬н•  мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙл ҮкІҢ н•ҳл©ҙ м•ұмқҳ м—¬лҹ¬ л°ҳліө лІ„м „мқ„ м •лҰ¬н•ҳкі  л№„көҗн•ҳлҠ” лҚ° лҸ„мӣҖмқҙ лҗ©лӢҲлӢӨ.

ModelsлҠ” мҪ”л“ң лІ„м „ кҙҖлҰ¬мҷҖ мһ…л Ҙ/м¶ңл Ҙ мәЎмІҳм—җ лҚ”н•ҙ, м• н”ҢлҰ¬мјҖмқҙм…ҳ лҸҷмһ‘мқ„ м ңм–ҙн•ҳлҠ” кө¬мЎ°нҷ”лҗң л§Өк°ңліҖмҲҳлҸ„ н•Ёк»ҳ мҲҳм§‘н•ҳлҜҖлЎң м–ҙл–Ө л§Өк°ңліҖмҲҳк°Җ к°ҖмһҘ нҡЁкіјм Ғмқҙм—ҲлҠ”м§Җ мүҪкІҢ м°ҫмқ„ мҲҳ мһҲмҠөлӢҲлӢӨ. Weave ModelsлҠ” `serve` л°Ҹ EvaluationsмҷҖ н•Ёк»ҳ мӮ¬мҡ©н•  мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ.

м•„лһҳ мҳҲм ңм—җм„ңлҠ” лӢӨм–‘н•ң лӘЁлҚёкіј temperatureлЎң мӢӨн—ҳн•  мҲҳ мһҲмҠөлӢҲлӢӨ:

```python lines {4,6,10}
import litellm
import weave

weave.init('weave_litellm_integration')

class TranslatorModel(weave.Model):
    model: str
    temperature: float
  
    @weave.op()
    def predict(self, text: str, target_language: str):
        response = litellm.completion(
            model=self.model,
            messages=[
                {"role": "system", "content": f"You are a translator. Translate the given text to {target_language}."},
                {"role": "user", "content": text}
            ],
            max_tokens=1024,
            temperature=self.temperature
        )
        return response.choices[0].message.content

# лӢӨм–‘н•ң лӘЁлҚёлЎң мқёмҠӨн„ҙмҠӨ мғқм„ұ
gpt_translator = TranslatorModel(model="gpt-3.5-turbo", temperature=0.3)
claude_translator = TranslatorModel(model="claude-3-5-sonnet-20240620", temperature=0.1)

# лІҲм—ӯм—җ лӢӨм–‘н•ң лӘЁлҚё мӮ¬мҡ©
english_text = "Hello, how are you today?"

print("GPT-3.5 н”„лһ‘мҠӨм–ҙ лІҲм—ӯ:")
print(gpt_translator.predict(english_text, "French"))

print("\nClaude-3.5 Sonnet мҠӨнҺҳмқём–ҙ лІҲм—ӯ:")
print(claude_translator.predict(english_text, "Spanish"))
```

  ## н•ЁмҲҳ нҳём¶ң

LiteLLMмқҖ нҳёнҷҳлҗҳлҠ” лӘЁлҚём—җм„ң н•ЁмҲҳ нҳём¶ңмқ„ м§Җмӣҗн•©лӢҲлӢӨ. WeaveлҠ” мқҙлҹ¬н•ң н•ЁмҲҳ нҳём¶ңмқ„ мһҗлҸҷмңјлЎң м¶”м Ғн•©лӢҲлӢӨ.

```python lines {4}
import litellm
import weave

weave.init("weave_litellm_integration")

response = litellm.completion(
    model="gpt-3.5-turbo",
    messages=[{"role": "user", "content": "Translate 'Hello, how are you?' to French"}],
    functions=[
        {
            "name": "translate",
            "description": "Translate text to a specified language",
            "parameters": {
                "type": "object",
                "properties": {
                    "text": {
                        "type": "string",
                        "description": "The text to translate",
                    },
                    "target_language": {
                        "type": "string",
                        "description": "The language to translate to",
                    }
                },
                "required": ["text", "target_language"],
            },
        },
    ],
)

print(response)
```

н”„лЎ¬н”„нҠём—җм„ң мӮ¬мҡ©н•ң н•ЁмҲҳлҘј мһҗлҸҷмңјлЎң мәЎмІҳн•ҳкі  лІ„м „ кҙҖлҰ¬н•©лӢҲлӢӨ.
