Weights & Biases (W&B) Weave 는 Microsoft Azure OpenAI 서비스와 통합되어 팀이 Azure AI 애플리케이션을 최적화할 수 있도록 지원합니다. W&B를 사용하여 다음을 수행할 수 있습니다.

<Tip>
최신 튜토리얼을 보려면 [Weights & Biases on Microsoft Azure](https://wandb.ai/site/partners/azure)를 방문하세요.
</Tip>

## 시작하기

Weave 와 함께 Azure를 사용하려면, 추적하려는 함수에 `weave.op` 데코레이터를 추가하기만 하면 됩니다.

```python lines
@weave.op()
def call_azure_chat(model_id: str, messages: list, max_tokens: int = 1000, temperature: float = 0.5):
    # Azure OpenAI 클라이언트를 사용하여 채팅 완성 생성
    response = client.chat.completions.create(
        model=model_id,
        messages=messages,
        max_tokens=max_tokens,
        temperature=temperature
    )
    return {"status": "success", "response": response.choices[0].message.content}
```
