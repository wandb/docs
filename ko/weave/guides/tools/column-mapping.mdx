---
title: "데이터셋에서 컬럼 매핑하기"
description: "데이터셋의 컬럼 이름을 다른 이름으로 매핑합니다. 이를 통해 데이터셋의 컬럼 이름을 scorer가 기대하는 컬럼 이름과 일치시킬 수 있습니다."
---

<Tabs>
  <Tab title="Python">
    Weave Python SDK에서는 scorer의 `column_map` 속성을 사용하여 scorer가 기대하는 파라미터 이름을 데이터셋의 컬럼 이름에 매핑합니다. 매핑 형식은 `{scorer_parameter: dataset_column}`입니다.

    다음 예시는 `output`과 `target` 파라미터를 데이터셋의 `model_output` 및 `answer` 컬럼에 매핑합니다:

    ```python
    from weave.scorers import EmbeddingSimilarityScorer

    similarity_scorer = EmbeddingSimilarityScorer()

    similarity_scorer.column_map = {
        "output": "model_output",  # 모델이 생성한 텍스트
        "target": "answer"         # 기대되거나 기준이 되는 응답
    }
    ```

    scorer 컬럼 매핑에 대한 자세한 내용은 [Mapping Column Names with `column_map`](../evaluation/scorers#mapping-column-names-with-column_map)를 참고하세요.
  </Tab>
  <Tab title="TypeScript">
    Weave TypeScript SDK에서는 컬럼 매핑을 개별 scorer가 아니라 `Evaluation` 객체의 `columnMapping` 옵션에서 설정합니다. 매핑 형식은 `{scorer_key: dataset_column}`입니다.

    다음 예시는 scorer에서 사용하는 `expectedOutputTimesTwo`를 데이터셋의 `expected` 컬럼에 매핑합니다:

    ```typescript
    import * as weave from 'weave';

    const myScorer = weave.op(
      ({modelOutput, datasetRow}) => {
        return modelOutput * 2 === datasetRow.expectedOutputTimesTwo;
      },
      {name: 'myScorer'}
    );

    const myEval = new weave.Evaluation({
      id: 'my-evaluation',
      dataset: [{expected: 2}],
      scorers: [myScorer],
      columnMapping: {expectedOutputTimesTwo: 'expected'},
    });
    ```

    TypeScript scorer 인자에 대한 자세한 내용은 [Scorer Keyword Arguments](../evaluation/scorers#scorer-keyword-arguments)를 참고하세요.
  </Tab>
</Tabs>
