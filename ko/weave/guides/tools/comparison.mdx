---
title: "트레이스와 기타 로깅된 정보 비교"
description: "코드, 트레이스, 프롬프트, 모델, 설정을 시각적으로 비교하고 차이를 확인합니다"
---

W&amp;B Weave Comparison 기능을 사용하면 코드, 트레이스, 프롬프트, 모델, 모델 설정을 시각적으로 비교하고 차이를 확인할 수 있습니다. 두 객체를 나란히 비교하거나 더 큰 객체 집합을 분석하여 차이, 패턴, 추세를 파악할 수 있습니다.

이 가이드에서는 비교를 시작하는 step과 기준선 비교, 숫자 diff 형식 지정 등 비교 뷰를 맞춤 설정하는 데 사용할 수 있는 액션을 설명합니다.

<div id="access-the-comparison-view">
  ## 뷰에 액세스하기
</div>

1. 사이드바에서 비교할 객체의 유형(예: **트레이스** 또는 **Models**)을 선택하세요.
2. 비교할 객체를 선택하세요. 선택 방법은 비교하는 객체의 유형에 따라 달라집니다.
   * **트레이스**의 경우, Trace 열에서 해당 행의 체크박스를 선택해 비교할 트레이스를 고르세요.
   * **Models**와 같은 객체의 경우, model Versions 페이지로 이동한 다음 비교할 버전 옆의 체크박스를 선택하세요.
3. **Compare**를 선택하여 뷰를 여세요. 페이지 상단의 비교 막대에는 대화형 call ID token(예: `f02b`, `4a98`)이 포함되어 있으며, 이를 드래그하거나 순서를 바꾸거나 제거하여 비교를 변경할 수 있습니다.
4. [사용 가능한 액션](#available-actions)을 사용해 뷰를 세부적으로 조정할 수 있습니다.

<div id="display-summary">
  ### 요약 표시
</div>

선택한 객체 비교의 기본 뷰는 요약입니다. 이 요약에는 LLM 입력과 출력의 미리보기와 함께 토큰, 비용, 지연 시간이 포함됩니다.

<Frame>
  ![두 객체의 비교 요약 뷰](/weave/guides/tools/imgs/comparison-2objs-summary.png)
</Frame>

트레이스의 경우, 비교 테이블 상단에 표시되는 색상 막대는 Ops에 적용한 맞춤형 kind와 색상을 반영합니다. 이처럼 향상된 공개 범위를 사용하는 방법에 대한 자세한 내용은 [kind와 색상 적용](/ko/weave/guides/tracking/ops#apply-kinds-and-colors)을 참조하세요.

<div id="display-a-calls-view">
  ### Call 뷰 표시
</div>

각 객체의 Call를 비교하려면 **Call**를 선택합니다. 이 뷰에서는 다음을 수행할 수 있습니다:

* Call에 대한 전체 트레이스 트리를 표시합니다.
* 기본 메모리 내 텍스트 검색 기능을 제공합니다.
* Call별 지연 시간, 비용, 토큰, Op 유형, 피드백 표시를 숨기거나 표시할 수 있습니다.

<Frame>
  ![두 객체의 Call 비교 뷰](/weave/guides/tools/imgs/comparison-2obs-calls.png)
</Frame>

<div id="change-the-diff-display">
  ### diff 표시 방식 변경
</div>

기본적으로 **Diff only**는 꺼져 있습니다. 변경된 행만 표시하도록 테이블 행을 필터링하려면 **Diff only**를 켜세요.

**Diff only**는 나란히 보기와 통합 뷰에만 적용되며, 다른 뷰에서는 비활성화됩니다.

<div id="display-side-by-side">
  ### 나란히 표시
</div>

각 객체를 별도의 열에 나란히 표시하여 비교하려면 **Side-by-side**를 선택하세요.

<Frame>
  ![두 객체를 나란히 비교한 뷰](/weave/guides/tools/imgs/comparison-2objs-sidebyside.png)
</Frame>

<div id="display-in-a-unified-view">
  ### 통합 뷰로 표시
</div>

두 개의 객체를 통합 뷰에서 비교하려면 **Unified**를 선택합니다. 이 뷰는 두 개를 초과하는 객체를 비교할 때는 사용할 수 없습니다.

<Frame>
  ![두 객체의 Unified 비교 뷰](/weave/guides/tools/imgs/comparison-2objs-unified.png)

<div id="set-a-baseline">
  ### 기준선 설정하기
</div>

기본적으로 뷰의 각 객체는 왼쪽에 있는 객체와 비교됩니다. 하지만 객체를 &#95;기준선(baseline)&#95;으로 설정할 수 있으며, 이렇게 하면 모든 객체가 뷰의 가장 왼쪽에 있는 객체와 비교됩니다.

객체를 기준선으로 설정하려면:

1. 비교 막대에서 기준선으로 설정하려는 객체 위에 포인터를 올리세요.
2. ID 오른쪽의 줄임표(<Icon icon="ellipsis-vertical" iconType="solid" />)를 클릭하세요.
   ![비교 막대에서 선택한 객체의 Make baseline 옵션이 표시된 뷰.](/weave/guides/tools/imgs/comparison-2objs-baseline.png)
3. 목록에서 **Make baseline**을 선택하세요. UI가 새로고침되며 기준선 객체가 비교 막대의 가장 왼쪽 위치로 이동하고 ID 옆에 `Baseline`이 표시됩니다.
   ![비교 막대의 가장 왼쪽 위치에 선택한 기준선 객체가 표시된 뷰.](/weave/guides/tools/imgs/comparison-2objs-baseline-set.png)

<div id="remove-a-baseline">
  ### 기준선 제거
</div>

기준선으로 설정된 객체를 제거하려면:

1. 비교 막대에서 마우스 포인터를 기준선 객체 위에 올립니다.
2. ID 오른쪽에 있는 줄임표(<Icon icon="ellipsis-vertical" iconType="solid" />)를 클릭합니다.
3. 목록에서 **Remove baseline**을 선택합니다. 이제 `Baseline`이 더 이상 Call ID 옆에 표시되지 않습니다.

<div id="change-the-comparison-order">
  ### 비교 순서 변경하기
</div>

비교 막대에서 객체를 드래그해 순서를 변경합니다. 그러면 비교 테이블의 해당 열 순서도 함께 변경됩니다.

비교 순서를 변경하려면:

1. 비교 막대에서, 순서를 변경하려는 ID 위에 마우스 포인터를 올려 둡니다.
2. ID 왼쪽에 있는 여섯 개의 점을 클릭한 상태로 필요한 만큼 왼쪽 또는 오른쪽으로 드래그합니다.
3. 원하는 순서 위치에 ID를 놓습니다. 비교 데이터가 새 비교 순서에 맞게 업데이트됩니다.

<div id="change-numeric-diff-display-format">
  ### 숫자 diff 표시 형식 변경
</div>

`completion_tokens` 및 `total_tokens`와 같은 숫자 값의 경우 diff를 정수 또는 백분율로 확인할 수 있습니다. 또한 양수인 숫자 값은 배수로도 확인할 수 있습니다.

숫자 diff의 표시 형식을 변경하려면 다음 단계를 따르세요.

1. Comparison 테이블에서 diff 표시 형식을 업데이트하려는 숫자 값을 찾으세요.
   ![정수로 표시된 숫자 값입니다.](/weave/guides/tools/imgs/comparison-2objs-numericdiffformat.png)
2. diff 값을 클릭하세요. 형식이 자동으로 정수 또는 백분율로 업데이트됩니다.
   ![백분율로 업데이트된 숫자 값입니다.](/weave/guides/tools/imgs/comparison-2objs-numericdiffformat-updated.png)

<div id="compare-with-baseline-or-previous">
  ### 기준선 또는 이전 항목과 비교
</div>

이 옵션은 객체를 3개 이상 비교할 때만 사용할 수 있습니다. [기준선을 설정](#set-a-baseline)하거나 [ID 오른쪽의 점 3개를 클릭해 기존 기준선을 제거](#remove-a-baseline)할 수도 있습니다.

객체 3개 이상으로 기준선 비교를 수행하려면 다음과 같이 하세요:

1. Side-by-side 또는 Unified Comparison 뷰의 오른쪽 상단에서 목록을 클릭하세요. 현재 뷰 설정에 따라 목록 제목은 **Compare with previous** 또는 **Compare with baseline**입니다.
2. 현재 뷰 설정에 따라 **Compare with previous** 또는 **Compare with baseline**을 선택하세요.
   * **Compare with baseline**: 가장 왼쪽 객체를 기준선으로 설정합니다. 테이블이 업데이트되어 가장 왼쪽 열이 기준선이 됩니다.
   * **Compare with previous**: 어떤 객체도 기준선으로 설정되지 않습니다.

<div id="compare-a-pair-from-a-multi-object-comparison">
  ### 다중 객체 비교에서 쌍으로 비교하기
</div>

이 옵션은 3개 이상의 객체를 비교할 때만 사용 가능합니다.

3개 이상의 객체를 비교할 때는 단일 객체를 인접한 객체 또는 기준선과 비교할 수 있습니다. 그러면 Comparison 테이블 뷰가 변경되어, 두 객체 비교와 동일한 형태로 표시됩니다.

다중 객체 비교에서 두 객체를 쌍으로 비교하려면 다음을 수행합니다.

1. 비교 막대에서 비교하려는 ID를 선택합니다. 기준선을 설정한 경우, 선택한 객체와 기준선을 비교합니다. 기준선을 설정하지 않은 경우, 선택한 객체와 비교 막대에서 바로 왼쪽에 있는 ID를 비교합니다.
2. 항목을 선택하려면 ID를 클릭합니다. 그러면 UI가 새로 고쳐지며 두 객체 비교 테이블이 표시됩니다.
   ![다중 객체 비교에서 두 객체를 비교하는 예.](/weave/guides/tools/imgs/comparison-7objs-diffonly-subset.png)

비교를 위해 처음 선택된 6개 객체가 테이블에 다시 표시되도록 뷰를 재설정하려면 ID를 다시 클릭합니다.

<div id="remove-an-object-from-comparison">
  ### 비교에서 객체 제거
</div>

이 옵션은 3개 이상의 객체를 비교할 때만 사용 가능합니다.

비교에서 객체를 제거하려면:

1. 비교 막대에서 비교에서 제거하려는 객체를 찾습니다.
2. ID 오른쪽에 있는 줄임표(<Icon icon="ellipsis-vertical" iconType="solid" />) 아이콘을 클릭합니다.
3. 목록에서 **Remove object from comparison**을 선택합니다. UI가 새로 고침되며, 제거된 객체가 더 이상 포함되지 않은 업데이트된 테이블이 표시됩니다.

<div id="usage-notes">
  ## 사용 참고 사항
</div>

* Comparison 기능은 UI에서만 사용 가능합니다.
* 원하는 만큼 많은 객체를 비교할 수 있습니다. 하지만 UI에는 최대 6개만 표시됩니다. 6개가 넘는 객체를 비교할 때 보이지 않는 객체를 comparison table에서 보려면, 해당 객체가 왼쪽에서 오른쪽 기준 처음 6개 객체 중 하나가 되도록 [비교 순서를 변경](#change-the-comparison-order)하거나, 쉽게 볼 수 있도록 [다중 객체 비교에서 한 쌍 비교](#compare-a-pair-from-a-multi-object-comparison)를 하세요.
