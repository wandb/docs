---
title: "트레이스와 기타 로깅된 정보 비교"
description: "코드, 트레이스, 프롬프트, 모델, 구성 정보를 시각적으로 비교하고 차이 확인"
---

W&amp;B Weave의 Comparison 기능을 사용하면 코드, 트레이스, 프롬프트, 모델, 그리고 모델 구성 정보를 시각적으로 비교하고 차이를 확인할 수 있습니다. 두 개의 객체를 나란히 비교하거나 더 큰 객체 집합을 분석하여 차이점, 패턴, 추세를 파악할 수 있습니다.

이 가이드는 비교를 시작하는 단계와 기준선 비교, 수치 diff 서식 지정 등 Comparison 뷰를 원하는 대로 조정할 수 있는 기능을 다룹니다.

<div id="access-the-comparison-view">
  ## Comparison 뷰 열기
</div>

1. 사이드바에서 비교할 객체 유형을 선택합니다(예: **Traces** 또는 **Models**).
2. 비교하려는 객체를 선택합니다. 선택 방법은 비교하는 객체 유형에 따라 다릅니다.
   * **Traces**의 경우 Trace 열에서 해당 행의 체크박스를 선택해 비교할 trace를 선택합니다.
   * **Models**와 같은 객체의 경우 모델 Versions 페이지로 이동한 뒤, 비교하려는 버전 옆의 체크박스를 선택합니다.
3. **Compare**를 선택해 Comparison 뷰를 엽니다. 페이지 상단의 비교 바에는 인터랙티브 call ID 토큰(예: `f02b`, `4a98`)이 있으며, 이를 끌어서 순서를 바꾸거나 제거해 비교 대상을 변경할 수 있습니다.
4. [사용 가능한 작업](#available-actions)을 사용해 뷰를 더 세밀하게 조정할 수 있습니다.

<div id="display-summary">
  ### 요약 표시
</div>

선택한 객체 비교의 기본 보기는 요약입니다. 이 요약에는 LLM 입력과 출력의 미리보기뿐 아니라 토큰, 비용, 지연 시간도 포함됩니다.

<Frame>
  ![두 객체의 비교 요약 보기](/weave/guides/tools/imgs/comparison-2objs-summary.png)
</Frame>

Trace의 경우, 비교 테이블 상단에 표시되는 색 막대는 Ops에 적용한 모든 사용자 정의 kind와 색상을 반영합니다. 이 확장된 가시성 기능을 활용하는 방법에 대한 자세한 내용은 [Apply kinds and colors](/ko/weave/guides/tracking/ops#apply-kinds-and-colors)를 참고하세요.

<div id="display-a-calls-view">
  ### Calls 뷰 표시하기
</div>

각 오브젝트의 Calls를 비교하려면 **Calls**를 선택합니다. 이 뷰에서는 다음 작업을 할 수 있습니다:

* 해당 Call의 전체 트레이스 트리를 표시합니다.
* 기본적인 인메모리 텍스트 검색 기능을 제공합니다.
* 각 Call의 지연 시간, 비용, 토큰 수, Op 종류, 피드백 표시를 켜거나 끌 수 있습니다.

<Frame>
  ![두 오브젝트의 Calls 비교 뷰](/weave/guides/tools/imgs/comparison-2obs-calls.png)
</Frame>

<div id="change-the-diff-display">
  ### Diff 표시 변경
</div>

기본적으로 **Diff only** 는 꺼져 있습니다. 표에서 변경된 행만 표시되도록 필터링하려면 **Diff only** 를 켭니다.

**Diff only** 는 나란히 보기와 통합 보기(unified view)에만 적용되며, 다른 보기에서는 사용할 수 없습니다.

<div id="display-side-by-side">
  ### 나란히 보기
</div>

각 객체를 별도의 열에 나란히 비교하려면 **Side-by-side**를 선택하세요.

<Frame>
  ![두 객체를 나란히 비교하는 뷰](/weave/guides/tools/imgs/comparison-2objs-sidebyside.png)
</Frame>

<div id="display-in-a-unified-view">
  ### 통합 보기로 표시
</div>

두 객체를 통합 보기에서 비교하려면 **Unified**를 선택합니다. 세 개 이상 객체를 비교할 때는 이 보기를 사용할 수 없습니다.

<div id="set-a-baseline">
  ### 기준선 설정
</div>

기본적으로 Comparison 뷰의 각 객체는 왼쪽에 있는 객체와 비교됩니다. 하지만 특정 객체를 &#95;기준선&#95;으로 지정하면, 뷰에서 모든 객체가 가장 왼쪽 객체와 비교됩니다.

객체를 기준선으로 설정하려면:

1. 비교 막대에서 기준선으로 설정하려는 객체 위로 포인터를 가져갑니다.
2. ID 오른쪽에 있는 줄임표(<Icon icon="ellipsis-vertical" iconType="solid" />)를 클릭합니다.
   ![비교 막대에서 선택한 객체에 대해 Make baseline 옵션이 표시된 Comparison 뷰.](/weave/guides/tools/imgs/comparison-2objs-baseline.png)
3. 목록에서 **Make baseline**을 선택합니다. 그러면 UI가 새로고침되면서 기준선 객체가 비교 막대의 가장 왼쪽 위치로 이동하고 ID 옆에 `Baseline`이 표시됩니다.
   ![비교 막대의 가장 왼쪽 위치에 선택된 기준선 객체가 표시된 Comparison 뷰.](/weave/guides/tools/imgs/comparison-2objs-baseline-set.png)

<div id="remove-a-baseline">
  ### 기준선 제거
</div>

객체를 기준선에서 제거하려면:

1. 비교 막대에서 마우스 포인터를 기준선 객체 위에 올립니다.
2. ID 오른쪽에 있는 줄임표 아이콘(<Icon icon="ellipsis-vertical" iconType="solid" />)을 클릭합니다.
3. 목록에서 **Remove baseline**을 선택합니다. 이제 호출 ID 옆에 `Baseline`이 더 이상 표시되지 않습니다.

<div id="change-the-comparison-order">
  ### 비교 순서 변경
</div>

비교 막대에서 객체를 끌어 순서를 변경합니다. 이렇게 하면 비교 표의 해당 열 순서도 함께 변경됩니다.

비교 순서를 변경하려면:

1. 비교 막대에서 순서를 변경하려는 ID 위에 포인터를 올려 둡니다.
2. ID 왼쪽에 있는 여섯 개의 점을 클릭한 채로 필요한 만큼 왼쪽 또는 오른쪽으로 끌어 이동합니다.
3. ID를 원하는 위치에 놓습니다. 그러면 비교 데이터가 새 비교 순서에 맞게 자동으로 새로고침됩니다.

<div id="change-numeric-diff-display-format">
  ### 숫자 차이 표시 형식 변경
</div>

`completion_tokens` 및 `total_tokens`와 같은 숫자 값의 차이는 정수 또는 백분율로 볼 수 있습니다. 추가로, 양수 값은 배수(배율)로도 볼 수 있습니다.

숫자 차이의 표시 형식을 변경하려면:

1. Comparison 테이블에서 표시 형식을 변경하려는 숫자 값을 찾습니다.
   ![정수로 표시된 숫자 값.](/weave/guides/tools/imgs/comparison-2objs-numericdiffformat.png)
2. diff 값(차이 값)을 클릭합니다. 표시 형식이 자동으로 정수 또는 백분율로 전환됩니다.
   ![백분율로 업데이트된 숫자 값.](/weave/guides/tools/imgs/comparison-2objs-numericdiffformat-updated.png)

<div id="compare-with-baseline-or-previous">
  ### 베이스라인 또는 이전 항목과 비교
</div>

이 옵션은 3개 이상의 객체를 비교할 때만 사용할 수 있습니다. 또한 [베이스라인을 설정](#set-a-baseline)하거나 [ID 오른쪽의 점 세 개 아이콘을 클릭해 기존 베이스라인을 제거](#remove-a-baseline)할 수 있습니다.

3개 이상의 객체로 베이스라인 비교를 수행하려면:

1. Side-by-side 또는 Unified Comparison 뷰의 오른쪽 상단에서 목록을 클릭합니다. 현재 뷰 구성에 따라 목록 제목은 **Compare with previous** 또는 **Compare with baseline**입니다.
2. 현재 뷰 구성에 따라 **Compare with previous** 또는 **Compare with baseline**을 선택합니다.
   * **Compare with baseline**: 가장 왼쪽 객체를 베이스라인으로 설정합니다. 테이블이 업데이트되어 가장 왼쪽 열이 베이스라인이 됩니다.
   * **Compare with previous**: 어떤 객체도 베이스라인으로 설정하지 않습니다.

<div id="compare-a-pair-from-a-multi-object-comparison">
  ### 다중 객체 비교에서 객체 쌍 비교하기
</div>

이 옵션은 3개 이상의 객체를 비교할 때만 사용할 수 있습니다.

3개 이상의 객체를 비교할 때는 단일 객체를 인접한 객체 또는 기준선과 비교할 수 있습니다. 그러면 비교 테이블 뷰(Comparison table view)가 두 객체 비교와 동일한 형태로 변경됩니다.

다중 객체 비교에서 두 객체만 선택해 비교하려면:

1. 비교 바에서 비교하려는 ID를 선택합니다. 기준선을 설정한 경우, 선택한 객체를 기준선과 비교합니다. 기준선을 설정하지 않은 경우, 비교 바에서 선택한 객체의 왼쪽에 바로 있는 ID와 비교합니다.
2. 항목을 선택하려면 ID를 클릭합니다. 그러면 UI가 새로 고쳐지면서 두 객체 비교 테이블이 표시됩니다.
   ![다중 객체 비교에서 한 쌍을 비교하는 모습.](/weave/guides/tools/imgs/comparison-7objs-diffonly-subset.png)

비교를 위해 처음 선택한 6개의 객체가 표에 다시 표시되도록 보기를 초기화하려면 ID를 다시 클릭합니다.

<div id="remove-an-object-from-comparison">
  ### 비교에서 객체 제거
</div>

이 옵션은 3개 이상의 객체를 비교할 때만 사용할 수 있습니다.

비교에서 객체를 제거하려면:

1. 비교 표시줄에서 비교에서 제거하려는 객체를 찾습니다.
2. ID 오른쪽에 있는 말줄임표(<Icon icon="ellipsis-vertical" iconType="solid" />) 아이콘을 클릭합니다.
3. 목록에서 **비교에서 객체 제거**를 선택합니다. UI가 새로 고쳐지며 제거된 객체가 더 이상 포함되지 않은 업데이트된 테이블이 표시됩니다.

<div id="usage-notes">
  ## 사용 시 참고사항
</div>

* 비교 기능은 UI에서만 사용할 수 있습니다.
* 원하는 만큼 많은 객체를 비교할 수 있습니다. 다만 UI에는 최대 6개까지만 표시됩니다. 6개를 초과하여 객체를 비교할 때 비교 테이블에 보이지 않는 객체를 확인하려면, 해당 객체가 왼쪽에서 오른쪽으로 첫 6개 객체 안에 들어가도록 [비교 순서를 변경](#change-the-comparison-order)하거나, 더 쉽게 보기 위해 [다중 객체 비교에서 쌍 비교](#compare-a-pair-from-a-multi-object-comparison)를 수행하세요.
