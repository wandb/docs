<div id="deploy-to-google-cloud">
  ## Google Cloud에 배포
</div>

<Note>
  `weave deploy`를 사용하려면 머신에 `gcloud`가 설치되어 있고 설정이 완료되어 있어야 합니다. `weave deploy gcp`는 명령줄 인수로 직접 지정하지 않은 경우 사전 설정된 설정을 사용합니다.
</Note>

Weave 모델을 Google Cloud에 배포하려면 다음을 실행하세요:

```
weave deploy gcp <ref>
```

배포가 완료되면 터미널에 다음 스키마를 사용하는 URL이 반환됩니다:

```
Service URL: <PATH_TO_MODEL>`. `<PATH_TO_MODEL>/docs`를 방문하여 모델과 상호작용하세요!
```
