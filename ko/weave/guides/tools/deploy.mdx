## Google Cloud에 배포하기

<Note>
`weave deploy`를 사용하려면 머신에 `gcloud`가 설치되어 있고 설정되어 있어야 합니다. `weave deploy gcp`는 커맨드라인 인수로 직접 지정하지 않은 경우 사전 설정된 설정을 사용합니다.
</Note>

Weave 모델을 Google Cloud에 배포하려면 다음 명령을 실행하세요:

```
weave deploy gcp <ref>
```

배포가 완료되면, 터미널은 다음 형식의 URL을 반환합니다:

``` 
Service URL: <PATH_TO_MODEL>`. Visit `<PATH_TO_MODEL>/docs` to interact with your model!
```
