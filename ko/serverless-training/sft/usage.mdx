[OpenPipe의 ART 프레임워크](https://art.openpipe.ai/getting-started/about)와 Serverless Training API를 통해 Serverless 지도 파인튜닝(SFT)을 사용하세요.

Serverless SFT를 시작하려면 W&amp;B 도구의 [사전 요구 사항](/ko/serverless-training/prerequisites)을 충족한 후 [ART Serverless SFT 문서](https://art.openpipe.ai/fundamentals/sft-training)를 참고하세요.

Serverless SFT의 API endpoint에 대해 알아보려면 [Serverless Training API 레퍼런스](/ko/serverless-training/api-reference)를 참고하세요.
