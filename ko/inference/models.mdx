<div id="using-model-ids">
  ## 모델 ID 사용하기
</div>

API를 사용할 때는 위 표에 있는 `Model ID`로 모델을 지정합니다. 예를 들면 다음과 같습니다:

```python
response = client.chat.completions.create(
    model="meta-llama/Llama-3.1-8B-Instruct",
    messages=[...]
)
```
