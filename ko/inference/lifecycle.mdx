새로운 모델은 W&B Inference 카탈로그의 일부로 자주 평가되고 배포됩니다.

기존 모델은 정기적으로 검토되며, 수요가 가장 높고 품질이 뛰어난 모델을 우선 제공하기 위해 사용이 중단될 수 있습니다.

<div id="model-lifecycle-stages">
  ## 모델 수명 주기 단계
</div>

W&B Inference의 모델은 수명 주기의 각 단계를 다음과 같은 용어로 구분합니다:

* **Generally Available**: 모델에 대해 완전한 지원이 제공되며, 사용을 권장하는 단계입니다.
* **Deprecated**: 모델이 제거될 예정인 단계입니다. 고객은 가능한 한 빨리 권장되는 대체 모델로 전환해야 합니다.
* **Retired**: 모델을 더 이상 사용할 수 없는 단계입니다. Retired 상태의 모델에 대한 모든 요청은 실패하며, `HTTP 404` 상태 코드를 반환합니다.

<div id="deprecation-process">
  ## 사용 중단 프로세스
</div>

모델을 사용 중단할 때 W&B는 해당 모델이 완전히 사용 종료되기 최소 2주 전에 이 페이지에 공지합니다.

또한, 사용 중단된 모델을 최근에 사용한 사용자에게 이메일로 알림을 보내고
대체 가능한 모델을 추천합니다.

**사용 중단(deprecated)** 단계에 있는 모델은 계속해서 요청을 처리합니다.
