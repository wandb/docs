새로운 모델은 W&amp;B Inference 카탈로그의 일부로 자주 평가되고 릴리스됩니다.

기존 모델은 가장 수요가 많고 품질이 높은 모델을 우선적으로 제공하기 위해 정기적으로 검토되며, 서비스 종료될 수 있습니다.

<div id="model-lifecycle-stages">
  ## 모델 수명 주기 단계
</div>

W&amp;B Inference의 모델는 수명 주기의 각 단계에 대해 다음 용어를 사용합니다:

* **정식 지원**: 모델이 정식으로 지원되며 사용이 권장됩니다.
* **사용 중단**: 모델이 제거될 예정입니다. 고객은 가능한 한 빨리 권장되는 대체 모델로 전환해야 합니다.
* **서비스 종료**: 모델을 더 이상 사용할 수 없습니다. 서비스 종료된 모델에 대한 모든 요청은 실패하며 `HTTP 404` 상태 코드를 반환합니다.

<div id="deprecation-process">
  ## 사용 중단 절차
</div>

모델을 사용 중단할 때 W&amp;B는 해당 모델이 서비스 종료되기 최소 2주 전에 이 페이지를 통해 안내합니다.

또한 최근에 사용 중단된 모델을 사용한 사용자에게 이메일로 알림을 보내고
대체 모델을 권장하기 위해 노력합니다.

**사용 중단** 단계의 모델은 계속 요청을 서빙합니다.
