이 페이지에는 API 또는 UI에서 Serverless Inference 서비스를 호출하기 전에 설정해야 하는 계정, 자격 증명, 도구가 나와 있습니다. 이후 요청이 인증에 성공하고 유효한 프로젝트로 라우팅되도록 다음 섹션의 모든 항목을 완료하세요.

<Tip>
  시작하기 전에 비용과 제한 사항을 파악할 수 있도록 [사용 정보 및 제한 사항](/ko/inference/usage-limits/)을 검토하세요.
</Tip>

<div id="set-up-your-wb-account-and-project">
  ## W&amp;B 계정 및 프로젝트 설정하기
</div>

Serverless Inference를 사용하려면 다음 항목이 필요합니다:

1. **W&amp;B 계정.** [W&amp;B](https://app.wandb.ai/login?signup=true)에서 가입하세요.

2. **W&amp;B API 키.**

   <ApiKeyCreate />

3. **W&amp;B 프로젝트.** 사용 내역을 추적할 수 있도록 W&amp;B 계정에서 프로젝트를 생성하세요.

이 세 가지를 모두 완료하면 Inference API에 인증할 수 있는 W&amp;B ID와 사용 내역을 기록하는 프로젝트가 준비됩니다.

<div id="set-up-your-environment-for-python">
  ## Python용 환경 설정
</div>

Python에서 Inference API를 호출할 계획이라면 일반 요구 사항 외에 다음도 설정해야 합니다.

1. 앞선 섹션의 일반 요구 사항을 완료합니다.

2. 필요한 라이브러리를 설치합니다.

   ```bash
   pip install openai weave
   ```

<Note>
  **참고**

  `weave` 라이브러리는 선택 사항이지만 권장됩니다. 이 라이브러리를 사용하면 LLM 애플리케이션을 트레이스할 수 있습니다. 자세한 내용은 [Weave 퀵스타트](/ko/models/quickstart/)를 참조하세요.
</Note>
