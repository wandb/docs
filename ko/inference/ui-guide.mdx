웹 UI에서 W&amp;B Inference 서비스를 사용하는 방법을 알아보세요. UI를 사용하기 전에 [사전 준비 사항](/ko/inference/prerequisites/)을 완료하세요.

<div id="access-the-inference-service">
  ## Inference 서비스에 접근하기
</div>

Inference 서비스에는 다음 세 가지 경로를 통해 접근할 수 있습니다:

<div id="direct-link">
  ### 직접 링크
</div>

[https://wandb.ai/inference](https://wandb.ai/inference) 페이지로 이동하세요.

<div id="from-the-inference-tab">
  ### Inference 탭에서
</div>

1. [https://wandb.ai/](https://wandb.ai/)에 접속해 자신의 W&amp;B 계정 페이지로 이동합니다.
2. 왼쪽 사이드바에서 **Inference**를 선택합니다.
3. 사용 가능한 모델과 해당 모델 정보가 표시된 페이지가 나타납니다.

<Frame>
  <img src="/images/inference/inference-playground-single.png" alt="Playground에서 Inference 모델 사용하기" />
</Frame>

<div id="from-the-playground-tab">
  ### Playground 탭에서
</div>

1. 왼쪽 사이드바에서 **Playground**를 선택합니다. Playground 채팅 UI가 표시됩니다.
2. LLM 드롭다운 목록에서 **W&amp;B Inference** 위에 마우스를 올려둡니다. 오른쪽에 사용 가능한 모델이 포함된 드롭다운이 표시됩니다.
3. 모델 드롭다운에서 다음 작업을 할 수 있습니다:
   * 아무 모델 이름이나 클릭하여 [Playground에서 사용해 보기](#try-a-model-in-the-playground)
   * [여러 모델 비교](#compare-multiple-models)

<Frame>
  <img src="/images/inference/inference-playground.png" alt="Playground의 Inference 모델 드롭다운" />
</Frame>

<div id="try-a-model-in-the-playground">
  ## Playground에서 모델 시험해 보기
</div>

[모델을 선택한](#access-the-inference-service) 후 Playground에서 테스트해 볼 수 있습니다. 다음 작업을 수행할 수 있습니다:

* [모델 설정과 파라미터 사용자 정의](/ko/weave/guides/tools/playground#customize-settings)
* [메시지 추가, 재시도, 편집, 삭제](/ko/weave/guides/tools/playground#message-controls)
* [사용자 정의 설정으로 모델 저장 및 재사용](/ko/weave/guides/tools/playground#saved-models)
* [여러 모델 비교](#compare-multiple-models)

<div id="compare-multiple-models">
  ## 여러 모델 비교
</div>

Playground에서 Inference 모델을 나란히 비교할 수 있습니다. Compare 뷰는 다음 두 곳에서 열 수 있습니다:

<div id="from-the-inference-tab">
  ### Inference 탭에서
</div>

1. 왼쪽 사이드바에서 **Inference**를 선택합니다. 사용 가능한 모델 목록 페이지가 표시됩니다.
2. 모델 카드를 선택하려면 (모델 이름을 제외한) 카드 아무 곳이나 클릭합니다. 카드 테두리가 파란색으로 바뀝니다.
3. 비교하려는 각 모델에 대해 이 작업을 반복합니다.
4. 선택한 카드 중 아무 카드에서나 **Playground에서 N개 모델 비교**를 클릭합니다. `N`은 선택한 모델 수를 나타냅니다.
5. 비교 뷰가 열립니다.

이제 [Playground에서 모델 사용해 보기](#try-a-model-in-the-playground)에 설명된 모든 기능을 사용하면서 모델을 비교할 수 있습니다.

<Frame>
  <img src="/images/inference/inference-playground-compare.png" alt="Playground에서 비교할 여러 모델 선택" />
</Frame>

<div id="from-the-playground-tab">
  ### Playground 탭에서
</div>

1. 왼쪽 사이드바에서 **Playground**를 선택합니다. Playground 채팅 UI가 표시됩니다.
2. LLM 드롭다운 목록에서 **W&amp;B Inference** 위에 마우스를 올립니다. 오른쪽에 모델 드롭다운이 표시됩니다.
3. 드롭다운에서 **Compare**를 선택합니다. **Inference** 탭이 표시됩니다.
4. 모델 카드에서 모델 이름을 제외한 아무 곳이나 클릭해 선택합니다. 카드 테두리가 파란색으로 바뀝니다.
5. 비교하려는 각 모델에 대해 이 작업을 반복합니다.
6. 선택된 카드 중 하나에서 **Compare N models in the Playground**를 클릭합니다. 비교 뷰가 열립니다.

이제 모델을 비교하고 [Playground에서 모델 사용해 보기](#try-a-model-in-the-playground)의 모든 기능을 사용할 수 있습니다.

<div id="view-billing-and-usage-information">
  ## 결제 및 사용량 정보 확인
</div>

조직 관리자는 W&amp;B UI에서 크레딧 잔액, 사용 이력, 예정 청구서를 확인할 수 있습니다:

1. UI에서 W&amp;B **Billing** 페이지로 이동합니다.
2. 오른쪽 하단에서 Inference 결제 정보 카드를 찾습니다.
3. 여기에서 다음을 수행할 수 있습니다:
   * **View usage**를 클릭해 시간 경과에 따른 사용량을 확인합니다.
   * 예정된 inference 청구 금액(유료 플랜의 경우)을 확인합니다.

<Tip>
  모델별 요금에 대한 자세한 내용은 [Inference pricing page](https://wandb.ai/site/pricing/inference)를 방문하세요.
</Tip>
