웹 UI에서 W&amp;B Inference 서비스를 사용하는 방법을 알아보세요. UI를 사용하기 전에 [사전 요구 사항](/ko/inference/prerequisites/)을 완료하세요.

<div id="access-the-inference-service">
  ## Inference 서비스에 액세스하기
</div>

Inference 서비스에는 세 가지 경로로 액세스할 수 있습니다:

<div id="direct-link">
  ### 직접 링크
</div>

[https://wandb.ai/inference](https://wandb.ai/inference)로 이동합니다.

<div id="from-the-inference-tab">
  ### Inference 탭에서
</div>

1. [https://wandb.ai/](https://wandb.ai/)에서 W&amp;B 계정으로 이동합니다.
2. 왼쪽 사이드바에서 **Inference**를 선택합니다.
3. 사용 가능한 모델과 모델 정보가 표시된 페이지가 열립니다.

<Frame>
  <img src="/images/inference/inference-playground-single.png" alt="플레이그라운드에서 Inference 모델 사용하기" />
</Frame>

<div id="from-the-playground-tab">
  ### 플레이그라운드 탭에서
</div>

1. 왼쪽 사이드바에서 **플레이그라운드**를 선택하세요. 플레이그라운드 채팅 UI가 나타납니다
2. LLM 드롭다운 목록에서 **W&amp;B Inference**에 마우스를 올리세요. 오른쪽에 사용 가능한 모델 목록이 있는 드롭다운이 나타납니다
3. 모델 드롭다운에서 다음 작업을 할 수 있습니다:
   * 원하는 모델 이름을 클릭하여 [플레이그라운드에서 사용해 보기](#try-a-model-in-the-playground)
   * [여러 모델 비교하기](#compare-multiple-models)

  <img src="/images/inference/inference-playground.png" alt="플레이그라운드의 Inference 모델 드롭다운" />

<div id="compare-multiple-models">
  ## 여러 모델 비교하기
</div>

플레이그라운드에서 Inference 모델을 나란히 비교할 수 있습니다. 다음 두 위치에서 비교 뷰에 액세스할 수 있습니다:

<div id="from-the-inference-tab">
  ### Inference 탭에서
</div>

1. 왼쪽 사이드바에서 **Inference**를 선택하세요. 사용 가능한 모델 페이지가 표시됩니다
2. 모델 카드를 선택하려면 카드의 아무 곳이나 클릭하세요(모델 이름 제외). 카드 테두리가 파란색으로 바뀝니다
3. 비교하려는 각 모델에 대해 반복하세요
4. 선택한 카드에서 **플레이그라운드에서 모델 N개 비교**를 클릭하세요. `N`은 선택한 모델 수를 나타냅니다
5. 비교 뷰가 열립니다

이제 모델을 비교하고 [플레이그라운드에서 모델 사용해 보기](#try-a-model-in-the-playground)의 모든 기능을 사용할 수 있습니다.

<Frame>
  <img src="/images/inference/inference-playground-compare.png" alt="플레이그라운드에서 비교할 여러 모델 선택" />
</Frame>

<div id="from-the-playground-tab">
  ### 플레이그라운드 탭에서
</div>

1. 왼쪽 사이드바에서 **플레이그라운드**를 선택합니다. 플레이그라운드 채팅 UI가 나타납니다.
2. LLM 드롭다운 목록의 **W&amp;B Inference**에 마우스를 올립니다. 오른쪽에 모델 드롭다운이 나타납니다.
3. 드롭다운에서 **비교**를 선택합니다. **Inference** 탭이 나타납니다.
4. 모델 카드(모델 이름 제외)의 아무 곳이나 클릭하여 선택합니다. 카드 테두리가 파란색으로 변합니다.
5. 비교하려는 각 모델에 대해 이 과정을 반복합니다.
6. 선택된 카드 중 하나에서 **비교 N models in the 플레이그라운드**를 클릭합니다. 비교 뷰가 열립니다.

이제 모델을 비교하고 [Try a model in the 플레이그라운드](#try-a-model-in-the-playground)의 모든 기능을 사용할 수 있습니다.

<div id="view-billing-and-usage-information">
  ## 결제 및 사용 정보 보기
</div>

조직 관리자는 W&amp;B UI에서 크레딧 잔액, 사용 이력, 그리고 예정된 청구 내역을 확인할 수 있습니다:

1. W&amp;B UI의 **Billing** 페이지로 이동합니다
2. 오른쪽 하단에서 Inference 결제 정보 카드를 찾습니다
3. 여기서 다음을 할 수 있습니다:
   * **View usage**를 클릭하여 시간 경과에 따른 사용을 확인합니다
   * 유료 플랜의 경우 예정된 Inference 요금을 확인합니다

<Tip>
  모델별 가격에 대한 자세한 내용은 [Inference 가격 페이지](https://wandb.ai/site/pricing/inference)를 방문하세요.
</Tip>
