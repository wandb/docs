W&amp;B Weave는 LLM 애플리케이션을 추적·평가하고 개선할 수 있도록 해주는 관측 및 평가 플랫폼입니다. Weave를 사용하면 다음과 같은 작업을 수행할 수 있습니다:

* LLM 애플리케이션을 [관찰하고 디버깅](/ko/weave/quickstart)할 수 있습니다
* LLM judge와 사용자 정의 scorer를 사용해 애플리케이션의 응답을 [평가](/ko/weave/tutorial-eval)할 수 있습니다

<div id="install-weave">
  ## Weave 설치
</div>

W&amp;B Weave는 Python 및 TypeScript용 라이브러리를 제공합니다. Weave 라이브러리를 설치하려면 다음 명령을 실행하세요:

<Tabs>
  <Tab title="Python">
    ```bash
    pip install weave
    ```
  </Tab>

  <Tab title="TypeScript">
    ```bash
    pnpm install weave
    ```
  </Tab>
</Tabs>

Weave 라이브러리를 사용하려면 먼저 [Weights &amp; Biases (W&amp;B) 계정](https://wandb.ai)과 [User Settings에서 API 키](https://wandb.ai/settings)를 생성해야 합니다. 이 API 키로 W&amp;B 계정에 인증한 다음 데이터 전송을 시작할 수 있습니다.
