W&B Weave 는 LLM 애플리케이션을 추적, 평가 및 개선하는 데 도움을 주는 관측성(observability) 및 평가 플랫폼입니다. Weave 를 사용하여 다음을 수행할 수 있습니다:

* LLM 애플리케이션 [관찰 및 디버깅](/weave/quickstart)
* LLM judge 및 커스텀 scorer를 사용하여 애플리케이션의 응답 [평가](/weave/tutorial-eval)

## Weave 설치

W&B Weave 는 Python 및 TypeScript 라이브러리를 제공합니다. Weave 라이브러리를 설치하려면 다음 코맨드를 실행하세요:

<Tabs>
  <Tab title="Python">
    ```bash
    pip install weave
    ```
  </Tab>
  <Tab title="TypeScript">
    ```bash
    pnpm install weave
    ```
  </Tab>
</Tabs>

Weave 라이브러리 사용을 시작하려면 [Weights & Biases (W&B) 계정](https://wandb.ai)을 생성하고 [사용자 설정(User Settings)에서 API 키](https://wandb.ai/settings)를 발급받으세요. API 키를 통해 W&B 계정에 인증하고 데이터를 전송하기 시작할 수 있습니다.
