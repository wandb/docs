<div id="product-comparison">
  ## 제품 비교
</div>

Weights &amp; Biases에 오신 것을 환영합니다! 제품을 사용하기 전에, 어떤 제품이 사용 사례에 가장 적합한지 파악하는 것이 중요합니다.

| 제품                                        | 적합한 용도           | 주요 기능                                          |
| ----------------------------------------- | ---------------- | ---------------------------------------------- |
| **[W&amp;B Models](#w%26b-models)**       | ML 모델을 처음부터 트레이닝 | 실험 추적, 하이퍼파라미터 최적화, 모델 레지스트리, 시각화              |
| **[W&amp;B Weave](#w%26b-weave)**         | LLM 애플리케이션 구축    | 트레이스, 프롬프트 관리, 평가, 프로덕션 AI 앱의 비용 추적 |
| **[W&amp;B Inference](#w%26b-inference)** | 사전 트레이닝된 모델 사용   | 호스팅된 오픈소스 모델, API 액세스, 테스트용 모델 플레이그라운드         |
| **[W&amp;B Training](#w%26b-training)**   | 모델 파인튜닝          | LoRA와 강화 학습 기반 맞춤형 모델 적응 생성 및 배포               |
