## 제품 비교

Weights & Biases 에 오신 것을 환영합니다! 제품을 시작하기 전에, 여러분의 유스 케이스에 어떤 제품이 가장 적합한지 확인하는 것이 중요합니다. 

| 제품 | 적합한 용도 | 주요 기능 |
|---------|----------|--------------|
| **[W&B Models](#w%26b-models)** | 처음부터 ML 모델 트레이닝 | 실험 트래킹, 하이퍼파라미터 최적화, 모델 레지스트리, 시각화 |
| **[W&B Weave](#w%26b-weave)** | LLM 애플리케이션 구축 | 프로덕션 AI 앱을 위한 Tracing, 프롬프트 관리, 평가, 비용 추적 |
| **[W&B Inference](#w%26b-inference)** | 사전학습된 모델 사용 | 호스팅된 오픈 소스 모델, API 엑세스, 테스트용 모델 플레이그라운드 |
| **[W&B Training](#w%26b-training)** | 모델 파인튜닝 | 강화학습을 통한 LoRA 및 커스텀 모델 적응(adaptation) 생성 및 배포 |
