<div id="product-comparison">
  ## 제품 비교
</div>

Weights &amp; Biases에 오신 것을 환영합니다! 제품을 사용하기 전에 어떤 제품이 사용자의 사용 사례에 가장 잘 맞는지 파악하는 것이 중요합니다.

| Product | Best For | Key Features |
|---------|----------|--------------|
| **[W&amp;B Models](#w%26b-models)** | ML 모델을 처음부터 트레이닝할 때 | 실험 추적, 하이퍼파라미터 최적화, 모델 레지스트리, 시각화 |
| **[W&amp;B Weave](#w%26b-weave)** | LLM 애플리케이션 구축 | 트레이싱, 프롬프트 관리, 평가, 프로덕션 AI 애플리케이션 비용 추적 |
| **[W&amp;B Inference](#w%26b-inference)** | 사전 학습된 모델 사용 | 호스팅되는 오픈 소스 모델, API 액세스, 테스트용 모델 플레이그라운드 |
| **[W&amp;B Training](#w%26b-training)** | 모델 파인튜닝 | 강화학습을 사용해 LoRA 및 커스텀 모델 변형을 생성하고 배포 |
