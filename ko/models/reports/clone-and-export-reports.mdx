<div id="export-reports">
  ## 리포트 내보내기
</div>

리포트를 PDF 또는 LaTeX 형식으로 내보낼 수 있습니다. 리포트에서 케밥 메뉴 아이콘을 선택해 드롭다운 메뉴를 연 다음 **Download and**를 선택하고, PDF 또는 LaTeX 출력 형식 중 하나를 선택합니다.

<div id="cloning-reports">
  ## 리포트 복제하기
</div>

<Tabs>
<Tab title="W&amp;B 앱">
리포트에서 케밥 아이콘을 선택해 드롭다운 메뉴를 펼칩니다. **Clone this report** 버튼을 선택합니다. 모달에서 리포트 복제본을 저장할 위치를 선택합니다. **Clone report**를 선택합니다.

<Frame>
    <img src="/images/reports/clone_reports.gif" alt="리포트 복제하기"  />
</Frame>

리포트를 복제하면 프로젝트의 템플릿과 형식을 다시 사용할 수 있습니다. 팀 계정에서 리포트를 복제하면 생성된 리포트 복제본은 팀 전체에 표시됩니다. 개인 계정에서 리포트를 복제하면 해당 복제본은 그 사용자에게만 표시됩니다.
</Tab>
<Tab title="리포트 및 워크스페이스 API">
템플릿으로 사용하기 위해 URL에서 Report를 로드합니다.

```python
report = wr.Report(
    project=PROJECT, title="Quickstart Report", description="That was easy!"
)  # 생성
report.save()  # 저장
new_report = wr.Report.from_url(report.url)  # 로드
```

`new_report.blocks` 안의 내용을 수정합니다.

```python
pg = wr.PanelGrid(
    runsets=[
        wr.Runset(ENTITY, PROJECT, "First Run Set"),
        wr.Runset(ENTITY, PROJECT, "Elephants Only!", query="elephant"),
    ],
    panels=[
        wr.LinePlot(x="Step", y=["val_acc"], smoothing_factor=0.8),
        wr.BarPlot(metrics=["acc"]),
        wr.MediaBrowser(media_keys="img", num_columns=1),
        wr.RunComparer(diff_only="split", layout={"w": 24, "h": 9}),
    ],
)
new_report.blocks = (
    report.blocks[:1] + [wr.H1("Panel Grid Example"), pg] + report.blocks[1:]
)
new_report.save()
```
