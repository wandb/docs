## Reports 내보내기

Reports를 PDF 또는 LaTeX 형식으로 내보낼 수 있습니다. Report 내에서 케밥 아이콘을 클릭하여 드롭다운 메뉴를 확장합니다. **Download**를 선택한 후 PDF 또는 LaTeX 출력 형식을 선택하세요.

## Reports 복제하기

<Tabs>
<Tab title="W&B App">
Report 내에서 케밥 아이콘을 클릭하여 드롭다운 메뉴를 확장합니다. **Clone this report** 버튼을 선택하세요. 모달 창에서 복제된 Report를 저장할 대상을 선택한 뒤, **Clone report**를 클릭합니다.

프로젝트의 템플릿과 형식을 재사용하려면 Report를 복제하세요. Team 계정 내에서 프로젝트를 복제하면 팀원들이 해당 프로젝트를 볼 수 있습니다. 개인 계정 내에서 복제된 프로젝트는 해당 User에게만 보입니다.
</Tab>
<Tab title="Report and Workspace API">
URL에서 Report를 로드하여 템플릿으로 사용합니다.

```python
report = wr.Report(
    project=PROJECT, title="Quickstart Report", description="That was easy!"
)  # 생성
report.save()  # 저장
new_report = wr.Report.from_url(report.url)  # 로드
```

`new_report.blocks` 내의 콘텐츠를 편집합니다.

```python
pg = wr.PanelGrid(
    runsets=[
        wr.Runset(ENTITY, PROJECT, "First Run Set"),
        wr.Runset(ENTITY, PROJECT, "Elephants Only!", query="elephant"),
    ],
    panels=[
        wr.LinePlot(x="Step", y=["val_acc"], smoothing_factor=0.8),
        wr.BarPlot(metrics=["acc"]),
        wr.MediaBrowser(media_keys="img", num_columns=1),
        wr.RunComparer(diff_only="split", layout={"w": 24, "h": 9}),
    ],
)
new_report.blocks = (
    report.blocks[:1] + [wr.H1("Panel Grid Example"), pg] + report.blocks[1:]
)
new_report.save()
```
