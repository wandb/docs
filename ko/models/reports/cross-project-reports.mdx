  [프로젝트 간 run 비교를 시연하는 동영상](https://www.youtube.com/watch?v=uD4if_nGrs4)을 시청하세요 (2분).
</Note>

프로젝트 간 Reports를 사용해 서로 다른 두 프로젝트의 run을 비교할 수 있습니다. run 세트 테이블의 프로젝트 선택기를 사용해 프로젝트를 선택하세요.

<Frame>
  <img src="/images/reports/howto_pick_a_different_project_to_draw_runs_from.gif" alt="서로 다른 프로젝트 간 run 비교" />
</Frame>

이 섹션의 시각화는 첫 번째 활성 runset에서 열을 가져옵니다. 선 그래프에서 원하는 메트릭이 보이지 않는다면, 이 섹션에서 가장 먼저 체크된 run 세트에 해당 열이 있는지 확인하세요.

이 기능은 시계열 선 그래프에서 이력 데이터를 지원하지만, 서로 다른 프로젝트에서 서로 다른 summary 메트릭을 가져오는 것은 지원하지 않습니다. 다시 말해, 다른 프로젝트에서만 로깅된 열로 산점도를 만들 수는 없습니다.

두 프로젝트의 run을 비교해야 하는데 열이 제대로 동작하지 않는다면, 한 프로젝트의 run에 태그를 추가한 다음 그 run들을 다른 프로젝트로 옮기세요. 여전히 각 프로젝트의 run만 필터링할 수 있지만, 리포트에는 두 run 세트의 모든 열이 포함됩니다.

<div id="view-only-report-links">
  ## 보기 전용 리포트 링크
</div>

비공개 프로젝트나 팀 프로젝트에 있는 리포트에 대해 보기 전용 링크를 공유합니다.

<Frame>
  <img src="/images/reports/magic-links.gif" alt="보기 전용 리포트 링크" />
</Frame>

보기 전용 리포트 링크는 URL에 비밀 액세스 토큰을 추가하므로, 링크를 여는 사람은 누구나 페이지를 볼 수 있습니다. 누구나 로그인하지 않고도 매직 링크를 사용해 리포트를 열람할 수 있습니다. [W&amp;B Local](/ko/platform/hosting/) 프라이빗 클라우드 설치를 사용하는 고객의 경우, 이러한 링크는 여전히 방화벽 뒤에 있으므로, 프라이빗 인스턴스에 대한 액세스 권한 *및* 보기 전용 링크에 대한 액세스 권한을 가진 팀 구성원만 리포트를 볼 수 있습니다.

**보기 전용 모드**에서는 로그인하지 않은 사람도 차트를 보고, 값에 대한 툴팁을 확인하기 위해 마우스를 올리거나, 차트를 확대/축소하거나, 테이블에서 열을 스크롤할 수 있습니다. 보기 전용 모드에서는 데이터를 탐색하기 위한 새 차트나 새 테이블 쿼리를 만들 수 없습니다. 리포트 링크의 보기 전용 방문자는 run을 클릭해 run 페이지로 이동할 수 없습니다. 또한, 보기 전용 방문자는 공유 모달을 볼 수 없으며, 대신 마우스를 올렸을 때 `Sharing not available for view only access`라는 툴팁이 표시됩니다.

<Info>
  매직 링크는 “Private” 및 “Team” 프로젝트에서만 사용할 수 있습니다. “Public”(누구나 보기 가능) 또는 “Open”(누구나 보기 및 run 기여 가능) 프로젝트의 경우, 해당 프로젝트는 이미 공개 상태이므로 링크를 가진 누구나 접근할 수 있으며, 링크를 통해 공개 여부를 켜거나 끌 수 없습니다.
</Info>

<div id="send-a-graph-to-a-report">
  ## 그래프를 리포트로 보내기
</div>

워크스페이스에서 리포트로 그래프를 보내 진행 상황을 추적하세요. 리포트에 복사하려는 차트나 패널에서 드롭다운 메뉴를 클릭한 다음 **Add to report**를 클릭하여 대상 리포트를 선택합니다.
