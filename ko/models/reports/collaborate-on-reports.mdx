이 페이지에서는 팀 멤버들과 Reports 에서 협업하는 다양한 방법을 설명합니다.

## Report 공유하기
Report를 보고 있을 때 **Share**를 클릭한 후 다음을 수행하세요:
- 이메일 주소나 사용자 이름을 사용하여 Report 링크를 공유하려면 **Invite**를 클릭하세요. 이메일 주소 또는 사용자 이름을 입력하고, **Can view** 또는 **Can edit**을 선택한 다음 **Invite**를 클릭합니다. 이메일로 공유하는 경우, 해당 이메일 주소는 조직이나 팀의 멤버가 아니어도 됩니다.
- 대신 공유 링크를 생성하려면 **Share**를 클릭하세요. 링크에 대한 권한을 조정한 다음 **Copy report link**를 클릭합니다. 해당 멤버에게 링크를 공유하세요.

Report를 보는 동안 패널을 클릭하면 전체 화면 모드로 열립니다. 브라우저의 URL을 복사하여 다른 사용자에게 공유하면, 상대방이 해당 링크에 엑세스할 때 패널이 즉시 전체 화면 모드로 열립니다.

## Report 편집하기
팀 멤버가 **Edit** 버튼을 클릭하여 Report 편집을 시작하면 초안이 자동으로 저장됩니다. 변경 사항을 게시하려면 **Save to report**를 선택하세요.

두 명의 팀 멤버가 동시에 Report를 편집하는 것과 같은 편집 충돌이 발생하면, 경고 알림을 통해 충돌을 해결할 수 있도록 도와줍니다.

## Report에 댓글 달기
Report에 댓글을 남기려면 **Comment**를 클릭하세요.

패널에 직접 댓글을 달려면 패널 위로 마우스를 가져간 다음 말풍선 모양의 댓글 버튼을 클릭하세요.

## Report에 별표 표시하기 (Star)
팀에 Reports 양이 많은 경우, Report 상단의 **Star**를 클릭하여 즐겨찾기에 추가하세요. 팀의 Report 목록을 볼 때, 해당 Report 행의 별표를 클릭하여 즐겨찾기에 추가할 수도 있습니다. 별표 표시된 Reports 는 목록의 상단에 나타납니다.

Report 목록에서 각 Report에 얼마나 많은 멤버가 별표를 표시했는지 확인하여 상대적인 인기도를 가늠할 수 있습니다.
