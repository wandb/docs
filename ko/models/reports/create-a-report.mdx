<Note>

</Note>

아래 탭을 선택해 W&amp;B App에서 또는 W&amp;B Report 및 Workspace API를 사용해 프로그래밍 방식으로 리포트를 생성하는 방법을 확인하세요.

리포트를 프로그래밍 방식으로 생성하는 예시는 이 [Google Colab](https://colab.research.google.com/github/wandb/examples/blob/master/colabs/intro/Report_API_Quickstart.ipynb)을 참고하세요.

<Tabs>
  <Tab title="W&B App">
    1. W&amp;B App에서 프로젝트 워크스페이스로 이동합니다.

    2. 워크스페이스 오른쪽 상단에서 **Create report**를 클릭합니다.

       <Frame>
         <img src="/images/reports/create_a_report_button.png" alt="리포트 생성 버튼" />
       </Frame>

    3. 모달이 나타납니다. 시작할 때 사용할 차트를 선택합니다. 나중에 리포트 인터페이스에서 차트를 추가하거나 삭제할 수 있습니다.

       <Frame>
         <img src="/images/reports/create_a_report_modal.png" alt="리포트 생성 모달" />
       </Frame>

    4. 새 run이 리포트에 추가되지 않도록 하려면 **Filter run sets** 옵션을 선택합니다. 이 옵션은 언제든지 켜거나 끌 수 있습니다. **Create report**를 클릭하면, 계속 작업할 수 있는 초안 리포트가 리포트 탭에서 확인할 수 있는 상태로 생성됩니다.
  </Tab>

  <Tab title="Report tab">
    1. W&amp;B App에서 프로젝트 워크스페이스로 이동합니다.
    2. 프로젝트의 **Reports** 탭(클립보드 아이콘)을 선택합니다.
    3. 리포트 페이지에서 **Create Report** 버튼을 선택합니다.

       <Frame>
         <img src="/images/reports/create_report_button.png" alt="리포트 생성 버튼" />
       </Frame>
  </Tab>

  <Tab title="Report and Workspace API">
    다음과 같이 프로그래밍 방식으로 리포트를 생성할 수 있습니다:

    1. W&amp;B SDK (`wandb`)와 Report 및 Workspace API (`wandb-workspaces`)를 설치합니다:
       ```bash
       pip install wandb wandb-workspaces
       ```

    2. 다음으로 workspaces를 임포트합니다.
       ```python
       import wandb
       import wandb_workspaces.reports.v2 as wr
       ```

    3. `wandb_workspaces.reports.v2.Report`로 리포트를 생성합니다. Report 클래스 Public API([`wandb.apis.reports`](/ko/models/ref/python/public-api/api#reports))를 사용해 리포트 인스턴스를 생성합니다. 프로젝트 이름을 지정합니다.
       ```python
       report = wr.Report(project="report_standard")
       ```

    4. 리포트를 저장합니다. `.save()` 메서드를 호출하기 전까지는 리포트가 W&amp;B 서버에 업로드되지 않습니다:
       ```python
       report.save()
       ```

    App UI에서 인터랙티브하게 또는 프로그래밍 방식으로 리포트를 편집하는 방법은 [리포트 편집](/ko/models/reports/edit-a-report/)을 참고하세요.
  </Tab>
</Tabs>
