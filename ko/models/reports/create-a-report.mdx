<Note>
W&B Report 및 Workspace API는 현재 퍼블릭 프리뷰 단계입니다.
</Note>

아래 탭 중 하나를 선택하여 W&B 앱에서 Reports를 생성하거나 W&B Report 및 Workspace API를 사용하여 프로그래밍 방식으로 생성하는 방법을 확인하세요.

프로그래밍 방식으로 리포트를 생성하는 예제는 이 [Google Colab](https://colab.research.google.com/github/wandb/examples/blob/master/colabs/intro/Report_API_Quickstart.ipynb)을 참조하세요.


<Tabs>
<Tab title="W&B App">
1. W&B 앱에서 해당 Projects의 Workspace로 이동합니다.
2. Workspace의 오른쪽 상단에 있는 **Create report**를 클릭합니다.

    <Frame>
    <img src="/images/reports/create_a_report_button.png" alt="Create report button"  />
    </Frame>

3. 모달 창이 나타납니다. 시작할 때 포함할 차트들을 선택하세요. 나중에 리포트 인터페이스에서 차트를 추가하거나 삭제할 수 있습니다.

    <Frame>
    <img src="/images/reports/create_a_report_modal.png" alt="Create report modal"  />
    </Frame>

4. 새로운 Runs가 리포트에 자동으로 추가되지 않도록 하려면 **Filter run sets** 옵션을 선택하세요. 이 옵션은 나중에 켜거나 끌 수 있습니다. **Create report**를 클릭하면 리포트 탭에 초안 리포트가 생성되어 작업을 계속할 수 있습니다.
</Tab>

3. 리포트 페이지에서 **Create Report** 버튼을 선택합니다.

<Tab title="Report and Workspace API">
프로그래밍 방식으로 리포트 생성하기:

1. W&B SDK(`wandb`)와 Report 및 Workspace API(`wandb-workspaces`)를 설치합니다:
    ```bash
    pip install wandb wandb-workspaces
    ```
2. 다음으로 workspaces를 임포트합니다.
    ```python
    import wandb
    import wandb_workspaces.reports.v2 as wr
    ```       
3. `wandb_workspaces.reports.v2.Report`를 사용하여 리포트를 생성합니다. Report Class Public API ([`wandb.apis.reports`](/models/ref/python/public-api/api#reports))를 사용하여 리포트 인스턴스를 생성하세요. 프로젝트 이름을 지정해야 합니다.   
    ```python
    report = wr.Report(project="report_standard")
    ```  

4. 리포트를 저장합니다. 리포트는 `save()` 메소드를 호출하기 전까지는 W&B 서버에 업로드되지 않습니다:
    ```python
    report.save()
    ```

앱 UI를 통해 대화형으로 또는 프로그래밍 방식으로 리포트를 편집하는 방법에 대한 자세한 내용은 [Edit a report](/models/reports/edit-a-report/)를 참조하세요.
</Tab>
