description: run이 활성 상태일 때 또는 Public API만 사용하여 기존 artifact를 업데이트합니다.
title: artifact 업데이트
---

artifact의 `description`, `metadata`, `alias`를 업데이트하려면 원하는 값을 지정하세요. W&amp;B Public API([`wandb.Api`](/ko/models/ref/python/public-api/api))를 사용해 이전에 W&amp;B에 로깅된 run을 업데이트하세요. artifact가 처음 초기화된 후 아직 활성 상태라면 `wandb.Run.save()`를 사용해 업데이트하세요.

<Note>
  **Artifact.save() 또는 wandb.Run.log&#95;artifact()를 사용해야 하는 경우**

  * 새 run을 만들지 않고 기존 artifact를 업데이트하려면 `Artifact.save()`를 사용하세요.
  * 새 artifact를 만들고 특정 run에 연결하려면 `wandb.Run.log_artifact()`를 사용하세요.
</Note>

artifact를 업데이트하려면 W&amp;B Public API([`wandb.Api`](/ko/models/ref/python/public-api/api))를 사용하세요. run이 활성 상태인 동안에는 wandb.Artifact([`wandb.Artifact`](/ko/models/ref/python/experiments/artifact)) Class를 사용하세요.

<Warning>
  Model Registry의 모델에 연결된 artifact의 alias는 업데이트할 수 없습니다.
</Warning>

<Tabs>
  <Tab title="run 중">
    다음 코드 예제는 [`wandb.Artifact`](/ko/models/ref/python/experiments/artifact) API를 사용해 artifact의 설명을 업데이트하는 방법을 보여줍니다:

    ```python
    import wandb

    with wandb.init(project="<example>") as run:
        artifact = run.use_artifact("<artifact-name>:<alias>")
        artifact.description = "<description>"
        artifact.save()
    ```
  </Tab>

  </Tab>

  <Tab title="collection과 함께">
    단일 artifact와 같은 방식으로 Artifact collection도 업데이트할 수 있습니다:

    ```python
    import wandb
    with wandb.init(project="<example>") as run:
        api = wandb.Api()
        artifact = api.artifact_collection(type="<type-name>", collection="<collection-name>")
        artifact.name = "<new-collection-name>"
        artifact.description = "<This is where you'd describe the purpose of your collection.>"
        artifact.save()
    ```

    자세한 내용은 [Artifacts Collection](/ko/models/ref/python/public-api/api) 레퍼런스를 참조하세요.
  </Tab>
