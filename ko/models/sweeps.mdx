W&amp;B Sweeps를 사용해 하이퍼파라미터 검색을 자동화하고 풍부하고 인터랙티브한 실험 추적 결과를 시각화하세요. Bayesian, 그리드 서치, 랜덤과 같은 널리 쓰이는 검색 방법을 선택해 하이퍼파라미터 공간을 탐색할 수 있습니다. 스윕을 하나 이상의 머신으로 확장해 병렬로 실행하세요.

<div id="how-it-works">
  ## 동작 방식
</div>

다음 두 개의 [W&amp;B CLI](/ko/models/ref/cli/) 명령을 사용해 스윕을 생성합니다.

1. 스윕을 초기화합니다.

```bash
  wandb sweep --project <project-name> <path-to-config file>
```

2. 스윕 에이전트를 시작합니다.

```bash
  wandb agent <sweep-ID>
```

<Note>
  앞의 코드 스니펫과 이 페이지에 링크된 Colab 노트북은 W&amp;B CLI를 사용해 스윕을 초기화하고 생성하는 방법을 보여줍니다. Python SDK를 사용해 스윕을 구성, 초기화 및 실행하는 방법은 [Sweeps walkthrough](/ko/models/sweeps/walkthrough/)를 참조하세요.
</Note>

  * [코드에 W&amp;B 추가하기](/ko/models/sweeps/add-w-and-b-to-your-code/)
  * [스윕 설정 정의하기](/ko/models/sweeps/define-sweep-configuration/)
  * [스윕 초기화하기](/ko/models/sweeps/initialize-sweeps/)
  * [스윕 에이전트 시작하기](/ko/models/sweeps/start-sweep-agents/)
  * [스윕 결과 시각화하기](/ko/models/sweeps/visualize-sweep-results/)
