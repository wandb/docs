<div id="how-it-works">
  ## 동작 방식
</div>

두 개의 [W&amp;B CLI](/ko/models/ref/cli/) 명령을 사용해 sweep을 생성합니다:

1. sweep을 초기화합니다.

```bash
  wandb sweep --project <project-name> <path-to-config file>
```

2. 스윕 에이전트를 실행합니다.

```bash
  wandb agent <sweep-ID>
```

<Note>
  앞의 코드 스니펫과 이 페이지에 링크된 Colab 노트북은 W&amp;B CLI를 사용해 스윕을 초기화하고 생성하는 방법을 보여줍니다. Python SDK를 사용해 스윕을 구성하고, 초기화하고, 실행하는 방법은 [Sweeps walkthrough](/ko/models/sweeps/walkthrough/)를 참고하세요.
</Note>

<div id="how-to-get-started">
  ## 시작하기
</div>

사용 사례에 따라 아래 리소스를 활용해 W&amp;B Sweeps를 시작하세요:

* W&amp;B Python SDK 명령을 사용해 스윕 구성 정의, 스윕 초기화, 스윕 시작까지 단계별로 정리한 [sweeps walkthrough](/ko/models/sweeps/walkthrough/)를 읽어 보세요.
* 이 챕터를 살펴보면서 다음 내용을 학습하세요:
  * [코드에 W&amp;B 추가하기](/ko/models/sweeps/add-w-and-b-to-your-code/)
  * [스윕 구성 정의하기](/ko/models/sweeps/define-sweep-configuration/)
  * [스윕 초기화하기](/ko/models/sweeps/initialize-sweeps/)
  * [스윕 에이전트 시작하기](/ko/models/sweeps/start-sweep-agents/)
  * [스윕 결과 시각화하기](/ko/models/sweeps/visualize-sweep-results/)
* W&amp;B Sweeps로 하이퍼파라미터 최적화를 수행한 [선별된 Sweep 실험 목록](/ko/models/sweeps/useful-resources/)을 살펴보세요. 결과는 W&amp;B Reports에 저장됩니다.

단계별 동영상 가이드는 다음을 참고하세요: [Tune Hyperparameters Easily with W&amp;B Sweeps](https://www.youtube.com/watch?v=9zrmUIlScdY\&ab_channel=Weights%26Biases).
