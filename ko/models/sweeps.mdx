W&B Sweeps 를 사용하여 하이퍼파라미터 검색을 자동화하고 풍부하고 인터랙티브한 실험 트래킹을 시각화하세요. 베이지안 (Bayesian), 그리드 검색 (grid search), 랜덤 (random)과 같은 인기 있는 검색 메소드 중에서 선택하여 하이퍼파라미터 공간을 탐색할 수 있습니다. 하나 이상의 머신에서 스윕을 확장하고 병렬화하세요.

### 작동 방식
두 개의 [W&B CLI](/models/ref/cli/) 코맨드로 스윕을 생성합니다:

1. 스윕 초기화

```bash
wandb sweep --project <project-name> <path-to-config file>
```

2. 스윕 에이전트 시작

```bash
wandb agent <sweep-ID>
```

<Note>
위의 코드조각과 이 페이지에 링크된 Colab은 W&B CLI를 사용하여 스윕을 초기화하고 생성하는 방법을 보여줍니다. 스윕 구성을 정의하고, 스윕을 초기화하고, 스윕을 시작하는 데 사용하는 W&B Python SDK 코맨드에 대한 단계별 개요는 [Sweeps walkthrough](/models/sweeps/walkthrough/)를 참조하세요.
</Note>
