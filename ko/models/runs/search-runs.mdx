---
title: "run 검색"
description: "프로젝트의 Runs 테이블 또는 Workspace에서 이름이나 ID로 특정 run을 검색하는 방법을 알아봅니다."
---

프로젝트의 Runs 테이블 또는 Workspace에 있는 검색 상자를 사용해 특정 [이름 또는 ID로 run](/ko/models/runs/run-identifiers)을 찾아보세요.

<Note>
  기본적으로 검색 상자는 정규 표현식(RegEx)을 사용해 쿼리를 run 이름이나 ID와 일치시킵니다.
</Note>

<div id="search-for-runs-by-name-or-id">
  ## 이름 또는 ID로 run 검색
</div>

1. 프로젝트 사이드바에서 **Runs** 탭 또는 **Workspace**를 클릭합니다.
2. Runs 테이블 상단의 검색 상자를 클릭합니다.
3. 검색하려는 run 이름 또는 run ID를 입력합니다.

<div id="turn-off-regular-expressions-search">
  ## 정규식 검색 끄기
</div>

1. 프로젝트 사이드바에서 **Runs** 탭 또는 **Workspace**를 클릭합니다.
2. Runs 테이블 상단의 검색 상자를 클릭합니다.
3. **RegEx** 토글 (.*)이 회색이 되도록 끕니다.
