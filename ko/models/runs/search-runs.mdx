---

프로젝트의 Runs table 또는 Workspace의 검색 상자를 사용해 [이름이나 ID로 특정 run을 찾으세요](/ko/models/runs/run-identifiers).

<Note>
  기본적으로 검색 상자는 정규식(RegEx)을 사용해 쿼리를 run 이름이나 ID와 매칭합니다.
</Note>

<div id="search-for-runs-by-name-or-id">
  ## 이름 또는 ID로 run 검색하기
</div>

1. 프로젝트 사이드바에서 **Runs** 탭 또는 **Workspace**를 클릭합니다.
2. Runs table 상단의 검색 상자를 클릭합니다.
3. 검색할 run name 또는 run ID를 입력합니다.

<div id="turn-off-regular-expressions-search">
  ## 정규식 검색 끄기
</div>

1. 프로젝트 사이드바에서 **Runs** 탭 또는 **Workspace**를 클릭합니다.
2. Runs table 상단의 검색 상자를 클릭합니다.
3. **RegEx** 표시/숨기기(.*)를 회색으로 비활성화합니다.
