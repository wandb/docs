프로젝트의 Runs 테이블 또는 Workspace 내에 있는 검색 상자를 사용하여 이름이나 ID로 특정 [runs](/models/runs/run-identifiers)를 찾을 수 있습니다.

<Note>
기본적으로 검색 상자는 정규 표현식 (RegEx) 을 사용하여 쿼리를 run 이름 또는 ID와 매칭합니다.
</Note>

## 이름 또는 ID로 runs 검색하기
1. 프로젝트 사이드바에서 **Runs** 탭 또는 **Workspace**를 클릭합니다.
2. runs 테이블 상단에 있는 검색 상자를 클릭합니다.
3. 검색하려는 run 이름 또는 run ID를 입력합니다.

## 정규 표현식 검색 끄기

1. 프로젝트 사이드바에서 **Runs** 탭 또는 **Workspace**를 클릭합니다.
2. runs 테이블 상단에 있는 검색 상자를 클릭합니다.
3. **RegEx** 토글 (.*) 을 클릭하여 회색으로 표시되도록 끕니다.
