---
title: "실행 검색"
description: "프로젝트의 Runs 테이블 또는 Workspace에서 이름이나 ID로 특정 실행을 검색하는 방법을 알아보세요."
---

프로젝트의 Runs 테이블 또는 Workspace에 있는 검색 상자를 사용해 특정 [이름 또는 ID로 식별되는 실행](/ko/models/runs/run-identifiers)을 찾을 수 있습니다.

<Note>
  기본적으로 검색 상자는 정규식(RegEx)을 사용해 검색어를 실행 이름 또는 ID와 매칭합니다.
</Note>

<div id="search-for-runs-by-name-or-id">
  ## 실행 이름 또는 ID로 검색하기
</div>

1. 프로젝트 사이드바에서 **Runs** 탭 또는 **Workspace**를 클릭합니다.
2. 실행 테이블 상단에 있는 검색 상자를 클릭합니다.
3. 검색하려는 실행 이름 또는 실행 ID를 입력합니다.

<div id="turn-off-regular-expressions-search">
  ## 정규식 검색 끄기
</div>

1. 프로젝트 사이드바에서 **Runs** 탭 또는 **Workspace**를 클릭합니다.
2. Runs 테이블 상단의 검색 창을 클릭합니다.
3. **RegEx** 토글(.*)이 회색으로 표시되도록 끕니다.
