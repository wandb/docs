---
title: 실행에 태그를 추가해 레이블 지정하기
---

로그된 지표나 아티팩트 데이터만으로는 드러나지 않는 특정 특성을 실행에 레이블로 표시하려면 태그를 추가하세요.

예를 들어, 한 실행의 모델이 `in_production` 상태임을 나타내거나, 해당 실행이 `preemptible`인지, 이 실행이 `baseline`을 나타내는지 등을 표시하기 위해 실행에 태그를 추가할 수 있습니다.

<div id="add-tags-to-one-or-more-runs">
  ## 하나 이상의 실행에 태그 추가하기
</div>

프로그래밍 방식이나 인터랙티브하게 실행에 태그를 추가할 수 있습니다.

사용 사례에 따라 아래 탭에서 목적에 가장 잘 맞는 방법을 선택하세요:

<Tabs>
  <Tab title="W&B Python SDK">
    실행이 생성될 때 태그를 추가할 수 있습니다:

    ```python
    import wandb

    with wandb.init(
      entity="entity",
      project="<project-name>",
      tags=["tag1", "tag2"]
    ) as run:
        # 여기에 학습 코드를 작성하세요
    ```

    실행을 초기화한 이후에도 태그를 업데이트할 수 있습니다. 예를 들어, 다음 코드 스니펫은 특정 메트릭이 사전에 정의한 임계값을 넘었을 때 태그를 업데이트하는 방법을 보여줍니다:

    ```python
    import wandb

    with wandb.init(entity="entity", project="capsules", tags=["debug"]) as run:

      # 모델을 학습하는 Python 로직
      if current_loss < threshold:
          run.tags = run.tags + ("release_candidate",)
    ```
  </Tab>

  <Tab title="Public API">
    실행을 생성한 이후에는 [Public API](/ko/models/track/public-api-guide/)를 사용해 태그를 업데이트할 수 있습니다. 예를 들어 다음과 같습니다:

    ```python
    with wandb.Api().run("{entity}/{project}/{run-id}") as run:
      run.tags.append("tag1")  # 여기에서 실행 데이터를 기반으로 태그를 선택할 수 있습니다
      run.update()
    ```
  </Tab>

  <Tab title="Project page">
    이 방법은 동일한 태그(또는 태그 집합)를 많은 실행에 한 번에 추가할 때 가장 적합합니다.

    1. 프로젝트 워크스페이스로 이동합니다.
    2. 프로젝트 사이드바에서 **Runs**를 선택합니다.
    3. 테이블에서 하나 이상의 실행을 선택합니다.
    4. 하나 이상의 실행을 선택한 후, 테이블 위에 있는 **Tag** 버튼을 선택합니다.
    5. 추가하려는 태그를 입력하고 **Create new tag** 체크박스를 선택하여 태그를 추가합니다.
  </Tab>

  <Tab title="Run page">
    이 방법은 단일 실행에 수동으로 하나 또는 여러 개의 태그를 적용할 때 가장 적합합니다.

    1. 프로젝트 워크스페이스로 이동합니다.
    2. 프로젝트 워크스페이스 내 실행 목록에서 하나의 실행을 선택합니다.
    3. 프로젝트 사이드바에서 **Overview**를 선택합니다.
    4. **Tags** 옆의 회색 플러스 아이콘(**+**) 버튼을 선택합니다.
    5. 추가하려는 태그를 입력하고 텍스트 상자 아래의 **Add**를 선택해 새 태그를 추가합니다.
  </Tab>
</Tabs>

<div id="remove-tags-from-one-or-more-runs">
  ## 하나 이상의 실행에서 태그 제거하기
</div>

W&amp;B App에서 실행의 태그를 제거하려면 다음 단계를 따르세요.

<Tabs>
  <Tab title="프로젝트 페이지">
    이 방법은 많은 수의 실행에서 태그를 제거할 때 가장 적합합니다.

    1. 프로젝트의 실행 사이드바에서 오른쪽 상단의 테이블 아이콘을 선택합니다. 그러면 사이드바가 전체 실행 테이블로 확장됩니다.
    2. 테이블에서 실행 위에 마우스를 올려 왼쪽에 표시되는 체크박스를 확인하거나, 모든 실행을 선택하려면 헤더 행의 체크박스를 찾습니다.
    3. 체크박스를 선택하여 일괄 작업을 활성화합니다.
    4. 태그를 제거하려는 실행들을 선택합니다.
    5. 실행 목록 상단에 있는 **Tag** 버튼을 선택합니다.
    6. 실행에서 태그를 제거하려면 해당 태그 옆의 체크 표시를 해제합니다.
  </Tab>

  <Tab title="실행 페이지">
    1. 실행 페이지의 왼쪽 사이드바에서 맨 위의 **Overview** 탭을 선택합니다. 여기에서 실행에 설정된 태그를 볼 수 있습니다.
    2. 태그 위에 마우스를 올린 다음 &quot;x&quot;를 클릭하여 실행에서 제거합니다.
  </Tab>
</Tabs>
