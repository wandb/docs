W&amp;B 프로젝트에 로깅된 모든 run은 프로젝트 사이드바의 **Runs** 탭에서 볼 수 있습니다. Runs 탭 안에는 *Runs table*이 표시됩니다. Runs table은 프로젝트의 모든 run에 대한 세부 정보를 보여줍니다. Runs table을 사용하면 run을 비교하고, 특정 컬럼을 기준으로 run을 정렬하며, run을 그룹으로 구성할 수 있습니다.

<div id="manage-columns">
  ## 열 관리
</div>

다음 섹션에서는 Runs 테이블을 사용자 지정하는 방법을 설명합니다.

<div id="add-columns">
  ### 열 추가
</div>

Runs 테이블에 열을 추가하여 프로젝트와 연관된 어떤 속성을 표시할지 사용자 정의할 수 있습니다.

Runs 테이블에 열을 추가하려면:

1. 프로젝트 사이드바에서 **Runs** 탭을 선택합니다.
1. run 목록 위에서 **Columns**(가로 줄 여섯 개) 버튼을 클릭합니다.
1. 모달의 **Hidden** 섹션에서 속성 이름을 선택합니다.
1. 열을 드래그하여 순서를 변경합니다.
1. **Close**를 클릭합니다.

<div id="remove-columns">
  ### 컬럼 제거
</div>

Runs 테이블에서 컬럼을 제거하여 프로젝트와 연관된 어떤 속성을 표시할지 원하는 대로 구성할 수 있습니다.

Runs 테이블에서 컬럼을 제거하려면:

1. 프로젝트 사이드바에서 **Runs** 탭을 선택합니다.
1. run 목록 위에 있는 **Columns**(가로 줄 여섯 개) 버튼을 클릭합니다.
1. 모달 창의 **Visible & Pinned** 섹션에서 속성 이름을 선택합니다.
1. **Close**를 클릭합니다.

<div id="move-columns">
  ### 열 이동
</div>

Runs 테이블에서 열을 이동하려면:

1. 프로젝트 사이드바에서 **Runs** 탭을 선택합니다.
1. 열을 왼쪽이나 오른쪽으로 드래그합니다.

<div id="pin-columns">
  ### 열 고정(Pin columns)
</div>

고정된 열은 Runs 탭의 왼쪽에 표시되고, 고정되지 않은 열은 오른쪽에 표시됩니다. **Runs** 탭에서 열을 고정하면 **Workspace** 탭에서도 고정됩니다. 마찬가지로 **Workspace** 탭에서 열을 고정하면 **Runs** 탭에서도 고정됩니다.

열을 고정하려면:

1. 프로젝트 사이드바에서 **Runs** 탭으로 이동합니다.
1. **Columns**(가로 줄 6개) 버튼을 클릭합니다.
1. 모달의 **Visible & Pinned** 섹션에서, 고정하려는 열 이름 옆에 있는 핀 아이콘을 클릭합니다.

열 고정을 해제하려면:

1. 프로젝트 사이드바에서 **Runs** 탭으로 이동합니다.
1. 열 이름 위에 마우스를 올린 다음 나타나는 작업 `...` 메뉴를 클릭하거나, **Columns**(가로 줄 6개) 버튼을 클릭한 뒤 고정 해제하려는 열 이름 옆의 핀 아이콘을 클릭합니다.
1. **Unpin column**을 클릭합니다.

<Note>
W&B는 Runs 테이블에서 사용자가 고정하거나 고정 해제한 열 설정을 Workspace 탭의 Runs 선택기에서도 그대로 유지합니다.
</Note>

<div id="hide-columns">
  ### 열 숨기기
</div>

Runs 테이블에서 열을 숨기려면:

1. 프로젝트 사이드바에서 **Runs** 탭을 선택합니다.
1. 열 이름 위에 마우스를 올린 다음, 나타나는 작업 메뉴 `...`를 클릭합니다.
1. **Hide column**을 클릭합니다.

현재 숨겨져 있는 모든 열을 보려면 **Columns**를 클릭합니다.

<div id="sort-runs-by-column">
  ## 열별로 run 정렬하기
</div>

Runs 테이블에서 표시된 어떤 열이든 기준으로 run을 정렬할 수 있습니다. 기록된 값 중 가장 좋은(또는 가장 나쁜) 값을 보고 싶을 때 특히 유용합니다.

표시된 열을 기준으로 run 목록을 정렬하려면:

1. 열 이름 위에 마우스를 올린 다음, 작업 `...` 메뉴를 클릭합니다.
1. 필요하다면 **Show latest** 위에 마우스를 올린 뒤, 드롭다운에서 **latest**, **min**, 또는 **max**를 선택합니다.
1. **Sort ascending** 또는 **Sort descending**을 클릭합니다.

다음 애니메이션은 기록된 메트릭의 최대값을 기준으로 run을 정렬하는 방법을 보여줍니다:

</Note>

<div id="export-runs-table-to-csv">
  ## Runs 테이블을 CSV로 내보내기
</div>

다운로드 버튼을 사용해 모든 run, 하이퍼파라미터, 요약 메트릭이 포함된 테이블을 CSV로 내보낼 수 있습니다.

1. 프로젝트 사이드바에서 **Runs** 탭을 선택합니다.
1. run 목록 상단에서 **Download**(아래쪽 화살표) 버튼을 클릭합니다.
