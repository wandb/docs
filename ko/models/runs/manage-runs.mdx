title: 다른 프로젝트 또는 팀으로 run 이동하기
description: W&B App을 사용하여 프로젝트 또는 Teams 간에 Runs 를 이동할 수 있습니다.
---

<Note>
시작하기 전에 Runs 를 Projects 또는 Teams 간에 이동할 수 있는 권한이 있는지 확인하세요. 해당 Run 의 현재 위치와 이동하려는 위치 모두에 대한 엑세스 권한이 있어야 합니다.
</Note>

Runs 를 한 Project 에서 다른 Project 로, 또는 Teams 간에 이동하는 방법은 다음과 같습니다:

1. 이동하려는 Runs 가 포함된 Project 로 이동합니다.
2. Project 사이드바에서 **Runs** 탭을 선택합니다.
3. 이동하려는 Runs 옆의 체크박스를 선택합니다.
4. 테이블 상단의 **Move to project** 버튼을 클릭합니다.
5. 드롭다운에서 대상 Team 과 Project 를 선택합니다.

<Note>
Run 을 이동할 때, 해당 Run 과 연관된 과거의 Artifacts 는 함께 이동되지 않습니다. Artifact 를 수동으로 이동하려면, [`wandb artifact get`](/models/ref/cli/wandb-artifact/wandb-artifact-get/) SDK 코맨드나 [`Api.artifact` API](/models/ref/python/public-api/api/#artifact) 를 사용하여 Artifact 를 다운로드한 후, [`wandb artifact put`](/models/ref/cli/wandb-artifact/wandb-artifact-put/) 또는 `Api.artifact` API 를 사용하여 Run 의 새 위치로 업로드할 수 있습니다.
</Note>
