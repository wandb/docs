title: 실행을 다른 프로젝트나 팀으로 이동하기
description: W&B App을 사용해 실행을 프로젝트 또는 팀 간에 이동합니다.
---

<Note>
  시작하기 전에 프로젝트 또는 팀 간에 실행을 이동할 수 있는 필요한 권한이 있는지 확인하세요. 현재 위치와 새 위치 모두에서 해당 실행에 액세스할 수 있어야 합니다.
</Note>

실행을 한 프로젝트에서 다른 프로젝트로, 또는 팀 간에 이동하려면:

1. 이동하려는 실행이 포함된 프로젝트로 이동합니다.
2. 프로젝트 사이드바에서 **Runs** 탭을 선택합니다.
3. 이동하려는 실행 옆의 체크박스를 선택합니다.
4. 테이블 상단의 **Move to project** 버튼을 클릭합니다.
5. 드롭다운에서 대상 팀과 프로젝트를 선택합니다.

<Note>
  실행을 이동해도 해당 실행과 연결된 기존 아티팩트는 함께 이동되지 않습니다. 아티팩트를 수동으로 이동하려면 [`wandb artifact get`](/ko/models/ref/cli/wandb-artifact/wandb-artifact-get/) SDK 명령어나 [`Api.artifact` API](/ko/models/ref/python/public-api/api/#artifact)를 사용해 아티팩트를 다운로드한 다음, [`wandb artifact put`](/ko/models/ref/cli/wandb-artifact/wandb-artifact-put/) 또는 `Api.artifact` API를 사용해 실행의 새 위치로 업로드하세요.
</Note>

  5. 모달 창에서 대상 그룹을 선택하거나 새 그룹을 생성합니다.
  6. **Move**를 클릭합니다. */}
