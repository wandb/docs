title: 실행 색상 사용자 지정
---

W&B는 프로젝트에서 생성하는 각 실행에 자동으로 색상을 할당합니다. 실행의 기본 색상을 변경하여 테이블과 그래프에서 다른 실행과 시각적으로 구분할 수 있습니다. 프로젝트 워크스페이스를 재설정하면 테이블에 있는 모든 실행의 색상이 기본값으로 복원됩니다.

실행 색상은 로컬 범위로 적용됩니다. 프로젝트 페이지에서 사용자 지정 색상은 본인의 워크스페이스에만 적용됩니다. 리포트에서는 실행에 대한 사용자 지정 색상이 섹션 수준에만 적용됩니다. 동일한 실행을 서로 다른 섹션에서 시각화할 수 있으며, 섹션마다 서로 다른 사용자 지정 색상을 사용할 수 있습니다.

<div id="edit-default-run-colors">
  ## 기본 실행 색상 편집
</div>

1. 프로젝트 사이드바에서 **Runs** 탭을 클릭합니다.
2. **Name** 열에서 실행 이름 옆에 있는 색상 점 아이콘을 클릭합니다.
3. 색상 팔레트나 색상 선택기에서 색상을 선택하거나, Hex 색상 코드를 입력합니다.

<div id="randomize-run-colors">
  ## 실행 색상 무작위화
</div>

테이블에 있는 모든 실행의 색상을 무작위로 변경하려면:

1. 프로젝트 사이드바에서 **Runs** 탭을 클릭합니다.
2. **Name** 열 헤더 위에 마우스를 올리고, 가로 점 세 개(**...**)를 클릭한 다음 드롭다운 메뉴에서 **Randomize run colors**를 선택합니다.

<Note>
실행 색상 무작위화 옵션은 정렬, 필터링, 검색, 그룹화 등과 같이 실행 테이블을 어떤 방식으로든 변경한 이후에만 사용할 수 있습니다.
</Note>

<div id="reset-run-colors">
  ## 실행 색상 재설정
</div>

{/* 테이블이나 선택기에서 실행이 최소 두 개 이상 있고, 보기 설정(정렬, 필터링, 검색 또는 그룹화)을 어떤 방식으로든 변경한 경우에만 실행 색상을 무작위로 설정하는 옵션을 사용할 수 있습니다. */}

테이블에서 모든 실행의 색을 기본값으로 되돌리려면:

1. 프로젝트 사이드바에서 **Runs** 탭을 클릭합니다.
2. **Name** 열 헤더 위에 마우스를 올리고 가로 점 세 개(**...**)를 클릭한 다음, 드롭다운 메뉴에서 **Reset colors**를 선택합니다.
