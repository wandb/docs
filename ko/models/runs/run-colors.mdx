title: run 색상 사용자 지정
---

W&amp;B는 프로젝트에서 생성하는 각 run에 색상을 자동으로 할당합니다. 테이블과 그래프에서 다른 run과 시각적으로 쉽게 구분할 수 있도록 run의 기본 색상을 변경할 수 있습니다. 프로젝트 워크스페이스를 재설정하면 테이블의 모든 run 색상이 기본값으로 복원됩니다.

run 색상은 로컬 범위에서 적용됩니다. 프로젝트 페이지에서는 사용자 지정 색상이 본인 워크스페이스에만 적용됩니다. Reports에서는 run의 사용자 지정 색상이 섹션 수준에서만 적용됩니다. 동일한 run을 여러 섹션에서 시각화할 수 있으며, 각 섹션마다 서로 다른 사용자 지정 색상을 사용할 수 있습니다.

<div id="edit-default-run-colors">
  ## 기본 run 색상 편집
</div>

1. 프로젝트 사이드바에서 **Runs** 탭을 클릭합니다.
2. **Name** 열에서 run 이름 옆의 색상 점을 클릭합니다.
3. 색상 팔레트나 색상 선택기에서 색상을 선택하거나 16진수 코드를 입력합니다.

<Image src="/images/runs/run-color-palette.png" alt="프로젝트 워크스페이스에서 기본 run 색상 편집" />

<div id="randomize-run-colors">
  ## run 색상 무작위로 지정
</div>

테이블의 모든 run 색상을 무작위로 지정하려면 다음 단계를 따르세요.

1. 프로젝트 사이드바에서 **Runs** 탭을 클릭합니다.
2. **Name** 열 헤더 위에 마우스를 올리고 **액션 (<Icon icon="ellipsis" iconType="solid" />)** 메뉴를 클릭한 다음, 드롭다운 메뉴에서 **Randomize run colors**를 선택합니다.

<Note>
  정렬, 필터링, 검색 또는 그룹화와 같이 run 테이블을 어떤 방식으로든 수정한 후에만 run 색상을 무작위로 지정하는 옵션을 사용할 수 있습니다.
</Note>

<div id="reset-run-colors">
  ## run 색상 재설정
</div>

{/* run 색상을 무작위로 설정하는 옵션은 테이블이나 선택기에 run이 최소 2개 이상 있고, 뷰를 어떤 식으로든 변경한 경우(정렬, 필터링, 검색 또는 그룹화)에만 사용할 수 있습니다. */}

테이블의 모든 run에 기본 색상을 복원하려면 다음과 같이 하세요.

1. 프로젝트 사이드바에서 **Runs** 탭을 클릭합니다.
2. **Name** 열 헤더에 마우스를 올리고 **액션 (<Icon icon="ellipsis" iconType="solid" />)** 메뉴를 클릭한 다음, 드롭다운 메뉴에서 **Reset colors**를 선택합니다.
