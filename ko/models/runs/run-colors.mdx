---
title: Run 색상 사용자 지정하기
---

W&B는 프로젝트에서 생성하는 각 Run 에 색상을 자동으로 할당합니다. 테이블과 그래프에서 다른 Run 들과 시각적으로 구분하기 위해 Run 의 기본 색상을 변경할 수 있습니다. 테이블에 있는 모든 Run 의 기본 색상을 복원하려면 프로젝트 Workspace 를 리셋하세요.

Run 색상은 로컬 범위로 적용됩니다. 프로젝트 페이지에서 설정한 커스텀 색상은 본인의 Workspace 에만 적용됩니다. Reports 에서는 Run 의 커스텀 색상이 섹션 단위로만 적용됩니다. 동일한 Run 이라도 섹션마다 서로 다른 커스텀 색상을 사용하여 시각화할 수 있습니다.

## 기본 Run 색상 편집하기

1. 프로젝트 사이드바에서 **Runs** 탭을 클릭합니다.
2. **Name** 컬럼의 Run 이름 옆에 있는 색상 점을 클릭합니다.
3. 색상 팔레트나 컬러 피커에서 색상을 선택하거나, 헥스(hex) 코드를 입력합니다.

<Image src="/images/runs/run-color-palette.png" alt="Edit default run color in project workspace" />

## Run 색상 무작위로 변경하기

테이블에 있는 모든 Run 의 색상을 무작위로 변경하려면 다음 단계를 따르세요:

1. 프로젝트 사이드바에서 **Runs** 탭을 클릭합니다.
2. **Name** 컬럼 헤더에 마우스를 올리고, 점 세 개 아이콘 (**...**)을 클릭한 뒤 드롭다운 메뉴에서 **Randomize run colors**를 선택합니다.

<Note>
Run 색상을 무작위로 변경하는 옵션은 정렬, 필터링, 검색 또는 그룹화와 같이 Run 테이블을 어떤 방식으로든 수정한 후에만 사용할 수 있습니다.
</Note>

## Run 색상 초기화하기

테이블에 있는 모든 Run 의 기본 색상을 복원하려면 다음 단계를 따르세요:

1. 프로젝트 사이드바에서 **Runs** 탭을 클릭합니다.
2. **Name** 컬럼 헤더에 마우스를 올리고, 점 세 개 아이콘 (**...**)을 클릭한 뒤 드롭다운 메뉴에서 **Reset colors**를 선택합니다.
