title: run 색상 사용자 지정
---

W&B는 프로젝트에서 생성하는 각 run에 자동으로 색상을 할당합니다. 테이블과 그래프에서 다른 run과 시각적으로 구분하기 쉽도록 run의 기본 색상을 변경할 수 있습니다. 테이블에 표시된 모든 run의 기본 색상을 복원하려면 프로젝트 워크스페이스를 재설정하세요.

run 색상은 로컬에만 적용됩니다. 프로젝트 페이지에서는 사용자 지정 색상이 본인의 워크스페이스에만 적용됩니다. 리포트에서는 run에 대한 사용자 지정 색상이 섹션 수준에만 적용됩니다. 동일한 run을 서로 다른 섹션에서 시각화할 수 있으며, 섹션마다 서로 다른 사용자 지정 색상을 사용할 수 있습니다.

<div id="edit-default-run-colors">
  ## 기본 run 색상 편집
</div>

1. 프로젝트 사이드바에서 **Runs** 탭을 클릭합니다.
2. **Name** 열에서 run 이름 옆의 점 아이콘 색상을 클릭합니다.
3. 색상 팔레트나 색상 선택기에서 색상을 선택하거나, 16진수(hex) 코드를 입력합니다.

<Image src="/images/runs/run-color-palette.png" alt="프로젝트 워크스페이스에서 기본 run 색상 편집" />

<div id="randomize-run-colors">
  ## run 색상 무작위화
</div>

테이블에 있는 모든 run의 색상을 무작위로 설정하려면:

1. 프로젝트 사이드바에서 **Runs** 탭을 클릭합니다.
2. **Name** 열 헤더 위에 마우스를 올리고, 가로 점 세 개(**...**)를 클릭한 후 드롭다운 메뉴에서 **Randomize run colors**를 선택합니다.

<Note>
run 색상 무작위화 옵션은 정렬, 필터링, 검색, 그룹화 등과 같이 run 테이블을 어떤 방식으로든 수정한 후에만 사용할 수 있습니다.
</Note>

<div id="reset-run-colors">
  ## run 색상 초기화
</div>

{/* run 색상을 무작위로 변경하는 옵션은 테이블 또는 선택기에 run이 최소 두 개 이상 있고, 정렬, 필터링, 검색 또는 그룹화와 같이 뷰를 일부 수정한 경우에만 사용할 수 있습니다. */}

테이블의 모든 run 색상을 기본값으로 되돌리려면:

1. 프로젝트 사이드바에서 **Runs** 탭을 클릭합니다.
2. **Name** 열 헤더 위에 마우스를 올린 다음, 가로 점 세 개(**...**)를 클릭하고 드롭다운 메뉴에서 **Reset colors**를 선택합니다.
