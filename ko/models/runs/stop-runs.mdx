---

W&amp;B Python SDK를 사용해 코드로 실행을 중지하거나 W&amp;B App에서 직접 실행을 중지할 수 있습니다.

<Tabs>
  <Tab title="Python SDK">
    1. 실행을 초기화한 터미널 또는 코드 에디터로 이동합니다.
    2. `Ctrl+D`를 눌러 실행을 중지합니다.
  </Tab>

  <Tab title="W&B App">
    1. 해당 실행이 로깅되고 있는 프로젝트로 이동합니다.
    2. 실행 선택기에서 중지하려는 실행을 선택합니다.
    3. **Overview** 탭을 선택합니다.
    4. **State** 필드 옆의 중지 버튼을 선택합니다.

    **State** 필드 옆에서 실행의 상태가 `running`에서 `Killed`로 변경됩니다.
  </Tab>
</Tabs>

가능한 실행 상태 전체 목록은 [State fields](/ko/models/runs/run-states)를 참고하세요.
