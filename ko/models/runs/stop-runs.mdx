---
title: Run 중지하기
description: W&B Python SDK를 사용하여 프로그래밍 방식으로 또는 W&B App에서 수동으로 run 을 중지합니다.
---

W&B Python SDK를 사용하여 프로그래밍 방식으로 또는 W&B App에서 대화형으로 run을 중지합니다.

<Tabs>
<Tab title="Python SDK">
1. run을 초기화한 터미널 또는 코드 에디터로 이동합니다.
2. `Ctrl+D`를 눌러 run을 중지합니다.

</Tab>
<Tab title="W&B App">
1. run이 로그를 기록 중인 프로젝트로 이동합니다.
2. run 선택기에서 중지하려는 run을 선택합니다.
3. **Overview** 탭을 선택합니다.
4. **State** 필드 옆의 중지 버튼을 선택합니다.

**State** 필드 옆에서 run의 상태가 `running`에서 `Killed`로 변경됩니다.

</Tab>
</Tabs>

가능한 전체 run 상태 목록은 [State fields](/models/runs/run-states)를 참조하세요.
