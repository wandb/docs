---
title: CoreWeave 인프라 알림 시각화
description: "W&B 실험 run 플롯에서 GPU 장애 및 열 한계 위반과 같은 CoreWeave 인프라 알림을 확인하세요."
---

W&amp;B에 기록하는 머신 러닝 실험 중에 GPU 장애, 열 한계 위반 등과 같은 인프라 알림을 관찰하세요. [W&amp;B run](/ko/models/runs) 중에는 [CoreWeave Mission Control](https://coreweave.com/mission-control)이 컴퓨트 인프라를 모니터링합니다.

<Note>
  이 기능은 Preview 상태이며 CoreWeave cluster에서 트레이닝할 때만 사용 가능합니다. 액세스하려면 W&amp;B 담당자에게 문의하세요.
</Note>

오류가 발생하면 CoreWeave가 해당 정보를 W&amp;B로 보냅니다. W&amp;B는 프로젝트 Workspace의 run 플롯에 인프라 정보를 채웁니다. CoreWeave는 일부 문제를 자동으로 해결하려고 시도하며, W&amp;B는 해당 정보를 run 페이지에 표시합니다.
