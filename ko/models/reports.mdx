W&amp;B Reports를 사용하면 다음을 수행할 수 있습니다:

* 실행을 체계적으로 구성합니다.
* 시각화를 임베드하고 자동화합니다.
* 분석 결과를 설명합니다.
* LaTeX zip 파일이나 PDF 형식으로 공동 작업자와 업데이트를 공유합니다.

<div id="how-it-works">
  ## 작동 방식
</div>

몇 번만 클릭하면 공동 작업용 리포트를 만들 수 있습니다.

1. W&amp;B App에서 W&amp;B 프로젝트 워크스페이스로 이동합니다.
2. 워크스페이스 오른쪽 상단에서 **Create report** 버튼을 클릭합니다.

<Frame>
  <img src="/images/reports/create_a_report_button.png" alt="Create report button" max-width="90%" />
</Frame>

3. **Create Report**라는 제목의 모달 창이 나타납니다. 리포트에 추가할 차트와 패널을 선택합니다. (차트와 패널은 나중에 추가하거나 제거할 수 있습니다.)
4. **Create report**를 클릭합니다.
5. 리포트를 원하는 형태로 편집합니다.
6. **Publish to project**를 클릭합니다.
7. **Share** 버튼을 클릭해 리포트를 협업자와 공유합니다.

W&amp;B Python SDK를 사용해 리포트를 대화형 방식이나 프로그래밍 방식으로 생성하는 방법에 대한 자세한 내용은 [Create a report](/ko/models/reports/create-a-report/) 페이지를 참고하세요.
