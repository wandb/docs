<div id="initialize-a-run-and-track-hyperparameters">
  ## run을 초기화하고 하이퍼파라미터를 추적하세요
</div>

Python script or notebook에서 [`wandb.init()`](/ko/models/ref/python/experiments/run/)을 사용해 W&amp;B run 객체를 초기화하세요. `config` 파라미터에는 딕셔너리를 사용하여
하이퍼파라미터 이름과 값을 지정하세요. `with` 문 안에서는 메트릭 및 기타 정보를 W&amp;B에 로깅할 수 있습니다.

```python
import wandb

wandb.login()

# run이 기록되는 프로젝트
project = "my-awesome-project"

# 하이퍼파라미터가 포함된 딕셔너리
config = {
    'epochs' : 10,
    'lr' : 0.01
}

with wandb.init(project=project, config=config) as run:
    # 트레이닝 코드를 여기에 작성하세요
    # run.log()를 사용하여 W&B에 값을 로깅합니다
    run.log({"accuracy": 0.9, "loss": 0.1})
```

완전한 예제는 다음 섹션을 참조하세요. 이 예제에서는 트레이닝 run을 시뮬레이션하고 정확도 및 loss 메트릭을 W&amp;B에 로깅합니다.

<Info>
  [run](/ko/models/runs/)은 W&amp;B의 핵심 요소입니다. runs를 사용해 [메트릭을 추적](/ko/models/track/)하고, [로깅하고](/ko/models/track/log/), 아티팩트를 추적하는 등 다양한 작업을 수행할 수 있습니다.
</Info>

<div id="create-a-machine-learning-training-experiment">
  ## 머신 러닝 트레이닝 실험 만들기
</div>

이 모의 트레이닝 스크립트는 시뮬레이션된 정확도와 loss 메트릭을 W&amp;B에 로깅합니다. 다음 코드를 복사하여 Python script or notebook의 notebook cell에 붙여 넣고 실행하세요:

```python
import wandb
import random

wandb.login()

# Run이 기록되는 프로젝트
project = "my-awesome-project"

# 하이퍼파라미터가 포함된 딕셔너리
config = {
    'epochs' : 10,
    'lr' : 0.01
}

with wandb.init(project=project, config=config) as run:
    offset = random.random() / 5
    print(f"lr: {config['lr']}")
    
    # 트레이닝 run 시뮬레이션
    for epoch in range(2, config['epochs']):
        acc = 1 - 2**-config['epochs'] - random.random() / config['epochs'] - offset
        loss = 2**-config['epochs'] + random.random() / config['epochs'] + offset
        print(f"epoch={config['epochs']}, accuracy={acc}, loss={loss}")
        run.log({"accuracy": acc, "loss": loss})
```

기록된 메트릭(예: 정확도와 loss)과 각 트레이닝 step 동안 어떻게 변경되었는지 보려면 [wandb.ai/home](https://wandb.ai/home)에 방문하세요. 다음 이미지는 각 run에서 추적된 loss와 정확도를 보여줍니다. 각 run 객체는 생성된 이름과 함께 **Runs** column에 표시됩니다.
