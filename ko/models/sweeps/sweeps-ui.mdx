---
title: Sweeps UI
description: Sweeps UI 의 다양한 구성 요소에 대해 설명합니다.
---

Sweeps UI에는 상태 (**State**), 생성 시간 (**Created**), Sweeps를 시작한 엔티티 (**Creator**), 완료된 Runs의 수 (**Run count**), 그리고 Sweeps 계산에 소요된 시간 (**Compute time**)이 표시됩니다. 이산 탐색 공간(discrete search space)에 대해 그리드 검색을 수행할 때, Sweeps가 생성할 것으로 예상되는 Runs의 수 (**Est. Runs**)가 제공됩니다. 또한 인터페이스에서 특정 Sweeps를 클릭하여 일시 중지, 재개, 중지 또는 삭제(kill)할 수 있습니다.
