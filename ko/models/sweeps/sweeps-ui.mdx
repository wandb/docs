description: Sweeps UI의 다양한 구성 요소를 설명합니다.
title: Sweeps UI
---

Sweeps UI에는 상태(**State**), 생성 시간(**Created**), 스윕을 시작한 엔터티(**Creator**), 완료된 run 수(**Run count**), 그리고 스윕을 계산하는 데 걸린 시간(**Compute time**)이 표시됩니다. 스윕이 생성할 것으로 예상되는 run 수(**Est. Runs**)는 이산 탐색 공간에서 그리드 서치를 수행할 때 제공됩니다. 인터페이스에서 스윕을 클릭하여 일시 중지, 재개, 중지 또는 강제 종료할 수도 있습니다.
