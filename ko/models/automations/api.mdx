<Note>
  `wandb.Api().create_automation()` 및 관련 헬퍼를 통한 자동화의 프로그래밍 방식 **생성** 및 **업데이트**는 클라이언트 측 기능 검사(예: run 상태 자동화) 때문에 최신 `wandb` Python 클라이언트에서 실패할 수 있습니다. 수정된 SDK가 배포될 때까지는 [W&amp;B App](/ko/models/automations/create-automations)에서 자동화를 생성하고 편집하세요. 사용 중인 버전에서는 목록, 조회, 삭제가 계속 작동할 수도 있지만, 오류가 발생하면 App을 사용하세요. 내부 추적: WB-34263.
</Note>
