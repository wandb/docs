<div id="install-and-import-packages">
  ### 패키지 설치 및 임포트
</div>

W&amp;B Report와 Workspaces 라이브러리를 설치합니다.

```python
pip install wandb-workspaces
```

<div id="create-a-report">
  ### 리포트 만들기
</div>

리포트를 만들려면 팀의 엔터티를 지정하고 리포트 이름을 입력하세요. 꺾쇠 괄호(&lt; &gt;) 안의 텍스트를 실제 값으로 바꾸세요:

```python
import wandb_workspaces.reports.v2 as wr 
# 생성
report = wr.Report(
    entity="<team_entity>",
    project="<project_name>",
    title='Quickstart Report',
    description="That was easy!"
)

# 보고서 저장
report.save()
```

이제 보고서에 블록과 패널을 추가하세요. 예를 들어, 다음 코드는 목차, 헤더, 문단이 포함된 보고서를 만듭니다:

```python
report.blocks = [
    wr.TableOfContents(),
    wr.H1("Text and images example"),
    wr.P("Lorem ipsum dolor sit amet."),
]
report.save()
```

엔드 투 엔드 예시는 [Reports API Quickstart](https://colab.research.google.com/github/wandb/examples/blob/master/colabs/intro/Report_API_Quickstart.ipynb) Google Colab 노트북을 참고하세요.

<div id="create-a-workspace">
  ### 워크스페이스 생성
</div>

다음 코드는 선 그래프, 막대 그래프, 스칼라 차트로 구성된 세 개의 패널을 포함한 섹션이 있는 워크스페이스를 생성하는 방법을 보여줍니다. 꺾쇠 괄호 안의 텍스트를 여러분의 값으로 바꾸세요:

```python
# 가져오기 방법
import wandb_workspaces.workspaces as ws

# 워크스페이스 생성
ws.Workspace(
     entity="<team_entity>", # 워크스페이스를 소유하는 엔터티
     project="<project_name>", # 워크스페이스와 연결된 프로젝트
     sections=[
         ws.Section(
             name="<Validation Metrics>",
             panels=[
                 wr.LinePlot(x="Step", y=["<val_loss>"]),
                 wr.BarPlot(metrics=["<val_accuracy>"]),
                 wr.ScalarChart(metric="<f1_score>", groupby_aggfunc="<mean>"),
             ],
             is_open=True,
         ),
     ],
)
workspace.save()
```

전체 과정을 다루는 예제는 [Workspace API Quickstart](https://colab.research.google.com/github/wandb/wandb-workspaces/blob/Update-wandb-workspaces-tuturial/Workspace_tutorial.ipynb#scrollTo=MmxL0wjvrNtQ) Google Colab을 참고하세요.
