`wandb`를 통해 엑세스할 수 있는 W&B Python SDK를 사용하면 모델을 트레이닝 및 파인튜닝하고, 실험 단계부터 프로덕션 단계까지 Models 를 관리할 수 있습니다.

> 이 SDK를 사용하여 트레이닝 및 파인튜닝 작업을 수행한 후에는, [Public API](/models/ref/python/public-api)를 사용하여 로그된 데이터를 쿼리하고 분석할 수 있으며, [Reports 및 Workspaces API](/models/ref/wandb_workspaces)를 사용하여 작업 결과를 요약하고 웹에 게시 가능한 [report](/models/reports/)를 생성할 수 있습니다.

## 설치 및 설정

### 가입 및 API 키 생성

W&B에 머신을 인증하려면 먼저 [사용자 설정(User Settings)](https://wandb.ai/settings)에서 API 키를 생성해야 합니다.

### 패키지 설치 및 임포트

W&B 라이브러리를 설치합니다.

```
pip install wandb
```

### W&B Python SDK 임포트:

```python
import wandb

# 팀 entity를 지정합니다
entity = "<team_entity>"

# run이 기록될 Project를 지정합니다
project = "my-awesome-project"

with wandb.init(entity=entity, project=project) as run:
   run.log({"accuracy": 0.9, "loss": 0.1})
```
