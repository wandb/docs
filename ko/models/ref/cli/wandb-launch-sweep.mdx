W&B Launch Sweep 을 실행합니다 (Experimental).

## Usage

```bash
wandb launch-sweep [CONFIG] [OPTIONS]
```

## Arguments

| Argument | Description | Required |
| :--- | :--- | :--- |
| `CONFIG` | 설명 없음 | No |

## Options

| Option | Description |
| :--- | :--- |
| `--queue`, `-q` | Sweep 을 푸시할 큐의 이름 |
| `--project`, `-p` | 에이전트가 모니터링할 Project 이름. 입력 시, 설정 파일을 통해 전달된 프로젝트 값을 재정의합니다. |
| `--entity`, `-e` | 사용할 Entity. 기본값은 현재 로그인된 사용자입니다. |
| `--resume_id`, `-r` | 8자리의 Sweep ID를 전달하여 Launch Sweep 을 재개합니다. 큐 지정이 필요합니다. |
| `--prior_run`, `-R` | 이 Sweep 에 추가할 기존 Run 의 ID |
