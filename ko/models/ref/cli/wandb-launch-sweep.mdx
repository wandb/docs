W&amp;B Launch sweep을 실행합니다 (실험 기능).


<div id="usage">
  ## 사용
</div>

```bash
wandb launch-sweep [CONFIG] [OPTIONS]
```


<div id="arguments">
  ## 인수
</div>

| 인수       | 설명           | 필수  |
| :------- | :----------- | :-- |
| `CONFIG` | 제공된 설명이 없습니다 | 아니요 |

<div id="options">
  ## 옵션
</div>

| Option              | 설명                                                    |
| :------------------ | :---------------------------------------------------- |
| `--queue`, `-q`     | sweep를 보낼 큐 이름                                        |
| `--project`, `-p`   | agent가 감시할 프로젝트 이름입니다. 지정하면 설정 파일로 전달된 프로젝트 값을 재정의합니다 |
| `--entity`, `-e`    | 사용할 entity입니다. 기본값은 현재 로그인한 사용자입니다                    |
| `--resume_id`, `-r` | 8자 sweep ID를 전달해 Launch sweep를 재개합니다. 큐가 필요합니다        |
| `--prior_run`, `-R` | 이 sweep에 추가할 기존 run의 ID                               |
