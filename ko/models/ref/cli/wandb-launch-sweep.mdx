<div id="usage">
  ## 사용
</div>

```bash
launch-sweep [OPTIONS] [CONFIG]
```

<div id="description">
  ## 설명
</div>

W&amp;B Launch 스윕을 실행합니다 (실험 기능).

<div id="arguments">
  ## 인수
</div>

| 이름       | 기본값  | 유형    |
| -------- | ---- | ----- |
| `config` | PATH | False |

<div id="options">
  ## 옵션
</div>

| 플래그               | 유형  | 설명                                                                         |
| ----------------- | --- | -------------------------------------------------------------------------- |
| `--queue, -q`     | STR | 스윕을 푸시할 큐 이름입니다. **기본값**: None                                             |
| `--project, -p`   | STR | 에이전트가 모니터링할 프로젝트 이름입니다. 이 값을 지정하면 설정 파일로 전달한 프로젝트 값을 재정의합니다. **기본값**: None |
| `--entity, -e`    | STR | 사용할 entity입니다. 기본값은 현재 로그인한 사용자입니다. **기본값**: None                          |
| `--resume_id, -r` | STR | 8자 스윕 ID를 전달해 Launch 스윕을 재개합니다. 큐가 필요합니다. **기본값**: None                    |
| `--prior_run, -R` | STR | 이 스윕에 추가할 기존 run의 ID입니다. **기본값**: None                                     |
