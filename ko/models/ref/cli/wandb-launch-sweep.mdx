W&amp;B Launch 스윕을 실행합니다 (실험적 기능).


<div id="usage">
  ## 사용 방법
</div>

```bash
wandb launch-sweep [CONFIG] [OPTIONS]
```


<div id="arguments">
  ## 인자
</div>

| 인자 | 설명 | 필수 여부 |
| :--- | :--- | :--- |
| `CONFIG` | 설명이 제공되지 않습니다 | 아니요 |

<div id="options">
  ## 옵션
</div>

| Option | Description |
| :--- | :--- |
| `--queue`, `-q` | 스윕을 푸시할 큐 이름 |
| `--project`, `-p` | 에이전트가 모니터링할 프로젝트 이름. 이 값을 전달하면 설정 파일로 전달된 프로젝트 값을 덮어쓴다 |
| `--entity`, `-e` | 사용할 entity. 기본값은 현재 로그인된 사용자 |
| `--resume_id`, `-r` | 8자 스윕 ID를 전달하여 Launch 스윕을 재개. 큐 필수 |
| `--prior_run`, `-R` | 이 스윕에 추가할 기존 run ID |
