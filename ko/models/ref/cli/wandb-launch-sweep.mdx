W&amp;B Launch 스위프를 실행합니다(실험적 기능).


<div id="usage">
  ## 사용 방법
</div>

```bash
wandb launch-sweep [CONFIG] [OPTIONS]
```


<div id="arguments">
  ## 인자
</div>

| 인자 | 설명 | 필수 여부 |
| :--- | :--- | :--- |
| `CONFIG` | 설명이 제공되지 않습니다 | 아니요 |

<div id="options">
  ## 옵션
</div>

| 옵션 | 설명 |
| :--- | :--- |
| `--queue`, `-q` | 스윕을 보낼 큐의 이름 |
| `--project`, `-p` | 에이전트가 모니터링할 프로젝트 이름. 이 값을 전달하면 구성 파일로 지정된 프로젝트 값을 덮어씁니다 |
| `--entity`, `-e` | 사용할 엔터티. 기본값은 현재 로그인한 사용자입니다 |
| `--resume_id`, `-r` | 8자 스윕 ID를 전달하여 Launch 스윕을 재개합니다. 큐가 필요합니다 |
| `--prior_run`, `-R` | 이 스윕에 추가할 기존 실행의 ID |
