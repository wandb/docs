<div id="usage">
  ## 사용
</div>

```bash
wandb sync [PATH] [OPTIONS]
```


<div id="arguments">
  ## 인수
</div>

| 인수     | 설명    | 필수  |
| :----- | :---- | :-- |
| `PATH` | 설명 없음 | 아니요 |

<div id="options">
  ## 옵션
</div>

| 옵션                   | 설명                                                                          |
| :------------------- | :-------------------------------------------------------------------------- |
| `--id`               | 업로드할 run입니다.                                                                |
| `--project`, `-p`    | 업로드할 프로젝트입니다.                                                               |
| `--entity`, `-e`     | 범위를 지정할 entity입니다.                                                          |
| `--job_type`         | 관련 run을 함께 그룹화하기 위한 run의 유형을 지정합니다.                                         |
| `--sync-tensorboard` | tfevent 파일을 wandb로 스트리밍합니다.                                                 |
| `--include-globs`    | 포함할 glob의 쉼표로 구분된 목록입니다.                                                    |
| `--exclude-globs`    | 제외할 glob의 쉼표로 구분된 목록입니다.                                                    |
| `--include-online`   | 온라인 run을 포함합니다                                                              |
| `--include-offline`  | 오프라인 run을 포함합니다                                                             |
| `--include-synced`   | 동기화된 run을 포함합니다                                                             |
| `--mark-synced`      | run을 동기화됨으로 표시합니다(기본값: True)                                                |
| `--sync-all`         | 모든 run을 동기화합니다(기본값: False)                                                  |
| `--clean`            | 동기화된 run을 삭제합니다(기본값: False)                                                 |
| `--clean-old-hours`  | 지정한 시간(시간 단위) 이전에 생성된 run을 삭제합니다. `--clean` 플래그와 함께 사용합니다. (기본값: 24)        |
| `--clean-force`      | 확인 프롬프트 없이 삭제합니다. (기본값: False)                                              |
| `--show`             | 표시할 run 수입니다(기본값: 5)                                                        |
| `--append`           | run을 추가합니다(기본값: False)                                                      |
| `--skip-console`     | 콘솔 로그를 건너뜁니다(기본값: False)                                                    |
| `--replace-tags`     | &#39;old&#95;tag1=new&#95;tag1,old&#95;tag2=new&#95;tag2&#39; 형식으로 태그를 바꿉니다 |
