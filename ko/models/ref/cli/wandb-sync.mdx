<div id="usage">
  ## 사용
</div>

```bash
wandb sync [PATH] [OPTIONS]
```


<div id="arguments">
  ## 인수
</div>

| 인수     | 설명            | 필수 여부 |
| :----- | :------------ | :---- |
| `PATH` | 설명이 제공되지 않습니다 | 아니요   |

<div id="options">
  ## 옵션
</div>

| 옵션                   | 설명                                                                       |
| :------------------- | :----------------------------------------------------------------------- |
| `--id`               | 업로드할 run.                                                                |
| `--project`, `-p`    | 업로드할 프로젝트.                                                               |
| `--entity`, `-e`     | 범위를 지정할 entity.                                                          |
| `--job_type`         | 관련 run들을 함께 그룹화하기 위한 run 유형.                                             |
| `--sync-tensorboard` | tfevent 파일을 wandb로 스트림.                                                  |
| `--include-globs`    | 포함할 glob 패턴의 쉼표로 구분된 목록.                                                 |
| `--exclude-globs`    | 제외할 glob 패턴의 쉼표로 구분된 목록.                                                 |
| `--include-online`   | 온라인 run 포함                                                               |
| `--include-offline`  | 오프라인 run 포함                                                              |
| `--include-synced`   | 동기화된 run 포함                                                              |
| `--mark-synced`      | run을 동기화된 것으로 표시 (기본값: True)                                             |
| `--sync-all`         | 모든 run 동기화 (기본값: False)                                                  |
| `--clean`            | 동기화된 run 삭제 (기본값: False)                                                 |
| `--clean-old-hours`  | 이 시간(시간 단위) 이전에 생성된 run 삭제. --clean 플래그와 함께 사용. (기본값: 24)                |
| `--clean-force`      | 확인 프롬프트 없이 정리. (기본값: False)                                              |
| `--show`             | 표시할 run 수 (기본값: 5)                                                       |
| `--append`           | run 추가 (기본값: False)                                                      |
| `--skip-console`     | 콘솔 로그 건너뛰기 (기본값: False)                                                  |
| `--replace-tags`     | &#39;old&#95;tag1=new&#95;tag1,old&#95;tag2=new&#95;tag2&#39; 형식으로 태그 교체 |
