<div id="description">
  ## 설명
</div>

`wandb disabled`로 비활성화한 W&amp;B를 다시 활성화합니다.

모드를 `online`으로 설정하면 클라우드 동기화와 아티팩트 저장소를 포함한
W&amp;B의 전체 기능이 복원됩니다.

W&amp;B가 `disabled`로 설정되어 있는 동안 로깅된 데이터는 업로드되지 않지만,
새 데이터는 로깅하고 동기화할 수 있습니다.

W&amp;B를 완전히 비활성화하지 않고 online 모드와 offline 모드 사이를 전환하려면,
대신 `wandb online` 또는 `wandb offline`을 사용하세요.

예를 들어, `wandb disabled`로 비활성화한 뒤 W&amp;B 기능을 복원하려면 다음과 같이 하세요:

```bash
wandb enabled
```

W&amp;B 로깅 및 동기화가 다시 활성화된 상태로 트레이닝 스크립트를 실행하려면:

```bash
python train.py # W&B에 데이터를 로그하고 동기화합니다
```
