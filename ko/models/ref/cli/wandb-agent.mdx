W&B 에이전트를 실행합니다.

## 사용법

```bash
wandb agent SWEEP_ID [OPTIONS]
```

## 인수

| 인수 | 설명 | 필수 여부 |
| :--- | :--- | :--- |
| `SWEEP_ID` | 설명 없음 | 예 |

## 옵션

| 옵션 | 설명 |
| :--- | :--- |
| `--project`, `-p` | Sweeps 에서 생성된 W&B Runs 가 전송될 Projects 의 이름입니다. Projects 를 지정하지 않으면 Run 은 'Uncategorized'로 표시된 프로젝트로 전송됩니다. |
| `--entity`, `-e` | Sweeps 에 의해 생성된 W&B Runs 를 전송할 Entities (사용자 이름 또는 팀 이름)입니다. 지정하는 Entity 가 이미 존재하는지 확인하세요. Entity 를 지정하지 않으면 Run 은 대개 사용자 이름인 기본 Entity 로 전송됩니다. |
| `--count` | 이 에이전트가 실행할 최대 Runs 수입니다. |
| `--forward-signals`, `-f` | 에이전트에 전달된 신호(예: SIGINT/SIGTERM)를 자식 Runs 에 전달하여 정상적으로 종료될 수 있도록 합니다. (기본값: False) |
