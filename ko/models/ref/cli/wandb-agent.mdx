W&amp;B agent 실행하기


<div id="usage">
  ## 사용
</div>

```bash
wandb agent SWEEP_ID [OPTIONS]
```


<div id="arguments">
  ## 인수
</div>

| 인수         | 설명    | 필수 |
| :--------- | :---- | :- |
| `SWEEP_ID` | 설명 없음 | 예  |

<div id="options">
  ## 옵션
</div>

| 옵션                        | 설명                                                                                                                                |
| :------------------------ | :-------------------------------------------------------------------------------------------------------------------------------- |
| `--project`, `-p`         | sweep에서 생성된 W&amp;B run이 전송될 프로젝트 이름입니다. 프로젝트를 지정하지 않으면 run은 &#39;Uncategorized&#39;로 표시된 프로젝트로 전송됩니다.                            |
| `--entity`, `-e`          | sweep에서 생성된 W&amp;B run을 전송할 사용자 이름 또는 팀 이름입니다. 지정한 entity가 이미 존재하는지 확인하세요. entity를 지정하지 않으면 run은 기본 entity(일반적으로 사용자 이름)로 전송됩니다. |
| `--count`                 | 이 agent의 최대 run 수입니다.                                                                                                             |
| `--forward-signals`, `-f` | agent에 전달된 시그널(예: SIGINT/SIGTERM)을 하위 run에도 전달하여 정상적으로 종료할 수 있게 합니다. (기본값: False)                                                 |
