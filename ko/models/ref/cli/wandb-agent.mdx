W&amp;B 에이전트 실행하기


<div id="usage">
  ## 사용법
</div>

```bash
wandb agent SWEEP_ID [OPTIONS]
```


<div id="arguments">
  ## 인수
</div>

| 인수 | 설명 | 필수 여부 |
| :--- | :--- | :--- |
| `SWEEP_ID` | 사용 가능한 설명이 없습니다 | 예 |

<div id="options">
  ## 옵션
</div>

| 옵션 | 설명 |
| :--- | :--- |
| `--project`, `-p` | 스위프로 생성된 W&B 실행을 보낼 프로젝트 이름입니다. 프로젝트를 지정하지 않으면 실행은 'Uncategorized' 레이블이 지정된 프로젝트로 전송됩니다. |
| `--entity`, `-e` | 스위프로 생성된 W&B 실행을 보낼 사용자 이름 또는 팀 이름입니다. 지정한 엔터티가 이미 존재하는지 확인하십시오. 엔터티를 지정하지 않으면 실행은 기본 엔터티(보통 사용자 이름)로 전송됩니다. |
| `--count` | 이 에이전트에 대한 최대 실행 횟수입니다. |
| `--forward-signals`, `-f` | 에이전트에 전달된 신호(예: SIGINT/SIGTERM)를 자식 실행에도 전달하여 자식 실행이 정상적으로 종료될 수 있도록 합니다. (기본값: False) |
