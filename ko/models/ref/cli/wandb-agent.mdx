W&amp;B 에이전트를 실행하세요


<div id="usage">
  ## 사용
</div>

```bash
wandb agent SWEEP_ID [OPTIONS]
```


<div id="arguments">
  ## 인수
</div>

| 인수         | 설명            | 필수 |
| :--------- | :------------ | :- |
| `SWEEP_ID` | 설명이 제공되지 않습니다 | 예  |

<div id="options">
  ## 옵션
</div>

| Option                    | 설명                                                                                                                                |
| :------------------------ | :-------------------------------------------------------------------------------------------------------------------------------- |
| `--project`, `-p`         | 스윕에서 생성된 W&amp;B run이 전송될 프로젝트 이름입니다. 프로젝트를 지정하지 않으면 run은 &#39;Uncategorized&#39; 라벨이 붙은 프로젝트로 전송됩니다.                             |
| `--entity`, `-e`          | 스윕에서 생성된 W&amp;B run을 전송하려는 사용자 이름 또는 팀 이름입니다. 지정한 entity가 이미 존재하는지 확인하십시오. entity를 지정하지 않으면 run은 기본 entity(일반적으로 사용자 이름)로 전송됩니다. |
| `--count`                 | 이 에이전트가 실행할 최대 run 수입니다.                                                                                                          |
| `--forward-signals`, `-f` | 에이전트에 전달된 시그널(예: SIGINT/SIGTERM)를 자식 run으로 전달하여 정상적으로 종료할 수 있도록 합니다. (기본값: False)                                                 |
