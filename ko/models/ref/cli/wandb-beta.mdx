wandb CLI 코맨드의 베타 버전입니다.

이 코맨드들은 wandb의 향후 릴리즈에서 변경되거나 완전히 기능이 중단될 수 있습니다.

## Usage

```bash
wandb beta COMMAND [ARGS]...
```

## Commands

| 코맨드 | 설명 |
| :--- | :--- |
| [leet](/models/ref/cli/wandb-beta/wandb-beta-leet) | W&B LEET(Lightweight Experiment Exploration Tool)를 실행합니다. |
| [sync](/models/ref/cli/wandb-beta/wandb-beta-sync) | PATHS로 지정된 .wandb 파일을 업로드합니다. |
