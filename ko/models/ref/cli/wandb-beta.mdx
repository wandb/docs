title: "wandb 베타"
---

wandb CLI 명령어의 베타 버전입니다.

이 명령어는 wandb의 어느 릴리스에서든 변경되거나 아예 작동하지 않을 수 있습니다.

<div id="usage">
  ## 사용
</div>

```bash
wandb beta COMMAND [ARGS]...
```

<div id="commands">
  ## 명령어
</div>

| 명령어                                                | 설명                            |
| :------------------------------------------------- | :---------------------------- |
| [leet](/ko/models/ref/cli/wandb-beta/wandb-beta-leet) | W&amp;B LEET: 경량 실험 탐색 도구.    |
| [sync](/ko/models/ref/cli/wandb-beta/wandb-beta-sync) | PATHS에 지정된 .wandb 파일을 업로드합니다. |
