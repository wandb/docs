```bash
beta [OPTIONS] COMMAND [ARGS]...
```

<div id="description">
  ## 설명
</div>

wandb CLI 명령어의 베타 버전입니다.

이 명령어는 wandb의 어느 릴리스에서든 변경되거나 아예 작동하지 않을 수 있습니다.

<div id="subcommands">
  ## 하위 명령어
</div>

| 명령어               | 설명                                              |
| ----------------- | ----------------------------------------------- |
| `wandb beta core` | 다중 프로세스 워크로드를 위해 공유되는 로컬 wandb-core 서비스를 관리합니다. |
| `wandb beta leet` | W&amp;B LEET: 경량 실험 탐색 도구.                      |
| `wandb beta sync` | PATHS에 지정된 .wandb 파일을 업로드합니다.                   |
