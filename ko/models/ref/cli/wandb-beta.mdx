---

wandb CLI 명령의 베타 버전입니다.

이 명령들은 향후 wandb의 어떤 릴리스에서든 변경되거나 완전히 동작하지 않을 수 있습니다.

<div id="usage">
  ## 사용 방법
</div>

```bash
wandb beta COMMAND [ARGS]...
```

<div id="commands">
  ## 명령어
</div>

| 명령어 | 설명 |
| :--- | :--- |
| [leet](/ko/models/ref/cli/wandb-beta/wandb-beta-leet) | W&amp;B LEET(경량 실험 탐색 도구)를 실행합니다. |
| [sync](/ko/models/ref/cli/wandb-beta/wandb-beta-sync) | PATHS로 지정한 .wandb 파일을 업로드합니다. |
