W&amp;B job을 관리하고 조회하는 명령어

  ## 사용법
</div>

```bash
wandb job COMMAND [ARGS]...
```

<div id="commands">
  ## 명령어
</div>

| 명령 | 설명 |
| :--- | :--- |
| [create](/ko/models/ref/cli/wandb-job/wandb-job-create) | wandb 실행 없이 소스에서 작업을 생성합니다. |
| [describe](/ko/models/ref/cli/wandb-job/wandb-job-describe) | Launch 작업을 설명합니다. |
| [list](/ko/models/ref/cli/wandb-job/wandb-job-list) | 프로젝트의 작업을 나열합니다. |
