---
title: wandb job
---

W&B jobs를 관리하고 확인하기 위한 코맨드

## Usage

```bash
wandb job COMMAND [ARGS]...
```

## Commands

| 코맨드 | 설명 |
| :--- | :--- |
| [create](/models/ref/cli/wandb-job/wandb-job-create) | wandb run 없이 소스로부터 job을 생성합니다. |
| [describe](/models/ref/cli/wandb-job/wandb-job-describe) | Launch job에 대한 상세 정보를 확인합니다. |
| [list](/models/ref/cli/wandb-job/wandb-job-list) | 프로젝트 내의 jobs 목록을 표시합니다. |
