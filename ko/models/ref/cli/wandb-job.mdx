<div id="description">
  ## 설명
</div>

W&amp;B 작업을 관리하고 확인하는 명령어.

<div id="subcommands">
  ## 하위 명령어
</div>

| Command              | Description                                       |
| -------------------- | ------------------------------------------------- |
| `wandb job create`   | 소스에서 wandb run 없이 작업을 생성합니다.                      |
| `wandb job describe` | Launch 작업의 세부 정보를 표시합니다. Launch 작업을 다음 형식으로 지정하세요 |
| `wandb job list`     | 프로젝트의 작업 목록을 표시합니다.                               |
