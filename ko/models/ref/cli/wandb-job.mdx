---

W&amp;B 작업을 관리하고 확인하는 명령어

<div id="usage">
  ## 사용
</div>

```bash
wandb job COMMAND [ARGS]...
```

<div id="commands">
  ## 명령어
</div>

| 명령어                                                      | 설명                           |
| :------------------------------------------------------- | :--------------------------- |
| [create](/ko/models/ref/cli/wandb-job/wandb-job-create)     | 소스에서 wandb run 없이 작업을 생성합니다. |
| [describe](/ko/models/ref/cli/wandb-job/wandb-job-describe) | Launch 작업의 세부 정보를 표시합니다.     |
| [list](/ko/models/ref/cli/wandb-job/wandb-job-list)         | 프로젝트의 작업 목록을 표시합니다.          |
