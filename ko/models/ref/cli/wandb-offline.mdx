W&amp;B에 로깅된 데이터를 클라우드로 업로드하지 않고 로컬에만 저장합니다.

오프라인 run을 업로드하려면 `wandb online` 또는 `wandb sync`를 사용하세요.


<div id="usage">
  ## 사용 방법
</div>

```bash
wandb offline
```
