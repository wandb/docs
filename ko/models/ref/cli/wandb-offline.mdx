W&B 에 로그된 데이터를 클라우드에 업로드하지 않고 로컬에 저장합니다.

오프라인 Runs 를 업로드하려면 `wandb online` 또는 `wandb sync` 를 사용하세요.

## 사용법

```bash
wandb offline
```
