W&amp;B에 기록된 데이터를 클라우드로 업로드하지 않고 로컬에만 저장합니다.

`wandb online` 또는 `wandb sync`를 사용하여 오프라인 실행을 업로드합니다.


<div id="usage">
  ## 사용 방법
</div>

```bash
wandb offline
```
