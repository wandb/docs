<div id="description">

</div>

W&amp;B LEET: 경량 실험 탐색 도구입니다.

로컬에서 W&amp;B Runs를 확인할 수 있는 터미널 UI입니다.

<div id="subcommands">
  ## 하위 명령어
</div>

| 명령어                      | 설명                     |
| ------------------------ | ---------------------- |
| `wandb beta leet config` | LEET 설정을 수정합니다.        |
| `wandb beta leet run`    | LEET TUI를 실행합니다.       |
| `wandb beta leet symon`  | 독립 실행형 시스템 모니터를 실행합니다. |
