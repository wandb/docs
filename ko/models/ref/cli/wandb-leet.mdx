<div id="description">
  ## 설명
</div>

W&amp;B LEET: 경량 실험 탐색 도구입니다.

로컬에서 W&amp;B run을 확인할 수 있는 터미널 UI입니다.



<div id="subcommands">
  ## 하위 명령어
</div>

| 명령어                 | 설명                         |
| ------------------- | -------------------------- |
| `wandb leet config` | LEET 설정을 편집합니다.            |
| `wandb leet run`    | LEET TUI를 실행합니다.           |
| `wandb leet symon`  | 독립형 system monitor를 실행합니다. |
