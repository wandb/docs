<div id="usage">
  ## 사용
</div>

```bash
controller [OPTIONS] SWEEP_ID
```

<div id="description">
  ## 설명
</div>

W&amp;B 하이퍼파라미터 스윕용 로컬 스윕 컨트롤러를 시작합니다.

지정한 스윕을 조정하는 로컬 프로세스를 시작합니다. W&amp;B에서
스윕 설정을 읽고, 구성된 검색 전략(그리드, 랜덤, 베이지안 등)에 따라
하이퍼파라미터 조합을 선택한 다음, run을 스윕 에이전트에 분배합니다.

기본적으로 W&amp;B는 관리형 인프라에서 스윕 컨트롤러를 실행합니다.
대신 컨트롤러를 로컬에서 실행하려면 이 명령을 사용하세요. 예를 들어,
이 명령을 사용해 동작을 디버그하거나 연결이 제한된 환경에서
운영할 수 있습니다.

스윕을 생성할 때 `wandb sweep`이 `sweep_id`를 출력합니다. 이는
스윕의 고유 식별자로 이루어지며, entity 및 프로젝트 경로
(entity/project/sweep&#95;id)를 포함할 수 있습니다.

예를 들어, sweep ID가 wbyz9876인 스윕의 로컬 스윕 컨트롤러를 시작하려면 다음과 같이 하세요:

```bash
wandb controller wbyz9876
```
