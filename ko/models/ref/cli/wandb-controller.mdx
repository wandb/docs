W&B 로컬 sweep controller를 실행합니다.

## 사용법

```bash
wandb controller SWEEP_ID [OPTIONS]
```

## 인수

| 인수 | 설명 | 필수 여부 |
| :--- | :--- | :--- |
| `SWEEP_ID` | 설명 없음 | 예 |

## 옵션

| 옵션 | 설명 |
| :--- | :--- |
| `--verbose` | 상세 출력을 표시합니다 (기본값: False) |
