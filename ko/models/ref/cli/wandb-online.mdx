<div id="description">
  ## 설명
</div>

W&amp;B run의 클라우드 동기화를 다시 활성화합니다.

오프라인 모드 설정을 해제하여 이 디렉터리에서 이후에 실행하는 run이
W&amp;B 클라우드에 데이터를 동기화하도록 합니다.

이전에 실행한 `wandb offline` 명령을 되돌립니다.

예를 들어, 오프라인으로 작업한 후 클라우드 동기화를 다시 활성화하려면 다음과 같이 하세요:

```bash
wandb online
```

클라우드 동기화를 다시 활성화한 후 트레이닝 스크립트를 실행하세요:

```bash
wandb online && python train.py
```
