<div id="usage">
  ## 사용
</div>

```bash
wandb login [KEY] [OPTIONS]
```


<div id="arguments">
  ## 인수
</div>

| 인수    | 설명           | 필수  |
| :---- | :----------- | :-- |
| `KEY` | 제공된 설명이 없습니다 | 아니요 |

<div id="options">
  ## 옵션
</div>

| 옵션                     | 설명                               |
| :--------------------- | :------------------------------- |
| `--cloud`              | 로컬이 아니라 클라우드에 로그인합니다(기본값: False) |
| `--host`, `--base-url` | 특정 W&amp;B 인스턴스에 로그인합니다          |
| `--relogin`            | 이미 로그인되어 있으면 강제로 다시 로그인합니다.      |
| `--verify`             | 로그인 자격 증명을 확인합니다(기본값: False)     |
