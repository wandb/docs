로컬 W&B 컨테이너를 시작합니다 (더 이상 사용되지 않음, wandb server --help를 참조하세요)

## 사용법

```bash
wandb local [OPTIONS]
```

## 옵션

| 옵션 | 설명 |
| :--- | :--- |
| `--port`, `-p` | W&B 로컬을 바인딩할 호스트 포트 (기본값: 8080) |
| `--env`, `-e` | wandb/local에 전달할 환경 변수 (기본값: []) |
| `--daemon` | 데몬 모드 실행 여부 설정 (기본값: True) |
| `--upgrade` | 최신 버전으로 업그레이드 (기본값: False) |
