<div id="usage">
  ## 사용 방법
</div>

```bash
wandb local [OPTIONS]
```


<div id="options">
  ## 옵션
</div>

| 옵션 | 설명 |
| :--- | :--- |
| `--port`, `-p` | W&B Local을 바인딩할 호스트 포트 (기본값: 8080) |
| `--env`, `-e` | wandb/local에 전달할 환경 변수들 (기본값: []) |
| `--daemon` | 데몬 모드로 실행할지 여부 (기본값: True) |
| `--upgrade` | 최신 버전으로 업그레이드할지 여부 (기본값: False) |
