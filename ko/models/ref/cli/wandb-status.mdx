<div id="usage">
  ## 사용
</div>

```bash
status [OPTIONS]
```

<div id="description">
  ## 설명
</div>

현재 W&amp;B 설정을 표시합니다.

base URL, API 키, 프로젝트, entity 및 기타 최종 적용된 값을 포함해
현재 활성화된 모든 W&amp;B 설정을 형식이 지정된 JSON으로 출력합니다.

예를 들어, 현재 설정을 표시하려면 다음과 같습니다:

```bash
wandb status
```

<div id="options">
  ## 옵션
</div>

| 플래그          | 유형       | 설명                          |
| ------------ | -------- | --------------------------- |
| `--settings` | BOOL 플래그 | 현재 설정을 표시합니다. **기본값**: True |
