<div id="description">
  ## 설명
</div>

W&amp;B 기능을 비활성화합니다.

W&amp;B가 `disabled` 모드인 동안에는 데이터를 로깅하거나 동기화하지 않습니다.

W&amp;B 기능을 복원하려면 `wandb enable`을 사용하세요.

로컬에는 계속 데이터를 로깅하면서 클라우드 동기화만 중지하려면 `wandb offline`을 사용하세요.

예를 들어, W&amp;B를 꺼서 train.py 스크립트가 W&amp;B에 데이터를 로깅하거나
동기화하지 않고 실행되도록 하려면 다음과 같이 하세요:

```bash
wandb disabled
```

다음으로, W&amp;B에 로깅하거나 동기화하지 않은 상태로 모델을 학습하세요:

```bash
python train.py  # W&B에 데이터를 로깅하거나 동기화하지 않습니다
```

다시 로깅하고 동기화할 준비가 되면 W&amp;B 기능을 복원하세요:

```bash
wandb enabled
```
