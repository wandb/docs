---
title: wandb 아티팩트
---

Artifacts와 상호작용하기 위한 코맨드

## Usage

```bash
wandb artifact COMMAND [ARGS]...
```

## Commands

| 코맨드 | 설명 |
| :--- | :--- |
| [cache](/models/ref/cli/wandb-artifact/wandb-artifact-cache) | Artifacts 캐시와 상호작용하기 위한 코맨드 |
| [get](/models/ref/cli/wandb-artifact/wandb-artifact-get) | W&B에서 아티팩트 다운로드 |
| [ls](/models/ref/cli/wandb-artifact/wandb-artifact-ls) | W&B 프로젝트의 모든 Artifacts 리스트 출력 |
| [put](/models/ref/cli/wandb-artifact/wandb-artifact-put) | W&B로 아티팩트 업로드 |
