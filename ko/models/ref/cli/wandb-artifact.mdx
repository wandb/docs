<div id="description">
  ## 설명
</div>

W&amp;B 아티팩트를 업로드하고 다운로드하며 관리합니다.

<div id="subcommands">
  ## 하위 명령어
</div>

| 명령                     | 설명                               |
| ---------------------- | -------------------------------- |
| `wandb artifact cache` | 로컬 아티팩트 캐시를 관리합니다.               |
| `wandb artifact get`   | 경로로 W&amp;B에서 아티팩트를 다운로드합니다.     |
| `wandb artifact ls`    | W&amp;B 프로젝트의 모든 아티팩트 목록을 표시합니다. |
| `wandb artifact put`   | W&amp;B에 아티팩트를 업로드합니다.           |
