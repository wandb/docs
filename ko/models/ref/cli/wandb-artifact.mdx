---
title: "wandb artifact"
---

artifact를 다루는 명령어

<div id="usage">
  ## 사용
</div>

```bash
wandb artifact COMMAND [ARGS]...
```

<div id="commands">
  ## 명령어
</div>

| 명령어                                                          | 설명                             |
| :----------------------------------------------------------- | :----------------------------- |
| [cache](/ko/models/ref/cli/wandb-artifact/wandb-artifact-cache) | artifact 캐시를 관리하는 명령어          |
| [get](/ko/models/ref/cli/wandb-artifact/wandb-artifact-get)     | wandb에서 artifact를 다운로드합니다      |
| [ls](/ko/models/ref/cli/wandb-artifact/wandb-artifact-ls)       | wandb 프로젝트의 모든 artifact를 나열합니다 |
| [put](/ko/models/ref/cli/wandb-artifact/wandb-artifact-put)     | artifact를 wandb에 업로드합니다        |
