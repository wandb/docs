Weights & Biases 에서 파일을 가져옵니다.

## 사용법

```bash
wandb pull RUN [OPTIONS]
```

## 인수

| 인수 | 설명 | 필수 여부 |
| :--- | :--- | :--- |
| `RUN` | 설명 없음 | 예 |

## 옵션

| 옵션 | 설명 |
| :--- | :--- |
| `--project`, `-p` | 다운로드할 프로젝트. |
| `--entity`, `-e` | 목록 범위를 지정할 엔티티. (기본값: models) |
