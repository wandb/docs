Weights &amp; Biases에서 파일 가져오기


<div id="usage">
  ## 사용
</div>

```bash
wandb pull RUN [OPTIONS]
```


<div id="arguments">
  ## 인자
</div>

| 인자    | 설명       | 필수 |
| :---- | :------- | :- |
| `RUN` | 설명이 없습니다 | 예  |

<div id="options">
  ## 옵션
</div>

| 옵션                | 설명                                   |
| :---------------- | :----------------------------------- |
| `--project`, `-p` | 다운로드할 프로젝트입니다.                       |
| `--entity`, `-e`  | 목록의 범위를 지정할 entity입니다. (기본값: models) |
