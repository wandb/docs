<div id="usage">
  ## 사용 방법
</div>

```bash
wandb pull RUN [OPTIONS]
```


<div id="arguments">
  ## 인수
</div>

| 인수 | 설명 | 필수 여부 |
| :--- | :--- | :--- |
| `RUN` | 설명이 제공되지 않습니다 | 예 |

<div id="options">
  ## 옵션
</div>

| 옵션 | 설명 |
| :--- | :--- |
| `--project`, `-p` | 다운로드할 프로젝트. |
| `--entity`, `-e` | 목록의 범위를 지정할 엔터티. (기본값: models) |
