로컬 W&amp;B 서버 시작


<div id="usage">
  ## 사용법
</div>

```bash
wandb server start [OPTIONS]
```


<div id="options">
  ## 옵션
</div>

| 옵션 | 설명 |
| :--- | :--- |
| `--port`, `-p` | W&B 서버를 바인딩할 호스트 포트 번호 (기본값: 8080) |
| `--env`, `-e` | wandb/local에 전달할 환경 변수들 (기본값: []) |
| `--daemon` | 데몬 모드로 실행할지 여부 (기본값: True) |
