로컬 W&B 서버를 시작합니다.

## 사용법

```bash
wandb server start [OPTIONS]
```

## 옵션

| 옵션 | 설명 |
| :--- | :--- |
| `--port`, `-p` | W&B 서버를 바인딩할 호스트 포트 (기본값: 8080) |
| `--env`, `-e` | wandb/local에 전달할 환경 변수 (기본값: []) |
| `--daemon` | 데몬 모드 실행 여부 설정 (기본값: True) |
