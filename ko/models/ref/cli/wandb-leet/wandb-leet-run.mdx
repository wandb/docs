<div id="usage">
  ## 사용
</div>

```bash
run [OPTIONS] [PATH]
```

<div id="description">
  ## 설명
</div>

LEET TUI를 실행합니다.

LEET는 선택 사항인 PATH로 지정된 W&amp;B run을 확인하는 터미널 UI입니다.

PATH에는 .wandb 파일, .wandb 파일이 포함된 run 디렉터리,
또는 W&amp;B run URL을 지정할 수 있습니다.
PATH를 지정하지 않으면 command가 최신 run을 찾습니다.

<div id="arguments">
  ## 인수
</div>

| 이름     | 기본값 | 유형    |
| ------ | --- | ----- |
| `path` | STR | False |
