W&amp;B Launch 에이전트를 실행합니다.


<div id="usage">
  ## 사용 방법
</div>

```bash
wandb launch-agent [OPTIONS]
```


<div id="options">
  ## 옵션
</div>

| 옵션 | 설명 |
| :--- | :--- |
| `--queue`, `-q` | 에이전트가 모니터링할 큐 이름입니다. 여러 개의 -q 플래그를 사용할 수 있습니다. |
| `--entity`, `-e` | 사용할 entity입니다. 기본값은 현재 로그인한 사용자입니다. |
| `--log-file`, `-l` | 내부 에이전트 로그의 저장 위치입니다. stdout으로 출력하려면 -를 사용하세요. 기본적으로 모든 에이전트 로그는 wandb/ 하위 디렉터리의 debug.log 파일로 기록되며, WANDB_DIR가 설정된 경우 해당 경로로 기록됩니다. |
| `--max-jobs`, `-j` | 이 에이전트가 동시에 실행할 수 있는 최대 Launch 작업 수입니다. 기본값은 1입니다. 상한을 두지 않으려면 -1로 설정하세요. |
| `--config`, `-c` | 사용할 에이전트 설정 YAML 파일의 경로입니다. |
| `--verbose`, `-v` | 자세한 출력을 표시합니다(기본값: 0). |
