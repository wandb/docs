---
title: "wandb launch-agent"
---

W&B launch 에이전트를 실행합니다.

## Usage

```bash
wandb launch-agent [OPTIONS]
```

## Options

| Option | Description |
| :--- | :--- |
| `--queue`, `-q` | 에이전트가 모니터링할 큐의 이름입니다. 여러 개의 -q 플래그를 지원합니다. |
| `--entity`, `-e` | 사용할 Entity입니다. 기본값은 현재 로그인된 사용자입니다. |
| `--log-file`, `-l` | 내부 에이전트 로그의 저장 위치입니다. stdout을 사용하려면 -를 입력하세요. 기본적으로 모든 에이전트 로그는 wandb/ 하위 디렉토리의 debug.log 또는 설정된 경우 WANDB_DIR로 전송됩니다. |
| `--max-jobs`, `-j` | 이 에이전트가 병렬로 실행할 수 있는 최대 launch 작업 수입니다. 기본값은 1입니다. 상한선을 없애려면 -1로 설정하세요. |
| `--config`, `-c` | 사용할 에이전트 설정 yaml 파일의 경로입니다. |
| `--verbose`, `-v` | 상세 출력을 표시합니다 (기본값: 0). |
