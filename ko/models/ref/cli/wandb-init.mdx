Weights &amp; Biases를 사용해 디렉터리 구성하기


<div id="usage">
  ## 사용 방법
</div>

```bash
wandb init [OPTIONS]
```


<div id="options">
  ## 옵션
</div>

| 옵션 | 설명 |
| :--- | :--- |
| `--project`, `-p` | 사용할 프로젝트. |
| `--entity`, `-e` | 프로젝트 범위를 지정할 엔티티. |
| `--reset` | 설정을 초기화합니다(기본값: False). |
| `--mode`, `-m` | "online", "offline" 또는 "disabled"일 수 있습니다. 기본값은 "online"입니다. |
