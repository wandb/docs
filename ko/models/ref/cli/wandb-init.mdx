Weights &amp; Biases로 디렉터리 설정하기


<div id="usage">
  ## 사용
</div>

```bash
wandb init [OPTIONS]
```


<div id="options">
  ## 옵션
</div>

| 옵션                | 설명                                                                        |
| :---------------- | :------------------------------------------------------------------------ |
| `--project`, `-p` | 사용할 프로젝트입니다.                                                              |
| `--entity`, `-e`  | 프로젝트에 적용할 entity입니다.                                                      |
| `--reset`         | 설정을 재설정합니다(기본값: False).                                                   |
| `--mode`, `-m`    | `"online"`, `"offline"`, `"disabled"` 중 하나를 사용할 수 있습니다. 기본값은 `online`입니다. |
