Weights &amp; Biases용 디렉터리 설정


<div id="usage">
  ## 사용 방법
</div>

```bash
wandb init [OPTIONS]
```


<div id="options">
  ## 옵션
</div>

| 옵션 | 설명 |
| :--- | :--- |
| `--project`, `-p` | 사용할 프로젝트. |
| `--entity`, `-e` | 프로젝트의 범위를 지정할 엔터티. |
| `--reset` | 설정을 초기화 (기본값: False). |
| `--mode`, `-m` | "online", "offline" 또는 "disabled" 모드. 기본값은 "online". |
