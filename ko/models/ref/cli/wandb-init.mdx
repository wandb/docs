Weights & Biases를 사용하도록 디렉토리를 설정합니다.

## Usage

```bash
wandb init [OPTIONS]
```

## Options

| Option | Description |
| :--- | :--- |
| `--project`, `-p` | 사용할 Projects 를 지정합니다. |
| `--entity`, `-e` | 프로젝트의 범위를 지정할 Entities 를 설정합니다. |
| `--reset` | 설정을 초기화합니다. (기본값: False) |
| `--mode`, `-m` | "online", "offline" 또는 "disabled" 중 하나를 선택할 수 있습니다. 기본값은 online입니다. |
