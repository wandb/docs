Weights &amp; Biases를 사용해 디렉터리 설정하기


<div id="usage">
  ## 사용
</div>

```bash
wandb init [OPTIONS]
```


<div id="options">
  ## 옵션
</div>

| 옵션                | 설명                                                                                                   |
| :---------------- | :--------------------------------------------------------------------------------------------------- |
| `--project`, `-p` | 사용할 프로젝트.                                                                                            |
| `--entity`, `-e`  | 프로젝트 범위를 지정할 entity.                                                                                 |
| `--reset`         | 설정을 재설정합니다(기본값: False).                                                                              |
| `--mode`, `-m`    | &quot;online&quot;, &quot;offline&quot; 또는 &quot;disabled&quot;일 수 있습니다. 기본값은 &quot;online&quot;입니다. |
