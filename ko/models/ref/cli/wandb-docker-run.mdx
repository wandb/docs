<div id="usage">
  ## 사용
</div>

```bash
docker-run [OPTIONS] [DOCKER_RUN_ARGS]...
```

<div id="description">
  ## 설명
</div>

`docker run`을 래핑하고 W&amp;B 환경 변수를 자동으로 주입합니다.

다음을 주입하면서 모든 인수를 `docker run`에 그대로 전달합니다:

* WANDB&#95;API&#95;KEY: 로그인되어 있으면 현재 API 키를 주입하여
  컨테이너가 W&amp;B에 인증할 수 있도록 합니다.

  * WANDB&#95;DOCKER: 인수에서 이미지를 감지할 수 있으면 확인된 이미지 ID를 주입하여
    어떤 도커 이미지가 해당 run을 생성했는지 W&amp;B가 추적할 수 있도록 합니다.

호스트에서 `nvidia-docker`가 감지되고 `--runtime`이 아직 설정되지 않은 경우
`--runtime nvidia`를 자동으로 추가합니다.

예를 들어, &quot;my-image&quot; 컨테이너 내부에서 Python 스크립트 train.py를 실행하려면:

```bash
wandb docker-run my-image python train.py
```
