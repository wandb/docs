Artifacts 캐시와 상호작용하기 위한 코맨드

## Usage

```bash
wandb artifact cache COMMAND [ARGS]...
```

## Commands

| Command | Description |
| :--- | :--- |
| [cleanup](/models/ref/cli/wandb-artifact/wandb-artifact-cache/wandb-artifact-cache-cleanup) | Artifacts 캐시에서 사용 빈도가 낮은 파일들을 정리합니다. |
