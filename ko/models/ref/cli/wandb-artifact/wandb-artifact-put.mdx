wandb에 아티팩트 업로드하기


<div id="usage">
  ## 사용
</div>

```bash
wandb artifact put PATH [OPTIONS]
```


<div id="arguments">
  ## 인수
</div>

| 인수     | 설명    | 필수 |
| :----- | :---- | :- |
| `PATH` | 설명 없음 | 예  |

<div id="options">
  ## 옵션
</div>

| 옵션                    | 설명                                       |
| :-------------------- | :--------------------------------------- |
| `--name`, `-n`        | 푸시할 아티팩트의 이름: project/artifact&#95;name  |
| `--description`, `-d` | 이 아티팩트에 대한 설명                            |
| `--type`, `-t`        | 아티팩트의 유형 (기본값: 데이터셋)                     |
| `--alias`, `-a`       | 이 아티팩트에 적용할 별칭 (기본값: [&#39;latest&#39;]) |
| `--id`                | 업로드할 런 ID                                |
| `--resume`            | 현재 디렉토리의 마지막 런 재개                        |
| `--skip_cache`        | 아티팩트 파일 업로드 시 캐싱 건너뛰기 (기본값: False)       |
| `--policy`            | 아티팩트 파일 업로드 시 저장소 정책 설정 (기본값: mutable)   |
