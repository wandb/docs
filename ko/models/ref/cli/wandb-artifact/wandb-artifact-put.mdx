Artifacts 를 wandb에 업로드합니다.

## Usage

```bash
wandb artifact put PATH [OPTIONS]
```

## Arguments

| Argument | Description | Required |
| :--- | :--- | :--- |
| `PATH` | 설명 없음 | Yes |

## Options

| Option | Description |
| :--- | :--- |
| `--name`, `-n` | 푸시할 Artifact 의 이름: project/artifact_name |
| `--description`, `-d` | 이 Artifact 에 대한 설명 |
| `--type`, `-t` | Artifact 의 타입 (기본값: dataset) |
| `--alias`, `-a` | 이 Artifact 에 적용할 에일리어스 (기본값: ['latest']) |
| `--id` | 업로드하려는 대상 Run 입니다. |
| `--resume` | 현재 디렉토리의 마지막 Run 을 재개합니다. |
| `--skip_cache` | Artifact 파일을 업로드하는 동안 캐싱을 건너뜁니다. (기본값: False) |
| `--policy` | Artifact 파일을 업로드하는 동안 스토리지 정책을 설정합니다. (기본값: mutable) |
