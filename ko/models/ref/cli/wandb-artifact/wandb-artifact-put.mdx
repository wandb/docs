wandb에 artifact 업로드하기


<div id="usage">
  ## 사용
</div>

```bash
wandb artifact put PATH [OPTIONS]
```


<div id="arguments">
  ## 인수
</div>

| 인수     | 설명       | 필수 |
| :----- | :------- | :- |
| `PATH` | 설명이 없습니다 | 예  |

<div id="options">
  ## 옵션
</div>

| 옵션                    | 설명                                                      |
| :-------------------- | :------------------------------------------------------ |
| `--name`, `-n`        | 푸시할 artifact의 이름: 프로젝트/artifact&#95;name                |
| `--description`, `-d` | 이 artifact에 대한 설명                                       |
| `--type`, `-t`        | artifact의 유형 (기본값: 데이터셋)                                |
| `--alias`, `-a`       | 이 artifact에 적용할 alias (기본값: [&#39;latest&#39;])         |
| `--id`                | 업로드할 대상 run입니다.                                         |
| `--resume`            | 현재 디렉터리에서 마지막 run을 재개합니다.                               |
| `--skip_cache`        | artifact 파일을 업로드하는 동안 캐싱을 건너뜁니다. (기본값: False)           |
| `--policy`            | artifact 파일을 업로드하는 동안 저장소 policy를 설정합니다. (기본값: mutable) |
