Artifacts 캐시에서 사용 빈도가 낮은 파일을 정리합니다.

## 사용법

```bash
wandb artifact cache cleanup TARGET_SIZE [OPTIONS]
```

## 인수

| 인수 | 설명 | 필수 여부 |
| :--- | :--- | :--- |
| `TARGET_SIZE` | 설명 없음 | 예 |

## 옵션

| 옵션 | 설명 |
| :--- | :--- |
| `--remove-temp` | 임시 파일 삭제 (기본값: False) |
