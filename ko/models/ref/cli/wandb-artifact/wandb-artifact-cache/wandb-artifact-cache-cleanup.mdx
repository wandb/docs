artifacts 캐시에서 자주 사용되지 않는 파일 정리


<div id="usage">
  ## 사용
</div>

```bash
wandb artifact cache cleanup TARGET_SIZE [OPTIONS]
```


<div id="arguments">
  ## 인수
</div>

| 인수            | 설명    | 필수 |
| :------------ | :---- | :- |
| `TARGET_SIZE` | 설명 없음 | 예  |

<div id="options">
  ## 옵션
</div>

| 옵션              | 설명                       |
| :-------------- | :----------------------- |
| `--remove-temp` | 임시 파일을 제거합니다(기본값: False) |
