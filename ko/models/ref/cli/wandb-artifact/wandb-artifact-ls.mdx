W&B 프로젝트의 모든 Artifacts 목록을 출력합니다.

## Usage

```bash
wandb artifact ls PATH [OPTIONS]
```

## Arguments

| 인수 | 설명 | 필수 여부 |
| :--- | :--- | :--- |
| `PATH` | 설명 없음 | 예 |

## Options

| 옵션 | 설명 |
| :--- | :--- |
| `--type`, `-t` | 나열할 Artifacts의 타입 |
