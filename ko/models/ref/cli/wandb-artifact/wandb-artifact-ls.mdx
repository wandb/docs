wandb 프로젝트에 있는 모든 artifact를 나열합니다


<div id="usage">
  ## 사용
</div>

```bash
wandb artifact ls PATH [OPTIONS]
```


<div id="arguments">
  ## 인수
</div>

| 인수     | 설명    | 필수 |
| :----- | :---- | :- |
| `PATH` | 설명 없음 | 예  |

<div id="options">
  ## 옵션
</div>

| 옵션             | 설명               |
| :------------- | :--------------- |
| `--type`, `-t` | 나열할 artifact의 유형 |
