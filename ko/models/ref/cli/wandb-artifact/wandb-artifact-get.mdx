wandb에서 아티팩트 다운로드하기


<div id="usage">
  ## 사용법
</div>

```bash
wandb artifact get PATH [OPTIONS]
```


<div id="arguments">
  ## 인수
</div>

| 인수 | 설명 | 필수 여부 |
| :--- | :--- | :--- |
| `PATH` | 설명이 제공되지 않습니다 | 예 |

<div id="options">
  ## 옵션
</div>

| 옵션 | 설명 |
| :--- | :--- |
| `--root` | 다운로드할 아티팩트가 저장될 디렉터리 |
| `--type` | 다운로드할 아티팩트의 유형 |
