<div id="usage">
  ## 사용
</div>

```bash
get [OPTIONS] PATH
```

<div id="description">
  ## 설명
</div>

경로를 사용해 W&amp;B에서 아티팩트를 다운로드합니다.

PATH 형식은 entity/project/artifact&#95;name:version입니다. version을
생략하면 &quot;latest&quot; 별칭을 사용합니다.

예를 들어, &quot;team-awesome&quot; entity 아래의 &quot;foobar&quot; 프로젝트에서
&quot;processed-training-set&quot;라는 아티팩트의 최신 버전을 다운로드하려면 다음과 같습니다.

```bash
wandb artifact get team-awesome/foobar/processed-training-set:latest
```

&quot;processed-training-set&quot; 아티팩트의 특정 버전(v2)을
로컬 디렉터리(./data)에 다운로드하려면 다음과 같이 하세요:

```bash
wandb artifact get --root ./data team-awesome/foobar/processed-training-set:v2
```

<div id="arguments">
  ## 인수
</div>

| 이름     | 기본값 | 유형   |
| ------ | --- | ---- |
| `path` | STR | True |

<div id="options">
  ## 옵션
</div>

| 플래그      | 유형  | 설명                                                           |
| -------- | --- | ------------------------------------------------------------ |
| `--root` | STR | 아티팩트를 다운로드할 디렉터리입니다. 설정하지 않으면 기본 아티팩트 캐시를 사용합니다. **기본값**: 없음 |
| `--type` | STR | 예상되는 아티팩트 유형입니다. 아티팩트 유형이 일치하지 않으면 실패합니다. **기본값**: 없음        |
