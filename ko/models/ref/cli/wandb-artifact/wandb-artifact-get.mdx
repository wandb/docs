W&B에서 Artifacts 를 다운로드합니다.

## Usage

```bash
wandb artifact get PATH [OPTIONS]
```

## Arguments

| 인수 | 설명 | 필수 여부 |
| :--- | :--- | :--- |
| `PATH` | 설명 없음 | Yes |

## Options

| 옵션 | 설명 |
| :--- | :--- |
| `--root` | 아티팩트를 다운로드할 디렉토리 |
| `--type` | 다운로드하려는 아티팩트의 유형 |
