wandb에서 artifact 다운로드하기


<div id="usage">
  ## 사용
</div>

```bash
wandb artifact get PATH [OPTIONS]
```


<div id="arguments">
  ## 인수
</div>

| 인수     | 설명       | 필수 |
| :----- | :------- | :- |
| `PATH` | 설명이 없습니다 | 예  |

<div id="options">
  ## 옵션
</div>

| 옵션       | 설명                      |
| :------- | :---------------------- |
| `--root` | artifact를 다운로드할 대상 디렉터리 |
| `--type` | 다운로드할 artifact의 유형      |
