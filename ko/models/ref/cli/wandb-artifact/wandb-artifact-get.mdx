wandb에서 아티팩트 다운로드


<div id="usage">
  ## 사용
</div>

```bash
wandb artifact get PATH [OPTIONS]
```


<div id="arguments">
  ## 인수
</div>

| 인수     | 설명       | 필수 |
| :----- | :------- | :- |
| `PATH` | 설명이 없습니다 | 예  |

<div id="options">
  ## 옵션
</div>

| 옵션       | 설명               |
| :------- | :--------------- |
| `--root` | 아티팩트를 다운로드할 디렉터리 |
| `--type` | 다운로드하는 아티팩트의 유형  |
