프로젝트의 작업 목록 보기


<div id="usage">
  ## 사용
</div>

```bash
wandb job list [OPTIONS]
```


<div id="options">
  ## 옵션
</div>

| 옵션 | 설명 |
| :--- | :--- |
| `--project`, `-p` | 잡 목록을 표시할 프로젝트입니다. |
| `--entity`, `-e` | 잡이 속하는 엔터티입니다(기본값: Models). |
