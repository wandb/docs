프로젝트의 Job 나열


<div id="usage">
  ## 사용 방법
</div>

```bash
wandb job list [OPTIONS]
```


<div id="options">
  ## 옵션
</div>

| 옵션 | 설명 |
| :--- | :--- |
| `--project`, `-p` | 작업을 나열할 작업이 포함된 프로젝트입니다. |
| `--entity`, `-e` | 작업이 속한 엔터티입니다(기본값: models). |
