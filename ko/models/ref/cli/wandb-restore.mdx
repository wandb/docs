실행에 대한 코드, 설정 및 Docker 상태를 복원합니다. 코드가 `wandb.save()` 또는 `wandb.init(save_code=True)`로 저장되지 않은 경우에는 최신 커밋에서 코드를 가져옵니다.


<div id="usage">
  ## 사용 방법
</div>

```bash
wandb restore RUN [OPTIONS]
```


<div id="arguments">
  ## 인자
</div>

| 인자 | 설명 | 필수 여부 |
| :--- | :--- | :--- |
| `RUN` | 설명 없음 | 예 |

<div id="options">
  ## 옵션
</div>

| 옵션 | 설명 |
| :--- | :--- |
| `--no-git` | git 상태를 복원하지 않습니다 (기본값: False) |
| `--branch` | 브랜치를 생성할지 또는 detached 체크아웃을 할지 여부입니다 (기본값: True) |
| `--project`, `-p` | 업로드할 프로젝트를 지정합니다. |
| `--entity`, `-e` | 목록의 범위를 지정할 엔터티를 설정합니다. |
