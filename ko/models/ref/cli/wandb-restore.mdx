run의 코드, 설정, Docker 상태를 복원합니다. 코드가 `wandb.save()` 또는 `wandb.init(save_code=True)`로 저장되지 않은 경우 최신 커밋에서 코드를 조회합니다.


<div id="usage">
  ## 사용
</div>

```bash
wandb restore RUN [OPTIONS]
```


<div id="arguments">
  ## 인수
</div>

| 인수    | 설명            | 필수 |
| :---- | :------------ | :- |
| `RUN` | 설명이 제공되지 않습니다 | 예  |

<div id="options">
  ## 옵션
</div>

| 옵션                | 설명                                                   |
| :---------------- | :--------------------------------------------------- |
| `--no-git`        | git 상태를 복원하지 않습니다 (기본값: False)                       |
| `--branch`        | 브랜치를 생성할지 또는 분리(detached) 체크아웃을 할지 여부입니다 (기본값: True) |
| `--project`, `-p` | 업로드할 프로젝트입니다.                                        |
| `--entity`, `-e`  | 목록 범위를 지정할 entity입니다.                                |
