Run 의 코드, config 및 docker 상태를 복구합니다. `wandb.save()` 또는 `wandb.init(save_code=True)`를 통해 코드가 저장되지 않은 경우, 마지막 커밋에서 코드를 가져옵니다.

## Usage

```bash
wandb restore RUN [OPTIONS]
```

## Arguments

| 인수 | 설명 | 필수 여부 |
| :--- | :--- | :--- |
| `RUN` | 설명 없음 | Yes |

## Options

| 옵션 | 설명 |
| :--- | :--- |
| `--no-git` | git 상태를 복구하지 않습니다 (기본값: False) |
| `--branch` | 브랜치를 생성할지 또는 detached 상태로 체크아웃할지 여부 (기본값: True) |
| `--project`, `-p` | 업로드할 Projects 를 지정합니다. |
| `--entity`, `-e` | 리스트를 조회할 Entities 를 지정합니다. |
