<div id="usage">
  ## 사용
</div>

```bash
wandb [OPTIONS] COMMAND [ARGS]...
```

<div id="options">
  ## 옵션
</div>

| 옵션          | 설명                               |
| :---------- | :------------------------------- |
| `--version` | 버전을 표시하고 종료합니다. (default: False) |

| [enabled](/ko/models/ref/cli/wandb-enabled)           | W&amp;B를 활성화합니다.                                                          |

| [local](/ko/models/ref/cli/wandb-local)               | 로컬 W&amp;B 컨테이너를 시작합니다(사용 중단, `wandb server --help` 참조).                  |
| [login](/ko/models/ref/cli/wandb-login)               | W&amp;B 서비스 인증을 위해 API 키를 확인하고 저장합니다.                                     |
| [off](/ko/models/ref/cli/wandb-off)                   | 설명이 제공되지 않습니다.                                                            |
| [offline](/ko/models/ref/cli/wandb-offline)           | W&amp;B에 로깅된 데이터를 클라우드에 업로드하지 않고 로컬에 저장합니다.                               |
| [on](/ko/models/ref/cli/wandb-on)                     | 설명이 제공되지 않습니다.                                                            |

| [server](/ko/models/ref/cli/wandb-server)             | 로컬 W&amp;B 서버를 운영하기 위한 명령입니다.                                             |

| [sync](/ko/models/ref/cli/wandb-sync)                 | W&amp;B run 데이터를 클라우드에 동기화합니다.                                            |
| [verify](/ko/models/ref/cli/wandb-verify)             | 로컬 W&amp;B 인스턴스를 점검하고 확인합니다.                                              |
