<div id="usage">
  ## 사용
</div>

```bash
wandb [OPTIONS] COMMAND [ARGS]...
```

<div id="options">
  ## 옵션
</div>

| 옵션          | 설명                            |
| :---------- | :---------------------------- |
| `--version` | 버전을 표시한 후 종료합니다. (기본값: False) |

| [scheduler](/ko/models/ref/cli/wandb-scheduler)       | W&amp;B Launch 스윕 스케줄러를 실행합니다(실험적 기능).                   |
| [server](/ko/models/ref/cli/wandb-server)             | 로컬 W&amp;B 서버를 운영하는 명령어입니다                               |

| [sync](/ko/models/ref/cli/wandb-sync)                 | 기존 로컬 W&amp;B run 데이터를 클라우드에 업로드합니다.                     |
| [verify](/ko/models/ref/cli/wandb-verify)             | self-hosted W&amp;B 인스턴스를 대상으로 인테그레이션 검사를 실행합니다.         |
