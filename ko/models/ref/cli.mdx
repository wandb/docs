<div id="usage">
  ## 사용법
</div>

```bash
wandb [OPTIONS] COMMAND [ARGS]...
```

<div id="options">
  ## 옵션
</div>

| 옵션 | 설명 |
| :--- | :--- |
| `--version` | 버전을 표시한 후 종료합니다. (기본값: False) |
