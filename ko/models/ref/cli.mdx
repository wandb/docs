## 사용법

```bash
wandb [OPTIONS] COMMAND [ARGS]...
```

## 옵션 (Options)

| 옵션 | 설명 |
| :--- | :--- |
| `--version` | 버전을 표시하고 종료합니다. (기본값: False) |
