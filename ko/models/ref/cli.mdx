<div id="usage">
  ## 사용 방법
</div>

```bash
wandb [OPTIONS] COMMAND [ARGS]...
```

<div id="options">
  ## 옵션
</div>

| 옵션 | 설명 |
| :--- | :--- |
| `--version` | 버전 정보를 표시하고 종료합니다. (기본값: False) |
