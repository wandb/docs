### <a id="and"></a>`and`

두 값에 대한 논리적 `and` 결과를 반환합니다

| Argument | Description |
| :--- | :--- |
| `lhs` | 첫 번째 이진 값 |
| `rhs` | 두 번째 이진 값 |

#### 반환 값

두 값을 논리 AND한 결과

| :--- | :--- |

| `rhs` | 두 번째 불리언 값 |

#### 반환 값
