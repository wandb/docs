  # W&B 개요


  Weights & Biases(W&B)는 머신러닝 팀이 모델을 더 빠르게 구축하고 최적화할 수 있도록 돕는 AI 개발 플랫폼입니다. W&B의 도구
  모음은 Experiments 를 추적하고, 데이터를 시각화하며, 모델 성능을 공유하는 과정을 간소화합니다.


  W&B 플랫폼은 다음과 같은 핵심 구성 요소로 이루어져 있습니다:


  1. [**Experiments**](https://docs.wandb.ai/guides/runs): 머신러닝 실험의 하이퍼파라미터, 지표 및
  결과(Runs)를 추적하고 시각화합니다.

  2. [**Artifacts**](https://docs.wandb.ai/guides/artifacts): 데이터셋, 모델 및 기타 바이너리 파일을
  버전 관리하여 전체 머신러닝 파이프라인의 데이터 리니지를 추적합니다.

  3. [**Sweeps**](https://docs.wandb.ai/guides/sweeps): 하이퍼파라미터 튜닝을 자동화하여 모델의 최적 구성을
  신속하게 찾습니다.

  4. [**Models**](https://docs.wandb.ai/guides/models): 모델의 라이프사이클을 관리하고, 학습부터 프로덕션
  배포까지의 과정을 조율합니다.

  5. [**Reports**](https://docs.wandb.ai/guides/reports): 실험 결과와 분석 내용을 시각적으로 정리하여
  팀원들과 공유하고 협업합니다.

  6. [**Launch**](https://docs.wandb.ai/guides/launch): 모든 컴퓨팅 환경에서 머신러닝 워크플로우를 간편하게
  실행하고 관리합니다.

  7. [**Weave**](https://docs.wandb.ai/guides/weave): 생성형 AI(Generative AI) 애플리케이션을
  개발하고 반복 개선하기 위한 경량 워크플로우 도구입니다.


  W&B를 사용하면 복잡한 머신러닝 프로젝트를 체계적으로 관리하고, 팀 전체의 생산성을 높일 수 있습니다. 시작하려면 [빠른 시작 가이드](https://docs.wandb.ai/quickstart)를
  확인해 보세요.'
