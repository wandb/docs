## Chainable Ops
### <a id="asset-file"></a>`asset-file`

애셋의 _file_ 을 반환합니다.

| 인수 | 설명 |
| :--- | :--- |
| `asset` | 애셋 |

#### 반환 값
애셋의 _file_

### <a id="partitionedtable-file"></a>`partitionedtable-file`

_partitioned-table_ 의 _file_ 을 반환합니다.

| 인수 | 설명 |
| :--- | :--- |
| `partitionedTable` | 해당 _partitioned-table_ |

#### 반환 값
_partitioned-table_ 의 _file_

### <a id="partitionedtable-rows"></a>`partitionedtable-rows`

_partitioned-table_ 의 행(rows)을 반환합니다.

| 인수 | 설명 |
| :--- | :--- |
| `partitionedTable` | 행을 가져올 _partitioned-table_ |

#### 반환 값
_partitioned-table_ 의 행(rows)
