W&amp;B Public API는 W&amp;B에 저장된 데이터를 프로그래밍 방식으로 쿼리, 내보내기, 업데이트할 수 있도록 지원합니다. 이 API는 사후 분석, 데이터 내보내기, 그리고 run, 아티팩트, sweep의 프로그래밍 방식 관리에 사용하세요. 기본 SDK가 트레이닝 중 실시간 logging을 처리하는 반면, Public API를 사용하면 과거 데이터를 조회하고, 메타데이터를 업데이트하고, 아티팩트를 관리하고, 완료된 실험을 분석할 수 있습니다. 기본 `Api` 클래스는 대부분의 기능에 접근하는 진입점입니다.

<div id="authentication">
  ## 인증
</div>

Public API는 Python SDK와 동일한 인증 방식을 사용합니다. 다음과 같은 여러 방법으로 인증할 수 있습니다:

`WANDB_API_KEY` 환경 변수를 사용해 API 키를 설정합니다:

```bash
export WANDB_API_KEY=your_api_key
```

`Api` 클래스를 초기화할 때 API 키를 직접 전달하세요:

```python
api = Api(api_key="your_api_key")
```

또는 현재 세션을 인증하려면 `wandb.login()`을 사용하세요:

```python
import wandb

wandb.login()
api = Api()
```
