## <kbd>class</kbd> `MetricChangeFilter`
메트릭 value 의 **변화량**을 사용자가 정의한 임계값(threshold)과 비교하는 필터입니다.

변화량은 "tumbling" 윈도우를 통해 계산됩니다. 즉, 현재 윈도우와 겹치지 않는 이전 윈도우 사이의 차이를 계산합니다.
