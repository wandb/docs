## <kbd>class</kbd> `NewAutomation`

새로 생성되는 자동화입니다.

### <kbd>방법</kbd> `NewAutomation.__init__`

```python
__init__(
    name: 'str | None' = None,
    description: 'str | None' = None,
    enabled: 'bool | None' = None,
    event: 'Annotated | None' = None,
    action: 'Annotated | None' = None
) → None
```

**매개변수:**

* `name` (Optional[str]): 이 자동화의 이름입니다.
* `description` (Optional[str]): 이 자동화에 대한 설명(선택 사항)입니다.
* `enabled` (Optional[bool]): 이 자동화가 활성화되어 있는지 여부입니다. 활성화된 자동화만 트리거됩니다.
* `event` (Optional[Annotated]): 이 자동화를 트리거할 이벤트입니다.
* `action` (Optional[Annotated]): 이 자동화가 트리거될 때 실행될 액션입니다.

**반환값:**
`NewAutomation` 객체입니다.


### <kbd>속성</kbd> `NewAutomation.scope`

트리거 이벤트가 발생해야 하는 스코프입니다.

**반환값:**

* `Optional[AutomationScope]`: 스코프 속성 값입니다.
