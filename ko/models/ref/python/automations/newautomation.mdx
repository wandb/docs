## <kbd>class</kbd> `NewAutomation`

새로 생성되는 자동화입니다.

### <kbd>method</kbd> `NewAutomation.__init__`

```python
__init__(
    name: 'str | None' = None,
    description: 'str | None' = None,
    enabled: 'bool | None' = None,
    event: 'Annotated | None' = None,
    action: 'Annotated | None' = None
) → None
```

**인자:**

* `name` (Optional[str]): 이 automation의 이름.
* `description` (Optional[str]): 이 automation에 대한 선택적 설명.
* `enabled` (Optional[bool]): 이 automation이 활성화되어 있는지 여부. 활성화된 automation만 트리거된다.
* `event` (Optional[Annotated]): 이 automation을 트리거하는 이벤트.
* `action` (Optional[Annotated]): 이 automation이 트리거될 때 실행될 동작.

**반환값:**
`NewAutomation` 객체.
