## <kbd>클래스</kbd> `RunStateFilter`

Run 상태의 변경을 기준으로 이벤트를 트리거하는 필터를 나타냅니다.

### <kbd>방법</kbd> `RunStateFilter.__init__`

```python
__init__(
    run: 'And | Or | Nor | Not | Lt | Gt | Lte | Gte | Eq | Ne | In | NotIn | Exists | Regex | Contains | FilterExpr | dict[str, Any]' = And(()),
    state: 'StateFilter'
) → None
```

**매개변수:**

* `run` (Union[And, Or, Nor, Not, Lt, Gt, Lte, Gte, Eq, Ne, In, NotIn, Exists, Regex, Contains, FilterExpr, Dict[str, Any]]): 이 이벤트를 트리거할 모든 run과 일치해야 하는 필터입니다.
* `state` (StateFilter): 이 이벤트가 트리거되기 위해 충족되어야 하는 Run 상태 조건입니다.

**반환값:**
`RunStateFilter` 객체입니다.
