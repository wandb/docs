## <kbd>class</kbd> `MetricZScoreFilter`

메트릭의 z-점수를 사용자가 정의한 임계값과 비교하는 필터입니다.

### <kbd>메서드</kbd> `MetricZScoreFilter.__init__`

```python
__init__(
    name: 'str',
    window: 'int' = 30,
    threshold: 'Annotated | Annotated' = 3.0,
    change_dir: 'ChangeDir' = ANY
) → None
```

**인자:**

* `name` (str): 관측할 메트릭의 이름.
* `window` (int): 메트릭의 평균과 표준편차를 계산할 때 사용할 윈도우 크기.
* `threshold` (Union[Annotated, Annotated]): z-점수 임계값.
* `change_dir` (ChangeDir): 모니터링할 z-점수 변화 방향.

**반환값:**
`MetricZScoreFilter` 객체.
