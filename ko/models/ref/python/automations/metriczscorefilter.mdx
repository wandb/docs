## <kbd>class</kbd> `MetricZScoreFilter`

메트릭의 z-score를 사용자 정의 임계값과 비교하는 필터입니다.

### <kbd>메서드</kbd> `MetricZScoreFilter.__init__`

```python
__init__(
    name: 'str',
    window: 'int' = 30,
    threshold: 'Annotated | Annotated' = 3.0,
    change_dir: 'ChangeDir' = ANY
) → None
```

**인자:**

* `name` (str): 관측할 메트릭의 이름입니다.
* `window` (int): 메트릭의 평균과 표준편차를 계산할 때 사용할 윈도 크기입니다.
* `threshold` (Union[Annotated, Annotated]): z-점수(z-score)의 임계값입니다.
* `change_dir` (ChangeDir): 모니터링할 z-점수 변화 방향입니다.

**반환값:**
`MetricZScoreFilter` 객체를 반환합니다.
