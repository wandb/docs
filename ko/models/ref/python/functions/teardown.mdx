### <kbd>function</kbd> `teardown`

```python
teardown(exit_code: 'int | None' = None) → None
```

W&B 가 종료될 때까지 대기하고 리소스를 해제합니다.

`run.finish()` 를 사용하여 명시적으로 종료되지 않은 모든 Runs 를 완료하고 모든 데이터가 업로드될 때까지 대기합니다.

`wandb.setup()` 을 사용한 세션의 끝에서 이 함수를 호출하는 것을 권장합니다. `atexit` 훅 (hook) 에서 자동으로 호출되지만, 파이썬의 `multiprocessing` 모듈을 사용하는 경우와 같은 특정 설정에서는 동작이 보장되지 않을 수 있습니다.
