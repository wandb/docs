### <kbd>function</kbd> `sweep`

```python
sweep(
    sweep: Union[dict, Callable],
    entity: Optional[str] = None,
    project: Optional[str] = None,
    prior_runs: Optional[List[str]] = None
) → str
```

하이퍼파라미터 스윕을 초기화합니다.

머신러닝 모델의 비용 함수를 최적화하는 하이퍼파라미터를 찾기 위해 다양한 조합을 테스트합니다.

반환되는 고유 식별자인 `sweep_id`를 기록해 두십시오. 이후 단계에서 `sweep_id`를 스윕 에이전트에 전달합니다.

스윕을 정의하는 방법에 대한 정보는 [Sweep configuration structure](https://docs.wandb.ai/guides/sweeps/define-sweep-configuration)를 참조하십시오.

**Args:**

* `sweep`:  하이퍼파라미터 검색을 위한 설정(또는 설정 생성기)입니다.  callable을 제공하는 경우, 해당 callable이 인수를 받지 않고 W&amp;B sweep 설정 스펙을 따르는 사전을 반환하는지 확인하십시오.
* `entity`:  스윕으로 생성된 W&amp;B run을 전송할 사용자 이름 또는 팀 이름입니다.  지정한 entity가 이미 존재하는지 확인하십시오. entity를 지정하지 않으면 run은 기본 entity로 전송되며, 이는 보통 사용자 이름입니다.
* `project`:  스윕으로 생성된 W&amp;B run이 전송될 프로젝트 이름입니다. 프로젝트를 지정하지 않으면 run은 &#39;Uncategorized&#39; 라벨이 지정된 프로젝트로 전송됩니다.
* `prior_runs`:  이 스윕에 추가할 기존 run의 run ID들입니다.

**Returns:**

* `str`:  스윕에 대한 고유 식별자입니다.
