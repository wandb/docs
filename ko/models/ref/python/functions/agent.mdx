### <kbd>함수</kbd> `agent`

```python
agent(
    sweep_id: str,
    function: Optional[Callable] = None,
    entity: Optional[str] = None,
    project: Optional[str] = None,
    count: Optional[int] = None,
    forward_signals: bool = False
) → None
```

하나 이상의 sweep agent를 시작합니다.

sweep agent는 `sweep_id`를 사용해 자신이 속한 sweep이 무엇인지, 어떤 함수를 실행해야 하는지, 그리고 (선택적으로) 몇 개의 agent를 실행할지 결정합니다.

**Args:**

* `sweep_id`:  sweep에 대한 고유 식별자입니다. sweep ID는 W&amp;B CLI 또는 Python SDK에 의해 생성됩니다.
* `function`:  sweep 설정에서 지정된 &quot;program&quot; 대신 호출할 함수입니다.
* `entity`:  sweep에 의해 생성된 W&amp;B 실행을 전송할 사용자 이름 또는 팀 이름입니다. 지정한 엔터티가 이미 존재하는지 확인하십시오. 엔터티를 지정하지 않으면 실행은 기본 엔터티로 전송되며, 일반적으로 기본 엔터티는 사용자의 사용자 이름입니다.
* `project`:  sweep에서 생성된 W&amp;B 실행이 전송될 프로젝트 이름입니다. 프로젝트를 지정하지 않으면 실행은 &quot;Uncategorized&quot;라는 레이블의 프로젝트로 전송됩니다.
* `count`:  sweep 설정에서 시도할 trial 수입니다.
* `forward_signals`:  agent가 수신한 신호를 자식 프로세스에 전달할지 여부입니다. CLI agent에서만 지원됩니다.
