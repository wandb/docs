W&amp;B의 전역 함수는 `wandb.init()` 또는 `wandb.login()`처럼 직접 호출하는 최상위 함수입니다. 특정 클래스에 속한 메서드와 달리, 이러한 함수는 먼저 객체를 인스턴스화할 필요 없이 W&amp;B의 핵심 기능에 직접 액세스할 수 있게 해줍니다.

<div id="available-functions">
  ## 사용 가능한 함수
</div>

| 함수                                                        | 설명                                                                       |
| --------------------------------------------------------- | ------------------------------------------------------------------------ |
| [`init()`](/ko/models/ref/python/functions/init)             | W&amp;B에 추적하고 로깅하기 위해 새로운 런을 시작합니다. 보통 ML 트레이닝 파이프라인에서 가장 먼저 호출하는 함수입니다. |
| [`login()`](/ko/models/ref/python/functions/login)           | 플랫폼과 현재 머신을 인증하기 위해 W&amp;B 로그인 자격 증명을 설정합니다.                            |
| [`setup()`](/ko/models/ref/python/functions/setup)           | 현재 프로세스와 하위 프로세스에서 W&amp;B를 사용할 수 있도록 준비합니다. 멀티프로세스 애플리케이션에 유용합니다.       |
| [`teardown()`](/ko/models/ref/python/functions/teardown)     | W&amp;B 리소스를 정리하고 백엔드 프로세스를 종료합니다.                                       |
| [`sweep()`](/ko/models/ref/python/functions/sweep)           | 최적의 모델 설정을 검색하기 위해 하이퍼파라미터 스윕을 초기화합니다.                                   |
| [`agent()`](/ko/models/ref/python/functions/agent)           | 하이퍼파라미터 최적화 실험을 실행할 스윕 에이전트를 생성합니다.                                      |
| [`controller()`](/ko/models/ref/python/functions/controller) | 스윕 에이전트와 그 실행을 관리하고 제어합니다.                                               |
| [`restore()`](/ko/models/ref/python/functions/restore)       | 작업을 재개할 수 있도록 이전 런 또는 실험 상태를 복원합니다.                                      |
| [`finish()`](/ko/models/ref/python/functions/finish)         | 런을 종료하고 리소스를 정리합니다.                                                      |

<div id="example">
  ## 예시
</div>

가장 일반적인 워크플로는 W&amp;B에 인증하고 런을 초기화한 다음, 트레이닝 루프에서 정확도와 손실 같은 값을 로깅하는 것으로 시작합니다. 첫 단계는 `wandb`를 임포트하고 `login()` 및 `init()` 전역 함수를 사용하는 것입니다:

```python
import wandb

# W&B 인증
wandb.login()

# 하이퍼파라미터 및 메타데이터
config = {
   "learning_rate": 0.01,
   "epochs": 10,
}

# 런이 기록되는 프로젝트
project = "my-awesome-project"

# 새 런 초기화
with wandb.init(project=project, config=config) as run:
   # 트레이닝 코드를 여기에 작성하세요...
   
   # W&B에 값 로깅
   run.log({"accuracy": 0.9, "loss": 0.1})
```
