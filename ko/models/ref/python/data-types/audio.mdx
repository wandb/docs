## <kbd>class</kbd> `Audio`

오디오 클립을 위한 W&amp;B 클래스. 

### <kbd>방법</kbd> `Audio.__init__`

```python
__init__(
    data_or_path: 'str | pathlib.Path | list | np.ndarray',
    sample_rate: 'int | None' = None,
    caption: 'str | None' = None
)
```

오디오 파일 경로 또는 오디오 데이터가 담긴 NumPy 배열을 받습니다.

**매개변수:**

* `data_or_path`:  오디오 파일 경로 또는 오디오 데이터가 담긴 NumPy 배열입니다.
* `sample_rate`:  샘플 레이트입니다. 원시 NumPy 오디오 데이터 배열을 전달할 때 필수입니다.
* `caption`:  오디오와 함께 표시할 캡션입니다.

***

```python
durations(audio_list)
```
