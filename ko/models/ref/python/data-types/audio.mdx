## <kbd>class</kbd> `Audio`

오디오 클립을 나타내는 W&B 클래스입니다. 

### <kbd>method</kbd> `Audio.__init__`

```python
__init__(
    data_or_path: Union[str, pathlib.Path, list, ForwardRef('np.ndarray')],
    sample_rate: Optional[int] = None,
    caption: Optional[str] = None
)
```

오디오 파일 경로 또는 오디오 데이터가 담긴 NumPy 배열을 입력으로 받습니다.

**인자:**

* `data_or_path`:  오디오 파일의 경로 또는 오디오 데이터가 담긴 NumPy 배열.
* `sample_rate`:  샘플 레이트. 원시 NumPy 배열 형태의 오디오 데이터를 전달할 때 필요합니다.
* `caption`:  오디오와 함께 표시할 캡션.
