## <kbd>클래스</kbd> wandb.Html

```python
data: 'str | pathlib.Path | TextIO',
inject: 'bool' = True,
data_is_not_path: 'bool' = False
```

W&amp;B에 HTML 콘텐츠를 로깅하기 위한 W&amp;B 클래스입니다.

<div id="args">
  ## 인수
</div>

* `data`: 확장자가 &quot;.html&quot;인 파일 경로를 나타내는 문자열이거나, 리터럴 HTML이 들어 있는 문자열 또는 IO 객체입니다.
* `inject`: HTML 객체에 스타일시트를 추가합니다. False로 설정하면 HTML이 변경 없이 그대로 전달됩니다.
* `data_is_not_path`: False로 설정하면 `data`를 파일 경로로 처리합니다.
