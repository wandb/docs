## <kbd>클래스</kbd> wandb.Histogram

```python
sequence: 'Sequence | None' = None,
np_histogram: 'NumpyHistogram | None' = None,
num_bins: 'int' = 64
```

히스토그램용 W&amp;B 클래스입니다.

이 객체는 numpy의 histogram 함수와 동일한 방식으로 동작합니다
[https://docs.scipy.org/doc/numpy/reference/generated/numpy.histogram.html](https://docs.scipy.org/doc/numpy/reference/generated/numpy.histogram.html)

<div id="args">
  ## 인수
</div>

* `sequence`: 히스토그램의 입력 데이터입니다.
* `np_histogram`: 미리 계산된 히스토그램을 대체 입력으로 사용합니다.
* `num_bins`: 히스토그램의 bins 수입니다. 기본 bins 수는 64입니다. 최대 bins 수는 512입니다.

<div id="attributes">
  ## 속성
</div>

* `bins` (`[float]`): 각 bins의 경계
* `histogram` (`[int]`): 각 bin에 속하는 요소 수.
