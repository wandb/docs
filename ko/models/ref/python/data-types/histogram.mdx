## <kbd>class</kbd> `Histogram`

히스토그램을 위한 W&B 클래스입니다. 

이 객체는 NumPy의 `histogram` 함수와 동일하게 동작합니다. https://docs.scipy.org/doc/numpy/reference/generated/numpy.histogram.html 

**Attributes:**

- `bins` ([float]):  구간의 경계값
 - `histogram` ([int]):  각 구간에 포함되는 요소의 개수 

### <kbd>method</kbd> `Histogram.__init__`

```python
__init__(
    sequence: Optional[Sequence] = None,
    np_histogram: Optional[ForwardRef('NumpyHistogram')] = None,
    num_bins: int = 64
) → None
```

Histogram 객체를 초기화합니다.

**Args:**
sequence: 히스토그램을 생성할 입력 데이터입니다. np&#95;histogram: 미리 계산된 히스토그램을 나타내는 대체 입력입니다. num&#95;bins: 히스토그램의 bin 개수입니다. 기본 bin 개수는 64이고, 최대 bin 개수는 512입니다.

**Examples:**
시퀀스에서 히스토그램을 생성합니다.

```python
import wandb

wandb.Histogram([1, 2, 3])
```

np.histogram 결과로부터 효율적으로 초기화합니다.

```python
import numpy as np
import wandb

hist = np.histogram(data)
wandb.Histogram(np_histogram=hist)
```
