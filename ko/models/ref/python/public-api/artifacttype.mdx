## <kbd>class</kbd> `ArtifactType`

지정된 타입을 기준으로 하는 쿼리 조건을 만족하는 아티팩트 객체입니다. 

**Args:**

- `client`:  W&B에 대해 쿼리할 때 사용할 클라이언트 인스턴스. 
 - `entity`:  프로젝트를 소유한 엔터티(사용자 또는 팀). 
 - `project`:  아티팩트 타입을 쿼리할 프로젝트 이름. 
 - `type_name`:  아티팩트 타입 이름. 
 - `attrs`:  ArtifactType을 초기화할 때 사용할 선택적 속성 값입니다. 생략하면 객체는 초기화 시 W&B에서 해당 속성을 로드합니다. 

### <kbd>property</kbd> ArtifactType.id

아티팩트 타입의 고유 식별자입니다. 

**반환값:**

- `str`: id 속성 값입니다.

---

### <kbd>property</kbd> ArtifactType.name

Artifact 타입의 이름입니다. 

**Returns:**

- `str`: name 속성 값입니다.

---

### <kbd>메서드</kbd> `ArtifactType.collection`

```python
collection(name: 'str') → ArtifactCollection
```

이름을 기준으로 특정 아티팩트 컬렉션을 가져옵니다.

**Args:**

* `name` (str):  가져올 아티팩트 컬렉션의 이름입니다.

***


### <kbd>메서드</kbd> `ArtifactType.collections`

```python
collections(per_page: 'int' = 50) → ArtifactCollections
```

이 Artifact 타입과 관련된 모든 아티팩트 컬렉션을 가져옵니다.

**인수:**

* `per_page` (int):  페이지당 가져올 아티팩트 컬렉션의 개수입니다. 기본값은 50입니다.

***
