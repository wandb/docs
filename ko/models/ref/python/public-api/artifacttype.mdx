## <kbd>class</kbd> `ArtifactType`
지정된 타입에 따라 쿼리를 충족하는 Artifacts 오브젝트입니다.

**Args:**
 
 - `client`: W&B 쿼리에 사용할 클라이언트 인스턴스입니다.
 - `entity`: Projects 를 소유한 Entity (사용자 또는 팀)입니다.
 - `project`: Artifacts 타입을 쿼리할 Project 의 이름입니다.
 - `type_name`: Artifacts 타입의 이름입니다.
 - `attrs`: ArtifactType 을 초기화하기 위한 선택적 속성입니다. 생략 시, 오브젝트는 초기화 중에 W&B로부터 속성을 로드합니다.

### <kbd>property</kbd> ArtifactType.id

Artifacts 타입의 고유 식별자입니다.

**Returns:**
 - `str`: id 속성 값.
---

### <kbd>property</kbd> ArtifactType.name

Artifacts 타입의 이름입니다.

**Returns:**
 - `str`: name 속성 값.
---

### <kbd>method</kbd> `ArtifactType.collection`

```python
collection(name: 'str') → ArtifactCollection
```

이름으로 특정 아티팩트 컬렉션을 가져옵니다.

**Args:**
 
 - `name` (str): 가져올 아티팩트 컬렉션의 이름입니다.

---

### <kbd>method</kbd> `ArtifactType.collections`

```python
collections(per_page: 'int' = 50) → ArtifactCollections
```

이 Artifacts 타입과 관련된 모든 아티팩트 컬렉션을 가져옵니다.

**Args:**
 
 - `per_page` (int): 페이지당 가져올 아티팩트 컬렉션의 수입니다. 기본값은 50입니다.
