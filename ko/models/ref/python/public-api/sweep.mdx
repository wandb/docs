## <kbd>class</kbd> `Sweep`

Sweep과 연결된 실행 집합입니다. 

**Attributes:**

- `runs` (Runs):  실행 목록 
 - `id` (str):  Sweep ID 
 - `project` (str):  Sweep이 속한 프로젝트 이름 
 - `config` (dict):  Sweep 설정이 담긴 딕셔너리 
 - `state` (str):  Sweep의 상태. "Finished", "Failed", "Crashed", "Running" 중 하나일 수 있습니다. 
 - `expected_run_count` (int):  Sweep에 대해 예상되는 실행 수

### <kbd>property</kbd> Sweep.expected_run_count

sweep에서 예상되는 실행 횟수를 반환하며, 실행 횟수가 무한대인 경우에는 None을 반환합니다. 

**Returns:**

- `int | None`: expected_run_count 속성의 값.
---

### <kbd>property</kbd> Sweep.name

Sweep의 이름입니다. 

다음 우선순위에 따라 존재하는 첫 번째 이름을 반환합니다: 

1. 사용자가 수정한 표시 이름 2. 생성 시 설정된 이름 3. 스윕 ID 

---

### <kbd>property</kbd> Sweep.order

---

### <kbd>property</kbd> Sweep.path

프로젝트의 경로를 반환합니다. 

이 경로는 엔터티, 프로젝트 이름, 스윕 ID로 구성된 목록입니다. 

---

### <kbd>property</kbd> Sweep.url

스윕의 URL입니다. 

스윕 URL은 엔터티, 프로젝트, 문자열 "sweeps", 그리고 스윕 ID.run_id를 조합해 생성됩니다. SaaS 사용자의 경우 `https://wandb.ai/entity/project/sweeps/sweeps_ID` 형식을 따릅니다. 

---
