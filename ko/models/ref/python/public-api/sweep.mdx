<div id="attributes">
  ## 속성
</div>

### <kbd>속성</kbd> config

해당 스윕에 사용되는 설정입니다.

### <kbd>속성</kbd> entity

스윕에 연결된 entity입니다.

### <kbd>속성</kbd> expected_run_count

스윕의 예상 run 수를 반환합니다. run 수가 무한한 경우 `None`을 반환합니다.

### <kbd>속성</kbd> name

스윕의 이름입니다.

다음 우선순위에 따라 가장 먼저 존재하는 이름을 반환합니다:

1. 사용자가 수정한 표시 이름
2. 생성 시 설정한 이름
3. 스윕 ID

### <kbd>속성</kbd> order

스윕의 순서 키를 반환합니다.

### <kbd>속성</kbd> path

프로젝트 경로를 반환합니다.

이 경로는 entity, 프로젝트 이름, 스윕 ID로 구성된 목록입니다.

### <kbd>속성</kbd> url

스윕의 URL입니다.

스윕 URL은 entity, 프로젝트, &quot;sweeps&quot;라는 문자열, 그리고 스윕 ID.run&#95;id를 조합해 생성됩니다. SaaS 사용자의 경우 `https://wandb.ai/entity/project/sweeps/sweeps_ID` 형식을 사용합니다.

### <kbd>속성</kbd> username

사용 중단되었습니다. 대신 `Sweep.entity`를 사용하세요.
