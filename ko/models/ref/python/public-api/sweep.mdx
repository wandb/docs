## <kbd>class</kbd> `Sweep`

스윕과 연결된 run들의 집합입니다.

**속성:**

* `runs` (Runs):  run 목록
* `id` (str):  스윕 ID
* `project` (str):  스윕이 속한 프로젝트의 이름
* `config` (dict):  스윕 설정을 포함하는 딕셔너리
* `state` (str):  스윕의 상태. &quot;Finished&quot;, &quot;Failed&quot;, &quot;Crashed&quot;, &quot;Running&quot; 중 하나일 수 있습니다.
* `expected_run_count` (int):  스윕에 대해 예상되는 run 수

### <kbd>property</kbd> Sweep.config

***

### <kbd>property</kbd> Sweep.entity

스윕과 연결된 entity입니다.

**반환값:**

* `str`: entity 속성 값입니다.

***

### <kbd>property</kbd> Sweep.expected_run_count

스윕에서 예상되는 run 수를 반환하며, run 수가 무한대인 경우 None을 반환합니다.

**반환값:**

* `int | None`: expected&#95;run&#95;count 속성 값.

***

### <kbd>property</kbd> Sweep.name

스윕의 이름입니다.

다음 우선순위에 따라 존재하는 첫 번째 이름을 반환합니다:

1. 사용자가 편집한 표시 이름 2. 생성 시 설정된 이름 3. Sweep ID

***

### <kbd>property</kbd> Sweep.order

스윕의 순서를 나타내는 키를 반환합니다.

***

### <kbd>property</kbd> Sweep.path

프로젝트의 경로를 반환합니다.

이 경로는 entity, 프로젝트 이름, 스윕 ID로 구성된 목록입니다.

***

### <kbd>property</kbd> Sweep.url

스윕의 URL입니다.

스윕 URL은 entity, 프로젝트, &quot;sweeps&quot;라는 용어, 그리고 스윕 ID.run&#95;id로부터 생성됩니다. SaaS 사용자의 경우 `https://wandb.ai/entity/project/sweeps/sweeps_ID` 형태가 됩니다.

***

### <kbd>property</kbd> Sweep.username

사용 중단되었습니다. 대신 `Sweep.entity`를 사용하세요.

**반환값:**

* `str`: `username` 속성 값입니다.
