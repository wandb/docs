## <kbd>class</kbd> `Sweeps`

`Sweep` 객체 컬렉션을 순회하는 lazy 이터레이터입니다.

**예시:**

### <kbd>방법</kbd> `Sweeps.__init__`

```python
__init__(
    client: 'RetryingClient',
    entity: 'str',
    project: 'str',
    per_page: 'int' = 50
) → Sweeps
```

반복 가능한 `Sweep` 객체 컬렉션입니다.

**매개변수:**

* `client`:  W&amp;B를 쿼리하는 데 사용하는 API 클라이언트입니다.
* `entity`:  Sweep를 소유한 entity입니다.
* `project`:  Sweep가 포함된 프로젝트입니다.
* `per_page`:  API에 대한 각 요청마다 가져올 Sweep 수입니다.
