## <kbd>class</kbd> `Sweeps`

`Sweep` 객체 컬렉션에 대한 지연(lazy) 반복자입니다.

**예제:**

```python
from wandb.apis.public import Api

sweeps = Api().project(name="project_name", entity="entity").sweeps()

# sweep을 반복하며 세부 정보 출력
for sweep in sweeps:
     print(f"Sweep name: {sweep.name}")
     print(f"Sweep ID: {sweep.id}")
     print(f"Sweep URL: {sweep.url}")
     print("----------")
```


### <kbd>method</kbd> `Sweeps.__init__`

```python
__init__(
    client: 'RetryingClient',
    entity: 'str',
    project: 'str',
    per_page: 'int' = 50
) → Sweeps
```

반복 가능한 `Sweep` 객체 컬렉션입니다.

**Args:**

* `client`:  W&amp;B를 쿼리하는 데 사용되는 API 클라이언트입니다.
* `entity`:  Sweep을 소유하는 엔터티입니다.
* `project`:  Sweep을 포함하는 프로젝트입니다.
* `per_page`:  API 요청당 가져올 Sweep 개수입니다.
