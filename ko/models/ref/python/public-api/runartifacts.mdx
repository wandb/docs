## <kbd>class</kbd> `RunArtifacts`

특정 run과 연결된 artifact로 이루어진 순회 가능한 collection입니다. 

### <kbd>속성</kbd> RunArtifacts.cursor

다음에 가져올 페이지의 시작 위치를 나타내는 불투명한 커서입니다.

이 값은 저장해 두었다가 이후의 페이지네이션된 쿼리에서 `start=`로 전달하면, 이 paginator가 멈춘 지점부터 반복을 재개할 수 있습니다.

**반환값:**

* `str | None`: 커서 속성 값입니다.
