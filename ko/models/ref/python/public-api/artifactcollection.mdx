## <kbd>class</kbd> `ArtifactCollection`

관련된 아티팩트들의 집합을 나타내는 아티팩트 컬렉션입니다. 

**Args:**

- `client`:  W&B에 쿼리할 때 사용할 클라이언트 인스턴스입니다. 
 - `entity`:  프로젝트를 소유하는 엔터티(사용자 또는 팀)입니다. 
 - `project`:  아티팩트 컬렉션을 조회할 프로젝트 이름입니다. 
 - `name`:  아티팩트 컬렉션의 이름입니다. 
 - `type`:  아티팩트 컬렉션의 유형입니다(예: "dataset", "model"). 
 - `organization`:  해당되는 경우의 선택적 조직 이름입니다. 
 - `attrs`:  아티팩트 컬렉션을 초기화하기 위한 선택적 속성 매핑입니다. 제공되지 않은 경우 객체는 초기화 시점에 W&B에서 속성을 로드합니다. 

### <kbd>property</kbd> ArtifactCollection.aliases

이 컬렉션에 포함된 모든 아티팩트 버전의 별칭(alias)입니다. 

**반환값:**

- `list[str]`: `aliases` 프로퍼티 값입니다.
