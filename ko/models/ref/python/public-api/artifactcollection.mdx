## <kbd>class</kbd> `ArtifactCollection`

관련된 artifact를 그룹으로 나타내는 artifact collection입니다.

**매개변수:**

* `client`:  W&amp;B를 쿼리하는 데 사용할 클라이언트 인스턴스입니다.
* `entity`:  프로젝트를 소유한 entity(사용자 또는 팀)입니다.
* `project`:  artifact collection을 쿼리할 프로젝트 이름입니다.
* `name`:  artifact collection의 이름입니다.
* `type`:  artifact collection의 유형입니다(예: &quot;dataset&quot;, &quot;model&quot;).
* `organization`:  해당하는 경우 조직 이름(선택 사항)입니다.
* `attrs`:  artifact collection을 초기화하는 속성 매핑(선택 사항)입니다. 제공하지 않으면 초기화 시 객체가 W&amp;B에서 해당 속성을 로드합니다.

### <kbd>방법</kbd> `ArtifactCollection.artifacts`

```python
artifacts(per_page: 'int' = 50, start: 'str | None' = None) → Artifacts
```

컬렉션에 있는 모든 아티팩트를 조회합니다.

**인수:**

* `per_page`:  페이지당 가져올 아티팩트 수입니다.
* `start`:  이전 paginator의 `.cursor` 속성에서 가져온, 이전 쿼리를 다시 시작하기 위한 페이지네이션 커서입니다.

***
