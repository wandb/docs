## <kbd>class</kbd> `ArtifactCollection`

서로 관련된 아티팩트 그룹을 나타내는 아티팩트 컬렉션입니다. 

**Args:**

- `client`:  W&B에 쿼리할 때 사용할 클라이언트 인스턴스입니다. 
- `entity`:  프로젝트를 소유한 엔터티(사용자 또는 팀)입니다. 
- `project`:  아티팩트 컬렉션을 쿼리할 프로젝트 이름입니다. 
- `name`:  아티팩트 컬렉션의 이름입니다. 
- `type`:  아티팩트 컬렉션의 유형입니다(예: "dataset", "model"). 
- `organization`:  해당되는 경우 사용할 선택적인 organization 이름입니다. 
- `attrs`:  아티팩트 컬렉션을 초기화하기 위한 속성 매핑(선택 사항)입니다. 제공되지 않은 경우, 객체는 초기화 시점에 W&B에서 속성을 로드합니다.
