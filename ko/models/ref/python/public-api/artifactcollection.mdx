## <kbd>class</kbd> `ArtifactCollection`
관련된 Artifacts 그룹을 나타내는 아티팩트 컬렉션입니다.

**Args:**
 
 - `client`: W&B 쿼리에 사용할 클라이언트 인스턴스입니다.
 - `entity`: Projects 를 소유한 엔티티 (사용자 또는 팀)입니다.
 - `project`: 아티팩트 컬렉션을 쿼리할 프로젝트의 이름입니다.
 - `name`: 아티팩트 컬렉션의 이름입니다.
 - `type`: 아티팩트 컬렉션의 유형 (예: "dataset", "model")입니다.
 - `organization`: 해당되는 경우 선택적인 조직 이름입니다.
 - `attrs`: 아티팩트 컬렉션을 초기화하기 위한 속성 매핑 (선택 사항)입니다. 제공되지 않으면 초기화 시 W&B에서 속성을 로드합니다.
