## <kbd>class</kbd> `ArtifactCollections`

프로젝트 내 특정 유형의 아티팩트 컬렉션을 나타냅니다. 

**인자:**

- `client`:  W&B에 쿼리할 때 사용할 클라이언트 인스턴스입니다. 
 - `entity`:  프로젝트를 소유한 엔터티(사용자 또는 팀)입니다. 
 - `project`:  아티팩트 컬렉션을 쿼리할 프로젝트의 이름입니다. 
 - `type_name`:  컬렉션을 가져올 아티팩트 유형의 이름입니다. 
 - `per_page`:  페이지당 가져올 아티팩트 컬렉션의 개수입니다. 기본값은 50입니다.
