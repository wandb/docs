## <kbd>class</kbd> `IncompleteRunHistoryError`

run 히스토리가 완전하지 않을 때 발생합니다. 

불완전한 히스토리는 일부 데이터가 아직 parquet 파일로 내보내지 않았을 때 발생합니다. 보통 실행 중인 run 때문에 발생합니다.
