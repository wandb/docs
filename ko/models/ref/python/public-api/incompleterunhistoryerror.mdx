## <kbd>class</kbd> `IncompleteRunHistoryError`

실행 이력이 완전하지 않을 때 발생합니다. 

불완전한 이력은 일부 데이터가 아직 parquet 파일로 내보내지지 않았을 때 발생합니다. 보통 실행이 아직 진행 중일 때입니다.
