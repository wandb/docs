## <kbd>클래스</kbd> wandb.apis.public.Team

```python
(service_api: 'ServiceApi',
name: 'str',
attrs: 'Mapping[str, Any] | None' = None)
```

W&amp;B 팀을 나타내는 클래스입니다.

이 클래스는 팀 생성, 구성원 초대, 서비스 계정 관리 등을 포함해 W&amp;B 팀을 관리하기 위한 메서드를 제공합니다. 팀 속성 관리를 위해 Attrs를 상속합니다.

<div id="args">
  ## 인수
</div>

* `service_api`: W&amp;B를 쿼리하는 데 사용할 서비스 API 인스턴스
* `name`: 팀 이름
* `attrs`: 선택적인 팀 속성 딕셔너리

<div id="examples">
  ## 예시
</div>

팀을 찾아보고 구성원을 초대하세요.

```python
import wandb

api = wandb.Api()
team = api.team("my-team")
team.invite("user@example.com")
```

팀을 만들고 서비스 계정을 추가하세요.

```python
import wandb

api = wandb.Api()
team = api.create_team("my-team")
team.create_service_account("CI service account")
```

### <kbd>메서드</kbd> Team.create()

```python
api: 'Api',
team: 'str',
admin_username: 'str | None' = None
```

새 팀을 생성합니다.

<div id="arguments">
  ##### 인수
</div>

* `api`: (`Api`) 사용할 API 인스턴스
* `team`: (str) 팀 이름
* `admin_username`: (str) 선택 사항. 팀 관리자 사용자 이름으로, 기본값은 현재 사용자입니다.

### <kbd>메서드</kbd> Team.create_service_account()

```python
self,
description: 'str'
```

팀용 서비스 계정을 생성합니다.

<div id="arguments">
  ##### 인수
</div>

* `description`: (str) 이 서비스 계정에 대한 설명

### <kbd>메서드</kbd> Team.invite()

```python
self,
username_or_email: 'str',
admin: 'bool' = False
```

팀에 사용자를 초대합니다.

<div id="arguments">
  ##### 인수
</div>

* `username_or_email`: (str) 초대할 사용자의 사용자 이름 또는 이메일 주소.
* `admin`: (bool) 이 사용자를 팀 관리자(admin)로 지정할지 여부. 기본값은 `False`입니다.
