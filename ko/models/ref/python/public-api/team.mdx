## <kbd>class</kbd> `Team`
W&B Teams 를 나타내는 클래스입니다.

이 클래스는 Teams 생성, 멤버 초대, 서비스 계정 관리 등 W&B Teams 를 관리하기 위한 메소드를 제공합니다. 팀 속성을 처리하기 위해 Attrs를 상속합니다.

### <kbd>method</kbd> `Team.__init__`

```python
__init__(
    client: 'RetryingClient',
    name: 'str',
    attrs: 'Mapping[str, Any] | None' = None
)
```

**Args:**

 - `client` (`wandb.apis.public.Api`): 사용할 api 인스턴스
 - `name` (str): 팀의 이름
 - `attrs` (dict): 팀 속성이 담긴 선택적 사전 (dictionary)

**Note:**

> Team 관리를 위해서는 적절한 권한이 필요합니다.

### <kbd>method</kbd> `Team.invite`

```python
invite(username_or_email: 'str', admin: 'bool' = False) → bool
```

Teams 에 사용자를 초대합니다.

**Args:**

 - `username_or_email`: (str) 초대하려는 사용자의 사용자 이름 또는 이메일 주소.
 - `admin`: (bool) 이 사용자를 팀 관리자로 설정할지 여부. 기본값은 `False` 입니다.

**Returns:**
 성공 시 `True`, 사용자가 이미 초대되었거나 존재하지 않는 경우 `False`를 반환합니다.
