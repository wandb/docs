## <kbd>class</kbd> `Project`
Project는 Runs를 위한 네임스페이스입니다.

### <kbd>method</kbd> `Project.__init__`

```python
__init__(
    client: 'RetryingClient',
    entity: 'str',
    project: 'str',
    attrs: 'Mapping[str, Any]'
) → Project
```

**Args:**
 
 - `client`: W&B API 클라이언트 인스턴스. 
 - `name` (str): 프로젝트의 이름. 
 - `entity` (str): 프로젝트를 소유한 엔티티 이름. 

엔티티와 연결된 단일 프로젝트입니다. 

**Args:**
 
 - `client`: W&B 쿼리에 사용되는 API 클라이언트. 
 - `entity`: 프로젝트를 소유한 엔티티. 
 - `project`: 쿼리할 프로젝트의 이름. 
 - `attrs`: 프로젝트의 속성.
