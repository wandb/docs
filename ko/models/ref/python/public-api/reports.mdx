## <kbd>class</kbd> `Reports`

Reports는 `BetaReport` 객체를 순회하는 lazy 이터레이터입니다.

### <kbd>방법</kbd> `Reports.__init__`

```python
__init__(
    client: 'RetryingClient',
    project: 'Project',
    name: 'str | None' = None,
    entity: 'str | None' = None,
    per_page: 'int' = 50
)
```

**매개변수:**

* `client` (`wandb.apis.internal.Api`):  사용할 API 클라이언트 인스턴스입니다.
* `project` (`wandb.sdk.internal.Project`):  Reports를 가져올 프로젝트입니다.
* `name` (str, 선택):  필터링할 Reports의 이름입니다. `None`이면 모든 Reports를 가져옵니다.
* `entity` (str, 선택):  프로젝트의 entity 이름입니다. 기본값은 프로젝트의 entity입니다.
* `per_page` (int):  페이지당 가져올 Reports 수입니다(기본값: 50).
