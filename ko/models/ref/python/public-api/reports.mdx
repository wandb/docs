## <kbd>class</kbd> `Reports`

Reports는 `BetaReport` 객체들의 지연(lazy) 이터레이터입니다.

### <kbd>메서드</kbd> `Reports.__init__`

```python
__init__(
    client: 'RetryingClient',
    project: 'Project',
    name: 'str | None' = None,
    entity: 'str | None' = None,
    per_page: 'int' = 50
)
```

**매개변수:**

* `client` (`wandb.apis.internal.Api`):  사용할 API 클라이언트 인스턴스입니다.
* `project` (`wandb.sdk.internal.Project`):  리포트를 가져올 프로젝트입니다.
* `name` (str, 선택적):  필터링할 리포트 이름입니다. `None`인 경우 모든 리포트를 가져옵니다.
* `entity` (str, 선택적):  프로젝트의 엔터티 이름입니다. 지정하지 않으면 해당 프로젝트의 엔터티가 사용됩니다.
* `per_page` (int):  페이지당 가져올 리포트 개수입니다(기본값은 50).
