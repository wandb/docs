## <kbd>class</kbd> `ArtifactFiles`

아티팩트 내 파일을 위한 페이지네이터입니다. 

### <kbd>속성</kbd> ArtifactFiles.cursor

다음에 가져올 페이지의 시작 위치를 나타내는 불투명한 커서입니다.

이 값은 저장해 두었다가 나중에 페이지가 매겨진 쿼리에서 `start=`로 전달하여, 이 페이지네이터가 멈춘 지점부터 반복을 재개할 수 있습니다.

**반환값:**

* `str | None`: 커서 속성 값입니다.
