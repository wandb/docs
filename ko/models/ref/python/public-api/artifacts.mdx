## <kbd>class</kbd> `Artifacts`

프로젝트와 연결된 아티팩트 버전들의 반복 가능한 컬렉션입니다. 

선택적으로 필터를 전달하여 특정 기준에 따라 결과를 좁힐 수 있습니다. 

**인자(Args):**

- `client`: W&B에 쿼리를 수행할 때 사용할 클라이언트 인스턴스.
- `entity`: 프로젝트를 소유한 엔터티(사용자 또는 팀).
- `project`: 아티팩트를 쿼리할 프로젝트의 이름.
- `collection_name`: 쿼리할 아티팩트 컬렉션의 이름.
- `type`: 쿼리할 아티팩트의 유형. 일반적인 예로는 "dataset" 또는 "model"이 있습니다.
- `filters`: 쿼리에 적용할 선택적 필터 매핑.
- `order`: 결과 정렬 순서를 지정하는 선택적 문자열.
- `per_page`: 페이지당 가져올 아티팩트 버전 개수. 기본값은 50.
- `tags`: 태그로 아티팩트를 필터링하기 위한 선택적 문자열 또는 문자열 목록. 

### <kbd>속성</kbd> Artifacts.cursor

다음에 가져올 페이지의 시작 지점을 나타내는 불투명한 커서입니다.

이 값은 저장해 두었다가 나중에 페이지가 매겨진 쿼리에서 `start=`로 전달하여, 이 페이지네이터가 멈춘 지점부터 반복을 재개하는 데 사용할 수 있습니다.

**반환값:**

* `str | None`: 커서 속성 값입니다.
