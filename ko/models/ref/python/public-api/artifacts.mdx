## <kbd>class</kbd> `Artifacts`

프로젝트와 연결된 아티팩트 버전의 이터러블 컬렉션입니다. 

필터를 선택적으로 전달해 특정 기준에 따라 결과를 좁힐 수 있습니다. 

**Args:**

- `client`:  W&B에 쿼리할 때 사용할 클라이언트 인스턴스. 
 - `entity`:  프로젝트를 소유한 엔터티(사용자 또는 팀). 
 - `project`:  아티팩트를 조회할 프로젝트의 이름. 
 - `collection_name`:  조회할 아티팩트 컬렉션의 이름. 
 - `type`:  조회할 아티팩트의 유형. 일반적인 예로는 "dataset"(데이터셋) 또는 "model"(모델)이 있습니다. 
 - `filters`:  쿼리에 적용할 선택적 필터 매핑. 
 - `order`:  결과의 정렬 순서를 지정하는 선택적 문자열. 
 - `per_page`:  한 페이지당 가져올 아티팩트 버전 수. 기본값은 50. 
 - `tags`:  태그로 아티팩트를 필터링하기 위한 선택적 문자열 또는 문자열 리스트(목록).
