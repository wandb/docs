### <kbd>함수</kbd> `scatter`

```python
scatter(
    table: 'wandb.Table',
    x: 'str',
    y: 'str',
    title: 'str' = '',
    split_table: 'bool' = False
) → CustomChart
```

`wandb.Table`의 데이터로 산점도를 만듭니다.

**매개변수:**

* `table`:  시각화할 데이터가 들어 있는 W&amp;B Table입니다.
* `x`:  x축에 사용할 열 이름입니다.
* `y`:  y축에 사용할 열 이름입니다.
* `title`:  산점도 차트의 제목입니다.
* `split_table`:  테이블을 W&amp;B UI의 별도 섹션으로 분리할지 여부입니다. `True`이면 테이블이 &quot;Custom Chart Tables&quot;라는 이름의 섹션에 표시됩니다. 기본값은 `False`입니다.

**반환값:**

* `CustomChart`:  W&amp;B에 로깅할 수 있는 맞춤형 차트 객체입니다. 차트를 로깅하려면 `wandb.log()`에 전달합니다.
