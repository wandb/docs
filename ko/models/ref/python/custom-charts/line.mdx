### <kbd>function</kbd> `line`

```python
line(
    table: 'wandb.Table',
    x: 'str',
    y: 'str',
    stroke: 'str | None' = None,
    title: 'str' = '',
    split_table: 'bool' = False
) → CustomChart
```

사용자 지정 가능한 선형 차트를 생성합니다.

**Args:**

* `table`:   차트에 사용할 데이터가 포함된 테이블.
* `x`:  x축 값에 사용할 열 이름.
* `y`:  y축 값에 사용할 열 이름.
* `stroke`:  선을 구분하기 위한 값(예: 여러 선을 그룹화할 때)으로 사용할 열 이름.
* `title`:  차트의 제목.
* `split_table`:  테이블을 W&amp;B UI에서 별도의 섹션으로 분리할지 여부. `True`인 경우 테이블은 &quot;Custom Chart Tables&quot;라는 이름의 섹션에 표시됩니다. 기본값은 `False`입니다.

**Returns:**

* `CustomChart`:  W&amp;B에 로깅할 수 있는 사용자 지정 차트 객체. 차트를 로깅하려면 `wandb.log()`에 전달합니다.

**Example:**

```python
import math
import random
import wandb

# 다양한 패턴의 여러 시리즈 데이터 생성
data = []
for i in range(100):
     # 시리즈 1: 랜덤 노이즈가 포함된 사인파 패턴
     data.append([i, math.sin(i / 10) + random.uniform(-0.1, 0.1), "series_1"])
     # 시리즈 2: 랜덤 노이즈가 포함된 코사인 패턴
     data.append([i, math.cos(i / 10) + random.uniform(-0.1, 0.1), "series_2"])
     # 시리즈 3: 랜덤 노이즈가 포함된 선형 증가
     data.append([i, i / 10 + random.uniform(-0.5, 0.5), "series_3"])

# 테이블의 열 정의
table = wandb.Table(data=data, columns=["step", "value", "series"])

# wandb 실행 초기화 및 선 차트 기록
with wandb.init(project="line_chart_example") as run:
     line_chart = wandb.plot.line(
         table=table,
         x="step",
         y="value",
         stroke="series",  # "series" 열 기준으로 그룹화
         title="Multi-Series Line Plot",
     )
     run.log({"line-chart": line_chart})
```
