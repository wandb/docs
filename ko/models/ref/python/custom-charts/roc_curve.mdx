### <kbd>함수</kbd> `roc_curve`

```python
roc_curve(
    y_true: 'Sequence[numbers.Number]',
    y_probas: 'Sequence[Sequence[float]] | None' = None,
    labels: 'list[str] | None' = None,
    classes_to_plot: 'list[numbers.Number] | None' = None,
    title: 'str' = 'ROC Curve',
    split_table: 'bool' = False
) → CustomChart
```

Receiver Operating Characteristic (ROC) 곡선 chart를 생성합니다.

**매개변수:**

* `y_true`:  대상 변수의 실제 클래스 레이블(ground truth)입니다. shape는 (num&#95;samples,)이어야 합니다.
* `y_probas`:  각 클래스의 예측 확률 또는 결정 점수입니다. shape는 (num&#95;samples, num&#95;classes)여야 합니다.
* `labels`:  `y_true`의 클래스 인덱스에 대응하는 사람이 읽기 쉬운 레이블입니다. 예를 들어 `labels=['dog', 'cat']`이면 plot에서 클래스 0은 &#39;dog&#39;, 클래스 1은 &#39;cat&#39;으로 표시됩니다. None이면 `y_true`의 원래 클래스 인덱스를 사용합니다. 기본값은 None입니다.
* `classes_to_plot`:  ROC 곡선에 포함할 고유 클래스 레이블의 하위 집합입니다. None이면 `y_true`의 모든 클래스가 plot됩니다. 기본값은 None입니다.
* `title`:  ROC 곡선 plot의 제목입니다. 기본값은 &quot;ROC Curve&quot;입니다.
* `split_table`:  테이블을 W&amp;B UI의 별도 섹션으로 분리할지 여부입니다. `True`이면 테이블이 &quot;Custom Chart Tables&quot;라는 이름의 섹션에 표시됩니다. 기본값은 `False`입니다.

**반환값:**

* `CustomChart`:  W&amp;B에 로깅할 수 있는 맞춤형 chart 객체입니다. chart를 로깅하려면 `wandb.log()`에 전달하세요.

**예외:**

* `wandb.Error`:  numpy, pandas 또는 scikit-learn을 찾을 수 없는 경우입니다.
