### <kbd>function</kbd> `histogram`

```python
histogram(
    table: 'wandb.Table',
    value: 'str',
    title: 'str' = '',
    split_table: 'bool' = False
) → CustomChart
```

W&amp;B Table에서 히스토그램 차트를 생성합니다.

**Args:**

* `table`:  히스토그램에 사용할 데이터를 포함하는 W&amp;B Table입니다.
* `value`:  구간 축(x축)의 레이블입니다.
* `title`:  히스토그램 플롯의 제목입니다.
* `split_table`:  이 테이블을 W&amp;B UI에서 별도의 섹션으로 분리할지 여부입니다. `True`인 경우 테이블은 &quot;Custom Chart Tables&quot;라는 섹션에 표시됩니다. 기본값은 `False`입니다.

**Returns:**

* `CustomChart`:  W&amp;B에 로그할 수 있는 커스텀 차트 객체입니다. 차트를 로그하려면 이 객체를 `wandb.log()`에 전달합니다.

**Example:**

```python
import math
import random
import wandb

# 랜덤 데이터 생성
data = [[i, random.random() + math.sin(i / 10)] for i in range(100)]

# W&B Table 생성
table = wandb.Table(
    data=data,
    columns=["step", "height"],
)

# 히스토그램 플롯 생성
histogram = wandb.plot.histogram(
    table,
    value="height",
    title="My Histogram",
)

# W&B에 히스토그램 플롯 기록
with wandb.init(...) as run:
    run.log({"histogram-plot1": histogram})
```
