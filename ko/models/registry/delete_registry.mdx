---
title: 레지스트리 삭제
description: "Python SDK 또는 W&B App UI를 사용하여 Team admin 또는 레지스트리 관리자로서 W&B Registry를 삭제합니다."
---

이 페이지에서는 Team admin 또는 레지스트리 관리자가 레지스트리를 삭제하는 방법을 설명합니다.

* Team admin은 조직의 모든 레지스트리를 삭제할 수 있습니다.
* 레지스트리 관리자는 자신이 생성한 레지스트리를 삭제할 수 있습니다.

레지스트리를 삭제하면 해당 레지스트리에 속한 컬렉션도 함께 삭제되지만, 레지스트리에 연결된 아티팩트는 삭제되지 않습니다. 해당 아티팩트는 아티팩트가 로깅된 원래 프로젝트에 그대로 남아 있습니다.

<Tabs>
  <Tab title="Python SDK">
    `wandb` API의 `delete()` 메서드를 사용하여 프로그래밍 방식으로 레지스트리를 삭제할 수 있습니다. 다음 예제에서는 다음 방법을 보여 줍니다.

    1. `api.registry()`를 사용해 삭제하려는 레지스트리를 가져옵니다.
    2. 반환된 레지스트리 객체에서 `delete()` 메서드를 호출해 레지스트리를 삭제합니다.

    ```python
    import wandb

    # W&B API를 초기화합니다
    api = wandb.Api()

    # 삭제하려는 레지스트리를 가져옵니다
    fetched_registry = api.registry("<registry_name>")

    # 레지스트리를 삭제합니다
    fetched_registry.delete()
    ```
  </Tab>

  <Tab title="W&B App">
    1. https://wandb.ai/registry/의 W&amp;B Registry로 이동합니다.
    2. 삭제하려는 레지스트리를 선택합니다.
    3. 오른쪽 상단의 톱니바퀴 아이콘을 클릭해 레지스트리의 Settings를 확인합니다.
    4. 레지스트리를 삭제하려면 Settings 페이지 오른쪽 상단의 휴지통 아이콘을 클릭합니다.
    5. 표시되는 모달에 삭제할 레지스트리의 이름을 입력해 확인한 다음, **Delete**를 클릭합니다.
  </Tab>
</Tabs>
