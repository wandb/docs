---
title: 레지스트리 삭제
---

이 페이지에서는 Team 관리자 또는 Registry 관리자가 레지스트리를 삭제하는 방법을 설명합니다.

- Team 관리자는 조직 내의 모든 레지스트리를 삭제할 수 있습니다.
- Registry 관리자는 자신이 생성한 레지스트리를 삭제할 수 있습니다.

레지스트리를 삭제하면 해당 레지스트리에 속한 컬렉션도 함께 삭제되지만, 레지스트리에 연결된 Artifacts는 삭제되지 않습니다. 이러한 아티팩트는 원래 로그된 프로젝트에 그대로 유지됩니다.

<Tabs>
<Tab title="Python SDK">
프로그래밍 방식으로 레지스트리를 삭제하려면 `wandb` API의 `delete()` 메소드를 사용하세요. 다음 예제는 그 방법을 보여줍니다.

1. `api.registry()`를 사용하여 삭제하려는 레지스트리를 가져옵니다.
1. 반환된 레지스트리 오브젝트에서 `delete()` 메소드를 호출하여 레지스트리를 삭제합니다.

```python
import wandb

# W&B API 초기화
api = wandb.Api()

# 삭제하려는 레지스트리 가져오기
fetched_registry = api.registry("<registry_name>")

# 레지스트리 삭제하기
fetched_registry.delete()
```
</Tab>
<Tab title="W&B App">
1. https://wandb.ai/registry/ 에서 W&B Registry로 이동합니다.
2. 삭제하려는 레지스트리를 선택합니다.
3. 오른쪽 상단에 있는 톱니바퀴 아이콘을 클릭하여 레지스트리의 설정( settings )을 확인합니다.
4. 레지스트리를 삭제하려면 설정 페이지 오른쪽 상단의 휴지통 아이콘을 클릭합니다.
5. 나타나는 모달창에 삭제할 레지스트리 이름을 입력하여 확인한 다음, **Delete**를 클릭합니다.
</Tab>
</Tabs>
