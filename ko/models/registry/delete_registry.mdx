---
title: 레지스트리 삭제
description: "Python SDK 또는 W&B App UI를 사용하여 팀 관리자 또는 레지스트리 관리자로서 W&B Registry를 삭제합니다."
---

이 페이지에서는 팀 관리자 또는 레지스트리 관리자가 레지스트리를 삭제하는 방법을 설명합니다.

* 팀 관리자는 조직의 모든 레지스트리를 삭제할 수 있습니다.
* 레지스트리 관리자는 자신이 생성한 레지스트리만 삭제할 수 있습니다.

레지스트리를 삭제하면 해당 레지스트리에 속한 컬렉션도 함께 삭제되지만, 레지스트리에 연결된 아티팩트는 삭제되지 않습니다. 이러한 아티팩트는 아티팩트가 로깅된 원래 프로젝트에 그대로 남습니다.

<Tabs>
  <Tab title="Python SDK">
    `wandb` API의 `delete()` 메서드를 사용해 프로그래밍 방식으로 레지스트리를 삭제하세요. 다음 예제는 다음 단계를 보여줍니다:

    1. `api.registry()`로 삭제하려는 레지스트리를 가져옵니다.
    2. 반환된 레지스트리 객체에서 `delete()` 메서드를 호출해 레지스트리를 삭제합니다.

    ```python
    import wandb

    # W&B API 초기화
    api = wandb.Api()

    # 삭제하려는 레지스트리 가져오기
    fetched_registry = api.registry("<registry_name>")

    # 레지스트리 삭제
    fetched_registry.delete()
    ```
  </Tab>

  <Tab title="W&B App">
    1. W&amp;B Registry(https://wandb.ai/registry/)로 이동하세요.
    2. 삭제하려는 레지스트리를 선택하세요.
    3. 오른쪽 상단의 톱니바퀴 아이콘을 클릭하여 레지스트리 설정 페이지를 엽니다.
    4. 레지스트리를 삭제하려면 설정 페이지 오른쪽 상단의 휴지통 아이콘을 클릭하세요.
    5. 표시되는 모달 창에서 레지스트리 이름을 입력하여 삭제를 확인한 다음 **Delete**를 클릭하세요.
  </Tab>
</Tabs>
