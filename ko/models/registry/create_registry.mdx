---
title: 레지스트리 생성
---

레지스트리는 사용할 수 있는 아티팩트 유형에 대한 유연성과 관리 기능을 제공하며, 레지스트리의 가시성을 제한하는 등 다양한 기능을 제공합니다.

<div id="create-a-registry">
  ## 레지스트리 생성
</div>

W&amp;B Registry UI 또는 W&amp;B Python SDK를 사용해 레지스트리를 생성할 수 있습니다.

<Tabs>
  <Tab title=" W&B Registry UI">
    1. https://wandb.ai/registry/ 에서 W&amp;B Registry로 이동합니다.
    2. **Create registry** 버튼을 클릭합니다.
    3. **Name** 필드에 레지스트리 이름을 입력합니다.
    4. 원한다면 레지스트리에 대한 설명을 입력합니다.
    5. **Registry visibility** 드롭다운에서 레지스트리를 볼 수 있는 대상을 선택합니다. 레지스트리 가시성 옵션에 대한 자세한 내용은 [레지스트리 가시성 유형](./configure_registry#registry-visibility-types)을 참조하세요.
    6. **Accepted artifacts type** 드롭다운에서 **All types** 또는 **Specify types**를 선택합니다.
    7. (**Specify types**를 선택한 경우) 레지스트리가 허용하는 하나 이상의 아티팩트 유형을 추가합니다.
    8. **Create registry** 버튼을 클릭합니다.
  </Tab>

  <Tab title="Python SDK">
    [`wandb.Api().create_registry()`](/ko/models/ref/python/#method-apicreate_registry) 메서드를 사용하여 프로그래밍 방식으로 레지스트리를 생성합니다. 레지스트리의 `name`과 `visibility` 매개변수에 각각 레지스트리 이름과 [visibility](#visibility-types)를 지정합니다.

    아래 코드 블록을 복사해 붙여넣고, `<>` 로 둘러싸인 값을 자신의 값으로 바꾸세요:

    ```python
    import wandb

    registry = wandb.Api().create_registry(
        name="<registry_name>",
        visibility="< 'restricted' | 'organization' >",
    )
    ```

    레지스트리를 생성할 때 제공할 수 있는 전체 매개변수 목록은 [`wandb.Api().create_registry()`](/ko/models/ref/python/#method-apicreate_registry) 메서드 참조 문서를 확인하세요.
  </Tab>
</Tabs>

<Note>
  아티팩트 유형은 레지스트리 설정에 한 번 저장되면 해당 레지스트리에서 제거할 수 없습니다.
</Note>

예를 들어, 다음 이미지는 사용자가 생성하려는 `Fine_Tuned_Models`라는 레지스트리를 보여줍니다. 이 레지스트리는 **Restricted**로 설정되어 있어 레지스트리에 수동으로 추가된 멤버만 접근할 수 있습니다.

<div id="visibility-types">
  ## 가시성 유형
</div>

레지스트리의 *가시성*은 어떤 사용자가 해당 레지스트리에 접근할 수 있는지를 결정합니다. 레지스트리의 가시성을 제한하면 지정된 구성원만 해당 레지스트리에 접근하도록 보장하는 데 도움이 됩니다.

레지스트리에는 두 가지 가시성 옵션이 있습니다:

| 가시성 | 설명 |
| --- | --- |
| 제한됨   | 초대된 조직 구성원만 레지스트리에 접근할 수 있습니다. |
| 조직 | 조직 내 모든 사용자가 레지스트리에 접근할 수 있습니다. |

팀 관리자 또는 레지스트리 관리자는 레지스트리의 가시성을 설정할 수 있습니다.

가시성이 `제한됨`으로 설정된 레지스트리를 생성한 사용자는 해당 레지스트리의 레지스트리 관리자로 자동 추가됩니다.

<div id="configure-the-visibility-of-a-registry">
  ## 레지스트리 공개 범위 설정
</div>

팀 관리자 또는 레지스트리 관리자는 레지스트리를 생성하는 동안이나 생성한 후에 레지스트리의 공개 범위를 설정할 수 있습니다.

기존 레지스트리의 공개 범위를 제한하려면 다음 단계를 따르세요.

1. https://wandb.ai/registry/ 에서 W&amp;B Registry로 이동합니다.
2. 레지스트리를 선택합니다.
3. 오른쪽 상단의 톱니바퀴 아이콘을 클릭합니다.
4. **Registry visibility** 드롭다운에서 원하는 레지스트리 공개 범위를 선택합니다.
5. **Restricted visibility** 를 선택한 경우:
   1. 이 레지스트리에 접근할 수 있도록 허용할 조직 구성원을 추가합니다. **Registry members and roles** 섹션으로 스크롤한 다음 **Add member** 버튼을 클릭합니다.
   2. **Member** 필드에 추가하려는 구성원의 이메일 또는 사용자 이름을 입력합니다.
   3. **Add new member** 를 클릭합니다.
