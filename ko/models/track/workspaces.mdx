W&amp;B 워크스페이스는 차트를 사용자 지정하고 모델 결과를 탐색할 수 있는 개인용 샌드박스입니다. W&amp;B 워크스페이스는 *Tables*와 *Panel sections*로 구성됩니다.

* **Tables**: 프로젝트에 로깅된 모든 run은 해당 프로젝트의 테이블에 나열됩니다. run을 켜고 끄거나 색상을 변경하고, 테이블을 확장하여 각 run의 노트, 설정, 요약 메트릭을 확인할 수 있습니다.
* **Panel sections**: 하나 이상의 [패널](/ko/models/app/features/panels/)을 포함하는 섹션입니다. 새 패널을 만들고 이를 구성한 다음, 워크스페이스 스냅샷을 저장하기 위해 리포트로 내보낼 수 있습니다.

<div id="workspace-types">
  ## 저장된 워크스페이스 뷰
</div>

맞춤형 워크스페이스 뷰를 통해 팀 협업을 강화하세요. 차트와 데이터를 선호하는 설정으로 정리할 수 있도록 저장된 뷰를 생성하세요.

### 새로 저장된 워크스페이스 뷰 만들기

1. 개인 워크스페이스나 저장된 뷰로 이동합니다.
2. 워크스페이스를 원하는 대로 편집합니다.
3. 워크스페이스 오른쪽 상단의 점 세 개 아이콘(미트볼 메뉴)을 클릭한 다음 **Save as a new view**를 클릭합니다.

새로 저장된 뷰는 워크스페이스 내비게이션 메뉴에 표시됩니다.

<div id="create-a-new-saved-workspace-view">
  ### 저장된 워크스페이스 뷰 업데이트하기
</div>

저장된 변경 사항은 이전에 저장된 뷰의 상태를 덮어씁니다. 저장되지 않은 변경 사항은 유지되지 않습니다. W&amp;B에서 저장된 워크스페이스 뷰를 업데이트하려면:

1. 저장된 뷰로 이동합니다.
2. 워크스페이스 내에서 차트와 데이터에 원하는 변경 사항을 적용합니다.
3. 변경 사항을 확정하려면 **Save** 버튼을 클릭합니다.

<Note>
  워크스페이스 뷰에 대한 업데이트를 저장하면 확인 대화상자가 표시됩니다. 앞으로 이 안내를 다시 보고 싶지 않다면, 저장을 확정하기 전에 **Do not show this modal next time** 옵션을 선택합니다.
</Note>

<div id="update-a-saved-workspace-view">
  ### 워크스페이스 뷰 공유하기
</div>

워크스페이스 URL을 직접 공유하여 맞춤형 워크스페이스를 팀과 공유하세요. 해당 워크스페이스 프로젝트에 액세스 권한이 있는 모든 사용자는 해당 워크스페이스의 저장된 뷰를 볼 수 있습니다.

## 워크스페이스 템플릿

<Note>
  이 기능을 사용하려면 [Enterprise](https://wandb.ai/site/pricing/) 라이선스가 필요합니다.
</Note>

&#95;워크스페이스 템플릿&#95;을 사용하면 [새 워크스페이스에 대한 기본 설정](#default-workspace-settings) 대신 기존 워크스페이스와 동일한 설정을 가진 워크스페이스를 만들 수 있습니다.

<div id="share-a-workspace-view">
  ### 기본 워크스페이스 설정
</div>

W&amp;B에서는 워크스페이스에 대해 다음 기본 설정을 사용합니다:

#### 검색 시 빈 섹션 숨기기

기본적으로 W&amp;B는 워크스페이스에서 빈 섹션을 표시하지 않습니다. 이 설정을 변경하여 워크스페이스에서 빈 섹션을 표시할 수 있습니다.

#### 패널을 알파벳순으로 정렬

기본적으로 W&amp;B는 패널을 플롯 제목을 기준으로 알파벳순으로 정렬하지 않습니다. 이 설정을 변경하면 패널을 플롯 제목을 기준으로 알파벳순으로 정렬할 수 있습니다.

<div id="hide-empty-sections-during-search">
  #### 섹션 구성
</div>

기본적으로 W&amp;B는 메트릭 이름의 첫 번째 접두사를 기준으로 패널을 섹션으로 구성합니다. 예를 들어, 워크스페이스에 메트릭 `a/b/c/d`와 `a/e/f`가 포함되어 있으면 W&amp;B는 이러한 메트릭을 `a`라는 섹션으로 구성합니다. 워크스페이스에 메트릭 `a/b/c/d`와 `e/f/g`가 포함되어 있으면 W&amp;B는 이러한 메트릭을 `a`와 `e`라는 섹션으로 구성합니다.

마지막 접두사를 기준으로 그룹화하도록 기본 섹션 구성을 변경할 수 있습니다. 예를 들어, 워크스페이스에 메트릭 `a/b/c/d`와 `a/e/f`가 포함되어 있으면 W&amp;B는 이러한 메트릭을 `d`와 `f`라는 섹션으로 구성합니다. 워크스페이스에 메트릭 `a/b/c/d`와 `e/f/g`가 포함되어 있으면 W&amp;B는 이러한 메트릭을 `d`와 `g`라는 섹션으로 구성합니다.

### 워크스페이스 템플릿 구성

1. 워크스페이스를 열거나 새로 생성하세요.
2. 워크스페이스 오른쪽 상단의 **Settings** 버튼을 클릭하세요.
3. 패널에서 **Workspace layout**을 선택하세요.
4. 원하는 대로 워크스페이스 설정을 구성하세요.

### 워크스페이스 템플릿 저장하기

1. 워크스페이스 상단에서 **Undo** 및 **Redo** 화살표 아이콘 근처의 작업 메뉴 `...`를 클릭합니다.
2. **Save personal workspace template**를 클릭합니다.
3. 템플릿 설정을 확인한 다음 **Save**를 클릭합니다.

새 워크스페이스는 기본값 대신 이 설정을 사용합니다.

<div id="configure-your-workspace-template">
  ### 워크스페이스 템플릿 보기
</div>

워크스페이스 템플릿의 현재 설정을 보려면:

1. 어느 페이지에서든 오른쪽 상단의 사용자 아이콘을 선택합니다. 드롭다운에서 **User Settings**를 선택합니다.
2. **개인 워크스페이스 템플릿** 섹션으로 이동합니다. 워크스페이스 템플릿을 사용 중인 경우 해당 설정이 표시됩니다. 그렇지 않으면 이 섹션에는 세부 정보가 표시되지 않습니다.

<div id="save-your-workspace-template">
  ### 워크스페이스 템플릿 업데이트하기
</div>

워크스페이스 템플릿을 업데이트하려면 다음을 수행하세요.

1. 아무 워크스페이스나 엽니다.
2. 워크스페이스 설정을 수정합니다. 예를 들어, 포함할 run 수를 `11`로 설정합니다.
3. 템플릿에 변경 내용을 저장하려면 **Undo** 및 **Redo** 화살표 아이콘 근처의 작업 메뉴 `...`를 클릭한 다음 **Update personal workspace template**을 클릭합니다.
4. 설정을 확인한 다음 **Update**를 클릭합니다. 템플릿이 업데이트되며, 이 템플릿을 사용하는 모든 워크스페이스에 다시 적용됩니다.

<div id="delete-your-workspace-template">
  ### 워크스페이스 뷰를 프로그래밍 방식으로 정의하고 저장
</div>

```python
import wandb_workspaces.reports.v2 as ws

workspace = ws.Workspace(entity="your-entity", project="your-project", views=[...])
workspace.save()
```

### 기존 뷰 수정

```python
existing_workspace = ws.Workspace.from_url("workspace-url")
existing_workspace.views[0] = ws.View(name="my-new-view", sections=[...])
existing_workspace.save()
```

<div id="install-workspace-api">
  ### 워크스페이스의 `저장된 뷰`를 다른 워크스페이스로 복사하기
</div>

```python
old_workspace = ws.Workspace.from_url("old-workspace-url")
old_workspace_view = old_workspace.views[0]
new_workspace = ws.Workspace(entity="new-entity", project="new-project", views=[old_workspace_view])

new_workspace.save()
```

자세한 워크스페이스 API 예시는 [`wandb-workspace examples`](https://github.com/wandb/wandb-workspaces/tree/main/examples/workspaces)를 참조하세요.
