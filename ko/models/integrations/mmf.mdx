[Meta AI의 MMF](https://github.com/facebookresearch/mmf) 라이브러리의 `WandbLogger` 클래스는 W&amp;B가 트레이닝/검증 메트릭, 시스템(GPU 및 CPU) 메트릭, 모델 체크포인트 및 설정 파라미터를 로깅할 수 있도록 합니다.

<div id="current-features">
  ## 현재 지원 기능
</div>

다음 기능은 현재 MMF의 `WandbLogger`에서 지원됩니다:

* 트레이닝 및 검증 메트릭
* 시간 경과에 따른 학습률
* 모델 체크포인트를 W&amp;B Artifacts에 저장
* GPU 및 CPU 시스템 메트릭
* 트레이닝 설정 파라미터

<div id="config-parameters">
  ## 설정 파라미터
</div>

wandb 로깅을 활성화하고 사용자 지정할 수 있도록 MMF 설정에서 다음 옵션을 사용할 수 있습니다:

```
training:
    wandb:
        enabled: true
        
        # entity는 runs을 전송할 사용자 이름 또는 팀 이름입니다.
        # 기본적으로 사용자 계정에 run을 로깅합니다.
        entity: null
        
        # wandb로 실험을 로깅할 때 사용할 프로젝트 이름
        project: mmf
        
        # wandb로 프로젝트 하위에 실험을 로깅할 때 사용할
        # 실험/run 이름. 기본 실험 이름은
        # ${training.experiment_name}입니다.
        name: ${training.experiment_name}
        
        # 모델 체크포인팅을 활성화하여 체크포인트를 W&B Artifacts에 저장
        log_model_checkpoint: true
        
        # wandb.init()에 전달할 추가 인수 값 (예시):
        # job_type: 'train'
        # tags: ['tag1', 'tag2']
        
env:
    # wandb 메타데이터가 저장될 디렉터리 경로를 변경하려면
    # (기본값: env.log_dir):
    wandb_logdir: ${env:MMF_WANDB_LOGDIR,}
```
