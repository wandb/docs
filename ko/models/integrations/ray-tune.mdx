W&B는 두 가지 가벼운 통합 방식을 통해 [Ray](https://github.com/ray-project/ray)와 연동됩니다.

- `WandbLoggerCallback` 함수는 Tune에 보고된 지표(metrics)를 자동으로 Wandb API에 기록합니다.
- 함수 API와 함께 사용할 수 있는 `setup_wandb()` 함수는 Tune의 학습 정보를 사용해 Wandb API를 자동으로 초기화합니다. 이후에는 평소처럼 Wandb API를 사용할 수 있으며, `run.log()`를 사용해 학습 과정을 기록할 수 있습니다.

<div id="configure-the-integration">
  ## 통합 설정
</div>

```python
from ray.air.integrations.wandb import WandbLoggerCallback
```

Wandb 구성은 `tune.run()`의 config 파라미터에 wandb 키를 전달해서 수행합니다(아래 예시 참조).

wandb config 항목의 내용은 `wandb.init()`에 키워드 인수로 전달됩니다. 다음 설정은 예외로, `WandbLoggerCallback` 자체를 구성하는 데 사용됩니다:


<div id="parameters">
  ### 매개변수
</div>

`project (str)`: wandb 프로젝트 이름입니다. 필수입니다.

`api_key_file (str)`: wandb API Key가 포함된 파일의 경로입니다.

`api_key (str)`: wandb API Key입니다. `api_key_file`을 설정하는 대신 사용할 수 있습니다.

`excludes (list)`: 로그에서 제외할 메트릭 목록입니다.

`log_config (bool)`: 결과 딕셔너리의 `config` 매개변수를 로깅할지 여부입니다. 기본값은 False입니다.

`upload_checkpoints (bool)`: True이면 모델 체크포인트를 아티팩트로 업로드합니다. 기본값은 False입니다.

<div id="example">
  ### 예제
</div>

```python
from ray import tune, train
from ray.air.integrations.wandb import WandbLoggerCallback


def train_fc(config):
    for i in range(10):
        train.report({"mean_accuracy": (i + config["alpha"]) / 10})


tuner = tune.Tuner(
    train_fc,
    param_space={
        "alpha": tune.grid_search([0.1, 0.2, 0.3]),
        "beta": tune.uniform(0.5, 1.0),
    },
    run_config=train.RunConfig(
        callbacks=[
            WandbLoggerCallback(
                project="<your-project>", api_key="<your-api-key>", log_config=True
            )
        ]
    ),
)

results = tuner.fit()
```


<div id="setup_wandb">
  ## setup_wandb
</div>

```python
from ray.air.integrations.wandb import setup_wandb
```

이 유틸리티 함수는 Ray Tune과 함께 사용할 wandb를 초기화하는 데 도움이 됩니다. 기본적으로는 학습 함수에서 `setup_wandb()`를 호출하면 됩니다.

```python
from ray.air.integrations.wandb import setup_wandb


def train_fn(config):
    # wandb 초기화
    wandb = setup_wandb(config)
    run = wandb.init(
        project=config["wandb"]["project"],
        api_key_file=config["wandb"]["api_key_file"],
    )

    for i in range(10):
        loss = config["a"] + config["b"]
        run.log({"loss": loss})
        tune.report(loss=loss)
    run.finish()


tuner = tune.Tuner(
    train_fn,
    param_space={
        # 탐색 공간 정의
        "a": tune.choice([1, 2, 3]),
        "b": tune.choice([4, 5, 6]),
        # wandb 설정
        "wandb": {"project": "Optimization_Project", "api_key_file": "/path/to/file"},
    },
)
results = tuner.fit()
```
