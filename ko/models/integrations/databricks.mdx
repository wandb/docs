W&amp;B는 Databricks 환경에서 W&amp;B Jupyter 노트북 환경을 맞춤형으로 구성하여 [Databricks](https://www.databricks.com/)와 통합됩니다.

<div id="configure-databricks">
  ## Databricks 설정
</div>

1. 클러스터에 wandb 설치

   클러스터 설정으로 이동하여 클러스터를 선택하고 **Libraries**를 클릭하세요. **Install New**를 클릭하고 **PyPI**를 선택한 후 `wandb` 패키지를 추가하세요.

2. 인증 설정

   W&amp;B 계정을 인증하려면 노트북에서 쿼리할 수 있는 Databricks secret을 추가하면 됩니다.

   ```bash
   # databricks cli 설치
   pip install databricks-cli

   # databricks UI에서 토큰 생성
   databricks configure --token

   # 다음 두 명령 중 하나로 scope 생성 (databricks에서 보안 기능 활성화 여부에 따라 다름):
   # 보안 애드온 사용 시
   databricks secrets create-scope --scope wandb
   # 보안 애드온 미사용 시
   databricks secrets create-scope --scope wandb --initial-manage-principal users

   # https://wandb.ai/settings 에서 API 키 생성
   databricks secrets put --scope wandb --key api_key
   ```

<div id="simple-example">
  ### 단순한 예시
</div>

```python
import os
import wandb

api_key = dbutils.secrets.get("wandb", "api_key")
wandb.login(key=api_key)

with wandb.init() as run:
    run.log({"foo": 1})
```


<div id="sweeps">
  ### Sweeps
</div>

노트북에서 wandb.sweep() 또는 wandb.agent()를 사용하려면 설정이 필요합니다(임시):

```python
import os

# 향후에는 이 설정이 필요하지 않을 예정입니다
os.environ["WANDB_ENTITY"] = "my-entity"
os.environ["WANDB_PROJECT"] = "my-project-that-exists"
```
