  현재 버전의 fastai를 사용하는 경우에는 [fastai 페이지](../)를 참고하세요.
</Note>

fastai v1을 사용하는 스크립트의 경우, 모델 구조, loss, metric, weight, gradient, 샘플 예측, 그리고 최적으로 학습된 모델을 자동으로 기록하는 콜백이 제공됩니다.

```python
import wandb
from wandb.fastai import WandbCallback

wandb.init()

learn = cnn_learner(data, model, callback_fns=WandbCallback)
learn.fit(epochs)
```

기록할 데이터는 콜백 생성자를 통해 설정할 수 있습니다.

```python
from functools import partial

learn = cnn_learner(
    data, model, callback_fns=partial(WandbCallback, input_type="images")
)
```

훈련을 시작할 때 WandbCallback만 사용하는 것도 가능합니다. 이 경우에는 인스턴스를 직접 생성해야 합니다.

```python
learn.fit(epochs, callbacks=WandbCallback(learn))
```

이 단계에서 커스텀 매개변수를 전달할 수도 있습니다.

```python
learn.fit(epochs, callbacks=WandbCallback(learn, input_type="images"))
```

<div id="options">
  ## Options
</div>

`WandbCallback()` 클래스는 여러 옵션을 지원합니다:

| Keyword argument | Default   | Description                                                                                              |
| ---------------- | --------- | -------------------------------------------------------------------------------------------------------- |
| learn            | N/A       | 연결할 fast.ai learner입니다.                                                                            |
| save_model       | True      | 각 단계에서 모델이 개선되면 모델을 저장합니다. 학습이 끝날 때 최상의 모델을 로드합니다.                  |
| mode             | auto      | `min`, `max`, 또는 `auto`: `monitor`에 지정된 학습 지표를 단계 간에 어떻게 비교할지 설정합니다.          |
| monitor          | None      | 최적의 모델을 저장하기 위해 성능을 측정하는 데 사용하는 학습 지표입니다. None인 경우 기본적으로 검증 손실을 사용합니다. |
| log              | gradients | `gradients`, `parameters`, `all`, 또는 None. 손실과 지표는 항상 로깅됩니다.                              |
| input_type       | None      | `images` 또는 `None`. 샘플 예측을 표시하는 데 사용됩니다.                                                 |
| validation_data  | None      | `input_type`가 설정된 경우 샘플 예측에 사용할 데이터입니다.                                               |
| predictions      | 36        | `input_type`가 설정되어 있고 `validation_data`가 `None`인 경우 생성할 예측의 개수입니다.                 |
| seed             | 12345     | `input_type`가 설정되어 있고 `validation_data`가 `None`인 경우 샘플 예측을 위한 난수 생성기를 초기화합니다. |
