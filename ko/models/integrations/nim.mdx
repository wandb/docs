---
title: NVIDIA NeMo Inference Microservice 배포 작업
---

W&amp;B에서 NVIDIA NeMo Inference Microservice로 모델 아티팩트를 배포합니다. 이를 위해 W&amp;B Launch를 사용합니다. W&amp;B Launch는 모델 아티팩트를 NVIDIA NeMo 모델로 변환한 후, 실행 중인 NIM/Triton 서버에 배포합니다.

현재 W&amp;B Launch는 다음과 같은 호환되는 모델 유형을 지원합니다:

1. [Llama2](https://llama.meta.com/llama2/)
2. [StarCoder](https://github.com/bigcode-project/starcoder)
3. NV-GPT (곧 지원 예정)

<Note>
  배포 시간은 모델과 머신 유형에 따라 달라집니다. 기본 Llama2-7b 설정은 Google Cloud의 `a2-ultragpu-1g`에서 약 1분 정도 소요됩니다.
</Note>

<div id="quickstart">
  ## 빠른 시작
</div>

1. 아직 Launch 큐가 없다면 [Launch 큐를 생성](/ko/platform/launch/add-job-to-queue/)하세요. 아래는 예시 큐 설정입니다.

   ```yaml
   net: host
   gpus: all # 특정 GPU 집합을 지정하거나 `all`로 모든 GPU 사용
   runtime: nvidia # nvidia 컨테이너 런타임도 필요
   volume:
     - model-store:/model-store/
   ```

   <Frame>
     <img src="/images/integrations/nim1.png" alt="이미지" />
   </Frame>

2. 프로젝트에 다음 Job을 생성합니다:

   ```bash
   wandb job create -n "deploy-to-nvidia-nemo-inference-microservice" \
      -e $ENTITY \
      -p $PROJECT \
      -E jobs/deploy_to_nvidia_nemo_inference_microservice/job.py \
      -g andrew/nim-updates \
      git https://github.com/wandb/launch-jobs
   ```

3. GPU 머신에서 agent를 실행합니다:
   ```bash
   wandb launch-agent -e $ENTITY -p $PROJECT -q $QUEUE
   ```

4. [Launch UI](https://wandb.ai/launch)에서 원하는 설정으로 배포 Launch job을 제출합니다.
   1. CLI로도 제출할 수 있습니다:
      ```bash
      wandb launch -d gcr.io/playground-111/deploy-to-nemo:latest \
        -e $ENTITY \
        -p $PROJECT \
        -q $QUEUE \
        -c $CONFIG_JSON_FNAME
      ```
      <Frame>
        <img src="/images/integrations/nim2.png" alt="이미지" />
      </Frame>

5. Launch UI에서 배포 프로세스를 추적할 수 있습니다.
   <Frame>
     <img src="/images/integrations/nim3.png" alt="이미지" />
   </Frame>

6. 배포가 완료되면 엔드포인트에 curl 요청을 보내 바로 모델을 테스트할 수 있습니다. 모델 이름은 항상 `ensemble`입니다.
   ```bash
    #!/bin/bash
    curl -X POST "http://0.0.0.0:9999/v1/completions" \
        -H "accept: application/json" \
        -H "Content-Type: application/json" \
        -d '{
            "model": "ensemble",
            "prompt": "Tell me a joke",
            "max_tokens": 256,
            "temperature": 0.5,
            "n": 1,
            "stream": false,
            "stop": "string",
            "frequency_penalty": 0.0
            }'
   ```
