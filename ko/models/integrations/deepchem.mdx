[DeepChem 라이브러리](https://github.com/deepchem/deepchem)는 약물 발견, 소재 과학, 화학, 생물학 분야에서 누구나 딥러닝을 활용할 수 있도록 하는 오픈 소스 도구를 제공합니다. 이 W&amp;B 통합은 DeepChem으로 모델을 학습할 때 간단하고 쉽게 실험을 추적하고 모델 체크포인트를 관리할 수 있게 해줍니다.

<div id="deepchem-logging-in-3-lines-of-code">
  ## 코드 3줄로 DeepChem 로깅하기
</div>

```python
logger = WandbLogger(…)
model = TorchModel(…, wandb_logger=logger)
model.fit(…)
```

<div id="log-your-training-and-evaluation-data-to-wb">
  ### 학습 및 평가 데이터를 W&amp;B에 로깅하기
</div>

학습 손실과 평가 지표를 W&amp;B에 자동으로 로깅할 수 있습니다. 선택적인 평가는 DeepChem [ValidationCallback](https://github.com/deepchem/deepchem/blob/master/deepchem/models/callbacks.py)을 사용해 활성화할 수 있으며, `WandbLogger`가 ValidationCallback 콜백을 감지해 생성된 지표를 로깅합니다.

<Tabs>
  <Tab title="TorchModel">
    ```python
    from deepchem.models import TorchModel, ValidationCallback

    vc = ValidationCallback(…)  # 선택 사항
    model = TorchModel(…, wandb_logger=logger)
    model.fit(…, callbacks=[vc])
    logger.finish()
    ```
  </Tab>

  <Tab title="KerasModel">
    ```python
    from deepchem.models import KerasModel, ValidationCallback

    vc = ValidationCallback(…)  # 선택 사항
    model = KerasModel(…, wandb_logger=logger)
    model.fit(…, callbacks=[vc])
    logger.finish()
    ```
  </Tab>
</Tabs>
