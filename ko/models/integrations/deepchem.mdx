[DeepChem library](https://github.com/deepchem/deepchem)는 약물 발견, 재료 과학, 화학, 생물학 분야에서 딥러닝을 더 쉽게 활용할 수 있도록 돕는 오픈 소스 도구를 제공합니다. 이 W&amp;B 인테그레이션은 DeepChem으로 모델을 트레이닝하는 동안 실험 추적과 모델 체크포인팅을 지원합니다.

이 페이지를 사용해 DeepChem 트레이닝 워크플로에 W&amp;B 로깅을 추가하면 여러 실험 전반에서 트레이닝 손실, 평가 메트릭, 모델 체크포인트를 추적할 수 있습니다. 이미 DeepChem으로 모델을 트레이닝하고 있다면 최소한의 코드 변경만으로 실험 추적을 추가할 수 있습니다.

<div id="deepchem-logging-in-3-lines-of-code">
  ## 코드 3줄로 DeepChem 로깅하기
</div>

```python
logger = WandbLogger(…)
model = TorchModel(…, wandb_logger=logger)
model.fit(…)
```

`WandbLogger` 인스턴스를 DeepChem 모델에 전달하면 트레이닝 run에 W&amp;B 로깅이 연결되어, `fit` 중에 생성되는 메트릭이 W&amp;B 프로젝트로 자동으로 스트리밍됩니다.

  ## 실험 추적

이 페이지의 나머지 부분에서는 API 키를 설정하고, `wandb` 라이브러리를 설치하며, `TorchModel` 또는 `KerasModel`에 대한 로깅을 활성화하는 방법을 설명합니다.

<div id="log-your-training-and-evaluation-data-to-wb">
  ### 트레이닝 및 평가 데이터를 W&amp;B에 기록하기
</div>

`wandb`를 설치하고 인증했다면 이제 DeepChem 모델에 `WandbLogger`를 연결하여 트레이닝 및 평가 데이터가 W&amp;B로 전달되도록 할 수 있습니다.

트레이닝 손실와 평가 메트릭은 자동으로 W&amp;B에 기록됩니다. 선택적 평가를 활성화하려면 DeepChem [`ValidationCallback`](https://github.com/deepchem/deepchem/blob/master/deepchem/models/callbacks.py)을 사용하세요. `WandbLogger`는 `ValidationCallback`을 감지하고, 이 callback이 생성한 메트릭을 기록합니다.

<Tabs>
  <Tab title="TorchModel">
    ```python
    from deepchem.models import TorchModel, ValidationCallback

    vc = ValidationCallback(…)  # 선택 사항
    model = TorchModel(…, wandb_logger=logger)
    model.fit(…, callbacks=[vc])
    logger.finish()
    ```
  </Tab>

  <Tab title="KerasModel">
    ```python
    from deepchem.models import KerasModel, ValidationCallback

    vc = ValidationCallback(…)  # 선택 사항
    model = KerasModel(…, wandb_logger=logger)
    model.fit(…, callbacks=[vc])
    logger.finish()
    ```
  </Tab>
</Tabs>

`model.fit`가 실행된 후에는 트레이닝 손실와 `ValidationCallback`이 생성한 모든 평가 메트릭이 W&amp;B 프로젝트에서 `WandbLogger`가 생성한 run 아래에 표시됩니다.
