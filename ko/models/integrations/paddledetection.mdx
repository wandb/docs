[PaddleDetection](https://github.com/PaddlePaddle/PaddleDetection)은 [PaddlePaddle](https://github.com/PaddlePaddle/Paddle)을 기반으로 한 엔드 투 엔드 객체 탐지 개발 키트입니다. 네트워크 컴포넌트, 데이터 증강, 손실 함수 등의 모듈을 구성해 다양한 주요 객체를 탐지하고, 인스턴스 분할을 수행하며, 키포인트를 추적 및 탐지할 수 있습니다.

PaddleDetection에는 W&amp;B와의 내장 통합 기능이 포함되어 있어, 모든 학습 및 검증 지표는 물론 모델 체크포인트와 해당 메타데이터까지 로깅합니다.

PaddleDetection의 `WandbLogger`는 학습 및 평가 지표를 W&amp;B에 로깅하고, 학습 중 모델 체크포인트도 함께 기록합니다.

`COCO2017` 데이터셋의 서브셋에서 PaddleDetection과 YOLOX 모델을 통합하는 방법을 보여 주는 [W&amp;B 블로그 글](https://wandb.ai/manan-goel/PaddleDetectionYOLOX/reports/Object-Detection-with-PaddleDetection-and-W-B--VmlldzoyMDU4MjY0)을 읽어 보세요.

<div id="install-the-wandb-library-and-log-in">
  ## `wandb` 라이브러리를 설치하고 로그인하기
</div>

로컬 환경에 `wandb` 라이브러리를 설치하고 로그인하려면 다음 단계를 따르세요:

<Tabs>
  <Tab title="Command Line">
    1. `WANDB_API_KEY` [환경 변수](/ko/models/track/environment-variables/)를 자신의 API 키 값으로 설정합니다.

       ```bash
       export WANDB_API_KEY=<your_api_key>
       ```

    2. `wandb` 라이브러리를 설치한 뒤 로그인합니다.

       ```shell
       pip install wandb

       wandb login
       ```
  </Tab>

  <Tab title="Python">
    ```bash
    pip install wandb
    ```

    ```python
    import wandb
    wandb.login()
    ```
  </Tab>

  <Tab title="Python notebook">

<div id="activate-the-wandblogger-in-your-training-script">
  ## 학습 스크립트에서 `WandbLogger` 활성화하기
</div>

<Tabs>
  <Tab title="명령줄">
    [PaddleDetection](https://github.com/PaddlePaddle/PaddleDetection/)에서 `train.py`에 전달하는 인자를 통해 wandb를 사용하려면:

    * `--use_wandb` 플래그를 추가합니다.
    * 첫 번째 wandb 인자는 반드시 `-o` 옵션 뒤에 위치해야 합니다(`-o`는 한 번만 지정하면 됩니다).
    * 각 인자에는 `"wandb-"` 접두사를 포함해야 합니다. 예를 들어 [`wandb.init()`](/ko/models/ref/python/functions/init)에 전달할 인자는 모두 `wandb-` 접두사를 갖습니다.

    ```shell
    python tools/train.py 
        -c config.yml \ 
        --use_wandb \
        -o \ 
        wandb-project=MyDetector \
        wandb-entity=MyTeam \
        wandb-save_dir=./logs
    ```
  </Tab>

  <Tab title="`config.yml`">
    `wandb` 키 아래에 wandb 인자를 config.yml 파일에 추가합니다:

    ```
    wandb:
      project: MyProject
      entity: MyTeam
      save_dir: ./logs
    ```

    `train.py` 파일을 실행하면 W&amp;B 대시보드로 연결되는 링크가 생성됩니다.

    </Frame>
  </Tab>
</Tabs>
