<div id="introduction">
  ## 소개
</div>

Microsoft Azure에서 W&B를 사용하여 GPT-3.5 또는 GPT-4 모델을 파인튜닝하면, 지표를 자동으로 수집하고 W&B의 실험 추적 및 평가 도구를 통해 체계적인 평가를 수행함으로써 모델 성능을 추적·분석·개선할 수 있습니다.

    <img src="/images/integrations/aoai_ft_plot.png" alt="Azure OpenAI 파인튜닝 지표"  />
</Frame>

<div id="prerequisites">
  ## 사전 준비 사항
</div>

- [공식 Azure 문서](https://wandb.me/aoai-wb-int)를 참고하여 Azure OpenAI 서비스를 설정합니다.
- W&B 계정을 생성하고 API 키를 설정합니다.

<div id="workflow-overview">
  ## 워크플로우 개요
</div>

<div id="1-fine-tuning-setup">
  ### 1. 파인튜닝 설정
</div>

- Azure OpenAI 요구 사항에 따라 학습 데이터를 준비합니다.
- Azure OpenAI에서 파인튜닝 작업을 설정합니다.
- W&amp;B가 파인튜닝 과정을 자동으로 추적하며, 메트릭과 하이퍼파라미터를 기록합니다.

<div id="2-experiment-tracking">
  ### 2. 실험 추적
</div>

파인튜닝 중에 W&B는 다음을 기록합니다:

- 학습 및 검증 지표
- 모델 하이퍼파라미터
- 리소스 사용량
- 학습 아티팩트

<div id="3-model-evaluation">
  ### 3. 모델 평가
</div>

파인튜닝이 끝난 후, [W&B Weave](https://weave-docs.wandb.ai)를 사용해 다음을 수행합니다:

- 모델 출력을 레퍼런스 데이터셋과 비교하여 평가합니다
- 서로 다른 파인튜닝 실행 간 성능을 비교합니다
- 특정 테스트 케이스에서의 모델 동작을 분석합니다
- 모델 선택을 위한 데이터 기반 의사결정을 내립니다
