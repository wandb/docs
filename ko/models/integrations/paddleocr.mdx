[PaddleOCR](https://github.com/PaddlePaddle/PaddleOCR)은 PaddlePaddle로 구현된, 다국어를 지원하는 뛰어나고 선도적인 실용적 OCR 도구를 만들어 사용자가 더 나은 모델을 학습하고 실제 환경에 적용할 수 있도록 하는 것을 목표로 합니다. PaddleOCR은 OCR과 관련된 다양한 최첨단 알고리즘을 지원하고, 산업용 솔루션도 개발했습니다. 현재 PaddleOCR에는 W&amp;B 통합 기능이 포함되어 있어, 모델 체크포인트와 해당 메타데이터와 함께 학습 및 평가 지표를 로깅할 수 있습니다.

<div id="add-wandb-to-your-configyml-file">
  ## `config.yml` 파일에 wandb 추가하기
</div>

PaddleOCR에서는 YAML 파일을 통해 구성 변수를 지정해야 합니다. 구성 YAML 파일의 끝에 다음 스니펫을 추가하면 모든 학습 및 검증 지표가 모델 체크포인트와 함께 자동으로 W&amp;B 대시보드에 로깅됩니다:

```python
Global:
    use_wandb: True
```

[`wandb.init()`](/ko/models/ref/python/functions/init)에 전달하려는 추가 선택적 인자는 YAML 파일의 `wandb` 헤더 아래에 추가로 지정할 수 있습니다:

```
wandb:  
    project: CoolOCR  # (선택 사항) wandb 프로젝트 이름입니다 
    entity: my_team   # (선택 사항) wandb 팀을 사용하는 경우 팀 이름을 여기에 입력하세요
    name: MyOCRModel  # (선택 사항) wandb 실행의 이름입니다
```

<div id="pass-the-configyml-file-to-trainpy">
  ## `config.yml` 파일을 `train.py`에 전달하기
</div>

그런 다음 해당 YAML 파일을 PaddleOCR 리포지토리에 있는 [training script](https://github.com/PaddlePaddle/PaddleOCR/blob/release/2.5/tools/train.py)에 인수로 전달합니다.

```bash
python tools/train.py -c config.yml
```

W&amp;B를 활성화한 상태에서 `train.py` 파일을 실행하면 W&amp;B 대시보드로 이동할 수 있는 링크가 생성됩니다:
