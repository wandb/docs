[PaddleOCR](https://github.com/PaddlePaddle/PaddleOCR)는 PaddlePaddle로 구현되어, 사용자가 더 나은 모델을 트레이닝하고 이를 실제에 적용할 수 있도록 돕는 다국어 지원의 뛰어나고 선도적이며 실용적인 OCR 도구를 만드는 것을 목표로 합니다. PaddleOCR은 OCR과 관련된 다양한 최첨단 알고리즘을 지원하며, 산업용 솔루션도 개발했습니다. 이제 PaddleOCR은 트레이닝 및 평가 메트릭과 해당 메타데이터가 포함된 모델 체크포인트를 로깅할 수 있는 W&amp;B 인테그레이션도 제공합니다.

API 키는 사용 중인 머신을 W&amp;B에 인증합니다. 사용자 프로필에서 API 키를 생성할 수 있습니다.

<div id="install-the-wandb-library-and-log-in">
  ## `wandb` 라이브러리 설치 및 로그인
</div>

로컬 환경에 `wandb` 라이브러리를 설치하고 로그인하려면 다음 단계를 따르세요.

<Tabs>
  <Tab title="명령줄">
    1. `WANDB_API_KEY` [환경 변수](/ko/models/track/environment-variables/)를 API 키로 설정합니다.

       ```bash
       export WANDB_API_KEY=<your_api_key>
       ```

    2. `wandb` 라이브러리를 설치하고 로그인합니다.

       ```shell
       pip install wandb

       wandb login
       ```
  </Tab>

<div id="add-wandb-to-your-configyml-file">
  ## `config.yml` 파일에 wandb 추가
</div>

PaddleOCR에서는 설정 변수를 yaml 파일로 지정해야 합니다. 설정 yaml 파일 끝에 다음 스니펫을 추가하면 모델 체크포인트와 함께 모든 트레이닝 및 검증 메트릭이 W&amp;B 대시보드에 자동으로 로깅됩니다:

```python
Global:
    use_wandb: True
```

[`wandb.init()`](/ko/models/ref/python/functions/init)에 추가로 전달하려는 선택 인수도 YAML 파일의 `wandb` 헤더 아래에 추가할 수 있습니다:

<div id="pass-the-configyml-file-to-trainpy">
  ## `config.yml` 파일을 `train.py`에 전달하기
</div>

그런 다음 YAML 파일을 PaddleOCR 저장소에서 사용할 수 있는 [트레이닝 스크립트](https://github.com/PaddlePaddle/PaddleOCR/blob/release/2.5/tools/train.py)의 인수로 전달합니다.

```bash
python tools/train.py -c config.yml
```

W&amp;B를 활성화한 상태로 `train.py` 파일을 실행하면 W&amp;B 대시보드로 이동할 수 있는 링크가 생성됩니다:
