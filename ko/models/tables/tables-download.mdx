모든 W&amp;B Artifacts와 마찬가지로, Tables는 데이터를 쉽게 내보낼 수 있도록 pandas DataFrame으로 변환할 수 있습니다.

<div id="convert-table-to-artifact">
  ## `table`을 `artifact`로 변환하기
</div>

먼저 테이블을 아티팩트로 변환해야 합니다. 이를 수행하는 가장 쉬운 방법은 `artifact.get(table, "table_name")`를 사용하는 것입니다.

```python
# 새 테이블을 생성하고 로그합니다.
with wandb.init() as r:
    artifact = wandb.Artifact("my_dataset", type="dataset")
    table = wandb.Table(
        columns=["a", "b", "c"], data=[(i, i * 2, 2**i) for i in range(10)]
    )
    artifact.add(table, "my_table")
    wandb.log_artifact(artifact)

# 생성한 아티팩트를 사용하여 테이블을 가져옵니다.
with wandb.init() as r:
    artifact = r.use_artifact("my_dataset:latest")
    table = artifact.get("my_table")
```

<div id="convert-artifact-to-dataframe">
  ## `artifact`를 데이터프레임으로 변환
</div>

그런 다음 테이블을 데이터프레임으로 변환합니다.

```python
# 앞의 코드 예제에 이어서:
df = table.get_dataframe()
```

<div id="export-data">
  ## 데이터 내보내기
</div>

이제 dataframe이 지원하는 모든 방식으로 데이터를 내보낼 수 있습니다:

```python
# 테이블 데이터를 .csv로 변환
df.to_csv("example.csv", encoding="utf-8")
```
