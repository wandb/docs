다음 섹션에서는 W&amp;B Tables를 활용하는 몇 가지 방법을 소개합니다:

<div id="view-your-data">
  ### 데이터 보기
</div>

모델 트레이닝이나 평가 중에 메트릭과 리치 미디어를 기록한 다음, 클라우드나 [호스팅 인스턴스](/ko/platform/hosting)와 동기화되는 영구 데이터베이스에서 결과를 시각화하세요.

<Frame>
  <img src="/images/data_vis/tables_see_data.png" alt="데이터 탐색 테이블" max-width="90%" />
</Frame>

예를 들어, [사진 데이터셋의 균형 잡힌 분할](https://wandb.ai/stacey/mendeleev/artifacts/balanced_data/inat_80-10-10_5K/ab79f01e007113280018/files/data_split.table.json)을 보여주는 다음 테이블을 확인해 보세요.

<div id="interactively-explore-your-data">
  ### 데이터를 대화형으로 탐색하기
</div>

정적 파일을 하나씩 살펴보거나 분석 스크립트를 다시 실행할 필요 없이, 테이블을 뷰에서 확인하고 정렬·필터링·그룹화·조인·쿼리해서 데이터와 모델 성능을 파악할 수 있습니다.

<Frame>
  <img src="/images/data_vis/explore_data.png" alt="오디오 비교" max-width="90%" />
</Frame>

예를 들어, [스타일이 변환된 오디오](https://wandb.ai/stacey/cshanty/reports/Whale2Song-W-B-Tables-for-Audio--Vmlldzo4NDI3NzM)에 대한 이 리포트를 참조하세요.

<div id="compare-model-versions">
  ### 모델 버전 비교
</div>

서로 다른 트레이닝 에포크, 데이터셋, 하이퍼파라미터 선택, 모델 아키텍처 등에 걸쳐 결과를 빠르게 비교하세요.

<Frame>
  <img src="/images/data_vis/compare_model_versions.png" alt="모델 비교" max-width="90%" />
</Frame>

예를 들어, [동일한 테스트 이미지에서의 두 모델](https://wandb.ai/stacey/evalserver_answers_2/artifacts/results/eval_Daenerys/c2290abd3d7274f00ad8/files/eval_results.table.json#b6dae62d4f00d31eeebf$eval_Bob)을 비교하는 이 테이블을 참조하세요.

<div id="track-every-detail-and-see-the-bigger-picture">
  ### 모든 세부 사항을 추적하고 큰 그림을 파악하세요
</div>

특정 step의 특정 예측을 확대해 시각화하세요. 축소하면 집계 통계를 확인하고, 오류 패턴을 파악하며, 개선 기회를 이해할 수 있습니다. 이 도구는 단일 모델 트레이닝의 step을 비교할 때도, 서로 다른 모델 버전 간 결과를 비교할 때도 사용할 수 있습니다.

<Frame>
  <img src="/images/data_vis/track_details.png" alt="실험 세부 사항 추적" />
</Frame>

예를 들어, [MNIST 데이터셋에서 한 에포크 후와 다섯 에포크 후의 결과](https://wandb.ai/stacey/mnist-viz/artifacts/predictions/baseline/d888bc05719667811b23/files/predictions.table.json#7dd0cd845c0edb469dec)를 분석한 이 예시 테이블을 참조하세요.

<div id="example-projects-with-wb-tables">
  ## W&amp;B Tables를 사용하는 예제 프로젝트
</div>

다음은 W&amp;B Tables를 사용하는 실제 W&amp;B 프로젝트의 일부입니다.

<div id="image-classification">
  ### 이미지 분류
</div>

[Visualize Data for Image Classification](https://wandb.ai/stacey/mendeleev/reports/Visualize-Data-for-Image-Classification--VmlldzozNjE3NjA)를 조회하고, [data visualization nature Colab](https://wandb.me/dsviz-nature-colab)을 따라 하거나, [아티팩트 context](https://wandb.ai/stacey/mendeleev/artifacts/val_epoch_preds/val_pred_gawf9z8j/2dcee8fa22863317472b/files/val_epoch_res.table.json)를 탐색해 보세요. CNN이 [iNaturalist](https://www.inaturalist.org/pages/developers) 사진에서 열 가지 종류의 생물(식물, 새, 곤충 등)을 어떻게 식별하는지 확인할 수 있습니다.

</Frame>

<div id="audio">
  ### Audio
</div>

[timbre transfer를 다루는 Whale2Song - W&amp;B Tables for Audio](https://wandb.ai/stacey/cshanty/reports/Whale2Song-W-B-Tables-for-Audio--Vmlldzo4NDI3NzM)에서 Audio 테이블을 탐색해 보세요. 녹음된 고래 소리를 바이올린이나 트럼펫 같은 악기로 같은 멜로디를 합성한 버전과 비교할 수 있습니다. 또한 자신의 노래를 녹음하고 [audio transfer Colab](http://wandb.me/audio-transfer)을 사용해 W&amp;B에서 합성된 버전도 탐색할 수 있습니다.

<div id="text">
  ### 텍스트
</div>

트레이닝 데이터나 생성된 출력의 텍스트 샘플을 살펴보고, 관련 필드를 기준으로 동적으로 그룹화하고, 모델 변형이나 실험 설정 전반에서 평가를 정렬해 비교하세요. 텍스트를 Markdown으로 렌더링하거나 시각적 diff 모드를 사용해 텍스트 간 차이를 비교할 수 있습니다. 문자 기반 RNN의 예시는 [Shakespeare 텍스트 생성 리포트](https://wandb.ai/stacey/nlg/reports/Visualize-Text-Data-Predictions--Vmlldzo1NzcwNzY)를 참조하세요.

</Frame>
