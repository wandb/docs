다음 퀵스타트에서는 데이터 테이블을 로깅하고, 데이터를 시각화하고, 쿼리하는 방법을 설명합니다.

아래 버튼을 선택하여 MNIST 데이터용 PyTorch 퀵스타트 예제 프로젝트를 사용해 보세요.

<div id="1-log-a-table">
  ## 1. 테이블 로깅하기
</div>

W&amp;B에 테이블을 로깅하세요. 새 테이블을 만들거나 Pandas 데이터프레임을 전달할 수 있습니다.

<Tabs>
  <Tab title="테이블 구성">
    새 Table을 구성하고 로깅하려면 다음을 사용하세요.

    * [`wandb.init()`](/ko/models/ref/python/functions/init): 결과를 추적할 [run](/ko/models/runs/)을 생성합니다.
    * [`wandb.Table()`](/ko/models/ref/python/data-types/table): 새 테이블 객체를 생성합니다.
      * `columns`: 열 이름을 설정합니다.
      * `data`: 각 행의 내용을 설정합니다.
    * [`wandb.Run.log()`](/ko/models/ref/python/experiments/run.md/#method-runlog): 테이블을 로깅하여 W&amp;B에 저장합니다.

    예시는 다음과 같습니다.

    ```python
    import wandb

    with wandb.init(project="table-test") as run:
        # 새 테이블을 생성하고 로그합니다.
        my_table = wandb.Table(columns=["a", "b"], data=[["a1", "b1"], ["a2", "b2"]])
        run.log({"Table Name": my_table})
    ```
  </Tab>

  <Tab title="Pandas 데이터프레임">
    Pandas 데이터프레임을 `wandb.Table()`에 전달하여 새 테이블을 만드세요.

    ```python
    import wandb
    import pandas as pd

    df = pd.read_csv("my_data.csv")

    with wandb.init(project="df-table") as run:
        # DataFrame에서 새 테이블을 생성하고
        # 이를 W&B에 로그합니다.
      my_table = wandb.Table(dataframe=df)
      run.log({"Table Name": my_table})
    ```

    지원되는 데이터 유형에 대한 자세한 내용은 W&amp;B API 레퍼런스 가이드의 [`wandb.Table`](/ko/models/ref/python/data-types/table)을 참조하세요.
  </Tab>
</Tabs>
