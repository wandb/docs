---

W&amp;B는 널리 사용되는 머신러닝 프레임워크, 클라우드 플랫폼, 워크플로 오케스트레이션 도구와 연동되어, 실험을 추적하고 지표를 기록하며 모델을 원활하게 관리할 수 있도록 도와줍니다.

<div id="popular-ml-integrations">
  ## 인기 있는 ML 통합:
</div>

<CardGroup cols={2}>
  <Card title="PyTorch Lightning" href="/ko/models/integrations/lightning">
    W&amp;B를 PyTorch Lightning 코드에 통합하여 파이프라인에 실험 추적 기능을 추가하세요.
  </Card>

  <Card title="HuggingFace Transformers" href="/ko/models/integrations/huggingface">
    W&amp;B와 함께 HuggingFace Transformers 모델을 최적화하여 실험 추적과 모델 관리를 수행하세요.
  </Card>
</CardGroup>

<CardGroup cols={2}>
  <Card title="Keras" href="/ko/models/integrations/keras">
    W&amp;B와 Keras를 사용해 머신 러닝 실험 추적, 데이터셋 버전 관리, 프로젝트 협업을 수행하세요.
  </Card>

  <Card title="YOLOv5" href="/ko/models/integrations/yolov5">
    &quot;You Only Look Once&quot;(일명 YOLOv5) 실시간 객체 감지 프레임워크와 W&amp;B를 함께 사용하여 모델 지표를 추적하고, 모델 출력을 검사하며, 중단된 실행을 다시 시작하세요.
  </Card>
</CardGroup>

사용 중인 라이브러리가 네이티브로 지원되지 않더라도 W&amp;B [Python SDK](/ko/models/ref/python)를 사용해 W&amp;B를 통합할 수 있습니다. 모범 사례와 구현 방법은 [어떤 라이브러리에도 W&amp;B 추가하기](/ko/models/integrations/add-wandb-to-any-library)를 참고하세요.
