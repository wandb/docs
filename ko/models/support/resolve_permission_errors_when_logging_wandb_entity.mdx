---
title: Run 을 로그할 때 발생하는 권한 오류를 어떻게 해결하나요?
---

W&B Entity 에 Run 을 로그할 때 발생하는 권한 오류를 해결하려면 다음 단계를 따르세요:

- **Entity 및 Project 이름 확인**: 코드에 입력한 W&B Entity 와 Project 이름의 철자와 대소문자가 정확한지 확인하세요.
- **권한 확인**: 관리자로부터 필요한 권한을 부여받았는지 확인하세요.
- **로그인 자격 증명 확인**: 올바른 W&B 계정으로 로그인했는지 확인하세요. 다음 코드로 Run 을 생성하여 테스트할 수 있습니다:
  ```python
  import wandb

  # entity와 project 이름을 본인의 정보로 수정하세요
  with wandb.init(entity="your_entity", project="your_project") as run:
      run.log({'example_metric': 1})
  ```
- **API 키 설정**: `WANDB_API_KEY` 환경 변수를 사용하세요:
  ```bash
  export WANDB_API_KEY='your_api_key'
  ```
- **호스트 정보 확인**: 커스텀 배포 환경인 경우 호스트 URL을 설정하세요:
  ```bash
  wandb login --relogin --host=<host-url>
  export WANDB_BASE_URL=<host-url>
  ```
