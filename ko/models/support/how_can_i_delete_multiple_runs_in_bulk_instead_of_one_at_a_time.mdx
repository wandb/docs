---
title: 한 번에 하나씩 삭제하지 않고 여러 개의 Runs 를 대량으로 삭제하려면 어떻게 해야 하나요?
---

[public API](/models/ref/python/public-api/api) 를 사용하여 단일 작업으로 여러 개의 Runs 를 삭제할 수 있습니다:

```python
import wandb

api = wandb.Api()
runs = api.runs('<entity>/<project>')
for run in runs:
    # 조건에 따라
    if <condition>:
        # run 삭제
        run.delete()
```
