---
title: 여러 개의 선을 범례와 함께 플롯하려면 어떻게 하나요?
---

`wandb.plot.line_series()`로 여러 선이 있는 커스텀 차트를 만드세요. 선 차트를 보려면 [프로젝트 페이지](/ko/models/track/project-page/)로 이동하세요. 범례를 추가하려면 `wandb.plot.line_series()`에 `keys` 인자를 포함하세요. 예를 들어:

```python

with wandb.init(project="my_project") as run:

    run.log(
        {
            "my_plot": wandb.plot.line_series(
                xs=x_data, ys=y_data, keys=["metric_A", "metric_B"]
            )
        }
    )
```

멀티라인 플롯에 대한 추가 내용은 **Multi-line** 탭의 [여기](/ko/models/track/log/plots/#basic-charts)에서 확인하세요.
