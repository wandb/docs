---
title: 범례(legend)와 함께 차트에 여러 개의 선을 그리는 방법은 무엇인가요?
---

`wandb.plot.line_series()`를 사용하여 다중 선형 커스텀 차트를 생성하세요. [project page](/models/track/project-page/)로 이동하여 라인 차트를 확인할 수 있습니다. 범례를 추가하려면 `wandb.plot.line_series()`에 `keys` 인수를 포함하세요. 예시:

```python

with wandb.init(project="my_project") as run:

    run.log(
        {
            "my_plot": wandb.plot.line_series(
                xs=x_data, ys=y_data, keys=["metric_A", "metric_B"]
            )
        }
    )
```

다중 선형 플롯에 대한 자세한 내용은 [여기](/models/track/log/plots/#basic-charts)의 **Multi-line** 탭에서 확인할 수 있습니다.
