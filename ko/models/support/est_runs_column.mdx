---
title: "Est. Runs 열은 무엇인가요?"
---

W&amp;B는 이산형 탐색 공간을 갖는 W&amp;B Sweep을 생성할 때 생성될 실행 수를 추정해 제공합니다. 이 값은 탐색 공간의 데카르트 곱을 반영합니다.

예를 들어, 다음과 같은 탐색 공간을 고려해 보세요:

<Frame>
  <img src="/images/sweeps/sweeps_faq_whatisestruns_1.png" alt="Estimated runs column" />
</Frame>

이 경우 데카르트 곱은 9가 됩니다. W&amp;B는 이 값을 앱 UI에서 추정 실행 수(**Est. Runs**)로 표시합니다:

<Frame>
  <img src="/images/sweeps/spaces_sweeps_faq_whatisestruns_2.webp" alt="Sweep run estimation" />
</Frame>

추정 실행 수를 프로그래밍 방식으로 가져오려면 W&amp;B SDK 내 Sweep 객체의 `expected_run_count` 속성을 사용하세요:

```python
sweep_id = wandb.sweep(
    sweep_configs, project="your_project_name", entity="your_entity_name"
)
api = wandb.Api()
sweep = api.sweep(f"your_entity_name/your_project_name/sweeps/{sweep_id}")
print(f"EXPECTED RUN COUNT = {sweep.expected_run_count}")
```
