---
title: 익명 사용자 가 사용할 수 없는 기능에는 어떤 것들이 있나요?
---

* **영구 데이터 없음**: Runs 데이터는 익명 계정에 7일 동안 저장됩니다. 익명 run 데이터를 실제 계정으로 저장하여 소유권을 가져올 수 있습니다.

<Frame>
<img src="/images/app_ui/anon_mode_no_data.png" alt="익명 모드 인터페이스" />
</Frame>

* **Artifacts 로그 불가**: 익명 run에 아티팩트를 로그하려고 하면 커맨드라인에 다음과 같은 경고가 표시됩니다:
    ```bash
    wandb: WARNING Artifacts logged anonymously cannot be claimed and expire after 7 days.
    ```

* **프로필 또는 설정 페이지 없음**: 실제 계정에서만 유효한 일부 UI 페이지는 제공되지 않습니다.
