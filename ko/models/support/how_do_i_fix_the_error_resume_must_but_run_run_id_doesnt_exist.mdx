---
title: "resume='must' but run (<run_id>) doesn't exist 오류를 어떻게 해결하나요?"
---

`resume='must' but run (<run_id>) doesn't exist` 오류가 발생하는 경우, 재개(resume)하려는 실행이 해당 프로젝트나 엔터티 내에 존재하지 않는다는 의미입니다. 올바른 W&amp;B 인스턴스에 로그인했는지, 그리고 프로젝트와 엔터티가 올바르게 설정되어 있는지 확인하세요:

```python
wandb.init(entity=<entity>, project=<project>, id=<run-id>, resume='must')
```

[`wandb login --relogin`](/ko/models/ref/cli/wandb-login) 명령을 실행해 인증이 완료되었는지 확인하세요.
