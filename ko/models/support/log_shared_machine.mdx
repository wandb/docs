---
title: 공용 머신에서 올바른 wandb 사용자 계정으로 로그를 기록하려면 어떻게 해야 하나요?
---

공용 머신을 사용할 때는 인증을 위해 `WANDB_API_KEY` 환경 변수를 설정하여 Runs 가 올바른 W&B 계정으로 로그되도록 하세요. 환경 설정 파일에 이 변수가 포함되어 있으면 로그인 시 올바른 자격 증명이 제공됩니다. 또는 스크립트에서 직접 환경 변수를 설정할 수도 있습니다.

`export WANDB_API_KEY=X` 코맨드를 실행하세요. 여기서 X는 사용자의 API 키로 대체합니다. API 키는 [wandb.ai/settings](https://wandb.ai/settings)에서 생성할 수 있습니다.
