---
title: 내 run과 연관된 git 커밋을 어떻게 저장할 수 있나요?
---

`wandb.init`이 호출되면 시스템이 원격 저장소 링크와 최신 커밋의 SHA를 포함한 git 정보를 자동으로 수집합니다. 이 정보는 [run 페이지](/ko/models/runs/#view-logged-runs)에 표시됩니다. 이 정보를 확인하려면 스크립트를 실행할 때 현재 작업 디렉터리가 git으로 관리되는 폴더 안인지 확인하세요.

git 커밋과 실험 실행에 사용한 명령은 해당 사용자에게는 계속 보이지만 외부 사용자에게는 숨겨집니다. 공개 프로젝트의 경우에도 이러한 세부 정보는 비공개로 유지됩니다.
