---
title: 실행과 연결된 git 커밋은 어떻게 저장할 수 있나요?
---

`wandb.init`이 호출되면, 시스템은 원격 저장소 링크와 최신 커밋의 SHA를 포함한 Git 정보를 자동으로 수집합니다. 이 정보는 [실행 페이지](/ko/models/runs/#view-logged-runs)에 표시됩니다. 이 정보를 확인하려면 스크립트를 실행할 때 현재 작업 디렉터리가 Git으로 관리되는 폴더 안인지 확인하세요.

Git 커밋과 실험을 실행하는 데 사용된 명령은 사용자에게는 계속 보이지만 외부 사용자에게는 숨겨집니다. 공개 프로젝트에서도 이러한 세부 정보는 비공개로 유지됩니다.
