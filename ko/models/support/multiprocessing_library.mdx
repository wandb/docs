---
title: W&B는 `multiprocessing` 라이브러리를 사용하나요?
---

네, W&amp;B는 `multiprocessing` 라이브러리를 사용합니다. 다음과 같은 오류 메시지는 문제가 발생했을 수 있음을 나타냅니다:

```
An attempt has been made to start a new process before the current process 
has finished its bootstrapping phase.
```

이 문제를 해결하려면 `if __name__ == "__main__":` 엔트리 포인트 보호 코드를 추가하세요. 스크립트에서 W&amp;B를 직접 실행할 때는 이 보호 코드가 필요합니다.
