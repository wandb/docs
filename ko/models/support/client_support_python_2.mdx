---
title: W&B 클라이언트는 Python 2를 지원하나요?
---

W&B 클라이언트 라이브러리는 버전 0.10까지 Python 2.7과 Python 3를 모두 지원했습니다. Python 2의 지원 종료(EOL)로 인해 Python 2.7 지원은 버전 0.11부터 중단되었습니다. Python 2.7 시스템에서 `pip install --upgrade wandb`를 실행하면 0.10.x 시리즈의 신규 릴리스만 설치됩니다. 0.10.x 시리즈에 대한 지원은 중요 버그 수정 및 패치에만 한정됩니다. Python 2.7을 지원하는 0.10.x 시리즈의 마지막 버전은 0.10.33입니다.
