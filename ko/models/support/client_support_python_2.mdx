---
title: W&B 클라이언트는 Python 2를 지원하나요?
---

W&B 클라이언트 라이브러리는 0.10 버전까지 Python 2.7과 Python 3를 모두 지원했습니다. Python 2가 수명 종료(EOL)에 도달함에 따라 0.11 버전부터는 Python 2.7 지원이 중단되었습니다. Python 2.7 시스템에서 `pip install --upgrade wandb`를 실행하면 0.10.x 시리즈의 새로운 릴리스만 설치됩니다. 0.10.x 시리즈에 대한 지원은 중요 버그 수정 및 패치로만 제한됩니다. Python 2.7을 지원하는 0.10.x 시리즈의 마지막 버전은 0.10.33입니다.
