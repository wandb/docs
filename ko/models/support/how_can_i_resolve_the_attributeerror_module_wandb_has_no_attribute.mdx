---
title: '`AttributeError: module ''wandb'' has no attribute ...`와 같은 오류를 어떻게 해결할 수 있나요?'
---

Python에서 `wandb`를 임포트할 때 `AttributeError: module 'wandb' has no attribute 'init'` 또는 `AttributeError: module 'wandb' has no attribute 'login'`와 같은 오류가 발생한다면, `wandb`가 설치되어 있지 않거나 설치가 손상된 상태인데 현재 작업 디렉터리에 `wandb` 디렉터리가 존재하기 때문일 수 있습니다. 이 오류를 해결하려면, `wandb`를 언인스톨(uninstall)하고 해당 디렉터리를 삭제한 뒤, 다시 `wandb`를 설치하세요:

```bash
pip uninstall wandb; rm -rI wandb; pip install wandb
```
