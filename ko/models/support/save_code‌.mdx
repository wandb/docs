---
title: 코드는 어떻게 저장하나요?‌
---

`wandb.init`에서 `save_code=True`를 사용하면 해당 run을 시작하는 메인 스크립트나 노트북을 저장할 수 있습니다. run의 모든 코드를 저장하려면 Artifacts로 코드를 버전 관리하세요. 다음 예시는 이 과정을 보여 줍니다.

```python
code_artifact = wandb.Artifact(type="code")
code_artifact.add_file("./train.py")
wandb.log_artifact(code_artifact)
```
