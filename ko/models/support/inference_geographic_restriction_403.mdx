---
title: W&B Inference에서 제 국가 또는 지역이 지원되지 않는다고 표시되는 이유는 무엇인가요?
---

"Country, region, or territory not supported" 메시지와 함께 발생하는 403 에러는 지원되지 않는 위치에서 W&B Inference에 엑세스하고 있음을 의미합니다.

## 발생 원인

W&B Inference는 준수 사항 및 규제 요구 사항으로 인해 지리적 제한이 있습니다. 이 서비스는 지원되는 지리적 위치에서만 엑세스할 수 있습니다.

## 해결 방법

1. **서비스 약관 확인**
   - 현재 지원되는 위치 목록은 [서비스 약관](https://docs.coreweave.com/docs/policies/terms-of-service/terms-of-use#geographic-restrictions)에서 확인하세요.

2. **지원되는 위치에서 사용**
   - 지원되는 국가 또는 지역에서 서비스에 엑세스하세요.
   - 지원되는 위치에 있는 조직의 리소스를 사용하는 것을 고려해 보세요.

3. **계정 담당 팀에 문의**
   - Enterprise 고객은 담당 Account Executive와 옵션에 대해 논의할 수 있습니다.
   - 일부 조직에는 별도의 특별한 계약이 있을 수 있습니다.

## 에러 상세 정보

다음과 같은 에러가 표시되는 경우:
```
{
  "error": {
    "code": 403,
    "message": "Country, region, or territory not supported"
  }
}
```

이 에러는 API 요청 시점의 IP 주소 위치에 따라 결정됩니다.
