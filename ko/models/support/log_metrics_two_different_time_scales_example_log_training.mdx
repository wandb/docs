---
title: 서로 다른 두 시간 단위로 지표를 기록할 수 있나요?
---

예를 들어, 배치마다 훈련 정확도(training accuracy)를, 에포크마다 검증 정확도(validation accuracy)를 기록하고 싶다고 가정해 봅시다.

가능합니다. 지표와 함께 `batch`, `epoch` 같은 인덱스를 같이 기록하세요. 한 번은 `wandb.Run.log()({'train_accuracy': 0.9, 'batch': 200})`를 사용하고, 다른 한 번은 `wandb.Run.log()({'val_accuracy': 0.8, 'epoch': 4})`를 사용하면 됩니다. UI에서 각 차트에 대해 원하는 값을 x축으로 설정하세요. 특정 인덱스에 대해 기본 x축을 설정하려면 [Run.define&#95;metric()](/ko/models/ref/python/experiments/run/#define_metric)을 사용하세요. 위 예시의 경우 다음 코드를 사용합니다:

```python
import wandb

with wandb.init() as run:
   run.define_metric("batch")
   run.define_metric("epoch")

   run.define_metric("train_accuracy", step_metric="batch")
   run.define_metric("val_accuracy", step_metric="epoch")
```
