---
title: 동일한 머신에서 계정을 전환하려면 어떻게 하나요?
---

동일한 머신에서 두 개의 W&amp;B 계정을 관리하려면 두 API 키를 모두 하나의 파일에 저장하세요. 각 리포지토리에서 아래 코드를 사용해 키를 안전하게 전환하면, API 키가 소스 코드 관리 시스템에 커밋되는 것을 방지할 수 있습니다.

```python
if os.path.exists("~/keys.json"):
    os.environ["WANDB_API_KEY"] = json.loads("~/keys.json")["work_account"]
```
