---
title: 동일한 장비에서 여러 계정 간에 어떻게 전환하나요?
---

한 대의 머신에서 두 개의 W&B 계정을 관리하려면, 두 API 키를 하나의 파일에 저장하세요. 소스 제어 시스템에 비밀 키가 노출되는 것을 방지하면서 안전하게 키를 전환하려면 리포지토리에서 다음 코드를 사용하세요.

```python
if os.path.exists("~/keys.json"):
    # keys.json 파일에서 work_account에 해당하는 API 키를 환경 변수로 설정합니다.
    os.environ["WANDB_API_KEY"] = json.loads("~/keys.json")["work_account"]
```
