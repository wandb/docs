---
title: 하나의 스크립트에서 여러 실행을 수행하려면 어떻게 하나요?
---

하나의 스크립트에서 여러 실행을 기록하려면, 새 실행을 시작하기 전에
이전 실행을 마쳐야 합니다.

이 작업을 수행하는 권장 방법은 `wandb.init()`을 컨텍스트 매니저로 사용하는 것입니다.
이렇게 하면 스크립트에서 예외가 발생했을 때 실행을 종료하고 실패로 표시합니다:

```python
import wandb

for x in range(10):
    with wandb.init() as run:
        for y in range(100):
            run.log({"metric": x + y})
```

또한 `run.finish()`를 명시적으로 호출할 수도 있습니다:

```python
import wandb

for x in range(10):
    run = wandb.init()

    try:
        for y in range(100):
            run.log({"metric": x + y})

    except Exception:
        run.finish(exit_code=1)
        raise

    finally:
        run.finish()
```

<div id="multiple-active-runs">
  ## 다중 활성 실행
</div>

wandb 0.19.10부터 `reinit` 설정을 `"create_new"`로 지정하면
여러 실행을 동시에 활성 상태로 둘 수 있습니다.

```python
import wandb

with wandb.init(reinit="create_new") as tracking_run:
    for x in range(10):
        with wandb.init(reinit="create_new") as run:
            for y in range(100):
                run.log({"x_plus_y": x + y})

            tracking_run.log({"x": x})
```

`reinit="create_new"` 옵션에 대한 자세한 설명과 W&amp;B 통합 시 주의해야 할 사항은
[프로세스당 여러 실행](/ko/models/runs/initialize-run/)을 참고하세요.
