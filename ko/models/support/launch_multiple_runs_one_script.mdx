---
title: 하나의 스크립트에서 여러 run을 실행하려면 어떻게 하나요?
---

단일 스크립트에서 여러 run을 로깅하려면 새 run을 시작하기 전에 이전 run을 먼저 종료해야 합니다.

권장되는 방식은 `wandb.init()`을 컨텍스트 매니저로 사용하는 것입니다. 이렇게 하면 스크립트에서 예외가 발생하는 경우 해당 run을 종료하고 실패로 표시합니다:

```python
import wandb

for x in range(10):
    with wandb.init() as run:
        for y in range(100):
            run.log({"metric": x + y})
```

또한 `run.finish()`를 명시적으로 호출할 수도 있습니다:

```python
import wandb

for x in range(10):
    run = wandb.init()

    try:
        for y in range(100):
            run.log({"metric": x + y})

    except Exception:
        run.finish(exit_code=1)
        raise

    finally:
        run.finish()
```

<div id="multiple-active-runs">
  ## 여러 개의 활성 run
</div>

wandb 0.19.10부터 `reinit` 설정을 `"create_new"`로 지정하면
동시에 활성화된 여러 개의 run을 생성할 수 있습니다.

```python
import wandb

with wandb.init(reinit="create_new") as tracking_run:
    for x in range(10):
        with wandb.init(reinit="create_new") as run:
            for y in range(100):
                run.log({"x_plus_y": x + y})

            tracking_run.log({"x": x})
```

`reinit="create_new"` 설정과 W&amp;B 인테그레이션 관련 주의 사항에 대해서는
[프로세스당 여러 run](/ko/models/runs/initialize-run/)을 참고하세요.
