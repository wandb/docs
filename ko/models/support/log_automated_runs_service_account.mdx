---
title: CI나 내부 도구에서 실행되는 실행은 어떻게 기록하나요?
---

자동화된 테스트나 내부 도구에서 W&B로 로그를 기록하려면 팀 설정 페이지에서 **Service Account**를 생성하세요. 이렇게 하면 CI(지속적 통합)를 통해 실행되는 작업을 포함한 자동화 작업에 서비스 API 키를 사용할 수 있습니다. Service Account로 수행된 작업을 특정 사용자에게 연결하려면 `WANDB_USERNAME` 또는 `WANDB_USER_EMAIL` 환경 변수를 설정하세요.
