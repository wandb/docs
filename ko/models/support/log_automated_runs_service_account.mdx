---
title: 지속적 통합(continuous integration)이나 내부 툴(internal tools)을 통해 실행된 Runs를 어떻게 로그하나요?
---

W&B에 로그를 남기는 자동화된 테스트나 내부 툴을 실행하려면 팀 설정 페이지에서 **Service Account**를 생성하세요. 이 기능을 통해 지속적 통합(CI) 환경에서 실행되는 작업을 포함한 모든 자동화된 작업에 서비스 API 키를 사용할 수 있습니다. Service Account 작업을 특정 사용자에게 할당하려면 `WANDB_USERNAME` 또는 `WANDB_USER_EMAIL` 환경 변수를 설정하세요.
