---
title: 계정이 없는 사람은 run 결과를 어떻게 보나요?
---

`anonymous="allow"`로 스크립트를 실행하면:

1. **임시 계정 자동 생성**: W&B는 로그인된 계정이 있는지 확인합니다. 없으면 W&B가 새로운 익명 계정을 만들고 그 세션을 위한 API 키를 저장합니다.
2. **결과를 빠르게 로깅**: 사용자는 스크립트를 여러 번 실행하고 W&B 대시보드에서 결과를 즉시 확인할 수 있습니다. 이 소유자가 없는 익명 run은 7일 동안 이용 가능합니다.
3. **필요할 때 데이터 가져오기**: 사용자가 W&B에서 가치 있는 결과를 찾으면, 페이지 상단 배너의 버튼을 클릭해 run 데이터를 실제 계정으로 가져와 저장할 수 있습니다. 가져오지 않으면 run 데이터는 7일 후 삭제됩니다.

<Warning>
**익명 run 링크는 민감한 정보입니다**. 이 링크를 사용하면 누구나 7일 동안 실험 결과를 조회하고 소유권을 주장할 수 있으므로, 신뢰할 수 있는 사람에게만 링크를 공유하세요. 작성자의 신원을 숨긴 채로 결과를 공개적으로 공유하려면 support@wandb.com으로 연락하여 도움을 받으세요.
</Warning>

W&B 사용자가 스크립트를 찾아 실행하면, 일반적인 run과 마찬가지로 결과는 해당 사용자의 계정에 정상적으로 로깅됩니다.
