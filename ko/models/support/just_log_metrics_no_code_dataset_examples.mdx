---
title: 코드나 데이터셋 예시 없이 메트릭만 로깅할 수 있나요?
---

기본적으로 W&B는 데이터셋 예시를 로깅하지 않습니다. 기본 설정에서는 코드와 시스템 메트릭을 로깅합니다.

환경 변수를 사용해 코드 로깅을 끄는 방법은 두 가지가 있습니다:

1. `WANDB_DISABLE_CODE`를 `true`로 설정해 모든 코드 추적을 끕니다. 이렇게 하면 git SHA와 diff 패치를 가져오지 않습니다.
2. `WANDB_IGNORE_GLOBS`를 `*.patch`로 설정해 diff 패치를 서버와 동기화하지 않도록 하되, `wandb restore`로 적용할 수 있도록 로컬에는 유지합니다.

관리자인 경우 팀 설정에서 팀에 대한 코드 저장 기능을 끌 수도 있습니다:

1. `https://wandb.ai/<team>/settings`에서 팀 설정 페이지로 이동합니다. 여기서 `<team>`은 팀 이름입니다.
2. Privacy 섹션까지 스크롤합니다.
3. **Enable code saving by default** 설정을 전환합니다.
