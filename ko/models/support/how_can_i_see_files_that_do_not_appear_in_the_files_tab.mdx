파일 탭은 최대 10,000개의 파일을 보여줍니다. 모든 파일을 다운로드하려면 [public API](/ko/models/ref/python/public-api/api)를 사용하세요.

```python
import wandb

api = wandb.Api()
run = api.run('<entity>/<project>/<run_id>')
run.file('<file>').download()

for f in run.files():
    if <condition>:
        f.download()
```
