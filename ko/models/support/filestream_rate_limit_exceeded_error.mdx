---
title: Filestream rate limit exceeded 오류를 어떻게 해결할 수 있나요?
---

W&B에서 "Filestream rate limit exceeded" 오류를 해결하려면 다음 단계를 수행하세요:

**로깅 최적화**:

- API 요청을 줄이기 위해 로깅 빈도를 낮추거나 로그를 배치로 전송하세요.
  - 실험 시작 시간을 분산해 동시에 API 요청이 몰리지 않도록 하세요.

**장애 여부 확인**:

- [W&B 상태 페이지](https://status.wandb.com)를 확인해 일시적인 서버 측 문제로 인해 발생한 것은 아닌지 확인하세요.

**지원팀에 문의**:

- 요청 속도 제한(rate limit) 상향을 요청하려면 실험 설정에 대한 상세 정보를 포함해 W&B 지원팀(support@wandb.com)에 문의하세요.
