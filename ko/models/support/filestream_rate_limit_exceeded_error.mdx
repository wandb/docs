---
title: Filestream rate limit exceeded 오류를 어떻게 해결할 수 있나요?
---

W&B에서 "Filestream rate limit exceeded" 오류를 해결하려면 다음 단계를 따르세요.

**로그 최적화**:
  - 로그 기록 빈도를 줄이거나 로그를 배치로 묶어 API 요청 횟수를 줄입니다.
  - API 요청이 동시에 몰리지 않도록 실험 시작 시간을 시차를 두고 설정합니다.

**서비스 중단 여부 확인**:
  - [W&B status updates](https://status.wandb.com)를 확인하여 해당 문제가 일시적인 서버 측 문제는 아닌지 확인합니다.

**지원팀에 문의**:
  - 실험 설정에 대한 상세 정보를 포함하여 W&B 지원팀(support@wandb.com)에 문의하고 요율 제한(rate limits) 상향을 요청하세요.
