title: 배치 단위로는 일부 메트릭만, 에포크 단위로는 다른 메트릭만 기록하고 싶으면 어떻게 하나요?
---

각 배치에서 특정 메트릭을 기록하고 플롯을 표준화하려면, 원하는 x축 값들을 메트릭과 함께 기록하세요. 커스텀 플롯에서 Edit을 클릭한 다음, 사용자 지정 x축을 선택하세요.

```python
import wandb

with wandb.init() as run:
    run.log({"batch": batch_idx, "loss": 0.3})
    run.log({"epoch": epoch, "val_acc": 0.94})
```
