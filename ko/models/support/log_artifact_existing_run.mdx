---
title: 기존에 존재하는 run 에 어떻게 아티팩트 (artifact) 를 로그 (log) 하나요?
---

때때로 이전에 로그된 Run 의 출력으로 Artifacts 를 표시해야 할 때가 있습니다. 이 경우, 다음과 같이 기존 Run 을 재구성하고 새로운 Artifacts 를 로그합니다:

```python
with wandb.init(id="existing_run_id", resume="allow") as run:
    # 기존 Run ID를 사용하여 초기화하고 resume을 허용합니다.
    artifact = wandb.Artifact("artifact_name", "artifact_type")
    artifact.add_file("my_data/file.txt")
    run.log_artifact(artifact)
```
