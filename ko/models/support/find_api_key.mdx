<div id="delete-an-api-key">
  ## API 키 삭제
</div>

<Warning>
  API 키를 삭제하면 해당 키를 사용 중인 모든 스크립트와 서비스의 접근 권한이 즉시 철회됩니다. 기존 키를 삭제하기 전에 모든 시스템에서 새 키를 사용하도록 이미 업데이트했는지 확인하세요.
</Warning>

접근 권한을 철회하려면 API 키를 삭제하세요. 자세한 내용은 **개인 API 키** 또는 **서비스 계정 API 키** 탭을 선택하세요.

<Tabs>
  <Tab title="개인 API 키">
    1. [wandb.ai/settings](https://wandb.ai/settings)로 이동합니다.
    2. **API Keys** 섹션에서 삭제하려는 키 옆의 삭제 버튼을 클릭합니다.
  </Tab>

  <Tab title="서비스 계정 API 키">
    <ServiceAccountApiKeyDelete />

    [서비스 계정을 사용하여 워크플로우를 자동화하는 방법](/ko/platform/hosting/iam/service-accounts)에 대해 자세히 알아보세요.
  </Tab>
</Tabs>

<div id="replace-an-api-key">
  ## API 키 교체
</div>

API 키를 교체(로테이션)하려면:

1. [새 API 키를 생성](#create-a-new-api-key)합니다.
2. 모든 스크립트, 서비스, 환경에서 새 API 키를 사용하도록 업데이트합니다.
3. 새 API 키가 모든 사용처에서 정상적으로 작동하는지 테스트합니다.
4. 기존 API 키를 삭제합니다.
