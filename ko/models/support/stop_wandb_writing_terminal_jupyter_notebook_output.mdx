---
title: How do I stop wandb from writing to my terminal or my Jupyter notebook output?
support:
- environment variables
toc_hide: true
type: docs
---

환경 변수 [`WANDB_SILENT`](/ko/models/support/environment_variables/) 를 `true` 로 설정하세요.

<Tabs>
<Tab title="Python">
```python
os.environ["WANDB_SILENT"] = "true"
```
</Tab>
<Tab title="Notebook">
```python
%env WANDB_SILENT=true
```
</Tab>
<Tab title="Command-Line">
```shell
WANDB_SILENT=true
```
</Tab>
</Tabs>
