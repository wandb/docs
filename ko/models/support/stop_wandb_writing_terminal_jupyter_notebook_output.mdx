---
title: wandb가 터미널이나 Jupyter 노트북에 메시지를 출력하지 않게 하려면 어떻게 하나요?
---

환경 변수 [`WANDB_SILENT`](/ko/models/track/environment-variables/)을 `true`로 설정하세요.

<Tabs>
  <Tab title="Python">
    ```python
    os.environ["WANDB_SILENT"] = "true"
    ```
  </Tab>

  <Tab title="Notebook">
    ```python
    %env WANDB_SILENT=true
    ```
  </Tab>

  <Tab title="Command-Line">
    ```shell
    WANDB_SILENT=true
    ```
  </Tab>
</Tabs>
