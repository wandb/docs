title: 미디어 패널
---

미디어 패널은 3D 객체, 오디오, 이미지, 비디오, 포인트 클라우드 등을 포함한 [미디어 객체에 대해 기록된 키](/ko/models/track/log/media/)를 시각화합니다. 이 페이지에서는 워크스페이스에 미디어 패널을 추가하고 관리하는 방법을 설명합니다.

<div id="add-a-media-panel">
  ## 미디어 패널 추가
</div>

로그된 키에 대해 기본 구성을 사용해 미디어 패널을 추가하려면 Quick Add를 사용합니다. 미디어 패널은 전역 또는 특정 섹션에 추가할 수 있습니다.

1. **Global**: 패널 검색 필드 근처의 컨트롤 바에서 **Add panels**를 클릭합니다.
2. **Section**: 섹션의 작업 `...` 메뉴를 클릭한 다음 **Add panels**를 클릭합니다.
3. 사용 가능한 패널 목록에서 패널에 사용할 키를 찾은 뒤 **Add**를 클릭합니다. 추가하려는 각 미디어 패널에 대해 이 단계를 반복한 다음, 오른쪽 상단의 **X**를 클릭해 **Quick Add** 목록을 닫습니다.
4. 필요하다면 [패널을 구성](#configure-a-media-panel)합니다.

미디어 패널은 전역 또는 특정 섹션에 추가할 수 있습니다:

1. **Global**: 패널 검색 필드 근처의 컨트롤 바에서 **Add panels**를 클릭합니다.
2. **Section**: 섹션의 작업 `...` 메뉴를 클릭한 다음 **Add panels**를 클릭합니다.
3. **Media** 섹션을 클릭해 확장합니다.
4. 패널에서 시각화할 미디어 유형(3D 객체, 이미지, 비디오 또는 오디오)을 선택합니다. 패널 구성 화면이 표시됩니다. 패널을 구성한 다음 **Apply**를 클릭합니다. [미디어 패널 구성](#configure-a-media-panel)을 참조하세요.

<div id="configure-a-media-panel">
  ## 미디어 패널 구성하기
</div>

모든 미디어 유형의 패널에는 동일한 옵션이 제공됩니다.

미디어 패널을 수동으로 추가하면, 미디어 유형을 선택하면 해당 패널의 구성 페이지가 열립니다. 기존 패널의 구성을 변경하려면 패널 위에 마우스를 올린 다음, 오른쪽 상단에 나타나는 톱니바퀴 아이콘을 클릭합니다. 이 섹션에서는 각 탭에서 사용할 수 있는 설정을 설명합니다.

<div id="overlays">
  ### 오버레이
</div>

세그멘테이션 마스크 또는 바운딩 박스가 포함된 이미지나 포인트 클라우드를 로그하면 이 탭이 표시됩니다.

* 이름으로 오버레이를 검색하고 필터링합니다.
* 오버레이 색상을 사용자 지정합니다.

<div id="sync">
  ### 동기화
</div>

이 탭은 워크스페이스와 섹션 설정에 표시됩니다.

* **키로 슬라이더 동기화**: 섹션에 있는 비디오의 step 슬라이더를 키 기준으로 동기화하여 함께 움직이도록 할지 설정합니다.
* **비디오 자동 재생**: 페이지가 로드될 때 비디오를 자동으로 재생할지 설정합니다.
* **비디오 반복 재생**: 섹션의 비디오가 자동으로 다시 시작되어 중지할 때까지 계속 재생되도록 할지 설정합니다. 섹션 단위로는 설정할 수 없습니다. 워크스페이스에 비디오 미디어 패널(video media panel)이 있는 경우에만 표시됩니다.

<div id="display">
  ### Display
</div>

패널의 전반적인 표시 방식과 동작을 설정합니다.

* 패널 제목을 설정합니다.
* 시각화할 미디어 키를 선택합니다.
* 패널의 슬라이더 및 재생 동작을 사용자 지정합니다.
  * 기본값이 **Step**인 슬라이더 키를 설정합니다.
  * 슬라이더를 한 번 클릭할 때마다 이동할 단계 수를 **Stride length**로 설정합니다.
  * **Snap to existing step**을 켜거나 끕니다. 켜져 있으면, 슬라이더는 **Stride length** 이후에 다음으로 존재하는 기존 step으로 이동합니다. 꺼져 있으면, 기존 step과 맞지 않더라도 정확히 **Stride length**만큼만 이동합니다.
* **Images**: 스무딩을 켜거나 끕니다.
* **3d objects**: 배경색과 포인트 색상을 설정합니다.

<div id="layout">
  ### 레이아웃
</div>

패널의 개별 항목 표시 방식을 설정합니다.

* **Grid mode**를 켜거나 끕니다.
  * 켜져 있으면 각 항목 위에 표시할 사용자 지정 X, Y 축을 선택할 수 있습니다. 각 행에 둘 이상의 항목이 표시되며, 표시할 행 수를 제한할 수 있습니다.
  * 꺼져 있으면 패널 콘텐츠에 사용할 열(column) 개수를 지정할 수 있고, 기본값이 **Run**인 패널 콘텐츠 자체도 구성할 수 있습니다.
* 패널에 포함할 **Max runs to include**(실행 최대 개수)를 선택적으로 제한합니다.
* 실행마다 포함할 미디어 항목 수를 제한하기 위해 선택적으로 **Media display limit**을 지정합니다.
* **Images and videos**: 전체 크기 미디어 표시 여부를 켜거나 끕니다.
* **Images**: **Fit media**가 켜져 있으면 패널 크기에 맞도록 패널의 미디어 크기를 조정합니다.
* **Point clouds**: 기본 왼손 좌표계 대신, 선택적으로 오른손 좌표계를 사용하여 포인트를 시각화합니다.

<div id="all-media-panels-in-a-section">
  ### 섹션의 모든 미디어 패널
</div>

섹션의 모든 미디어 패널에 대한 기본 설정을 워크스페이스의 미디어 패널 설정보다 우선하도록 커스터마이즈하려면:

1. 섹션의 톱니바퀴 아이콘을 클릭해서 설정을 엽니다.
2. **Media settings**를 클릭합니다.
3. 표시되는 드로어에서 **Display**, **Layout**, 또는 **Sync** 탭을 클릭해 섹션의 기본 미디어 설정을 구성합니다. 이미지, 비디오, 오디오, 3D 객체에 대한 설정을 구성할 수 있습니다. 표시되는 설정은 섹션의 현재 미디어 패널에 따라 달라집니다.

**Display** 또는 **Layout** 미디어 설정의 개별 설정에 대한 자세한 내용은 [Configure a media panel](#configure-a-media-panel)을 참조하세요. **Sync** 탭은 개별 미디어 패널이 아니라 섹션 또는 워크스페이스 수준에서만 사용할 수 있습니다.

**Step slider syncing**이 켜져 있으면, 동일한 Step 슬라이더를 사용하는 섹션의 미디어 패널이 서로 동기화됩니다. Step 슬라이더 동기화를 켜려면:

1. **Sync** 탭을 클릭합니다.
   1. **Sync slider by key (Step)**을 켭니다.

<div id="all-media-panels-in-a-workspace">
  ### 워크스페이스의 모든 미디어 패널
</div>

워크스페이스의 모든 미디어 패널에 대한 기본 설정을 변경하려면 다음 단계를 따르세요.

1. 톱니바퀴 아이콘과 **Settings** 레이블이 있는 워크스페이스 설정을 클릭합니다.
2. **Media settings**를 클릭합니다.
3. 표시되는 드로어에서 **Display** 또는 **Layout** 탭을 클릭해 워크스페이스의 기본 미디어 설정을 구성합니다. 이미지, 비디오, 오디오 및 3D 오브젝트에 대한 설정을 구성할 수 있습니다. 표시되는 설정은 워크스페이스의 현재 미디어 패널에 따라 달라집니다.

**Sync** 탭을 제외한 각 설정에 대한 자세한 내용은 [미디어 패널 구성](#configure-a-media-panel)을 참조하세요. **Sync** 탭은 개별 미디어 패널이 아니라 섹션 또는 워크스페이스 수준에서만 사용할 수 있습니다.

**Step slider syncing**을 켜면 동일한 스텝 슬라이더를 사용하는 섹션의 미디어 패널이 서로 동기화된 상태로 유지됩니다. 스텝 슬라이더 동기화를 켜려면:

1. **Sync** 탭을 클릭합니다.
2. **Sync slider by key (Step)**을 켭니다.

각 설정에 대한 자세한 내용은 [미디어 패널 구성](#configure-a-media-panel)을 참조하세요.

<div id="interact-with-a-media-panel">
  ## 미디어 패널과 상호작용하기
</div>

* 미디어 패널을 클릭하여 전체 화면 모드로 표시합니다. 패널 상단의 화살표 버튼을 클릭하여 전체 화면 모드를 종료합니다.
* 전체 화면 모드를 종료하지 않고 섹션 내 패널을 이동하려면, 패널 아래의 **Previous** 및 **Next** 버튼이나 키보드의 왼쪽/오른쪽 화살표 키를 사용합니다.
* 미디어 패널의 스텝 슬라이더를 이동하려면 **CMD + 왼쪽/오른쪽 화살표 키**(macOS) 또는 **Ctrl + 왼쪽/오른쪽 화살표 키**(Windows / Linux)를 사용합니다. 섹션 또는 워크스페이스에 대해 **Sync slider by key**가 켜져 있으면, 한 미디어 패널에서 스텝 슬라이더를 이동할 때 동일한 스텝 슬라이더 키를 사용하는 다른 미디어 패널의 스텝 슬라이더도 함께 이동합니다.
* 미디어 패널 상단의 스테퍼를 사용해 미디어 실행을 단계별로 살펴볼 수 있습니다. 스텝 슬라이더를 이동하려면 UI 컨트롤을 사용합니다.
* 미디어 컨트롤을 사용해 비디오를 재생, 일시 정지 또는 중지할 수 있습니다. **Sync video playback**가 켜져 있으면 섹션의 모든 비디오가 동기화되어 재생됩니다. **Loop videos**가 켜져 있으면 섹션의 비디오가 자동으로 다시 시작되며, 중지할 때까지 계속 반복 재생됩니다.
* 미디어 패널을 설정하려면 패널 위에 마우스를 올리고 상단의 톱니바퀴 아이콘을 클릭합니다.
* 세그멘테이션 마스크가 함께 로그된 이미지의 경우, 각 마스크의 표시 방식을 사용자 지정하거나 개별적으로 켜고 끌 수 있습니다. 패널 위에 마우스를 올린 다음, 하단의 톱니바퀴 아이콘을 클릭합니다.
* 바운딩 박스가 함께 로그된 이미지나 포인트 클라우드의 경우, 각 바운딩 박스의 표시 방식을 사용자 지정하거나 개별적으로 켜고 끌 수 있습니다. 패널 위에 마우스를 올린 다음, 하단의 톱니바퀴 아이콘을 클릭합니다.
