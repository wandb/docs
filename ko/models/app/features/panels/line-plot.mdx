시간 경과에 따라 `wandb.Run.log()`로 로깅된 메트릭은 기본적으로 선형 플롯으로 표시됩니다. 선형 플롯은 여러 메트릭을 플로팅하고, 맞춤형 축을 계산하는 등 다양한 기능을 지원합니다.

이 페이지에서는 [Workspace](/ko/models/track/workspaces)에서 선형 플롯을 생성, 설정 및 관리하는 방법을 설명합니다.

<div id="add-a-line-plot">
  ## 선형 플롯 추가
</div>

이 섹션에서는 단일 메트릭 또는 여러 메트릭에 대한 선형 플롯을 생성하는 방법을 설명합니다.

<Tabs>
  <Tab title="단일 메트릭 선형 플롯">
    [자동 Workspace](/ko/models/app/features/panels#workspace-modes)에서는 로깅된 각 메트릭에 대해 단일 메트릭 선형 플롯이 자동으로 생성됩니다. 다음 step을 따라 자동 Workspace에서 삭제된 선형 플롯을 다시 추가하거나, 수동 Workspace에 선형 플롯을 추가하세요.

    1. Workspace로 이동하세요.
    2. 선형 플롯을 전체적으로 추가하려면, 패널 검색 필드 근처의 컨트롤 바에서 **Add panels**를 클릭하세요.

       대신 섹션에 직접 선형 플롯을 추가하려면, 해당 섹션의 액션 `...` 메뉴를 클릭한 다음 **+ Add panels**를 클릭하세요.
    3. 기본 설정으로 단일 메트릭 플롯을 추가하려면, **Quick panel builder**를 클릭하세요.
       1. **Single-key panels** 탭에서 메트릭 위에 마우스를 올린 다음 **Add**를 클릭하세요. 추가하려는 각 패널에 대해 이 step을 반복하세요.
       2. **Create &lt;number&gt; panels**를 클릭하세요.
    4. 대신 맞춤형 선형 플롯을 추가하려면, **Line plot**을 클릭하세요.
       1. 해당 탭을 사용해 선형 플롯의 데이터, 그룹화, 표시 기본 설정을 설정하세요. 자세한 내용은 [선형 플롯 설정 편집](#edit-line-plot-settings)을 참조하세요.
       2. x축 또는 y축에 계산된 표현식을 추가하려면 **Expressions**를 클릭하세요. [JavaScript 정규 표현식](https://www.w3schools.com/js/js_regexp.asp)이 지원됩니다.
          추가할 패널 유형(예: 차트)을 선택하세요. 패널의 설정 세부 정보가 선택된 기본값과 함께 표시됩니다.
    5. 필요에 따라 패널과 해당 표시 기본 설정을 사용자 지정하세요. 설정 옵션은 선택한 패널 유형에 따라 달라집니다. 각 패널 유형의 옵션에 대해 자세히 알아보려면 아래의 관련 섹션(예: [Line plots](/ko/models/app/features/panels/line-plot/) 또는 [Bar plots](/ko/models/app/features/panels/bar-plot/))을 참고하세요.
    6. **Apply**를 클릭하세요.
  </Tab>

  <Tab title="다중 메트릭 선형 플롯">
    <Note>
      이 기능은 미리보기 상태이며, 초대를 받은 경우에만 사용할 수 있습니다. 등록을 요청하려면 [지원팀](mailto:support@wandb.com) 또는 AISE에 문의하세요.
    </Note>

    [automatic workspace](/ko/models/app/features/panels#workspace-modes)에서는 로깅된 각 메트릭에 대해 단일 메트릭 선형 플롯이 자동으로 생성됩니다. 이 섹션에서는 JavaScript 정규 표현식으로 정의된 여러 메트릭을 함께 표시하는 단일 선형 플롯을 생성하는 방법을 설명합니다. 필요에 따라 여러 단일 메트릭 플롯을 하나의 다중 메트릭 플롯으로 통합할 수 있습니다. 이렇게 하면 로깅된 메트릭이 많은 Workspace의 성능을 개선할 수 있고, Runs 결과를 효율적으로 분석하는 데 도움이 됩니다.

    1. Workspace로 이동합니다.
    2. 선형 플롯을 전역으로 추가하려면 패널 검색 필드 근처의 컨트롤 바에서 **Add panels**를 클릭합니다.

       대신 선형 플롯을 섹션에 직접 추가하려면 섹션의 액션 `...` 메뉴를 클릭한 다음 **+ Add panels**를 클릭합니다.
    3. **Quick panel builder**를 클릭한 다음 **Multi-metric panels** 탭을 클릭합니다.
    4. **Regex**에 [JavaScript 정규 표현식](https://www.w3schools.com/js/js_regexp.asp) 형식의 표현식을 입력합니다. 입력하는 동안 UI가 업데이트되어 어떤 메트릭이 표현식과 일치하는지 표시합니다. 기본적으로 플롯 이름에는 플롯에서 사용한 정규 표현식이 표시됩니다. 플롯에는 향후 로깅되는 메트릭을 포함해 표현식과 일치하는 모든 메트릭의 선이 포함됩니다.
    5. 다중 메트릭 플롯이 생성될 때 중복된 단일 메트릭 패널을 선택적으로 제거하려면 **Clean up auto-generated panels** 표시/숨기기를 사용합니다. 어떤 패널이 정리되는지 미리보기로 표시됩니다. <Note>이 옵션을 켜면 표현식과 일치하는 새로 로깅된 메트릭에 대해 단일 메트릭 플롯이 생성되지 않습니다. 대신 이 다중 메트릭 플롯에만 포함됩니다.</Note>
    6. **Create &lt;number&gt; panels**를 클릭합니다.

    ### 다중 메트릭 정규 표현식 자세히 알아보기

    다중 메트릭 선형 플롯은 메트릭 이름과 일치시키기 위해 [JavaScript 정규 표현식](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Regular_expressions)를 사용합니다. 이 섹션에서는 몇 가지 일반적인 사용 사례를 보여주고, 캡처 그룹이 생성되는 패널에 어떤 영향을 미치는지와 같이 정규 표현식이 작동하는 방식에 대한 자세한 내용을 설명합니다.

    #### 일반적인 사용 사례

    이 섹션에서는 다중 메트릭 패널을 사용해 실험 결과를 분석하는 데 도움이 되는 몇 가지 방법을 보여줍니다.

    **레이어 또는 모델 컴포넌트 전반의 메트릭 비교**
    각 레이어의 메트릭마다 별도의 패널을 만드는 대신, 하나의 패널에서 함께 볼 수 있습니다. 예를 들어, 이 Python 예시 코드에서 `layer_0_loss`, `layer_1_loss`, `layer_2_loss`처럼 일관된 이름으로 메트릭을 로깅하는 경우, 정규식 `layer_\d+_loss`를 사용해 모든 레이어 손실을 하나의 플롯에 표시할 수 있습니다.

    ```python
    with wandb.init(project="multi-layer-model") as run:
        for step in range(100):
            run.log({
                "layer_0_loss": loss_0,
                "layer_1_loss": loss_1,
                "layer_2_loss": loss_2,
                "step": step
            })
    ```

    **접두사 또는 접미사로 관련 메트릭 그룹화**
    공통된 명명 패턴을 공유하는 모든 메트릭을 일치시키세요. 예를 들면 다음과 같습니다.

    * `train_.*`는 `train_loss`, `train_accuracy`, `train_f1_score`와 같은 모든 트레이닝 메트릭과 일치합니다.
    * `.*_accuracy`는 `train_accuracy`, `val_accuracy`, `test_accuracy`처럼 서로 다른 데이터셋 전반의 정확도 메트릭과 일치합니다.

    **특정 메트릭 변형 일치**
    원하는 메트릭만 일치시키려면 alternation을 사용하세요. 예를 들어, non-capturing group `(?:layer_0|layer_10)_loss`는 중간 레이어를 제외하고 첫 번째와 열 번째 레이어의 loss에만 일치합니다.

<div id="individual-line-plot">
  ### 개별 선형 플롯
</div>

개별 선형 플롯 설정은 섹션 또는 Workspace 수준의 선형 플롯 설정보다 우선합니다. 선형 플롯을 사용자 지정하려면:

1. Workspace로 이동합니다.
2. 패널 위에 마우스를 올린 다음 톱니바퀴 아이콘을 클릭합니다.
3. 나타나는 drawer에서 설정을 편집할 탭을 선택합니다.
4. **Apply**를 클릭합니다.

선형 플롯 설정은 다음과 같은 탭으로 구성됩니다:

* **Data**: x축, y축, 샘플링 방법, 스무딩, 이상치, 차트 유형을 설정합니다.
* **Grouping**: 플롯에서 run을 어떻게 그룹화하고 집계할지 설정합니다.
* **Chart**: 패널과 축의 제목을 지정하고, 범례의 표시 여부와 위치를 설정합니다.
* **Legend**: 패널 범례의 모양과 내용을 사용자 지정합니다.
* **Expressions**: 축에 사용할 맞춤형 계산식을 추가합니다.

각 설정에 대한 자세한 내용은 [라인 플롯 레퍼런스](/ko/models/app/features/panels/line-plot/reference)를 참조하세요.

<div id="all-line-plots-in-a-section">
  ### 섹션의 모든 선형 플롯
</div>

섹션의 모든 선형 플롯에 대한 기본 설정을 사용자 지정하여, 선형 플롯에 대한 Workspace 설정을 재정의하려면:

1. Workspace로 이동합니다.
2. 섹션의 톱니바퀴 아이콘을 클릭하여 설정을 엽니다.
3. 나타나는 드로어에서 **Data** 또는 **Display preferences** 탭을 선택해 섹션의 기본 설정을 설정합니다. 각 **Data** 설정에 대한 자세한 내용은 [라인 플롯 레퍼런스](/ko/models/app/features/panels/line-plot/reference)를 참조하세요. 각 표시 환경설정에 대한 자세한 내용은 [Configure section layout](../#configure-section-layout)을 참조하세요.

<div id="all-line-plots-in-a-workspace">
  ### 워크스페이스의 모든 선형 플롯
</div>

워크스페이스의 모든 선형 플롯에 대한 기본 설정을 사용자 지정하려면 다음을 수행하세요.

1. 워크스페이스로 이동합니다.
2. 기어 아이콘과 **Settings** 레이블이 있는 Workspace 설정 아이콘을 클릭합니다.
3. **Line plots**를 클릭합니다.
4. 표시되는 드로어에서 **Data** 또는 **Display preferences** 탭을 선택하여 워크스페이스의 기본 설정을 설정합니다.
   * 각 **Data** 설정에 대한 자세한 내용은 [라인 플롯 레퍼런스](/ko/models/app/features/panels/line-plot/reference)를 참조하세요.
   * 각 **Display preferences** 섹션에 대한 자세한 내용은 [Workspace display preferences](../#configure-workspace-layout)를 참조하세요. Workspace 수준에서는 선형 플롯의 기본 **Zooming** 동작을 설정할 수 있습니다. 이 설정은 x-axis 키가 일치하는 선형 플롯 간에 확대/축소를 동기화할지 여부를 제어합니다. 기본적으로 비활성화되어 있습니다.

<Frame>
  <img src="/images/app_ui/howto_use_relative_time_or_wall_time.gif" alt="X-axis time options" />
</Frame>

맞춤형 x축을 사용하려면 y축을 로깅하는 것과 같은 `wandb.Run.log()` 호출에서 해당 메트릭을 함께 로깅하십시오. 예를 들면 다음과 같습니다:

```python
with wandb.init() as run:
    for i in range(100):
        run.log({"accuracy": acc, "custom_x": i * 10})
```

자세한 내용은 [로그 축 사용자 지정](/ko/models/track/log/customize-logging-axes#customize-log-axes)을 참조하세요.

  ## run 메트릭 알림 만들기
</div>

[Automations](/ko/models/automations/)를 사용하면 run 메트릭이 사용자가 지정한 조건을 충족할 때 팀에 알림을 보낼 수 있습니다. 자동화는 Slack 채널에 게시하거나 웹훅을 실행할 수 있습니다.

선형 플롯에 표시된 메트릭에 대한 [run 메트릭 알림](/ko/models/automations/automation-events/#run-events)을 빠르게 생성할 수 있습니다:

1. Workspace로 이동하세요.
2. 패널에 마우스를 올린 다음 종 아이콘을 클릭하세요.
3. 기본 또는 고급 설정 컨트롤을 사용해 자동화를 설정하세요. 예를 들어 자동화의 범위를 제한하도록 run 필터를 적용하거나 절대 임곗값을 설정할 수 있습니다.

[Automations](/ko/models/automations/)에 대해 자세히 알아보세요.
