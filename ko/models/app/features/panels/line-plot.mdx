선 그래프는 기본적으로 `wandb.Run.log()`으로 시간 경과에 따라 기록한 메트릭을 표시합니다. 선 그래프는 여러 메트릭을 동시에 표시하고, 사용자 지정 축을 계산하는 등 다양한 기능을 지원합니다.

이 페이지에서는 [워크스페이스](/ko/models/track/workspaces)에서 선 그래프를 생성, 구성 및 관리하는 방법을 설명합니다.

<div id="add-a-line-plot">
  ## 선 그래프 추가
</div>

이 섹션에서는 하나의 메트릭 또는 여러 메트릭에 대한 선 그래프를 만드는 방법을 보여줍니다.

<Tabs>
  <Tab title="단일 메트릭 선형 차트">
    [자동 워크스페이스](/ko/models/app/features/panels#workspace-modes)에서는 로그된 각 메트릭마다 단일 메트릭 선 그래프가 자동으로 생성됩니다. 자동 워크스페이스에서 삭제된 선 그래프를 다시 추가하거나, 수동 워크스페이스에 선 그래프를 추가하려면 다음 단계를 따르세요.

    1. 워크스페이스로 이동합니다.
    2. 워크스페이스 전체에 선 그래프를 추가하려면 패널 검색 필드 근처의 컨트롤 바에서 **Add panels**를 클릭합니다.

       대신 특정 섹션에 바로 선 그래프를 추가하려면, 섹션의 작업 `...` 메뉴를 클릭한 다음 **+ Add panels**를 클릭합니다.
    3. 기본 설정으로 단일 메트릭 플롯을 추가하려면 **Quick panel builder**를 클릭합니다.
       1. **Single-key panels** 탭에서 메트릭 위에 마우스를 올린 다음 **Add**를 클릭합니다. 추가하려는 각 패널에 대해 이 단계를 반복합니다.
       2. **Create &lt;number&gt; panels**를 클릭합니다.
    4. 대신 커스텀 선 그래프를 추가하려면 **Line plot**을 클릭합니다.
       1. 해당 탭을 사용하여 선 그래프의 데이터, 그룹화, 표시 설정을 구성합니다. 자세한 내용은 [선 그래프 설정 편집](#edit-line-plot-settings)을 참조하세요.
       2. x 또는 y 축에 계산된 표현식을 추가하려면 **Expressions**를 클릭합니다. [JavaScript 정규식](https://www.w3schools.com/js/js_regexp.asp)을 지원합니다.
          차트와 같이 추가할 패널 유형을 선택합니다. 선택된 기본값과 함께 패널의 구성 세부 정보가 표시됩니다.
    5. 필요에 따라 패널과 표시 설정을 사용자 정의합니다. 구성 옵션은 선택한 패널 유형에 따라 달라집니다. 각 패널 유형에 대한 옵션을 더 알아보려면 [Line plots](/ko/models/app/features/panels/line-plot/) 또는 [Bar plots](/ko/models/app/features/panels/bar-plot/) 등의 아래 관련 섹션을 참조하세요.
    6. **Apply**를 클릭합니다.
  </Tab>

  <Tab title="다중 지표 선 그래프">
    <Note>
      이 기능은 초대를 받은 사용자에게만 제공되는 프리뷰 기능입니다. 사용을 원하면 [support](mailto:support@wandb.com) 또는 담당 AISE에게 문의하세요.
    </Note>

    [자동 워크스페이스](/ko/models/app/features/panels#workspace-modes)에서는 각 기록된 메트릭마다 단일 메트릭 선형 플롯이 자동으로 생성됩니다. 이 섹션에서는 JavaScript 정규식을 사용해 여러 메트릭을 함께 보여주는 단일 선형 플롯을 생성하는 방법을 설명합니다. 선택적으로, 여러 단일 메트릭 플롯을 하나의 다중 메트릭 플롯으로 통합할 수 있습니다. 이렇게 하면 많은 메트릭이 기록된 워크스페이스의 성능을 개선하고, 실행 결과를 더 효율적으로 분석하는 데 도움이 됩니다.

    1. 워크스페이스로 이동합니다.
    2. 워크스페이스 전체에 선형 플롯을 추가하려면, 패널 검색 필드 근처에 있는 컨트롤 바에서 **Add panels**를 클릭합니다.

       대신 특정 섹션에 직접 선형 플롯을 추가하려면, 섹션의 `...` 작업 메뉴를 클릭한 다음 **+ Add panels**를 클릭합니다.
    3. **Quick panel builder**를 클릭한 다음 **Multi-metric panels** 탭을 클릭합니다.
    4. **Regex**에 [JavaScript 정규식](https://www.w3schools.com/js/js_regexp.asp) 형식의 표현식을 입력합니다. 입력하는 동안 UI가 해당 표현식과 일치하는 메트릭을 표시하도록 업데이트됩니다. 기본적으로 플롯 이름에는 플롯에서 사용된 정규식이 표시됩니다. 이 플롯에는 표현식과 일치하는 모든 메트릭의 선이 포함되며, 앞으로 기록될 메트릭도 자동으로 포함됩니다.
    5. 다중 메트릭 플롯을 생성할 때 중복되는 단일 메트릭 패널을 선택적으로 제거하려면 **Clean up auto-generated panels**를 토글합니다. 미리 보기에 어떤 패널이 정리될지 표시됩니다. <Note>이 옵션이 켜져 있으면, 표현식과 일치하는 새로 기록된 메트릭에 대해서는 단일 메트릭 플롯이 생성되지 않습니다. 대신 이 다중 메트릭 플롯에만 포함됩니다.</Note>
    6. **Create &lt;number&gt; panels**를 클릭합니다.

    ### 다중 메트릭 정규식에 대한 추가 설명

    다중 메트릭 선형 플롯은 메트릭 이름을 매칭하기 위해 [JavaScript 정규식](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Regular_expressions)을 사용합니다. 이 섹션에서는 몇 가지 일반적인 사용 사례를 보여 주고, 캡처 그룹이 생성되는 패널에 어떤 영향을 미치는지 등 정규식이 작동하는 방식에 대해 더 자세히 설명합니다.

    #### 일반적인 사용 사례

    이 섹션에서는 실험 결과를 분석하는 데 다중 메트릭 패널을 활용하는 몇 가지 방법을 보여 줍니다.

    **레이어 또는 모델 컴포넌트 간 메트릭 비교**
    각 레이어의 메트릭마다 별도의 패널을 만드는 대신, 하나의 패널에서 함께 볼 수 있습니다. 예를 들어, 이 Python 예제 코드처럼 `layer_0_loss`, `layer_1_loss`, `layer_2_loss`와 같이 일관된 이름 규칙으로 메트릭을 기록하는 경우, 정규식 `layer_\d+_loss`를 사용하여 모든 레이어의 loss를 하나의 플롯에 표시할 수 있습니다.

    ```python
    with wandb.init(project="multi-layer-model") as run:
        for step in range(100):
            run.log({
                "layer_0_loss": loss_0,
                "layer_1_loss": loss_1,
                "layer_2_loss": loss_2,
                "step": step
            })
    ```

    **접두사나 접미사로 관련 메트릭 그룹화**
    공통된 이름 패턴을 가진 모든 메트릭을 정규식으로 한 번에 매칭할 수 있습니다. 예를 들어:

    * `train_.*` 은 `train_loss`, `train_accuracy`, `train_f1_score` 와 같은 모든 학습 메트릭과 매칭됩니다.
    * `.*_accuracy` 는 `train_accuracy`, `val_accuracy`, `test_accuracy` 와 같이 서로 다른 데이터셋에 대한 정확도 메트릭과 매칭됩니다.

    **특정 메트릭 변형만 매칭하기**
    원하는 메트릭만 매칭하기 위해 alternation(또는 `|` 연산자)을 사용합니다. 예를 들어, 비캡처 그룹 `(?:layer_0|layer_10)_loss` 는 중간 레이어를 제외하고 첫 번째와 열 번째 레이어의 loss 만 매칭합니다.

이 섹션에서는 개별 라인 플롯 패널, 섹션 내 모든 라인 플롯 패널, 또는 워크스페이스 내 모든 라인 플롯 패널의 설정을 편집하는 방법을 설명합니다. 라인 플롯 설정에 대한 자세한 내용은 [라인 플롯 참고](/ko/models/app/features/panels/line-plot/reference)를 참조하세요.

<div id="individual-line-plot">
  ### 개별 라인 플롯
</div>

개별 라인 플롯 설정은 섹션 또는 워크스페이스의 라인 플롯 설정보다 우선 적용됩니다. 라인 플롯을 사용자 지정하려면 다음을 수행합니다:

1. 워크스페이스로 이동합니다.
2. 패널 위에 마우스를 올린 다음 톱니바퀴 아이콘을 클릭합니다.
3. 표시되는 드로어에서 설정을 편집할 탭을 선택합니다.
4. **Apply**를 클릭합니다.

라인 플롯 설정은 다음과 같은 탭으로 구성됩니다:

* **Data**: x축, y축, 샘플링 방법, 스무딩, 이상치, 차트 유형을 구성합니다.
* **Grouping**: 플롯에서 실행을 그룹화하고 집계할지 여부와 방식을 구성합니다.
* **Chart**: 패널과 축의 제목을 지정하고, 범례의 표시 여부와 위치를 구성합니다.
* **Legend**: 패널 범례의 모양과 내용을 사용자 지정합니다.
* **Expressions**: 축에 사용할 사용자 정의 계산식을 추가합니다.

각 설정 항목에 대한 자세한 내용은 [Line plot reference](/ko/models/app/features/panels/line-plot/reference)를 참조하세요.

<div id="all-line-plots-in-a-section">
  ### 섹션의 모든 선 그래프
</div>

섹션의 모든 선 그래프에 대한 기본 설정을 사용자 지정하여 워크스페이스의 선 그래프 설정을 재정의하려면:

1. 워크스페이스로 이동합니다.
2. 섹션의 톱니바퀴 아이콘을 클릭하여 설정을 엽니다.
3. 표시되는 드로어에서 **Data** 또는 **Display preferences** 탭을 선택해 섹션의 기본 설정을 구성합니다. 각 **Data** 설정에 대한 자세한 내용은 [Line plot reference](/ko/models/app/features/panels/line-plot/reference)를 참조하세요. 각 표시 기본 설정에 대한 자세한 내용은 [Configure section layout](../#configure-section-layout)을 참조하세요.

<div id="all-line-plots-in-a-workspace">
  ### 워크스페이스의 모든 선형 플롯
</div>

워크스페이스에 있는 모든 선형 플롯의 기본 설정을 사용자 지정하려면:

1. 워크스페이스로 이동합니다.
2. 기어 모양 아이콘과 **Settings** 레이블이 있는 워크스페이스 설정 아이콘을 클릭합니다.
3. **Line plots**를 클릭합니다.
4. 나타나는 드로어 패널에서 **Data** 또는 **Display preferences** 탭을 선택하여 워크스페이스의 기본 설정을 구성합니다.
   * 각 **Data** 설정에 대한 자세한 내용은 [Line plot reference](/ko/models/app/features/panels/line-plot/reference)를 참조하세요.
   * 각 **Display preferences** 섹션에 대한 자세한 내용은 [Workspace display preferences](../#configure-workspace-layout)를 참조하세요. 워크스페이스 수준에서는 선형 플롯에 대한 기본 **Zooming** 동작을 구성할 수 있습니다. 이 설정은 동일한 x-axis key를 가진 선형 플롯 간에 줌을 동기화할지 여부를 제어합니다. 기본적으로 비활성화되어 있습니다.

<div id="create-a-run-metrics-notification">
  ## 실행 메트릭 알림 만들기
</div>

[Automations](/ko/models/automations/)를 사용하여 실행 메트릭이 지정한 조건을 충족할 때 팀에 알림을 보낼 수 있습니다. 자동화 규칙은 Slack 채널에 메시지를 보내거나 웹훅을 실행할 수 있습니다.

라인 플롯에서, 플롯에 표시된 메트릭에 대해 [실행 메트릭 알림](/ko/models/automations/automation-events/#run-events)을 빠르게 만들 수 있습니다:

1. 워크스페이스로 이동합니다.
2. 패널 위에 마우스를 올린 다음, 종 모양 아이콘을 클릭합니다.
3. 기본 또는 고급 구성 옵션을 사용하여 자동화를 설정합니다. 예를 들어 실행 필터를 적용해 자동화의 범위를 제한하거나, 절대 임계값을 설정할 수 있습니다.

[Automations](/ko/models/automations/)에 대해 더 알아보세요.
