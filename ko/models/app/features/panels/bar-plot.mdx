막대 그래프는 범주형 데이터를 직사각형 막대로 나타내며, 세로 또는 가로 방향으로 그릴 수 있습니다. 기록된 모든 값의 길이가 1일 때 `wandb.Run.log()`를 사용하면 기본적으로 막대 그래프가 표시됩니다.

    <img src="/images/app_ui/bar_plot.png" alt="W&B에서 박스 플롯과 수평 막대 그래프 그리기"  />
</Frame>

차트 설정을 사용해 표시할 실행 수의 최대값을 제한하고, 원하는 config 기준으로 실행을 그룹화하며, 레이블 이름을 변경할 수 있습니다.

<Frame>
    <img src="/images/app_ui/bar_plot_custom.png" alt="사용자 지정된 막대 그래프"  />

<div id="customize-bar-plots">
  ## 막대형 차트 사용자 지정
</div>

여러 개의 요약 통계를 하나의 차트 유형으로 결합하기 위해 **Box** 또는 **Violin** 플롯을 생성할 수도 있습니다.

1. Runs 테이블을 통해 실행을 그룹화합니다.
2. Workspace에서 'Add panel'을 클릭합니다.
3. 표준 'Bar Chart'를 추가하고 표시할 메트릭을 선택합니다.
4. 'Grouping' 탭에서 'Box plot' 또는 'Violin' 등을 선택해 이 중 하나의 스타일로 시각화합니다.
