평행좌표 차트는 다수의 하이퍼파라미터와 모델 지표 간의 관계를 한눈에 요약해 보여줍니다.

<Frame>
  <img src="/images/app_ui/parallel_coordinates.gif" alt="평행좌표 플롯" />
</Frame>

* **축(Axes)**: [`wandb.Run.config`](/ko/models/evaluate-models/)의 다양한 하이퍼파라미터와 [`wandb.Run.log()`](/ko/models/evaluate-models/)에서 기록한 메트릭입니다.
* **선(Lines)**: 각 선은 하나의 실행을 나타냅니다. 선 위에 마우스를 올리면 해당 실행에 대한 세부 정보를 툴팁으로 볼 수 있습니다. 현재 필터와 일치하는 모든 선이 표시되지만, 눈 아이콘을 끄면 선이 회색으로 표시됩니다.

<div id="create-a-parallel-coordinates-panel">
  ## 평행좌표 패널 만들기
</div>

1. 워크스페이스 시작 페이지로 이동합니다.
2. **Add Panels**을 클릭합니다.
3. **Parallel coordinates**를 선택합니다.

<div id="panel-settings">
  ## 패널 설정
</div>

패널을 설정하려면 패널 오른쪽 상단의 편집 버튼을 클릭하세요.

* **툴팁**: 마우스를 올리면 각 실행에 대한 정보가 담긴 범례가 표시됩니다
* **제목**: 축 제목을 더 읽기 쉽게 편집하세요
* **그라디언트**: 원하는 색상 범위로 그라디언트를 커스터마이즈하세요
* **로그 스케일**: 각 축을 서로 독립적으로 로그 스케일로 설정할 수 있습니다
* **축 뒤집기**: 축 방향을 전환합니다. 정확도와 손실을 모두 열로 사용할 때 유용합니다
