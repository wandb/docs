W&amp;B는 키보드 단축키를 지원하여 Experiments, Workspace, 데이터 전반을 더 효율적으로 탐색하고 상호작용할 수 있도록 돕습니다. 이 가이드에서는 W&amp;B App과 W&amp;B LEET (Lightweight Experiment Exploration Tool) 터미널 UI의 키보드 단축키를 설명합니다.

<Tabs>
  <Tab title="App">
    W&amp;B App에는 현재 컨텍스트에서 사용 가능한 모든 단축키를 나열하는 키보드 단축키 드로어가 있습니다. **Cmd+/** (macOS) 또는 **Ctrl+/** (Windows/Linux)로 열거나, 상단 내비게이션의 **?** (도움말) 메뉴에서 **Keyboard shortcuts**를 선택하세요. 드로어의 검색 필드를 사용해 이름으로 단축키를 필터링할 수 있습니다.

    **Workspace** 및 **Media browser modal** 섹션은 컨텍스트에 따라 달라지며, 해당 영역이 활성화된 경우에만 표시됩니다.

    ## 필수

    | 단축키                                            | 설명                           |
    | ---------------------------------------------- | ---------------------------- |
    | **Cmd+/** (macOS) / **Ctrl+/** (Windows/Linux) | 키보드 단축키 드로어를 엽니다.            |
    | **Cmd+K** (macOS) / **Ctrl+K** (Windows/Linux) | 전역 검색 대화상자를 엽니다.             |
    | **Alt+M**                                      | 밝은 앱 테마와 어두운 앱 테마 사이를 전환합니다. |

    ## Workspace

    | 단축키                                                        | 설명                                    |
    | ---------------------------------------------------------- | ------------------------------------- |
    | **Cmd+Z** (macOS) / **Ctrl+Z** (Windows/Linux)             | 가장 최근의 Workspace 액션을 실행 취소합니다.        |
    | **Cmd+Shift+Z** (macOS) / **Ctrl+Shift+Z** (Windows/Linux) | 가장 최근에 실행 취소한 Workspace 액션을 다시 실행합니다. |
    | **Cmd+S** (macOS) / **Ctrl+S** (Windows/Linux)             | 현재 Workspace를 저장합니다.                  |
    | **Cmd+J** (macOS) / **Ctrl+J** (Windows/Linux)             | Runs 사이드바를 펼치거나 접습니다.                 |
    | **Cmd+.** (macOS) / **Ctrl+.** (Windows/Linux)             | 최소화된 run 선택기를 표시하거나 숨깁니다.             |
    | **Alt+V**                                                  | 현재 패널에 시각화된 Runs만 표시합니다.              |

    ## Media browser modal

    이 단축키는 전체 화면 모달에서 미디어를 보고 있을 때 사용할 수 있습니다.

    | 단축키                                            | 설명                   |
    | ---------------------------------------------- | -------------------- |
    | **Cmd++** (macOS) / **Ctrl++** (Windows/Linux) | 확대합니다.               |
    | **Cmd+-** (macOS) / **Ctrl+-** (Windows/Linux) | 축소합니다.               |
    | **Cmd+0** (macOS) / **Ctrl+0** (Windows/Linux) | 확대/축소를 100%로 재설정합니다. |
    | **Shift+1**                                    | 뷰포트에 맞게 확대/축소합니다.    |
  </Tab>

  <Tab title="LEET">
    이 키보드 단축키는 W&amp;B LEET (Lightweight Experiment Exploration Tool) 터미널 UI에서 작동합니다. LEET를 시작하려면 터미널에서 `wandb leet`를 실행하세요.

    LEET에서 `h` 또는 `?`를 눌러 현재 뷰의 앱 내 도움말 오버레이를 여세요. 제품 개요와 실행 방법은 [LEET 터미널 UI](/ko/models/app/leet-tui)를 참조하세요.
