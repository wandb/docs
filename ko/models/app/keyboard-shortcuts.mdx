W&amp;B는 키보드 단축키를 지원하여 실험, 워크스페이스, 데이터 탐색과의 상호작용을 더 효율적으로 할 수 있도록 돕습니다. 이 참고 가이드는 W&amp;B 앱과 W&amp;B LEET(Lightweight Experiment Exploration Tool) 터미널 UI에서 사용할 수 있는 키보드 단축키를 다룹니다.

<Tabs>
  <Tab title="앱">
    다음 키보드 단축키는 W&amp;B 앱에서 동작합니다.

    ## 워크스페이스 관리

    | Shortcut                                             | Description                                 |
    | ---------------------------------------------------- | ------------------------------------------- |
    | **Cmd+Z** (macOS) / **Ctrl+Z** (Windows/Linux)       | 워크스페이스나 패널 수정과 같이 UI에서 수행한 변경 내용을 실행 취소합니다. |
    | **Cmd+Shift+Z** (macOS) / **Ctrl+Y** (Windows/Linux) | 워크스페이스에서 방금 실행 취소했던 변경 내용을 다시 수행합니다.        |

    ## 탐색

    | Shortcut                                       | Description                                                                            |
    | ---------------------------------------------- | -------------------------------------------------------------------------------------- |
    | **Tab**                                        | 상호작용 가능한 요소 사이를 이동합니다.                                                                 |
    | **Cmd+J** (macOS) / **Ctrl+J** (Windows/Linux) | 사이드바에서 Workspaces 탭과 Runs 탭을 전환합니다.                                                    |
    | **Cmd+K** (macOS) / **Ctrl+K** (Windows/Linux) | 프로젝트, 실행, 기타 리소스를 통합적으로 검색할 수 있는 빠른 검색 대화 상자를 엽니다.                                     |
    | **Esc**                                        | W&amp;B 앱 전반에서 전체 화면 패널 보기를 종료하고, 설정 서랍을 닫고, 빠른 검색 대화 상자를 닫고, 편집기를 닫거나, 기타 오버레이를 닫습니다. |

    ## 패널 탐색

    | Shortcut                                                                                                 | Description                            |
    | -------------------------------------------------------------------------------------------------------- | -------------------------------------- |
    | **Left Arrow / Right Arrow**                                                                             | 전체 화면 모드일 때 섹션 내 패널을 순서대로 이동합니다.       |
    | **Esc**                                                                                                  | 전체 화면 패널 보기를 종료하고 워크스페이스로 돌아갑니다.       |
    | **Cmd+Left Arrow / Cmd+Right Arrow** (macOS)<br />**Ctrl+Left Arrow / Ctrl+Right Arrow** (Windows/Linux) | 전체 화면 모드에서 미디어 패널을 볼 때 스텝 슬라이더를 이동합니다. |

    ## 리포트

    | Shortcut               | Description                                        |
    | ---------------------- | -------------------------------------------------- |
    | **Delete / Backspace** | 리포트에서 선택한 패널 그리드를 제거합니다.                           |
    | **Enter**              | 리포트에서 &quot;/mark&quot;를 입력한 뒤 Markdown 블록을 삽입합니다. |
    | **Esc**                | 리포트 편집기를 종료합니다.                                    |
    | **Tab**                | 리포트 내 상호작용 가능한 요소 사이를 이동합니다.                       |

    ## 참고

    * 대부분의 키보드 단축키는 macOS에서는 **Cmd**, Windows/Linux에서는 **Ctrl** 키를 사용합니다.
    * W&amp;B 앱은 일부 브라우저 기본 단축키에 대해 사용자 정의 동작을 구현합니다.
    * 일부 단축키는 컨텍스트에 따라 달라지며, 애플리케이션의 특정 영역에서만 동작합니다.
