---

W&amp;B는 키보드 단축키를 지원하여 Experiments, 워크스페이스, 데이터를 더 효율적으로 탐색하고 상호작용할 수 있도록 돕습니다. 이 레퍼런스 가이드는 W&amp;B App과 W&amp;B LEET(Lightweight Experiment Exploration Tool) 터미널 UI의 키보드 단축키를 설명합니다.

<Tabs>
  <Tab title="App">
    이 키보드 단축키는 W&amp;B App에서 사용할 수 있습니다.

    ## 워크스페이스 관리

    | Shortcut                                             | Description                            |
    | ---------------------------------------------------- | -------------------------------------- |
    | **Cmd+Z** (macOS) / **Ctrl+Z** (Windows/Linux)       | 워크스페이스나 패널 수정 등 UI에서 변경한 내용을 실행 취소합니다. |
    | **Cmd+Shift+Z** (macOS) / **Ctrl+Y** (Windows/Linux) | 워크스페이스에서 이전에 실행 취소한 변경 사항을 다시 적용합니다.   |

    ## 내비게이션

    | Shortcut                                       | Description                                                                             |
    | ---------------------------------------------- | --------------------------------------------------------------------------------------- |
    | **Tab**                                        | 대화형 요소 사이를 이동합니다.                                                                       |
    | **Cmd+J** (macOS) / **Ctrl+J** (Windows/Linux) | 프로젝트 사이드바에서 Workspaces 탭과 Runs 탭 사이를 전환합니다.                                             |
    | **Cmd+.** (macOS) / **Ctrl+.** (Windows/Linux) | 화면 공간을 확보할 수 있도록 Runs 선택기 사이드바를 최소화하거나 복원합니다.                                           |
    | **Cmd+K** (macOS) / **Ctrl+K** (Windows/Linux) | 빠른 검색 대화상자를 열어 프로젝트, Runs 및 기타 리소스를 검색합니다.                                              |
    | **Esc**                                        | W&amp;B App 전반에서 전체 화면 패널 뷰를 종료하고, 설정 드로어를 닫고, 빠른 검색 대화상자를 닫고, 편집기를 닫거나, 기타 오버레이를 닫습니다. |

    ## 패널 내비게이션

    | Shortcut                                                                                                 | Description                              |
    | -------------------------------------------------------------------------------------------------------- | ---------------------------------------- |
    | **Left Arrow / Right Arrow**                                                                             | 전체 화면 모드에서 섹션 내 패널 사이를 이동합니다.            |
    | **Esc**                                                                                                  | 전체 화면 패널 뷰를 종료하고 워크스페이스로 돌아갑니다.          |
    | **Cmd+Left Arrow / Cmd+Right Arrow** (macOS)<br />**Ctrl+Left Arrow / Ctrl+Right Arrow** (Windows/Linux) | 전체 화면 모드에서 미디어 패널을 볼 때 step 슬라이더를 이동합니다. |

    ## 미디어 패널

    | Shortcut                                                  | Description                                                                |
    | --------------------------------------------------------- | -------------------------------------------------------------------------- |
    | **Cmd + +/-** (macOS)<br />**Ctrl + +/-** (Windows/Linux) | 전체 화면에서 이미지를 볼 때 확대하거나 축소합니다.<br />확대한 상태에서는 클릭한 채로 드래그하여 이미지를 이동할 수 있습니다. |
    | **Cmd + 0** (macOS)<br />**Ctrl + 0** (Windows/Linux)     | 전체 화면에서 이미지를 볼 때 확대 수준을 100%로 재설정합니다.                                      |
    | **Shift + L**                                             | 전체 화면에서 이미지를 볼 때 화면에 맞게 표시합니다.                                             |

    ## Reports

    | Shortcut               | Description                                            |
    | ---------------------- | ------------------------------------------------------ |
    | **Delete / Backspace** | Reports에서 선택한 패널 그리드를 제거합니다.                           |
    | **Enter**              | Reports에서 &quot;/mark&quot;를 입력한 후 Markdown 블록을 삽입합니다. |
    | **Esc**                | Reports 편집기를 종료합니다.                                    |
    | **Tab**                | Reports에서 대화형 요소 사이를 이동합니다.                            |

    ## 참고

    * 대부분의 키보드 단축키는 macOS에서는 **Cmd**, Windows/Linux에서는 **Ctrl**를 사용합니다.
    * W&amp;B App은 일부 브라우저 기본 단축키를 맞춤형으로 처리합니다.
    * 일부 단축키는 컨텍스트에 따라 달라지며 애플리케이션의 특정 영역에서만 작동합니다.
