W&amp;B는 키보드 단축키를 지원하여 실험, 워크스페이스, 데이터 전반을 더 효율적으로 탐색하고 상호작용할 수 있게 해줍니다. 이 참고 가이드는 W&amp;B App과 W&amp;B LEET (Lightweight Experiment Exploration Tool) 터미널 UI에서 사용할 수 있는 키보드 단축키를 다룹니다.

<Tabs>
  <Tab title="앱">
    다음 키보드 단축키는 W&amp;B App에서 동작합니다.

    ## Workspace 관리

    | Shortcut                                             | Description                                 |
    | ---------------------------------------------------- | ------------------------------------------- |
    | **Cmd+Z** (macOS) / **Ctrl+Z** (Windows/Linux)       | 워크스페이스나 패널 수정과 같이 UI에서 수행한 변경 사항을 실행 취소합니다. |
    | **Cmd+Shift+Z** (macOS) / **Ctrl+Y** (Windows/Linux) | 워크스페이스에서 이전에 실행 취소한 변경 사항을 다시 실행합니다.        |

    ## 탐색

    | Shortcut                                       | Description                                                                                     |
    | ---------------------------------------------- | ----------------------------------------------------------------------------------------------- |
    | **Tab**                                        | 상호작용 가능한 요소 간을 이동합니다.                                                                           |
    | **Cmd+J** (macOS) / **Ctrl+J** (Windows/Linux) | 프로젝트 사이드바에서 Workspaces 탭과 Runs 탭 사이를 전환합니다.                                                     |
    | **Cmd+.** (macOS) / **Ctrl+.** (Windows/Linux) | 화면 공간을 확보할 수 있도록 Runs 선택기 사이드바를 최소화하거나 복원합니다.                                                   |
    | **Cmd+K** (macOS) / **Ctrl+K** (Windows/Linux) | 빠른 검색 대화 상자를 열어 프로젝트, Runs 및 기타 리소스를 검색합니다.                                                     |
    | **Esc**                                        | W&amp;B App 전반에서 전체 화면 패널 보기를 종료하거나, 설정 drawer를 닫거나, 빠른 검색 대화 상자를 닫거나, 편집기를 닫거나, 기타 오버레이를 닫습니다. |

    ## 패널 탐색

    | Shortcut                                                                                                 | Description                              |
    | -------------------------------------------------------------------------------------------------------- | ---------------------------------------- |
    | **Left Arrow / Right Arrow**                                                                             | 전체 화면 모드에서 section 내 패널을 순서대로 탐색합니다.     |
    | **Esc**                                                                                                  | 전체 화면 패널 보기를 종료하고 워크스페이스로 돌아갑니다.         |
    | **Cmd+Left Arrow / Cmd+Right Arrow** (macOS)<br />**Ctrl+Left Arrow / Ctrl+Right Arrow** (Windows/Linux) | 전체 화면 모드에서 미디어 패널을 볼 때 step 슬라이더를 이동합니다. |

    ## 미디어 패널

    | Shortcut                                                  | Description                                                         |
    | --------------------------------------------------------- | ------------------------------------------------------------------- |
    | **Cmd + +/-** (macOS)<br />**Ctrl + +/-** (Windows/Linux) | 전체 화면에서 이미지를 볼 때 확대하거나 축소합니다.<br />확대한 상태에서는 클릭하고 드래그하여 이미지를 이동합니다. |
    | **Cmd + 0** (macOS)<br />**Ctrl + 0** (Windows/Linux)     | 전체 화면에서 이미지를 볼 때 확대 비율을 100%로 재설정합니다.                               |
    | **Shift + L**                                             | 전체 화면에서 이미지를 볼 때 화면에 맞게 확대합니다.                                      |

    ## Reports

    | Shortcut               | Description                                        |
    | ---------------------- | -------------------------------------------------- |
    | **Delete / Backspace** | 선택한 패널 그리드를 리포트에서 제거합니다.                           |
    | **Enter**              | 리포트에서 &quot;/mark&quot;를 입력한 뒤 Markdown 블록을 삽입합니다. |
    | **Esc**                | 리포트 편집기를 종료합니다.                                    |
    | **Tab**                | 리포트 내 상호작용 가능한 요소 간을 이동합니다.                        |

    ## 참고 사항

    * 대부분의 키보드 단축키는 macOS에서는 **Cmd**, Windows/Linux에서는 **Ctrl**을 사용합니다.
    * W&amp;B App은 일부 브라우저 기본 단축키에 대해 맞춤형 처리를 구현합니다.
    * 일부 단축키는 컨텍스트에 따라 동작하며, 애플리케이션의 특정 영역에서만 사용할 수 있습니다.
