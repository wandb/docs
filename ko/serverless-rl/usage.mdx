[OpenPipe의 ART 프레임워크](https://art.openpipe.ai/getting-started/about)와 [Serverless RL API](/ko/serverless-rl/api-reference)를 통해 Serverless RL을 사용할 수 있습니다.

Serverless RL을 시작하려면 먼저 W&amp;B 도구를 사용하기 위한 [사전 요구 사항](/ko/serverless-rl/prerequisites)을 충족한 후, ART [퀵스타트](https://art.openpipe.ai/getting-started/quick-start)를 따라 하세요.

* 코드 예시와 워크플로는 [Google Colab 노트북](https://colab.research.google.com/github/openpipe/art-notebooks/blob/main/examples/2048/2048.ipynb)에서 확인하세요.
* Serverless RL의 API 엔드포인트에 대해 알아보려면 [Serverless RL API 레퍼런스](/ko/serverless-rl/api-reference)를 참조하세요.
