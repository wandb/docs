[OpenPipe의 ART 프레임워크](https://art.openpipe.ai/getting-started/about)와 [Serverless RL API](/ko/serverless-rl/api-reference)를 통해 Serverless SFT를 사용하세요.

Serverless SFT 사용을 시작하려면 먼저 W&amp;B 도구 사용을 위한 [사전 요구 사항](/ko/serverless-rl/prerequisites)을 충족한 다음, ART [Serverless SFT 문서](https://art.openpipe.ai/fundamentals/sft-training)를 확인하세요.

* Serverless SFT의 API endpoint에 대해 알아보려면 [Serverless RL API 레퍼런스](/ko/serverless-rl/api-reference)를 참조하세요.
