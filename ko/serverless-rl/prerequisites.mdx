OpenPipe ART 프레임워크 또는 API를 통해 Serverless RL 특성을 사용하기 전에 다음 단계를 완료하세요.

<Tip>
  시작하기 전에 비용과 제한 사항을 이해할 수 있도록 [사용 정보 및 제한 사항](/ko/serverless-rl/usage-limits)을 검토하세요.
</Tip>

<div id="sign-up-and-create-an-api-key">
  ## 회원가입하고 API 키 생성하기
</div>

W&amp;B에서 머신을 인증하려면 먼저 API 키를 생성해야 합니다.

<ApiKeyCreate />

<div id="create-a-project-in-wb">
  ## W&amp;B에서 프로젝트 만들기
</div>

사용량을 추적하고 트레이닝 메트릭을 기록하며 학습된 모델을 저장하려면 W&amp;B 계정에 프로젝트를 만드세요. 자세한 내용은 [Projects 가이드](https://docs.wandb.ai/models/track/project-page)를 참조하세요.

<div id="next-steps">
  ## 다음 단계
</div>

사전 요구 사항을 완료한 후에는:

* 사용 가능한 엔드포인트를 알아보려면 [API 레퍼런스](/ko/serverless-rl/api-reference)를 확인하세요
* [ART 퀵스타트](https://art.openpipe.ai/getting-started/quick-start)를 사용해 보세요
