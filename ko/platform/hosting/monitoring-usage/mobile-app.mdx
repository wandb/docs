iOS용 W&amp;B 모바일 앱을 사용하면 어디서든 W&amp;B Models 프로젝트에 계속 연결된 상태를 유지할 수 있습니다. iPhone 또는 iPad에서 훈련 run을 실시간으로 추적하고, 선형 플롯을 보고, 프로젝트의 이력을 탐색하고, 팀의 진행 상황을 확인해 보세요.

<Info>
  The mobile app is only available for [Multi-tenant Cloud](/ko/platform/hosting/hosting-options/multi_tenant_cloud) accounts. It is not available for [Dedicated Cloud](/ko/platform/hosting/hosting-options/dedicated-cloud) or [Self-Managed](/ko/platform/hosting/hosting-options/self-managed) deployments.
</Info>

<div id="features">
  ## 기능
</div>

* **트레이닝 run 추적**: run 상태, 메트릭, 선형 플롯을 실시간으로 확인합니다.
* **프로젝트 이력 탐색**: W&amp;B 프로젝트 전반을 탐색하고 검색합니다. **프로젝트 패널**의 메트릭 차트에는 최신 run뿐 아니라 프로젝트의 **모든 run** 데이터가 포함됩니다.
* **팀 전환**: **팀 선택기**를 사용해 팀을 전환합니다. 선택기에는 더 빠르게 탐색할 수 있도록 업데이트된 레이아웃이 적용되었습니다.
* **최신 정보 유지**: 알림을 사용하면 노트북을 열지 않고도 실험 업데이트를 받을 수 있습니다. 더 이상 필요 없는 **알림 제거**는 **Notifications** 탭에서 해당 알림을 스와이프해 처리합니다.

<div id="set-up-notifications">
  ## 알림 설정
</div>

다음 두 가지 방법으로 알림을 설정할 수 있습니다.

**메트릭 임계값 알림**

1. run으로 이동합니다.
2. 그래프를 탭해 전체 화면 뷰로 전환합니다.
3. 오른쪽 상단의 종 아이콘을 탭합니다.
4. 이후 run이 지정한 메트릭 임계값을 넘을 때 트리거되는 알림을 설정합니다.

**run 실패 알림**

1. 프로젝트로 이동합니다.
2. 오른쪽 상단의 점 3개 메뉴를 탭합니다.
3. 해당 프로젝트에서 이후 run 실패에 대한 알림을 받으려면 **Run failed alert**를 선택합니다.

**Manage notifications** 화면에는 두 종류의 설정이 한곳에 나열되며, 각 섹션에서 제어하는 내용을 설명하는 짧은 설명도 함께 표시됩니다.
