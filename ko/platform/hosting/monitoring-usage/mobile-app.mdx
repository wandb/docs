iOS용 W&amp;B 모바일 앱을 사용하면 어디서든 W&amp;B Models 프로젝트에 계속 연결된 상태를 유지할 수 있습니다. iPhone 또는 iPad에서 트레이닝 run을 실시간으로 추적하고, 선 그래프를 확인하고, 프로젝트 기록을 살펴보고, 팀의 진행 상황을 확인하세요.

<Info>
  모바일 앱은 [Multi-tenant Cloud](/ko/platform/hosting/hosting-options/multi_tenant_cloud) 계정에서만 사용할 수 있습니다. [Dedicated Cloud](/ko/platform/hosting/hosting-options/dedicated-cloud) 또는 [Self-Managed](/ko/platform/hosting/hosting-options/self-managed) deployment에서는 사용할 수 없습니다.
</Info>

<div id="features">
  ## 기능
</div>

* **트레이닝 run 추적**: run 상태, 메트릭, 라인 플롯을 실시간으로 뷰합니다
* **프로젝트 이력 탐색**: W&amp;B 프로젝트 전체에서 둘러보고 검색합니다.
* **최신 정보 유지**: 알림을 사용하면 노트북을 열지 않고도 Experiments 업데이트를 받아볼 수 있습니다.

<div id="set-up-notifications">
  ## 알림 설정
</div>

다음 두 가지 방법으로 알림을 설정할 수 있습니다.

**메트릭 임계값 알림**

1. run으로 이동합니다.
2. 그래프를 탭해 전체 화면 뷰로 전환합니다.
3. 오른쪽 상단의 종 아이콘을 탭합니다.
4. 이후 run이 지정한 메트릭 임계값을 넘을 때 트리거되는 알림을 설정합니다.

**run 실패 알림**

1. 프로젝트로 이동합니다.
2. 오른쪽 상단의 점 3개 메뉴를 탭합니다.
3. 해당 프로젝트에서 이후 run 실패에 대한 알림을 받으려면 **Run failed alert**를 선택합니다.
