iOS용 W&amp;B Mobile App을 사용하면 어디서든 W&amp;B Models 프로젝트와 계속 연결된 상태를 유지할 수 있습니다. iPhone 또는 iPad에서 바로 트레이닝 run을 실시간으로 추적하고, 선형 플롯을 확인하고, 프로젝트의 이력을 탐색하고, 팀의 진행 상황을 살펴보세요.

<Info>
  모바일 앱은 [Multi-tenant Cloud](/ko/platform/hosting/hosting-options/multi_tenant_cloud) 계정에서만 사용할 수 있습니다. [Dedicated Cloud](/ko/platform/hosting/hosting-options/dedicated-cloud) 또는 [Self-Managed](/ko/platform/hosting/hosting-options/self-managed) 배포에서는 사용할 수 없습니다.
</Info>

<div id="features">
  ## 기능
</div>

* **Track training runs**: 실시간으로 run 상태, 메트릭, 선형 플롯을 확인합니다.
* **프로젝트 이력 탐색**: W&amp;B 프로젝트 전반을 둘러보고 검색합니다. **프로젝트 패널**의 메트릭 차트에는 최신 run뿐만 아니라 프로젝트의 **모든 run** 데이터가 포함됩니다.
* **팀 전환**: **팀 선택기**를 사용해 팀을 변경합니다. 이 선택기는 더 빠르게 탐색할 수 있도록 업데이트된 레이아웃을 사용합니다.
* **계속 확인하기**: 알림을 사용하면 노트북을 열지 않고도 실험 업데이트를 받아볼 수 있습니다. 더 이상 필요하지 않은 **알림**은 **Notifications** 탭에서 스와이프해 제거하세요.

<div id="set-up-notifications">
  ## 알림 설정
</div>

알림은 두 가지 방법으로 설정할 수 있습니다.

**메트릭 임계값 알림**

1. run으로 이동합니다.
2. 그래프를 탭해 전체 화면으로 전환합니다.
3. 오른쪽 상단의 종 아이콘을 탭합니다.
4. 이후 run이 지정한 메트릭 임계값을 넘을 때 알림이 트리거되도록 설정합니다.

**run 실패 알림**

1. 프로젝트로 이동합니다.
2. 오른쪽 상단의 점 3개 메뉴를 탭합니다.
3. 해당 프로젝트에서 이후 run 실패에 대한 알림을 받으려면 **Run failed alert**를 선택합니다.

**Manage notifications** 화면에서는 두 종류의 설정을 한곳에서 확인할 수 있으며, 각 섹션에서 제어하는 항목을 설명하는 짧은 설명도 함께 제공됩니다.
