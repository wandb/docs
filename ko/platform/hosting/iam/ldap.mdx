---
title: LDAP로 SSO 구성
description: "연결 매개변수와 환경 변수를 포함해 W&B Server용 LDAP 기반 SSO 인증을 설정합니다."
---

W&amp;B Server LDAP 서버를 사용해 자격 증명을 인증합니다. 다음 가이드에서는 W&amp;B Server의 LDAP 설정 방법을 설명합니다. 필수 및 선택 설정은 물론, 시스템 Settings UI에서 LDAP 연결을 설정하는 방법도 다룹니다. 또한 주소, 기본 DN, 속성 등 LDAP 설정의 여러 입력 항목에 대한 정보도 제공합니다. 이러한 속성은 W&amp;B App UI에서 지정하거나 환경 변수를 사용해 지정할 수 있습니다. 익명 바인드를 설정하거나 관리자 DN 및 비밀번호로 바인드할 수 있습니다.

{/* <Note>
  W&B Team Admin이라면 익명 바인드를 설정하거나 관리자 DN 및 비밀번호를 사용한 바인드를 설정할 수 있습니다.
  </Note> */}

<Note>
  LDAP 인증을 활성화하고 설정할 수 있는 것은 W&amp;B Admin 역할뿐입니다.
</Note>


<div id="configure-ldap-connection">
  ## LDAP 연결 구성
</div>

<Tabs>
  <Tab title="W&B App">
    1. W&amp;B App으로 이동합니다.
    2. 오른쪽 상단의 프로필 아이콘을 선택합니다. 드롭다운에서 **System Settings**를 선택합니다.
    3. **Configure LDAP Client**를 켭니다.
    4. 양식에 세부 정보를 입력합니다. 각 입력 항목에 대한 자세한 내용은 **Configuring Parameters** 섹션을 참고하세요.
    5. 설정을 테스트하려면 **Update Settings**를 클릭합니다. 그러면 W&amp;B 서버에 대한 테스트 클라이언트/연결이 설정됩니다.
    6. 연결이 확인되면 **Enable LDAP Authentication**를 켜고 **Update Settings** 버튼을 선택합니다.
  </Tab>

  <Tab title="환경 변수">
    다음 환경 변수로 LDAP 연결을 설정합니다:

    | 환경 변수                         | 필수  | 예시                              |
    | ----------------------------- | --- | ------------------------------- |
    | `LOCAL_LDAP_ADDRESS`          | 예   | `ldaps://ldap.example.com:636`  |
    | `LOCAL_LDAP_BASE_DN`          | 예   | `email=mail,group=gidNumber`    |
    | `LOCAL_LDAP_BIND_DN`          | 아니요 | `cn=admin`, `dc=example,dc=org` |
    | `LOCAL_LDAP_BIND_PW`          | 아니요 |                                 |
    | `LOCAL_LDAP_ATTRIBUTES`       | 예   | `email=mail`, `group=gidNumber` |
    | `LOCAL_LDAP_TLS_ENABLE`       | 아니요 |                                 |
    | `LOCAL_LDAP_GROUP_ALLOW_LIST` | 아니요 |                                 |
    | `LOCAL_LDAP_LOGIN`            | 아니요 |                                 |

    각 환경 변수의 정의는 [설정 매개변수](#configuration-parameters) 섹션을 참조하세요. 명확성을 위해 정의 이름에서는 환경 변수 접두사 `LOCAL_LDAP`를 생략했습니다.
  </Tab>
</Tabs>

<div id="configuration-parameters">
  ## 설정 매개변수
</div>

다음 표에는 필수 및 선택 LDAP 설정 항목과 각 항목에 대한 설명이 나와 있습니다.

| 환경 변수              | 설명                                                                                                                                                       | 필수 여부 |
| ------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------- | ----- |
| `ADDRESS`          | W&amp;B Server를 호스팅하는 VPC 내 LDAP 서버의 주소입니다.                                                                                                              | 예     |
| `BASE_DN`          | 검색이 시작되는 루트 경로이며, 이 디렉터리에 쿼리하려면 필요합니다.                                                                                                                   | 예     |
| `BIND_DN`          | LDAP 서버에 등록된 관리자 사용자의 경로입니다. LDAP 서버가 인증되지 않은 바인딩을 지원하지 않는 경우 필요합니다. 지정하면 W&amp;B Server는 이 사용자로 LDAP 서버에 연결합니다. 그렇지 않으면 W&amp;B Server는 익명 바인딩으로 연결합니다. | 아니요   |
| `BIND_PW`          | 관리자 사용자의 비밀번호이며 바인딩 인증에 사용됩니다. 비워 두면 W&amp;B Server는 익명 바인딩으로 연결합니다.                                                                                     | 아니요   |
| `ATTRIBUTES`       | 이메일 및 그룹 ID 속성 이름을 쉼표로 구분된 문자열 값으로 지정합니다.                                                                                                                | 예     |
| `TLS_ENABLE`       | TLS를 활성화합니다.                                                                                                                                             | 아니요   |
| `GROUP_ALLOW_LIST` | 그룹 허용 목록입니다.                                                                                                                                             | 아니요   |
| `LOGIN`            | W&amp;B Server가 LDAP를 사용해 인증하도록 지정합니다. `True` 또는 `False`로 설정합니다. 필요하면 LDAP 설정을 테스트하기 위해 `false`로 설정할 수 있습니다. LDAP 인증을 시작하려면 `true`로 설정합니다.               | 아니요   |
