---

조직 관리자와 팀 관리자는 각각 조직 수준과 팀 수준에서 개인정보 보호 설정을 구성할 수 있습니다. 조직 수준에서 설정하면 조직 관리자가 해당 조직의 모든 팀에 이 설정을 강제 적용합니다.

<Note>
  팀 워크플로의 예기치 않은 변경을 방지하기 위해, W&amp;B는 조직 관리자가 개인정보 보호 설정을 강제 적용하기 전에 조직의 모든 팀 관리자와 사용자에게 이 사실을 미리 안내할 것을 권장합니다.
</Note>

<div id="enforce-privacy-settings-for-all-teams">
  ## 모든 팀에 개인정보 보호 설정 강제 적용
</div>

조직 관리자는 계정 또는 조직 대시보드의 **Settings** 탭에 있는 **Privacy** 섹션에서 조직 내 모든 팀의 개인정보 보호 설정을 강제 적용할 수 있습니다. 조직 관리자가 설정을 강제 적용하면 각 팀의 관리자는 해당 설정을 팀 내에서 구성할 수 없습니다.

* **팀 가시성 제한 강제 적용** - 조직 전체에서 멤버가 아닌 사용자에게 모든 팀를 숨깁니다
* **향후 프로젝트에 대한 개인정보 보호 강제 적용** - 모든 팀의 모든 새 프로젝트를 비공개 또는 [제한됨](./iam/access-management/restricted-projects)으로 설정하도록 요구합니다
* **초대 제어 강제 적용** - 관리자가 아닌 사용자가 어떤 팀에도 멤버를 초대하지 못하도록 합니다
* **Reports 공유 제어 강제 적용** - 비공개 프로젝트에서 Reports의 공개 공유를 비활성화하고 기존 매직 링크도 비활성화합니다
* **팀 자동 참여 제한 강제 적용** - 조직 이메일 도메인이 일치하는 Users가 어떤 팀에도 자동으로 참여하지 못하도록 제한합니다
  <Note>이 설정은 [Multi-tenant Cloud](./hosting-options/multi_tenant_cloud) 배포에서만 사용할 수 있습니다.</Note>
* **기본 코드 저장 제한 강제 적용** - 모든 팀에서 기본적으로 코드 저장을 비활성화합니다

조직 개인정보 보호 설정을 열려면:

1. 조직 관리자로 로그인하세요.
2. Multi-tenant Cloud에서는 `https://wandb.ai/account-settings/<organization>/settings`로 이동하세요. `<organization>`을 조직 이름으로 바꾸세요. Dedicated Cloud 또는 Self-Managed 배포에서는 인스턴스 조직 대시보드를 여세요. 환경별 URL은 [Add and manage users](/ko/platform/hosting/iam/access-management/manage-organization#add-and-manage-users)를 참조하세요.
3. **Privacy** 섹션을 열고 **기본 코드 저장 제한 강제 적용**과 필요한 기타 조직 전체 제어를 구성하세요.

<div id="configure-privacy-settings-for-a-team">
  ## 팀의 개인정보 보호 설정 구성하기
</div>

팀 관리자는 팀 **Settings** 탭의 **Privacy** 섹션에서 각 팀의 개인정보 보호 설정을 구성할 수 있습니다. 각 설정은 조직 범위에서 강제 적용되지 않은 경우에만 구성할 수 있습니다.

* **이 팀을 모든 비구성원에게 숨기기** - 구성원이 아닌 사용자에게 팀이 보이지 않도록 합니다
* **향후 생성되는 모든 팀 프로젝트를 비공개로 만들기** - 팀에서 새로 생성되는 모든 프로젝트가 비공개가 되도록 합니다(공개 공유는 허용되지 않음)
* **모든 팀 구성원이 다른 구성원을 초대할 수 있도록 허용하기** - 관리자뿐 아니라 모든 팀 구성원이 새 구성원을 초대할 수 있도록 합니다
* **비공개 프로젝트의 Reports에 대한 공개 공유 끄기** - Reports의 공개 공유를 비활성화하고 기존 매직 링크도 비활성화합니다
* **조직 이메일 도메인이 일치하는 사용자가 이 팀에 참여하도록 허용하기** - 동일한 이메일 도메인을 사용하는 사용자가 자동으로 팀에 참여할 수 있도록 합니다
  <Note>이 설정은 [Multi-tenant Cloud](./hosting-options/multi_tenant_cloud) 배포 환경에서만 사용할 수 있습니다.</Note>
* **기본적으로 코드 저장 활성화하기** - 팀의 모든 Runs에 대해 코드를 자동으로 저장합니다

팀 개인정보 보호 설정을 열려면 다음 단계를 따르세요.

1. `https://wandb.ai/<team>`로 이동하세요. `<team>`을 팀 이름으로 바꾸세요.
2. 왼쪽 내비게이션에서 **Team settings**를 선택하세요.
3. **Privacy** 섹션을 열고 **Enable code saving by default**를 구성하세요.
