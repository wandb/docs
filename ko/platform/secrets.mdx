title: 시크릿 관리
---

W&amp;B Secret Manager를 사용하면 액세스 토큰, 베어러 토큰, API 키, 비밀번호와 같은 민감한 문자열인 &#95;시크릿&#95;을 안전하게 중앙에서 저장, 관리, 주입할 수 있습니다. W&amp;B Secret Manager를 사용하면 민감한 문자열을 코드에 직접 추가하거나 웹훅 헤더 또는 [payload](/ko/models/automations/)를 설정할 때 직접 입력할 필요가 없습니다.

시크릿은 각 팀의 Secret Manager에 저장 및 관리되며, [team settings](/ko/platform/app/settings-page/teams/)의 **Team secrets** 섹션에서 확인할 수 있습니다.

<Note>
  * 시크릿은 W&amp;B Admin만 생성, 편집, 삭제할 수 있습니다.
  * 시크릿은 W&amp;B의 핵심 기능에 포함되며, Azure, Google Cloud 또는 AWS에서 호스팅하는 [W&amp;B Server deployments](/ko/platform/hosting/)에서도 사용할 수 있습니다. 다른 deployment 유형을 사용하는 경우, W&amp;B에서 시크릿을 사용하는 방법은 W&amp;B account team에 문의하세요.
  * W&amp;B Server에서는 보안 요구 사항을 충족하는 보안 조치를 직접 구성할 책임이 있습니다.

    * W&amp;B는 AWS, Google Cloud 또는 Azure에서 제공하는 고급 보안 기능이 구성된 클라우드 공급자의 시크릿 관리자용 W&amp;B 인스턴스에 시크릿을 저장할 것을 강력히 권장합니다.

    * W&amp;B는 클라우드 시크릿 관리자(AWS, Google Cloud 또는 Azure)용 W&amp;B 인스턴스를 사용할 수 없고, cluster 사용 시 발생할 수 있는 보안 취약점을 방지하는 방법을 이해하고 있는 경우가 아니라면, Kubernetes cluster를 시크릿 저장소의 백엔드로 사용하는 것을 권장하지 않습니다.
</Note>

<div id="add-a-secret">
  ## 시크릿 추가
</div>

시크릿을 추가하려면 다음 단계를 따르세요.

1. 수신 서비스가 들어오는 웹훅을 인증하는 데 필요하다면, 필요한 token 또는 API 키를 생성하세요. 필요한 경우 비밀번호 관리자와 같은 안전한 곳에 이 민감한 문자열을 저장하세요.
2. W&amp;B에 로그인한 다음 팀의 **Settings** 페이지로 이동하세요.
3. **Team Secrets** 섹션에서 **New secret**을 클릭하세요.
4. 문자, 숫자, 밑줄(`_`)을 사용해 시크릿 이름을 입력하세요.
5. 민감한 문자열을 **시크릿** 필드에 붙여넣으세요.
6. **시크릿 추가**를 클릭하세요.

웹훅을 구성할 때 webhook automation에 사용할 시크릿을 지정하세요. 자세한 내용은 [웹훅 구성](#configure-a-webhook) 섹션을 참조하세요.

<Note>
  시크릿을 만든 후에는 [webhook automation의 payload](/ko/models/automations/create-automations/webhook/)에서 `${SECRET_NAME}` 형식을 사용해 해당 시크릿에 액세스할 수 있습니다.
</Note>

<div id="rotate-a-secret">
  ## 시크릿 교체
</div>

시크릿을 교체하고 값을 업데이트하려면 다음과 같이 하세요.

1. 시크릿의 세부 정보를 열려면 해당 시크릿 행의 연필 아이콘을 클릭합니다.
2. **시크릿**을 새 값으로 설정합니다. 필요하면 **시크릿 표시**를 클릭해 새 값을 확인합니다.
3. **시크릿 추가**를 클릭합니다. 시크릿 값이 업데이트되며 더 이상 이전 값으로 사용되지 않습니다.

<Note>
  시크릿을 생성하거나 업데이트한 후에는 현재 값을 더 이상 표시할 수 없습니다. 대신 시크릿을 새 값으로 교체하세요.
</Note>

<div id="delete-a-secret">
  ## 시크릿 삭제하기
</div>

시크릿을 삭제하려면 다음 단계를 따르세요.

1. 시크릿 행에서 휴지통 아이콘을 클릭합니다.
2. 확인 대화 상자를 확인한 다음 **Delete**를 클릭합니다. 시크릿은 즉시 영구적으로 삭제됩니다.

<div id="manage-access-to-secrets">
  ## 시크릿 접근 권한 관리
</div>

팀의 자동화는 팀 시크릿을 사용할 수 있습니다. 시크릿을 제거하기 전에 해당 시크릿을 사용하는 자동화를 업데이트하거나 제거해 작동이 중단되지 않도록 하세요.
