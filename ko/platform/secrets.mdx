description: W&B 시크릿의 개요, 작동 방식, 그리고 사용 방법을 설명합니다.
title: 시크릿 관리
---

W&amp;B Secret Manager를 사용하면 액세스 토큰, 베어러 토큰, API 키, 비밀번호와 같은 민감한 문자열인 &#95;시크릿&#95;을 안전하게 중앙에서 저장·관리하고 애플리케이션에 주입할 수 있습니다. W&amp;B Secret Manager를 사용하면 코드에 직접 민감한 문자열을 추가하거나 웹훅의 헤더나 [payload](/ko/models/automations/)를 구성할 때 직접 입력할 필요가 없습니다.

시크릿은 각 팀의 Secret Manager에서, [team settings](/ko/platform/app/settings-page/teams/)의 **Team secrets** 섹션을 통해 저장 및 관리됩니다.

<Note>
  * W&amp;B Admin만 시크릿을 생성, 편집 또는 삭제할 수 있습니다.
  * 시크릿은 Azure, Google Cloud, AWS에 호스팅하는 [W&amp;B Server 배포](/ko/platform/hosting/)를 포함하여 W&amp;B의 핵심 기능에 포함됩니다. 다른 배포 유형을 사용하는 경우, W&amp;B에서 시크릿을 어떻게 사용할 수 있는지 논의하려면 W&amp;B 계정 팀에 문의하세요.
  * W&amp;B Server에서는 보안 요구 사항을 충족하도록 보안 조치를 구성하는 책임이 사용자에게 있습니다.

    * W&amp;B는 AWS, Google Cloud, Azure 등 클라우드 제공자가 제공하는 시크릿 매니저와 연동되고 고급 보안 기능으로 구성된 W&amp;B 인스턴스에 시크릿을 저장할 것을 강력히 권장합니다.

    * W&amp;B는 AWS, Google Cloud, Azure와 같은 클라우드 시크릿 매니저와 연동된 W&amp;B 인스턴스를 사용할 수 없고, 클러스터를 사용할 때 발생할 수 있는 보안 취약점을 방지하는 방법을 충분히 이해한 경우가 아니라면 시크릿 저장소의 백엔드로 Kubernetes 클러스터를 사용하는 것을 권장하지 않습니다.
</Note>

<div id="add-a-secret">
  ## 시크릿 추가
</div>

시크릿을 추가하려면:

1. 수신 서비스에서 들어오는 웹훅을 인증하기 위해 토큰이나 API 키가 필요한 경우, 해당 토큰이나 API 키를 생성합니다. 필요하다면 비밀번호 관리자 같은 안전한 곳에 이 민감한 문자열을 저장해 두세요.
2. W&amp;B에 로그인한 다음 팀의 **Settings** 페이지로 이동합니다.
3. **Team Secrets** 섹션에서 **New secret**을 클릭합니다.
4. 문자, 숫자, 밑줄(`_`)을 사용해 시크릿 이름을 지정합니다.
5. 민감한 문자열을 **Secret** 필드에 붙여넣습니다.
6. **Add secret**을 클릭합니다.

웹훅을 구성할 때, 웹훅 자동화에 사용할 시크릿을 지정합니다. 자세한 내용은 [Configure a webhook](#configure-a-webhook) 섹션을 참조하세요.

<Note>
  시크릿을 생성하면 `${SECRET_NAME}` 형식을 사용해 [webhook automation의 payload](/ko/models/automations/create-automations/webhook/)에서 해당 시크릿에 접근할 수 있습니다.
</Note>

<div id="rotate-a-secret">
  ## 시크릿 교체하기
</div>

시크릿을 교체하고 값을 업데이트하려면:

1. 시크릿의 행에서 연필 아이콘을 클릭하여 시크릿 세부 정보를 엽니다.
2. **Secret**에 새 값을 설정합니다. 필요하다면 **Reveal secret**을 클릭하여 새 값을 확인합니다.
3. **Add secret**을 클릭합니다. 시크릿 값이 업데이트되며 더 이상 이전 값으로 참조되지 않습니다.

<Note>
  시크릿이 생성되거나 업데이트된 후에는 현재 값을 다시 확인할 수 없습니다. 대신 시크릿을 새 값으로 교체하세요.
</Note>

<div id="delete-a-secret">
  ## 시크릿 삭제
</div>

시크릿을 삭제하려면:

1. 시크릿 행에서 휴지통 아이콘을 클릭합니다.
2. 확인 대화 상자의 내용을 확인한 다음 **Delete**를 클릭합니다. 시크릿은 즉시 영구적으로 삭제됩니다.

<div id="manage-access-to-secrets">
  ## 시크릿 액세스 관리
</div>

팀의 자동화는 팀의 시크릿을 사용할 수 있습니다. 시크릿을 제거하기 전에 해당 시크릿을 사용하는 자동화를 미리 수정하거나 삭제하여 자동화가 중단되지 않도록 하세요.
