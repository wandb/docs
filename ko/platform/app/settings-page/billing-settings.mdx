사용자 프로필 페이지로 이동한 다음 오른쪽 상단의 사용자 아이콘을 선택합니다. 드롭다운에서 **Billing**을 선택하거나 **Settings**를 선택한 다음 **Billing** 탭을 선택합니다.

<div id="plan-details">
  ## 플랜 세부 정보
</div>

**플랜 세부 정보** 섹션에서는 조직의 현재 플랜, 요금, 한도, 사용 현황을 요약해서 보여줍니다.

* 세부 정보와 사용자 목록을 보려면 **Manage users**를 클릭합니다.
* 사용 현황의 세부 정보를 보려면 **View usage**를 클릭합니다.
* 조직에서 사용 중인 저장소 용량(무료 및 유료 포함)입니다. 여기에서 추가 저장소를 구매하고 현재 사용 중인 저장소를 관리할 수 있습니다. 자세한 내용은 [저장소 설정](/ko/platform/app/settings-page/storage/)을 참조하세요.

여기에서 플랜을 비교하거나 Sales에 문의할 수 있습니다.

<div id="plan-usage">
  ## 플랜 사용
</div>

이 섹션에서는 현재 사용량을 시각적으로 요약하고, 예정된 사용 요금을 표시합니다. 월별 사용량에 대한 자세한 정보를 보려면 각 타일에서 **View usage**를 클릭하세요. 캘린더 월별, 팀별 또는 프로젝트별 사용량을 내보내려면 **Export CSV**를 클릭하세요.

사용량 또는 청구 관련 문의는 AISE 또는 [지원팀](mailto:support@wandb.com)에 연락하세요.

<div id="usage-alerts">
  ### 사용량 알림
</div>

사용량 알림은 현재 청구 기간 동안 특정 카테고리의 사용량이 조직의 사용 한도의 85% 및 100%에 도달하면 조직 관리자와 청구 관리자에게 이메일로 알립니다.

| Category  | Description                                                                                                                                                                   |
| --------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 저장소       | 조직에 저장된 artifact 및 run 데이터의 총 누적 용량이며, GB 단위로 측정됩니다.                                                                                                                          |
| 추적 시간     | 청구 기간 동안의 실제 경과 트레이닝 시간이며, 시간 단위로 측정됩니다.                                                                                                                                      |
| Weave     | 청구 기간 동안 수집된 데이터의 총량이며, MB 단위로 측정됩니다. 배포에서 Weave가 활성화된 경우에만 표시됩니다.                                                                                                            |
| Inference | Inference에서 사용한 총 입력 및 출력 token 수입니다. [Inference Pricing](https://wandb.ai/site/pricing/inference/)을 참조하세요. 배포에서 Inference가 활성화된 경우에만 표시됩니다.                                  |
| 트레이닝      | 토큰 기반 Inference 및 컴퓨팅 사용량이며, GPU/시간 단위로 측정됩니다. [Serverless RL Pricing](https://wandb.ai/site/pricing/reinforcement-learning/)을 참조하세요. 배포에서 W&amp;B Training이 활성화된 경우에만 표시됩니다. |

**Enterprise** 플랜의 조직에서는 모든 **조직 관리자**와 **청구 관리자**가 사용량 알림을 받습니다. [Pro plan](https://wandb.ai/site/pricing/)에서는 **청구 관리자**만 사용량 알림을 받습니다.

메시지는 `support@wandb.com`에서 발송됩니다. 임계값은 설정할 수 없지만, 카테고리별로 두 알림 중 하나 또는 둘 다 선택적으로 끌 수 있습니다.

**조직 관리자** 또는 **청구 관리자**가 조직의 사용량 알림을 설정할 수 있습니다.

1. 오른쪽 상단의 사용자 프로필 아이콘을 클릭합니다.
2. **Account** 섹션에서 **Usage &amp; Alerts**를 클릭합니다. 또는 **Settings**를 클릭한 다음 **Usage &amp; Alerts** 탭을 클릭합니다.
3. 사이드바에서 카테고리를 선택합니다.
4. **Usage** 탭을 클릭해 해당 카테고리의 조직 사용량과 알림 설정을 확인합니다.
5. 필요에 따라 해당 카테고리의 **85% usage alert** 또는 **100% usage alert**를 켜거나 끕니다. <Warning>두 사용량 알림을 모두 끄면 해당 카테고리의 사용량에 대해서는 알림이 전송되지 않습니다.</Warning>

<div id="spending-alerts">
  ### 지출 알림
</div>

특정 청구 기간 동안 지출 알림은 조직이 USD 기준의 사용자 지정 가능한 지출 임계값에 하나 이상 도달하면 조직 관리자와 청구 관리자에게 이메일로 알립니다. 카테고리별로 서로 다른 지출 알림을 최대 5개까지 설정할 수 있습니다:

| Category  | Description                                                                                                                                                                              |
| --------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 저장소       | 청구 기간 동안 조직의 저장소 [사용](#usage-alerts)에 대한 요금입니다. [가격](https://wandb.ai/site/pricing/)을 참조하세요.                                                                                             |
| 추적 시간     | 청구 기간 동안 조직의 트레이닝 [사용](#usage-alerts)에 대한 요금입니다. [가격](https://wandb.ai/site/pricing/)을 참조하세요.                                                                                            |
| Weave     | 청구 기간 동안 조직의 Weave [사용](#usage-alerts)에 대한 요금입니다. [가격](https://wandb.ai/site/pricing/)을 참조하세요. 배포에서 Weave가 활성화된 경우에만 표시됩니다.                                                              |
| Inference | 청구 기간 동안 조직의 Inference [사용](#usage-alerts)에 대한 요금입니다. [Inference Pricing](https://wandb.ai/site/pricing/inference/)을 참조하세요. 배포에서 Inference가 활성화된 경우에만 표시됩니다.                             |
| 트레이닝      | 청구 기간 동안 조직의 Serverless RL [사용](#usage-alerts)에 대한 요금입니다. [Serverless RL Pricing](https://wandb.ai/site/pricing/reinforcement-learning/)을 참조하세요. 배포에서 W&amp;B Training이 활성화된 경우에만 표시됩니다. |

메시지는 `support@wandb.com`에서 발송됩니다.

**Enterprise** 플랜을 사용하는 조직의 경우:

* 모든 **조직 관리자**와 **청구 관리자**가 지출 알림을 받습니다.
* 기본적으로 설정된 지출 알림은 없습니다.

[Pro 플랜](https://wandb.ai/site/pricing/)의 경우:

* **청구 관리자**만 지출 알림을 받습니다.
* 기본적으로 지출 알림은 $200, $450, $700, $1000으로 설정됩니다.

<Note>
  추적 시간에는 요금이 청구되지 않으며, 모든 플랜에서 무제한으로 제공됩니다. 추적 시간은 모니터링 목적으로 **Usage** 페이지에 표시됩니다.
</Note>

<div id="view-spending-alerts">
  #### 지출 알림 보기
</div>

조직의 지출 알림을 보려면 다음 단계를 따르세요.

1. 오른쪽 상단의 사용자 프로필 아이콘을 클릭합니다.
2. **Account** 섹션에서 **Usage &amp; Alerts**를 클릭합니다. 또는 **Settings**를 클릭한 다음 **Usage &amp; Alerts** 탭을 클릭합니다.
3. 사이드바에서 카테고리를 선택합니다.
4. 처음에는 **Usage** 탭이 표시됩니다. 지출 알림을 보려면 **Spend alerts**를 클릭합니다. 선택한 카테고리에 대해 구성된 모든 지출 알림이 표시됩니다.

<div id="add-or-delete-a-spending-alert">
  #### 지출 알림 추가 또는 삭제
</div>

지출 알림을 관리하려면 다음 단계를 따르세요.

1. 오른쪽 상단의 사용자 프로필 아이콘을 클릭합니다.
2. **Account** 섹션에서 **Usage &amp; Alerts**을 클릭합니다. 또는 **설정**을 클릭한 다음 **Usage &amp; Alerts** 탭을 클릭합니다.
3. 사이드바에서 **저장소**와 같은 카테고리를 선택합니다.
4. 알림을 추가하려면 **알림 추가**를 클릭합니다.
   1. USD 기준 임계값 금액을 입력합니다.
   2. **알림 생성**을 클릭합니다.
5. 지출 알림을 삭제하려면 해당 알림 옆의 휴지통 아이콘을 클릭합니다.

<div id="payment-methods">
  ## 결제 방법
</div>

이 섹션에서는 조직에 등록된 결제 방법을 확인할 수 있습니다. 결제 방법을 아직 추가하지 않았다면, 플랜을 업그레이드하거나 유료 저장소를 추가할 때 결제 방법을 추가하라는 안내가 표시됩니다.

<div id="billing-admin">
  ## 청구 관리자
</div>

이 섹션에는 현재 청구 관리자가 표시됩니다. 청구 관리자는 조직 관리자이며, 청구 관련 이메일을 모두 수신하고 결제 방법을 확인하고 관리할 수 있습니다.

<Note>
  W&amp;B Dedicated Cloud에서는 여러 사용자가 청구 관리자일 수 있습니다. W&amp;B Multi-tenant Cloud에서는 한 번에 한 사용자만 청구 관리자일 수 있습니다.
</Note>

청구 관리자를 변경하거나 추가 사용자에게 이 역할을 할당하려면 다음 단계를 따르세요.

1. **Manage roles**를 클릭합니다.
2. 사용자를 검색합니다.
3. 해당 사용자의 행에서 **청구 관리자** 필드를 클릭합니다.
4. 요약을 조회한 다음 **Change billing user**를 클릭합니다.

<div id="invoices">
  ## 청구서
</div>

신용카드로 결제하는 경우 이 섹션에서 월별 청구서를 확인할 수 있습니다.

* 계좌이체로 결제하는 Enterprise 계정의 경우 이 섹션은 비어 있습니다. 문의 사항이 있으면 account team에 문의하세요.
* 조직에 청구된 금액이 없으면 청구서가 생성되지 않습니다.
