사용자 프로필 페이지로 이동한 다음 오른쪽 상단의 사용자 아이콘을 선택하세요. 드롭다운에서 **Billing**을 선택하거나 **Settings**를 선택한 다음 **Billing** 탭을 선택하세요.

<div id="plan-details">
  ## 플랜 세부 정보
</div>

**플랜 세부 정보** 섹션에서는 조직의 현재 플랜, 요금, 한도, 사용량을 요약해 보여줍니다.

* 사용자 목록과 상세 정보를 보려면 **Manage users**를 클릭합니다.
* 사용량에 대한 상세 정보를 보려면 **View usage**를 클릭합니다.
* 조직에서 사용하는 저장소 용량(무료 및 유료 포함)을 보여줍니다. 여기에서 추가 저장소를 구매하고 현재 사용 중인 저장소를 관리할 수 있습니다. 자세한 내용은 [저장소 설정](/ko/platform/app/settings-page/storage/)을 참조하세요.

이곳에서 플랜을 비교하거나 Sales에 문의할 수 있습니다.

<div id="plan-usage">
  ## 플랜 사용
</div>

이 섹션에서는 현재 사용을 시각적으로 요약하며 향후 청구될 사용 요금을 표시합니다. 월별 사용에 대한 자세한 정보를 보려면 개별 타일에서 **View usage**를 클릭하세요. 달력 월, 팀 또는 프로젝트별 사용을 내보내려면 **Export CSV**를 클릭하세요.

사용이나 결제에 대한 질문이 있으시면 AISE 또는 [지원팀](mailto:support@wandb.com)로 문의하세요.

<div id="usage-alerts">
  ### 사용량 알림
</div>

사용량 알림은 현재 청구 기간 동안 각 범주에서 조직의 사용량이 한도의 85% 및 100%에 도달했을 때 조직 관리자와 결제 관리자에게 이메일을 보냅니다:

| Category      | 설명                                                                                                                                                                            |
| ------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 저장소       | 조직에서 저장한 아티팩트와 run 데이터의 총 누적 용량(GB 단위).                                                                                                                                       |
| Tracked hours | 청구 기간 동안의 경과 시간 기준 트레이닝 시간(시간 단위).                                                                                                                                            |
| Weave         | 청구 기간 동안의 총 데이터 수집량(MB 단위). 배포 환경에서 Weave가 활성화된 경우에만 표시됩니다.                                                                                                                   |
| Inference     | Inference에서 사용된 입력 및 출력 토큰의 총량입니다. [Inference Pricing](https://wandb.ai/site/pricing/inference/)을 참조하세요. 배포 환경에서 Inference가 활성화된 경우에만 표시됩니다.                                  |
| 트레이닝      | GPU/hour 단위로 측정되는 토큰 기반 inference 및 컴퓨트입니다. [Serverless RL Pricing](https://wandb.ai/site/pricing/reinforcement-learning/)을 참조하세요. 배포 환경에서 W&amp;B Training이 활성화된 경우에만 표시됩니다. |

**Enterprise** 플랜의 조직에서는 모든 **조직 관리자**과 **결제 관리자**이 사용량 알림을 받습니다. [Pro 플랜](https://wandb.ai/site/pricing/)에서는 **결제 관리자**만 사용량 알림을 받습니다.

메시지는 `support@wandb.com`에서 발송됩니다. 임곗값은 변경할 수 없지만, 범주별로 두 가지 알림 중 하나 또는 둘 다를 선택적으로 끌 수 있습니다.

**조직 관리자** 또는 **결제 관리자**은 조직의 사용량 알림을 구성할 수 있습니다:

1. 오른쪽 상단에서 사용자 프로필 아이콘을 클릭합니다.
2. **Account** 섹션에서 **Usage &amp; Alerts**를 클릭합니다. 또는 **Settings**를 클릭한 다음 **Usage &amp; Alerts** 탭을 클릭합니다.
3. 사이드바에서 범주를 선택합니다.
4. **Usage** 탭을 클릭하여 해당 범주에 대한 조직의 사용량 및 알림 설정을 확인합니다.
5. 원하는 경우, 해당 범주에 대해 **85% usage alert** 또는 **100% usage alert**를 켜거나 끕니다. <Warning>두 사용량 알림을 모두 끄면 해당 사용량 범주에 대해서는 어떤 알림도 전송되지 않습니다.</Warning>

<div id="spending-alerts">
  ### 지출 알림
</div>

청구 기간 중 조직이 USD 기준으로 설정한 하나 이상의 맞춤형 지출 임곗값에 도달하면, 지출 알림이 조직 관리자와 결제 관리자에게 이메일로 알립니다. 범주별로 서로 다른 지출 알림을 최대 5개까지 설정할 수 있습니다:

| 범주            | 설명                                                                                                                                                                                               |
| ------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 저장소           | 청구 기간 동안 조직의 저장소 [사용](#usage-alerts)에 대한 요금입니다. [가격](https://wandb.ai/site/pricing/)을 참조하세요.                                                                                                     |
| Tracked hours | 청구 기간 동안 조직의 트레이닝 [사용](#usage-alerts)에 대한 요금입니다. [가격](https://wandb.ai/site/pricing/)을 참조하세요.                                                                                                    |
| Weave         | 청구 기간 동안 조직의 Weave [사용](#usage-alerts)에 대한 요금입니다. [가격](https://wandb.ai/site/pricing/)을 참조하세요. 배포에서 Weave가 활성화된 경우에만 표시됩니다.                                                              |
| Inference     | 청구 기간 동안 조직의 Inference [사용](#usage-alerts)에 대한 요금입니다. [Inference Pricing](https://wandb.ai/site/pricing/inference/)을 참조하세요. 배포에서 Inference가 활성화된 경우에만 표시됩니다.                             |
| 트레이닝          | 청구 기간 동안 조직의 Serverless RL [사용](#usage-alerts)에 대한 요금입니다. [Serverless RL Pricing](https://wandb.ai/site/pricing/reinforcement-learning/)을 참조하세요. 배포에서 W&amp;B Training이 활성화된 경우에만 표시됩니다. |

메시지는 `support@wandb.com`에서 발송됩니다.

**Enterprise** 플랜을 사용하는 조직의 경우:

* 모든 **조직 관리자**와 **결제 관리자**가 지출 알림을 받습니다.
* 기본으로 설정된 지출 알림은 없습니다.

[Pro 플랜](https://wandb.ai/site/pricing/)에서는:

* **결제 관리자**만 지출 알림을 받습니다.
* 기본적으로 지출 알림은 $200, $450, $700, $1000으로 설정되어 있습니다.

<div id="view-spending-alerts">
  #### 지출 알림 보기
</div>

조직의 지출 알림을 보려면:

1. 오른쪽 상단에 있는 사용자 프로필 아이콘을 클릭하세요.
2. **Account** 섹션에서 **Usage &amp; Alerts**를 클릭하세요. 또는 **Settings**를 클릭한 다음 **Usage &amp; Alerts** 탭을 클릭하세요.
3. 사이드바에서 범주를 선택하세요.
4. 처음에는 **Usage** 탭이 표시됩니다. 지출 알림을 보려면 **Spend alerts**를 클릭하세요. 선택한 범주에 대해 설정된 모든 지출 알림이 표시됩니다.

<div id="add-or-delete-a-spending-alert">
  #### 지출 알림 추가 또는 삭제
</div>

지출 알림을 관리하려면 다음 단계를 따르세요.

1. 오른쪽 상단에 있는 사용자 프로필 아이콘을 클릭하세요.
2. **Account** 섹션에서 **Usage &amp; Alerts**를 클릭하세요. 또는 **Settings**를 클릭한 다음 **Usage &amp; Alerts** 탭을 클릭하세요.
3. 사이드바에서 **저장소**와 같은 범주를 선택하세요.
4. 알림을 추가하려면 **Add alert**를 클릭하세요.
   1. USD로 임곗값 금액을 입력하세요.
   2. **Create alert**를 클릭하세요.
5. 지출 알림을 삭제하려면 해당 알림 옆의 휴지통 아이콘을 클릭하세요.

<div id="payment-methods">
  ## 결제 수단
</div>

이 섹션에는 조직에 등록된 결제 수단가 표시됩니다. 결제 방법을 추가하지 않은 경우, 플랜을 업그레이드하거나 유료 저장소를 추가할 때 추가하라는 메시지가 표시됩니다.

<div id="billing-admin">
  ## 결제 관리자
</div>

이 섹션에는 현재 결제 관리자가 표시됩니다. 결제 관리자는 조직 관리자이며, 모든 결제 관련 이메일을 받고 결제 수단을 조회하고 관리할 수 있습니다.

<Note>
  W&amp;B Dedicated Cloud에서는 여러 사용자가 결제 관리자가 될 수 있습니다. W&amp;B Multi-tenant Cloud에서는 한 번에 한 명의 사용자만 결제 관리자가 될 수 있습니다.
</Note>

결제 관리자를 변경하거나 추가 사용자에게 이 역할을 부여하려면:

1. **Manage roles**를 클릭하세요.
2. 사용자를 검색하세요.
3. 해당 사용자의 행에서 **결제 관리자** 필드를 클릭하세요.
4. 요약을 조회한 다음 **Change billing user**를 클릭하세요.

<div id="invoices">
  ## 청구서
</div>

신용 카드를 사용하여 결제하는 경우, 이 섹션에서 월별 청구서를 확인할 수 있습니다.

* 계좌이체로 결제하는 Enterprise 계정의 경우, 이 섹션은 비어 있습니다. 문의 사항이 있으면 아카운트 담당자에게 문의하세요.
* 조직에 청구된 금액이 없으면 청구서가 생성되지 않습니다.
