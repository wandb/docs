사용자 프로필 페이지로 이동한 다음 오른쪽 상단의 사용자 아이콘을 선택합니다. 드롭다운에서 **Billing**을 선택하거나 **Settings**를 선택한 후 **Billing** 탭을 선택합니다.

<div id="plan-details">
  ## 플랜 세부 정보
</div>

**플랜 세부 정보** 섹션에서는 조직의 현재 플랜, 요금, 한도, 사용량을 요약해 보여줍니다.

* 사용자 목록과 상세 정보를 보려면 **Manage users**를 클릭합니다.
* 사용량에 대한 상세 정보를 보려면 **View usage**를 클릭합니다.
* 조직에서 사용하는 저장소 용량(무료 및 유료 포함)을 보여줍니다. 여기에서 추가 저장소를 구매하고 현재 사용 중인 저장소를 관리할 수 있습니다. 자세한 내용은 [저장소 설정](/ko/platform/app/settings-page/storage/)을 참조하세요.

이곳에서 플랜을 비교하거나 영업팀에 문의할 수 있습니다.

<div id="plan-usage">
  ## 플랜 사용량
</div>

이 섹션에서는 현재 사용량을 시각적으로 요약해 보여주고 향후 청구될 사용 요금을 표시합니다. 월별 사용량을 자세히 확인하려면 각 타일에서 **View usage**를 클릭합니다. 달력 기준 월, 팀, 또는 프로젝트 단위로 사용량을 내보내려면 **Export CSV**를 클릭합니다.

사용량 또는 청구 관련 질문이 있으면 AISE 또는 [Support](mailto:support@wandb.com)로 문의하세요.

<div id="usage-alerts">
  ### 사용량 알림
</div>

사용량 알림은 현재 청구 기간 동안 각 카테고리에서 조직의 사용량이 한도의 85% 및 100%에 도달했을 때 조직 관리자와 결제 관리자에게 이메일을 보냅니다:

| Category      | Description                                                                                                                                                                   |
| ------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Storage       | 조직에서 저장한 아티팩트와 run 데이터의 총 누적 용량(GB 단위).                                                                                                                                       |
| Tracked hours | 청구 기간 동안의 경과 시간 기준 트레이닝 시간(시간 단위).                                                                                                                                            |
| Weave         | 청구 기간 동안의 총 데이터 수집량(MB 단위). 배포 환경에서 Weave가 활성화된 경우에만 표시됩니다.                                                                                                                   |
| Inference     | Inference에서 사용된 입력 및 출력 토큰의 총량입니다. [Inference Pricing](https://wandb.ai/site/pricing/inference/)을 참조하세요. 배포 환경에서 Inference가 활성화된 경우에만 표시됩니다.                                  |
| Training      | GPU/hour 단위로 측정되는 토큰 기반 inference 및 컴퓨트입니다. [Serverless RL Pricing](https://wandb.ai/site/pricing/reinforcement-learning/)을 참조하세요. 배포 환경에서 W&amp;B Training이 활성화된 경우에만 표시됩니다. |

**Enterprise** 플랜의 조직에서는 모든 **Organization admin**과 **Billing admin**이 사용량 알림을 받습니다. [Pro 플랜](https://wandb.ai/site/pricing/)에서는 **Billing admin**만 사용량 알림을 받습니다.

메시지는 `support@wandb.com`에서 발송됩니다. 임계값은 변경할 수 없지만, 카테고리별로 두 가지 알림 중 하나 또는 둘 다를 선택적으로 끌 수 있습니다.

**Organization admin** 또는 **Billing admin**은 조직의 사용량 알림을 구성할 수 있습니다:

1. 오른쪽 상단에서 사용자 프로필 아이콘을 클릭합니다.
2. **Account** 섹션에서 **Usage &amp; Alerts**를 클릭합니다. 또는 **Settings**를 클릭한 다음 **Usage &amp; Alerts** 탭을 클릭합니다.
3. 사이드바에서 카테고리를 선택합니다.
4. **Usage** 탭을 클릭하여 해당 카테고리에 대한 조직의 사용량 및 알림 설정을 확인합니다.
5. 원하는 경우, 해당 카테고리에 대해 **85% usage alert** 또는 **100% usage alert**를 켜거나 끕니다. <Warning>두 사용량 알림을 모두 끄면 해당 사용량 카테고리에 대해서는 어떤 알림도 전송되지 않습니다.</Warning>

<div id="spending-alerts">
  ### 지출 알림
</div>

특정 청구 기간 동안 지출 알림은 조직이 USD 기준으로 설정한 하나 이상의 지출 임계값에 도달했을 때 조직 관리자와 결제 관리자에게 이메일로 알려 줍니다. 카테고리별로 최대 5개의 고유한 지출 알림을 설정할 수 있습니다:

| Category      | Description                                                                                                                                                                                     |
| ------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Storage       | 청구 기간 동안 조직의 스토리지 [usage](#usage-alerts) 사용량에 대한 요금입니다. [Pricing](https://wandb.ai/site/pricing/)을 참고하세요.                                                                                       |
| Tracked hours | 청구 기간 동안 조직의 트레이닝 [usage](#usage-alerts) 사용량에 대한 요금입니다. [Pricing](https://wandb.ai/site/pricing/)을 참고하세요.                                                                                       |
| Weave         | 청구 기간 동안 조직의 Weave [usage](#usage-alerts) 사용량에 대한 요금입니다. [Pricing](https://wandb.ai/site/pricing/)을 참고하세요. 배포에서 Weave가 활성화된 경우에만 표시됩니다.                                                         |
| Inference     | 청구 기간 동안 조직의 Inference [usage](#usage-alerts) 사용량에 대한 요금입니다. [Inference Pricing](https://wandb.ai/site/pricing/inference/)을 참고하세요. 배포에서 Inference가 활성화된 경우에만 표시됩니다.                             |
| Training      | 청구 기간 동안 조직의 Serverless RL [usage](#usage-alerts) 사용량에 대한 요금입니다. [Serverless RL Pricing](https://wandb.ai/site/pricing/reinforcement-learning/)을 참고하세요. 배포에서 W&amp;B Training이 활성화된 경우에만 표시됩니다. |

메시지는 `support@wandb.com`에서 발송됩니다.

**Enterprise** 플랜의 조직의 경우:

* 모든 **Organization admins**와 **Billing admins**가 지출 알림을 받습니다.
* 기본으로 설정된 지출 알림은 없습니다.

[Pro 플랜](https://wandb.ai/site/pricing/)에서는:

* **Billing admin**만 지출 알림을 받습니다.
* 기본으로 지출 알림은 $200, $450, $700, $1000에 설정되어 있습니다.

<div id="view-spending-alerts">
  #### 지출 알림 보기
</div>

조직의 지출 알림을 보려면:

1. 오른쪽 상단에 있는 사용자 프로필 아이콘을 클릭합니다.
2. **Account** 섹션에서 **Usage &amp; Alerts**를 클릭합니다. 또는 **Settings**를 클릭한 다음 **Usage &amp; Alerts** 탭을 클릭합니다.
3. 사이드바에서 범주를 선택합니다.
4. 처음에는 **Usage** 탭이 표시됩니다. 지출 알림을 보려면 **Spend alerts**를 클릭합니다. 선택한 범주에 대해 설정된 모든 지출 알림이 표시됩니다.

<div id="add-or-delete-a-spending-alert">
  #### 지출 알림 추가 또는 삭제
</div>

지출 알림을 관리하려면 다음 단계를 따르세요.

1. 오른쪽 상단에 있는 사용자 프로필 아이콘을 클릭합니다.
2. **Account** 섹션에서 **Usage &amp; Alerts**를 클릭합니다. 또는 **Settings**를 클릭한 다음 **Usage &amp; Alerts** 탭을 클릭합니다.
3. 사이드바에서 **Storage**와 같은 카테고리를 선택합니다.
4. 알림을 추가하려면 **Add alert**를 클릭합니다.
   1. 임계 금액을 USD로 입력합니다.
   2. **Create alert**를 클릭합니다.
5. 지출 알림을 삭제하려면 해당 알림 옆의 휴지통 아이콘을 클릭합니다.

<div id="payment-methods">
  ## 결제 수단
</div>

이 섹션에는 조직에 등록된 결제 수단이 표시됩니다. 결제 수단을 아직 추가하지 않은 경우, 요금제를 업그레이드하거나 유료 스토리지를 추가할 때 결제 수단을 추가하라는 안내가 표시됩니다.

<div id="billing-admin">
  ## 결제 관리자
</div>

이 섹션에는 현재 결제 관리자가 표시됩니다. 결제 관리자는 조직 관리자이며, 모든 결제 관련 이메일을 받고 결제 수단을 조회하고 관리할 수 있습니다.

<Note>
  W&amp;B Dedicated Cloud에서는 여러 사용자가 결제 관리자가 될 수 있습니다. W&amp;B Multi-tenant Cloud에서는 한 번에 한 명의 사용자만 결제 관리자가 될 수 있습니다.
</Note>

결제 관리자를 변경하거나 추가 사용자에게 이 역할을 지정하려면:

1. **Manage roles**를 클릭합니다.
2. 사용자를 검색합니다.
3. 해당 사용자의 행에서 **Billing admin** 필드를 클릭합니다.
4. 요약을 읽은 다음 **Change billing user**를 클릭합니다.

<div id="invoices">
  ## 청구서
</div>

신용카드로 결제하는 경우, 이 섹션에서 월별 청구서를 확인할 수 있습니다.

* 계좌 이체(전신 송금)로 결제하는 Enterprise 계정의 경우, 이 섹션에는 아무 내용도 표시되지 않습니다. 궁금한 점이 있으면 계정 담당 팀에 문의하세요.
* 조직에 청구가 발생하지 않은 경우, 청구서는 생성되지 않습니다.
