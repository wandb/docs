W&B 프로필 설정 페이지에서 이메일 유형과 기본 이메일 주소를 추가, 삭제 및 관리할 수 있습니다. W&B 대시보드 오른쪽 상단에 있는 프로필 아이콘을 선택합니다. 드롭다운 메뉴에서 **Settings**를 선택합니다. **Settings** 페이지에서 아래로 스크롤해 Emails 대시보드로 이동합니다:

<Frame>
    <img src="/images/app_ui/manage_emails.png" alt="이메일 관리 대시보드"  />
</Frame>

<div id="manage-primary-email">
  ## 기본 이메일 관리
</div>

기본 이메일은 😎 이모지로 표시됩니다. 기본 이메일은 W&amp;B 계정을 생성할 때 입력한 이메일로 자동 설정됩니다.

W&amp;B 계정에 연결된 기본 이메일을 변경하려면 케밥 메뉴 드롭다운을 선택하세요.

<Note>
인증된 이메일만 기본 이메일로 설정할 수 있습니다.
</Note>

<Frame>
    <img src="/images/app_ui/primary_email.png" alt="기본 이메일 드롭다운"  />
</Frame>

<div id="add-emails">
  ## 이메일 추가
</div>

이메일을 추가하려면 **+ Add Email**을 선택합니다. Auth0 페이지로 이동하며, 여기에서 새 이메일의 로그인 정보를 입력하거나 단일 로그인(SSO)을 사용해 연결할 수 있습니다.

<div id="delete-emails">
  ## 이메일 삭제
</div>

케밥 드롭다운을 열고 **Delete Emails**를 선택하여 W&B 계정에 등록된 이메일 주소를 삭제합니다.

<Note>
기본 이메일은 삭제할 수 없습니다. 삭제하기 전에 다른 이메일을 기본 이메일로 먼저 설정해야 합니다.
</Note>

<div id="log-in-methods">
  ## 로그인 방법
</div>

로그인 방법 열에는 계정에 연결된 로그인 방법이 표시됩니다.

W&B 계정을 생성하면 확인 이메일이 이메일 계정으로 전송됩니다. 이메일 주소를 확인하기 전까지는 이메일 계정이 미확인 상태로 간주됩니다. 미확인 이메일은 빨간색으로 표시됩니다.

이메일 계정으로 전송된 첫 번째 확인 이메일이 더 이상 없는 경우, 이메일 주소로 다시 로그인 시도하여 두 번째 확인 이메일을 받으십시오.

계정 로그인 관련 문제가 있으면 support@wandb.com으로 문의하십시오.
