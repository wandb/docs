---
title: W&B가 컨테이너를 빌드하지 않았으면 합니다. 그래도 Launch를 사용할 수 있나요?
---

미리 빌드된 Docker 이미지로 실행하려면 다음 명령을 실행하세요. `<>`로 표시된 플레이스홀더를 사용자의 구체적인 정보로 바꾸세요:

```bash
wandb launch -d <docker-image-uri> -q <queue-name> -E <entrypoint>
```

이 명령은 Job을 생성하고 실행을 시작합니다.

이미지로부터 Job을 생성하려면 다음 명령을 실행하세요:

```bash
wandb job create image <image-name> -p <project> -e <entity>
```
