---
title: W&B Launch는 이미지를 어떻게 빌드하나요?
---

이미지를 빌드하는 단계는 잡 소스와 리소스 구성에 지정된 가속기 기본 이미지에 따라 달라집니다.

<Note>
큐를 구성하거나 잡을 제출할 때, 큐 또는 잡 리소스 구성에 가속기 기본 이미지를 포함해야 합니다:
```json
{
    "builder": {
        "accelerator": {
            "base_image": "image-name"
        }
    }
}
```
</Note>

빌드 프로세스에는 잡 유형과 제공된 가속기 기본 이미지에 따라 다음 작업이 포함됩니다:

| | apt를 사용한 Python 설치 | Python 패키지 설치 | 사용자 및 작업 디렉터리 생성 | 코드를 이미지에 복사 | 엔트리포인트 설정 | |
