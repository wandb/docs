---
title: W&B Launch에서 이미지를 어떻게 빌드하나요?
---

이미지 빌드 단계는 작업 소스와 리소스 설정에서 지정한 가속기 베이스 이미지에 따라 달라집니다.

<Note>
  큐를 설정하거나 작업을 제출할 때는 큐 또는 작업 리소스 설정에 가속기 베이스 이미지를 포함하세요:

  ```json
  {
      "builder": {
          "accelerator": {
              "base_image": "image-name"
          }
      }
  }
  ```
</Note>

빌드 프로세스에는 작업 유형과 지정된 가속기 베이스 이미지에 따라 다음 작업이 포함됩니다:

| | apt를 사용한 Python 설치 | Python 패키지 설치 | 사용자 및 작업 디렉터리 생성 | 이미지에 코드 복사 | 엔트리포인트 설정 | |
