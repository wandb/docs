---
title: 작업/자동화에 시크릿을 지정할 수 있나요? 예를 들어 사용자에게 직접
  보이지 않게 하고 싶은 API 키가 있을 때요?
---

네. 다음 단계를 따르세요:

1. 다음 명령을 사용해, 해당 실행이 수행될 네임스페이스에 Kubernetes 시크릿을 생성합니다:  
   `kubectl create secret -n <namespace> generic <secret_name> <secret_value>`

2. 시크릿을 생성한 후, 실행이 시작될 때 이 시크릿이 주입되도록 큐를 설정합니다. 클러스터 관리자만 이 시크릿을 볼 수 있으며, 최종 사용자는 볼 수 없습니다.
