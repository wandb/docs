---
title: 작업/자동화에 사용할 시크릿을 지정할 수 있나요? 예를 들어, 사용자에게
  직접 보이지 않게 하고 싶은 API 키 같은 값이 있을 때요?
---

네. 다음 단계를 따르세요:

1. 다음 명령을 사용해 해당 run이 사용하는 지정된 네임스페이스에 Kubernetes 시크릿을 생성합니다:  
   `kubectl create secret -n <namespace> generic <secret_name> <secret_value>`

2. 시크릿을 생성한 후, run이 시작될 때 이 시크릿이 주입되도록 큐를 구성합니다. 클러스터 관리자만 시크릿을 볼 수 있으며, 일반 사용자는 볼 수 없습니다.
