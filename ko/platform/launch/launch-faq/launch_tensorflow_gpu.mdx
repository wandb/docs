---
title: W&B Launch를 TensorFlow GPU와 함께 사용하려면 어떻게 해야 하나요?
---

GPU를 사용하는 TensorFlow 작업의 경우, 컨테이너 빌드를 위해 커스텀 베이스 이미지를 지정해야 합니다. 이렇게 하면 실행 중에 GPU를 적절히 활용할 수 있습니다. 리소스 구성에서 `builder.accelerator.base_image` 키 아래에 이미지 태그를 추가하십시오. 예를 들면 다음과 같습니다:

```json
{
    "gpus": "all",
    "builder": {
        "accelerator": {
            "base_image": "tensorflow/tensorflow:latest-gpu"
        }
    }
}
```

W&amp;B 0.15.6 이전 버전에서는 `base_image`의 상위 키로 `accelerator`가 아니라 `cuda`를 사용해야 합니다.
