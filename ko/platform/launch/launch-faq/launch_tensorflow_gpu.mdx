---
title: Tensorflow와 GPU를 사용하여 W&B Launch를 작동시키려면 어떻게 해야 하나요?
---

GPU 를 사용하는 TensorFlow 작업의 경우, 컨테이너 빌드를 위한 커스텀 베이스 이미지를 지정하세요. 이를 통해 Runs 가 진행되는 동안 GPU 가 적절하게 활용되도록 보장할 수 있습니다. 리소스 설정의 `builder.accelerator.base_image` 키 아래에 이미지 태그를 추가하세요. 예시는 다음과 같습니다:

```json
{
    "gpus": "all",
    "builder": {
        "accelerator": {
            "base_image": "tensorflow/tensorflow:latest-gpu"
        }
    }
}
```

W&B 0.15.6 이전 버전에서는 `base_image` 의 부모 키로 `accelerator` 대신 `cuda` 를 사용하세요.
