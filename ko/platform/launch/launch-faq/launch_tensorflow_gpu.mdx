---
title: TensorFlow에서 GPU와 함께 W&B Launch를 사용하려면 어떻게 해야 하나요?
---

GPU를 사용하는 TensorFlow 작업에서는 컨테이너 빌드를 위해 커스텀 베이스 이미지를 지정하세요. 이렇게 하면 run 동안 GPU가 제대로 활용됩니다. 리소스 설정에서 `builder.accelerator.base_image` 키 아래에 이미지 태그를 추가하세요. 예를 들어:

```json
{
    "gpus": "all",
    "builder": {
        "accelerator": {
            "base_image": "tensorflow/tensorflow:latest-gpu"
        }
    }
}
```

W&amp;B 0.15.6 이전 버전에서는 `base_image`의 상위 키로 `accelerator`가 아니라 `cuda`를 사용하세요.
