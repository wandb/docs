---
title: W&B Launch가 GPU에서 TensorFlow와 함께 작동하도록 하려면 어떻게 하나요?
---

GPU를 사용하는 TensorFlow 작업의 경우 컨테이너 빌드에 맞춤형 베이스 이미지를 지정하세요. 이렇게 하면 run 중에 GPU를 올바르게 활용할 수 있습니다. 예를 들어, 리소스 설정의 `builder.accelerator.base_image` 키 아래에 이미지 태그를 추가합니다.

```json
{
    "gpus": "all",
    "builder": {
        "accelerator": {
            "base_image": "tensorflow/tensorflow:latest-gpu"
        }
    }
}
```

W&amp;B 0.15.6 이전 버전에서는 `base_image`의 부모 키로 `accelerator` 대신 `cuda`를 사용하세요.
