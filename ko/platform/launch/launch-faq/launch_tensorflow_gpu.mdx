---
title: GPU에서 TensorFlow와 함께 W&B Launch를 사용하려면 어떻게 해야 하나요?
---

GPU를 사용하는 TensorFlow 작업의 경우, 컨테이너 빌드에 사용할 맞춤형 기본 이미지를 지정하십시오. 이렇게 하면 실행 중 GPU 사용량이 적절히 확보됩니다. 리소스 설정의 `builder.accelerator.base_image` 키 아래에 이미지 태그를 추가하십시오. 예를 들면 다음과 같습니다.

```json
{
    "gpus": "all",
    "builder": {
        "accelerator": {
            "base_image": "tensorflow/tensorflow:latest-gpu"
        }
    }
}
```

W&amp;B 0.15.6 이전 버전에서는 `base_image`의 상위 키로 `accelerator` 대신 `cuda`를 사용하십시오.
