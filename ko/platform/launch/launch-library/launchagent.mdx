주어진 run queue를 폴링(poll)하고 wandb Launch 를 위해 run 을 실행하는 Launch 에이전트 클래스입니다.

```python
LaunchAgent(
    api: Api,
    config: Dict[str, Any]
)
```

| 인수 | 설명 |
| :--- | :--- |
| `api` | 백엔드에 요청을 보내는 데 사용할 Api 오브젝트입니다. |
| `config` | 에이전트를 위한 설정 사전(Config dictionary)입니다. |

| 속성 | 설명 |
| :--- | :--- |
| `num_running_jobs` | 스케줄러를 제외하고 실행 중인 job의 수를 반환합니다. |
| `num_running_schedulers` | 실행 중인 스케줄러의 수만 반환합니다. |
| `thread_ids` | 에이전트에서 실행 중인 스레드 ID의 키 리스트를 반환합니다. |
