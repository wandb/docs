title: "Service API overview"
description: "Use the Weave Service API, which provides REST endpoints for programmatic access to Weave functionality."
---

## Base URL

```
https://trace.wandb.ai
```

## Authentication

Include your W&B API key in the request headers:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" https://trace.wandb.ai/...
```
