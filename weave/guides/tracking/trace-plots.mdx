The _trace plots_ tool in W&B Weave lets you explore, visualize, and debug trace-level metrics (for example, latency, cost, and tokens) using interactive charts. You can use the default trace plots or create your own through the custom trace plot builder. This guide shows you how to open the trace plots panel, work with the default plots, and build custom plots tailored to your project's data.

## Get started

Follow these steps to open the trace plots side pane for your project:

1. Navigate to your project's **Traces** page.
2. Optional: Select **Filter** to filter selected traces (for example, by datetime or operation).
3. In the upper right corner of the **Traces** view, click the **Show Metrics** icon to open the trace plots side pane.

    ![The **Show Metrics** icon](/weave/guides/tracking/imgs/plots-show-metrics-icon.png)

    From here, you can:
   - View the [default trace plots](#default-trace-plots).
   - Create a [custom trace plot](#create-a-custom-trace-plot).
4. Charts update dynamically based on your trace filters and selections.

## Default trace plots

When you first open the trace plots panel, Weave automatically generates trace plots based on your available project trace data:

- Bar chart (cost or latency grouped over time bins).
- Line chart (latency over time).
- Scatter plot (for example, prompt tokens versus completion tokens).

Each trace plot is interactive:

- Hover for tooltips.
- Drag to zoom.
- Double-click to reset.
- Click points in scatter plots to open a specific trace.

## Create a custom trace plot

If the default trace plots don't show the metrics or relationships you need, you can build your own. To create a custom trace plot, do the following:

1. From the trace plots side pane, click **➕ Add Chart**.

    ![**➕ Add Chart** button.](/weave/guides/tracking/imgs/plots-add-chart.png)

2. In the pop-up, select one of the available trace plot types:

    ![Available custom trace plot types](/weave/guides/tracking/imgs/plots-custom-chart-types.png)
    
3. Configure your trace plot for the selected type. For information on configuration options by trace plot type, see [Trace plot settings by plot type](#trace-plot-settings-by-plot-type).
4. Click **Save chart** to save your chart.

Your custom trace plot now appears in the trace plots side pane alongside the default plots and updates dynamically with your trace filters and selections.

### Trace plot settings by plot type

When you add or edit a custom trace plot, the available configuration options vary depending on the selected trace plot type. The following table describes the configurable options.

| Setting         | Scatter Plot         | Line Chart           | Bar Chart            |
| --------------- | -------------------- | -------------------- | -------------------- |
| Y-axis      | ✓ Required           | ✓ Required           | ✓ Required           |
| X-axis      | ✓ Selectable         | Fixed (`started at`) | Fixed (`started at`) |
| Grouping    | ✓ Optional           | ✓ Optional           | ✓ Optional           |
| Binning     | ✗ Not used           | ✓ Used               | ✓ Used               |
| Aggregation | ✗ Not used           | ✓ Used               | ✓ Used               |
