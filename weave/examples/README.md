# Weave examples

This directory contains runnable examples that show how to use Weave, from quick interactive notebooks to more advanced proofs-of-concept. Use these examples to explore Weave's features, learn common patterns, and adapt sample code for your own projects.

## Directory structure

The following list describes what you can find in each subdirectory of the `examples/` folder:

- `get_started.ipynb`: A quick interactive notebook overview of Weave.
- `getting_started`: Learn how to build interactive data applications.
- `reference`: Syntax and usage for Weave types, ops, and panels.
- `monitoring`: Examples and documentation for LLM monitoring.
- `experimental`: Everything else, from focused proofs-of-concept to experimental works-in-progress. Use these examples to inspire your exploration, and submit PRs to suggest improvements.
- `apps`: Example applications.
- `guides`: Guides to use specific features for advanced users.

## Run examples

You can run these examples either locally with Jupyter or in Google Colab.

### Run locally with Jupyter notebook

Run `jupyter notebook` or `jupyter lab` in the root directory `examples`, then use the Jupyter browser to find and open a notebook.

### Run in Google Colab

Run the same code in Google Colab. Install Weave from PyPI first with `!pip install weave`.

Use the following Google Colab notebooks to get started:

- [Weave Quickstart](https://colab.research.google.com/drive/1TwlhvvoWIHKDtRUu6eW0NMRq0GzGZ9oX)
- [Weave Board Quickstart: Generative AI Playground](https://colab.research.google.com/drive/1gcR-ucIgjDbDEBFykEpJ3kkBoBQ84Ipr)
