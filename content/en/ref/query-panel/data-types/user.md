---
title: User
---

Represents a W&B user.

```typescript
const user: User = {
  id: "user_123",
  username: "john_doe",
  name: "John Doe",
  email: "john@example.com"
};
```

| Property | Type | Description |
| :------- | :--- | :---------- |
| `id` | `string` | User ID |
| `username` | `string` | Username |
| `name` | `string` | *Optional*. User's full name |
| `email` | `string` | *Optional*. User email |
