Commands for operating a local W&B server

## Usage

```bash
wandb server COMMAND [ARGS]...
```

## Commands

| Command | Description |
| :--- | :--- |
| [start]({{< relref "wandb-server-start" >}}) | Start a local W&B server |
| [stop]({{< relref "wandb-server-stop" >}}) | Stop a local W&B server |
