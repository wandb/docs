---
title: wandb artifact ls
---

List all artifacts in a wandb project

## Usage

```bash
wandb ls PATH [OPTIONS]
```

## Arguments

| Argument | Description | Required |
| :--- | :--- | :--- |
| `PATH` | No description available | Yes |

## Options

| Option | Description |
| :--- | :--- |
| `--type`, `-t` | The type of artifacts to list |
