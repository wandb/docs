Use the [public API]({{< relref "/ref/python/public-api/api.md" >}}) to delete multiple runs in a single operation:

```python
import wandb

api = wandb.Api()
runs = api.runs('<entity>/<project>')
for run in runs:
    if <condition>:
        run.delete()
```
