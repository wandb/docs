**사용법**

`wandb server [옵션] 코맨드 [ARG]...`

**요약**

로컬 W&B 서버를 운영하기 위한 코맨드입니다.

**옵션**

| **옵션** | **설명** |
| :--- | :--- |

**코맨드**

| **코맨드** | **설명** |
| :--- | :--- |
| start | 로컬 W&B 서버를 시작합니다 |
| stop | 로컬 W&B 서버를 중지합니다 |
