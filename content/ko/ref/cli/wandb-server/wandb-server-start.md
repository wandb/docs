**사용법**

`wandb server start [OPTIONS]`

**요약**

로컬 W&B server 를 시작합니다

**옵션**

| **옵션** | **설명** |
| :--- | :--- |
| `-p, --port` | W&B server 를 바인딩할 호스트 포트 |
| `-e, --env` | wandb/local 에 전달할 환경 변수 |
| `--daemon / --no-daemon` | 데몬 모드로 실행하거나 실행하지 않음 |
