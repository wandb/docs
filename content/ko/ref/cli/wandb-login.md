**사용법**

`wandb login [옵션] [키]...`

**요약**

Weights & Biases에 로그인합니다

**옵션**

| **옵션** | **설명** |
| :--- | :--- |
| `--cloud` | 로컬이 아닌 클라우드에 로그인합니다 |
| `--host, --base-url` | 특정 W&B 인스턴스에 로그인합니다 |
| `--relogin` | 이미 로그인되어 있어도 다시 로그인합니다. |
| `--anonymously` | 익명으로 로그인합니다 |
| `--verify / --no-verify` | 로그인 자격 증명을 확인합니다 |
