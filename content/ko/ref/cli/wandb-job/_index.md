**사용법**

`wandb job [OPTIONS] 코맨드 [ARG]...`

**요약**

W&B job 을 관리하고 조회하는 코맨드


**옵션**

| **옵션** | **설명** |
| :--- | :--- |


**코맨드**

| **코맨드** | **설명** |
| :--- | :--- |
| create | wandb run 없이 소스에서 job 을 생성합니다. |
| describe | launch job 을 설명합니다. |
| list | 프로젝트 내의 job 을 나열합니다. |
