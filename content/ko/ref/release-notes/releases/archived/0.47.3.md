# 수정 사항

v0.47.2에서 출시된 _**Custom Roles**_ 프리뷰 기능과 관련하여 중요한 몇 가지 수정 사항을 안내드립니다. 더 세분화된 역할을 만들고 최소 권한 원칙을 더욱 잘 준수하고 싶으시다면 반드시 이번 최신 서버 릴리스를 이용해 주시기 바랍니다. 최신 엔터프라이즈 라이선스가 필요하신 경우 W&B 팀에 문의해 주세요.
