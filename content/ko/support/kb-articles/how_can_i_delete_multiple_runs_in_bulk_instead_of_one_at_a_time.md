[public API]({{< relref path="/ref/python/public-api/api.md" lang="ko" >}})를 사용하여 단일 작업에서 여러 개의 runs을 삭제합니다.

```python
import wandb

api = wandb.Api()
runs = api.runs('<entity>/<project>')
for run in runs:
    if <condition>:
        run.delete()
```
