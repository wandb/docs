W&B 클라이언트 라이브러리는 Python 3.5를 0.11 버전까지 지원했습니다. Python 3.5 지원은 0.12 버전부터 종료되었으며, 이는 공식 지원 종료 시점과 일치합니다. 자세한 내용은 [version 0.12 release notes](https://github.com/wandb/wandb/releases/tag/v0.12.0)를 참고하세요.
