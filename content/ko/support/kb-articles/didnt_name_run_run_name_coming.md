run 에 이름을 명시적으로 지정하지 않으면, W&B는 프로젝트 내에서 식별할 수 있도록 무작위 이름을 할당합니다. 예를 들어 `pleasant-flower-4` 와 `misunderstood-glade-2` 같은 이름이 있습니다.
