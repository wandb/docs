삭제된 run 을 복구하려면 다음 단계를 따라주세요.

- Project Overview 페이지로 이동합니다.
- 오른쪽 상단의 점 세 개 아이콘을 클릭합니다.
- **Undelete recently deleted runs**를 선택합니다.

**참고**:
- 최근 7일 이내에 삭제된 run 만 복구할 수 있습니다.
- 복구 옵션을 사용할 수 없는 경우 W&B API를 사용해 로그를 직접 업로드할 수 있습니다.
