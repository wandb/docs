一度 W&B Sweep が開始されると、その Sweep configuration を変更することはできません。ただし、任意のテーブルビューに移動し、チェックボックスを使って Run を選択し、**Create sweep** メニューオプションを選ぶことで、過去の Run をもとに新しい Sweep configuration を作成できます。
