W&B Sweep が開始されると、sweep configuration を変更することはできません。ただし、任意のテーブルビューに移動し、チェックボックスで runs を選択してから、メニューの **Create sweep** を選ぶことで、過去の runs に基づいた新しい sweep configuration を作成できます。
