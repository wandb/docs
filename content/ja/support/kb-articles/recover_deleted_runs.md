削除した run を復元するには、以下の手順を実行してください。

- Project Overview ページに移動します。
- 右上の三点リーダーをクリックします。
- **Undelete recently deleted runs** を選択します。

**注意**:
- 復元できるのは、過去7日以内に削除された run のみです。
- 復元ができない場合は、W&B API を使って手動でログをアップロードできます。
