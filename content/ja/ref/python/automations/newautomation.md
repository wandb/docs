新しく作成するオートメーション。

属性:
- action (任意): このオートメーションがトリガーされたときに実行されるアクション。
- description (任意): このオートメーションの説明。
- enabled (任意): このオートメーションが有効かどうか。有効なオートメーションのみがトリガーされます。
- event (任意): このオートメーションをトリガーするイベント。
- name (任意): このオートメーションの名前。
