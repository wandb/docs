**Usage**

`wandb artifact get [OPTIONS] PATH`

**Summary**

Download an artifact from wandb


**Options**

| **Option** | **Description** |
| :--- | :--- |
| `--root` | The directory you want to download the artifact to |
| `--type` | The type of artifact you are downloading |
