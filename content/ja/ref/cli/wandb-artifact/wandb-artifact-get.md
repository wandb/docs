**使用方法**

`wandb artifact get [OPTIONS] PATH`

**概要**

wandb から artifact をダウンロードします


**オプション**

| **オプション** | **説明** |
| :--- | :--- |
| `--root` | artifact をダウンロードするディレクトリー |
| `--type` | ダウンロードする artifact の種類 |
