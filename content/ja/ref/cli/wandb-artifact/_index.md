**使い方**

`wandb artifact [OPTIONS] COMMAND [ARGS]...`

**概要**

artifacts とやり取りするためのコマンド


**オプション**

| **オプション** | **説明** |
| :--- | :--- |


**コマンド**

| **コマンド** | **説明** |
| :--- | :--- |
| cache | artifact キャッシュを操作するためのコマンド |
| get | wandb から artifact をダウンロード |
| ls | wandb project 内のすべての artifacts を一覧表示 |
| put | artifact を wandb にアップロード |
