**使用方法**

`wandb init [OPTIONS]`

**概要**

Weights & Biases 用にディレクトリーを設定します

**オプション**

| **オプション** | **説明** |
| :--- | :--- |
| `-p, --project` | 使用するプロジェクト。 |
| `-e, --entity` | プロジェクトを紐付けるエンティティ。 |
| `--reset` | 設定をリセットします。 |
| `-m, --mode` | "online"、"offline"、または "disabled" にできます。デフォルトは online です。 |
