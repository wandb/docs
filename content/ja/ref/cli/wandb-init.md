**使用方法**

`wandb init [OPTIONS]`

**概要**

ディレクトリーを Weights & Biases で設定します

**オプション**

| **オプション** | **説明** |
| :--- | :--- |
| `-p, --project` | 使用する Project。 |
| `-e, --entity` | Project を紐づける Entity。 |
| `--reset` | 設定をリセットします |
| `-m, --mode` | "online"、"offline"、または "disabled" を指定可能。デフォルトは online。 |
