---
title: wandb init
---

**使い方**

`wandb init [OPTIONS]`

**概要**

ディレクトリーを Weights & Biases で設定します

**オプション**

| **オプション** | **説明** |
| :--- | :--- |
| `-p, --project` | 利用する Project を指定します。 |
| `-e, --entity` | Project のスコープとなる Entity を指定します。 |
| `--reset` | 設定をリセットします |
| `-m, --mode` | "online"、"offline"、"disabled" のいずれかを選択可能です。デフォルトは online です。 |
