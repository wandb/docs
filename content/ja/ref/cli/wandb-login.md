---
title: wandb login
---

**使用方法**

`wandb login [OPTIONS] [KEY]...`

**概要**

Weights & Biases にログインする

**オプション**

| **オプション** | **説明** |
| :--- | :--- |
| `--cloud` | ローカルではなくクラウドにログインする |
| `--host, --base-url` | W&B の特定インスタンスにログインする |
| `--relogin` | すでにログインしている場合でも再ログインを強制する |
| `--anonymously` | 匿名でログインする |
| `--verify / --no-verify` | ログイン情報を検証する |
