**使用方法**

`wandb job [OPTIONS] COMMAND [ARGS]...`

**概要**

W&B ジョブの管理および表示のためのコマンド

**オプション**

| **オプション** | **説明** |
| :--- | :--- |

**コマンド**

| **コマンド** | **説明** |
| :--- | :--- |
| create | wandb run を使わずにソースからジョブを作成します。|
| describe | Launch ジョブの詳細を表示します。|
| list | プロジェクト内のジョブを一覧表示します。|
