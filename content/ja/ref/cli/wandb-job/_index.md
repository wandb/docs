**使用方法**

`wandb job [OPTIONS] COMMAND [ARGS]...`

**概要**

W&B jobs を管理および表示するためのコマンド


**オプション**

| **オプション** | **説明** |
| :--- | :--- |


**コマンド**

| **コマンド** | **説明** |
| :--- | :--- |
| create | ソースから、wandb run を伴わずに job を作成します。 |
| describe | Launch job の詳細を表示します。 |
| list | project 内の job を一覧表示します。 |
