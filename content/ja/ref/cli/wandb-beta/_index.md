**使い方**

`wandb beta [OPTIONS] コマンド [arg]...`

**概要**

wandb CLI コマンドのベータ版です。wandb-core が必要です。

| **オプション** | **説明** |
| :--- | :--- |

**コマンド**

| **コマンド** | **説明** |
| :--- | :--- |
| sync | トレーニング run を W&B にアップロードします |
