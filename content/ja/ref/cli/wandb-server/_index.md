**使用方法**

`wandb server [OPTIONS] COMMAND [ARGS]...`

**概要**

ローカルの W&B サーバーを操作するためのコマンド

**オプション**

| **オプション** | **説明** |
| :--- | :--- |

**コマンド**

| **コマンド** | **説明** |
| :--- | :--- |
| start | ローカルの W&B サーバーを起動する |
| stop | ローカルの W&B サーバーを停止する |
