**使い方**

`wandb server [OPTIONS] コマンド [arg]...`

**概要**

ローカル W&B サーバー を操作するためのコマンド


**オプション**

| **オプション** | **説明** |
| :--- | :--- |


**コマンド**

| **コマンド** | **説明** |
| :--- | :--- |
| start | ローカル W&B サーバー を起動する |
| stop | ローカル W&B サーバー を停止する |
