**使い方**

`wandb server start [OPTIONS]`

**概要**

ローカル W&B サーバーを起動します


**オプション**

| **オプション** | **説明** |
| :--- | :--- |
| `-p, --port` | W&B サーバーをバインドするホストポート |
| `-e, --env` | wandb/local に渡す環境変数 |
| `--daemon / --no-daemon` | デーモンモードで実行するかどうか |
