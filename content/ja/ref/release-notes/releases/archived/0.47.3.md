# 修正

バージョン v0.47.2 の一部として提供開始した _**Custom Roles**_ のプレビュー機能に対する重要な修正をいくつかリリースしました。最小権限の原則により適合させるために、その機能でよりきめ細かなロールを作成したい場合は、この最新のサーバー リリースをご利用のうえ、更新されたエンタープライズ ライセンスについて W&B チームまでご連絡ください。
