`wandb.init(tags='your_tag')` を使用して一意のタグを設定します。これにより、Project Page の Runs Table で対応するタグを選択して、プロジェクトの run を効率的に絞り込めます。

`wandb.init()` の詳細については、[`wandb.init()` リファレンス](/ja/models/ref/python/functions/init) を参照してください。
