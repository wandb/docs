W&amp;B で `resume` パラメーターを使用するには、`entity`、`project`、`id` を指定して、`wandb.init()` の `resume` 引数を設定します。`resume` 引数には `"must"` または `"allow"` を指定できます。

```python
  run = wandb.init(entity="your-entity", project="your-project", id="your-run-id", resume="must")
```
