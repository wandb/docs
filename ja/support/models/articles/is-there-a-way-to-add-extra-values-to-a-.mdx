---
title: "sweep に値を追加する方法はありますか？それとも新しく作成する必要がありますか？"
keywords: ["Sweeps"]
---

W&amp;B Sweep が開始されると、Sweep の設定は変更できません。ただし、任意の表ビューに移動してチェックボックスで run を選択し、**Create sweep** メニューオプションを選択すると、以前の run に基づいて新しい Sweep の設定を生成できます。
