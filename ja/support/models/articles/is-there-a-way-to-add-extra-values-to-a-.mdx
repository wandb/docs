title: "sweep に値を追加する方法はありますか？それとも新しく開始する必要がありますか？"
keywords: ["Sweeps"]
---

W&amp;B Sweep は、いったん開始すると Sweep の設定を変更できません。ただし、任意の表ビューにアクセスし、チェックボックスを使って run を選択してから、**sweep を作成** メニューオプションを選ぶことで、以前の run に基づいた新しい Sweep の設定を生成できます。
