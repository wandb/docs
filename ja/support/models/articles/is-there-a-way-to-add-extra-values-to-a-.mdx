---
title: "sweep に追加の値を加える方法はありますか、それとも新しく開始する必要がありますか？"
keywords: ["Sweeps"]
---

W&amp;B Sweep は一度開始すると、Sweep の設定を変更できません。ただし、任意の表ビューにアクセスする、チェックボックスを使用して run を選択し、その後 **Create sweep** メニューオプションを選ぶことで、以前の run を 기준으로新しい Sweep 設定を生成できます。
