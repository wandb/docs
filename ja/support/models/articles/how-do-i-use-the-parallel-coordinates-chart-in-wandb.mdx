平行座標チャートは、多数の run にわたるハイパーパラメーターとメトリクスの関係を示します。各線は 1 つの run を表し、各軸は設定キーまたはサマリー メトリクスです。パネルの追加方法や、軸、色、ブラシの設定については、[Parallel coordinates](/ja/models/app/features/panels/parallel-coordinates) および [sweep の結果を可視化する](/ja/models/sweeps/visualize-sweep-results) を参照してください。

<div id="reading-the-chart">
  ## チャートの読み方
</div>

* 隣接する軸の間で線が**交差**している場合、それらのパラメーターの間に逆相関があることを示唆します。
* 線が**平行に走っている**場合、正の相関を示唆します。
* 良好なメトリクス値付近に線の**束**が密集している場合、有望なハイパーパラメーター領域を示します。

サマリー メトリクス (たとえば `val/accuracy`) で線を色分けすると、優れた run が寒色から暖色へのグラデーションの中で際立ちます。

<div id="filtering-with-brushes">
  ## ブラシによるフィルタリング
</div>

軸上をドラッグして、値の範囲をブラシで選択します。範囲内の Runs のみがハイライト表示されたままになります。複数の軸をブラシで選択すると、すべての範囲を満たす Runs に絞り込まれます。これは、どの学習率とバッチサイズの組み合わせが 90% を超える精度を達成するか、といった確認に便利です。

<div id="sweeps">
  ## Sweeps
</div>

sweep の実行後、パネルに試行 run のみが表示されるよう、sweep (**Filter &gt; Sweep &gt; your-sweep-id**) でフィルターします。チャートで探索された空間を確認し、より絞り込んだフォローアップの sweep を計画します。
