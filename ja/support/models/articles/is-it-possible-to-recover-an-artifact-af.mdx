---
title: "run とともに削除されたアーティファクトは復元できますか？"
keywords: ["Artifacts"]
---

run を削除すると、関連付けられたアーティファクトも削除するかどうかを確認するダイアログが表示されます。このオプションを選択すると、アーティファクトは完全に削除されるため、後から run 自体を復元しても復旧できません。
