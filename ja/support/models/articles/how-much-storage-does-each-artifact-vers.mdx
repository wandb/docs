title: "各アーティファクト バージョンはどれくらいのストレージを使用しますか？"
keywords: ["Artifacts", "ストレージ"]
---

2 つのアーティファクト バージョン間で変更されたファイルのみが、ストレージ コストの対象になります。

<Frame>
  <img src="/images/artifacts/artifacts-dedupe.PNG" alt="アーティファクトの重複排除" />
</Frame>

`animals` という名の画像アーティファクトに、`cat.png` と `dog.png` という 2 つの画像ファイルが含まれているとします。

```
images
|-- cat.png (2MB) # `v0` で追加
|-- dog.png (1MB) # `v0` で追加
```

このアーティファクトにはバージョン `v0` が割り当てられます。

新しい画像 `rat.png` を追加すると、次の内容を含む新しいアーティファクト バージョン `v1` が作成されます。

```
images
|-- cat.png (2MB) # `v0` で追加
|-- dog.png (1MB) # `v0` で追加
|-- rat.png (3MB) # `v1` で追加
```

バージョン `v1` は合計 6MB をトラッキングしますが、`v0` と残りの 3MB を共有しているため、占有する容量は 3MB のみです。`v1` を削除すると、`rat.png` に 연결된 3MB のストレージが解放されます。`v0` を削除すると、`cat.png` と `dog.png` のストレージコストが `v1` に移され、そのストレージサイズは 6MB に増加します。
