---
title: "各artifactバージョンはどのくらいのストレージを使用しますか?"
keywords: ["Artifacts", "ストレージ"]
---

2つのartifactバージョンの間で変更されたファイルにのみ、ストレージコストが発生します。

<Frame>
  <img src="/images/artifacts/artifacts-dedupe.PNG" alt="artifactの重複排除" />
</Frame>

`animals` という名前の画像artifactに、`cat.png` と `dog.png` の2つの画像ファイルが含まれているとします。

```
images
|-- cat.png (2MB) # `v0` で追加
|-- dog.png (1MB) # `v0` で追加
```

このartifactにはバージョン`v0`が付与されます。

新しい画像`rat.png`を追加すると、次の内容を含む新しいartifactバージョン`v1`が作成されます。

```
images
|-- cat.png (2MB) # `v0` で追加
|-- dog.png (1MB) # `v0` で追加
|-- rat.png (3MB) # `v1` で追加
```

バージョン `v1` は合計 6MB をトラッキングしていますが、`v0` と残りの 3MB を共有しているため、実際に使用する容量は 3MB のみです。`v1` を削除すると、`rat.png` に対応する 3MB のストレージが解放されます。`v0` を削除すると、`cat.png` と `dog.png` のストレージコストが `v1` に移り、そのストレージサイズは 6MB に増加します。
