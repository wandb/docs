---
title: "run がログしたアーティファクトや使用したアーティファクトをどうすれば見つけられますか？ アーティファクトをログした run や使用した run をどうすれば見つけられますか？"
keywords: ["Artifacts"]
---

W&amp;B は、各 run がログしたアーティファクトと、各 run が使用したアーティファクトを追跡して、アーティファクトグラフを構築します。このグラフは、run とアーティファクトを表すノードで構成される二部有向非巡回グラフです。例は [こちら](https://wandb.ai/shawn/detectron2-11/artifacts/dataset/furniture-small-val/06d5ddd4deeb2a6ebdd5/graph) で表示できます (グラフを展開するには &quot;Explode&quot; をクリックします)。

Public API を使用して、アーティファクトまたは run のいずれかを起点に、プログラムからグラフをたどることができます。

<Tabs>
  <Tab title="アーティファクトから">
    ```python
    api = wandb.Api()

    artifact = api.artifact("project/artifact:alias")

    # アーティファクトからグラフの上流をたどる:
    producer_run = artifact.logged_by()
    # アーティファクトからグラフの下流をたどる:
    consumer_runs = artifact.used_by()

    # run からグラフの下流をたどる:
    next_artifacts = consumer_runs[0].logged_artifacts()
    # run からグラフの上流をたどる:
    previous_artifacts = producer_run.used_artifacts()
    ```
  </Tab>

  <Tab title="run から">
    ```python
    api = wandb.Api()

    run = api.run("entity/project/run_id")

    # run からグラフの下流をたどる:
    produced_artifacts = run.logged_artifacts()
    # run からグラフの上流をたどる:
    consumed_artifacts = run.used_artifacts()

    # アーティファクトからグラフの上流をたどる:
    earlier_run = consumed_artifacts[0].logged_by()
    # アーティファクトからグラフの下流をたどる:
    consumer_runs = produced_artifacts[0].used_by()
    ```
  </Tab>
</Tabs>
