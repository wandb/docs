`Step` 以外を X 軸にしてメトリクスを可視化すると、表示されるデータ ポイントは少なくなります。同期を維持するには、メトリクスを同じ `Step` でログする必要があります。サンプル間を補間する際にサンプリングされるのは、同じ `Step` でログされたメトリクスだけです。

**ガイドライン**

メトリクスは 1 回の `log()` 呼び出しにまとめてください。例えば、次のようにする代わりに:

```python
import wandb
with wandb.init() as run:
    run.log({"Precision": precision})
    ...
    run.log({"Recall": recall})
```

使用する:

```python
import wandb
with wandb.init() as run:
    run.log({"Precision": precision, "Recall": recall})
```

step パラメーターを手動で制御するには、コード内で次のようにメトリクスを同期してください:

```python
with wandb.init() as run:
    step = 100  # step の値の例
    # 同じ step で Precision と Recall をログする
    run.log({"Precision": precision, "Recall": recall}, step=step)
```

メトリクスが同じ `step` にログされ、まとめてサンプリングされるように、両方の `log()` 呼び出しで `step` の値を同じにしてください。`step` の値は呼び出しごとに単調増加している必要があります。そうでない場合、`step` の値は無視されます。
