---
title: "run の状態は UI では `crashed` になっていますが、マシン上ではまだ実行中です。データを復旧するにはどうすればよいですか？"
keywords: ["Experiments", "run の異常終了"]
---

トレーニング中に、マシンとの接続が失われた可能性があります。[`wandb sync [PATH_TO_RUN]`](/ja/models/ref/cli/wandb-sync) を実行して、データを復旧してください。run のパスは、進行中の run の Run ID に一致する `wandb` ディレクトリ内のフォルダーです。
