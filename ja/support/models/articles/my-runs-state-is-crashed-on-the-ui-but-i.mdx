---
title: "UI では run の状態が `crashed` ですが、マシン上ではまだ実行中です。データを復元するにはどうすればよいですか？"
keywords: ["Experiments", "run のクラッシュ"]
---

トレーニング中にマシンとの接続が切れた可能性があります。[`wandb sync [PATH_TO_RUN]`](/ja/models/ref/cli/wandb-sync) を実行してデータを復元してください。run のパスは、`wandb` ディレクトリ内にある、実行中の run の Run ID と一致する名前のフォルダです。
