誰かが `anonymous="allow"` を指定してスクリプトを実行した場合:

1. **一時アカウントを自動作成**: W&amp;B はサインイン済みのアカウントがあるか確認します。存在しない場合、W&amp;B は新しい匿名アカウントを作成し、そのセッション用の APIキーを保存します。
2. **結果をすばやくログする**: ユーザーはスクリプトを繰り返し実行し、W&amp;B ダッシュボードですぐに結果を確認できます。これらの未請求の匿名 run は 7 日間利用できます。
3. **有用なデータをアカウントに引き取る**: ユーザーが W&amp;B で価値のある結果を見つけたら、ページ上部のバナーにあるボタンをクリックして、run データを通常のアカウントに保存できます。引き取らない場合、run データは 7 日後に削除されます。

<Warning>
  **匿名 run のリンクは機密性の高い情報です**。これらのリンクがあれば、誰でも 7 日間、実験結果を閲覧して引き取ることができます。そのため、リンクは信頼できる相手とのみ共有してください。作成者のアイデンティティを隠したまま結果を公開共有したい場合は、support@wandb.com までお問い合わせください。
</Warning>

W&amp;B ユーザーがそのスクリプトを見つけて実行すると、結果は通常の run と同様に、そのユーザーのアカウントへ正しくログされます。
