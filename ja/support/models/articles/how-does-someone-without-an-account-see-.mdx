---
title: "アカウントを持たない人はどのように run の結果を確認できますか？"
keywords: ["匿名"]
---

誰かが `anonymous="allow"` を指定してスクリプトを実行すると、次のようになります。

1. **一時アカウントを自動作成**: W&amp;B はサインイン済みのアカウントがあるかどうかを確認します。ない場合は、新しい匿名アカウントを作成し、そのセッション用の APIキー を保存します。
2. **すばやく結果をログする**: Users はスクリプトを繰り返し実行し、W&amp;B ダッシュボードで結果をすぐに確認できます。これらの未請求の匿名 run は 7 日間利用できます。
3. **有用な結果があればデータを請求する**: ユーザーが W&amp;B で価値のある結果を見つけたら、ページ上部のバナーにあるボタンをクリックして、自分の run データを正式なアカウントに保存できます。請求しない場合、run データは 7 日後に削除されます。

<Warning>
  **匿名 run のリンクは機密性の高い情報です**。これらのリンクを知っている人は誰でも 7 日間、実験結果を確認して請求できるため、リンクは信頼できる相手とのみ共有してください。作成者のアイデンティティを隠したまま結果を一般公開したい場合は、support@wandb.com に連絡してください。
</Warning>

W&amp;B ユーザーがそのスクリプトを見つけて実行すると、結果は通常の run と同様にそのアカウントへ正しくログされます。
