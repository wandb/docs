---
title: "InitStartError: wandb プロセスとの通信中にエラーが発生しました"
keywords: ["Experiments", "Run Crashes"]
---

このエラーは、ライブラリがサーバーにデータを同期するプロセスの起動時に問題が発生したことを示しています。

以下の回避策で、特定の環境におけるこの問題を解決できます：

<Tabs>
  <Tab title="Linux と OS X">
    ```python
    wandb.init(settings=wandb.Settings(start_method="fork"))
    ```
  </Tab>

  <Tab title="Google Colab">
    バージョン `0.13.0` より前では、次を使用してください。

    ```python
    wandb.init(settings=wandb.Settings(start_method="thread"))
    ```
  </Tab>
</Tabs>
