---
title: "トレーニング コードで run 名をどのように設定できますか？"
keywords: ["Experiments"]
featured: true
---

トレーニング スクリプトの先頭で、実験名を指定して `wandb.init` を呼び出します。例: `wandb.init(name="my_awesome_run")`。
