1 つの run には複数のタグを付けられるため、タグでグループ化することはサポートされていません。代わりに、これらの run の [`config`](/ja/models/track/config) オブジェクトに値を追加し、その config 値でグループ化してください。これは [API](/ja/models/track/config#set-the-configuration-after-your-run-has-finished) を使用して行えます。
