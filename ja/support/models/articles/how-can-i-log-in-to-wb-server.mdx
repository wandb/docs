---
title: "W&B Server にログインするにはどうすればよいですか？"
keywords: ["ユーザー管理"]
---

次のいずれかのmethodでログイン URL を設定します。

* [環境変数](/ja/models/track/environment-variables) `WANDB_BASE_URL` を Server URL に設定します。
* [`wandb login`](/ja/models/ref/cli/wandb-login) の `--host` フラグを Server URL に設定します。
