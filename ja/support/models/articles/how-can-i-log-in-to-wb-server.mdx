title: "W&B Server にログインするにはどうすればよいですか？"
keywords: ["ユーザー管理"]
---

次のいずれかの method でログイン URL を設定します。

* [環境変数](/ja/models/track/environment-variables) `WANDB_BASE_URL` を Server URL に設定します。
* [`wandb login`](/ja/models/ref/cli/wandb-login) の `--host` フラグに Server URL を指定します。
