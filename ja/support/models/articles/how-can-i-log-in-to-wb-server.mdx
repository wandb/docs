以下のいずれかの方法でログイン URL を設定します。

* [環境変数](/ja/models/track/environment-variables) `WANDB_BASE_URL` に W&amp;B Server の URL を設定します。
* [`wandb login`](/ja/models/ref/cli/wandb-login) の `--host` フラグに W&amp;B Server の URL を設定します。
