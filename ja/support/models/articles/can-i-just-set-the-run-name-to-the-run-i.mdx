---
title: "run 名を run ID に設定するだけでよいですか？"
keywords: ["Experiments"]
---

はい。run 名を run ID で上書きするには、次のコード スニペットを使用します。

```python
import wandb

with wandb.init() as run:
   run.name = run.id
   run.save()
```
