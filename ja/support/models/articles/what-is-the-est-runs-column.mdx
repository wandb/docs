title: "`Est. Runs` 列とは何ですか？"

---

W&amp;B では、離散的な search space を持つ W&amp;B sweep を作成するときに、生成される Runs の推定数を提供します。この合計は、search space のデカルト積を反映したものです。

たとえば、次の search space を考えてみます。

<Frame>
  <img src="/images/sweeps/sweeps_faq_whatisestruns_1.png" alt="推定 run 数の列" />
</Frame>

この場合、デカルト積は 9 です。W&amp;B はこの値を App UI に推定 run 数 (**Est. Runs**) として表示します。

<Frame>
  <img src="/images/sweeps/spaces_sweeps_faq_whatisestruns_2.webp" alt="sweep の run 数推定" />
</Frame>

推定 Run 数をプログラムで取得するには、W&amp;B SDK 内の Sweep object の `expected_run_count` 属性を使用します。

```python
sweep_id = wandb.sweep(
    sweep_configs, project="your_project_name", entity="your_entity_name"
)
api = wandb.Api()
sweep = api.sweep(f"your_entity_name/your_project_name/sweeps/{sweep_id}")
print(f"EXPECTED RUN COUNT = {sweep.expected_run_count}")
```
