title: "`Est. Runs` 列とは？"

---

W&amp;B では、離散的な探索空間を持つ W&amp;B Sweep を作成する際に、生成される Runs の推定数を表示します。この合計値は、探索空間のデカルト積を反映したものです。

たとえば、次のような探索空間を考えます。

<Frame>
  <img src="/images/sweeps/sweeps_faq_whatisestruns_1.png" alt="推定Runs列" />
</Frame>

この場合、デカルト積は 9 です。W&amp;B は、この値を App UI に推定 run 数 (**Est. Runs**) として表示します。

<Frame>
  <img src="/images/sweeps/spaces_sweeps_faq_whatisestruns_2.webp" alt="Sweep の run 数推定" />
</Frame>

推定 Run 数をプログラムから取得するには、W&amp;B SDK の Sweep オブジェクトにある `expected_run_count` 属性を使用します。

```python
sweep_id = wandb.sweep(
    sweep_configs, project="your_project_name", entity="your_entity_name"
)
api = wandb.Api()
sweep = api.sweep(f"your_entity_name/your_project_name/sweeps/{sweep_id}")
print(f"EXPECTED RUN COUNT = {sweep.expected_run_count}")
```
