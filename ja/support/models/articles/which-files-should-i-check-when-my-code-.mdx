---
title: "コードがクラッシュした場合、どのファイルを確認すればよいですか？"
keywords: ["ログ"]
---

影響を受けた run について、コードを実行しているディレクトリ内の `wandb/run-<date>_<time>-<run-id>/logs` にある `debug.log` と `debug-internal.log` を確認してください。
