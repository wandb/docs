---
title: "run に名前を付けていません。run 名はどこで付けられているのですか？"
keywords: ["Experiments"]
---

run に明示的な名前が付けられていない場合、W&amp;B はプロジェクト内で識別できるよう、ランダムな名前を割り当てます。ランダムな名前の例には、`pleasant-flower-4` や `misunderstood-glade-2. があります。
