---
title: "run に名前を付けていません。run 名はどこから来るのでしょうか？"
keywords: ["Experiments"]
---

run に明示的な名前が付けられていない場合、W&amp;B はプロジェクト内で識別できるようにランダムな名前を割り当てます。ランダムな名前の例としては、`pleasant-flower-4` や `misunderstood-glade-2. があります
