title: "アーティファクトはどこにダウンロードされますか？また、その保存先を制御するにはどうすればよいですか？"
keywords: ["Artifacts", "環境変数"]
---

デフォルトでは、アーティファクトは `artifacts/` フォルダにダウンロードされます。保存先を変更するには:

* root を指定して [`wandb.Artifact().download`](/ja/models/ref/python/public-api/api) を呼び出します:

  ```python
  wandb.Artifact().download(root="<path_to_download>")
  ```

* `WANDB_ARTIFACT_DIR` [環境変数](/ja/models/track/environment-variables) を設定します:

  ```python
  import os
  os.environ["WANDB_ARTIFACT_DIR"] = "<path_to_download>"
  ```
