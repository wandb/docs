---
title: "自分のrunに関連付けられたgitコミットを保存するにはどうすればよいですか？"
keywords: ["Experiments"]
---

`wandb.init` が呼び出されると、システムはリモートrepositoryへのlinkや最新コミットのSHAを含むgit情報を自動的に収集します。この情報は [runページ](/ja/models/runs/#view-logged-runs) に表示されます。この情報を表示するには、スクリプト実行時のカレントワーキングディレクトリが git 管理下のフォルダー内にあることを確認してください。

gitコミットと実験の実行に使用したコマンドはユーザーには表示されますが、外部ユーザーには非表示です。公開プロジェクトでも、これらの詳細は非公開のままです。
