---
title: "自分のrunに関連付けられたgitコミットを保存するにはどうすればよいですか？"
keywords: ["Experiments"]
---

`wandb.init` を呼び出すと、W&amp;B はリモートrepositoryへのlinkや最新コミットのSHAを含むGit情報を自動的に収集します。この情報は [runページ](/ja/models/runs/#view-logged-runs) に表示されます。この情報を表示するには、スクリプト実行時のカレントワーキングディレクトリが Git 管理下のフォルダー内にあることを確認してください。

Gitコミットと実験の実行に使用したコマンドは確認できますが、公開プロジェクトであっても外部ユーザーには非表示です。
