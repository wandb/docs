`wandb.init` が呼び出されると、システムはリモート リポジトリへのリンクや最新コミットの SHA などの Git 情報を自動的に収集します。この情報は [run ページ](/ja/models/runs/#view-logged-runs) に表示されます。この情報を表示するには、スクリプトの実行時の現在の作業ディレクトリが Git 管理下のフォルダー内にあることを確認してください。

Git コミットと実験の実行に使用したコマンドはユーザーには表示されますが、外部ユーザーには表示されません。公開プロジェクトでも、これらの詳細は非公開のままです。
