title: "W&B の情報メッセージを非表示にするにはどうすればよいですか？"
keywords: ["ノートブック", "環境変数"]
---

W&amp;B はデフォルトで INFO レベルのログメッセージを出力するため、ノートブックの出力が煩雑になることがあります。このページでは、それらのメッセージを減らす、または非表示にする 2 つの方法を説明します。1 つは Python の logger level を調整する方法、もう 1 つは W&amp;B の環境変数を使用する方法です。

ノートブックで次のようなログメッセージを抑制するには:

```text
INFO SenderThread:11484 [sender.py:finish():979]
```

ログレベルを `logging.ERROR` に設定すると、エラーのみを表示し、INFO レベルのログ出力を抑制できます。

```python
import logging

logger = logging.getLogger("wandb")
logger.setLevel(logging.ERROR)
```

または、W&amp;B の環境変数を使用してログの詳細度を制御することもできます:

* ログ出力を減らすには、`WANDB_QUIET` 環境変数を `True` に設定します。
* ログ出力を完全に停止するには、`WANDB_SILENT` 環境変数を `True` に設定します。

ノートブックでは、`wandb.login` を実行する前に `WANDB_QUIET` または `WANDB_SILENT` を設定します。

<Tabs>
  <Tab title="ノートブック">
    ```python
    %env WANDB_SILENT=True
    ```
  </Tab>

  <Tab title="Python">
    ```python
    import os

    os.environ["WANDB_SILENT"] = "True"
    ```
  </Tab>
</Tabs>
