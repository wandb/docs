---
title: "W&B の情報メッセージを抑制するにはどうすればよいですか?"
keywords: ["ノートブック", "環境変数"]
---

次のようなログメッセージをノートブックで抑制するには:

```
INFO SenderThread:11484 [sender.py:finish():979]
```

ログレベルを `logging.ERROR` に設定すると、エラーのみを表示し、info レベルのログ出力を抑制します。

```python
import logging

logger = logging.getLogger("wandb")
logger.setLevel(logging.ERROR)
```

ログ出力を大幅に減らすには、`WANDB_QUIET` 環境変数を `True` に設定します。ログ出力を完全に停止するには、`WANDB_SILENT` 環境変数を `True` に設定します。ノートブックでは、`wandb.login` を実行する前に `WANDB_QUIET` または `WANDB_SILENT` を設定します。

<Tabs>
  <Tab title="ノートブック">
    ```python
    %env WANDB_SILENT=True
    ```
  </Tab>

  <Tab title="Python">
    ```python
    import os

    os.environ["WANDB_SILENT"] = "True"
    ```
  </Tab>
</Tabs>
