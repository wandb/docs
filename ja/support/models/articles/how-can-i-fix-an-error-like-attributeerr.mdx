---
title: "`AttributeError: module 'wandb' has no attribute ...` のようなエラーを修正するにはどうすればよいですか？"
keywords: ["run のクラッシュ"]
---

Python で `wandb` をインポートしたときに、`AttributeError: module 'wandb' has no attribute 'init'` または `AttributeError: module 'wandb' has no attribute 'login'` のようなエラーが発生する場合、`wandb` がインストールされていないか、インストールが破損しており、現在の作業ディレクトリに `wandb` ディレクトリが存在している可能性があります。このエラーを修正するには、`wandb` をアンインストールし、ディレクトリを削除してから、`wandb` をインストールしてください。

```bash
pip uninstall wandb; rm -rI wandb; pip install wandb
```
