W&amp;B クライアントライブラリは、バージョン 0.10 までは Python 2.7 と Python 3 の両方をサポートしていました。Python 2 のサポート終了に伴い、Python 2.7 のサポートはバージョン 0.11 で終了しました。Python 2.7 のシステムで `pip install --upgrade wandb` を実行すると、0.10.x 系列の新しいリリースのみがインストールされます。0.10.x 系列のサポートには、重大なバグ修正とパッチのみが含まれます。Python 2.7 をサポートする 0.10.x 系列の最後のバージョンは 0.10.33 です。
