---
title: "W&B クライアントは Python 2 をサポートしていますか？"
keywords: ["Python"]
---

W&amp;B クライアントライブラリは、バージョン 0.10 までは Python 2.7 と Python 3 の両方をサポートしていました。バージョン 0.11 では、Python 2 のサポート終了に伴い、Python 2.7 のサポートが削除されました。Python 2.7 環境で `pip install --upgrade wandb` を実行すると、重大なバグ修正とパッチのみが提供される 0.10.x 系列の新しいリリースだけがインストールされます。Python 2.7 をサポートする 0.10.x 系列の最後のバージョンは 0.10.33 です。
