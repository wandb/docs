---
title: "自分の run にログしたデータに、直接プログラムからアクセスするにはどうすればよいですか？"
keywords: ["Experiments"]
---

history オブジェクトは、`wandb.log` でログしたメトリクスを追跡します。API を使用して history オブジェクトにアクセスできます。

```python
api = wandb.Api()
run = api.run("username/project/run_id")
print(run.history())
```
