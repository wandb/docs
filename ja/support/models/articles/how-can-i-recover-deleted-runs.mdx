---
title: "削除した run を復元するにはどうすればよいですか？"
keywords: ["Runs"]
---

削除された run を復元するには、次の手順を行います。

* Project Overview ページに移動します。
* 画面右上の 3 点メニューをクリックします。
* **最近削除された run を復元** を選択します。

**メモ**:

* 復元できるのは、過去 7 日以内に削除された run のみです。
* 復元できない場合は、W&amp;B API を使用してログを手動でアップロードできます。
