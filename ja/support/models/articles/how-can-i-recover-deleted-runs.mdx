---
title: "削除した run を復元するにはどうすればよいですか？"
keywords: ["Runs"]
description: "W&B App で最近削除した run を、最大 7 日間復元する方法について説明します。"
---

誤って run を削除した場合でも、experiment の履歴を失わないように W&amp;B App から復元できます。削除した run を復元するには:

1. プロジェクトのOverviewページにアクセスします。
2. 画面右上の **action (<Icon icon="ellipsis" iconType="solid" />)** メニューをクリックします。
3. **Undelete recently deleted runs** を選択します。

復元した run は project に再び表示されます。

<Note>
  * 復元できるのは、過去 7 日以内に削除した run のみです。
  * 復元ができない場合は、W&amp;B API を使用してログを手動でアップロードできます。
</Note>
