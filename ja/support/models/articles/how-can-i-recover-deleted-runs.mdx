---
title: "削除した run を復元するにはどうすればよいですか？"
keywords: ["Runs"]
---

削除した run を復元するには、次の手順に従ってください。

* プロジェクトのOverviewページにアクセスします。
* 画面右上の三点リーダーをクリックします。
* **Undelete recently deleted runs** を選択します。

**メモ**:

* 復元できるのは、過去 7 日以内に削除した run のみです。
* 復元ができない場合は、W&amp;B API を使用してログを手動でアップロードできます。
