---
title: "完了した run に割り当てられたグループは後から変更できますか？"
keywords: ["Runs"]
---

完了した run に割り当てられたグループは、API を使用して変更できます。この機能は Web UI には表示されません。グループを更新するには、次のコードを使用してください。

```python
import wandb

api = wandb.Api()
run = api.run("<ENTITY>/<PROJECT>/<RUN_ID>")
run.group = "NEW-GROUP-NAME"
run.update()
```
