---
title: "人が読める形式の run 名にプログラムからアクセスするにはどうすればよいですか?"
keywords: ["Experiments"]
---

[`wandb.Run`](/ja/models/ref/python/experiments/run) の `.name` 属性には、次のようにアクセスできます。

```python
import wandb

with wandb.init() as run:
   run_name = run.name
   print(f"人間が読めるrun名は: {run_name}")
```
