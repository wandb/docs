---
title: "「Group」機能を使わずに run をグループ化できますか？"
keywords: ["Workspaces", "Runs"]
---

はい、タグやカスタム メタデータを使用して run を分類することもできます。これは、プロジェクトの Workspace ビューと Runs ビューで使用できる `Group` ボタンから行えます。
