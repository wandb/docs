---
title: "グリッドサーチを再実行できますか？"
keywords: ["Sweeps", "ハイパーパラメーター", "Runs"]
---

グリッドサーチが完了していても、クラッシュによって一部の W&amp;B Runs を再実行する必要がある場合は、再実行したい該当の W&amp;B Runs を削除します。次に、[W&amp;B App の Sweep 一覧](/ja/models/sweeps/visualize-sweep-results)で **Resume** ボタンをクリックします。新しい Sweep ID を使用して、新しい W&amp;B Sweep agent を起動します。

すでに完了している W&amp;B Run のパラメーターの組み合わせは、再実行されません。
