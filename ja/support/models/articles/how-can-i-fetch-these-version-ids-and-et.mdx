W&amp;B でアーティファクト参照がログされ、バケットでバージョニングが有効になっている場合、バージョン ID が Amazon S3 UI に表示されます。W&amp;B でこれらのバージョン ID と ETag を取得するには、アーティファクトを取得して、対応するマニフェスト エントリにアクセスします。たとえば、次のとおりです。

```python
artifact = run.use_artifact("my_table:latest")
for entry in artifact.manifest.entries.values():
    versionID = entry.extra.get("versionID")
    etag = entry.extra.get("etag")
```
