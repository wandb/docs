---
title: "選択した run が異なる複数のチャートを表示するには？"
keywords: ["Reports"]
---

W&amp;B Reports では、個別のパネルグリッドを使用することで、同じ report 内に異なる run set のチャートを並べて表示できます。これは、コンテキストを切り替えずに、異なる run の選択内容で結果を比較したい場合に便利です。

これを設定するには、次の手順に従います。

1. 複数のパネルグリッドを作成します。
2. 各パネルグリッドで run set を選択するためにフィルターを適用します。各パネルグリッドはそれぞれ独自の run の選択を保持するため、異なるチャートに異なる run を反映できます。
3. パネルグリッド内でチャートを作成します。

これらの手順を完了すると、report には複数のパネルグリッドが含まれ、それぞれに独自にフィルターされた run set のチャートが表示されます。
