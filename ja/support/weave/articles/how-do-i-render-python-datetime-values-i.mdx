---
title: "UI で Python の datetime 値を表示するにはどうすればよいですか？"
keywords: ["UI 表示"]
---

タイムゾーン情報を含む Python の `datetime.datetime` を使用し、`weave.publish(...)` でオブジェクトを公開します。Weave はこのタイプを認識し、タイムスタンプとして表示します。
