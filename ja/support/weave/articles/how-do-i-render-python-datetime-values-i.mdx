---
title: "UI で Python の datetime 値を表示するにはどうすればよいですか？"
keywords: ["UI レンダリング"]
---

Python の `datetime.datetime` (タイムゾーン情報付き) を使用し、`weave.publish(...)` でオブジェクトを公開します。Weave はこのタイプを認識し、タイムスタンプとして表示します。
