---
title: "UI で Markdown を表示するにはどうすればよいですか?"
keywords: ["UI レンダリング"]
---

保存する前に、文字列を `weave.Markdown(...)` でラップし、`weave.publish(...)` を使用して保存します。Weave はオブジェクトのタイプに基づいて表示方法を決定し、`weave.Markdown` は既知の UI レンダラーにマッピングされます。この値は、UI で書式設定された Markdown オブジェクトとして表示されます。完全なコードサンプルについては、[Call の表示](/ja/weave/guides/tracking/tracing#viewing-calls)を参照してください。
