---
title: "Markdown を UI で表示するにはどうすればよいですか?"
keywords: ["UI レンダリング"]
---

保存前に、文字列を `weave.Markdown(...)` でラップし、`weave.publish(...)` で保存します。Weave はオブジェクトの type に基づいて表示方法を判定し、`weave.Markdown` は既知の UI レンダラーに対応しています。この値は、UI で整形済みの Markdown オブジェクトとして表示されます。完全なコード サンプルについては、[Call の表示](/ja/weave/guides/tracking/tracing#viewing-calls) を参照してください。
