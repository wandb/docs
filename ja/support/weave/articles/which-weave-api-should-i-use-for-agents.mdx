---
title: "エージェントをトレースするには、どの Weave API を使用すべきですか？"
keywords: ["Agents", "Tracing", "weave.op"]
---

何を作成しているかによって異なります。

* 既製のエージェントフレームワーク (Claude Code など) を実行している場合は、対応する [Weave integration](/ja/weave/guides/integrations) をインストールしてください。トレースは自動的に **Agents** タブに表示されます。
* セッションとターンを使って独自のエージェントを作成している場合は、[Agents 向け Weave SDK](/ja/weave/guides/tracking/trace-agents) (`weave.start_session`、`weave.start_turn`) を使用してください。結果は **Agents** タブに表示されます。
* エージェントの会話ではなく、個々の関数やパイプラインをトレースする場合は、[`@weave.op`](/ja/weave/guides/tracking/tracing) を使用してください。結果は **Traces** タブに表示されます。
