---
title: "API エラーコード 403 - 組織で推論ゲートウェイが有効になっていません"
keywords: ["認証とアクセス", "管理者"]
---

「The inference gateway is not enabled for your organization」というメッセージの 403 エラーは、組織で推論ゲートウェイが有効になっていないことを示しています。W&amp;B Inference を使用するには、推論ゲートウェイを有効にする必要があります。

<div id="why-this-happens">
  ## この問題が発生する理由
</div>

W&amp;B Inference を使用するには、組織レベルで推論ゲートウェイを有効にする必要があります。組織でこの機能が有効になっていない場合、すべての API リクエストは 403 エラーで拒否されます。

<div id="what-you-can-do">
  ## 対応方法
</div>

1. **W&amp;B 管理者に連絡する**
   * 組織の W&amp;B 管理者に、推論ゲートウェイを有効にしてもらうよう依頼してください

2. **W&amp;B サポートに問い合わせる**
   * 管理者が誰かわからない場合や、ゲートウェイの有効化でサポートが必要な場合は、[W&amp;B サポート](mailto:support@wandb.com) までお問い合わせください
