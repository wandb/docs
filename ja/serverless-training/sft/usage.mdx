[OpenPipe の ART フレームワーク](https://art.openpipe.ai/getting-started/about)と Serverless Training API を使用して、Serverless の教師ありファインチューニング (SFT) を利用できます。

Serverless SFT の利用を開始するには、まず W&amp;B ツールの[前提条件](/ja/serverless-training/prerequisites)を満たし、その後 [ART Serverless SFT ドキュメント](https://art.openpipe.ai/fundamentals/sft-training)を参照してください。

Serverless SFT の API エンドポイントについては、[Serverless Training API リファレンス](/ja/serverless-training/api-reference)を参照してください。
