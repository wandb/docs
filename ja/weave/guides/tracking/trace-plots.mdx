W&B Weave の _trace plots_ ツールを使うと、インタラクティブなチャートでトレースレベルのメトリクス（レイテンシ、コスト、トークンなど）を探索・可視化・デバッグできます。標準のトレースプロットを使うことも、カスタムトレースプロットビルダーで独自のプロットを作成することもできます。

<Frame>
![Trace plots in action](/weave/guides/tracking/imgs/plots-example.png)
</Frame>

<div id="get-started">
  ## はじめに
</div>

1. プロジェクトの **Traces** ページに移動します。
2. （任意）**Filter** を選択して、日時やオペレーションなどでトレースをフィルタリングします。
3. **Traces** ビューの右上隅にある **Show Metrics** アイコンをクリックして、トレースプロットのサイドペインを開きます。 

    ![The  **Show Metrics** icon](/weave/guides/tracking/imgs/plots-show-metrics-icon.png)

    ここからは、次の操作ができます:
   - [デフォルトのトレースプロット](#default-trace-plots)を表示する。
   - [カスタムトレースプロット](#create-a-custom-trace-plot)を作成する。
4. チャートは、トレースのフィルターと選択内容に応じて動的に更新されます。

<div id="default-trace-plots">
  ## デフォルトのトレースプロット
</div>

トレースプロットパネルを最初に開くと、利用可能なプロジェクトのトレースデータに基づいて、Weave がいくつかのトレースプロットを自動生成します:

- 棒グラフ（コストまたはレイテンシを時間ビンごとに集計）
- 折れ線グラフ（時間に対するレイテンシ）
- 散布図（例: プロンプトトークン数 vs. 完了トークン数）

各トレースプロットはインタラクティブです:

- ホバーしてツールチップを表示
- ドラッグしてズーム
- ダブルクリックしてリセット
- 散布図内のポイントをクリックして特定のトレースを開く

<div id="create-a-custom-trace-plot">
  ## カスタムトレースプロットを作成する
</div>

カスタムトレースプロットを作成することもできます。カスタムトレースプロットを作成するには、次の手順に従います。

1. トレースプロットのサイドペインで **➕ Add Chart** をクリックします。

    ![**➕ Add Chart** button.](/weave/guides/tracking/imgs/plots-add-chart.png)

2. ポップアップウィンドウで、利用可能なトレースプロットタイプのいずれかを選択します。

    ![Available custom trace plot types](/weave/guides/tracking/imgs/plots-custom-chart-types.png)
    
3. 選択したトレースプロットタイプに応じて、トレースプロットを設定します。トレースプロットタイプごとの設定オプションについては、[Trace plot settings by plot type](#trace-plot-settings-by-plot-type) を参照してください。
4. **Save chart** をクリックしてチャートを保存します。

<div id="trace-plot-settings-by-plot-type">
  ### プロットタイプ別のトレースプロット設定
</div>

カスタムトレースプロットを追加または編集する際、利用可能な設定オプションは、選択したトレースプロットタイプによって多少異なります。以下の表では、設定可能なオプションの内訳を示します。

| 設定            | Scatter Plot         | Line Chart           | Bar Chart            |
| --------------- | -------------------- | -------------------- | -------------------- |
| Y-axis      | ✅ 必須               | ✅ 必須               | ✅ 必須               |
| X-axis      | ✅ 選択可能           | 固定 (`started at`)  | 固定 (`started at`)  |
| Grouping    | ✅ 任意               | ✅ 任意               | ✅ 任意               |
| Binning     | ❌ 使用しない         | ✅ 使用する           | ✅ 使用する           |
| Aggregation | ❌ 使用しない         | ✅ 使用する           | ✅ 使用する           |
