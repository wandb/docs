W&B Weave の _trace plots_ ツールを使用すると、インタラクティブなチャートでトレースレベルのメトリクス（例: レイテンシ、コスト、トークン）を探索・可視化・デバッグできます。デフォルトのトレースプロットを使用することも、カスタム トレースプロット ビルダーを使って独自のプロットを作成することもできます。

<Frame>
![動作中のトレースプロット](/weave/guides/tracking/imgs/plots-example.png)
</Frame>

<div id="get-started">
  ## はじめに
</div>

1. プロジェクトの **Traces** ページに移動します。
2. （任意）**Filter** を選択して、対象の trace をフィルタリングします（例：日時や operation で絞り込みます）。
3. **Traces** ビュー右上の **Show Metrics** アイコンをクリックして、trace プロットのサイド ペインを開きます。 

    ![**Show Metrics** アイコン](/weave/guides/tracking/imgs/plots-show-metrics-icon.png)

    ここからは、次のことができます。
   - [デフォルトの trace プロット](#default-trace-plots)を表示する
   - [カスタム trace プロット](#create-a-custom-trace-plot)を作成する
4. チャートは、trace に対するフィルターや選択内容に基づいて動的に更新されます。

<div id="default-trace-plots">
  ## デフォルトの trace プロット
</div>

trace プロット パネルを最初に開くと、Weave は利用可能なプロジェクトの trace データに基づいて、いくつかの trace プロットを自動生成します：

- 棒グラフ（時間ビンごとにグループ化されたコストまたはレイテンシ）
- 折れ線グラフ（時間経過に伴うレイテンシ）
- 散布図（例: プロンプトトークン数と completion トークン数の関係）

各 trace プロットはインタラクティブです：

- カーソルを合わせてツールチップを表示
- ドラッグしてズーム
- ダブルクリックしてリセット
- 散布図上の点をクリックして特定の trace を開く

<div id="create-a-custom-trace-plot">
  ## カスタム trace plot を作成する
</div>

カスタム trace plot も作成できます。カスタム trace plot を作成するには、次の操作を行います。

1. trace plots サイドペインから **➕ Add Chart** をクリックします。

    ![**➕ Add Chart** ボタン。](/weave/guides/tracking/imgs/plots-add-chart.png)

2. ポップアップで、利用可能な trace plot の種類のいずれかを選択します。

    ![利用可能なカスタム trace plot の種類](/weave/guides/tracking/imgs/plots-custom-chart-types.png)
    
3. 選択した trace plot の設定を行います。trace plot タイプごとの設定オプションについては、[プロット タイプ別 trace plot 設定](#trace-plot-settings-by-plot-type) を参照してください。
4. **Save chart** をクリックしてチャートを保存します。

<div id="trace-plot-settings-by-plot-type">
  ### プロットタイプ別のトレースプロット設定
</div>

カスタム トレースプロットを追加または編集する際、利用可能な設定オプションは、選択した トレースプロットのタイプによってわずかに異なります。次の表は、設定可能なオプションの一覧です。

| 設定            | Scatter Plot        | Line Chart           | Bar Chart            |
| --------------- | -------------------- | -------------------- | -------------------- |
| Y-axis      | ✅ 必須              | ✅ 必須              | ✅ 必須              |
| X-axis      | ✅ 選択可            | 固定（`started at`） | 固定（`started at`） |
| Grouping    | ✅ 任意              | ✅ 任意              | ✅ 任意              |
| Binning     | ❌ 使用しない        | ✅ 使用              | ✅ 使用              |
| Aggregation | ❌ 使用しない        | ✅ 使用              | ✅ 使用              |
