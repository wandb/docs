W&amp;B Weave の *トレースプロット* ツールを使用すると、インタラクティブなチャートを使ってトレースレベルのメトリクス (レイテンシー、コスト、トークンなど) を探索、可視化、デバッグできます。デフォルトのトレースプロットを使用することも、custom trace plot builder で独自のプロットを作成することもできます。

<Frame>
  ![動作中のトレースプロット](/weave/guides/tracking/imgs/plots-example.png)
</Frame>

<div id="get-started">
  ## はじめに
</div>

1. プロジェクトの **Traces** ページにアクセスします。
2. (任意) **Filter** を選択して、対象のトレースをフィルタリングします (例：日時や operation で絞り込みます) 。
3. **Traces** ビュー右上の **Show Metrics** アイコンをクリックして、トレース プロットのサイド ペインを開きます。

   ![Show Metrics アイコン](/weave/guides/tracking/imgs/plots-show-metrics-icon.png)

   ここからは、次のことができます。

   * [デフォルトのトレース プロット](#default-trace-plots)を表示する
   * [カスタム トレース プロット](#create-a-custom-trace-plot)を作成する
4. チャートは、トレース に対するフィルターや選択内容に基づいて動的に更新されます。

<div id="default-trace-plots">
  ## デフォルトの トレースプロット
</div>

トレースプロット パネルを最初に開くと、Weave は利用可能なプロジェクトの Trace Data に基づいて、いくつかのトレースプロットを自動生成します：

* 棒チャート (時間ビンごとにグループ化されたコストまたはレイテンシ)
* 折れ線チャート (時間経過に伴うレイテンシ)
* 散布プロット (例: プロンプトトークン数と completion トークン数の関係)

各 トレースプロットはインタラクティブです：

* カーソルを合わせてツールチップを表示
* ドラッグしてズーム
* ダブルクリックしてリセット
* 散布プロット上の点をクリックして特定の trace を開く

<div id="create-a-custom-trace-plot">
  ## カスタム トレースプロット を作成する
</div>

カスタム トレースプロット も作成できます。カスタム トレースプロット を作成するには、次の手順を実行します。

1. トレースプロット side pane で **➕ Add Chart** をクリックします。

   ![➕ Add Chart ボタン。](/weave/guides/tracking/imgs/plots-add-chart.png)

2. ポップアップで、利用可能な トレースプロット タイプのいずれかを選択します。

   ![利用可能なカスタム トレースプロット タイプ](/weave/guides/tracking/imgs/plots-custom-chart-types.png)

3. 選択した トレースプロット タイプについて、トレースプロット を設定します。トレースプロット タイプごとの設定オプションについては、[プロット タイプごとの trace plot 設定](#trace-plot-settings-by-plot-type)を参照してください。

4. **Save chart** をクリックして、チャートを保存します。

<div id="trace-plot-settings-by-plot-type">
  ### プロットタイプ別のトレースプロット設定
</div>

カスタム トレースプロットを追加または編集する際、利用可能な設定オプションは、選択した トレースプロットのタイプによってわずかに異なります。次の表は、設定可能なオプションの一覧です。

| 設定       | 散布プロット | 折れ線チャート        | 棒チャート         |
| -------- | ------------ | ----------------- | ----------------- |
| Y 軸      | ✅ 必須         | ✅ 必須              | ✅ 必須              |
| X-axis   | ✅ 選択可        | 固定 (`started at`) | 固定 (`started at`) |
| Grouping | ✅ 任意         | ✅ 任意              | ✅ 任意              |
| Binning  | ❌ 使用しない      | ✅ 使用              | ✅ 使用              |
| 集約       | ❌ 使用しない      | ✅ 使用              | ✅ 使用              |
