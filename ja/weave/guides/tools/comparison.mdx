---
title: "トレースとその他のログ情報を比較する"
description: "コード、トレース、プロンプト、モデル、設定を視覚的に比較・差分表示する"
---

W&amp;B Weave Comparison機能を使用すると、コード、トレース、プロンプト、モデル、およびモデル設定を視覚的に比較し、差分を確認できます。2 つのオブジェクトを左右に並べて比較したり、より多くのオブジェクトを分析して、違い、パターン、傾向を特定したりできます。

このガイドでは、比較を開始するstepと、ベースライン比較、数値差分の表示形式など、Comparisonビューをニーズに合わせて調整するために使用できるアクションについて説明します。

<div id="access-the-comparison-view">
  ## 뷰にアクセスする
</div>

1. サイドバーで、比較するオブジェクトの種類を選択します (例: **Traces** や **Models**) 。
2. 比較したいオブジェクトを選択します。選択方法は、比較対象のオブジェクトの種類によって異なります。
   * **Traces** の場合は、Trace 列の該当する行のチェックボックスをオンにして、比較するトレースを選択します。
   * **Models** などのオブジェクトの場合は、model Versions ページにアクセスし、比較したいバージョンの横にあるチェックボックスをオンにします。
3. **Compare** を選択して 뷰を開きます。ページ上部の比較バーには、インタラクティブな Call ID トークン (例: `f02b`、`4a98`) が表示され、ドラッグして並べ替えたり削除したりして、比較内容を変更できます。
4. [利用可能なアクション](#available-actions) を使ってビューを絞り込むことができます。

<div id="display-summary">
  ### サマリーの表示
</div>

選択したオブジェクトの比較では、デフォルトビューとしてサマリーが表示されます。このサマリーには、LLM の入力と出力のプレビューに加えて、トークン、コスト、レイテンシが含まれます。

<Frame>
  ![2 つのオブジェクトの Comparison Summary ビュー](/weave/guides/tools/imgs/comparison-2objs-summary.png)
</Frame>

トレースでは、比較表の上部に表示される色付きのバーに、Ops に適用したカスタムの種類と色が反映されます。この拡張された表示機能の詳細については、[Apply kinds and colors](/ja/weave/guides/tracking/ops#apply-kinds-and-colors)を参照してください。

<div id="display-a-calls-view">
  ### Call ビューを表示する
</div>

各オブジェクトのCallを比較するには、**Call** を選択します。このビューでは次のことができます。

* Call の完全なトレース ツリーを表示します。
* 基本的なインメモリ テキスト検索を提供します。
* Call ごとのレイテンシ、コスト、トークン、Op の種類、フィードバックの表示を切り替えられます。

<Frame>
  ![2 つのオブジェクトの Calls Comparison view](/weave/guides/tools/imgs/comparison-2obs-calls.png)
</Frame>

<div id="change-the-diff-display">
  ### 差分表示を変更する
</div>

デフォルトでは、**Diff only** はオフに設定されています。変更された行のみが表示されるように表の行をフィルターするには、**Diff only** をオンに切り替えます。

**Diff only** は side-by-side ビューと Unified ビューに適用され、他のビューでは無効です。

<div id="display-side-by-side">
  ### 横に並べて表示
</div>

各オブジェクトを別々の列で横に並べて比較するには、**Side-by-side** を選択します。

<Frame>
  ![2 つのオブジェクトの横並び比較ビュー](/weave/guides/tools/imgs/comparison-2objs-sidebyside.png)
</Frame>

<div id="display-in-a-unified-view">
  ### 統合ビューで表示する
</div>

2 つのオブジェクトを統合ビューで比較するには、**Unified** を選択します。3 つ以上のオブジェクトを比較する場合、このビューは使用できません。

<Frame>
  ![2 つのオブジェクトの統合比較ビュー](/weave/guides/tools/imgs/comparison-2objs-unified.png)
</Frame>

<div id="set-a-baseline">
  ### ベースラインを設定する
</div>

デフォルトでは、뷰内の各オブジェクトはその左側のオブジェクトと比較されます。ただし、オブジェクトを *ベースライン* として設定することもできます。この場合、すべてのオブジェクトはビューの左端のオブジェクトと比較されます。

オブジェクトをベースラインとして設定するには:

1. 比較バーで、ベースラインとして設定するオブジェクトにポインターを重ねます。
2. ID の右側にある三点リーダー (<Icon icon="ellipsis-vertical" iconType="solid" />) をクリックします。
   ![比較バーで、選択したオブジェクトに対する Make baseline オプションを示す比較ビュー。](/weave/guides/tools/imgs/comparison-2objs-baseline.png)
3. リストで **Make baseline** を選択します。UI が更新され、ベースライン オブジェクトが比較バーの左端に配置され、ID の横に `Baseline` が表示されます。
   ![比較バーの左端の位置に、選択したベースライン オブジェクトを示す比較ビュー。](/weave/guides/tools/imgs/comparison-2objs-baseline-set.png)

<div id="remove-a-baseline">
  ### ベースラインを削除する
</div>

オブジェクトをベースラインから削除するには、次の手順を実行します。

1. 比較バーで、ベースラインのオブジェクトにポインターを合わせます。
2. ID の右側にある三点リーダー (<Icon icon="ellipsis-vertical" iconType="solid" />) をクリックします。
3. リストから **Remove baseline** を選択します。これで、Call ID の横に `Baseline` は表示されなくなります。

<div id="change-the-comparison-order">
  ### 比較順序を変更する
</div>

比較バー内のオブジェクトをドラッグして並び替えます。これにより、比較表の対応する列の順序も同様に変更されます。

比較順序を変更するには:

1. 比較バーで、並び替えたい ID の上にポインターを置きます。
2. ID の左にある 6 つの点をクリックしたまま、必要に応じて左または右にドラッグします。
3. ID を希望する順番になる位置までドラッグして配置します。比較データが更新され、新しい順序で比較が表示されます。

<div id="change-numeric-diff-display-format">
  ### 数値差分の表示形式を変更する
</div>

`completion_tokens` や `total_tokens` のような数値では、差分を整数またはパーセンテージとして表示できます。さらに、正の数値は倍率として表示することもできます。

数値差分の表示形式を変更するには:

1. 比較表で、差分の表示形式を変更したい数値を検索します。
   ![整数として表示されている数値。](/weave/guides/tools/imgs/comparison-2objs-numericdiffformat.png)
2. 差分の値をクリックします。形式は自動的に整数またはパーセンテージのいずれかに切り替わります。
   ![パーセンテージに更新された数値。](/weave/guides/tools/imgs/comparison-2objs-numericdiffformat-updated.png)

<div id="compare-with-baseline-or-previous">
  ### ベースラインまたは前のオブジェクトと比較する
</div>

このオプションは、3 つ以上のオブジェクトを比較している場合にのみ使用できます。また、[ベースラインを設定](#set-a-baseline) したり、[ID の右側にある 3 点アイコンをクリックして既存のベースラインを削除](#remove-a-baseline) したりすることもできます。

3 つ以上のオブジェクトでベースライン比較を行うには、次の手順を実行します：

1. Side-by-side ビューまたは Unified Comparison ビューの右上隅で、リストをクリックします。現在のビュー設定に応じて、リストのタイトルは **Compare with previous** または **Compare with baseline** になります。
2. 現在のビュー設定に応じて、**Compare with previous** または **Compare with baseline** を選択します。
   * **Compare with baseline**: 左端のオブジェクトをベースラインとして設定します。表が更新され、左端の列がベースラインになります。
   * **Compare with previous**: ベースラインとして設定されるオブジェクトはありません。

<div id="compare-a-pair-from-a-multi-object-comparison">
  ### 複数オブジェクト比較からペアを比較する
</div>

このオプションは、3 個以上のオブジェクトを比較している場合にのみ使用できます。

3 個以上のオブジェクトを比較している場合は、1 つのオブジェクトを隣接するオブジェクトまたはベースラインと比較できます。これにより、比較表の表示が 2 オブジェクト比較と同じ表示に切り替わります。

複数オブジェクト比較から 2 つのオブジェクトを比較するには、次の手順を実行します。

1. 比較バーで、比較する ID を選択します。ベースラインを設定している場合、表示では選択したオブジェクトとベースラインが比較されます。ベースラインを設定していない場合は、比較バーで選択したオブジェクトのすぐ左にある ID と比較されます。
2. 項目を選択するには、その ID をクリックします。UI が更新され、2 オブジェクトの比較表が表示されます。
   ![Comparing a pair from a multi-object comparison.](/weave/guides/tools/imgs/comparison-7objs-diffonly-subset.png)

表に最初の 6 個の比較対象オブジェクトを表示するように表示をリセットするには、同じ ID をもう一度クリックします。

<div id="remove-an-object-from-comparison">
  ### 比較からオブジェクトを削除する
</div>

このオプションは、3 つ以上のオブジェクトを比較している場合にのみ利用できます。

比較からオブジェクトを削除するには：

1. 比較バーで、比較から削除したいオブジェクトを検索します。
2. ID の右側にある三点リーダー (<Icon icon="ellipsis-vertical" iconType="solid" />) をクリックします。
3. リストから **Remove object from comparison** を選択します。UI が更新され、削除したオブジェクトを除いた更新された表が表示されます。

<div id="usage-notes">
  ## 使用に関するメモ
</div>

* Comparison 機能は UI でのみ利用できます。
* 比較できるオブジェクトの数に制限はありません。ただし、UI に表示されるのは最大 6 件までです。6 件を超えるオブジェクトを比較した場合に、表示されていないオブジェクトを比較表で確認するには、そのオブジェクトが左から右に並ぶ最初の 6 件に入るように[比較順を変更する](#change-the-comparison-order)か、見やすくするために[複数オブジェクトの比較からペアを比較する](#compare-a-pair-from-a-multi-object-comparison)方法を使用します。
