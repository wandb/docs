---
title: "Serve"
description: "Weave の ops とモデルを API エンドポイントとして公開し、本番環境での予測のサービングと推論の実行を可能にします。"
---

任意の Weave Model への Weave ref があれば、以下を実行できます:

```
weave serve <ref>
```

そのモデル用の FastAPI サーバーを実行するにはこれを使います。[http://0.0.0.0:9996/docs](http://0.0.0.0:9996/docs) にアクセスして、モデルにインタラクティブにクエリできます。


<div id="install-fastapi">
  ## FastAPI のインストール
</div>

```bash
pip install fastapi uvicorn
```


<div id="serve-model">
  ## モデルをサーブする
</div>

ターミナルで次を実行します：

```bash
weave serve <your model ref>
```

モデルのページに移動し、UI からコピーしてモデルの ref を取得します。次のような形式になります:
`weave://your_entity/project-name/YourModel:<hash>`

これを使用するには、Swagger UI のリンクを開き、predict エンドポイントをクリックしてから「Try it out!」をクリックします。
