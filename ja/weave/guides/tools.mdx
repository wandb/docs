Weaveでは、AIアプリケーションのワークフローやデプロイを支援する各種ツールやユーティリティを開発しています。これらは現在まだ初期アルファ段階にあり、今後変更される可能性があります。現在取り組んでいる内容の概要は次のとおりです。

<div id="serve-experimental">
  ## Serve (実験的)
</div>

[Serve](/ja/weave/guides/tools/serve) は、Weave の op やモデルを API endpoint として公開するための機能です。現在、次のような可能性を検討しています。

* Weaveコンポーネント向けの Web サービスを作成する
* Weaveコンポーネントを既存のアプリケーションに統合する
* より本番に近い環境でモデルをテストする手段を提供する

<div id="deploy-experimental">
  ## Deploy (実験的)
</div>

[Deploy](/ja/weave/guides/tools/deploy) は、Weave の op やモデルのデプロイを支援するために開発中の、別のアルファ段階のユーティリティです。現在検討している機能には、次のようなものがあります。

* モデルをクラウドプラットフォームにプッシュする
* さまざまなデプロイ環境を管理する
* デプロイプロセスの一部を自動化する方法を検討する

これらのツールは、まだ開発のごく初期段階にある点にご注意ください。十分に機能しない可能性があり、大幅に変更されたり、提供が終了されたりする場合もあります。現時点では、実験目的に限って使用することをお勧めします。
