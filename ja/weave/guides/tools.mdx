Weave は、AI アプリケーションのワークフローとデプロイ プロセスを支援するための一連のツールやユーティリティを開発しています。これらは現在、アルファ版の初期段階であり、今後変更される可能性があります。以下では、現在取り組んでいる内容の概要を紹介します。

<div id="serve-experimental">
  ## Serve (実験的)
</div>

[Serve](/ja/weave/guides/tools/serve) は、Weave の ops とモデルを API エンドポイントとして公開できる機能です。現在、次のような可能性を検討しています。

* Weave コンポーネント向けの Web サービスを作成する
* 既存のアプリケーションに Weave コンポーネントを統合する
* 本番環境に近い形でモデルをテストする方法を提供する

<div id="deploy-experimental">
  ## Deploy (実験的)
</div>

[Deploy](/ja/weave/guides/tools/deploy) は、Weave ops やモデルのデプロイを支援するために現在開発中の、もう 1 つのアルファ段階のユーティリティです。現在検討している機能には、次のようなものがあります。

* モデルをクラウドプラットフォームにプッシュする
* 複数のデプロイ環境を管理する
* デプロイプロセスの一部を自動化する方法を検討する

これらのツールは、まだ開発のごく初期段階にある点にご注意ください。完全には機能しない場合があり、大幅に変更されたり、提供が中止されたりする可能性もあります。現時点では、実験目的に限って使用することをおすすめします。
