---

Weave では、AI アプリケーションのワークフローおよびデプロイプロセスを支援する一連のツールとユーティリティを開発しています。これらの機能は現在、アルファ版の初期段階にあり、今後変更される可能性があります。ここでは、現在開発中の内容の概要を紹介します。

<div id="serve-experimental">
  ## サーブ (実験的)
</div>

[Serve](/ja/weave/guides/tools/serve) は、Weave の ops とモデルを API エンドポイントとして公開するための機能です。現在、次のような可能性を検討しています:

* Weave コンポーネントの Web サービスを作成する
* 既存のアプリケーションに Weave コンポーネントを統合する
* 本番環境により近い形でモデルをテストする手段を提供する

<div id="deploy-experimental">
  ## Deploy（実験的）
</div>

[Deploy](/ja/weave/guides/tools/deploy) は、Weave の ops やモデルのデプロイを支援するために開発中のアルファ段階のユーティリティです。今後の機能として検討しているものには、次のようなものがあります。

* モデルをクラウドプラットフォームにプッシュする
* 異なるデプロイ環境を管理する
* デプロイプロセスの一部を自動化する方法を検討する

これらのツールは、まだ開発のごく初期段階である点に注意してください。完全には動作しなかったり、大きく変更されたり、提供が中止されたりする可能性があります。現時点では、実験的な目的にのみ使用することを推奨します。
