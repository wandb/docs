Weaveでは、AIアプリケーションの構築、テスト、デプロイを支援する各種ツールやユーティリティを提供しています。このページでは、開発中の実験的なツールを紹介し、それらを試してご自身のワークフローに適しているかどうかを判断できるようにしています。これらは現在まだ初期アルファ段階にあり、今後変更される可能性があります。以下のセクションでは、各ツールについて説明し、専用ガイドへのリンクを掲載しています。

<div id="serve-experimental">
  ## Serve (実験的)
</div>

[Serve](/ja/weave/guides/tools/serve) は、Weave の op やモデルを API endpoint として公開します。用途には次のようなものがあります。

* Weaveコンポーネント向けの Web サービスを作成する
* Weaveコンポーネントを既存のアプリケーションに統合する
* より本番に近い環境でモデルをテストする

<div id="deploy-experimental">
  ## Deploy (実験的機能)
</div>

[Deploy](/ja/weave/guides/tools/deploy) は、Weave の op やモデルのデプロイを支援するアルファ段階のユーティリティです。想定される機能は以下のとおりです。

* クラウドプラットフォームへのモデルの push。
* 複数のデプロイ環境の管理。
* デプロイプロセスの一部の自動化。

<Warning>
  これらのツールは開発の初期段階にあります。完全に機能しない場合や、変更または廃止される可能性があります。実験的な目的にのみ使用してください。
</Warning>
