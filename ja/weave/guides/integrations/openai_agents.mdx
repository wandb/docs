OpenAI Agents SDK と W&amp;B Weave を使用して、エージェント型アプリケーションをトレースおよび監視できます。

<Tabs>
  <Tab title="Python">
    [OpenAI Agents Python SDK](https://github.com/openai/openai-agents-python) は、マルチエージェントのワークフローを構築するための軽量で強力なフレームワークです。

    ## インストール

    `pip` を使用して、必須の依存関係をインストールします。

    ```bash
    pip install weave openai-agents
    ```

    ## はじめに

    OpenAI Agents SDK を Weave で使用するには:

    * プロジェクト名で Weave を初期化します。
    * エージェントに Weave トレースプロセッサを追加します。
    * 通常どおりエージェントを作成して実行します。

    次のコード例では、OpenAI Agent を作成し、トレーサビリティのために Weave と統合します。Weave プロジェクトが初期化され、実行トレースを取得するために `WeaveTracingProcessor` が設定されます。`Weather` データモデルは天気情報を表します。`get_weather` 関数は、エージェントが使用できるツールとしてデコレートされ、サンプルの天気レポートを返します。`Hello world` という名前のエージェントは、基本的な指示と天気ツールへの 액세스 を使用して設定されます。`main` 関数は、サンプル入力 (`What's the weather in Tokyo?`) を使ってエージェントを非同期に実行し、最終応答を出力します。

    ```python lines
    from pydantic import BaseModel
    from agents import Agent, Runner, function_tool
    import agents
    import weave
    import asyncio

    weave.init("openai-agents")

    class Weather(BaseModel):
        city: str
        temperature_range: str
        conditions: str

    @function_tool
    def get_weather(city: str) -> Weather:
        return Weather(city=city, temperature_range="14-20C", conditions="Sunny with wind.")

    agent = Agent(
        name="Hello world",
        instructions="You are a helpful agent.",
        tools=[get_weather]
    )

    async def main():
        result = await Runner.run(agent, input="What's the weather in Tokyo?")    
        print(result.final_output)

    if __name__ == "__main__":
        asyncio.run(main())
    ```

    ## トレースを表示

    上記のコードサンプルを実行すると、Weave ダッシュボードへのリンクが生成されます。リンクを開いて、エージェントの run のトレースを確認します。
  </Tab>

  <Tab title="TypeScript">
    [OpenAI Agents Node SDK](https://github.com/openai/openai-node) は、マルチエージェント ワークフローを構築するための軽量で強力なフレームワークです。

    ## インストール

    `npm` を使用して必要な依存関係をインストールします。

    ```bash
    npm install weave @openai/agents zod
    ```

    ## はじめに

    ほとんどの Node.js 環境では、手動インストルメンテーションは不要です。
    モジュールが読み込まれると、Weave は `@openai/agents` を自動的にインストルメントします。

    自動インストルメンテーションは、モジュールローダーフックを登録することで機能します。

    * **CommonJS プロジェクト:** 追加の設定は不要です。
    * **ESM プロジェクト:** 他のモジュールより前にインストルメンテーションが読み込まれるように、Node を `--import=weave/instrument` フラグ付きで起動する必要があります。

    プロジェクトの設定方法や、使用しているタイプの判別について支援が必要な場合は、[TypeScript SDK: Third-Party Integration Guide](/ja/weave/guides/integrations/js) を参照してください。

    次のコードサンプルでは、OpenAI Agent を作成し、トレーサビリティのために Weave と統合します。Weave プロジェクトの初期化にある `your-team-name/your-project-name` は、自身の entity とプロジェクトに置き換えてください。このサンプルでは、サンプルの天気レポートを返す `get_weather` ツールを定義し、その後、基本的な指示と天気ツールへの 액세스 を持つ `Hello world` という名前のエージェントを設定します。`main` function は、サンプル入力 (`What's the weather in Tokyo?`) でエージェントを実行し、最終的な Response をログします。

    ```typescript lines
    import * as weave from "weave";
    import { Agent, run, tool } from "@openai/agents";
    import { z } from "zod";

    const getWeather = tool({
      name: "get_weather",
      description: "指定した都市の現在の天気を取得します。",
      parameters: z.object({
        city: z.string().describe("都市の名前"),
      }),
      async execute({ city }) {
        return `${city}: 14-20C, Sunny with wind.`;
      },
    });

    async function main() {
      // プロジェクト情報に更新してください。
      await weave.init("your-team-name/your-project-name");

      const agent = new Agent({
        name: "Hello world",
        instructions: "あなたは役立つエージェントです。",
        tools: [getWeather],
      });

      const result = await run(agent, [
        { role: "user", content: "東京の天気はどうですか？" },
      ]);
      console.log(result.finalOutput);
    }

    main();
    ```

    CommonJS では、自動インストルメンテーションを行うため、`import * as weave from "weave"` ステートメントは `import { Agent, run, tool } from "@openai/agents"` より**前**に記述する必要があります。コードでこの順序を守れない場合は、以下の手動インストルメンテーション手法を使用してください。

    ### 手動インストルメンテーション

    手動インストルメンテーションが必要なのは、次のようにモジュールローダーフックを実行できない場合に限られます。

    * 依存関係を単一ファイルにバンドルするバンドラー。
    * Node CLI フラグを渡せない環境。
    * ローダーフックをバイパスする動的モジュール読み込みパターン。

    このような場合は、`instrumentOpenAIAgents()` を使用してインストルメンテーションを明示的に登録できます。

    ```typescript lines
    import * as weave from "weave";

    await weave.init("openai-agents");
    await weave.instrumentOpenAIAgents();
    ```

    カスタムのプロセッサ設定や条件付き登録など、トレースプロセッサを完全に制御するには、手動で作成して登録してください:

    ```typescript lines
    import { addTraceProcessor } from "@openai/agents";
    import { createOpenAIAgentsTracingProcessor } from "weave";

    ...

    const processor = createOpenAIAgentsTracingProcessor();
    addTraceProcessor(processor);
    ```
