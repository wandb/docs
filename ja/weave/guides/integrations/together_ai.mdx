Together AI は、生成 AI モデルを構築およびファインチューニングするためのプラットフォームで、オープンソース LLM に焦点を当て、顧客が独自のモデルをファインチューニングおよびホストできるようにします。

<Info>
  完全な Weave `together` python パッケージのサポートは現在開発中です
</Info>

`together` python パッケージに対する完全な Weave のサポートは現在開発中ですが、Together は OpenAI SDK との互換性 ([docs](https://docs.together.ai/docs/openai-api-compatibility)) をサポートしており、Weave がこれを自動的に検出して統合します。

Together API を使用するように切り替えるには、単に APIキーを [Together API](https://docs.together.ai/docs/quickstart) のキー、`base_url` を `https://api.together.xyz/v1` に、モデルを [チャットモデル](https://docs.together.ai/docs/inference-models#chat-models) の一つに切り替えるだけです。

```python lines {5, 10-13}
import os
import openai
import weave

weave.init('together-weave')

system_content = "You are a travel agent. Be descriptive and helpful."
user_content = "Tell me about San Francisco"

client = openai.OpenAI(
    api_key=os.environ.get("TOGETHER_API_KEY"),
    base_url="https://api.together.xyz/v1",
)
chat_completion = client.chat.completions.create(
    model="mistralai/Mixtral-8x7B-Instruct-v0.1",
    messages=[
        {"role": "system", "content": system_content},
        {"role": "user", "content": user_content},
    ],
    temperature=0.7,
    max_tokens=1024,
)
response = chat_completion.choices[0].message.content
print("Together response:\n", response)
```

これは簡単な例ですが、Weave を独自の関数と統合してより複雑なユースケースに対応する方法の詳細については、[OpenAI](/ja/weave/guides/integrations/openai#track-your-own-ops) ガイドを参照してください。
