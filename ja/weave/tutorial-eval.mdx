評価は、変更を加えた後に一連の例に対してテストすることで、アプリケーションを繰り返し改善するのに役立ちます。Weave は、[`Model`](/ja/weave/guides/core-types/models) と [`Evaluation`](/ja/weave/guides/core-types/evaluations) クラスを使用して、評価のトラッキングを第一級のサポートで提供します。これらの API は、仮定を最小限に抑えて設計されているため、幅広いユースケースに対応する柔軟性を備えています。

  ## このガイドで学ぶこと

このガイドでは、次の内容を説明します。

* `Model` を設定する
* LLM の応答をテストするためのデータセットを作成する

* Weave UI で評価結果を確認する

<div id="build-a-model">
  ## `Model` を構築する
</div>

Weave では、[`Models` はオブジェクトです](/ja/weave/guides/core-types/models)。これらのオブジェクトには、モデル / エージェントの動作 (ロジック、プロンプト、パラメーター) と、バージョン管理されたメタデータ (パラメーター、コード、マイクロ設定) の両方が含まれるため、信頼性高くトラッキング、比較、評価、反復を行えます。

`Model` をインスタンス化すると、Weave はその設定と動作を自動的に取得し、変更があるたびにバージョンを更新します。これにより、反復を重ねながら、時間の経過に伴うパフォーマンスの推移を追跡できます。

`Model` は、`Model` をサブクラス化し、1 つのサンプルを受け取ってレスポンスを返す `predict` 関数を実装して宣言します。

次のサンプルモデルでは、OpenAI を使用して、入力された文からエイリアンフルーツの名前、色、風味を抽出します。

<Tabs>
  <Tab title="Python">
    ```python lines {1,5}
    class ExtractFruitsModel(weave.Model):
        model_name: str
        prompt_template: str

        @weave.op()
        async def predict(self, sentence: str) -> dict:
            client = openai.AsyncClient()

            response = await client.chat.completions.create(
                model=self.model_name,
                messages=[
                    {"role": "user", "content": self.prompt_template.format(sentence=sentence)}
                ],
            )
            result = response.choices[0].message.content
            if result is None:
                raise ValueError("No response from model")
            parsed = json.loads(result)
            return parsed
    ```
  </Tab>

  <Tab title="TypeScript">
    ```typescript lines {9}
    // メモ: weave.Model は TypeScript ではまだサポートされていません。
    // 代わりに、モデルのような関数を weave.op でラップしてください

    import * as weave from 'weave';
    import OpenAI from 'openai';

    const openaiClient = new OpenAI();

    const model = weave.op(async function myModel({datasetRow}) {
      const prompt = `Extract fields ("fruit": <str>, "color": <str>, "flavor") from the following text, as json: ${datasetRow.sentence}`;
      const response = await openaiClient.chat.completions.create({
        model: 'gpt-3.5-turbo',
        messages: [{ role: 'user', content: prompt }],
        response_format: { type: 'json_object' }
      });
      return JSON.parse(response.choices[0].message.content);
    });
    ```
  </Tab>
</Tabs>

`ExtractFruitsModel` クラスは `weave.Model` を継承 (サブクラス化) しているため、Weave はインスタンス化されたオブジェクトをトラッキングできます。`@weave.op` は `predict` 関数をデコレートし、その入力と出力をトラッキングします。

`Model` オブジェクトは次のようにインスタンス化できます。

<Tabs>
  <Tab title="Python">
    ```python lines
    # チーム名とプロジェクト名を設定
    weave.init('<team-name>/eval_pipeline_quickstart')

    model = ExtractFruitsModel(
        model_name='gpt-3.5-turbo-1106',
        prompt_template='Extract fields ("fruit": <str>, "color": <str>, "flavor": <str>) from the following text, as json: {sentence}'
    )

    sentence = "There are many fruits that were found on the recently discovered planet Goocrux. There are neoskizzles that grow there, which are purple and taste like candy."

    print(asyncio.run(model.predict(sentence)))
    # Jupyter Notebook を使用している場合は、次を実行します:
    # await model.predict(sentence)
    ```
  </Tab>

  <Tab title="TypeScript">
    ```typescript
    await weave.init('eval_pipeline_quickstart');

    const sentence = "There are many fruits that were found on the recently discovered planet Goocrux. There are neoskizzles that grow there, which are purple and taste like candy.";

    const result = await model({ datasetRow: { sentence } });

    console.log(result);
    ```
  </Tab>
</Tabs>

<div id="create-a-dataset">
  ## データセットを作成する
</div>

次に、モデルを評価するためのデータセットが必要です。[`Dataset` は、Weave オブジェクトとして保存されるサンプルのコレクションです](/ja/weave/guides/core-types/datasets)。

次のサンプルデータセットでは、3 つの入力文とそれぞれの正解ラベル (`labels`) を定義し、それらをスコアリング関数が読み取れる JSON テーブル形式に整えます。

この例では、コード内でサンプルのリストを作成していますが、実行中のアプリケーションから 1 件ずつログすることもできます。

<Tabs>
  <Tab title="Python">
    ```python lines
    sentences = ["There are many fruits that were found on the recently discovered planet Goocrux. There are neoskizzles that grow there, which are purple and taste like candy.",
    "Pounits are a bright green color and are more savory than sweet.",
    "Finally, there are fruits called glowls, which have a very sour and bitter taste which is acidic and caustic, and a pale orange tinge to them."]
    labels = [
        {'fruit': 'neoskizzles', 'color': 'purple', 'flavor': 'candy'},
        {'fruit': 'pounits', 'color': 'bright green', 'flavor': 'savory'},
        {'fruit': 'glowls', 'color': 'pale orange', 'flavor': 'sour and bitter'}
    ]
    examples = [
        {'id': '0', 'sentence': sentences[0], 'target': labels[0]},
        {'id': '1', 'sentence': sentences[1], 'target': labels[1]},
        {'id': '2', 'sentence': sentences[2], 'target': labels[2]}
    ]
    ```
  </Tab>

  <Tab title="TypeScript">
    ```typescript
    const sentences = [
    ```

<div id="define-custom-scoring-functions">
  ## カスタムスコアリング関数を定義する
</div>

Weave の評価を使用する場合、Weave は `output` と比較するための `target` を期待します。次のスコアリング関数は、2 つの辞書 (`target` と `output`) を受け取り、出力が `target` と一致するかどうかを示すブール値を含む辞書を返します。`@weave.op()` デコレーターにより、Weave はこのスコアリング関数の実行をトラッキングできるようになります。

<Tabs>
  <Tab title="Python">
    ```python lines
    @weave.op()
    def fruit_name_score(target: dict, output: dict) -> dict:
        return {'correct': target['fruit'] == output['fruit']}
    ```
  </Tab>

  <Tab title="TypeScript">
    ```typescript
    import * as weave from 'weave';

    const fruitNameScorer = weave.op(
      function fruitNameScore({target, output}) {
        return { correct: target.fruit === output.fruit };
      }
    );
    ```
  </Tab>
</Tabs>

独自のスコアリング関数を作成するには、[Scorers](/ja/weave/guides/evaluation/scorers) ガイドの詳細をご覧ください。

一部のアプリケーションでは、カスタム `Scorer` クラスを作成したくなる場合があります。例えば、チャットモデルやプロンプトなどの特定の `パラメーター`、行ごとのスコアリング、およびスコアの集計計算を備えた標準化された `LLMJudge` クラスを作成するなどです。`Scorer` クラスの定義方法については、次の章の [RAG アプリケーションのモデルベース評価](/ja/weave/tutorial-rag#optional-defining-a-scorer-class) に関するチュートリアルを参照してください。

<div id="use-a-built-in-scorer-and-run-the-evaluation">
  ## 組み込み Scorer を使って評価を実行する
</div>

カスタムのスコアリング関数に加えて、[Weave の組み込み Scorers](/ja/weave/guides/evaluation/builtin_scorers) も使用できます。次の評価では、`weave.Evaluation()` が前のセクションで定義した `fruit_name_score` 関数と、[F1 スコア](https://en.wikipedia.org/wiki/F-score) を計算する組み込みの `MultiTaskBinaryClassificationF1` Scorer を使用します。

次の例では、`ExtractFruitsModel` を `fruits` データセット上で、これら 2 種類のスコアリング関数で評価を行い、その結果を Weave にログします。

<Tabs>
  <Tab title="Python">
    ```python lines {3-10}
    weave.init('eval_pipeline_quickstart')

    evaluation = weave.Evaluation(
        name='fruit_eval',
        dataset=dataset, 
        scorers=[
            MultiTaskBinaryClassificationF1(class_names=["fruit", "color", "flavor"]), 
            fruit_name_score
        ],
    )
    print(asyncio.run(evaluation.evaluate(model)))
    # Jupyter Notebook の場合は次を実行します:
    # await evaluation.evaluate(model)
    ```
  </Tab>

  <Tab title="TypeScript">
    ```typescript lines {5-9}
    import * as weave from 'weave';

    await weave.init('eval_pipeline_quickstart');

    const evaluation = new weave.Evaluation({
      name: 'fruit_eval',
      dataset: dataset,
      scorers: [fruitNameScorer],
    });
    const results = await evaluation.evaluate(model);
    console.log(results);
    ```
  </Tab>
</Tabs>

<Note>
  Python スクリプトから実行している場合は、`asyncio.run` を使う必要があります。一方、Jupyter Notebook から実行している場合は、`await` を直接使用できます。
</Note>

<div id="complete-example">
  ### 完全なサンプル
</div>

<Accordion title="1 つのスクリプトで完結する評価パイプライン:">
  <Tabs>
    <Tab title="Python">
      ```python lines
      import json
      import asyncio
      import openai
      import weave
      from weave.scorers import MultiTaskBinaryClassificationF1

      # Weave を一度初期化する
      weave.init('eval_pipeline_quickstart')

      # 1. モデルを定義する
      class ExtractFruitsModel(weave.Model):
          model_name: str
          prompt_template: str

          @weave.op()
          async def predict(self, sentence: str) -> dict:
              client = openai.AsyncClient()
              response = await client.chat.completions.create(
                  model=self.model_name,
                  messages=[{"role": "user", "content": self.prompt_template.format(sentence=sentence)}],
              )
              result = response.choices[0].message.content
              if result is None:
                  raise ValueError("No response from model")
              return json.loads(result)

      # 2. モデルをインスタンス化する
      model = ExtractFruitsModel(
          model_name='gpt-3.5-turbo-1106',
          prompt_template='Extract fields ("fruit": <str>, "color": <str>, "flavor": <str>) from the following text, as json: {sentence}'
      )

      # 3. データセットを作成する
      sentences = ["There are many fruits that were found on the recently discovered planet Goocrux. There are neoskizzles that grow there, which are purple and taste like candy.",
      "Pounits are a bright green color and are more savory than sweet.",
      "Finally, there are fruits called glowls, which have a very sour and bitter taste which is acidic and caustic, and a pale orange tinge to them."]
      labels = [
          {'fruit': 'neoskizzles', 'color': 'purple', 'flavor': 'candy'},
          {'fruit': 'pounits', 'color': 'bright green', 'flavor': 'savory'},
          {'fruit': 'glowls', 'color': 'pale orange', 'flavor': 'sour and bitter'}
      ]
      examples = [
          {'id': '0', 'sentence': sentences[0], 'target': labels[0]},
          {'id': '1', 'sentence': sentences[1], 'target': labels[1]},
          {'id': '2', 'sentence': sentences[2], 'target': labels[2]}
      ]

      dataset = weave.Dataset(name='fruits', rows=examples)
      weave.publish(dataset)

      # 4. スコアリング関数を定義する
      @weave.op()
      def fruit_name_score(target: dict, output: dict) -> dict:
          return {'correct': target['fruit'] == output['fruit']}

      # 5. 評価を実行する
      evaluation = weave.Evaluation(
          name='fruit_eval',
          dataset=dataset,
          scorers=[
              MultiTaskBinaryClassificationF1(class_names=["fruit", "color", "flavor"]),
              fruit_name_score
          ],
      )
      print(asyncio.run(evaluation.evaluate(model)))
      ```
    </Tab>

    <Tab title="TypeScript">
      ```typescript lines
      import * as weave from 'weave';
      import OpenAI from 'openai';

      // Weave を一度初期化する
      await weave.init('eval_pipeline_quickstart');

      // 1. モデルを定義する

          messages: [{ role: 'user', content: prompt }],
          response_format: { type: 'json_object' }
        });
        return JSON.parse(response.choices[0].message.content);
      });

      // 2. データセットを作成する
      const sentences = [
        "There are many fruits that were found on the recently discovered planet Goocrux. There are neoskizzles that grow there, which are purple and taste like candy.",

        { fruit: 'neoskizzles', color: 'purple', flavor: 'candy' },
        { fruit: 'pounits', color: 'bright green', flavor: 'savory' },
        { fruit: 'glowls', color: 'pale orange', flavor: 'sour and bitter' }
      ];
      const examples = sentences.map((sentence, i) => ({
        id: i.toString(),
        sentence,
        target: labels[i]

      const dataset = new weave.Dataset({

        rows: examples

      // 4. 評価を実行する
      const evaluation = new weave.Evaluation({
        name: 'fruit_eval',
        dataset: dataset,
        scorers: [fruitNameScorer],
      });
      const results = await evaluation.evaluate(model);
      console.log(results);
      ```
