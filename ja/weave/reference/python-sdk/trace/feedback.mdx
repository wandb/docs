### <kbd>method</kbd> `add`

```python
add(
    feedback_type: 'str',
    payload: 'dict[str, Any] | None' = None,
    creator: 'str | None' = None,
    annotation_ref: 'str | None' = None,
    **kwargs: 'dict[str, Any]'
) → str
```

ref にフィードバックを追加します。

feedback_type: フィードバックの種類を識別する文字列。"wandb." プレフィックスは予約済みです。creator: フィードバックの作成者として表示する名前。
