<div id="base-url">
  ## ベース URL
</div>

```
https://trace.wandb.ai
```


<div id="authentication">
  ## 認証
</div>

リクエストヘッダーに W&amp;B の APIキーを含めてください。

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" https://trace.wandb.ai/...
```


<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="calls">
  ### コール
</div>

- **[POST /call/end](https://docs.wandb.ai/weave/reference/service-api/calls/call-end)** - 呼び出し終了
- **[POST /call/read](https://docs.wandb.ai/weave/reference/service-api/calls/call-read)** - 呼び出し読み取り
- **[POST /call/start](https://docs.wandb.ai/weave/reference/service-api/calls/call-start)** - 呼び出し開始
- **[POST /call/update](https://docs.wandb.ai/weave/reference/service-api/calls/call-update)** - 呼び出し更新
- **[POST /call/upsert_batch](https://docs.wandb.ai/weave/reference/service-api/calls/call-start-batch)** - 呼び出し一括開始
- **[POST /calls/delete](https://docs.wandb.ai/weave/reference/service-api/calls/calls-delete)** - 呼び出し削除
- **[POST /calls/query_stats](https://docs.wandb.ai/weave/reference/service-api/calls/calls-query-stats)** - 呼び出し統計のクエリ
- **[POST /calls/stream_query](https://docs.wandb.ai/weave/reference/service-api/calls/calls-query-stream)** - 呼び出しストリームクエリ

- **[POST /cost/query](https://docs.wandb.ai/weave/reference/service-api/costs/cost-query)** - コストのクエリ
