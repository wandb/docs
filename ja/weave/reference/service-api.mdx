<div id="base-url">
  ## ベース URL
</div>

[W&amp;B Multi-tenant Cloud](/ja/platform/hosting/hosting-options/multi_tenant_cloud) と [W&amp;B 専用クラウド](/ja/platform/hosting/hosting-options/dedicated-cloud) では、次のベース URL を使用します:

```
https://trace.wandb.ai
```

[W&amp;B セルフマネージド インスタンス](/ja/weave/guides/platform/weave-self-managed) では、次を使用します:

```
https://<your-subdomain>.wandb.io/traces
```

<div id="authentication">
  ## 認証
</div>

リクエストヘッダーに W&amp;B APIキーを含めてください:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" https://trace.wandb.ai/...
```
