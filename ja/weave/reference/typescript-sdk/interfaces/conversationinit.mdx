## プロパティ

<div id="agentdescription">
  ### agentDescription
</div>

• `Optional` **agentDescription**: `string`

人が読んで理解できるエージェントの説明です。ターン で独自の `agentDescription` を設定しない限り、`startTurn()` で作成されるすべての `Turn` に、デフォルトの
`agentDescription` として引き継がれます。また、各 ターン の `invoke_agent` スパン に
`gen_ai.agent.description` として出力されます。

<div id="agentversion">
  ### agentVersion
</div>

• `Optional` **agentVersion**: `string`

エージェントのバージョン文字列です。各 `Turn` で独自の値が設定されていない限り、`startTurn()` で作成されるすべての `Turn` にデフォルトの `agentVersion` として引き継がれます。また、各 `Turn` の `invoke_agent` スパン で `gen_ai.agent.version` として出力されます。

  ### 属性

• `Optional` **属性**: `Attributes`

会話で生成されるすべての span に付与されるカスタム属性です。

ここで指定したキーが span 自身の `gen_ai.*` / `weave.*`
属性と衝突する場合はサポートされず、span 側の値が優先されます。
