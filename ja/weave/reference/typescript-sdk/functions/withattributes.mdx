<div id="withattributes">
  # withAttributes
</div>

▸ **withAttributes**&lt;`T`&gt;(`attrs`, `fn`): `Promise`&lt;`T`&gt; | `T`

現在の実行コンテキストに属性を付加すると、`fn` の内部で作成されるあらゆる Call が
自動的にそれらを継承します。属性は trace サーバー上の Call レコードに書き込まれ、
Weave UI やフィルターに表示されるため、Request ID、テナント、Experiments などで
run にタグ付けするのに最適です。

例:

```ts
await withAttributes({requestId: 'abc'}, async () => {
  await myOp();
});
```
