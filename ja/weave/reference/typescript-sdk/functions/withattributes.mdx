<div id="withattributes">
  # withAttributes
</div>

▸ **withAttributes**&lt;`T`&gt;(`attrs`, `fn`): `Promise`&lt;`T`&gt; | `T`

現在の実行コンテキストに属性を関連付け、`fn` 内で作成されるすべての call が
自動的にそれらを継承するようにします。属性は trace server 上の call レコードに
書き込まれ、Weave の UI やフィルタリング機能にも表示されるため、run をリクエスト ID、
テナント、実験などでタグ付けするのに最適です。

例:

```ts
await withAttributes({requestId: 'abc'}, async () => {
  await myOp();
});
```
