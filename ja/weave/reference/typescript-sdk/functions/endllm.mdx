<div id="endllm">
  # endLLM
</div>

▸ **endLLM**(`opts?`): `void`

現在アクティブなLLMを終了します。アクティブなLLMがない場合は何も行いません。

<div id="parameters">
  #### パラメーター
</div>

| 名       | タイプ              |
| :------ | :--------------- |
| `opts?` | `SpanEndOptions` |

<div id="returns">
  #### 戻り値
</div>

`void`

`例`

```ts twoslash
// @noErrors
weave.endLLM();
```

`例`

```ts twoslash
// @noErrors
weave.endLLM({endTime: new Date('2026-05-29T10:00:00.800Z')});
```

`例`

```ts twoslash
// @noErrors
weave.endLLM({error: new Error('llm call failed')});
```
