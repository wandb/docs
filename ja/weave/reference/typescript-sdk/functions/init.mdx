<div id="init">
  # init
</div>

▸ **init**(`project`, `settings?`): `Promise`&lt;[`WeaveClient`](../classes/weaveclient)&gt;

Weave クライアントを初期化します。Weave トレースを有効にするには、この初期化が必要です。

<div id="parameters">
  #### パラメーター
</div>

| 名           | タイプ            | 説明                                                                                                                                                                                                                                                          |
| :---------- | :------------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `project`   | `string`       | W&amp;Bのプロジェクト名 (`project` または `entity/project` を指定できます) 。W&amp;Bチーム (例: `team/project`) を指定しない場合は、デフォルトのentityが使用されます。デフォルトのentityを検索または更新するには、https://docs.wandb.ai/guides/models/app/settings-page/user-settings/#default-team の User Settings を参照してください |
| `settings?` | `SettingsInit` |  (省略可能) Weave トレース設定                                                                                                                                                                                                                                        |

<div id="returns">
  #### 戻り値
</div>

`Promise`&lt;[`WeaveClient`](../classes/weaveclient)&gt;

初期化された Weave クライアントで解決される Promise。

`Throws`

初期化に失敗した場合にスローされます
