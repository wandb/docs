<div id="init">
  # init
</div>

▸ **init**(`project`, `settings?`): `Promise`\<[`WeaveClient`](../classes/weaveclient)\>

Weave クライアントを初期化します。Weave のトレーシングを有効にするには、この初期化が必要です。

<div id="parameters">
  #### パラメーター
</div>

| Name | Type | Description |
| :------ | :------ | :------ |
| `project` | `string` | W&B プロジェクト名（`project` または `entity/project` の形式）。W&B team（例: `team/project`）を指定しない場合は、デフォルトの entity が使用されます。デフォルトの entity を確認または更新するには、https://docs.wandb.ai/guides/models/app/settings-page/user-settings/#default-team の User Settings を参照してください。 |
| `settings?` | `SettingsInit` | （オプション）Weave tracing 設定 |

<div id="returns">
  #### 戻り値
</div>

`Promise`\<[`WeaveClient`](../classes/weaveclient)\>

初期化された Weave クライアントで解決される Promise。

`Throws`

初期化に失敗した場合にスローされます
