<div id="login">
  # login
</div>

▸ **login**(`apiKey`, `host?`): `Promise`&lt;`void`&gt;

指定されたAPIキーを使用して、Weights &amp; Biases (W&amp;B) にログインします。
この関数は、今後の使用に備えて認証情報を`netrc`ファイルに保存しようとしますが、
ファイルシステムに書き込めない場合でも処理は継続されます。

<div id="parameters">
  #### パラメーター
</div>

| 名        | タイプ      | 説明                                                     |
| :------- | :------- | :----------------------------------------------------- |
| `apiKey` | `string` | あなたの W&amp;B APIキー。                                    |
| `host?`  | `string` |  (省略可能) ホスト名 (通常は、カスタム W&amp;B サーバーを使用している場合にのみ必要です) 。 |

<div id="returns">
  #### 戻り値
</div>

`Promise`&lt;`void`&gt;

`例外`

APIキーが指定されていない場合、またはWeave trace serverへの接続を確認できない場合。
