<div id="login">
  # login
</div>

▸ **login**(`apiKey`, `host?`): `Promise`\<`void`\>

指定された API キーを使用して Weights & Biases (W&B) にログインします。
この関数は、今後の利用のために認証情報を netrc ファイルに保存しようとしますが、
ファイルシステムに書き込めない場合でも処理は継続されます。

<div id="parameters">
  #### パラメーター
</div>

| Name | Type | Description |
| :------ | :------ | :------ |
| `apiKey` | `string` | あなたの W&B API キー。 |
| `host?` | `string` | （オプション）ホスト名（通常はカスタム W&B サーバーを使用している場合にのみ必要です）。 |

<div id="returns">
  #### 戻り値
</div>

`Promise`\<`void`\>

`Throws`

API キー が指定されていないか、Weave trace サーバーへの接続を検証できない場合にスローされます。
