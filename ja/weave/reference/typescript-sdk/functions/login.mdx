<div id="login">
  # login
</div>

▸ **login**(`apiKey`, `host?`): `Promise`&lt;`void`&gt;

指定された APIキーを使用して Weights &amp; Biases (W&amp;B) にログインします。
この関数は、今後の利用のために認証情報を netrc ファイルに保存しようとしますが、
ファイルシステムに書き込めない場合でも処理は継続されます。

<div id="parameters">
  #### パラメーター
</div>

| Name     | Type     | Description                                            |
| :------- | :------- | :----------------------------------------------------- |
| `apiKey` | `string` | あなたの W&amp;B APIキー。                                    |
| `host?`  | `string` |  (オプション) ホスト名 (通常はカスタム W&amp;B サーバーを使用している場合にのみ必要です) 。 |

<div id="returns">
  #### 戻り値
</div>

`Promise`&lt;`void`&gt;

`Throws`

APIキーが指定されていないか、Weave trace サーバーへの接続を検証できない場合にスローされます。
