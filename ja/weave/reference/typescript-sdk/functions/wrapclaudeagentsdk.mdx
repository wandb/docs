<div id="wrapclaudeagentsdk">
  # wrapClaudeAgentSdk
</div>

▸ **wrapClaudeAgentSdk**&lt;`T`&gt;(`sdk`): `T`

`@anthropic-ai/claude-agent-sdk` モジュールを手動でインストルメントします。

自動インストルメンテーションを適用できない場合に使用します。たとえば、CJS/ESM の hooks で監視できないモジュール読み込みを行うバンドラを使っている場合や、それらでカバーされない import パスを使用している場合です。`@anthropic-ai/claude-agent-sdk` &gt;= 0.3.178 が必要です。`query` export が trace されるモジュールの view を返します。元の import ではなく、返されたオブジェクトを使用してください (SDK の `query` は getter-only export であるため、元の binding をその場で patch することはできません) :

// @noErrors
