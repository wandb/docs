<div id="wrapopenai">
  # wrapOpenAI
</div>

▸ **wrapOpenAI**\<`T`\>(`openai`): `T`

OpenAI API をラップして、OpenAI 呼び出しの function tracing を有効にします。

<div id="type-parameters">
  #### 型パラメーター
</div>

| 名前 | 型 |
| :------ | :------ |
| `T` | extends `OpenAIAPI` |

<div id="parameters">
  #### パラメーター
</div>

| 名前 | 型 |
| :------ | :------ |
| `openai` | `T` |

<div id="returns">
  #### 戻り値
</div>

`T`

`例`

```ts
const openai = wrapOpenAI(new OpenAI());
const result = await openai.chat.completions.create({
  model: 'gpt-3.5-turbo',
  messages: [{ role: 'user', content: 'Hello, world!' }]
});
```
