<div id="wrapopenai">
  # wrapOpenAI
</div>

▸ **wrapOpenAI**&lt;`T`&gt;(`openai`): `T`

OpenAI の call で関数トレースを有効にするために、OpenAI API をラップします。

<div id="type-parameters">
  #### 型パラメーター
</div>

| 名   | タイプ                 |
| :-- | :------------------ |
| `T` | extends `OpenAIAPI` |

<div id="parameters">
  #### パラメーター
</div>

| 名        | タイプ |
| :------- | :-- |
| `openai` | `T` |

<div id="returns">
  #### 戻り値
</div>

`T`

`例`

```ts twoslash
// @noErrors
const openai = wrapOpenAI(new OpenAI());
const result = await openai.chat.completions.create({
  model: 'gpt-3.5-turbo',
  messages: [{ role: 'user', content: 'Hello, world!' }]
});
```
