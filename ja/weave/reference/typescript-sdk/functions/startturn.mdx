<div id="startturn">
  # startTurn
</div>

▸ **startTurn**(`opts?`): [`Turn`](../classes/turn)

新しいターンを開始します。Conversation がアクティブな場合、ターンはその
`conversationId` を引き継ぎます。アクティブでない場合、ターンには conversation ID がありません。

<div id="parameters">
  #### パラメーター
</div>

| 名      | タイプ                                  |
| :----- | :----------------------------------- |
| `opts` | [`TurnInit`](../interfaces/turninit) |

<div id="returns">
  #### 戻り値
</div>

[`Turn`](../classes/turn)

`例`

```ts twoslash
// @noErrors
weave.startTurn({agentName: 'research-bot'});
```

`例`

```ts twoslash
// @noErrors
weave.startTurn({
  agentName: 'research-bot',
  startTime: new Date('2026-05-29T10:00:00.000Z'),
});
```
