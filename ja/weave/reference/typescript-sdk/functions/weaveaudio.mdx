<div id="weaveaudio">
  # weaveAudio
</div>

▸ **weaveAudio**(`options`): [`WeaveAudio`](../interfaces/weaveaudio)

新しい WeaveAudio オブジェクトを作成します

<div id="parameters">
  #### パラメーター
</div>

| 名         | タイプ               | Description                                                                                                       |
| :-------- | :---------------- | :---------------------------------------------------------------------------------------------------------------- |
| `options` | `WeaveAudioInput` | このメディア タイプのオプション - data: Buffer としての生のオーディオデータ - audioType:  (オプション) オーディオファイルの種類。現在は &#39;wav&#39; のみがサポートされています |

<div id="returns">
  #### 戻り値
</div>

[`WeaveAudio`](../interfaces/weaveaudio)

`例`

```ts
const audioBuffer = fs.readFileSync('path/to/audio.wav');
const weaveAudio = weaveAudio({ data: audioBuffer });
```
