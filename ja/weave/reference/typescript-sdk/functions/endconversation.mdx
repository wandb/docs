<div id="endconversation">
  # endConversation
</div>

▸ **endConversation**(`opts?`): `void`

現在のConversationを終了します。アクティブなConversationがない場合、この操作は何も行いません。

<div id="parameters">
  #### パラメーター
</div>

| 名       | タイプ              |
| :------ | :--------------- |
| `opts?` | `SpanEndOptions` |

<div id="returns">
  #### 戻り値
</div>

`void`

`例`

```ts twoslash
// @noErrors
weave.endConversation();
```

`例`

```ts twoslash
// @noErrors
weave.endConversation({endTime: new Date('2026-05-29T10:00:01.700Z')});
```
