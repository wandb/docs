<div id="weaveimage">
  # weaveImage
</div>

▸ **weaveImage**(`options`): [`WeaveImage`](../interfaces/weaveimage)

WeaveImage オブジェクトを新規作成します

<div id="parameters">
  #### パラメーター
</div>

| 名         | タイプ               | 説明                                                                                                   |
| :-------- | :---------------- | :--------------------------------------------------------------------------------------------------- |
| `options` | `WeaveImageInput` | このメディアタイプのオプションです。- data: Buffer 形式の生の画像データ - imageType:  (任意) 画像ファイルのタイプ。現在サポートされているのは `'png'` のみです |

<div id="returns">
  #### 戻り値
</div>

[`WeaveImage`](../interfaces/weaveimage)

`例`

```ts
const imageBuffer = fs.readFileSync('path/to/image.png');
const weaveImage = weaveImage({ data: imageBuffer });
```
