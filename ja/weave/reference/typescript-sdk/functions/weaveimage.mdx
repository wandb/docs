<div id="weaveimage">
  # weaveImage
</div>

▸ **weaveImage**(`options`): [`WeaveImage`](../interfaces/weaveimage)

新しい WeaveImage オブジェクトを作成します

<div id="parameters">
  #### パラメーター
</div>

| Name | Type | Description |
| :------ | :------ | :------ |
| `options` | `WeaveImageInput` | このメディア タイプ用のオプション - data: Buffer 型の生の画像データ - imageType: （任意）画像ファイルの種類。現在は 'png' のみサポート |

<div id="returns">
  #### 戻り値
</div>

[`WeaveImage`](../interfaces/weaveimage)

`例`

```ts
const imageBuffer = fs.readFileSync('path/to/image.png');
const weaveImage = weaveImage({ data: imageBuffer });
```
