<div id="opdecorator">
  # OpDecorator
</div>

Ƭ **OpDecorator**\<`T`\>: (`value`: `T`, `context`: `ClassMethodDecoratorContext`) => `T` \| `void` & (`target`: `Object`, `propertyKey`: `string` \| `symbol`, `descriptor`: `TypedPropertyDescriptor`\<`T`\>) => `TypedPropertyDescriptor`\<`T`\> \| `void`

デコレーターのためのヘルパー型です。  
これは、従来のデコレーターと Stage 3 デコレーターの両方で使用できるデコレーター関数を表します。

Stage 3 デコレーターの場合:
  target: デコレートされる関数 (T)
  context: MethodDecoratorContext

従来のデコレーターの場合:
  target: プロトタイプ (インスタンス メソッド) またはコンストラクター (静的メソッド)
  propertyKey: メソッド名
  descriptor: メソッドを含むプロパティディスクリプター
