<div id="opdecorator">
  # OpDecorator
</div>

Ƭ **OpDecorator**&lt;`T`&gt;: (`value`: `T`, `context`: `ClassMethodDecoratorContext`) =&gt; `T` | `void` &amp; (`target`: `Object`, `propertyKey`: `string` | `symbol`, `descriptor`: `TypedPropertyDescriptor`&lt;`T`&gt;) =&gt; `TypedPropertyDescriptor`&lt;`T`&gt; | `void`

デコレーターのためのヘルパー型です。
これは、従来のデコレーターと Stage 3 デコレーターの両方で使用できるデコレーター関数を表します。

Stage 3 デコレーターの場合:
target: デコレートされる関数 (T)
context: MethodDecoratorContext

従来のデコレーターの場合:
target: プロトタイプ (インスタンス メソッド) またはコンストラクター (静的メソッド)
propertyKey: メソッド名
descriptor: メソッドを含むプロパティディスクリプター
