[weave](../) / ScoreLogger

ScoreLogger は、1 つの予測に対するスコアの管理を行います。
EvaluationLogger.logPrediction() から返されます。

`例`

```ts
const pred = await ev.logPrediction(example, output);
await pred.logScore("accuracy", 0.95);
await pred.logScore("relevance", 0.8);
await pred.finish(); // 予測を確定する
```
