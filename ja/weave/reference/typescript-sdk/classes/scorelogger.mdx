[weave](../) / ScoreLogger

ScoreLogger は、単一の予測に対するスコアリングを管理します。
EvaluationLogger.logPrediction() から返されます。

`Example`

```ts
const pred = await ev.logPrediction(example, output);
await pred.logScore("accuracy", 0.95);
await pred.logScore("relevance", 0.8);
await pred.finish(); // 予測を確定する
```

  ### finish
</div>

▸ **finish**(): `Promise`&lt;`void`&gt;

予測に対するスコアリング処理を終了します。
蓄積されたスコアを用いて `predict_and_score` 呼び出しを最終確定します。
段階的な集計を更新し、メモリを解放します。

▸ **logScore**(`scorerName`, `score`): `Promise`&lt;`void`&gt;
