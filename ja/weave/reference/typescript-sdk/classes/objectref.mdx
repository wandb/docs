[weave](../) / ObjectRef

保存された Weave オブジェクトへの参照を表します。

通常、エンドユーザーがこのクラスを直接扱う必要はありません。

ObjectRef には、Weave のストレージシステム内で保存されたオブジェクトを一意に識別するための
プロジェクト ID、オブジェクト ID、およびダイジェスト値が含まれます。

`例`

```ts
const ref = new ObjectRef('my-project', 'abc123', 'def456');
const uri = ref.uri(); // weave:///my-project/object/abc123:def456
```

▸ **fromUri**(`uri`): [`ObjectRef`](./objectref)

Weave URI 文字列から ObjectRef を生成します。

<div id="parameters">
  #### パラメーター
</div>

| Name | Type | Description |
| :------ | :------ | :------ |
| `uri` | `string` | weave:///entity/project/object/name:digest の形式の Weave URI |

<div id="returns">
  #### 戻り値
</div>

[`ObjectRef`](./objectref)

新しい ObjectRef インスタンス

`Throws`

URI の形式が無効、またはオブジェクト参照ではない場合にエラーをスローします

`Example`

```ts
const ref = ObjectRef.fromUri('weave:///my-entity/my-project/object/my-dataset:abc123');
```
