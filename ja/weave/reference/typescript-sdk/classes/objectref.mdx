[weave](../) / ObjectRef

保存された Weave オブジェクトへの参照を表します。

通常、エンドユーザーがこのクラスを直接操作する必要はありません。

ObjectRef には、Weave のストレージシステム内で保存されたオブジェクトを一意に識別するための、プロジェクト ID、オブジェクト ID、およびダイジェストが含まれています。

`Example`

```ts
const ref = new ObjectRef('my-project', 'abc123', 'def456');
const uri = ref.uri(); // weave:///my-project/object/abc123:def456
```

### fromUri

▸ **fromUri**(`uri`): [`ObjectRef`](./objectref)

Weave URI 文字列から ObjectRef を作成します。

#### パラメータ

| 名前 | 型 | 説明 |
| :------ | :------ | :------ |
| `uri` | `string` | Weave URI（形式: weave:///entity/project/object/name:digest） |

#### 返り値

[`ObjectRef`](./objectref)

新しい ObjectRef インスタンス

`Throws`

URI の形式が無効な場合、またはオブジェクト参照でない場合にエラーを投げます。

`Example`

```ts
const ref = ObjectRef.fromUri('weave:///my-entity/my-project/object/my-dataset:abc123');
```
