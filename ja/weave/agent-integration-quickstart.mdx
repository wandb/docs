W&amp;B Weave は、一般的な SDK やハーネスで構築されたマルチターンのエージェントを、各ターンを手動でインストルメンテーションすることなくトレースします。エージェントハーネス用のプラグインをインストールするか、サポートされるエージェント SDK を使用するコードで `weave.init()` を呼び出すと、Weave がフレームワークに自動でパッチを適用します。セッション、ターン、LLM Call、ツールコール は、project の **Agents** ビュー に表示されます。

以下のサポートされるインテグレーションのいずれかを選択してください。
