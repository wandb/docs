W&amp;B Reports を使用すると、次のことができます:

* Runs を整理する。
* 可視化を埋め込み、自動化する。
* 得られた知見を記述する。
* LaTeX の zip ファイルまたは PDF としてエクスポートし、共同編集者と最新情報を共有する。

<div id="how-it-works">
  ## 動作概要
</div>

数回クリックするだけで共同作業用のレポートを作成できます。

1. W&amp;B App で、対象の W&amp;B プロジェクトのワークスペースに移動します。
2. ワークスペース右上の **Create report** ボタンをクリックします。

<Frame>
  <img src="/images/reports/create_a_report_button.png" alt="Create report ボタン" max-width="90%" />
</Frame>

3. **Create Report** というタイトルのモーダルウィンドウが表示されます。レポートに追加したいグラフとパネルを選択します（グラフとパネルは後から追加・削除できます）。
4. **Create report** をクリックします。
5. レポートが目的の内容になるまで編集します。
6. **Publish to project** をクリックします。
7. **Share** ボタンをクリックして、レポートを共同作業者と共有します。

W&amp;B Python SDK を使って、対話的およびプログラムによってレポートを作成する方法の詳細は、[Create a report](/ja/models/reports/create-a-report/) ページを参照してください。
