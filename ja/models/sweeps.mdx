W&amp;B Sweeps を使用すると、ハイパーパラメーター探索を自動化し、充実したインタラクティブな実験管理を可視化できます。Bayesian、グリッドサーチ、ランダムなど、人気の探索手法からハイパーパラメーター空間を探索する方法を選択できます。1台または複数のマシンにわたって sweep をスケールアップし、並列実行することも可能です。

<div id="how-it-works">
  ## 仕組み
</div>

[W&amp;B CLI](/ja/models/ref/cli/) の 2 つのコマンドを使用して sweep を作成します：

1. sweep を初期化します。

```bash
  wandb sweep --project <project-name> <path-to-config file>
```

2. sweep エージェントを起動します。

```bash
  wandb agent <sweep-ID>
```

<Note>
  直前のコードスニペットと、このページからリンクされている Colab ノートブックは、W&amp;B CLI を使用して sweep を初期化および作成する方法を示しています。Python SDK を使用して sweep を設定、初期化、実行するには、[Sweeps walkthrough](/ja/models/sweeps/walkthrough/) を参照してください。
</Note>
