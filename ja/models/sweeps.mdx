W&amp;B Sweeps を使ってハイパーパラメータ探索を自動化し、リッチでインタラクティブな形で実験をトラッキング・可視化できます。ベイズ探索、グリッドサーチ、ランダムサーチなどの一般的な探索手法から選択して、ハイパーパラメータ空間を探索できます。1 台または複数台のマシンにまたがってスイープをスケールさせ、並列実行できます。

<div id="how-it-works">
  ## 仕組み
</div>

2 つの [W&amp;B CLI](/ja/models/ref/cli/) コマンドを使ってスイープを作成します：

1. スイープを初期化します。

```bash
  wandb sweep --project <project-name> <path-to-config file>
```

2. スイープエージェントを起動します。

```bash
  wandb agent <sweep-ID>
```

<Note>
  前述のコードスニペットとこのページに掲載されている Colab では、W&amp;B CLI を使ってスイープを初期化して作成する方法を示しています。Python SDK を使ってスイープを設定、初期化し、実行する方法については、[Sweeps walkthrough](/ja/models/sweeps/walkthrough/) を参照してください。
</Note>
