W&amp;B Python SDK を使用して Eval Table を作成するか、既存の [W&amp;B 表](/ja/models/tables) を [ARIA](/ja/aria) を使って変換できます。

<div id="create-an-eval-table">
  ## Eval Table を作成する
</div>

`wandb.EvalTable` クラスを使用して、Eval Table を作成します。

たとえば、次の pandas DataFrame があるとします。

```python
import pandas as pd
import wandb

df = pd.DataFrame(
    [
        {
            "image_id": "img_001",
            "true_label": "cat",
            "predicted_label": "cat",
            "confidence": 0.97,
            "correct": True,
        },
        {
            "image_id": "img_002",
            "true_label": "dog",
            "predicted_label": "cat",
            "confidence": 0.72,
            "correct": False,
        },
        {
            "image_id": "img_003",
            "true_label": "car",
            "predicted_label": "car",
            "confidence": 0.89,
            "correct": True,
        },
    ]
)
```

{/* ```python
  with wandb.init(project="classifier-table-demo") as run:
    table = wandb.Table(dataframe=df)
    run.log({"validation_predictions": table})
  ``` */}

DataFrame を `wandb.EvalTable` に渡して、Eval Table を作成します。各列の役割は、`input_columns`、`output_columns`、`score_columns` の各引数で指定します。

次の例では、以下を使用します。

* `image_id` を入力として使用
* `predicted_label` を出力として使用
* `correct` と `confidence` をスコアとして使用

```python
with wandb.init(project="classifier-eval-table-demo") as run:
    eval_table = wandb.EvalTable(
        dataframe=df,
        input_columns=["image_id"],
        output_columns=["predicted_label"],
        score_columns=["correct", "confidence"],
    )
    run.log({"validation_predictions_eval": eval_table})    
```

<div id="convert-a-wb-table-to-an-eval-table">
  ## W&amp;B 表を Eval Table に変換する
</div>

変換バナーを使用して、既存の W&amp;B 表を Eval Table に変換できます。

1. W&amp;B 表を含む project にアクセスします。
2. 変換バナーの **Try now** ボタンを選択します。すると、変換対象の表の情報が入力された状態で [ARIA](/ja/aria) が開きます。
3. ARIA の会話ウィンドウの案内に従います。

バナーを閉じた場合や表示されない場合は、ARIA を手動で開きます。

1. ページ右上の青い円を選択して ARIA を開きます。
2. ARIA の会話で `/convert-eval-table` と入力します。
3. 会話の右下にある上向き矢印 (<Icon icon="arrow-up" iconType="solid" />) で示された Send を選択します。

W&amp;B 表を変換しても、元の表は変更されません。
