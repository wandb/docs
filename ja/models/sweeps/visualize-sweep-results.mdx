---

W&amp;B Sweeps の結果を W&amp;B App で可視化します。[W&amp;B App](https://wandb.ai/home) にアクセスします。sweep を初期化したときに指定したプロジェクトを選択します。プロジェクトの [Workspace](/ja/models/track/workspaces/) にリダイレクトされます。プロジェクトのサイドバーで **Sweep アイコン** (ほうきアイコン) を選択します。Sweep UI で、リストから対象の Sweep 名を選択します。

sweep リストには、各 sweep の状態 (**State**) 、作成時刻 (**Created**) 、開始したユーザー (**Creator**) 、完了した run 数 (**Run count**) 、および合計の **Compute time** が表示されます。離散的な探索空間に対するグリッドサーチでは、W&amp;B は **Est. Runs** (想定される run 数) も表示します。リストから sweep を開くと、アプリから一時停止、再開、停止、または強制終了できます。CLI で同じ操作を行うには、[Manage sweeps](/ja/models/sweeps/pause-resume-and-cancel-sweeps/) を参照してください。

デフォルトでは、W&amp;B Sweep ジョブを開始すると、W&amp;B により並列座標プロット、パラメーター重要度プロット、および散布図が自動的に作成されます。

並列座標プロットは、多数のハイパーパラメーターとモデルのメトリクスの関係を一目で把握できるように要約して表示します。並列座標プロットの詳細については、[Parallel coordinates](/ja/models/app/features/panels/parallel-coordinates/) を参照してください。

<Frame>
  <img src="/images/sweeps/example_parallel_coordiantes_plot.png" alt="並列座標プロットの例。" />
</Frame>

散布図 (左) は、Sweep 中に生成された W&amp;B Runs を比較します。散布図の詳細については、[Scatter Plots](/ja/models/app/features/panels/scatter-plot/) を参照してください。

パラメーター重要度プロット (右) には、メトリクスの望ましい値を最もよく予測し、それらと高い相関を持つハイパーパラメーターが一覧表示されます。パラメーター重要度プロットの詳細については、[Parameter Importance](/ja/models/app/features/panels/parameter-importance/) を参照してください。

<Frame>
  <img src="/images/sweeps/scatter_and_parameter_importance.png" alt="散布図とパラメーター重要度" />
</Frame>

自動的に使用される従属変数と独立変数 (x 軸と y 軸) は変更できます。各パネルには **Edit panel** と呼ばれる鉛筆アイコンがあります。**Edit panel** を選択します。モーダルが表示されます。モーダル内で、グラフの動作を変更できます。

デフォルトの W&amp;B 可視化オプションの詳細については、[Panels](/ja/models/app/features/panels/) を参照してください。W&amp;B Sweep に含まれていない W&amp;B Runs からプロットを作成する方法については、[Data Visualization docs](/ja/models/tables/) を参照してください。
