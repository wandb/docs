title: エージェントを並列実行する
---

マルチコアまたはマルチ GPU マシン上で W&amp;B Sweep エージェントを並列実行します。作業を始める前に、W&amp;B Sweep を初期化していることを確認してください。W&amp;B Sweep の初期化方法については、[Sweeps を初期化する](./initialize-sweeps) を参照してください。

<div id="parallelize-on-a-multi-cpu-machine">
  ### マルチ CPU マシンでの並列化
</div>

ユースケースに応じて、以下のタブを参照し、CLI や Jupyter Notebook 内で W&amp;B Sweep エージェントを並列化する方法を確認してください。

<Tabs>
  <Tab title="CLI">
    [`wandb agent`](/ja/models/ref/cli/wandb-agent) コマンドを使用して、ターミナルから sweep エージェントを複数の CPU 間で並列実行します。[sweep を初期化](./initialize-sweeps) したときに返される sweep ID を指定します。

    1. ローカル マシン上で複数のターミナル ウィンドウを開きます。
    2. 以下のコードスニペットをコピーして貼り付け、`sweep_id` を自分の sweep ID に置き換えます:

    ```bash
    wandb agent sweep_id
    ```
  </Tab>

  <Tab title="Jupyter Notebook">
    W&amp;B Python SDK ライブラリを使用して、Jupyter Notebook 内で W&amp;B Sweep エージェントを複数の CPU 間で並列実行します。[sweep を初期化](./initialize-sweeps) したときに返される sweep ID を用意しておいてください。加えて、`function` パラメーターには sweep が実行する関数名を指定します:

    1. 複数の Jupyter Notebook を開きます。
    2. 複数の Jupyter Notebook 上で同じ W&amp;B Sweep ID をコピーして貼り付けることで、W&amp;B Sweep を並列化します。たとえば、`sweep_id` という変数に sweep ID が格納されており、関数名が `function_name` の場合は、以下のコードスニペットを複数の Jupyter Notebook に貼り付けることで sweep を並列化できます:

    ```python
    wandb.agent(sweep_id=sweep_id, function=function_name)
    ```
  </Tab>
</Tabs>

<div id="parallelize-on-a-multi-gpu-machine">
  ### マルチ GPU マシンで並列化する
</div>

CUDA Toolkit を利用してターミナルから複数の GPU にまたがって W&amp;B Sweep エージェントを並列化するには、次の手順に従います。

1. ローカルマシンで複数のターミナル ウィンドウを開きます。
2. W&amp;B Sweep ジョブ（[`wandb agent`](/ja/models/ref/cli/wandb-agent)）を開始するときに、`CUDA_VISIBLE_DEVICES` で使用する GPU インスタンスを指定します。使用する GPU インスタンスに対応する整数値を `CUDA_VISIBLE_DEVICES` に割り当てます。

たとえば、ローカルマシンに NVIDIA GPU が 2 つ搭載されているとします。1 つ目のターミナル ウィンドウを開き、`CUDA_VISIBLE_DEVICES` を `0`（`CUDA_VISIBLE_DEVICES=0`）に設定します。次の例の `sweep_ID` は、W&amp;B Sweep を初期化したときに返される W&amp;B Sweep ID に置き換えてください。

Terminal 1

```bash
CUDA_VISIBLE_DEVICES=0 wandb agent sweep_ID
```

2 つ目のターミナル ウィンドウを開きます。`CUDA_VISIBLE_DEVICES` を `1`（`CUDA_VISIBLE_DEVICES=1`）に設定します。次のコードスニペットで示されている `sweep_ID` に、同じ W&amp;B sweep ID を貼り付けます。

Terminal 2

```bash
CUDA_VISIBLE_DEVICES=1 wandb agent sweep_ID
```
