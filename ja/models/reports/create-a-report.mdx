タブを選択して、W&amp;B App でレポートを作成する方法、または Report and Workspace API を使用してプログラムで作成する方法を確認してください。

プログラムでレポートを作成する例については、この [Google Colab](https://colab.research.google.com/github/wandb/examples/blob/master/colabs/intro/Report_API_Quickstart.ipynb) を参照してください。

<Tabs>
  <Tab title="W&B App">
    1. W&amp;B App で project Workspace にアクセスします。

    2. Workspace の右上隅で、**Create report** をクリックします。

       <Frame>
         <img src="/images/reports/create_a_report_button.png" alt="レポート作成ボタン" />
       </Frame>

    3. モーダルが表示されます。最初に追加するチャートを選択します。チャートは後でレポートインターフェイスから追加または削除できます。

       <Frame>
         <img src="/images/reports/create_a_report_modal.png" alt="レポート作成モーダル" />
       </Frame>

    4. 新しい Runs がレポートに追加されないようにするには、**Filter run sets** オプションを選択します。このオプションはオンとオフを切り替えられます。レポートの下書きは自動的に保存されます。**Reports** タブからアクセスできます。
  </Tab>

  <Tab title="Report タブ">
    1. W&amp;B App で project Workspace にアクセスします。
    2. project で **Reports** タブを選択します。
    3. **Create report** をクリックします。

       <Frame>
         <img src="/images/reports/create_report_button.png" alt="レポート作成ボタン" />
       </Frame>
  </Tab>

  <Tab title="Report and Workspace API">
    プログラムでレポートを作成します。

    1. W&amp;B SDK (`wandb`) と Report and Workspace API (`wandb-workspaces`) をインストールします。
       ```bash
       pip install wandb wandb-workspaces
       ```

    2. W&amp;B Python SDK と Report and Workspace API をインポートします。
       ```python
       import wandb
       import wandb_workspaces.reports.v2 as wr
       ```

    3. `wandb_workspaces.reports.v2.Report` でレポートを作成します。Report Class Public API ([`wandb.apis.reports`](/ja/models/ref/python/public-api/api#reports)) を使用してレポートインスタンスを作成します。project 名を指定します。
       ```python
       report = wr.Report(project="report_standard")
       ```

    4. レポートを保存します。`.save()` メソッドを呼び出すまで、Reports は W&amp;B にアップロードされません。
       ```python
       report.save()
       ```

    詳細は、[レポートを編集する](/ja/models/reports/edit-a-report) を参照してください。
  </Tab>
</Tabs>
