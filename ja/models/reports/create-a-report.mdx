<Note>
  W&amp;B Report and Workspace API は Public Preview です。
</Note>

W&amp;B Reports を使用すると、Runs の知見、結果、可視化をコラボレーターと共有できます。project のワークスペース、**Reports** タブ、または Report and Workspace API を使ってプログラムから report を作成できます。開始するには、タブを選択してください。

プログラムによる例については、[Report API クイックスタート Google Colab](https://colab.research.google.com/github/wandb/examples/blob/master/colabs/intro/Report_API_Quickstart.ipynb) を参照してください。

<Tabs>
  <Tab title="W&B App">
    1. W&amp;B App で project のワークスペースにアクセスします。

    2. ワークスペースの右上隅で、**Create report** をクリックします。

       <Frame>
         <img src="/images/reports/create_a_report_button.png" alt="report 作成ボタン" />
       </Frame>

    3. モーダルが表示されます。最初に含めるチャートを選択します。report インターフェースから後でチャートを追加または削除できます。

       <Frame>
         <img src="/images/reports/create_a_report_modal.png" alt="チャート選択オプションを含む report 作成モーダル" />
       </Frame>

    4. report にこれ以上 Runs が追加されないようにするには、**Filter run sets** を選択します。W&amp;B は下書き report を自動的に保存します。**Reports** タブからアクセスできます。
  </Tab>

  <Tab title="Report tab">
    1. W&amp;B App で project のワークスペースにアクセスします。
    2. project で **Reports** タブを選択します。
    3. **Create report** をクリックします。

       <Frame>
         <img src="/images/reports/create_report_button.png" alt="report 作成ボタン" />
       </Frame>
  </Tab>

  <Tab title="Report and Workspace API">
    プログラムで report を作成します。

    1. W&amp;B SDK (`wandb`) と Report and Workspace API (`wandb-workspaces`) をインストールします。
       ```bash
       pip install wandb wandb-workspaces
       ```

    2. W&amp;B Python SDK と Report and Workspace API を import します。
       ```python
       import wandb
       import wandb_workspaces.reports.v2 as wr
       ```

    3. `wandb_workspaces.reports.v2.Report` または Report Class Public API ([`wandb.apis.reports`](/ja/models/ref/python/public-api/api#reports)) を使用して report インスタンスを作成します。report に関連付ける project を指定します。
       ```python
       report = wr.Report(project="report_standard")
       ```

    4. report を保存します。W&amp;B は `.save()` method を呼び出すまで report を upload しません。
       ```python
       report.save()
       ```

    詳細は、[report を編集する](/ja/models/reports/edit-a-report) を参照してください。
  </Tab>
</Tabs>
