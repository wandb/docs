</Note>

このページでは、W&amp;B Report をポータブルなファイル形式にエクスポートする方法について説明します。また、既存の report を複製して、その構造を新しい作業の出発点として再利用する方法についても説明します。

<div id="export-a-report">
  ## report をエクスポート
</div>

W&amp;B の外部で内容を共有したり、分析の静的バージョンをアーカイブしたりするために、report を PDF または LaTeX ファイルとしてエクスポートできます。report 内で **action (<Icon icon="ellipsis-vertical" iconType="solid" />)** メニューを選択します。**Download** を選択し、出力形式として PDF または LaTeX を選びます。

<div id="clone-a-report">
  ## report の複製
</div>

既存のプロジェクトのテンプレートと書式を新しい report のベースとして再利用するには、report を複製します。report は W&amp;B App で複製することも、Report and Workspace API を使用してプログラムから複製することもできます。

<Tabs>
  <Tab title="W&B App">
    report 内で:

    1. **action (<Icon icon="ellipsis-vertical" iconType="solid" />)** メニューを選択し、**Make a copy** を選択します。
    2. モーダルで、複製した report の保存先を選びます。
    3. **Clone report** を選択します。

       保存先のドロップダウンはデフォルトで現在の project に設定されており、自分が所属している各チームの project が一覧表示されます。別の project またはチームに report をコピーするには、ドロップダウンを開いて選択します。

    <Frame>
      <img src="/images/reports/clone_reports.gif" alt="W&B App で report を複製" />
    </Frame>

    report を複製するときは、保存先を指定します。チームに複製した場合は、チームのすべてのメンバーがその report を表示できます。個人のアカウントに複製した場合は、デフォルトでは自分だけが表示できます。
  </Tab>

  <Tab title="Report and Workspace API">
    既存の report を URL から読み込み、新しい report のテンプレートとして再利用するには、Report and Workspace API を使用します。

    URL から report を読み込み、テンプレートとして使用します。`PROJECT` は W&amp;B のプロジェクト名に置き換えてください。

    ```python
    report = wr.Report(
        project=PROJECT, title="Quickstart Report", description="That was easy!"
    )  # 作成
    report.save()  # 保存
    new_report = wr.Report.from_url(report.url)  # 読み込み
    ```

    report を読み込んだら、複製した report をカスタマイズするために `new_report.blocks` の内容を編集してから保存します。`ENTITY` は W&amp;B entity 名に置き換えてください。

    ```python
    pg = wr.PanelGrid(
        runsets=[
            wr.Runset(ENTITY, PROJECT, "First Run Set"),
            wr.Runset(ENTITY, PROJECT, "Elephants Only!", query="elephant"),
        ],
        panels=[
            wr.LinePlot(x="Step", y=["val_acc"], smoothing_factor=0.8),
            wr.BarPlot(metrics=["acc"]),
            wr.MediaBrowser(media_keys="img", num_columns=1),
            wr.RunComparer(diff_only="split", layout={"w": 24, "h": 9}),
        ],
    )
    new_report.blocks = (
        report.blocks[:1] + [wr.H1("Panel Grid Example"), pg] + report.blocks[1:]
    )
    new_report.save()
    ```

    保存すると、新しい report が W&amp;B のプロジェクトに表示されます。
  </Tab>
</Tabs>
