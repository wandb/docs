---
title: 実験を再現する
description: "run から関連するコード、依存関係、設定をダウンロードして、チームメンバーの W&B 実験を再現します。"
---

チームメンバーが作成した実験を再現し、その結果を検証して妥当性を確認します。

実験を再現する前に、次の項目をメモしておく必要があります。

* run がログされたプロジェクトの名
* 再現したい run の名

実験を再現するには、次の手順を実行します。

1. run がログされているプロジェクトにアクセスします。
2. プロジェクトのサイドバーから **Workspace** タブを選択します。
3. 再現したい run をクリックします。run ページが開き、デフォルトで **Overview** タブが表示されます。

続行するには、特定のハッシュに対応する実験のコードをダウンロードするか、実験で使用されたリポジトリ全体をクローンします。

<Tabs>
  <Tab title="Python script または notebook をダウンロード">
    実験の Python script または notebook をダウンロードします。

    1. **Overview** タブ (デフォルトで表示) で、**Command** フィールドにある、実験の作成に使用された script の名をメモします。
    2. run ページで **Code** タブを選択します。
    3. script または notebook に対応するファイルの横にある **Download** をクリックします。
  </Tab>

  <Tab title="GitHub">
    チームメンバーが実験の作成時に使用した GitHub repository をクローンします。手順は次のとおりです。

    1. 必要に応じて、チームメンバーが実験の作成に使用した GitHub repository へのアクセス権を取得します。
    2. GitHub repository の URL が含まれている **Git repository** フィールドをコピーします。
    3. リポジトリをクローンします。
       ```bash
       git clone https://github.com/your-repo.git && cd your-repo
       ```
    4. **Git state** フィールドをターミナルにコピー＆ペーストします。Git state は、チームメンバーが実験の作成に使用した正確なコミットをチェックアウトする一連の Git command です。次の code snippet にある値は、自分の値に置き換えてください。
       ```bash
       git checkout -b "<run-name>" 0123456789012345678901234567890123456789
       ```
  </Tab>
</Tabs>

5. run ページで **Files** タブを選択します。

6. `requirements.txt` ファイルをダウンロードし、作業ディレクトリに保存します。このディレクトリには、クローンした GitHub repository またはダウンロードした Python script もしくは notebook のいずれかが含まれている必要があります。

7.  (推奨) Python の仮想環境を作成します。

8. `requirements.txt` ファイルに記載された依存関係をインストールします。
   ```bash
   pip install -r requirements.txt
   ```

9. コードと依存関係の準備ができたら、script または notebook を実行して実験を再現できます。リポジトリをクローンした場合は、script または notebook があるディレクトリにアクセスする必要があることがあります。それ以外の場合は、作業ディレクトリから script または notebook を実行できます。

<Tabs>
  <Tab title="Python notebook">
    Python notebook をダウンロードした場合は、notebook をダウンロードしたディレクトリにアクセスし、ターミナルで次の command を実行します。

    ```bash
    jupyter notebook
    ```
  </Tab>

  <Tab title="Python script">
    Python script をダウンロードした場合は、script をダウンロードしたディレクトリにアクセスし、ターミナルで次の command を実行します。`<>` で囲まれた値は自分の値に置き換えてください。

    ```bash
    python <your-script-name>.py
    ```
  </Tab>
</Tabs>
