---
title: 実験を再現する
description: "run に関連付けられたコード、依存関係、設定をダウンロードして、チームメンバーの W&B 実験を再現します。"
---

チームメンバーが作成した実験を再現して、その結果を検証および妥当性確認できます。実験を再現するには、チームメンバーが実験の作成に使用した GitHub リポジトリをクローンするか、実験の作成に使用した Python スクリプトまたはノートブックをダウンロードします。ご自身に最適な方法を選択してください。

実験の作成に使用されたコードと、そのコードの実行に必要な依存関係を取得します。これを行うには、以下の手順に従ってください。

<Tabs>
  <Tab title="GitHub リポジトリでコードの状態を再現する">
    以下の手順では、GitHub リポジトリのコード状態を再現する方法を示します。

    1. run がログされている project にアクセスします。
    2. プロジェクトのサイドバーから **Workspace** タブを選択します。
    3. 再現したい run をクリックします。run ページが開き、デフォルトで **Overview** タブが表示されます。
    4. **Command** フィールドの横にある **Reproduce this run** ボタンにマウスカーソルを合わせます。実験を再現するための手順を含むモーダルが表示されます。このモーダルには、コード状態を設定する方法と、実験を実行するためのコマンドが記載されています。

       <img src="/images/experiments/reproduce_run_button.png" alt="run を再現するボタン" width="500" />

    または、次の手順で実験を手動で再現することもできます。

    1. **Git repository** フィールドをターミナルにコピー＆ペーストして、リポジトリをクローンします。次の code snippet に例を示します。`your-repo` はご自身のリポジトリ名に置き換えてください。
       ```bash
       git clone https://github.com/your-repo.git && cd your-repo
       ```
    2. **Git state** フィールドをターミナルにコピー＆ペーストして、チームメンバーが実験の作成に使用した正確なコミットをチェックアウトします。次の code snippet にその例を示します。以下の code snippet で指定されている値は、ご自身の値に置き換えてください。
       ```bash
       git checkout -b "<run-name>" 0123456789012345678901234567890123456789
       ```
  </Tab>

  <Tab title="ソースコードをダウンロードする">
    以下の手順では、実験の作成に使用された Python スクリプトまたはノートブックをダウンロードする方法を示します。

    1. run がログされている project にアクセスします。
    2. プロジェクトのサイドバーから **Workspace** タブを選択します。
    3. 再現したい run をクリックします。run ページが開き、デフォルトで **Overview** タブが表示されます。
    4. **Overview** タブ (デフォルトで表示) で、**Command** フィールドにある実験の作成に使用されたスクリプト名をメモしておきます。
    5. run ページで **Code** タブを選択します。
    6. スクリプトまたはノートブックに対応する file の横にある **Download** をクリックします。
    7. run ページで **Files** タブを選択します。
    8. `requirements.txt` file をダウンロードし、ご自身の作業ディレクトリに保存します。このディレクトリには、クローンした GitHub リポジトリまたはダウンロードした Python スクリプトもしくはノートブックを含めておく必要があります。
    9. `requirements.txt` file に指定されている requirements をインストールします。
       ```bash
       pip install -r requirements.txt
       ```
  </Tab>
</Tabs>
