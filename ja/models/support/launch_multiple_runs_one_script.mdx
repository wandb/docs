---
title: 1つのスクリプトから複数のrunを起動するにはどうすればよいですか？
---

1つのスクリプト内で複数のrunをログするには、新しいrunを開始する前に
前のrunを完了させてください。

推奨される方法は、`wandb.init()` をコンテキストマネージャーとして使うことです。
こうすることで、スクリプトが例外を送出した場合に、そのrunを終了して失敗として
マークできます。

```python
import wandb

for x in range(10):
    with wandb.init() as run:
        for y in range(100):
            run.log({"metric": x + y})
```

`run.finish()` を明示的に呼び出すこともできます。

```python
import wandb

for x in range(10):
    run = wandb.init()

    try:
        for y in range(100):
            run.log({"metric": x + y})

    except Exception:
        run.finish(exit_code=1)
        raise

    finally:
        run.finish()
```

<div id="multiple-active-runs">
  ## 複数のアクティブな run
</div>

wandb 0.19.10 以降では、`reinit` 設定を `"create_new"` にすることで、
同時にアクティブな複数の run を作成できます。

```python
import wandb

with wandb.init(reinit="create_new") as tracking_run:
    for x in range(10):
        with wandb.init(reinit="create_new") as run:
            for y in range(100):
                run.log({"x_plus_y": x + y})

            tracking_run.log({"x": x})
```

`reinit="create_new"` の詳細や W&amp;B インテグレーションに関する注意事項については、
[プロセスごとの複数の run](/ja/models/runs/initialize-run/)
を参照してください。
