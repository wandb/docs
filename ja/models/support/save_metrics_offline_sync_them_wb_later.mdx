---
title: メトリクスをオフラインで保存して、後から W&B に同期することはできますか？
---

デフォルトでは、`wandb.init` はメトリクスをリアルタイムでクラウドに同期するプロセスを開始します。オフラインで利用するには、2つの環境変数を設定してオフラインモードを有効にし、後から同期できるようにします。

次の環境変数を設定します：

1. `WANDB_API_KEY=$KEY`（ここで `$KEY` は [User Settings](https://wandb.ai/settings) で作成した APIキー です）
2. `WANDB_MODE="offline"`。

以下は、スクリプト内での実装例です：

```python
import wandb
import os

os.environ["WANDB_API_KEY"] = "YOUR_KEY_HERE"
os.environ["WANDB_MODE"] = "offline"

config = {
    "dataset": "CIFAR10",
    "machine": "offline cluster",
    "model": "CNN",
    "learning_rate": 0.01,
    "batch_size": 128,
}

with wandb.init(project="offline-demo") as run:
    for i in range(100):
        run.log({"accuracy": i})
```

```shell
wandb sync wandb/dryrun-folder-name
```
