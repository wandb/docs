---
title: コードやデータセットのサンプルなしで、メトリクスだけをログできますか？
---

既定では、W&B はデータセットのサンプルをログしません。既定では、W&B はコードとシステムメトリクスをログします。

環境変数を使ってコードのロギングを無効にする方法が 2 つあります。

1. `WANDB_DISABLE_CODE` を `true` に設定して、すべてのコードの追跡を無効にします。この操作により、git の SHA と diff パッチが取得されなくなります。
2. `WANDB_IGNORE_GLOBS` を `*.patch` に設定して、diff パッチをサーバーと同期しないようにしますが、`wandb restore` で適用できるようローカルには保持されます。

管理者であれば、チームの設定からチーム全体のコード保存を無効にすることもできます。

1. `https://wandb.ai/<team>/settings` にあるチームの設定ページに移動します。ここで `<team>` はあなたのチーム名です。
2. Privacy セクションまでスクロールします。
3. **Enable code saving by default** を切り替えます。
