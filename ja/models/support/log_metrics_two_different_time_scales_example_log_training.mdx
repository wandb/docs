---
title: 異なる2つの時間スケールでメトリクスを記録できますか？
---

たとえば、学習精度をバッチごとに、検証精度をエポックごとに記録したいとします。

はい、`batch` や `epoch` のようなインデックスをメトリクスと一緒にログします。あるステップでは `wandb.Run.log()({'train_accuracy': 0.9, 'batch': 200})` を、別のステップでは `wandb.Run.log()({'val_accuracy': 0.8, 'epoch': 4})` を使用します。UIでは、それぞれのチャートで任意の値を x 軸として設定します。特定のインデックスに対するデフォルトの x 軸を設定するには、[Run.define&#95;metric()](/ja/models/ref/python/experiments/run/#define_metric) を使用します。上記の例では、次のコードを使用します。

```python
import wandb

with wandb.init() as run:
   run.define_metric("batch")
   run.define_metric("epoch")

   run.define_metric("train_accuracy", step_metric="batch")
   run.define_metric("val_accuracy", step_metric="epoch")
```
