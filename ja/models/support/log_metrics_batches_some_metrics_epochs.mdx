---
title: 一部のメトリクスはバッチごとに、別のメトリクスはエポックごとにだけログしたい場合はどうすればよいですか？
---

各バッチで特定のメトリクスをログし、プロットを標準化するには、メトリクスと一緒に目的の x 軸の値もログします。カスタムプロットで「edit」をクリックし、カスタム x 軸を選択します。

```python
import wandb

with wandb.init() as run:
    run.log({"batch": batch_idx, "loss": 0.3})
    run.log({"epoch": epoch, "val_acc": 0.94})
```
