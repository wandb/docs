デフォルトでは、Artifacts は `artifacts/` フォルダにダウンロードされます。場所を変更するには次のようにします。

- [`wandb.Artifact().download`](/ja/models/ref/python/public-api/api) に渡します:

    ```python
    # ダウンロードのためのパスを指定します
    wandb.Artifact().download(root="<path_to_download>")
    ```

- `WANDB_ARTIFACT_DIR` [環境変数](/ja/models/track/environment-variables/) を設定します:

    ```python
    # 環境変数を設定します
    import os
    os.environ["WANDB_ARTIFACT_DIR"] = "<path_to_download>"
    ```
