---
title: 匿名ユーザーが利用できない機能は何ですか？
---

* **データを永続的に保存できない**: 匿名アカウントでは run は 7 日間しか保持されません。匿名 run のデータは、実アカウントに保存（紐付け）することで引き継げます。

<Frame>
<img src="/images/app_ui/anon_mode_no_data.png" alt="匿名モードのインターフェース" />
</Frame>

* **アーティファクトのログ記録はできない**: 匿名 run にアーティファクトをログしようとすると、コマンドラインに次の警告が表示されます:
    ```bash
    wandb: WARNING Artifacts logged anonymously cannot be claimed and expire after 7 days.
    ```

* **プロフィールや設定ページがない**: これらは実アカウントでのみ有用なため、UI にはこれらのページは表示されません。
