W&B（Weights and バイアス）の APIキーを見つけるには、次の手順に従います。

- W&Bにログイン：[https://wandb.ai/authorize](https://wandb.ai/authorize)
- または、プロフィールにアクセス：
  1. 右上のユーザープロフィールをクリック。
  2. 「ユーザー設定」を選択。
  3. 「Danger Zone」セクションまでスクロール。
  4. 「APIキー」の横にある「Reveal」をクリック。
