---
title: なぜ W&B Inference で「国または地域がサポートされていません」と表示されるのですか？
---

「Country, region, or territory not supported」というメッセージを伴う 403 エラーは、サポート対象外の場所から W&B Inference にアクセスしていることを意味します。

<div id="why-this-happens">
  ## この問題が発生する理由
</div>

W&B Inference には、コンプライアンスおよび規制要件に基づく地域制限があります。サービスには、サポート対象の地域からのみアクセスできます。

<div id="what-you-can-do">
  ## 対応方法
</div>

1. **利用規約を確認する**
   - 現在サポートされているロケーションの一覧については、[Terms of Service](https://docs.coreweave.com/docs/policies/terms-of-service/terms-of-use#geographic-restrictions) を参照する

2. **サポート対象の場所から利用する**
   - サポート対象の国または地域にいるときにサービスにアクセスする
   - サポート対象ロケーションにある組織のリソースの利用を検討する

3. **アカウントチームに連絡する**
   - エンタープライズ顧客はアカウントエグゼクティブと利用可能なオプションについて相談できる
   - 一部の組織には特別な取り決めが設けられている場合がある

<div id="error-details">
  ## エラーの詳細
</div>

次のエラーが表示された場合:

```
{
  "error": {
    "code": 403,
    "message": "Country, region, or territory not supported"
  }
}
```

これは、API リクエストを送信した時点の IP アドレスから特定される位置情報に基づいて決定されます。
