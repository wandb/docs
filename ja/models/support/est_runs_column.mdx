---
title: "Est. Runs 列とは何ですか？"
---

W&amp;B では、離散探索空間を持つ W&amp;B Sweep を作成する際に生成される Runs の推定数を提供します。この総数は探索空間のデカルト積を表します。

たとえば、次のような探索空間を考えます。

<Frame>
  <img src="/images/sweeps/sweeps_faq_whatisestruns_1.png" alt="Estimated Runs 列" />
</Frame>

この場合、デカルト積は 9 になります。W&amp;B は、この値を App UI 上で推定 Runs 数（**Est. Runs**）として表示します。

<Frame>
  <img src="/images/sweeps/spaces_sweeps_faq_whatisestruns_2.webp" alt="Sweep の Runs 推定" />
</Frame>

推定された Runs 数をプログラムから取得するには、W&amp;B SDK 内の Sweep オブジェクトの `expected_run_count` 属性を使用します。

```python
sweep_id = wandb.sweep(
    sweep_configs, project="your_project_name", entity="your_entity_name"
)
api = wandb.Api()
sweep = api.sweep(f"your_entity_name/your_project_name/sweeps/{sweep_id}")
print(f"EXPECTED RUN COUNT = {sweep.expected_run_count}")
```
