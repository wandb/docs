<div id="initialize-a-run-and-track-hyperparameters">
  ## run を初期化してハイパーパラメーターを追跡する
</div>

Python スクリプトまたはノートブック内で、[`wandb.init()`](/ja/models/ref/python/experiments/run/) を使って W&amp;B run オブジェクトを初期化します。`config` パラメーターには辞書を指定して、ハイパーパラメーター名と値を指定します。`with` 文のブロック内では、メトリクスやその他の情報を W&amp;B にログできます。

```python
import wandb

wandb.login()

# run の記録先プロジェクト
project = "my-awesome-project"

# ハイパーパラメーターの辞書
config = {
    'epochs' : 10,
    'lr' : 0.01
}

with wandb.init(project=project, config=config) as run:
    # トレーニングコードをここに記述
    # run.log() で W&B に値を記録する
    run.log({"accuracy": 0.9, "loss": 0.1})
```

次のセクションでは、トレーニング run をシミュレートし、accuracy と loss のメトリクスを W&amp;B に記録する完全な例を示します。

<Info>
  [run](/ja/models/runs/) は W&amp;B の中核要素です。run を使うと、[track metrics](/ja/models/track/)、[create logs](/ja/models/track/log/)、アーティファクトを追跡するなどのことができます。
</Info>

<div id="create-a-machine-learning-training-experiment">
  ## 機械学習のトレーニング実験を作成する
</div>

このモックのトレーニングスクリプトは、シミュレートした精度と損失のメトリクスを W&amp;B にログします。次のコードを Python スクリプトまたはノートブック セルにコピー＆ペーストして実行してください。

```python
import wandb
import random

wandb.login()

# run の記録先プロジェクト
project = "my-awesome-project"

# ハイパーパラメーターの辞書
config = {
    'epochs' : 10,
    'lr' : 0.01
}

with wandb.init(project=project, config=config) as run:
    offset = random.random() / 5
    print(f"lr: {config['lr']}")
    
    # トレーニング run のシミュレーション
    for epoch in range(2, config['epochs']):
        acc = 1 - 2**-config['epochs'] - random.random() / config['epochs'] - offset
        loss = 2**-config['epochs'] + random.random() / config['epochs'] + offset
        print(f"epoch={config['epochs']}, accuracy={acc}, loss={loss}")
        run.log({"accuracy": acc, "loss": loss})
```

記録されたメトリクス（精度や損失など）と、それらが各トレーニング ステップ中にどのように変化したかを確認するには、[wandb.ai/home](https://wandb.ai/home) にアクセスします。次の画像は、各 run で追跡された損失と精度を示しています。各 run オブジェクトは、自動生成された名前とともに **Runs** 列に表示されます。

<div id="next-steps">
  ## 次のステップ

W&amp;B エコシステムの他の機能も試してみてください。

1. PyTorch のようなフレームワーク、Hugging Face のようなライブラリ、SageMaker のようなサービスと W&amp;B を組み合わせた [W&amp;B Integration tutorials](/ja/models/integrations) を読みましょう。
2. [W&amp;B Reports](/ja/models/reports) を使って run を整理し、可視化を自動化し、結果を要約し、共同作業者と最新情報を共有します。
3. 機械学習パイプライン全体でデータセット、モデル、依存関係、結果を追跡するために [W&amp;B Artifacts](/ja/models/artifacts) を作成しましょう。
4. [W&amp;B Sweeps](/ja/models/sweeps) を使ってハイパーパラメーター探索を自動化し、モデルを最適化しましょう。
5. 共通の [central dashboard](/ja/models/tables) 上で run を分析し、モデル予測を可視化し、インサイトを共有しましょう。
6. [W&amp;B AI Academy](https://wandb.ai/site/courses/) を訪れて、LLM、MLOps、W&amp;B Models についてハンズオンコースで学びましょう。
7. [weave-docs.wandb.ai](/ja/weave) を訪れて、Weave を使って LLM ベースのアプリケーションを追跡、実験、評価、デプロイし、改善する方法を学びましょう。
