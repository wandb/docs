DeepChecks を使うと、機械学習モデルとデータを最小限の手間で検証できます。データの整合性を確認し、分布を調査し、データ分割を検証し、モデルを評価し、異なるモデルを比較できます。このページでは、DeepChecks と W&amp;B のインテグレーションの使い方を説明します。これにより、検証結果やテストスイートを Experiments とあわせてログできます。

詳細は、[DeepChecks W&amp;B インテグレーションガイド](https://docs.deepchecks.com/stable/general/usage/exporting_results/auto_examples/plot_exports_output_to_wandb.html)を参照してください。

<div id="get-started">
  ## はじめに
</div>

DeepChecks で W&amp;B を使用するには、まず [W&amp;B アカウント](https://wandb.ai/site) に登録してください。DeepChecks の W&amp;B インテグレーションでは、1 つのチェックを実行し、その結果を W&amp;B に送信するだけで使い始められます。

```python
import wandb

wandb.login()

# deepchecksからチェックをインポートする
from deepchecks.checks import ModelErrorAnalysis

# チェックを実行する
result = ModelErrorAnalysis()

# 結果をwandbにプッシュする
result.to_wandb()
```

個々のチェックをログするだけでなく、DeepChecks のテストスイート全体を W&amp;B にログすることもできます:

```python
import wandb

wandb.login()

# deepchecksからfull_suiteテストをインポートする
from deepchecks.suites import full_suite

# DeepChecksテストスイートを作成して実行する
suite_result = full_suite().run(...)

# 結果をwandbにpushする
# wandb.initに必要な設定や引数をここで渡すことができます
suite_result.to_wandb(project="my-suite-project", config={"suite-name": "full-suite"})
```
