[Prodigy](https://prodi.gy/) は、機械学習モデル向けのトレーニングデータや評価データの作成、エラー分析、データの確認とクリーニングに使用するアノテーションツールです。[W&amp;B Tables](/ja/models/tables/tables-walkthrough/) を使用すると、W&amp;B 内でデータセット (その他も含む) をログし、可視化、分析、共有できます。

[W&amp;B インテグレーション with Prodigy](https://github.com/wandb/wandb/blob/master/wandb/integration/prodigy/prodigy.py) は、シンプルで使いやすい機能を追加し、Prodigy でアノテーションしたデータセットを Tables で使用するために、W&amp;B に直接アップロードできるようにします。

次のような数行のコードを実行します。

```python
import wandb
from wandb.integration.prodigy import upload_dataset

with wandb.init(project="prodigy"):
    upload_dataset("news_headlines_ner")
```

<div id="quickstart">
  ## クイックスタート
</div>

`wandb.integration.prodigy.upload_dataset` を使用すると、ローカルの Prodigy データベースから注釈付きの Prodigy データセットを、W&amp;B の [表](/ja/models/ref/python/data-types/table) 形式で直接アップロードできます。インストールやセットアップを含む Prodigy の詳細については、[Prodigy documentation](https://prodi.gy/docs/)を参照してください。

W&amp;B は、画像フィールドと固有表現フィールドをそれぞれ自動的に [`wandb.Image`](/ja/models/ref/python/data-types/image) と [`wandb.Html`](/ja/models/ref/python/data-types/html) に変換しようとします。これらの可視化を含めるため、生成される表に追加の列が加えられる場合があります。
