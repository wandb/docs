[Prodigy](https://prodi.gy/) は、機械学習モデルの学習および評価用データの作成、エラー解析、データの検査およびクレンジングのためのアノテーションツールです。[W&amp;B Tables](/ja/models/tables/tables-walkthrough/) を使うと、W&amp;B 内でデータセット（など）をログとして記録、可視化、分析、および共有できます。

[W&amp;B integration with Prodigy](https://github.com/wandb/wandb/blob/master/wandb/integration/prodigy/prodigy.py) は、Prodigy でアノテーションしたデータセットを Tables で利用するために、W&amp;B に直接アップロードできるシンプルで使いやすい機能を提供します。

次のような数行のコードを実行してください:

```python
import wandb
from wandb.integration.prodigy import upload_dataset

with wandb.init(project="prodigy"):
    upload_dataset("news_headlines_ner")
```

<div id="quickstart">
  ## クイックスタート
</div>

`wandb.integration.prodigy.upload_dataset` を使用すると、ローカルの Prodigy データベースからアノテーション済みの Prodigy データセットを、W&amp;B の [Table](/ja/models/ref/python/data-types/table) 形式で直接アップロードできます。インストールやセットアップを含む Prodigy の詳細については、[Prodigy のドキュメント](https://prodi.gy/docs/) を参照してください。

W&amp;B は、画像フィールドおよび固有表現フィールドを自動的に [`wandb.Image`](/ja/models/ref/python/data-types/image) および [`wandb.Html`](/ja/models/ref/python/data-types/html) にそれぞれ変換しようとします。これらの可視化を含めるために、変換後のテーブルに追加のカラムが含まれる場合があります。
