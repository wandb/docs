W&amp;B は、Databricks 環境での W&amp;B Jupyter ノートブックの操作性をカスタマイズすることで、[Databricks](https://www.databricks.com/) と統合します。

<div id="configure-databricks">
  ## Databricks を設定する
</div>

1. クラスターに wandb をインストールする

   クラスターの設定にアクセスし、クラスターを選択して **Libraries** をクリックします。**Install New** をクリックし、**PyPI** を選択して、パッケージ `wandb` を追加します。

2. 認証を設定する

   W&amp;B アカウントを認証するには、ノートブックからクエリできる Databricks secret を追加します。

   ```bash
   # Databricks CLI をインストール
   pip install databricks-cli

   # Databricks の UI からトークンを発行
   databricks configure --token

   # スコープを作成（Databricks でセキュリティ機能が有効かどうかに応じて、次の 2 つのコマンドのいずれかを使用）
   # セキュリティアドオンあり
   databricks secrets create-scope --scope wandb
   # セキュリティアドオンなし
   databricks secrets create-scope --scope wandb --initial-manage-principal users

   # https://wandb.ai/settings で APIキーを作成
   databricks secrets put --scope wandb --key api_key
   ```

<div id="examples">
  ## 例
</div>

<div id="simple-example">
  ### シンプルな例
</div>

```python
import os
import wandb

api_key = dbutils.secrets.get("wandb", "api_key")
wandb.login(key=api_key)

with wandb.init() as run:
    run.log({"foo": 1})
```


<div id="sweeps">
  ### Sweeps
</div>

wandb.sweep() または wandb.agent() を使用するノートブックで必須の設定 (一時的) :

```python
import os

# これらは将来的には不要になります
os.environ["WANDB_ENTITY"] = "my-entity"
os.environ["WANDB_PROJECT"] = "my-project-that-exists"
```
