[DeepChem library](https://github.com/deepchem/deepchem) は、創薬、材料科学、化学、生物学におけるディープラーニングの活用を民主化するオープンソースツール群を提供します。この W&amp;B インテグレーションを使うと、DeepChem でモデルを学習する際に、シンプルで使いやすい実験トラッキング機能とモデルのチェックポイント機能を追加できます。

<div id="deepchem-logging-in-3-lines-of-code">
  ## 3行のコードで DeepChem をログに記録する
</div>

```python
logger = WandbLogger(…)
model = TorchModel(…, wandb_logger=logger)
model.fit(…)
```

<div id="log-your-training-and-evaluation-data-to-wb">
  ### 学習および評価データを W&amp;B にログする
</div>

学習損失と評価メトリクスは自動的に W&amp;B にログできます。オプションの評価は DeepChem の [ValidationCallback](https://github.com/deepchem/deepchem/blob/master/deepchem/models/callbacks.py) を使って有効化できます。`WandbLogger` は ValidationCallback を検出し、生成されたメトリクスをログします。

<Tabs>
  <Tab title="TorchModel">
    ```python
    from deepchem.models import TorchModel, ValidationCallback

    vc = ValidationCallback(…)  # optional
    model = TorchModel(…, wandb_logger=logger)
    model.fit(…, callbacks=[vc])
    logger.finish()
    ```
  </Tab>

  <Tab title="KerasModel">
    ```python
    from deepchem.models import KerasModel, ValidationCallback

    vc = ValidationCallback(…)  # optional
    model = KerasModel(…, wandb_logger=logger)
    model.fit(…, callbacks=[vc])
    logger.finish()
    ```
  </Tab>
</Tabs>
