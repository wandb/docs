---
title: NVIDIA NeMo Inference Microservice Deploy Job
---

W&amp;B から NVIDIA NeMo Inference Microservice にモデルアーティファクトをデプロイします。そのためには W&amp;B Launch を使用します。W&amp;B Launch はモデルアーティファクトを NVIDIA NeMo Model に変換し、稼働中の NIM/Triton サーバーにデプロイします。

W&amp;B Launch は現在、次の互換性のあるモデルタイプに対応しています:

1. [Llama2](https://llama.meta.com/llama2/)
2. [StarCoder](https://github.com/bigcode-project/starcoder)
3. NV-GPT (coming soon)

<Note>
  デプロイ時間はモデルとマシンタイプによって異なります。標準的な Llama2-7b の設定では、Google Cloud の `a2-ultragpu-1g` で約 1 分かかります。
</Note>

<div id="quickstart">
  ## クイックスタート
</div>

1. まだ Launch キューがない場合は、[Launch キューを作成](/ja/platform/launch/add-job-to-queue/)します。以下にキュー設定の例を示します。

   ```yaml
   net: host
   gpus: all # 特定の GPU を指定するか、`all` を指定してすべてを使用できます
   runtime: nvidia # NVIDIA コンテナランタイムも必要です
   volume:
     - model-store:/model-store/
   ```

   <Frame>
     <img src="/images/integrations/nim1.png" alt="画像" />
   </Frame>

2. 次のジョブをプロジェクト内に作成します:

   ```bash
   wandb job create -n "deploy-to-nvidia-nemo-inference-microservice" \
      -e $ENTITY \
      -p $PROJECT \
      -E jobs/deploy_to_nvidia_nemo_inference_microservice/job.py \
      -g andrew/nim-updates \
      git https://github.com/wandb/launch-jobs
   ```

3. GPU マシン上でエージェントを起動します:
   ```bash
   wandb launch-agent -e $ENTITY -p $PROJECT -q $QUEUE
   ```

4. 希望する設定で、[Launch UI](https://wandb.ai/launch) からデプロイ用の Launch ジョブを送信します。
   1. CLI から送信することもできます:
      ```bash
      wandb launch -d gcr.io/playground-111/deploy-to-nemo:latest \
        -e $ENTITY \
        -p $PROJECT \
        -q $QUEUE \
        -c $CONFIG_JSON_FNAME
      ```
      <Frame>
        <img src="/images/integrations/nim2.png" alt="画像" />
      </Frame>

5. Launch UI でデプロイプロセスを追跡できます。
   <Frame>
     <img src="/images/integrations/nim3.png" alt="画像" />
   </Frame>

6. 完了したら、すぐに curl コマンドでエンドポイントにリクエストを送り、モデルをテストできます。モデル名は常に `ensemble` です。
   ```bash
    #!/bin/bash
    curl -X POST "http://0.0.0.0:9999/v1/completions" \
        -H "accept: application/json" \
        -H "Content-Type: application/json" \
        -d '{
            "model": "ensemble",
            "prompt": "Tell me a joke",
            "max_tokens": 256,
            "temperature": 0.5,
            "n": 1,
            "stream": false,
            "stop": "string",
            "frequency_penalty": 0.0
            }'
   ```
