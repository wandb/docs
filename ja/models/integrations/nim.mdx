---

W&amp;B から NVIDIA NeMo Inference Microservice にモデル アーティファクトをデプロイします。これを行うには、W&amp;B Launch を使用します。W&amp;B Launch はモデル アーティファクトを NVIDIA NeMo Model に変換し、実行中の NIM/Triton サーバーにデプロイします。

現在、W&amp;B Launch は次の互換性のあるモデル タイプをサポートしています。

1. [Llama2](https://llama.meta.com/llama2/)
2. [StarCoder](https://github.com/bigcode-project/starcoder)
3. NV-GPT (近日対応予定)

<Note>
  Deployment time varies by model and machine type. The base Llama2-7b config takes about 1 minute on Google Cloud&#39;s `a2-ultragpu-1g`.
</Note>

<div id="quickstart">
  ## クイックスタート
</div>

1. まだ Launch キューがない場合は、[Launch キューを作成](/ja/platform/launch/add-job-to-queue/) してください。以下にキュー設定の例を示します。

   ```yaml
   net: host
   gpus: all # 特定の GPU セット、または `all` を指定してすべてを使用できます
   runtime: nvidia # NVIDIA コンテナランタイムも必要です
   volume:
     - model-store:/model-store/
   ```

   <Frame>
     <img src="/images/integrations/nim1.png" alt="画像" />
   </Frame>

2. プロジェクト内にこのジョブを作成してください。

   ```bash
   wandb job create -n "deploy-to-nvidia-nemo-inference-microservice" \
      -e $ENTITY \
      -p $PROJECT \
      -E jobs/deploy_to_nvidia_nemo_inference_microservice/job.py \
      -g andrew/nim-updates \
      git https://github.com/wandb/launch-jobs
   ```

3. GPU マシン上でエージェントを起動してください。
   ```bash
   wandb launch-agent -e $ENTITY -p $PROJECT -q $QUEUE
   ```

4. [Launch UI](https://wandb.ai/launch) から、ご希望の設定でデプロイ用の Launch ジョブを送信してください。
   1. CLI を使用して送信することもできます。
      ```bash
      wandb launch -d gcr.io/playground-111/deploy-to-nemo:latest \
        -e $ENTITY \
        -p $PROJECT \
        -q $QUEUE \
        -c $CONFIG_JSON_FNAME
      ```
      <Frame>
        <img src="/images/integrations/nim2.png" alt="画像" />
      </Frame>

5. Launch UI でデプロイプロセスを追跡できます。
   <Frame>
     <img src="/images/integrations/nim3.png" alt="画像" />
   </Frame>

6. 完了したら、すぐにエンドポイントを curl してモデルをテストできます。モデル名は常に `ensemble` です。
   ```bash
    #!/bin/bash
    curl -X POST "http://0.0.0.0:9999/v1/completions" \
        -H "accept: application/json" \
        -H "Content-Type: application/json" \
        -d '{
            "model": "ensemble",
            "prompt": "Tell me a joke",
            "max_tokens": 256,
            "temperature": 0.5,
            "n": 1,
            "stream": false,
            "stop": "string",
            "frequency_penalty": 0.0
            }'
   ```
