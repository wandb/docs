<div id="usage">
  ## 使い方
</div>

```bash
wandb [OPTIONS] COMMAND [ARGS]...
```

<div id="options">
  ## オプション
</div>

| オプション       | 説明                               |
| :---------- | :------------------------------- |
| `--version` | バージョンを表示して終了します。 (デフォルト: False)  |

<div id="commands">
  ## コマンド
</div>

| コマンド                                               | 説明                                                                        |
| :------------------------------------------------- | :------------------------------------------------------------------------ |
| [agent](/ja/models/ref/cli/wandb-agent)               | W&amp;B エージェントを実行します                                                      |
| [artifact](/ja/models/ref/cli/wandb-artifact)         | artifact を操作するためのコマンド                                                     |
| [beta](/ja/models/ref/cli/wandb-beta)                 | wandb CLI コマンドのベータ版。                                                      |
| [controller](/ja/models/ref/cli/wandb-controller)     | W&amp;B のローカル sweep コントローラーを実行します                                         |
| [disabled](/ja/models/ref/cli/wandb-disabled)         | W&amp;B を無効にします。                                                          |
| [docker](/ja/models/ref/cli/wandb-docker)             | docker コンテナーでコードを実行します。                                                   |
| [docker-run](/ja/models/ref/cli/wandb-docker-run)     | `docker run` をラップし、WANDB&#95;API&#95;KEY および WANDB&#95;DOCKER 環境変数を追加します。 |
| [enabled](/ja/models/ref/cli/wandb-enabled)           | W&amp;B を有効にします。                                                          |
| [init](/ja/models/ref/cli/wandb-init)                 | ディレクトリーを Weights &amp; Biases 用に設定します                                     |
| [job](/ja/models/ref/cli/wandb-job)                   | W&amp;B ジョブの管理と表示のためのコマンド                                                 |
| [launch](/ja/models/ref/cli/wandb-launch)             | W&amp;B Job を Launch するか、キューに追加します。                                       |
| [launch-agent](/ja/models/ref/cli/wandb-launch-agent) | W&amp;B Launch エージェントを実行します。                                              |
| [launch-sweep](/ja/models/ref/cli/wandb-launch-sweep) | W&amp;B Launch sweep を実行します (実験的機能) 。                                     |
| [local](/ja/models/ref/cli/wandb-local)               | ローカル W&amp;B コンテナーを起動します (非推奨。`wandb server --help` を参照)                  |
| [login](/ja/models/ref/cli/wandb-login)               | W&amp;B サービスで認証するための APIキー を確認して保存します。                                    |
| [off](/ja/models/ref/cli/wandb-off)                   | 説明はありません                                                                  |
| [offline](/ja/models/ref/cli/wandb-offline)           | W&amp;B にログされたデータをクラウドにアップロードせずにローカルへ保存します。                               |
| [on](/ja/models/ref/cli/wandb-on)                     | 説明はありません                                                                  |
| [online](/ja/models/ref/cli/wandb-online)             | `wandb offline` を解除します。                                                   |
| [projects](/ja/models/ref/cli/wandb-projects)         | Projects を一覧表示します                                                         |
| [pull](/ja/models/ref/cli/wandb-pull)                 | Weights &amp; Biases からファイルを取得します                                         |
| [purge-cache](/ja/models/ref/cli/wandb-purge-cache)   | ローカル W&amp;B キャッシュから、キャッシュされたログ、run 履歴、Artifacts を削除します。                  |
| [restore](/ja/models/ref/cli/wandb-restore)           | run のコード、設定、docker の状態を復元します。                                             |
| [scheduler](/ja/models/ref/cli/wandb-scheduler)       | W&amp;B Launch sweep スケジューラーを実行します (実験的機能)                                |
| [server](/ja/models/ref/cli/wandb-server)             | ローカル W&amp;B サーバーを運用するためのコマンド                                             |
| [status](/ja/models/ref/cli/wandb-status)             | 設定を表示します                                                                  |
| [sweep](/ja/models/ref/cli/wandb-sweep)               | ハイパーパラメーター sweep を初期化します。                                                 |
| [sync](/ja/models/ref/cli/wandb-sync)                 | W&amp;B run データをクラウドに同期します。                                               |
| [verify](/ja/models/ref/cli/wandb-verify)             | W&amp;B のローカルインスタンスをチェックして検証します。                                          |
