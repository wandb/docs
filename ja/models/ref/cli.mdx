<div id="usage">
  ## 使い方
</div>

```bash
wandb [OPTIONS] COMMAND [ARGS]...
```

<div id="options">
  ## オプション
</div>

| オプション | 説明 |
| :--- | :--- |
| `--version` | バージョンを表示して終了します (デフォルト: False)  |

  ## コマンド
</div>

| コマンド                                               | 説明                                                                      |
| :------------------------------------------------- | :---------------------------------------------------------------------- |
| [agent](/ja/models/ref/cli/wandb-agent)               | W&amp;B agent を実行します。                                                   |
| [artifact](/ja/models/ref/cli/wandb-artifact)         | アーティファクトを操作するコマンドです。                                                    |
| [beta](/ja/models/ref/cli/wandb-beta)                 | wandb CLI コマンドのベータ版です。                                                  |
| [controller](/ja/models/ref/cli/wandb-controller)     | W&amp;B のローカル sweep コントローラーを実行します。                                      |
| [disabled](/ja/models/ref/cli/wandb-disabled)         | W&amp;B を無効化します。                                                        |
| [docker](/ja/models/ref/cli/wandb-docker)             | コードを Docker コンテナ内で実行します。                                                |
| [docker-run](/ja/models/ref/cli/wandb-docker-run)     | `docker run` をラップし、WANDB&#95;API&#95;KEY と WANDB&#95;DOCKER 環境変数を追加します。 |
| [enabled](/ja/models/ref/cli/wandb-enabled)           | W&amp;B を有効化します。                                                        |
| [init](/ja/models/ref/cli/wandb-init)                 | Weights &amp; Biases 用にディレクトリを設定します。                                    |
| [job](/ja/models/ref/cli/wandb-job)                   | W&amp;B Job の管理および表示用コマンドです。                                            |
| [launch](/ja/models/ref/cli/wandb-launch)             | W&amp;B Job を起動またはキューに追加します。                                            |
| [launch-agent](/ja/models/ref/cli/wandb-launch-agent) | W&amp;B launch agent を実行します。                                            |
| [launch-sweep](/ja/models/ref/cli/wandb-launch-sweep) | W&amp;B launch sweep を実行します (実験的機能) 。                                   |
| [local](/ja/models/ref/cli/wandb-local)               | ローカルの W&amp;B コンテナを起動します (非推奨。`wandb server --help` を参照) 。              |
| [login](/ja/models/ref/cli/wandb-login)               | W&amp;B サービスでの認証のために API キーを検証して保存します。                                  |
| [off](/ja/models/ref/cli/wandb-off)                   | 利用可能な説明はありません。                                                          |
| [offline](/ja/models/ref/cli/wandb-offline)           | W&amp;B にログされたデータをクラウドにアップロードせずローカルに保存します。                              |
| [on](/ja/models/ref/cli/wandb-on)                     | 利用可能な説明はありません。                                                          |
| [online](/ja/models/ref/cli/wandb-online)             | `wandb offline` を取り消します。                                                |
| [projects](/ja/models/ref/cli/wandb-projects)         | プロジェクトを一覧表示します。                                                         |
| [pull](/ja/models/ref/cli/wandb-pull)                 | Weights &amp; Biases からファイルを取得します。                                      |
| [restore](/ja/models/ref/cli/wandb-restore)           | run のコード、設定、Docker の状態を復元します。                                           |
| [scheduler](/ja/models/ref/cli/wandb-scheduler)       | W&amp;B launch sweep スケジューラーを実行します (実験的機能) 。                            |
| [server](/ja/models/ref/cli/wandb-server)             | ローカル W&amp;B サーバーの操作用コマンドです。                                            |
| [status](/ja/models/ref/cli/wandb-status)             | 設定内容を表示します。                                                             |
| [sweep](/ja/models/ref/cli/wandb-sweep)               | ハイパーパラメーター sweep を初期化します。                                               |
| [sync](/ja/models/ref/cli/wandb-sync)                 | W&amp;B run データをクラウドに同期します。                                             |
| [verify](/ja/models/ref/cli/wandb-verify)             | ローカルの W&amp;B インスタンスをチェックして検証します。                                       |
