<div id="usage">
  ## 使い方
</div>

```bash
wandb [OPTIONS] COMMAND [ARGS]...
```

<div id="options">
  ## オプション
</div>

| オプション       | 説明                               |
| :---------- | :------------------------------- |
| `--version` | バージョンを表示して終了します。 (デフォルト: False)  |

<div id="commands">
  ## コマンド
</div>

| Command                                            | Description                                                   |
| :------------------------------------------------- | :------------------------------------------------------------ |
| [agent](/ja/models/ref/cli/wandb-agent)               | sweep エージェントを起動します。                                           |
| [artifact](/ja/models/ref/cli/wandb-artifact)         | W&amp;B Artifacts のアップロード、ダウンロード、管理を行います。                     |
| [beta](/ja/models/ref/cli/wandb-beta)                 | wandb CLI コマンドのベータ版です。                                        |
| [controller](/ja/models/ref/cli/wandb-controller)     | W&amp;B ハイパーパラメーター sweep 用のローカル sweep コントローラーを起動します。          |
| [disabled](/ja/models/ref/cli/wandb-disabled)         | W&amp;B の機能を無効にします。                                           |
| [docker](/ja/models/ref/cli/wandb-docker)             | W&amp;B が設定された Docker コンテナーでコードを実行します。                        |
| [docker-run](/ja/models/ref/cli/wandb-docker-run)     | `docker run` をラップし、W&amp;B 環境変数を自動的に注入します。                    |
| [enabled](/ja/models/ref/cli/wandb-enabled)           | `wandb disabled` で無効化した W&amp;B を再度有効にします。                    |
| [init](/ja/models/ref/cli/wandb-init)                 | 現在のディレクトリーに対する W&amp;B 設定を初期化または更新します。                        |
| [job](/ja/models/ref/cli/wandb-job)                   | W&amp;B ジョブを管理および表示するためのコマンドです。                               |
| [launch](/ja/models/ref/cli/wandb-launch)             | W&amp;B Job を Launch するか、キューに追加します。                           |
| [launch-agent](/ja/models/ref/cli/wandb-launch-agent) | W&amp;B Launch エージェントを実行します。                                  |
| [launch-sweep](/ja/models/ref/cli/wandb-launch-sweep) | W&amp;B Launch sweep を実行します (実験的) 。                           |
| [login](/ja/models/ref/cli/wandb-login)               | このマシンを W&amp;B に対して認証します。                                     |
| [offline](/ja/models/ref/cli/wandb-offline)           | W&amp;B にログしたデータをクラウドにアップロードせず、ローカルに保存します。                    |
| [online](/ja/models/ref/cli/wandb-online)             | W&amp;B Runs のクラウド同期を再度有効にします。                                |
| [pull](/ja/models/ref/cli/wandb-pull)                 | W&amp;B run からファイルをダウンロードします。                                 |
| [purge-cache](/ja/models/ref/cli/wandb-purge-cache)   | ローカル W&amp;B キャッシュから、キャッシュされたログ、run history、Artifacts を削除します。 |
| [restore](/ja/models/ref/cli/wandb-restore)           | 以前の W&amp;B run からコード、設定、または Docker 環境を復元します。                 |
| [scheduler](/ja/models/ref/cli/wandb-scheduler)       | W&amp;B Launch sweep スケジューラーを実行します (実験的) 。                    |
| [server](/ja/models/ref/cli/wandb-server)             | ローカル W&amp;B server を操作するためのコマンドです                            |
| [status](/ja/models/ref/cli/wandb-status)             | 現在の W&amp;B 設定を表示します。                                         |
| [sweep](/ja/models/ref/cli/wandb-sweep)               | ハイパーパラメーター sweep を作成、更新、または管理します。                             |
| [sync](/ja/models/ref/cli/wandb-sync)                 | 既存のローカル W&amp;B run データをクラウドにアップロードします。                       |
| [verify](/ja/models/ref/cli/wandb-verify)             | セルフホスト型 W&amp;B インスタンスに対してインテグレーション チェックを実行します。               |
