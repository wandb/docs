Weights & Biases からファイルをプルします。

## Usage（使用法）

```bash
wandb pull RUN [OPTIONS]
```

## Arguments（引数）

| 引数 | 説明 | 必須 |
| :--- | :--- | :--- |
| `RUN` | 説明なし | はい |

## Options（オプション）

| オプション | 説明 |
| :--- | :--- |
| `--project`, `-p` | ダウンロードしたい Projects。 |
| `--entity`, `-e` | リストのスコープを制限する Entities。（デフォルト: models） |
