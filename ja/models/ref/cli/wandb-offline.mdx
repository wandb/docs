クラウドにアップロードせずに、W&amp;B にログしたデータをローカルに保存します。

`wandb online` または `wandb sync` を使用して、オフラインのRunsをアップロードします。


<div id="usage">
  ## 使用方法
</div>

```bash
wandb offline
```
