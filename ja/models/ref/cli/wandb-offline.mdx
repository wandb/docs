クラウドにアップロードせず、W&amp;B にログしたデータをローカルに保存できます。

オフライン run をアップロードするには、`wandb online` または `wandb sync` を実行します。


<div id="usage">
  ## 使用方法
</div>

```bash
wandb offline
```
