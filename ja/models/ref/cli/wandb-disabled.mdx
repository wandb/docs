W&amp;B を無効にする。


<div id="usage">
  ## 使用方法
</div>

```bash
wandb disabled [OPTIONS]
```


<div id="options">
  ## オプション
</div>

| オプション | 説明 |
| :--- | :--- |
| `--service` | W&B サービスを無効にします（デフォルト: True） |
