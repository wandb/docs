W&amp;Bを無効にします。


<div id="usage">
  ## 使用法
</div>

```bash
wandb disabled [OPTIONS]
```


<div id="options">
  ## オプション
</div>

| オプション       | 説明                                    |
| :---------- | :------------------------------------ |
| `--service` | W&amp;B serviceを無効にします (デフォルト: True)  |
