W&amp;B ジョブを管理・表示するコマンド

<div id="usage">
  ## 使用法
</div>

```bash
wandb job COMMAND [ARGS]...
```

<div id="commands">
  ## コマンド
</div>

| Command                                                  | 説明                             |
| :------------------------------------------------------- | :----------------------------- |
| [create](/ja/models/ref/cli/wandb-job/wandb-job-create)     | wandb run を使わずにソースからジョブを作成します。 |
| [describe](/ja/models/ref/cli/wandb-job/wandb-job-describe) | Launch ジョブの詳細を表示します。           |
| [list](/ja/models/ref/cli/wandb-job/wandb-job-list)         | プロジェクト内のジョブを一覧表示します            |
