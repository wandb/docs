---
title: "wandb job"
---

W&amp;B ジョブを管理および表示するためのコマンド

<div id="usage">
  ## 使い方
</div>

```bash
wandb job COMMAND [ARGS]...
```

<div id="commands">
  ## コマンド
</div>

| コマンド | 説明 |
| :--- | :--- |
| [create](/ja/models/ref/cli/wandb-job/wandb-job-create) | wandb run を作成せずに、ソースからジョブを作成します。 |
| [describe](/ja/models/ref/cli/wandb-job/wandb-job-describe) | Launch のジョブの詳細を表示します。 |
| [list](/ja/models/ref/cli/wandb-job/wandb-job-list) | プロジェクト内のジョブを一覧表示します。 |
