<div id="description">
  ## 説明
</div>

W&amp;B ジョブを管理・表示するコマンド

<div id="subcommands">
  ## サブコマンド
</div>

| Command              | 説明                                        |
| -------------------- | ----------------------------------------- |
| `wandb job create`   | wandb run を使わずにソースからジョブを作成します。            |
| `wandb job describe` | Launch ジョブの詳細を表示します。Launch ジョブを次の形式で指定します |
| `wandb job list`     | プロジェクト内のジョブを一覧表示します。                      |
