---
title: "wandb ベータ版"
---

wandb CLI コマンドのベータ版です。

これらのコマンドは、今後の wandb のレポートで変更されたり、完全に動作しなくなったりする可能性があります。

<div id="usage">
  ## 使用
</div>

```bash
wandb beta COMMAND [ARGS]...
```

<div id="commands">
  ## コマンド
</div>

| Command                                            | Description                                                |
| :------------------------------------------------- | :--------------------------------------------------------- |
| [leet](/ja/models/ref/cli/wandb-beta/wandb-beta-leet) | W&amp;B LEET: the Lightweight Experiment Exploration Tool. |
| [sync](/ja/models/ref/cli/wandb-beta/wandb-beta-sync) | PATHS で指定された .wandb ファイルをアップロードします。                        |
