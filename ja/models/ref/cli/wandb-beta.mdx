---
title: "wandb ベータ"
---

wandb CLI コマンドのベータ版です。

これらのコマンドは、wandb のどの version でも変更されたり、完全に動作しなくなったりする可能性があります。

<div id="usage">
  ## 使用法
</div>

```bash
wandb beta COMMAND [ARGS]...
```

<div id="commands">
  ## コマンド
</div>

| コマンド                                               | 説明                                  |
| :------------------------------------------------- | :---------------------------------- |
| [leet](/ja/models/ref/cli/wandb-beta/wandb-beta-leet) | W&amp;B LEET: 軽量実験探索ツール。            |
| [sync](/ja/models/ref/cli/wandb-beta/wandb-beta-sync) | PATHS で指定された .wandb ファイルをアップロードします。 |
