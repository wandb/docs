<div id="usage">
  ## 使用法
</div>

```bash
beta [OPTIONS] COMMAND [ARGS]...
```

<div id="description">
  ## 説明
</div>

wandb CLI コマンドのベータ版です。

これらのコマンドは、wandb のどの version でも変更されたり、完全に動作しなくなったりする可能性があります。

<div id="subcommands">
  ## サブコマンド
</div>

| コマンド              | 説明                                              |
| ----------------- | ----------------------------------------------- |
| `wandb beta core` | マルチプロセスのワークロード向けに、共有ローカル wandb-core サービスを管理します。 |
| `wandb beta leet` | W&amp;B LEET: 軽量実験探索ツール。                        |
| `wandb beta sync` | PATHS で指定された .wandb ファイルをアップロードします。             |
