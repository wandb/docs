---
title: "wandb ベータ版"
---

wandb CLI コマンドのベータ版です。

これらのコマンドは、今後の wandb リリースで変更されたり、完全に動作しなくなったりする可能性があります。

<div id="usage">
  ## 使用方法
</div>

```bash
wandb beta COMMAND [ARGS]...
```

<div id="commands">
  ## コマンド
</div>

| コマンド | 説明 |
| :--- | :--- |
| [leet](/ja/models/ref/cli/wandb-beta/wandb-beta-leet) | W&amp;B LEET（Lightweight Experiment Exploration Tool）を起動します。 |
| [sync](/ja/models/ref/cli/wandb-beta/wandb-beta-sync) | PATHS で指定された .wandb ファイルをアップロードします。 |
