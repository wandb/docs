W&amp;B エージェントを実行する


<div id="usage">
  ## 使い方
</div>

```bash
wandb agent SWEEP_ID [OPTIONS]
```


<div id="arguments">
  ## 引数
</div>

| 引数         | 説明   | 必須 |
| :--------- | :--- | :- |
| `SWEEP_ID` | 説明なし | はい |

<div id="options">
  ## オプション
</div>

| Option                    | Description                                                                                                                             |
| :------------------------ | :-------------------------------------------------------------------------------------------------------------------------------------- |
| `--project`, `-p`         | sweep で作成された W&amp;B run の送信先となるプロジェクト名。プロジェクトを指定しない場合、run は「Uncategorized」というラベルのプロジェクトに送信されます。                                        |
| `--entity`, `-e`          | sweep で作成された W&amp;B run の送信先にするユーザー名またはチーム名。指定する entity は、あらかじめ存在している必要があります。entity を指定しない場合、run はデフォルトの entity (通常は自分のユーザー名) に送信されます。 |
| `--count`                 | このエージェントで作成する run の最大数。                                                                                                                |
| `--forward-signals`, `-f` | エージェントに送られたシグナル (例: SIGINT/SIGTERM) を子 run に転送し、それらが正常にシャットダウンできるようにします。 (デフォルト: False)                                                 |
