ローカルで W&amp;B コンテナを起動（非推奨。`wandb server --help` を参照）


<div id="usage">
  ## 使い方
</div>

```bash
wandb local [OPTIONS]
```


<div id="options">
  ## オプション
</div>

| オプション | 説明 |
| :--- | :--- |
| `--port`, `-p` | W&B Local をバインドするホストポート（デフォルト：8080） |
| `--env`, `-e` | wandb/local に渡す環境変数（デフォルト：[]） |
| `--daemon` | デーモンモードで実行するかどうか（デフォルト：True） |
| `--upgrade` | 最新バージョンにアップグレード（デフォルト：False） |
