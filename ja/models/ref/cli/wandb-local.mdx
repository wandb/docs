ローカルの W&amp;B コンテナを起動する（非推奨。wandb server --help を参照）


<div id="usage">
  ## 使用方法
</div>

```bash
wandb local [OPTIONS]
```


<div id="options">
  ## オプション
</div>

| オプション | 説明 |
| :--- | :--- |
| `--port`, `-p` | W&B Local をバインドするホストポート（デフォルト: 8080） |
| `--env`, `-e` | wandb/local に渡す環境変数（デフォルト: []） |
| `--daemon` | デーモン モードで実行するかどうか（デフォルト: True） |
| `--upgrade` | 最新バージョンにアップグレードするかどうか（デフォルト: False） |
