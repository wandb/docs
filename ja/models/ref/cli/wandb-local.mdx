ローカルの W&amp;B コンテナーを起動します (非推奨。wandb server --help を参照) 


<div id="usage">
  ## 使用方法
</div>

```bash
wandb local [OPTIONS]
```


<div id="options">
  ## オプション
</div>

| オプション          | 説明                                           |
| :------------- | :------------------------------------------- |
| `--port`, `-p` | W&amp;B local をバインドするホスト側のポート (デフォルト: 8080)  |
| `--env`, `-e`  | wandb/local に渡す環境変数 (デフォルト: [])              |
| `--daemon`     | デーモンモードで実行するかどうか (デフォルト: True)               |
| `--upgrade`    | 最新バージョンにアップグレードするかどうか (デフォルト: False)         |
