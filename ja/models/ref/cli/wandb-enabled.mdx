<div id="usage">
  ## 使用方法
</div>

```bash
wandb enabled [OPTIONS]
```


<div id="options">
  ## オプション
</div>

| オプション       | 説明                                     |
| :---------- | :------------------------------------- |
| `--service` | W&amp;B Service を有効にします (デフォルト: True)  |
