W&amp;Bを有効にします。


<div id="usage">
  ## 使い方
</div>

```bash
wandb enabled [OPTIONS]
```


<div id="options">
  ## オプション
</div>

| オプション       | 説明                                 |
| :---------- | :--------------------------------- |
| `--service` | W&amp;B サービスを有効にします (デフォルト: True)  |
