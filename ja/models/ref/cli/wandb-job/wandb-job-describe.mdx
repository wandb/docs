Launch ジョブの詳細を表示します。 Launch ジョブは `entity/project/job-name:alias-or-version` の形式で指定してください。

## Usage

```bash
wandb job describe JOB
```

## Arguments

| 引数 | 説明 | 必須 |
| :--- | :--- | :--- |
| `JOB` | 説明なし | はい |
