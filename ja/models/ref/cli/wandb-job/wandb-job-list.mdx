Project 内の job を一覧表示します。

## 使用法

```bash
wandb job list [OPTIONS]
```

## オプション

| オプション | 説明 |
| :--- | :--- |
| `--project`, `-p` | job を一覧表示したい Project。 |
| `--entity`, `-e` | job が属する Entity (デフォルト: models) |
