Weights &amp; Biases 用ディレクトリを設定する


<div id="usage">
  ## 使用方法
</div>

```bash
wandb init [OPTIONS]
```


<div id="options">
  ## オプション
</div>

| オプション | 説明 |
| :--- | :--- |
| `--project`, `-p` | 使用するプロジェクト。 |
| `--entity`, `-e` | プロジェクトの対象とする entity。 |
| `--reset` | 設定をリセットします (デフォルト: False)。 |
| `--mode`, `-m` | `"online"`、`"offline"`、`"disabled"` のいずれか。デフォルトは `online`。 |
