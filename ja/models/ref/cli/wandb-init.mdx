Weights &amp; Biases 用のディレクトリを設定する


<div id="usage">
  ## 使用法
</div>

```bash
wandb init [OPTIONS]
```


<div id="options">
  ## オプション
</div>

| オプション             | 説明                                                                                        |
| :---------------- | :---------------------------------------------------------------------------------------- |
| `--project`, `-p` | 使用するプロジェクト。                                                                               |
| `--entity`, `-e`  | プロジェクトのスコープを設定するentity。                                                                   |
| `--reset`         | 設定をリセットする (デフォルト: False)                                                                  |
| `--mode`, `-m`    | &quot;online&quot;、&quot;offline&quot;、または &quot;disabled&quot; を指定できます。デフォルトは online です。 |
