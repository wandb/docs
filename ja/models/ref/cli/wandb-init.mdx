Weights &amp; Biases 用のディレクトリを設定する


<div id="usage">
  ## 使用方法
</div>

```bash
wandb init [OPTIONS]
```


<div id="options">
  ## オプション
</div>

| オプション | 説明 |
| :--- | :--- |
| `--project`, `-p` | 使用するプロジェクト。 |
| `--entity`, `-e` | プロジェクトを紐づける entity。 |
| `--reset` | 設定をリセットする（デフォルト: False）。 |
| `--mode`, `-m` | `"online"`、`"offline"`、`"disabled"` のいずれか。デフォルトは `online`。 |
