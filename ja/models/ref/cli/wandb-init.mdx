<div id="usage">
  ## 使用法
</div>

```bash
init [OPTIONS]
```

<div id="description">
  ## 説明
</div>

現在のディレクトリの W&amp;B 設定を初期化または更新します。

project と entity を設定し、ローカルの W&amp;B 設定を作成して、
実験管理を行うための準備をディレクトリに対して行います。

たとえば、チームと project を選択するための対話形式のプロンプトを使用して、
現在のディレクトリに W&amp;B を設定します。

```bash
wandb init
```

デフォルトのprojectを&quot;foobar&quot;に、デフォルトのentityを&quot;team-awesome&quot;に、確認プロンプトを表示せずに設定するには:

```bash
wandb init --project foobar --entity team-awesome
```

W&amp;B をオフライン モードに設定するには:

```bash
wandb init --mode offline
```

現在のディレクトリにある既存のW&amp;B設定をリセットするには:

```bash
wandb init --reset
```

<div id="options">
  ## オプション
</div>

| フラグ             | タイプ       | 説明                                                                                                 |
| --------------- | --------- | -------------------------------------------------------------------------------------------------- |
| `--project, -p` | STR       | runs のアップロード先の project を設定します。**デフォルト**: なし                                                        |
| `--entity, -e`  | STR       | project のスコープ先となる entity を設定します。**デフォルト**: なし                                                      |
| `--reset`       | BOOL Flag | このディレクトリの既存の W&amp;B 設定をリセットします。**デフォルト**: False                                                   |
| `--mode, -m`    | STR       | W&amp;B のモードを設定します。&#39;online&#39;、&#39;offline&#39;、または &#39;disabled&#39; のいずれかです。**デフォルト**: なし |
