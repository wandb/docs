<div id="usage">
  ## 使用方法
</div>

```bash
artifact [OPTIONS] COMMAND [ARGS]...
```

<div id="description">
  ## 説明
</div>

W&amp;B アーティファクトのアップロード、ダウンロード、管理を行います。

<div id="subcommands">
  ## サブコマンド
</div>

| コマンド                   | 説明                                        |
| ---------------------- | ----------------------------------------- |
| `wandb artifact cache` | ローカルのアーティファクトキャッシュを管理します。                 |
| `wandb artifact get`   | パスを指定して W&amp;B からアーティファクトをダウンロードします。     |
| `wandb artifact ls`    | W&amp;B の project 内のすべてのアーティファクトを一覧表示します。 |
| `wandb artifact put`   | アーティファクトを W&amp;B にアップロードします。             |
