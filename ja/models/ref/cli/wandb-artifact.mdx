title: "wandb artifact"
---

アーティファクト を操作するコマンド

<div id="usage">
  ## 使用方法
</div>

```bash
wandb artifact COMMAND [ARGS]...
```

<div id="commands">
  ## コマンド
</div>

| コマンド                                                         | 説明                                 |
| :----------------------------------------------------------- | :--------------------------------- |
| [cache](/ja/models/ref/cli/wandb-artifact/wandb-artifact-cache) | アーティファクトキャッシュを操作するコマンド             |
| [get](/ja/models/ref/cli/wandb-artifact/wandb-artifact-get)     | wandb からアーティファクトをダウンロードします         |
| [ls](/ja/models/ref/cli/wandb-artifact/wandb-artifact-ls)       | wandb プロジェクト内のすべてのアーティファクトを一覧表示します |
| [put](/ja/models/ref/cli/wandb-artifact/wandb-artifact-put)     | アーティファクトを wandb にアップロードします         |
