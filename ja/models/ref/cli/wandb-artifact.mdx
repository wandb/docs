アーティファクトを扱うためのコマンド

<div id="usage">
  ## 使い方
</div>

```bash
wandb artifact COMMAND [ARGS]...
```

<div id="commands">
  ## コマンド
</div>

| Command | Description |
| :--- | :--- |
| [cache](/ja/models/ref/cli/wandb-artifact/wandb-artifact-cache) | アーティファクトキャッシュを操作するコマンド |
| [get](/ja/models/ref/cli/wandb-artifact/wandb-artifact-get) | wandb からアーティファクトをダウンロードする |
| [ls](/ja/models/ref/cli/wandb-artifact/wandb-artifact-ls) | wandb プロジェクト内のすべてのアーティファクトを一覧表示する |
| [put](/ja/models/ref/cli/wandb-artifact/wandb-artifact-put) | アーティファクトを wandb にアップロードする |
