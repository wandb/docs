W&amp;B Launch エージェントを起動します。


<div id="usage">
  ## 使用法
</div>

```bash
wandb launch-agent [OPTIONS]
```


<div id="options">
  ## オプション
</div>

| オプション              | 説明                                                                                                                                   |
| :----------------- | :----------------------------------------------------------------------------------------------------------------------------------- |
| `--queue`, `-q`    | エージェントが監視するキューの名。`-q` フラグは複数指定できます。                                                                                                  |
| `--entity`, `-e`   | 使用する entity。デフォルトは現在ログイン中のユーザーです                                                                                                     |
| `--log-file`, `-l` | エージェント内部ログの出力先。標準出力を指定するには `-` を使用します。デフォルトでは、すべてのエージェントログは `wandb/` サブディレクトリ内の `debug.log` に出力され、`WANDB_DIR` が設定されている場合はそちらに出力されます。 |
| `--max-jobs`, `-j` | このエージェントが並列で実行できる Launch ジョブの最大数。デフォルトは 1 です。上限をなくすには `-1` を設定します                                                                    |
| `--config`, `-c`   | 使用するエージェント設定 YAML へのパス                                                                                                               |
| `--verbose`, `-v`  | 詳細な出力を表示します (デフォルト: 0)                                                                                                               |
