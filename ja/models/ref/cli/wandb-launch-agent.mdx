<div id="usage">
  ## 使用法
</div>

```bash
launch-agent [OPTIONS]
```

<div id="description">
  ## 説明
</div>

W&amp;B Launch エージェントを起動します。

<div id="options">
  ## オプション
</div>

| フラグ              | タイプ | 説明                                                                                                                                          |
| ---------------- | --- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| `--queue, -q`    | STR | エージェントが監視するキューの名です。`-q` フラグは複数指定できます。**デフォルト**: None                                                                                        |
| `--entity, -e`   | STR | 使用する entity です。デフォルトでは、現在ログイン中のユーザーが使用されます。**デフォルト**: None                                                                                  |
| `--log-file, -l` | STR | エージェント内部ログの出力先です。標準出力には `-` を使用します。デフォルトでは、すべてのエージェントのログは `wandb/` サブディレクトリ、または設定されている場合は `WANDB_DIR` の `debug.log` に出力されます。**デフォルト**: None |
| `--max-jobs, -j` | STR | このエージェントが並列で実行できる Launch ジョブの最大数です。デフォルトは 1 です。上限を設けない場合は `-1` を設定します。**デフォルト**: None                                                       |
| `--config, -c`   | STR | 使用するエージェント設定 YAML へのパスです。**デフォルト**: None                                                                                                    |
| `--verbose, -v`  | INT | 詳細な出力を表示します。**デフォルト**: 0                                                                                                                    |
