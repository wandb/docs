<div id="usage">
  ## 使用方法
</div>

```bash
leet [PATH] | COMMAND [ARGS]...
```

<div id="description">
  ## 説明
</div>

W&amp;B LEET: 軽量な実験探索ツール。

ローカル環境で W&amp;B の runs を表示するためのターミナル UI。



<div id="subcommands">
  ## サブコマンド
</div>

| コマンド                | 説明                      |
| ------------------- | ----------------------- |
| `wandb leet config` | LEET の設定を編集します。         |
| `wandb leet run`    | LEET の TUI を起動します。      |
| `wandb leet symon`  | スタンドアロンのシステムモニターを起動します。 |
