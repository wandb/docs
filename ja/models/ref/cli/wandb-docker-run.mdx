<div id="usage">
  ## 使い方
</div>

```bash
docker-run [OPTIONS] [DOCKER_RUN_ARGS]...
```

<div id="description">
  ## 説明
</div>

`docker run` をラップし、W&amp;B の環境変数を自動的に挿入します。

以下を挿入しつつ、すべての引数を `docker run` にそのまま渡します。

* WANDB&#95;API&#95;KEY: ログイン済みであれば現在の APIキー を挿入し、
  コンテナーが W&amp;B に対して認証できるようにします。

  * WANDB&#95;DOCKER: 引数からイメージを検出できる場合は、解決されたイメージ ID を挿入し、
    W&amp;B がどの Docker image から run が生成されたかをトラッキングできるようにします。

ホスト上で `nvidia-docker` が検出され、かつ `--runtime` がまだ設定されていない場合は、
`--runtime nvidia` を自動的に追加します。

たとえば、&quot;my-image&quot; コンテナー内で Python スクリプト train.py を実行するには、次のようにします。

```bash
wandb docker-run my-image python train.py
```
