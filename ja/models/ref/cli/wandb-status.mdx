<div id="usage">
  ## 使い方
</div>

```bash
status [OPTIONS]
```

<div id="description">
  ## 説明
</div>

現在のW&amp;B設定を表示します。

ベースURL、APIキー、project、entity、そのほかの最終的に解決された値を含む、現在有効なW&amp;B設定を整形されたJSONとして出力します。

たとえば、現在の設定を表示するには次のようにします。

```bash
wandb status
```

<div id="options">
  ## オプション
</div>

| フラグ          | タイプ      | 説明                           |
| ------------ | -------- | ---------------------------- |
| `--settings` | BOOL フラグ | 現在の設定を表示します。 **デフォルト**: True |
