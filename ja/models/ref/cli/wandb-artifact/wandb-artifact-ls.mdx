wandb プロジェクト内のアーティファクトをすべて一覧表示する


<div id="usage">
  ## 使い方
</div>

```bash
wandb artifact ls PATH [OPTIONS]
```


<div id="arguments">
  ## 引数
</div>

| 引数 | 説明 | 必須 |
| :--- | :--- | :--- |
| `PATH` | 説明はありません | はい |

<div id="options">
  ## オプション
</div>

| オプション | 説明 |
| :--- | :--- |
| `--type`, `-t` | 一覧表示するアーティファクトの種類 |
