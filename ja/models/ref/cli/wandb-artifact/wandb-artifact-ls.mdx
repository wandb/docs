W&B Project 内のすべての Artifacts を一覧表示します。

## 使用法

```bash
wandb artifact ls PATH [OPTIONS]
```

## 引数

| 引数 | 説明 | 必須 |
| :--- | :--- | :--- |
| `PATH` | 説明なし | はい |

## オプション

| オプション | 説明 |
| :--- | :--- |
| `--type`, `-t` | 表示する Artifacts のタイプ |
