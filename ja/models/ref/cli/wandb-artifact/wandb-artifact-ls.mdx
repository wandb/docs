wandb プロジェクト内のすべての Artifacts を一覧表示する


<div id="usage">
  ## 使用方法
</div>

```bash
wandb artifact ls PATH [OPTIONS]
```


<div id="arguments">
  ## 인数
</div>

| 인数     | 説明       | 必須  |
| :----- | :------- | :-- |
| `PATH` | 説明はありません | Yes |

  ## オプション

| :------------- | :----------------- |
