wandb プロジェクト内のすべての Artifacts を一覧表示する

<div id="usage">
  ## 使用方法
</div>

```bash
wandb artifact ls PATH [OPTIONS]
```

<div id="arguments">
  ## 引数
</div>

| 引数 | 説明 | 必須 |
| :--- | :--- | :--- |
| `PATH` | 説明はありません | はい |

<div id="options">
  ## オプション
</div>

| オプション | 説明 |
| :--- | :--- |
| `--type`, `-t` | 一覧表示する Artifacts の種類 |
