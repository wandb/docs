wandb からアーティファクトをダウンロードする


<div id="usage">
  ## 使い方
</div>

```bash
wandb artifact get PATH [OPTIONS]
```


<div id="arguments">
  ## 引数
</div>

| 引数 | 説明 | 必須 |
| :--- | :--- | :--- |
| `PATH` | 説明は利用できません | はい |

<div id="options">
  ## オプション
</div>

| オプション | 説明 |
| :--- | :--- |
| `--root` | アーティファクトをダウンロードするディレクトリ |
| `--type` | ダウンロードするアーティファクトの種類 |
