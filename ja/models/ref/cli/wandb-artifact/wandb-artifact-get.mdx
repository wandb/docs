W&B から Artifacts をダウンロードします。

## Usage

```bash
wandb artifact get PATH [OPTIONS]
```

## Arguments

| 引数 | 説明 | 必須 |
| :--- | :--- | :--- |
| `PATH` | 説明はありません | はい |

## Options

| オプション | 説明 |
| :--- | :--- |
| `--root` | Artifacts をダウンロードする先の ディレクトリー |
| `--type` | ダウンロードする Artifacts のタイプ |
