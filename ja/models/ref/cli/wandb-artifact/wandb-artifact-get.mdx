<div id="usage">
  ## 使い方
</div>

```bash
get [OPTIONS] PATH
```

<div id="description">
  ## 説明
</div>

パスを指定して、W&amp;B からアーティファクトをダウンロードします。

PATH の形式は entity/project/artifact&#95;name:version です。version が
省略されている場合は、&quot;latest&quot; エイリアスを使用します。

たとえば、&quot;team-awesome&quot; entity 配下の &quot;foobar&quot; project から、
&quot;processed-training-set&quot; という名前のアーティファクトの最新バージョンをダウンロードするには、次のようにします。

```bash
wandb artifact get team-awesome/foobar/processed-training-set:latest
```

&quot;processed-training-set&quot; アーティファクトの特定のバージョン (v2) を
ローカルディレクトリ (./data) にダウンロードするには、次のようにします。

```bash
wandb artifact get --root ./data team-awesome/foobar/processed-training-set:v2
```

<div id="arguments">
  ## 引数
</div>

| 名      | デフォルト | タイプ  |
| ------ | ----- | ---- |
| `path` | STR   | True |

<div id="options">
  ## オプション
</div>

| フラグ      | タイプ | 説明                                                                              |
| -------- | --- | ------------------------------------------------------------------------------- |
| `--root` | STR | アーティファクト のダウンロード先ディレクトリです。設定されていない場合は、デフォルトの アーティファクト キャッシュを使用します。**デフォルト**: なし |
| `--type` | STR | 想定される アーティファクト のタイプです。一致しない場合は失敗します。**デフォルト**: なし                               |
