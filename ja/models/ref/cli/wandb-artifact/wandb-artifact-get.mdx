wandb からアーティファクトをダウンロードする


<div id="usage">
  ## 使用方法
</div>

```bash
wandb artifact get PATH [OPTIONS]
```


<div id="arguments">
  ## 인수
</div>

| 인수     | 説明       | 必須  |
| :----- | :------- | :-- |
| `PATH` | 説明はありません | Yes |

<div id="options">
  ## オプション
</div>

| オプション    | 説明                     |
| :------- | :--------------------- |
| `--root` | アーティファクトのダウンロード先ディレクトリ |
| `--type` | ダウンロードするアーティファクトのタイプ   |
