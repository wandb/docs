artifact を wandb にアップロードする


<div id="usage">
  ## 使い方
</div>

```bash
wandb artifact put PATH [OPTIONS]
```


<div id="arguments">
  ## 引数
</div>

| 引数     | 説明   | 必須 |
| :----- | :--- | :- |
| `PATH` | 説明なし | はい |

<div id="options">
  ## オプション
</div>

| オプション                 | 説明                                                      |
| :-------------------- | :------------------------------------------------------ |
| `--name`, `-n`        | プッシュするartifactの名: `project/artifact_name`               |
| `--description`, `-d` | このartifactの説明                                           |
| `--type`, `-t`        | artifactのタイプ (デフォルト: データセット)                            |
| `--alias`, `-a`       | このartifactに適用するエイリアス (デフォルト: [&#39;latest&#39;])        |
| `--id`                | アップロード先のrun。                                            |
| `--resume`            | 現在のディレクトリから直前のrunを再開します。                                |
| `--skip_cache`        | artifactファイルのアップロード中はキャッシュを使用しません。 (デフォルト: False)       |
| `--policy`            | artifactファイルのアップロード時のストレージポリシーを設定します。 (デフォルト: mutable)  |
