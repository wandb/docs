アーティファクトを wandb にアップロードする


<div id="usage">
  ## 使用方法
</div>

```bash
wandb artifact put PATH [OPTIONS]
```


<div id="arguments">
  ## 引数
</div>

| 引数 | 説明 | 必須 |
| :--- | :--- | :--- |
| `PATH` | 説明はありません | 必須 |

<div id="options">
  ## オプション
</div>

| オプション | 説明 |
| :--- | :--- |
| `--name`, `-n` | プッシュするアーティファクトの名前を指定します: project/artifact_name |
| `--description`, `-d` | このアーティファクトの説明を指定します |
| `--type`, `-t` | アーティファクトのタイプを指定します（デフォルト: dataset） |
| `--alias`, `-a` | このアーティファクトに適用するエイリアスを指定します（デフォルト: ['latest']） |
| `--id` | アップロード先の run を指定します。 |
| `--resume` | 現在のディレクトリで最後の run を再開します。 |
| `--skip_cache` | アーティファクトファイルをアップロードするときにキャッシュをスキップします（デフォルト: False）。 |
| `--policy` | アーティファクトファイルをアップロードするときのストレージポリシーを設定します（デフォルト: mutable）。 |
