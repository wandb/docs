wandb にアーティファクトをアップロードする


<div id="usage">
  ## 使用法
</div>

```bash
wandb artifact put PATH [OPTIONS]
```


<div id="arguments">
  ## 인수
</div>

| 인수     | 説明       | 必須  |
| :----- | :------- | :-- |
| `PATH` | 説明はありません | Yes |

<div id="options">
  ## オプション
</div>

| Option                | Description                                             |
| :-------------------- | :------------------------------------------------------ |
| `--name`, `-n`        | プッシュするアーティファクトの名: プロジェクト/artifact&#95;name              |
| `--description`, `-d` | このアーティファクトの説明                                           |
| `--type`, `-t`        | アーティファクトのタイプ (デフォルト: データセット)                            |
| `--alias`, `-a`       | このアーティファクトに適用する별칭 (デフォルト: [&#39;latest&#39;])           |
| `--id`                | アップロード先のrun。                                            |
| `--resume`            | 現在のディレクトリから最後のrunを再開します。                                |
| `--skip_cache`        | アーティファクトファイルのアップロード中は캐싱をスキップします。 (デフォルト: False)         |
| `--policy`            | アーティファクトファイルのアップロード中にストレージポリシーを設定します。 (デフォルト: mutable)  |
