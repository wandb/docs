Artifacts キャッシュから、使用頻度の低いファイルをクリーンアップします。

## Usage（使用法）

```bash
wandb artifact cache cleanup TARGET_SIZE [OPTIONS]
```

## Arguments（引数）

| 引数 | 説明 | 必須 |
| :--- | :--- | :--- |
| `TARGET_SIZE` | 説明なし | はい |

## Options（オプション）

| オプション | 説明 |
| :--- | :--- |
| `--remove-temp` | 一時ファイルを削除する（デフォルト：False） |
