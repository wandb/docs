使用頻度の低いファイルを Artifacts キャッシュから削除する


<div id="usage">
  ## Usage
</div>

```bash
wandb artifact cache cleanup TARGET_SIZE [OPTIONS]
```


<div id="arguments">
  ## 引数
</div>

| 引数            | 説明       | 必須 |
| :------------ | :------- | :- |
| `TARGET_SIZE` | 説明はありません | はい |

<div id="options">
  ## オプション
</div>

| オプション           | 説明                        |
| :-------------- | :------------------------ |
| `--remove-temp` | 一時ファイルを削除 (デフォルト: False)  |
