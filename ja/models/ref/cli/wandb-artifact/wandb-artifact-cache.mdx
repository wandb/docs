  ## 使用法
</div>

```bash
cache [OPTIONS] COMMAND [ARGS]...
```

<div id="description">
  ## 説明
</div>

ローカルのアーティファクト キャッシュを管理します。

ダウンロード済みのartifactファイルをローカルにキャッシュし、不要な再ダウンロードを防ぎます。

サブコマンドを使用して、キャッシュが使用しているディスク容量を確認し、解放します。

<div id="subcommands">
  ## サブコマンド
</div>

| コマンド                           | 説明                               |
| ------------------------------ | -------------------------------- |
| `wandb artifact cache cleanup` | ローカルの アーティファクト キャッシュのサイズを削減します。 |
