<div id="usage">
  ## 使い方
</div>

```bash
online [OPTIONS]
```

<div id="description">
  ## 説明
</div>

W&amp;B Runs のクラウド同期を再び有効にします。

オフライン モードの設定を解除すると、このディレクトリで以降に作成される Runs のデータが
W&amp;B クラウドに同期されます。

以前に実行した `wandb offline` を取り消します。

たとえば、オフラインで作業した後にクラウド同期を再び有効にするには、次のようにします。

```bash
wandb online
```

クラウド同期を再度有効にしてから、トレーニングスクリプトを実行します:

```bash
wandb online && python train.py
```
