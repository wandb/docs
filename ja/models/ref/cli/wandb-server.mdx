---
title: wandb サーバー
---

ローカルの W&B サーバー を操作するための コマンド

## 使用法

```bash
wandb server COMMAND [ARGS]...
```

## コマンド

| コマンド | 説明 |
| :--- | :--- |
| [start](/models/ref/cli/wandb-server/wandb-server-start) | ローカルの W&B サーバー を起動します |
| [stop](/models/ref/cli/wandb-server/wandb-server-stop) | ローカルの W&B サーバー を停止します |
