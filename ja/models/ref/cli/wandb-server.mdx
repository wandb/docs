<div id="description">
  ## 説明
</div>

ローカルのW&amp;B Serverを運用するコマンド

<div id="subcommands">
  ## サブコマンド
</div>

| コマンド                 | 説明                                              |
| -------------------- | ----------------------------------------------- |
| `wandb server start` | Dockerコンテナー内でローカルの W&amp;B Server インスタンスを起動します。 |
| `wandb server stop`  | 実行中のローカル W&amp;B Server を停止します。                 |
