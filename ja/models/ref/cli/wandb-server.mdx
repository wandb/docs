<div id="usage">
  ## 使用方法
</div>

```bash
wandb server COMMAND [ARGS]...
```

<div id="commands">
  ## コマンド
</div>

| コマンド                                                     | 説明                        |
| :------------------------------------------------------- | :------------------------ |
| [start](/ja/models/ref/cli/wandb-server/wandb-server-start) | ローカルのW&amp;B Serverを起動します |
| [stop](/ja/models/ref/cli/wandb-server/wandb-server-stop)   | ローカルのW&amp;B Serverを停止します |
