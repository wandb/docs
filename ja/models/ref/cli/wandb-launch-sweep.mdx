<div id="usage">
  ## 使い方
</div>

```bash
launch-sweep [OPTIONS] [CONFIG]
```

<div id="description">
  ## 説明
</div>

W&amp;B の Launch sweep を実行する (実験的) 。

<div id="arguments">
  ## 引数
</div>

| 名        | デフォルト | タイプ   |
| -------- | ----- | ----- |
| `config` | PATH  | False |

<div id="options">
  ## オプション
</div>

| Flag              | Type | Description                                                                |
| ----------------- | ---- | -------------------------------------------------------------------------- |
| `--queue, -q`     | STR  | sweep をプッシュする先のキュー名。**デフォルト**: None                                        |
| `--project, -p`   | STR  | エージェントが監視する project 名。指定した場合、設定ファイルで渡された project の値を上書きします。**デフォルト**: None |
| `--entity, -e`    | STR  | 使用する entity。デフォルトでは、現在ログインしているユーザーが使用されます。**デフォルト**: None                  |
| `--resume_id, -r` | STR  | 8 文字の sweep ID を指定して Launch sweep を再開します。キューは必須です。**デフォルト**: None          |
| `--prior_run, -R` | STR  | この sweep に追加する既存の run の ID。**デフォルト**: None                                 |
