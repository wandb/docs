W&amp;B Launch スイープを実行します（実験的な機能）。


<div id="usage">
  ## 使い方
</div>

```bash
wandb launch-sweep [CONFIG] [OPTIONS]
```


<div id="arguments">
  ## 引数
</div>

| 引数 | 説明 | 必須 |
| :--- | :--- | :--- |
| `CONFIG` | 説明はありません | 必須ではありません |

<div id="options">
  ## オプション
</div>

| オプション | 説明 |
| :--- | :--- |
| `--queue`, `-q` | スイープを投入するキューの名前 |
| `--project`, `-p` | エージェントが監視するプロジェクト名。指定した場合、設定ファイルで指定されたプロジェクトの値を上書きする |
| `--entity`, `-e` | 使用するエンティティ。デフォルトは現在ログイン中のユーザー |
| `--resume_id`, `-r` | 8文字のスイープ ID を渡して Launch スイープを再開する。キューの指定が必須 |
| `--prior_run`, `-R` | このスイープに追加する既存の run の ID |
