<div id="usage">
  ## 使い方
</div>

```bash
wandb login [KEY] [OPTIONS]
```


<div id="arguments">
  ## 引数
</div>

| 引数    | 説明   | 必須  |
| :---- | :--- | :-- |
| `KEY` | 説明なし | いいえ |

<div id="options">
  ## オプション
</div>

| オプション                  | 説明                                   |
| :--------------------- | :----------------------------------- |
| `--cloud`              | ローカルではなくクラウドにログインします (デフォルト: False)  |
| `--host`, `--base-url` | 特定の W&amp;B インスタンスにログインします           |
| `--relogin`            | すでにログインしている場合でも、再ログインを強制します。         |
| `--verify`             | ログイン認証情報を確認します (デフォルト: False)        |
