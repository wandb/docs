<div id="usage">
  ## 使い方
</div>

```bash
wandb login [KEY] [OPTIONS]
```


<div id="arguments">
  ## 引数
</div>

| 引数 | 説明 | 必須 |
| :--- | :--- | :--- |
| `KEY` | 説明はありません | いいえ |

<div id="options">
  ## オプション
</div>

| オプション | 説明 |
| :--- | :--- |
| `--cloud` | ローカルではなく cloud にログインする（デフォルト: False） |
| `--host`, `--base-url` | 特定の W&amp;B インスタンスにログインする |
| `--relogin` | すでにログインしている場合でも再ログインを強制する |
| `--verify` | ログイン認証情報を検証する（デフォルト: False） |
