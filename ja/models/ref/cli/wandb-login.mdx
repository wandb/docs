<div id="usage">
  ## 使用方法
</div>

```bash
wandb login [KEY] [OPTIONS]
```


<div id="arguments">
  ## 引数
</div>

| 引数 | 説明 | 必須 |
| :--- | :--- | :--- |
| `KEY` | 説明はありません | いいえ |

<div id="options">
  ## オプション
</div>

| オプション | 説明 |
| :--- | :--- |
| `--cloud` | ローカルではなくクラウドにログインします（デフォルト: False） |
| `--host`, `--base-url` | 特定の W&B インスタンスにログインします |
| `--relogin` | すでにログイン済みの場合でも再ログインを強制します。 |
| `--verify` | ログイン用の認証情報を検証します（デフォルト: False） |
