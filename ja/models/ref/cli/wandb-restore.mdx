run のコード、設定、および Docker の状態を復元します。コードが `wandb.save()` や `wandb.init(save_code=True)` で保存されていない場合は、最新のコミットからコードを取得します。


<div id="usage">
  ## 使い方
</div>

```bash
wandb restore RUN [OPTIONS]
```


<div id="arguments">
  ## 引数
</div>

| 引数 | 説明 | 必須 |
| :--- | :--- | :--- |
| `RUN` | 説明はありません | はい |

<div id="options">
  ## オプション
</div>

| オプション | 説明 |
| :--- | :--- |
| `--no-git` | git の状態を復元しない (デフォルト: False) |
| `--branch` | ブランチを作成するか、detached HEAD でチェックアウトするか (デフォルト: True) |
| `--project`, `-p` | アップロード先のプロジェクト。 |
| `--entity`, `-e` | 一覧のスコープとするエンティティ。 |
