run のコード、設定、Docker の状態を復元します。コードが `wandb.save()` または `wandb.init(save_code=True)` で保存されていない場合は、最新のコミットからコードを取得します。


<div id="usage">
  ## 使用方法
</div>

```bash
wandb restore RUN [OPTIONS]
```


<div id="arguments">
  ## 인수
</div>

| 인수    | 説明        | 必須 |
| :---- | :-------- | :- |
| `RUN` | 説明はありません。 | はい |

<div id="options">
  ## オプション
</div>

| オプション             | 説明                                             |
| :---------------- | :--------------------------------------------- |
| `--no-git`        | git の状態を復元しない (デフォルト: False)                   |
| `--branch`        | ブランチを作成するか、detached でチェックアウトするか (デフォルト: True)  |
| `--project`, `-p` | アップロード先のプロジェクト。                                |
| `--entity`, `-e`  | リストの対象とする entity。                              |
