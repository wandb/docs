  ## 使用方法

```bash
stop [OPTIONS]
```

<div id="description">
  ## 説明
</div>

実行中のローカル W&amp;B サーバーを停止します。

`wandb server start` で起動した `wandb-local` という名前の Docker コンテナーを停止します。Docker がインストールされている必要があります。

たとえば、ローカル W&amp;B サーバーを停止するには、次のようにします。

```bash
wandb server stop
```
