ローカルのW&amp;Bサーバーを起動する


<div id="usage">
  ## 使い方
</div>

```bash
wandb server start [OPTIONS]
```


<div id="options">
  ## オプション
</div>

| オプション          | 説明                                           |
| :------------- | :------------------------------------------- |
| `--port`, `-p` | W&amp;Bサーバー をバインドするホストのポート (デフォルト: 8080)  |
| `--env`, `-e`  | wandb/local に渡す環境変数 (デフォルト: [])              |
| `--daemon`     | デーモンモードで実行するかどうか (デフォルト: True)               |
