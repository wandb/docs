ローカルのW&amp;Bサーバーを起動します


<div id="usage">
  ## 使用方法
</div>

```bash
wandb server start [OPTIONS]
```


<div id="options">
  ## オプション
</div>

| オプション          | 説明                                        |
| :------------- | :---------------------------------------- |
| `--port`, `-p` | W&amp;Bサーバーをバインドするホストポート (デフォルト: 8080) |
| `--env`, `-e`  | wandb/local に渡す環境変数 (デフォルト: [])           |
| `--daemon`     | デーモンモードで実行するかどうか (デフォルト: True)            |
