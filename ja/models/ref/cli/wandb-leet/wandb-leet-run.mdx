<div id="usage">
  ## 使い方
</div>

```bash
run [OPTIONS] [PATH]
```

<div id="description">
  ## 説明
</div>

LEET TUI を起動します。

LEET は、オプションの PATH で指定した W&amp;B Run を表示するためのターミナル UI です。

PATH には、.wandb file、.wandb file を含む run ディレクトリ、
または W&amp;B Run の URL を指定できます。
PATH が指定されていない場合、コマンドは最新の run を探します。

<div id="arguments">
  ## 引数
</div>

| 名      | デフォルト | タイプ   |
| ------ | ----- | ----- |
| `path` | STR   | False |

  ## オプション
</div>

| フラグ          | タイプ      | 説明                                  |
| ------------ | -------- | ----------------------------------- |
| `-h, --help` | BOOL フラグ | このメッセージを表示して終了します。 **デフォルト**: False |
