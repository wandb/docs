<div id="usage">
  ## 使用方法
</div>

```bash
wandb sync [PATH] [OPTIONS]
```


<div id="arguments">
  ## 引数
</div>

| 引数 | 説明 | 必須 |
| :--- | :--- | :--- |
| `PATH` | 説明はありません | いいえ |

<div id="options">
  ## Options
</div>

| Option | Description |
| :--- | :--- |
| `--id` | アップロードしたい run。 |
| `--project`, `-p` | アップロードしたいプロジェクト。 |
| `--entity`, `-e` | スコープ対象とするエンティティ。 |
| `--job_type` | 関連する runs をまとめるための run の種類を指定します。 |
| `--sync-tensorboard` | tfevent ファイルを wandb にストリーミングします。 |
| `--include-globs` | 含める glob パターンのカンマ区切りリスト。 |
| `--exclude-globs` | 除外する glob パターンのカンマ区切りリスト。 |
| `--include-online` | オンラインの runs を含めます。 |
| `--include-offline` | オフラインの runs を含めます。 |
| `--include-synced` | 同期済みの runs を含めます。 |
| `--mark-synced` | runs を同期済みとしてマークします (デフォルト: True)。 |
| `--sync-all` | すべての runs を同期します (デフォルト: False)。 |
| `--clean` | 同期済みの runs を削除します (デフォルト: False)。 |
| `--clean-old-hours` | 指定した時間より前に作成された runs を削除します。`--clean` フラグと併用してください (デフォルト: 24)。 |
| `--clean-force` | 確認プロンプトなしで削除を実行します (デフォルト: False)。 |
| `--show` | 表示する run の数 (デフォルト: 5)。 |
| `--append` | run を追加します (デフォルト: False)。 |
| `--skip-console` | コンソールログをスキップします (デフォルト: False)。 |
| `--replace-tags` | 'old_tag1=new_tag1,old_tag2=new_tag2' の形式でタグを置き換えます。 |
