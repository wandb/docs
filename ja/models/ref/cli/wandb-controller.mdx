W&B ローカル Sweep controller を実行します。

## Usage（使用法）

```bash
wandb controller SWEEP_ID [OPTIONS]
```

## Arguments（引数）

| 引数 | 説明 | 必須 |
| :--- | :--- | :--- |
| `SWEEP_ID` | 説明なし | はい |

## Options（オプション）

| オプション | 説明 |
| :--- | :--- |
| `--verbose` | 詳細な出力を表示（デフォルト: False） |
