W&amp;B ローカルスイープコントローラーを実行する


<div id="usage">
  ## 使い方
</div>

```bash
wandb controller SWEEP_ID [OPTIONS]
```


<div id="arguments">
  ## 引数
</div>

| 引数 | 説明 | 必須 |
| :--- | :--- | :--- |
| `SWEEP_ID` | 説明はありません | はい |

<div id="options">
  ## オプション
</div>

| オプション | 説明 |
| :--- | :--- |
| `--verbose` | 詳細な出力を表示（デフォルト: False） |
