W&amp;B のローカル sweep コントローラーを実行する


<div id="usage">
  ## 使用方法
</div>

```bash
wandb controller SWEEP_ID [OPTIONS]
```


<div id="arguments">
  ## 引数
</div>

| 引数         | 説明   | 必須 |
| :--------- | :--- | :- |
| `SWEEP_ID` | 説明なし | はい |

<div id="options">
  ## オプション
</div>

| オプション       | 説明                         |
| :---------- | :------------------------- |
| `--verbose` | 詳細出力を表示します (デフォルト: False)  |
