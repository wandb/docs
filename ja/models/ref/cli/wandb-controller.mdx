W&amp;B のローカル sweep コントローラーを実行する


<div id="usage">
  ## 使用方法
</div>

```bash
wandb controller SWEEP_ID [OPTIONS]
```


<div id="arguments">
  ## 인수
</div>

| 인수         | 説明       | 必須  |
| :--------- | :------- | :-- |
| `SWEEP_ID` | 説明はありません | Yes |

<div id="options">
  ## オプション
</div>

| オプション       | 説明                          |
| :---------- | :-------------------------- |
| `--verbose` | 詳細な出力を表示します (デフォルト: False)  |
