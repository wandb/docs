<div id="usage">
  ## 使用方法

```bash
leet [PATH] | COMMAND [ARGS]...
```

<div id="description">
  ## 説明
</div>

W&amp;B LEET: 軽量な実験探索ツール。

ローカルで W&amp;B Runs を表示するためのターミナル UI です。

<div id="subcommands">
  ## サブコマンド
</div>

| Command                  | Description             |
| ------------------------ | ----------------------- |
| `wandb beta leet config` | LEET の設定を編集します。         |
| `wandb beta leet run`    | LEET の TUI を起動します。      |
| `wandb beta leet symon`  | スタンドアロン システムモニターを起動します。 |
