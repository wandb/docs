LEET TUI を起動します。

PATH には `.wandb` ファイル、run ディレクトリ、または wandb ディレクトリを指定できます。省略した場合は、最新の run を検索します。


<div id="usage">
  ## 使用方法
</div>

```bash
wandb beta leet run [PATH] [OPTIONS]
```
