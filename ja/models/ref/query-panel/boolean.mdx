## Chainable Ops
### <a id="and"></a>`and`

2つの 値 の論理積（ `and` ）を返します。

| 引数 | 説明 |
| :--- | :--- |
| `lhs` | 1つ目のバイナリ 値 |
| `rhs` | 2つ目のバイナリ 値 |

#### 返り値
2つの 値 の論理積

### <a id="or"></a>`or`

2つの 値 の論理和（ `or` ）を返します。

| 引数 | 説明 |
| :--- | :--- |
| `lhs` | 1つ目のバイナリ 値 |
| `rhs` | 2つ目のバイナリ 値 |

#### 返り値
2つの 値 の論理和

### <a id="boolean-not"></a>`boolean-not`

値 の論理否定を返します。

| 引数 | 説明 |
| :--- | :--- |
| `bool` | boolean 値 |

#### 返り値
値 の論理否定

### <a id="boolean-not"></a>`boolean-not`

値 の論理否定を返します。

| 引数 | 説明 |
| :--- | :--- |
| `bool` | boolean 値 |

#### 返り値
値 の論理否定
