<div id="chainable-ops">
  ## チェーン可能な op
</div>

### <a id="and" />`and`

2 つの値の論理積 `and` を返します

| 引数    | 説明        |
| :---- | :-------- |
| `lhs` | 1 つ目の二値 |
| `rhs` | 2 つ目の二値 |

<div id="return-value">
  #### 戻り値
</div>

2 つの値に対する論理積の結果

### <a id="or" />`or`

2 つの値の論理和 (`or`) を返します

| Argument | Description |
| :------- | :---------- |
| `lhs`    | 1 番目の二値    |
| `rhs`    | 2 番目の二値    |

<div id="return-value">
  #### 戻り値
</div>

2 つの値の論理和 (`or`)

### <a id="boolean-not" />`boolean-not`

値の論理否定を返します

| 引数     | 説明  |
| :----- | :-- |
| `bool` | 真偽値 |

<div id="return-value">
  #### 戻り値
</div>

指定された値の論理否定

### <a id="boolean-not" />`boolean-not`

与えられた値の論理否定を返します

| Argument | Description |
| :------- | :---------- |
| `bool`   | 真偽値         |

<div id="return-value">
  #### 戻り値
</div>

指定された値の論理否定
