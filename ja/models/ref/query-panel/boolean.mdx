title: ブール値

<div id="chainable-ops">
  ## チェーン可能な演算
</div>

### <a id="and"></a>`and`

2 つの値の論理積（`and`）を返します

| 引数 | 説明 |
| :--- | :--- |
| `lhs` | 1 つ目のブール値 |
| `rhs` | 2 つ目のブール値 |

#### 戻り値

2 つの値に対する論理演算 `and` の結果

### <a id="or"></a>`or`

2 つの値の論理和を返します

| 引数 | 説明 |
| :--- | :--- |
| `lhs` | 1 つ目のブール値 |
| `rhs` | 2 つ目のブール値 |

#### 戻り値

2 つの値の論理和 (`or`)

### <a id="boolean-not"></a>`boolean-not`

値の論理値を反転して返します

| Argument | Description |
| :--- | :--- |
| `bool` | ブール値 |

#### 戻り値

値の論理否定

### <a id="boolean-not"></a>`boolean-not`

与えられた値の論理否定を返します

| 引数 | 説明 |
| :--- | :--- |
| `bool` | 論理値（boolean） |

#### 戻り値

値の論理否定値

<div id="list-ops">
  ## リスト演算
</div>

### <a id="and"></a>`and`

2 つの値の論理積 `and` を返します

| 引数 | 説明 |
| :--- | :--- |
| `lhs` | 1 つ目のブール値 |
| `rhs` | 2 つ目のブール値 |

#### 戻り値

2 つの値に対する論理演算 `and` の結果

### <a id="or"></a>`or`

2 つの値の論理和（`or`）を返します

| Argument | Description |
| :--- | :--- |
| `lhs` | 1 番目の真偽値 |
| `rhs` | 2 番目の真偽値 |

#### 戻り値

2 つの値の論理和（`or`）

### <a id="boolean-not"></a>`boolean-not`

値の論理否定を返します

| 引数 | 説明 |
| :--- | :--- |
| `bool` | 真偽値 |

  #### Return Value

与えられた値の論理否定を返します

| Argument | Description |
| :--- | :--- |
| `bool` | 真偽値 |

#### 戻り値

指定された値の論理否定
