<div id="chainable-ops">
  ## チェーン可能な演算子
</div>

### <a id="and"></a>`and`

2つの値の論理 AND を返します

| Argument | Description |
| :--- | :--- |
| `lhs` | 1つ目のブール値 |
| `rhs` | 2つ目のブール値 |

#### 戻り値

2 つの値の論理積

### <a id="or"></a>`or`

2 つの値の論理和 (`or`) を返します

| 引数 | 説明 |
| :--- | :--- |
| `lhs` | 1 つ目の二値 |
| `rhs` | 2 つ目の二値 |

#### 戻り値

2 つの値に対する論理演算 `or` の結果

### <a id="boolean-not"></a>`boolean-not`

値の論理否定を返します

| 引数 | 説明 |
| :--- | :--- |
| `bool` | ブール値 |

#### 戻り値

値の論理否定値

### <a id="boolean-not"></a>`boolean-not`

値の論理否定を返します

| 引数 | 説明 |
| :--- | :--- |
| `bool` | ブール値 |

#### 戻り値

値の論理否定

<div id="list-ops">
  ## リスト操作
</div>

### <a id="and"></a>`and`

2 つの値の論理積を返します

| Argument | Description |
| :--- | :--- |
| `lhs` | 1 つ目のブール値 |
| `rhs` | 2 つ目のブール値 |

#### 戻り値

2 つの値に対する論理演算 `and` の結果

### <a id="or"></a>`or`

2 つの値の論理和（`or`）を返します

| Argument | Description |
| :--- | :--- |
| `lhs` | 1 つ目のブール値 |
| `rhs` | 2 つ目のブール値 |

#### 戻り値

2 つの値の論理和（`or`）

### <a id="boolean-not"></a>`boolean-not`

与えられた値の論理否定を返します

| 引数 | 説明 |
| :--- | :--- |
| `bool` | 真偽値 |

<div id="return-value">
  #### Return Value
</div>

値の論理否定の結果

### <a id="boolean-not"></a>`boolean-not`

値の論理否定を返します

| 引数 | 説明 |
| :--- | :--- |
| `bool` | ブール値（真偽値） |

#### 戻り値

与えられた値の論理否定
