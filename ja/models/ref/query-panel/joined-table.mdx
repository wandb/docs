### <a id="joinedtable-rows"></a>`joinedtable-rows`

| Argument | Description |
| :--- | :--- |
| `joinedTable` | _結合テーブル_ |
| `leftOuter` | 右側テーブルに対応する行が存在しない場合でも左側テーブルの行を含めるかどうか |
| `rightOuter` | 左側テーブルに対応する行が存在しない場合でも右側テーブルの行を含めるかどうか |
