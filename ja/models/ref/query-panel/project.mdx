### <a id="project-artifact" />`project-artifact`

指定した[プロジェクト](/ja/models/ref/query-panel/project/)内で、指定された名前の[アーティファクト](/ja/models/ref/query-panel/artifact/)を返します。

| Argument | Description |
| :--- | :--- |
| `project` | [プロジェクト](/ja/models/ref/query-panel/project/) |
| `artifactName` | [アーティファクト](/ja/models/ref/query-panel/artifact/)の名前 |

<div id="return-value">
  #### 戻り値
</div>

指定した名前の、ある[プロジェクト](/ja/models/ref/query-panel/project/)内の[アーティファクト](/ja/models/ref/query-panel/artifact/)

### <a id="project-artifactType" />`project-artifactType`

指定した名前の [artifactType](/ja/models/ref/query-panel/artifact-type) を、指定した [プロジェクト](/ja/models/ref/query-panel/project/) 内から返します。

| Argument | Description |
| :--- | :--- |
| `project` | 対象の [プロジェクト](/ja/models/ref/query-panel/project/) |
| `artifactType` | 取得する [artifactType](/ja/models/ref/query-panel/artifact-type) の名前 |

<div id="return-value">
  #### 戻り値
</div>
