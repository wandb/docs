## Chainable Ops
### <a id="asset-file"></a>`asset-file`

| 引数 | 説明 |
| :--- | :--- |
| `asset` | アセット |

#### 返り値

### <a id="partitionedtable-file"></a>`partitionedtable-file`

| 引数 | 説明 |
| :--- | :--- |
| `partitionedTable` | _partitioned-table_ |

#### 返り値
_partitioned-table_ の _file_

### <a id="partitionedtable-rows"></a>`partitionedtable-rows`

| 引数 | 説明 |
| :--- | :--- |
| `partitionedTable` | 行を取得する対象の _partitioned-table_ |

#### 返り値
