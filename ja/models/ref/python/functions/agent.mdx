### <kbd>関数</kbd> `agent`

```python
agent(
    sweep_id: 'str',
    function: 'Callable | None' = None,
    entity: 'str | None' = None,
    project: 'str | None' = None,
    count: 'int | None' = None,
    forward_signals: 'bool' = False
) → None
```

1 つ以上の sweep エージェントを開始します。

sweep エージェントは `sweep_id` を使用して、どの sweep の一部であるか、どの function を実行するか、また (省略可能で) いくつのエージェントを実行するかを把握します。

**引数:**

* `sweep_id`:  sweep の一意の ID です。sweep ID は W&amp;B CLI または Python SDK によって生成されます。
* `function`:  sweep 設定で指定された &quot;program&quot; の代わりに呼び出す function です。
* `entity`:  sweep によって作成された W&amp;B run の送信先となるユーザー名またはチーム名です。指定する entity がすでに存在していることを確認してください。entity を指定しない場合、run は通常はユーザー名であるデフォルトの entity に送信されます。
* `project`:  sweep から作成された W&amp;B run の送信先となるプロジェクトの名前です。project が指定されていない場合、run は &quot;Uncategorized&quot; というラベルの付いたプロジェクトに送信されます。
* `count`:  試行する sweep 設定の試行回数です。
* `forward_signals`:  エージェントが受信したシグナルを子プロセスに転送するかどうかです。CLI エージェントでのみサポートされます。
