## <kbd>関数</kbd> wandb.teardown

```python
exit_code: 'int | None' = None
```

W&amp;B の処理が完了するまで待機し、リソースを解放します。

`run.finish()` で明示的に終了されなかった Runs を完了し、すべてのデータがアップロードされるまで待機します。

`wandb.setup()` を使用したセッションの最後にこれを呼び出すことを推奨します。これは `atexit` フックで自動的に呼び出されますが、Python&#39;s `multiprocessing` モジュールを使用している場合など、一部の環境では確実に動作しません。
