run のメトリクスは、ユーザー定義の条件を満たします。

<div id="args">
  ## 引数
</div>

* `event_type`:
* `scope`: イベントのスコープ。project または team/org entity である必要があります。
* `filter`: このイベントがトリガーされるために満たす必要がある、run および/またはメトリクスの条件。

<div id="examples">
  ## 例
</div>

メトリクス &quot;my-metric&quot; の直近 5 個の値の平均が 123.45 を超えたときに、project &quot;my-project&quot; 内の任意の run に対してトリガーされるイベントを定義します:

```python
from wandb import Api
from wandb.automations import OnRunMetric, RunEvent

api = Api()
project = api.project(name="my-project")

event = OnRunMetric(
    scope=project,
    filter=RunEvent.metric("my-metric").avg(5).gt(123.45),
)
```
