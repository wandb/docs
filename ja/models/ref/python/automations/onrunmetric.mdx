## <kbd>class</kbd> `OnRunMetric`

run のメトリクスが、ユーザー定義の条件を満たした場合です。

例:
メトリクス &quot;my-metric&quot; の直近 5 個の値の平均が 123.45 を超えたときに、プロジェクト &quot;my-project&quot; 内の任意の run に対してトリガーされるイベントを定義します:

```python
from wandb import Api
from wandb.automations import OnRunMetric, RunEvent

api = Api()
project = api.project(name="my-project")

event = OnRunMetric(
    scope=project,
    filter=RunEvent.metric("my-metric").avg(5).gt(123.45),
)
```

**引数:**

* `event_type` (Literal[RUN&#95;METRIC&#95;THRESHOLD, RUN&#95;METRIC&#95;CHANGE, RUN&#95;METRIC&#95;ZSCORE]):
* `scope` (ProjectScope): イベントのスコープ。プロジェクトである必要があります。
* `filter` (RunMetricFilter): このイベントがトリガーされるために満たす必要がある、run および/またはメトリクスの条件。

**戻り値:**
