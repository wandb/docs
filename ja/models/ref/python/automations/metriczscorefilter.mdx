```python
__init__(
    name: 'str',
    window: 'int' = 30,
    threshold: 'Annotated | Annotated' = 3.0,
    change_dir: 'ChangeDir' = ANY

```

**引数:**

* `name` (str): 観測するメトリクスの名前。

* `threshold` (Union[Annotated, Annotated]): zスコア のしきい値。
