## <kbd>クラス</kbd> wandb.Html

```python
data: 'str | pathlib.Path | TextIO',
inject: 'bool' = True,
data_is_not_path: 'bool' = False
```

W&amp;B に HTML コンテンツをログするための W&amp;B クラスです。

<div id="args">
  ## 引数
</div>

* `data`: 拡張子が &quot;.html&quot; のファイルへのパスを指定する文字列、または生の HTML を含む文字列もしくは IO オブジェクト。
* `inject`: HTML オブジェクトにスタイルシートを追加します。False に設定すると、HTML は変更せずそのまま渡されます。
* `data_is_not_path`: False に設定すると、data はファイルへのパスとして扱われます。
