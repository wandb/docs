## <kbd>class</kbd> `ArtifactTypes`

特定のプロジェクトにおける`ArtifactType`オブジェクトの遅延評価イテレータ。 

### <kbd>プロパティ</kbd> ArtifactTypes.cursor

次に取得するページの先頭を示す不透明なカーソルです。

この値は保存しておき、後続のページネーション付きクエリで `start=` として渡すことで、このページネーターが中断した位置からイテレーションを再開できます。

**戻り値:**

* `str | None`: cursor のプロパティ値です。
