## <kbd>class</kbd> `BetaReport`

BetaReport は、W&amp;B で作成されたレポートに 연결된 クラスです。

レポートの属性 (名、description、user、spec、timestamps) への 액세스や、関連する run やセクションの取得、レポートを HTML として表示するための method を提供します。

**Attributes:**

* `id` (string):  レポートの一意の ID。
* `display_name` (string):  レポートの人間が読める表示名。
* `name` (string):  レポートの名。よりユーザーフレンドリーな名前には `display_name` を使用します。
* `description` (string):  レポートの説明。
* `user` (User):  レポートを作成したユーザー情報 (username、email) を含む dict。
* `spec` (dict):  レポートの spec。
* `url` (string):  レポートの URL。
* `updated_at` (string):  最終更新時刻のタイムスタンプ。
* `created_at` (string):  レポートの作成時刻のタイムスタンプ。
