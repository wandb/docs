### <kbd>プロパティ</kbd> Registry.allow_all_artifact_types

Registry であらゆる種類のアーティファクトが許可されているかどうかを返します。

`True` の場合、任意の種類のアーティファクトを追加できます。`False` の場合、アーティファクトは `artifact_types` に列挙された유형に制限されます。

**戻り値:**

* `bool`: allow&#95;all&#95;artifact&#95;types プロパティの値。

***

### <kbd>プロパティ</kbd> Registry.artifact_types

戻り値 Registry で許可されているアーティファクト 유형。

`allow_all_artifact_types` が `True` の場合、`artifact_types` には、以前に保存されたか、現在 Registry で使用されている 유형 が反映されます。`allow_all_artifact_types` が `False` の場合、アーティファクトは `artifact_types` 内の 유형 に制限されます。

**注記:**

> 以前に保存されたアーティファクト 유형 は削除できません。

**예시:**

```python
import wandb

registry = wandb.Api().create_registry()
registry.artifact_types.append("model")
registry.save()  # 一度保存すると、アーティファクトタイプ `model` は削除できません
registry.artifact_types.append("accidentally_added")
registry.artifact_types.remove(
     "accidentally_added"
)  # タイプは、まだ保存されていない場合にのみ削除できます
```

**戻り値:**

* `AddOnlyArtifactTypesList`: artifact&#95;types プロパティ値。

### <kbd>プロパティ</kbd> Registry.visibility

Registry の公開範囲。

**戻り値:**

* `Literal["organization", "restricted"]`:  公開範囲のレベル。
  * &quot;organization&quot;: 組織内の誰でもこの Registry を閲覧できます。後から UI の設定画面でロールを編集できます。
  * &quot;restricted&quot;: UI 経由で招待されたメンバーのみがこの Registry にアクセスできます。公開共有は無効です。

**戻り値:**

* `Literal['organization', 'restricted']`: 公開範囲プロパティの値。

### <kbd>method</kbd> `Registry.add_members`

```python
add_members(*members: 'User | UserMember | Team | TeamMember | str') → Self
```

この Registry にユーザーまたはチームを追加します。

**引数:**

* `members`:  Registry に追加するユーザーまたはチームです。`User` オブジェクト、`Team` オブジェクト、またはそれらの文字列 ID を受け入れます。

**戻り値:**
必要に応じてさらに method チェーンを行うための、この Registry です。

**例外:**

* `TypeError`:  引数としてメンバーが渡されなかった場合。
* `ValueError`:  ユーザーまたはチームの ID を infer or parse できない場合。

**例:**

```python
import wandb

api = wandb.Api()

# 既存の registry を取得する
registry = api.registry(name="my-registry", organization="my-org")

user1 = api.user(username="some-user")
user2 = api.user(username="other-user")
registry.add_members(user1, user2)

my_team = api.team(name="my-team")
registry.add_members(my_team)
```

### <kbd>クラスメソッド</kbd> `Registry.create`

```python
create(
    client: 'RetryingClient',
    organization: 'str',
    name: 'str',
    visibility: "Literal['organization', 'restricted']",
    description: 'str | None' = None,
    artifact_types: 'list[str] | None' = None
) → Self
```

新しいレジストリを作成します。

レジストリ名は、組織内で一意である必要があります。この関数は `api.create_registry()` を使用して呼び出す必要があります。

**引数:**

* `client`:  GraphQL クライアント。
* `organization`:  組織の名前。
* `name`:  レジストリの名前 (`wandb-registry-` プレフィックスを除く) 。
* `visibility`:  公開範囲 (&#39;organization&#39; または &#39;restricted&#39;) 。
* `description`:  レジストリの任意の説明。
* `artifact_types`:  許可される유형の任意のリスト。

**戻り値:**

* `Registry`:  新しく作成された Registry オブジェクト。

**例外:**

* `ValueError`:  組織内に同じ名前のレジストリがすでに存在する場合、または作成に失敗した場合。

### <kbd>method</kbd> `Registry.remove_members`

```python
remove_members(*members: 'User | UserMember | Team | TeamMember | str') → Self
```

この Registry からユーザーまたはTeamsを削除します。

**引数:**

* `members`:  この Registry から削除するユーザーまたはTeams。`User` オブジェクト、`Team` オブジェクト、またはそれらの文字列 ID を指定できます。

**戻り値:**
必要に応じてさらに method チェーンを行うための、この Registry 自身。

**例外:**

* `TypeError`:  `members` 引数が 1 つも渡されなかった場合。
* `ValueError`:  ユーザーまたはTeamの ID を推論または解析できなかった場合。

**例:**

```python
import wandb

api = wandb.Api()

# 既存の Registry を取得します
registry = api.registry(name="my-registry", organization="my-org")

user1 = api.user(username="some-user")
user2 = api.user(username="other-user")
registry.remove_members(user1, user2)

old_team = api.team(name="old-team")
registry.remove_members(old_team)
```

### <kbd>method</kbd> `Registry.update_member`

```python
update_member(
    member: 'User | UserMember | Team | TeamMember | str',
    role: 'MemberRole | str'
) → Self
```

この Registry 内のメンバー (ユーザーまたはチーム) のロールを更新します。

**引数:**

* `member`:  ロールを更新する対象のユーザーまたはチームです。`User` object、`Team` object、またはそれらの文字列 ID を受け入れます。
* `role`:  メンバーに付与する新しいロールです。次のいずれかを指定できます。
  * &quot;admin&quot;
  * &quot;member&quot;
  * &quot;viewer&quot;
  * &quot;restricted&#95;viewer&quot; (W&amp;B server でサポートされる場合) 

**戻り値:**
必要に応じて、さらに method チェーンを行うためのこの Registry です。

**例外:**

* `ValueError`:  ユーザーまたはチームの ID を推測できない場合。

**例:**
Registry 内のすべてのユーザーを管理者にします ```python
import wandb

api = wandb.Api()

  # 既存のRegistryを取得する
</div>

registry = api.registry(name=&quot;my-registry&quot;, organization=&quot;my-org&quot;)

for member in registry.user&#95;members():
registry.update&#95;member(member.user, role=&quot;admin&quot;)

````
