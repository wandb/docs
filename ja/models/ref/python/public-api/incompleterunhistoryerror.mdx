## <kbd>class</kbd> `IncompleteRunHistoryError`

run の履歴が不完全な場合に発生します。

不完全な履歴とは、まだ parquet ファイルにエクスポートされていないデータがある場合に発生します。通常は、進行中の run が原因です。
