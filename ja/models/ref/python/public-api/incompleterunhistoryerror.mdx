## <kbd>クラス</kbd> wandb.apis.public.IncompleteRunHistoryError

run history が不完全な場合に発生します。

履歴が不完全になるのは、通常、run がまだ実行中で、一部のデータがまだ
parquet ファイルにエクスポートされていないためです。
