## <kbd>class</kbd> `File`

W&amp;B に保存されたファイルです。

W&amp;B に保存された単一ファイルを表します。ファイルのメタデータへのアクセスを含みます。ファイルは特定の run に関連付けられており、テキストファイル、モデルの重み、データセット、可視化、その他のアーティファクトを含めることができます。ファイルのダウンロード、削除、およびファイルプロパティへのアクセスが可能です。

特定の run にログされた特定のファイルを検索するには、辞書内に1つ以上の属性を指定します。以下のキーを使用して検索できます。

* id (str): ファイルを含む run の ID
* name (str): ファイルの名前
* url (str): ファイルへのパス
* direct&#95;url (str): バケット内のファイルへのパス
* sizeBytes (int): ファイルのサイズ (バイト単位) 
* md5 (str): ファイルの md5
* mimetype (str): ファイルの mimetype
* updated&#95;at (str): 最終更新のタイムスタンプ
* path&#95;uri (str): バケット内のファイルへのパス。現在は S3 オブジェクトおよび参照ファイルのみで利用可能です

**引数:**

* `client`:  ファイルを含む run オブジェクト
* `attrs` (dict):  ファイルを定義する属性の辞書
* `run`:  ファイルを含む run オブジェクト 

### <kbd>プロパティ</kbd> File.path_uri

ストレージバケット内のファイルへの URI パスを返します。

**戻り値**

* `str`:  ファイルが S3 に保存されている場合は S3 URI (例: &#39;s3://bucket/path/to/file&#39;) 、参照ファイルの場合は直接アクセス可能な URL、取得できない場合は空文字列を返します。

**戻り値**

* `str`: `path_uri` プロパティの値。

***

### <kbd>プロパティ</kbd> File.size

ファイルのサイズをバイト単位で返します。

**戻り値:**

* `int`: size プロパティの値。

***

delete() → None

### <kbd>method</kbd> `File.download`

```python
download(
    root: 'str' = '.',
    replace: 'bool' = False,
    exist_ok: 'bool' = False,
    api: 'Api | None' = None
) → io.TextIOWrapper
```

wandb serverから、以前runによって保存されたファイルをダウンロードします。

**引数:**

* `root`:  ファイルを保存するローカルディレクトリです。デフォルトは現在の作業ディレクトリ (`"."`) です。
* `replace`:  `True` の場合、ローカルファイルが存在するとダウンロードで上書きされます。デフォルトは `False` です。
* `exist_ok`:  `True` の場合、ファイルがすでに存在していても ValueError を送出せず、`replace=True` でない限り再ダウンロードもしません。デフォルトは `False` です。
* `api`:  指定した場合、ファイルのダウンロードに使用する `Api` インスタンスです。

**例外:**
ファイルがすでに存在し、`replace=False` かつ `exist_ok=False` の場合は `ValueError`。
