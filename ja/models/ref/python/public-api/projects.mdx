## <kbd>class</kbd> wandb.apis.public.Projects

```python
service_api: 'ServiceApi',
entity: 'str',
per_page: 'int' = 50
```

`Project` オブジェクトの遅延イテレーター。

entity が作成して保存したプロジェクトにアクセスするためのイテラブルなインターフェース。

<div id="args">
  ## 引数
</div>

* `service_api`: このコレクションに対して W&amp;B API Call を実行する wandb-core サービスへのインターフェース。
* `entity`: プロジェクトを所有する entity。
* `per_page`: API へのリクエストごとに取得するプロジェクト数。

<div id="examples">
  ## 例
</div>

```python
from wandb.apis.public.api import Api

# この entity に属するプロジェクトを検索する
projects = Api().projects(entity="entity")

# ファイルを反復処理する
for project in projects:
    print(f"Project: {project.name}")
    print(f"- URL: {project.url}")
    print(f"- Created at: {project.created_at}")
    print(f"- Is benchmark: {project.is_benchmark}")
```
