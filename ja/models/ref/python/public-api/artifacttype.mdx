## <kbd>class</kbd> `ArtifactType`

指定したタイプに基づくクエリを満たすアーティファクト オブジェクト。

**引数:**

* `client`:  W&amp;B のクエリに使用する `client` インスタンス。
* `entity`:  プロジェクトを所有する entity (ユーザーまたはチーム) 。
* `project`:  アーティファクト タイプをクエリする対象のプロジェクト名。
* `type_name`:  アーティファクト タイプの名前。
* `attrs`:  ArtifactType を初期化するための省略可能な属性です。省略した場合、オブジェクトは初期化時に W&amp;B から属性を読み込みます。 

### <kbd>property</kbd> ArtifactType.id

Artifacts タイプの一意の識別子。

**戻り値:**

- `str`: id プロパティの値。

---

### <kbd>property</kbd> ArtifactType.name

Artifacts タイプの名前。

**戻り値:**

- `str`: `name` プロパティの値。

---

### <kbd>method</kbd> `ArtifactType.collection`

```python
collection(name: 'str') → ArtifactCollection
```

指定した名前の Artifacts コレクションを取得します。

**引数:**

* `name` (str):  取得する Artifacts コレクションの名前。

***

### <kbd>method</kbd> `ArtifactType.collections`

```python
collections(
    filters: 'Mapping[str, Any] | None' = None,
    order: 'str | None' = None,
    per_page: 'int' = 50
) → ArtifactCollections
```

このアーティファクト タイプに関連付けられているすべてのアーティファクト コレクションを取得します。

**引数:**

- `filters` (dict): クエリに適用するフィルターのマッピング（省略可能）。
 - `order` (str): 結果の並び順を指定する文字列（省略可能）。`order` の先頭に `+` を付けると昇順（デフォルト）になり、`-` を付けると降順になります。デフォルトの並び順は、アーティファクト コレクション ID の降順です。
 - `per_page` (int): 1 ページあたりに取得するアーティファクト コレクション数。デフォルトは 50 です。
