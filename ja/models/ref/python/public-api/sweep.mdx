## <kbd>class</kbd> `Sweep`

sweep に関連付けられた run のセット。

**属性:**

* `runs` (Runs):  run のリスト
* `id` (str):  Sweep ID
* `project` (str):  この sweep が属するプロジェクトの名
* `config` (dict):  sweep の設定を含む辞書
* `state` (str):  sweep の状態。&quot;Finished&quot;、&quot;Failed&quot;、&quot;Crashed&quot;、または &quot;Running&quot; のいずれかになります。
* `expected_run_count` (int):  この sweep で想定される run 数

### <kbd>プロパティ</kbd> Sweep.config

この sweep で使用される設定。

***

### <kbd>プロパティ</kbd> Sweep.entity

sweep に関連付けられた entity。

**戻り値:**

* `str`: entity プロパティの値。

***

### <kbd>プロパティ</kbd> Sweep.expected_run_count

このsweepで想定されるrun数を返します。run数が無限の場合は None を返します。

**戻り値:**

* `int | None`: expected&#95;run&#95;count プロパティの値。

***

### <kbd>プロパティ</kbd> Sweep.name

sweep の名前。

以下の優先順位で、存在する最初の名前を返します。

1. ユーザーが編集した表示名 2. 作成時に設定された名前 3. Sweep ID

***

### <kbd>プロパティ</kbd> Sweep.order

sweep の並び順のキーを返します。

***

### <kbd>プロパティ</kbd> Sweep.path

プロジェクトのパスを返します。

このパスは、entity、プロジェクト名、sweep ID からなるリストです。

***

### <kbd>プロパティ</kbd> Sweep.url

sweep の URL。

sweep の URL は、entity、プロジェクト、文字列 `"sweeps"`、および sweep ID.run&#95;id から生成されます。SaaS ユーザーの場合、形式は `https://wandb.ai/entity/project/sweeps/sweeps_ID` です。

***

### <kbd>プロパティ</kbd> Sweep.username

非推奨です。代わりに `Sweep.entity` を使用してください。

**戻り値:**

* `str`: username プロパティの値。

***
