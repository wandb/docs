## <kbd>class</kbd> `ArtifactCollections`

あるプロジェクト内の、特定のタイプのアーティファクトコレクション。

**引数:**

- `client`:  W&B へのクエリに使用するクライアントインスタンス。 
 - `entity`:  プロジェクトの所有者となるエンティティ（ユーザーまたはチーム）。 
 - `project`:  アーティファクトコレクションをクエリする対象のプロジェクト名。 
 - `type_name`:  コレクションを取得する対象のアーティファクトタイプ名。 
 - `per_page`:  1 ページあたりに取得するアーティファクトコレクション数。デフォルトは 50。
