## <kbd>クラス</kbd> wandb.apis.public.Team

```python
(service_api: 'ServiceApi',
name: 'str',
attrs: 'Mapping[str, Any] | None' = None)
```

W&amp;B のチームを表すクラス。

このクラスは、チームの作成、メンバーの招待、サービスアカウントの管理など、W&amp;B のチームを管理するための method を提供します。チーム の属性を扱うために Attrs を継承します。

<div id="args">
  ## 引数
</div>

* `service_api`: W&amp;B をクエリするために使用するサービス API インスタンス
* `name`: チーム の名
* `attrs`: チーム の属性を格納したオプションの辞書

<div id="examples">
  ## 例
</div>

チームを検索して、メンバーを招待します。

```python
import wandb

api = wandb.Api()
team = api.team("my-team")
team.invite("user@example.com")
```

チームを作成し、サービスアカウントを追加します。

```python
import wandb

api = wandb.Api()
team = api.create_team("my-team")
team.create_service_account("CI service account")
```

### <kbd>メソッド</kbd> Team.create()

```python
api: 'Api',
team: 'str',
admin_username: 'str | None' = None
```

新しいチームを作成します。

<div id="arguments">
  ##### 引数
</div>

* `api`: (`Api`) 使用するapiインスタンス
* `team`: (str) チームの名前
* `admin_username`: (str) チームの管理者ユーザーのユーザー名 (省略可能) 。デフォルトは現在のユーザーです。

### <kbd>メソッド</kbd> Team.create_service_account()

```python
self,
description: 'str'
```

チーム のサービスアカウントを作成します。

<div id="arguments">
  ##### 引数
</div>

* `description`: (str) このサービスアカウントの説明

### <kbd>メソッド</kbd> Team.invite()

```python
self,
username_or_email: 'str',
admin: 'bool' = False
```

ユーザーをチームに招待します。

<div id="arguments">
  ##### 引数
</div>

* `username_or_email`: (str) 招待するユーザーのユーザー名またはメールアドレス。
* `admin`: (bool) このユーザーをチームの管理者にするかどうか。デフォルトは `False` です。
