## <kbd>class</kbd> `Project`

プロジェクトは run の名前空間です。

### <kbd>method</kbd> `Project.__init__`

```python
__init__(
    client: 'RetryingClient',
    entity: 'str',
    project: 'str',
    attrs: 'Mapping[str, Any]'
) → Project
```

**引数:**

* `client`:  W&amp;B API client インスタンス。
* `name` (str):  プロジェクトの名。
* `entity` (str):  プロジェクトを所有する entity の名。

entity に関連付けられた単一のプロジェクト。

**引数:**

* `client`:  W&amp;B のクエリに使用する API client。
* `entity`:  プロジェクトを所有する entity。
* `project`:  クエリするプロジェクトの名。
* `attrs`:  プロジェクトの属性。

### <kbd>プロパティ</kbd> Project.owner

プロジェクトオーナーを User オブジェクトとして返します。

**発生する例外:**

* `ValueError`:  プロジェクトのユーザー情報が見つからない場合。

**戻り値:**

* `public.User`: owner プロパティの値。

***

### <kbd>プロパティ</kbd> Project.path

プロジェクトのパスを返します。パスは entity とプロジェクト名を含むリストです。

**戻り値:**

* `list[str]`: path プロパティの値。

***

### <kbd>プロパティ</kbd> Project.url

プロジェクトの URL を返します。

**戻り値:**

* `str`: URL プロパティの値。

```python
artifacts_types(per_page: 'int' = 50) → public.ArtifactTypes
```

このプロジェクトに関連付けられたすべてのアーティファクトタイプを返します。

***


### <kbd>method</kbd> `Project.collections`

```python
collections(
    filters: 'Mapping[str, Any] | None' = None,
    order: 'str | None' = None,
    per_page: 'int' = 50
) → public.ProjectArtifactCollections
```

このプロジェクトに関連付けられているすべてのアーティファクト コレクションを返します。

**引数:**

* `filters`:  クエリに適用するフィルターを指定するマッピング (省略可) 。
* `order`:  結果の並び順を指定する文字列 (省略可) 。先頭に `+` を付けると昇順 (デフォルト) 、`-` を付けると降順になります。
* `per_page`:  1 ページあたりに取得するアーティファクト コレクションの数。デフォルトは 50 です。

***


### <kbd>method</kbd> `Project.sweeps`

```python
sweeps(per_page: 'int' = 50) → Sweeps
```

このプロジェクト内の sweep のページネーションされたコレクションを返します。

**引数:**

* `per_page`:  API へのリクエストごとに取得する sweep の数。

**戻り値:**
`Sweep` オブジェクトからなるイテラブルなコレクションである `Sweeps` オブジェクト。
