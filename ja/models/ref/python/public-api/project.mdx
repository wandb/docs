## <kbd>class</kbd> `Project`

プロジェクトは run の名前空間を表します。

### <kbd>method</kbd> `Project.__init__`

```python
__init__(
    client: 'RetryingClient',
    entity: 'str',
    project: 'str',
    attrs: 'Mapping[str, Any]'
) → Project
```

**引数:**

* `client`:  W&amp;B API クライアント インスタンス。
* `name` (str):  プロジェクトの名前。
* `entity` (str):  プロジェクトを所有するエンティティ名。

エンティティに関連付けられた単一のプロジェクト。

**引数:**

* `client`:  W&amp;B をクエリするために使用される API クライアント。
* `entity`:  プロジェクトを所有するエンティティ。
* `project`:  クエリ対象のプロジェクトの名前。
* `attrs`:  プロジェクトの属性。

### <kbd>property</kbd> Project.path

プロジェクトのパスを返します。パスは entity 名とプロジェクト名からなるリストです。

### <kbd>property</kbd> Project.url

**戻り値:**

- `str`: URL プロパティの値。

### <kbd>method</kbd> `Project.artifacts_types`

```python
artifacts_types(per_page: 'int' = 50) → public.ArtifactTypes
```

このプロジェクトに関連するすべてのアーティファクト タイプを返します。

***


### <kbd>メソッド</kbd> `Project.sweeps`

```python
sweeps(per_page: 'int' = 50) → Sweeps
```

このプロジェクト内の sweep のページ分割されたコレクションを返します。

**Args:**

* `per_page`:  API へのリクエストごとに取得する sweep の数。

**Returns:**
`Sweeps` オブジェクト。`Sweep` オブジェクトの反復可能なコレクションです。
