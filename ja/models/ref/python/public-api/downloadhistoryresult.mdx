## <kbd>class</kbd> `DownloadHistoryResult`

run の履歴エクスポートをダウンロードした結果。 

**属性:**

- `paths`:  ダウンロードされた履歴ファイルへのパス。 
 - `errors`:  履歴ファイルのダウンロード中に発生したエラーの辞書。 
 - `contains_live_data`:  run にライブデータが含まれているかどうか。 
 - `not yet exported to parquet files`:  ライブデータがまだ parquet ファイルにエクスポートされていないかどうか。
