## <kbd>クラス</kbd> wandb.apis.public.Reports

```python
(service_api: 'ServiceApi',
project: 'Project',
name: 'str | None' = None,
entity: 'str | None' = None,
per_page: 'int' = 50)
```

Reports は、`BetaReport` オブジェクトを遅延的に返すイテレーターです。

<div id="args">
  ## 引数
</div>

* `service_api`: このコレクションに対して W&amp;B API Call を実行する wandb-core サービスへのインターフェース。
* `project`: Reports を取得する対象のプロジェクト。
* `name`: フィルター対象のレポート名。`None` の場合は、すべての Reports を取得します。
* `entity`: プロジェクトの entity 名。デフォルトでは、プロジェクトの entity が使用されます。
* `per_page`: 1 ページあたりに取得する Reports の数 (デフォルトは 50) 。
