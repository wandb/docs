## <kbd>class</kbd> `ArtifactCollection`

関連するアーティファクトのグループを表すアーティファクトコレクション。

**Args:**

* `client`:  W&amp;B へのクエリに使用するクライアントインスタンス。
* `entity`:  プロジェクトを所有する entity (ユーザーまたはチーム) 。
* `project`:  アーティファクトコレクションをクエリする対象のプロジェクト名。
* `name`:  アーティファクトコレクションの名前。
* `type`:  アーティファクトコレクションのタイプ (例: &quot;dataset&quot;, &quot;model&quot;) 。
* `organization`:  該当する場合の省略可能な組織名。
* `attrs`:  アーティファクトコレクションを初期化するための属性マッピング (省略可能) 。指定しない場合、オブジェクトは初期化時に W&amp;B から属性を読み込みます。 

### <kbd>property</kbd> ArtifactCollection.aliases

このコレクションに含まれるすべての Artifacts バージョンに対するエイリアス。 

**戻り値:**

- `list[str]`: aliases プロパティの値。

### <kbd>property</kbd> ArtifactCollection.created_at

Artifacts コレクションの作成日時。

**戻り値:**

- `str`: created_at プロパティの値。

プロジェクトの所有者である entity（ユーザーまたはチーム）。

**Returns:**

- `str`: entity プロパティの値。

### <kbd>property</kbd> ArtifactCollection.id

Artifacts コレクションの一意の識別子。

**Returns:**

- `str`: id プロパティの値。

### <kbd>property</kbd> ArtifactCollection.name

Artifacts コレクションの名前。

**戻り値:**

- `str`: name プロパティの値。

---

### <kbd>property</kbd> ArtifactCollection.project

Artifacts コレクションを含むプロジェクト。

- `str`: `project` プロパティの値。

---

### <kbd>property</kbd> ArtifactCollection.tags

この Artifacts コレクションに関連付けられているタグ。

**戻り値:**

- `list[str]`: `tags` プロパティの値。

---

### <kbd>property</kbd> ArtifactCollection.type

この Artifacts コレクションのタイプを返します。 

---

### <kbd>property</kbd> ArtifactCollection.updated_at

アーティファクトコレクションが最後に更新された日時。

**戻り値:**

- `str | None`: `updated_at` プロパティの値。

---

### <kbd>メソッド</kbd> `ArtifactCollection.artifacts`

```python
artifacts(per_page: 'int' = 50) → Artifacts
```

コレクション内のすべての Artifacts を取得します。

***

### <kbd>method</kbd> `ArtifactCollection.change_type`

```python
change_type(new_type: 'str') → None
```

非推奨です。代わりに、`save` でタイプを直接変更してください。

```python
delete() → None
```

### <kbd>method</kbd> `ArtifactCollection.is_sequence`

```python
is_sequence() → bool
```

アーティファクトコレクションがシーケンスであるかどうかを返します。

---

### <kbd>method</kbd> `ArtifactCollection.save`

```python
save() → None
```
