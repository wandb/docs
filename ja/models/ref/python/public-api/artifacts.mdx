## <kbd>class</kbd> `Artifacts`

あるプロジェクトに関連付けられたアーティファクトバージョンからなるイテラブルなコレクションです。

任意でフィルターを指定して、特定の条件に基づいて結果を絞り込むことができます。

**Args:**

- `client`:  W&B へのクエリに使用するクライアントインスタンス。 
 - `entity`:  プロジェクトを所有するエンティティ（ユーザーまたはチーム）。 
 - `project`:  アーティファクトをクエリする対象のプロジェクト名。 
 - `collection_name`:  クエリ対象のアーティファクトコレクション名。 
 - `type`:  クエリするアーティファクトのタイプ。一般的な例としては「データセット」や「モデル」など。 
 - `filters`:  クエリに適用するフィルターのオプションのマッピング。 
 - `order`:  結果の並び順を指定するオプションの文字列。 
 - `per_page`:  1ページあたりに取得するアーティファクトバージョン数。デフォルトは 50。 
 - `tags`:  タグでアーティファクトをフィルタリングするための、オプションの文字列または文字列のリスト。
