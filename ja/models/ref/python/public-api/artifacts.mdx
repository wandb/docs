## <kbd>class</kbd> `Artifacts`

プロジェクトに関連付けられたアーティファクト バージョンの反復可能なコレクションです。

必要に応じてフィルターを渡して、特定の条件に基づいて結果を絞り込めます。

**引数:**

* `client`:  W&amp;B のクエリに使用する client インスタンスです。
* `entity`:  プロジェクトを所有する entity (ユーザーまたはチーム) です。
* `project`:  アーティファクトをクエリする対象のプロジェクトの名です。
* `collection_name`:  クエリするアーティファクト コレクションの名です。
* `type`:  クエリするアーティファクトのタイプです。一般的な例には &quot;データセット&quot; や &quot;モデル&quot; があります。
* `filters`:  クエリに適用するフィルターの省略可能なマッピングです。
* `order`:  結果の順序を指定するための省略可能な文字列です。
* `per_page`:  1 ページあたりに取得するアーティファクト バージョン数です。デフォルトは 50 です。
* `tags`:  tags でアーティファクトをフィルターするための、省略可能な文字列または文字列の list です。
