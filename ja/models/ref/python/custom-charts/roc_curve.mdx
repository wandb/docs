### <kbd>function</kbd> `roc_curve`

```python
roc_curve(
    y_true: 'Sequence[numbers.Number]',
    y_probas: 'Sequence[Sequence[float]] | None' = None,
    labels: 'list[str] | None' = None,
    classes_to_plot: 'list[numbers.Number] | None' = None,
    title: 'str' = 'ROC Curve',
    split_table: 'bool' = False
) → CustomChart
```

ROC（受信者動作特性）曲線チャートを作成します。

**Args:**

* `y_true`:  目的変数に対する真のクラスラベル（正解ラベル）。形状は (num&#95;samples,) である必要があります。
* `y_probas`:  各クラスに対して予測された確率または決定スコア。形状は (num&#95;samples, num&#95;classes) である必要があります。
* `labels`:  `y_true` 内のクラスインデックスに対応する、人間が判読可能なラベル。たとえば `labels=['dog', 'cat']` の場合、プロット上でクラス 0 は &#39;dog&#39;、クラス 1 は &#39;cat&#39; と表示されます。None の場合、`y_true` の生のクラスインデックスが使用されます。デフォルトは None です。
* `classes_to_plot`:  ROC 曲線に含めるクラスラベルの部分集合。None の場合、`y_true` に含まれるすべてのクラスがプロットされます。デフォルトは None です。
* `title`:  ROC 曲線プロットのタイトル。デフォルトは &quot;ROC Curve&quot; です。
* `split_table`:  テーブルを W&amp;B UI 内で別セクションに分割するかどうか。`True` の場合、テーブルは &quot;Custom Chart Tables&quot; という名前のセクションに表示されます。デフォルトは `False` です。

**Returns:**

* `CustomChart`:  W&amp;B にログできるカスタムチャートオブジェクト。チャートをログするには、`wandb.log()` に渡します。

**Raises:**

* `wandb.Error`:  numpy、pandas、または scikit-learn が見つからない場合に発生します。

**Example:**

```python
import numpy as np
import wandb

# 3つの疾患を対象とした医療診断分類問題をシミュレートする
n_samples = 200
n_classes = 3

# 真のラベル: 各サンプルに「Diabetes」「Hypertension」「Heart Disease」のいずれかを割り当てる
disease_labels = ["Diabetes", "Hypertension", "Heart Disease"]
# 0: Diabetes, 1: Hypertension, 2: Heart Disease
y_true = np.random.choice([0, 1, 2], size=n_samples)

# 予測確率: 各サンプルの合計が1になるよう予測をシミュレートする
y_probas = np.random.dirichlet(np.ones(n_classes), size=n_samples)

# プロットするクラスを指定する（3つの疾患すべてをプロット）
classes_to_plot = [0, 1, 2]

# W&B の run を初期化し、疾患分類の ROC 曲線プロットをログに記録する
with wandb.init(project="medical_diagnosis") as run:
    roc_plot = wandb.plot.roc_curve(
         y_true=y_true,
         y_probas=y_probas,
         labels=disease_labels,
         classes_to_plot=classes_to_plot,
         title="ROC Curve for Disease Classification",
    )
    run.log({"roc-curve": roc_plot})
```
