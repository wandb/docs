## <kbd>関数</kbd> wandb.plot.scatter

```python
table: 'wandb.Table',
x: 'str',
y: 'str',
title: 'str' = '',
split_table: 'bool' = False
```

wandb.Table のデータから散布図を作成します。

<div id="args">
  ## 引数
</div>

* `table`: 可視化するデータを含む W&amp;B 表。
* `x`: x軸に使用する列の名。
* `y`: y軸に使用する列の名。
* `title`: 散布図のタイトル。
* `split_table`: 表を W&amp;B UI の別のセクションに分けて表示するかどうか。`True` の場合、表は &quot;Custom Chart Tables&quot; という名前のセクションに表示されます。デフォルトは `False` です。

<div id="returns">
  ## 戻り値
</div>

`CustomChart`: W&amp;B にログできるカスタムチャートオブジェクト。チャートをログするには、これを `wandb.log()` に渡します。

<div id="examples">
  ## 例
</div>

```python
import math
import random
import wandb

# 時間経過に伴う異なる高度での気温変化をシミュレートする
data = [
    [i, random.uniform(-10, 20) - 0.005 * i + 5 * math.sin(i / 50)]
    for i in range(300)
]

# 高度 (m) と気温 (°C) の列を持つ W&B 表を作成する
table = wandb.Table(data=data, columns=["altitude (m)", "temperature (°C)"])

# W&B run を初期化して散布図をログする
with wandb.init(project="temperature-altitude-scatter") as run:
    # 散布図を作成してログする
    scatter_plot = wandb.plot.scatter(
        table=table,
        x="altitude (m)",
        y="temperature (°C)",
        title="Altitude vs Temperature",
    )
    run.log({"altitude-temperature-scatter": scatter_plot})
```
