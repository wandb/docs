### <kbd>関数</kbd> `line`

```python
line(
    table: 'wandb.Table',
    x: 'str',
    y: 'str',
    stroke: 'str | None' = None,
    title: 'str' = '',
    split_table: 'bool' = False
) → CustomChart
```

カスタマイズ可能な折れ線グラフを作成します。

**引数:**

* `table`:   グラフ用のデータを含む表。
* `x`:  x軸の値の列名。
* `y`:  y軸の値の列名。
* `stroke`:  線を区別するための列名 (例: 線のグループ化) 。
* `title`:  グラフのタイトル。
* `split_table`:  表を W&amp;B UI 内の別のセクションに分けて表示するかどうか。`True` の場合、表は &quot;Custom Chart Tables&quot; という名前のセクションに表示されます。デフォルトは `False` です。

**戻り値:**

* `CustomChart`:  W&amp;B にログできるカスタムグラフオブジェクト。グラフをログするには、これを `wandb.log()` に渡します。
