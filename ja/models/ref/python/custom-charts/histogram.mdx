## <kbd>関数</kbd> wandb.plot.histogram

```python
table: 'wandb.Table',
value: 'str',
title: 'str' = '',
split_table: 'bool' = False
```

W&amp;B 表 からヒストグラムチャートを作成します。

<div id="args">
  ## 引数
</div>

* `table`: ヒストグラム用のデータを含む W&amp;B 表。
* `value`: ビン軸 (x軸) のラベル。
* `title`: ヒストグラムプロットのタイトル。
* `split_table`: 表を W&amp;B UI の別セクションとして分割するかどうか。`True` の場合、表は &quot;Custom Chart Tables&quot; という名前のセクションに表示されます。デフォルトは `False` です。

<div id="returns">
  ## 戻り値
</div>

`CustomChart`: W&amp;B にログできるカスタムチャートオブジェクト。チャートをログするには、これを `wandb.log()` に渡します。

<div id="examples">
  ## 例
</div>

```python
import math
import random
import wandb

# ランダムデータを生成する
data = [[i, random.random() + math.sin(i / 10)] for i in range(100)]

# W&B 表を作成する
table = wandb.Table(
    data=data,
    columns=["step", "height"],
)

# ヒストグラムプロットを作成する
histogram = wandb.plot.histogram(
    table,
    value="height",
    title="My Histogram",
)

# ヒストグラムプロットを W&B にログする
with wandb.init(...) as run:
    run.log({"histogram-plot1": histogram})
```
