  各 Workspace では、その設定にかかわらず、表示できる run は最大 1000 件です。
  </Note> */}

W&amp;B プロジェクトにログされたすべての run は、プロジェクトのサイドバーにある **Runs** タブで確認できます。Runs タブには *Runs table* があり、プロジェクト内のすべての run の詳細が表示されます。Runs table を使って run を比較したり、特定の列で run を並べ替えたり、run をグループに整理したりできます。

<div id="manage-columns">
  ## 列を管理する
</div>

以下のセクションでは、Runs テーブルをどのようにカスタマイズできるかを説明します。

<div id="add-columns">
  ### 列を追加する
</div>

Runs テーブルに列を追加して、プロジェクトに関連付けられているプロパティの表示内容をカスタマイズできます。

Runs テーブルに列を追加するには、次の手順に従います。

1. プロジェクトのサイドバーで **Runs** タブを選択します。
1. run の一覧の上にある **Columns**（横線が 6 本のアイコン）ボタンをクリックします。
1. モーダルウィンドウの **Hidden** セクション内で、プロパティ名を選択します。
1. 列をドラッグして順序を変更します。
1. **Close** をクリックします。

<div id="remove-columns">
  ### 列を削除する
</div>

Runs テーブルで列を削除して、プロジェクトに関連付けられた表示するプロパティをカスタマイズします。

Runs テーブルで列を削除するには、次の手順に従います。

1. プロジェクトサイドバーで **Runs** タブを選択します。
1. run の一覧の上にある **Columns**（横線 6 本）ボタンをクリックします。
1. モーダルの **Visible &amp; Pinned** セクション内で、プロパティ名を選択します。
1. **Close** をクリックします。

<div id="move-columns">
  ### 列を移動する
</div>

Runs テーブルの列を移動するには、次の手順に従います。

1. プロジェクトサイドバーで、**Runs** タブを選択します。
1. 列を左右にドラッグします。

<div id="pin-columns">
  ### 列をピン留めする
</div>

ピン留めされた列は左側に表示され、ピン留めされていない列は Runs タブの右側に表示されます。**Runs** タブで列をピン留めすると、**Workspace** タブでも同じ列がピン留めされます。同様に、**Workspace** タブで列をピン留めすると、**Runs** タブでも同じ列がピン留めされます。

列をピン留めするには、次の手順を実行します。

1. プロジェクトのサイドバーで **Runs** タブに移動します。
1. **Columns**（横線 6 本）ボタンをクリックします。
1. モーダルの **Visible & Pinned** セクションで、ピン留めしたい列名の横にあるピンアイコンをクリックします。

列のピン留めを解除するには、次の手順を実行します。

1. プロジェクトのサイドバーで **Runs** タブに移動します。
1. 列名の上にカーソルを合わせてアクションメニュー `...` をクリックするか、**Columns**（横線 6 本）ボタンをクリックして、ピン留めを解除したい列名の横にあるピンアイコンをクリックします。
1. **Unpin column** をクリックします。

<Note>
W&B は、Runs テーブルでピン留めまたはピン留めを解除した列を、Workspace タブの Runs セレクターに保存します。
</Note>

<div id="hide-columns">
  ### 列を非表示にする
</div>

Runs テーブルで列を非表示にするには、次のようにします。

1. プロジェクト サイドバーで **Runs** タブを選択します。
1. 列名にカーソルを合わせ、表示されるアクションメニュー `...` をクリックします。
1. **Hide column** をクリックします。

現在非表示になっている列をすべて表示するには、**Columns** をクリックします。

<div id="sort-runs-by-column">
  ## 列で run をソートする
</div>

Runs テーブルで、表示されている任意の列を使って run をソートできます。これは、記録された値のうち最良（または最悪）のものを確認したい場合に特に便利です。

表示されている任意の列で run の一覧をソートするには:

1. 列名の上にカーソルを置き、アクション `...` メニューをクリックします。
1. 必要に応じて **Show latest** にカーソルを置きます。ドロップダウンから **latest**、**min**、**max** のいずれかを選択します。
1. **Sort ascending** または **Sort descending** をクリックします。

<Note>
W&B は、Runs テーブルで選択したソート順を Workspace タブの Runs セレクターに保存します。
</Note>

<div id="export-runs-table-to-csv">
  ## run テーブルを CSV にエクスポートする
</div>

すべての run、ハイパーパラメーター、およびサマリー メトリクスを含むテーブルを、Download ボタンで CSV にエクスポートします。

1. プロジェクト サイドバーで **Runs** タブを選択します。
1. run の一覧の上にある **Download**（下向き矢印）ボタンをクリックします。
