  </Note> */}

W&amp;B プロジェクトのすべての run は、プロジェクトのサイドバーの **Runs** タブで確認できます。Runs タブ内には *Runs table* があります。Runs table には、プロジェクト内のすべての run の詳細が表示されます。Runs table を使用して、run を比較し、特定の列で run を並べ替え、run をグループに整理します。

<div id="manage-columns">
  ## 列を管理
</div>

以下のセクションでは、Runs table をカスタマイズする方法について説明します。

<div id="add-columns">
  ### 列を追加
</div>

Runs table に列を追加して、プロジェクトに関連付けられたどのプロパティを表示するかをカスタマイズします。

Runs table に列を追加するには:

1. プロジェクトのサイドバーで、**Runs** タブを選択します。
2. run のリストの上にある **Columns** (6 本の横線) ボタンをクリックします。
3. モーダルの **Hidden** セクション内で、プロパティの**名**を選択します。
4. 列をドラッグして順序を変更します。
5. **Close** をクリックします。

<div id="remove-columns">
  ### 列を削除
</div>

Runs table の列を削除して、プロジェクトに関連付けられたどのプロパティを表示するかをカスタマイズします。

Runs table の列を削除するには:

1. プロジェクトのサイドバーで、**Runs** タブを選択します。
2. run の一覧の上にある **Columns** (6 本の横線) ボタンをクリックします。
3. モーダルの **Visible &amp; Pinned** セクション内で、プロパティの名を選択します。
4. **Close** をクリックします。

<div id="move-columns">
  ### 列を移動する
</div>

Runs table 内の列を移動するには、次の手順に従います。

1. プロジェクトのサイドバーで、**Runs** タブを選択します。
2. 列を左または右にドラッグします。

<div id="pin-columns">
  ### 列をピン留めする
</div>

ピン留めされた列は左側に表示されます。ピン留めされていない列は、Runs タブの右側に表示されます。**Runs** タブで列をピン留めすると、**Workspace** タブでもその列がピン留めされます。同様に、**Workspace** タブで列をピン留めすると、**Runs** タブでもその列がピン留めされます。

列をピン留めするには、次の手順に従います。

1. プロジェクトのサイドバーで、**Runs** タブにアクセスします。
2. **Columns** (横棒 6 本) ボタンをクリックします。
3. モーダルの **Visible &amp; Pinned** セクションで、ピン留めしたい列名の横にあるピンアイコンをクリックします。

列のピン留めを解除するには、次の手順に従います。

1. プロジェクトのサイドバーで、**Runs** タブにアクセスします。
2. 列名にカーソルを合わせてその `...` 작업 메뉴をクリックするか、または **Columns** (横棒 6 本) ボタンをクリックし、ピン留めを解除したい列名の横にあるピンアイコンをクリックします。
3. **Unpin column** をクリックします。

<Note>
  W&amp;B では、Runs tableでピン留めまたはピン留め解除した列が、Workspace タブの Runs セレクターに保持されます。
</Note>

<div id="hide-columns">
  ### 列を非表示にする
</div>

Runs tableで列を非表示にするには:

1. プロジェクトのサイドバーで、**Runs** タブを選択します。
2. 列名にカーソルを合わせ、表示される작업 메뉴 `...` をクリックします。
3. **列を非表示**をクリックします。

現在非表示になっているすべての列を表示するには、**Columns**をクリックします。

<div id="sort-runs-by-column">
  ## 列でRunsを並べ替える
</div>

Runs tableの表示されている任意の列でRunsを並べ替えられます。これは、記録された最良値 (または最悪値) を確認したい場合に特に便利です。

表示されている任意の列でrunsのlistを並べ替えるには:

1. 列名にカーソルを合わせて、その작업 `...` メニューをクリックします。
2. 必要に応じて **Show latest** にカーソルを合わせます。ドロップダウンから、**latest**、**min**、または **max** を選択します。
3. **Sort ascending** または **Sort descending** をクリックします。

</Note>

<div id="export-runs-table-to-csv">
  ## Runs table を CSV にエクスポートする
</div>

ダウンロードボタンを使用して、すべての run、ハイパーパラメーター、サマリーメトリクスの表を CSV にエクスポートします。

1. プロジェクトのサイドバーで、**Runs** タブを選択します。
2. run の一覧の上にある **Download** (下向き矢印) ボタンをクリックします。
