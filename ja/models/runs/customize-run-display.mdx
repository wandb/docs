  ワークスペースでは、設定にかかわらず表示できる Runs は最大 1000 件までです。
  </Note> */}

W&amp;B プロジェクトにログされたすべての Runs は、プロジェクトのサイドバーにある **Runs** タブで確認できます。Runs タブには *Runs table* があります。Runs table には、プロジェクト内のすべての Runs の詳細が表示されます。Runs table を使用すると、Runs の比較、特定の列での並べ替え、グループへの整理を行えます。

<div id="manage-columns">
  ## 列の管理
</div>

以下のセクションでは、Runs table をカスタマイズする方法を説明します。

<div id="add-columns">
  ### 列を追加
</div>

Runs table に列を追加して、プロジェクトに関連付けられたどのプロパティを表示するかをカスタマイズできます。

Runs table に列を追加するには、次の手順に従います。

1. プロジェクトのサイドバーで、**Runs** タブを選択します。
2. Runs の一覧の上にある **Columns** (横線6本) ボタンをクリックします。
3. モーダルの **Hidden** セクションで、プロパティの名を選択します。
4. 列をドラッグして順序を変更します。
5. **Close** をクリックします。

<div id="remove-columns">
  ### 列を削除する
</div>

Runs table から列を削除して、プロジェクトに関連付けられたどのプロパティを表示するかをカスタマイズできます。

Runs table の列を削除するには、次の手順に従います。

1. プロジェクトのサイドバーで、**Runs** タブを選択します。
2. Runs のリストの上にある **Columns** (横線6本) ボタンをクリックします。
3. モーダルの **Visible &amp; Pinned** セクションで、プロパティ名を選択します。
4. **Close** をクリックします。

<div id="move-columns">
  ### 列を移動する
</div>

Runs table 内で列を移動するには、次のようにします。

1. プロジェクトのサイドバーで、**Runs** タブを選択します。
2. 列を左右にドラッグします。

<div id="pin-columns">
  ### 列を固定する
</div>

固定した列は左側に表示されます。固定していない列は **Runs** タブの右側に表示されます。**Runs** タブで列を固定すると、**Workspace** タブでも固定されます。同様に、**Workspace** タブで列を固定すると、**Runs** タブでも固定されます。

列を固定するには、次の手順に従います。

1. プロジェクトのサイドバーで、**Runs** タブにアクセスします。
2. **Columns** (横線6本) ボタンをクリックします。
3. モーダルの **Visible &amp; Pinned** セクションで、固定したい列名の横にあるピンアイコンをクリックします。

列の固定を解除するには、次の手順に従います。

1. プロジェクトのサイドバーで、**Runs** タブにアクセスします。
2. 列名にカーソルを合わせて `...` アクションメニューをクリックするか、**Columns** (横線6本) ボタンをクリックして固定を解除したい列名の横にあるピンアイコンをクリックします。
3. **Unpin column** をクリックします。

<Note>
  W&amp;B は、Runs table で固定または固定解除した列を、Workspace タブの Runs セレクターに保持します。
</Note>

<div id="hide-columns">
  ### 列を非表示にする
</div>

Runs table で列を非表示にするには、次の手順に従います。

1. プロジェクトのサイドバーで、**Runs** タブを選択します。
2. 列名にカーソルを合わせ、表示されたアクションメニュー `...` をクリックします。
3. **Hide column** をクリックします。

現在非表示になっているすべての列を確認するには、**Columns** をクリックします。

<div id="sort-runs-by-column">
  ## 列で Runs を並べ替える
</div>

Runs table の表示中の任意の列で Runs を並べ替えられます。これは、記録された最良値 (または最悪値) を確認したい場合に特に便利です。

表示中の任意の列で Runs の一覧を並べ替えるには、次の手順に従います。

1. 列名にカーソルを合わせ、`...` のアクションメニューをクリックします。
2. 必要に応じて **Show latest** にカーソルを合わせます。ドロップダウンから **latest**、**min**、または **max** を選択します。
3. **Sort ascending** または **Sort descending** をクリックします。

次のアニメーションは、ログされたメトリクスの最大値で Runs を並べ替える方法を示しています。

<Note>

<div id="export-runs-table-to-csv">
  ## Runs tableをCSVにエクスポートする
</div>

ダウンロードボタンを使って、すべてのRuns、ハイパーパラメーター、summaryメトリクスの表をCSVにエクスポートします。

1. プロジェクトのサイドバーで、**Runs** タブを選択します。
2. Runsの一覧の上にある **Download** (下向き矢印) ボタンをクリックします。
