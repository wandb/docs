title: run の ID または名前を見つけてカスタマイズする
description: run の一意の識別子と run 名の確認方法、カスタム run ID の作成方法、run 名のカスタマイズ方法について説明します。
---

W&amp;B の run を初期化すると、W&amp;B はその run に [*run ID* と呼ばれる一意の識別子](/ja/models/runs/run-identifiers#run-id)を割り当てます。各 run には、人間が読みやすくカスタマイズ可能な [一意ではない *run name*](/ja/models/runs/run-identifiers#run-name) もあります。

<div id="run-id">
  ## Run ID
</div>

run の ID は、その run を一意に識別します。デフォルトでは、新しい run を初期化するときに、[ランダムで一意な run ID](#autogenerated-run-ids) が自動的に生成されます。[run を初期化](/ja/models/runs/initialize-run)するときに、[独自の一意な run ID を指定](#create-a-custom-run-id)することもできます。

<div id="find-a-runs-id">
  ### run の ID を確認する
</div>

W&amp;B Python SDK を使ってプログラムから、または W&amp;B App でインタラクティブに run の一意の ID を取得できます。

<Tabs>
  <Tab title="Python SDK">
    run を初期化すると、W&amp;B はターミナルに一意の run ID を出力します。例えば、次のコードスニペットは W&amp;B run を初期化します:

    ```python
    import wandb
    entity = "nico"  # 自分の W&B エンティティに置き換えてください
    project = "awesome-project" 
    with wandb.init(entity=entity, project=project) as run:
        # ここに自分のコードを記述します
    ```

    ターミナルには次のように出力されます:

    ```bash
    wandb: Syncing run earnest-sunset-1
    wandb: ⭐️ View project at https://wandb.ai/nico/awesome-project
    wandb: 🚀 View run at https://wandb.ai/nico/awesome-project/runs/1jx1ud12
    ```

    run の URL の最後の部分（`1jx1ud12`）が一意の run ID です。
  </Tab>

  <Tab title="W&B App">
    W&amp;B App でも run の一意の ID を確認できます:

    1. [W&amp;B App](https://wandb.ai/home) に移動します。
    2. run を初期化したときに指定した W&amp;B プロジェクトに移動します。
    3. プロジェクトのワークスペース内で、**Workspace** または **Runs** タブを選択します。
    4. 表示したい run を選択します。
    5. **Overview** タブを選択します。

    W&amp;B は **Run path** フィールドに run ID を表示します。run path はチーム名、プロジェクト名、run ID で構成されます。一意の ID は run path の最後の部分です。

    例えば、次の画像では、一意の run ID は `9mxi1arc` です:

    <Frame>
      <img src="/images/runs/unique-run-id.png" alt="Run ID の場所" />
    </Frame>
  </Tab>
</Tabs>

<Tip>
  run の一意の ID を使うと、その run の概要ページに W&amp;B App から直接移動できます。次のコードブロックは、run 用の URL パスの形式を示しています:

  ```text title="特定の run 用の W&B App の URL"
  https://wandb.ai/<entity>/<project>/<run-id>
  ```

  山括弧（`< >`）で囲まれた値を、実際の entity、project、run ID の値に置き換えてください。
</Tip>

<div id="create-a-custom-run-id">
  ### カスタム run ID を作成する
</div>

run を初期化するときに、任意の run ID を文字列として `id` パラメータに指定します。

```python
import wandb

with wandb.init(entity="<project>", project="<project>", id="<run-id>") as run:
    # ここにコードを記述
```

<div id="run-name">
  ## Run name
</div>

各runには、人間が読みやすい（ただし一意ではない）run名があります。デフォルトでは、新しいrunを初期化するときにrun名を指定しなければ、W&amp;Bがランダムなrun名を自動生成します。runの名前は、プロジェクトのワークスペース内および[runの**Overview**ページ](#overview-tab)の上部に表示されます。

前の例から続けると、このrunの名前は `glowing-shadows-8` です。

<Frame>
  <img src="/images/runs/unique-run-id.png" alt="Run ID location" />
</Frame>

runを[初期化するとき](/ja/models/runs/run-identifiers#create-a-custom-run-name)に名前を付けることも、後から[名前を変更](/ja/models/runs/run-identifiers#rename-a-run)することもできます。

<div id="create-a-custom-run-name">
  ### カスタム run 名を指定する
</div>

[`wandb.init()`](/ja/models/ref/python/functions/init) メソッドに `name` パラメータを指定して、run の名前を設定します。

```python
import wandb

with wandb.init(entity="<project>", project="<project>", name="<run-name>") as run:
    # ここにコードを記述
```

<div id="rename-a-run">
  ### run の名前を変更する
</div>

Python SDK を使ってプログラムから、または W&amp;B App 上でインタラクティブに、初期化済みの run の名前を変更できます。

<Tabs>
  <Tab title="Python SDK">
    [`wandb.Api.Run`](/ja/models/ref/python/public-api/api#method-api-run) を使用して、W&amp;B にログされた run にアクセスします。このメソッドは [run オブジェクト](/ja/models/ref/python/public-api/run#property-run-name) を返し、run 名を更新するために使用できます。変更を永続化するには `wandb.Api.Run.update()` メソッドを呼び出します。

    山かっこ（`< >`）で囲まれた値を自身の値に置き換えてください。

    ```python
    import wandb

    api = wandb.Api()

    # パスを指定して run にアクセス
    run = api.run(path = "<entity>/<project>/<run-id>")

    # 新しい run 名を指定
    run.name = "<new-run-name>"
    run.update()
    ```
  </Tab>

  <Tab title="W&B App">
    1. 対象の W&amp;B プロジェクトに移動します。
    2. **Workspace** タブまたは **Runs** タブを選択します。
    3. 名前を変更したい run を検索するか、スクロールして探します。
    4. run 名にカーソルを合わせて縦三点リーダーアイコンをクリックし、**Rename run** をクリックします。
    5. run 名を変更するには、**Run name** フィールドを更新します。
    6. **Save** をクリックします。
  </Tab>
</Tabs>

<div id="run-display-name">
  ## run 表示名
</div>

各 run には、ワークスペースごとにカスタマイズできる *run 表示名* もあります。

<Note>
  あるワークスペースで run の表示名を変更しても、その変更が反映されるのはそのワークスペース内だけであり、他のワークスペースやプロジェクトには影響しません。
</Note>

表示名のデフォルト値は run 名と同じです。表示名は、その run の Workspace と runs テーブルに表示されます。

run 表示名を使うと、プロジェクト内の run 自体の名前は変更せずに、そのワークスペースで表示される run 名だけを上書きできます。

<div id="rename-a-runs-display-name">
  ### run の表示名を変更する
</div>

W&amp;B App から run の表示名を変更するには、次の手順を実行します:

1. W&amp;B プロジェクトに移動します。
2. **Workspace** または **Runs** タブを選択します。
3. 名前を変更したい run を検索するか、スクロールして探します。
4. run 名にカーソルを合わせて縦三点リーダーをクリックし、**Rename run** をクリックします。
5. **Display name** フィールドに新しい値を入力します。
6. **Save** をクリックします。

<div id="customize-run-name-truncation">
  ## run 名の省略方法をカスタマイズする
</div>

デフォルトでは、長い run 名は読みやすさのため中央部分が省略されます。run 名の省略方法をカスタマイズするには、次の手順を実行します。

1. Runs の一覧上部にあるアクション `...` メニューをクリックします。
2. **Run name cropping** で、末尾・中央・先頭のいずれを省略するかを選択します。
