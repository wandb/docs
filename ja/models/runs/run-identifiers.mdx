title: run の ID または名を検索してカスタマイズする
description: run の一意の識別子と run 名を検索する方法、カスタム run ID を作成する方法、および run の名をカスタマイズする方法について説明します。

W&amp;B Run を初期化すると、W&amp;B はその run に [*run ID* と呼ばれる一意の識別子](/ja/models/runs/run-identifiers#run-id) を 부여します。各 run には、カスタマイズ可能な、人が判読できる [一意ではない *run 名*](/ja/models/runs/run-identifiers#run-name) もあります。

<div id="run-id">
  ## Run ID
</div>

run ID は、その run を一意に識別します。デフォルトでは、新しい run を初期化するときに、[ランダムで一意な run ID](#autogenerated-run-ids)を W&amp;B が自動的に生成します。ただし、[run を初期化するとき](/ja/models/runs/initialize-run)に[独自の一意な run ID を指定](#create-a-custom-run-id)した場合は、その限りではありません。

<div id="find-a-runs-id">
  ### run の ID を検索する
</div>

W&amp;B Python SDK を使用してプログラムから、または W&amp;B App でインタラクティブに、run の一意の ID を検索できます。

<Tabs>
  <Tab title="Python SDK">
    run を初期化すると、W&amp;B はターミナルに一意の run ID を返します。たとえば、次のコード 스니펫では W&amp;B run を初期化しています。

    ```python
    import wandb
    entity = "nico"  # ご使用の W&B entity に置き換えてください
    project = "awesome-project" 
    with wandb.init(entity=entity, project=project) as run:
        # ここにコードを記述します
    ```

    ターミナルでは、W&amp;B は次を返します。

    ```bash
    wandb: Syncing run earnest-sunset-1
    wandb: ⭐️ View project at https://wandb.ai/nico/awesome-project
    wandb: 🚀 View run at https://wandb.ai/nico/awesome-project/runs/1jx1ud12
    ```

    run URL の最後の部分 (`1jx1ud12`) が一意の run ID です。
  </Tab>

  <Tab title="W&B App">
    W&amp;B App でも run の一意の ID を検索できます。

    1. [W&amp;B App](https://wandb.ai/home) にアクセスする。
    2. run の初期化時に指定した W&amp;B プロジェクトにアクセスする。
    3. プロジェクトの Workspace 内で、**Workspace** タブまたは **Runs** タブを選択します。
    4. 表示したい run を選択します。
    5. **Overview** タブを選択します。

    W&amp;B は **Run path** フィールドに run ID を表示します。run path は、チーム名、プロジェクト名、および run ID で構成されます。一意の ID は run path の最後の部分です。

    たとえば、次の画像では一意の run ID は `9mxi1arc` です。

    <Frame>
      <img src="/images/runs/unique-run-id.png" alt="Run ID の場所" />
    </Frame>
  </Tab>
</Tabs>

<Tip>
  run の一意の ID を使用すると、W&amp;B App 内のその run の概要ページに直接アクセスできます。次のコードブロックは、run の URL パスの形式を示しています。

  ```text title="特定の run の W&B App URL"
  https://wandb.ai/<entity>/<project>/<run-id>
  ```

  山かっこ (`< >`) で囲まれた値を、entity、プロジェクト、および run ID の実際の値に置き換えてください。
</Tip>

<div id="create-a-custom-run-id">
  ### カスタム run ID を作成する
</div>

run を初期化するときに、目的の run ID を文字列として `id` パラメーターに渡します:

```python
import wandb

with wandb.init(entity="<project>", project="<project>", id="<run-id>") as run:
    # ここにコードを記述します
```

<div id="run-name">
  ## run 名
</div>

各 run には、人が読める一意ではない run 名があります。デフォルトでは、新しい run を初期化する際に run 名を指定しないと、W&amp;B がランダムな run 名を生成します。run の名前は、プロジェクトの Workspace 内と、[run の **Overview** ページ](#overview-tab)の上部に表示されます。

前の例を続けると、この run の名前は `glowing-shadows-8` です。

<Frame>
  <img src="/images/runs/unique-run-id.png" alt="run ID の場所" />
</Frame>

run を[初期化するとき](/ja/models/runs/run-identifiers#create-a-custom-run-name)に名前を付けることも、後で[名前を変更する](/ja/models/runs/run-identifiers#rename-a-run)こともできます。

<div id="create-a-custom-run-name">
  ### カスタム run 名を作成する
</div>

[`wandb.init()`](/ja/models/ref/python/functions/init) method に `name` パラメーターを渡して、run の名を指定します。

```python
import wandb

with wandb.init(entity="<project>", project="<project>", name="<run-name>") as run:
    # ここにコードを記述します
```

<div id="rename-a-run">
  ### run の名前を変更する
</div>

Python SDK を使用してプログラムで初期化した後、または W&amp;B App でインタラクティブに、run の名前を変更できます。

<Tabs>
  <Tab title="Python SDK">
    [`wandb.Api.Run`](/ja/models/ref/python/public-api/api#method-api-run) を使用して、W&amp;B にログされた run にアクセスします。この method は、run 名を更新するために使用できる [run object](/ja/models/ref/python/public-api/run#property-run-name) を返します。変更を永続化するには、`wandb.Api.Run.update()` method を呼び出します。

    山かっこ (`< >`) で囲まれた値を、実際の値に置き換えます。

    ```python
    import wandb

    api = wandb.Api()

    # パスで run にアクセスする
    run = api.run(path = "<entity>/<project>/<run-id>")

    # 新しい run 名を指定する
    run.name = "<new-run-name>"
    run.update()
    ```
  </Tab>

  <Tab title="W&B App">
    1. W&amp;B のプロジェクトにアクセスします。
    2. **Workspace** または **Runs** タブを選択します。
    3. 名前を変更する run を検索するか、スクロールして見つけます。
    4. run 名にカーソルを合わせ、三つの点をクリックしてから、**Rename run** をクリックします。
    5. run 名を変更するには、**Run name** フィールドを更新します。
    6. **Save** をクリックします。
  </Tab>
</Tabs>

<div id="run-display-name">
  ## run の表示名
</div>

各 run には、Workspace ごとにカスタマイズできる *run の表示名* もあります。

<Note>
  1 つの Workspace で run の表示名を変更しても、表示名が変わるのはその Workspace だけで、他の Workspace やプロジェクトでは変わりません。
</Note>

表示名のデフォルト値は run 名と同じです。表示名は run の Workspace と Runs 表に表示されます。

run の表示名を使用すると、プロジェクト内の run 名を変更せずに、その Workspace に表示される run 名を上書きできます。

<div id="rename-a-runs-display-name">
  ### run の Display name を変更する
</div>

W&amp;B App で run の Display name を変更します。

1. W&amp;B のプロジェクトにアクセスします。
2. **Workspace** または **Runs** タブを選択します。
3. 名前を変更する run を検索するか、スクロールして見つけます。
4. run 名にカーソルを合わせ、三つの点をクリックしてから **Rename run** をクリックします。
5. **Display name** フィールドに新しい値を指定します。
6. **Save** をクリックします。

<div id="customize-run-name-truncation">
  ## run 名の切り詰めをカスタマイズする
</div>

デフォルトでは、長い run 名は読みやすさのために中央で切り詰められます。run 名の切り詰めをカスタマイズするには、次の手順を実行します。

1. Runs のリストの上部にあるアクション `...` メニューをクリックします。
2. **Run name cropping** を、末尾、中央、または先頭を切り詰めるように設定します。
