---
title: run を停止する
description: W&B Python SDK を使用してプログラムから、または W&B App から手動で run を停止します。
---

W&amp;B Python SDK を使用してプログラムから、または W&amp;B App でインタラクティブに run を停止します。

<Tabs>
  <Tab title="Python SDK">
    1. run を初期化したターミナルまたはコードエディタにアクセスします。
    2. `Ctrl+D` を押して run を停止します。
  </Tab>

  <Tab title="W&B App">
    1. run がログを記録しているプロジェクトにアクセスします。
    2. run selector で停止する run を選択します。
    3. **Overview** タブを選択します。
    4. **State** フィールドの横にある停止ボタンを選択します。

    **State** フィールドの横で、run の状態が `running` から `Killed` に変わります。
  </Tab>
</Tabs>

使用可能な run の状態の一覧については、[State fields](/ja/models/runs/run-states)を参照してください。
