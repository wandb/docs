title: CoreWeave インフラストラクチャアラートの可視化
description: 
---

W&amp;B にログした機械学習実験の実行中に、GPU 障害や温度異常などのインフラストラクチャアラートを確認できます。[W&amp;B run](/ja/models/runs) の実行中、[CoreWeave Mission Control](https://coreweave.com/mission-control) がコンピュートインフラストラクチャを監視します。

<Note>
  この機能はプレビュー版であり、CoreWeave クラスター上でトレーニングしている場合にのみ利用できます。アクセスについては W&amp;B の担当者にお問い合わせください。
</Note>

エラーが発生した場合、CoreWeave はその情報を W&amp;B に送信します。W&amp;B はインフラストラクチャ情報を、プロジェクトのワークスペース内にある run のプロットに表示します。CoreWeave は一部の問題を自動的に解決しようとし、W&amp;B はその情報を run ページに表示します。
