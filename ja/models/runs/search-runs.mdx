---
title: "run を検索する"
description: "プロジェクトの Runs テーブルや Workspace で、特定の run を名前または ID で検索する方法を説明します。"
---

プロジェクトの Runs テーブルまたは Workspace 内の検索ボックスを使用して、特定の[run を名前または ID で検索](/ja/models/runs/run-identifiers)します。

<Note>
  デフォルトでは、検索ボックスは正規表現（RegEx）を使用して、クエリを run 名または ID にマッチさせます。
</Note>

<div id="search-for-runs-by-name-or-id">
  ## run を名前または ID で検索する
</div>

1. プロジェクトのサイドバーから **Runs** タブまたは **Workspace** をクリックします。
2. Runs テーブル上部の検索ボックスをクリックします。
3. 検索する run の名前または run ID を入力します。

<div id="turn-off-regular-expressions-search">
  ## 正規表現検索をオフにする
</div>

1. プロジェクトのサイドバーから **Runs** タブまたは **Workspace** をクリックします。
2. Runs テーブル上部の検索ボックスをクリックします。
3. **RegEx** トグル (.*) がグレーになるようにオフに切り替えます。
