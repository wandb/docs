---
title: "run を検索する"
description: "プロジェクトの Runs 表 または Workspace で、名または ID を使って特定の run を検索する方法を説明します。"
---

プロジェクトの Runs 表 または Workspace 内の検索ボックスを使用して、特定の[名または ID による run](/ja/models/runs/run-identifiers)を検索します。

<Note>
  デフォルトでは、検索ボックスは正規表現 (RegEx) を使用して、run 名または ID に対してクエリを照合します。
</Note>

<div id="search-for-runs-by-name-or-id">
  ## run 名または ID で Runs を検索する
</div>

1. プロジェクトのサイドバーで、**Runs** タブまたは **Workspace** をクリックします。
2. Runs 表の上部にある検索ボックスをクリックします。
3. 検索する run 名または run ID を入力します。

<div id="turn-off-regular-expressions-search">
  ## 正規表現検索をオフにする
</div>

1. プロジェクトのサイドバーで、**Runs** タブまたは **Workspace** をクリックします。
2. Runs 表 の上部にある検索ボックスをクリックします。
3. **RegEx** トグル (.*) をオフにして、灰色の状態にします。
