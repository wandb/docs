description: "W&Bプロジェクトのワークスペースで、個々のrunに割り当てる色をカスタマイズ、ランダム化、リセットします。"
title: runの色をカスタマイズ
---

W&amp;Bでは、プロジェクトで作成した各runに色が自動的に割り当てられます。runのデフォルトの色を変更すると、表やグラフで他のrunと視覚的に区別しやすくなります。プロジェクトのワークスペースをリセットすると、表内のすべてのrunのデフォルトの色を復元できます。

runの色はローカルスコープです。プロジェクトページでは、カスタム色は自分のワークスペースにのみ適用されます。Reportsでは、runのカスタム色はセクション単位でのみ適用されます。同じrunを異なるセクションに表示でき、セクションごとに異なるカスタム色を使用できます。

<div id="edit-default-run-colors">
  ## デフォルトの run の色を編集する
</div>

1. プロジェクトのサイドバーで **Runs** タブをクリックします。
2. **名** 列で、run 名の横にある色のドットをクリックします。
3. カラーパレットまたはカラーピッカーから色を選択するか、16 進コードを入力します。

<Image src="/images/runs/run-color-palette.png" alt="ワークスペースでデフォルトの run の色を編集する" />

<div id="randomize-run-colors">
  ## run の色をランダム化する
</div>

表内のすべての run の色をランダム化するには、次の手順に従います。

1. プロジェクトのサイドバーで **Runs** タブをクリックします。
2. **名** 列ヘッダーにカーソルを合わせ、三つの点 (**...**) をクリックして、ドロップダウンメニューから **run の色をランダム化する** を選択します。

<Note>
  run の表を、並べ替え、フィルター、検索、グループ化などで何らかの形で変更した場合にのみ、run の色をランダム化するオプションを利用できます。
</Note>

<div id="reset-run-colors">
  ## run の色をリセット
</div>

{/* run の色をランダムにするオプションは、表またはセレクターに少なくとも 2 つの run があり、かつビューに何らかの変更（並べ替え、フィルタリング、検索、またはグループ化）を加えている場合にのみ使用できます。 */}

表内のすべての run の色をデフォルトに戻すには、次の手順に従います。

1. プロジェクトのサイドバーで **Runs** タブをクリックします。
2. **名** 列のヘッダーにカーソルを合わせ、三つの点 (**...**) をクリックして、ドロップダウンメニューから **色をリセット** を選択します。

<Image src="/images/runs/reset-run-colors.png" alt="ワークスペースで run の色をリセット" />
