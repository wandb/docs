---
description: 
title: run の色をカスタマイズする
---

W&B は、プロジェクト内で作成した各 run に自動的に色を割り当てます。テーブルやグラフ内でほかの run と視覚的に区別しやすくするために、run のデフォルトカラーを変更できます。テーブル内のすべての run の色を元に戻すには、プロジェクトのワークスペースをリセットしてデフォルトカラーを復元します。

run の色はローカルスコープで管理されます。プロジェクトページでは、カスタムカラーは自分のワークスペースにのみ適用されます。Reports では、run に対するカスタムカラーはセクション単位でのみ適用されます。同じ run を複数のセクションで可視化でき、セクションごとに異なるカスタムカラーを使用できます。

<div id="edit-default-run-colors">
  ## デフォルトの run の色を編集する
</div>

1. プロジェクトのサイドバーから **Runs** タブをクリックします。
2. **Name** 列で run 名の横にあるドットの色をクリックします。
3. カラーパレットまたはカラーピッカーから色を選択するか、hex コードを入力します。

<Image src="/images/runs/run-color-palette.png" alt="プロジェクト Workspace でデフォルトの run の色を編集する" />

<div id="randomize-run-colors">
  ## run の色をランダムに設定する
</div>

テーブル内のすべての run の色をランダムにするには、次のようにします。

1. プロジェクトのサイドバーから **Runs** タブをクリックします。
2. **Name** 列のヘッダーにカーソルを合わせ、省略記号アイコン (**...**) をクリックし、ドロップダウンメニューから **Randomize run colors** を選択します。

<Note>
run の色をランダムにするオプションは、ソート、フィルタ、検索、グループ化などで run テーブルを何らかの形で変更した後にのみ利用できます。
</Note>

<div id="reset-run-colors">
  ## run の色をリセットする
</div>

{/* run の色をランダムにするオプションは、テーブルまたはセレクタに少なくとも 2 つの run があり、かつビューに対して（ソート、フィルタ、検索、グループ化などの）何らかの変更を加えている場合にのみ利用できます。 */}

テーブル内のすべての run の色をデフォルトの色に戻すには、次の手順を実行します。

1. プロジェクトサイドバーで **Runs** タブをクリックします。
2. **Name** 列ヘッダーにカーソルを合わせ、三点アイコン（**...**）をクリックし、ドロップダウンメニューから **Reset colors** を選択します。

<Image src="/images/runs/reset-run-colors.png" alt="プロジェクト Workspace で run の色をリセットする" />
