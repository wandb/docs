description: "W&B プロジェクトのワークスペースで、個々の run に割り当てられた色をカスタマイズ、ランダム化、リセットします。"
title: run の色をカスタマイズする
---

W&amp;B は、プロジェクト内で作成するそれぞれの run に自動的に色を割り当てます。表やグラフ内で他の run と視覚的に区別しやすくするために、run のデフォルトの色を変更できます。ワークスペースをリセットすると、表内のすべての run の色をデフォルトに戻せます。

run の色はローカルスコープで管理されます。プロジェクトページでは、カスタムの色は自分自身のワークスペースにのみ適用されます。Reports では、run のカスタムの色はセクション単位にのみ適用されます。同じ run を複数のセクションで可視化でき、それぞれのセクションで異なるカスタムの色を使用できます。

<div id="edit-default-run-colors">
  ## デフォルトの run の色を変更する
</div>

1. プロジェクトのサイドバーから **Runs** タブをクリックします。
2. **名** 列の run 名の横にある丸印の色をクリックします。
3. カラーパレットまたはカラーピッカーから色を選択するか、16 進数コードを入力します。

<Image src="/images/runs/run-color-palette.png" alt="プロジェクト Workspace でデフォルトの run の色を編集する" />

<div id="randomize-run-colors">
  ## run の色をランダムに設定する
</div>

テーブル内のすべての run の色をランダムにするには、次の手順を実行します。

1. プロジェクトのサイドバーから **Runs** タブをクリックします。
2. **名** 列のヘッダーにカーソルを合わせ、三つの点 (**...**) をクリックし、ドロップダウンメニューから **Randomize run colors** を選択します。

<Note>
  run の色をランダムにするオプションは、ソート、フィルター、検索、グループ化などにより Runs 表を何らかの方法で変更した後にのみ利用できます。
</Note>

<div id="reset-run-colors">
  ## run の色をリセット
</div>

{/* テーブルまたはセレクター内に少なくとも 2 つの run があり、ビューに対してソート、フィルター、検索、グループ化などのいずれかの操作で変更を加えている場合にのみ、run の色をランダムに割り当て直すオプションを利用できます。 */}

表内のすべての run の色をデフォルト設定に戻すには、次の手順を実行します。

1. プロジェクトのサイドバーの **Runs** タブをクリックします。
2. **Name** 列ヘッダーにカーソルを合わせ、三つの点 (**...**) をクリックし、ドロップダウン メニューから **Reset colors** を選択します。

<Image src="/images/runs/reset-run-colors.png" alt="プロジェクトのワークスペースで run の色をリセットする" />
