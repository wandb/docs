description: Expression editor を使って Runs テーブル内の Runs をフィルターする方法を学びます。
title: Runs をフィルターする
---

Runs テーブル の Expression editor を使って、名、state、[tags](#filter-runs-with-tags)、その他のプロパティ기준으로 Runs をフィルターします。

フィルターを追加するときは、まずフィールド (例: tags、timestamp、entity) を選択します。各フィールドには、text、time、ID などの基となるタイプがあります。

表示される演算子の一覧 (例: is、in、≥、within last) は、そのタイプによって異なります。フィールドを選択すると、そのフィールドのタイプに対して有効な演算子だけが UI に表示されます。

<div id="common-operators-by-type">
  ## タイプごとの一般的な演算子
</div>

| フィルターの種類 | フィールド例              | 一般的な演算子                | 使用例                                |
| -------- | ------------------- | ---------------------- | ---------------------------------- |
| Tags     | `tags`              | is, is not, in, not in | `tags is "baseline"`               |
| Time     | `created timestamp` | ≤, ≥, within last      | `created timestamp` ≥ `01/16/2026` |
| String   | `state`             | =, ≠, IN, NOT IN       | `state = "finished"`               |

<Note>
  上の表では、利用可能なフィールドと演算子の一部のみを示しています。Expression editor には、利用可能なすべてのフィールドと演算子が表示されます。
</Note>

<div id="create-a-filter-expression">
  ## フィルター式を作成する
</div>

1. プロジェクトのサイドバーから **Runs** タブにアクセスします。
2. Runs テーブル上部の、じょうごの形をした **Filter** ボタンを選択します。
3. 左から右へ、列名、論理演算子、フィルター値を選択してフィルター式を作成します。
4. 任意で **New Filter** を選択して、追加のフィルターを適用します。

<div id="example-filter-runs-with-tags">
  ### 例：タグで Runs をフィルタリングする
</div>

タグに基づいて Runs をフィルタリングします。

1. プロジェクトのサイドバーから **Runs** タブをクリックします。
2. Runs テーブル上部にある、じょうごのアイコンの **Filter** ボタンをクリックします。
3. 左から右に向かって、最初のドロップダウン メニューで `"Tags"` を選択し、続いて論理演算子を選択します。
4. 2 つ目のドロップダウン メニューから is、is not、in、not in のいずれかを選択します。
5. 3 つ目のドロップダウン メニューで、フィルタリングに使いたいタグ名を入力します。

<div id="default-filters">
  ## デフォルトフィルター
</div>

デフォルトでは、W&amp;B は次のフィルターを提供します。

* **自分の Runs のみを表示**: 現在のユーザーが作成した Runs のみを表示します。
* **クラッシュした Runs を非表示**: `crashed` 状態の Runs を非表示にします。

デフォルトフィルターは、フィルター式エディター内の **New filter** ボタンの下にトグルとして表示されます。

<div id="remove-a-filter">
  ## フィルターを削除する
</div>

Runs テーブルからフィルターを削除するには、次の手順を実行します。

1. Runs テーブルの上にある、じょうごのようなアイコンの **Filter** ボタンをクリックします。
2. 削除したいフィルターの横にある `x` アイコンをクリックします。
