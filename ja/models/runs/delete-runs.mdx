---
title: Runs を削除
description: W&B App を使用して W&B プロジェクトから run を削除する方法を説明します。
---

W&amp;B App を使用して、プロジェクトから 1 つ以上の run を削除します。

1. 削除したい run を含むプロジェクトにアクセスします。
2. **Runs** タブを選択します。
3. 削除したい run の横にあるチェックボックスを選択します。
4. 表の上部にある **Delete** ボタン (ごみ箱アイコン) を選択します。
5. 表示されるドロワーで、**Delete** を選択します。

<Note>
  run ID は、run を削除した後でも再利用できません。再利用しようとすると、その run はエラーとなって失敗します。
</Note>

<Note>
  大量の run を含むプロジェクトでは、検索バーを使用して Regex で削除したい run をフィルタリングすることも、フィルターボタンを使用してステータス、タグ、その他のプロパティに基づいて run をフィルタリングすることもできます。
</Note>

```mermaid
graph TB
    Start([ユーザーが<br/>Run削除を開始]) --> RunSelect[削除するRuns<br/>を選択]
    
    RunSelect --> DeletePrompt{関連するArtifactsを<br/>削除しますか？}
    
    DeletePrompt -->|No| DeleteRunOnly[Runのみ削除<br/><br/>- Runのメタデータが削除される<br/>- Artifactsは引き続き利用可能<br/>- Artifactsには引き続きアクセス可能]
    
    DeletePrompt -->|Yes| CheckArtifacts[関連する<br/>Artifactsを確認]
    
    CheckArtifacts --> HasRegistry{ArtifactsはModel<br/>Registryにリンク済みですか？}
    
    HasRegistry -->|Yes| RegistryWarning[⚠️ 警告<br/><br/>Registryのモデルが削除されます<br/>本番エイリアスに影響があります]
    HasRegistry -->|No| DirectDelete
    
    RegistryWarning --> ConfirmRegistry{Registryのモデル<br/>削除を確認しますか？}
    
    ConfirmRegistry -->|No| DeleteRunOnly
    ConfirmRegistry -->|Yes| DirectDelete[Run + Artifactsを削除<br/><br/>- Runのメタデータが削除される<br/>- Artifactsが完全に削除される<br/>- Registryのリンクが削除される<br/>- 復元不可]
    
    DeleteRunOnly --> PartialEnd([Run削除済み<br/>Artifacts保持])
    DirectDelete --> FullEnd([Run + Artifacts<br/>完全削除済み])

```

<Warning>
  run を削除する際に、関連する Artifacts も削除するよう選択すると、それらの Artifacts は完全に削除され、後で run を復元しても復元できません。これには、Registry にリンクされている Artifacts も含まれます。
</Warning>
