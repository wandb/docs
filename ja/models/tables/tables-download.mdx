すべての W&amp;B Artifacts と同様に、Tables も pandas のデータフレームに変換して、データを簡単にエクスポートできます。

<div id="convert-table-to-artifact">
  ## `table` を `artifact` に変換する
</div>

まず、テーブルをアーティファクトに変換する必要があります。最も簡単なのは、`artifact.get(table, "table_name")` を使用する方法です。

```python
# 新しいテーブルを作成してログに記録する。
with wandb.init() as r:
    artifact = wandb.Artifact("my_dataset", type="dataset")
    table = wandb.Table(
        columns=["a", "b", "c"], data=[(i, i * 2, 2**i) for i in range(10)]
    )
    artifact.add(table, "my_table")
    wandb.log_artifact(artifact)

# 作成したアーティファクトを使用して、テーブルを取得する。
with wandb.init() as r:
    artifact = r.use_artifact("my_dataset:latest")
    table = artifact.get("my_table")
```

<div id="convert-artifact-to-dataframe">
  ## アーティファクトを DataFrame に変換する
</div>

次に、テーブルを DataFrame に変換します。

```python
# 前のコード例から続けて:
df = table.get_dataframe()
```

<div id="export-data">
  ## データをエクスポートする
</div>

これで、`dataframe` がサポートしている任意の方法でデータをエクスポートできます。

```python
# テーブルデータを .csv に変換する
df.to_csv("example.csv", encoding="utf-8")
```
