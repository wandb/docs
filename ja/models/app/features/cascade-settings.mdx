---
title: ワークスペース、セクション、パネルの設定を管理する
---

あるワークスペースページには、ワークスペース、セクション、パネルという 3 つの設定レベルがあります。[Workspace の設定](#workspace-settings)はワークスペース全体に適用されます。[セクションの設定](#section-settings)はセクション内のすべてのパネルに適用されます。[パネルの設定](#panel-settings)は個々のパネルに適用されます。

<div id="workspace-settings">
  ## Workspace 設定
</div>

Workspace 設定は、すべてのセクションおよびそのセクション内のすべてのパネルに適用されます。編集できる Workspace 設定は 2 種類あります。[Workspace layout](#workspace-layout-options) と [Line plots](#line-plots-options) です。**Workspace layout** はワークスペースの構造を決定し、**Line plots** 設定はワークスペース内の折れ線グラフのデフォルト設定を制御します。

このワークスペース全体の構造に適用される設定を編集するには、次の手順を実行します。

1. 自分のプロジェクトのワークスペースに移動します。
2. **New report** ボタンの横にある歯車アイコンをクリックして、Workspace 設定を表示します。
3. ワークスペースのレイアウトを変更するには **Workspace layout** を選択し、ワークスペース内の折れ線グラフのデフォルト設定を構成するには **Line plots** を選択します。

<Frame>
  <img src="/images/app_ui/workspace_settings.png" alt="Workspace 設定の歯車アイコン" />
</Frame>

<Note>
  ワークスペースをカスタマイズした後、*workspace templates* を使用して、同じ設定で新しいワークスペースをすばやく作成できます。詳細は [Workspace templates](/ja/models/track/workspaces/#workspace-templates) を参照してください。
</Note>

<div id="workspace-layout-options">
  ### Workspace レイアウトオプション
</div>

ワークスペースのレイアウトを設定して、ワークスペース全体の構造を定義します。これには、セクション分けのロジックやパネルの配置が含まれます。

<Frame>
  <img src="/images/app_ui/workspace_layout_settings.png" alt="Workspace layout options" />
</Frame>

Workspace レイアウトオプションのページでは、ワークスペースがパネルを自動生成するか、手動で生成するかを確認できます。ワークスペースのパネル生成モードを変更するには、[Panels](/ja/models/app/features/panels/) を参照してください。

次の表では、各 Workspace レイアウトオプションについて説明します。

| Workspace setting | 説明 |
| ----- | ----- |
| **検索時に空のセクションを非表示にする** | パネルを検索するとき、パネルを含まないセクションを非表示にします。|
| **パネルをアルファベット順に並べ替える** | ワークスペース内のパネルをアルファベット順に並べ替えます。 |
| **セクションの構成** | 既存のすべてのセクションとパネルを削除し、新しいセクション名で再構成します。新しく作成されたセクションを、先頭または末尾のプレフィックスでグループ化します。 |

<Note>
  W&amp;B では、末尾のプレフィックスでグループ化するのではなく、先頭のプレフィックスでグループ化してセクションを整理することを推奨します。先頭のプレフィックスでグループ化すると、セクション数を抑えられ、パフォーマンスの向上につながる場合があります。
</Note>

<div id="line-plots-options">
  ### 折れ線プロットのオプション
</div>

**Line plots** Workspace の設定を変更して、ワークスペース内の折れ線プロットに対するグローバルな既定値とカスタムルールを設定します。

<Frame>
  <img src="/images/app_ui/workspace_settings_line_plots.png" alt="Line plot settings" />
</Frame>

**Line plots** 設定内では、主に **Data** と **Display preferences** の 2 つの設定を編集できます。**Data** タブには次の設定があります:

| Line plot setting | Description |
| ----- | ----- |
| **X axis** |  折れ線プロットにおける x 軸のスケールです。x 軸はデフォルトで **Step** に設定されています。利用可能な x 軸オプションの一覧については、次の表を参照してください。 |
| **Range** |  x 軸に表示する最小値と最大値の設定です。 |
| **Smoothing** | 折れ線プロットのスムージングを変更します。スムージングの詳細については、[Smooth line plots](/ja/models/app/features/panels/line-plot/smoothing/) を参照してください。 |
| **Outliers** | デフォルトのプロットの最小値・最大値のスケールから外れ値を除外するように再スケーリングします。 |
| **Point aggregation method** | Data Visualization の表示精度とパフォーマンスを改善します。詳細については、[Point aggregation](/ja/models/app/features/panels/line-plot/sampling/) を参照してください。 |
| **Max number of runs or groups** | 折れ線プロットに表示される run またはグループの数を制限します。 |

**Step** に加えて、x 軸には次のオプションがあります:

| X axis option | Description |
| ------------- | ----------- |
| **Relative Time (Wall)**| プロセス開始からの経過時刻です。たとえば、ある run を開始して、翌日にその run を再開したとします。そのときに何かをログすると、その記録ポイントは 24 時間として記録されます。|
| **Relative Time (Process)** | 実行中のプロセス内での経過時刻です。たとえば、ある run を開始して 10 秒間実行したあと、翌日にその run を再開したとします。そのポイントは 10 秒として記録されます。 |
| **Wall Time** | グラフ上の最初の run の開始から経過した分数です。 |
| **Step** | `wandb.Run.log()` を呼び出すたびにインクリメントされます。|

<Note>
  個々の折れ線プロットを編集する方法については、Line plots の [Edit line panel settings](/ja/models/app/features/panels/line-plot/#edit-line-panel-settings) を参照してください。
</Note>

**Display preferences** タブ内では、次の設定を切り替えることができます:

| Display preference | Description |
| ----- | ----- |
| **Remove legends from all panels** | すべてのパネルから凡例を削除します |
| **Display colored run names in tooltips** | ツールチップ内に色付きのテキストとして run 名を表示します |
| **Only show highlighted run in companion chart tooltip** | チャートのツールチップにハイライトされた run のみを表示します |
| **Number of runs shown in tooltips** | ツールチップに表示する run の数を設定します |
| **Display full run names on the primary chart tooltip**| プライマリチャートのツールチップに run の完全な名前を表示します |

<div id="section-settings">
  ## セクション設定
</div>

セクション設定は、そのセクション内のすべてのパネルに適用されます。ワークスペースのセクション内では、パネルのソート、並べ替え、セクション名の変更ができます。

セクションの右上にある三点リーダー（**...**）アイコンを選択して、セクション設定を変更します。

<Frame>
  <img src="/images/app_ui/section_settings.png" alt="セクション設定メニュー" />
</Frame>

ドロップダウンから、セクション全体に適用される次の設定を編集できます。

| セクション設定 | 説明 |
| ----- | ----- |
| **Rename a section** | セクション名を変更します |
| **Sort panels A-Z** | セクション内のパネルをアルファベット順にソートします |
| **Rearrange panels** | セクション内のパネルを選択してドラッグし、任意の順序に並べ替えます |

次のアニメーションは、セクション内でパネルを並べ替える方法を示しています。

<Frame>
  <img src="/images/app_ui/rearrange_panels.gif" alt="パネルの並べ替え" />
</Frame>

<Note>
  前の表で説明した設定に加えて、**Add section below**、**Add section above**、**Delete section**、**Add section to report（レポートにセクションを追加）** など、ワークスペース内でのセクションの表示方法も編集できます。
</Note>

<div id="panel-settings">
  ## パネル設定
</div>

個々のパネルの設定をカスタマイズして、同じプロット上で複数のラインを比較したり、カスタム軸を設定したり、ラベルを変更したりできます。パネルの設定を編集するには、次の手順に従います。

1. 編集したいパネルの上にマウスカーソルを合わせます。
2. 表示される鉛筆アイコンを選択します。

<Frame>
  <img src="/images/app_ui/panel_settings.png" alt="パネル編集アイコン" />
</Frame>

3. 表示されるモーダル内で、パネルのデータ、表示設定などに関連する各種設定を編集できます。

<Frame>
  <img src="/images/app_ui/panel_settings_modal.png" alt="パネル設定モーダル" />
</Frame>

パネルに適用できる設定の一覧については、[ラインパネル設定の編集](/ja/models/app/features/panels/line-plot/#edit-line-panel-settings)を参照してください。
