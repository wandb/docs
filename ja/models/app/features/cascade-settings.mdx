特定の Workspace ページには、Workspace、セクション、パネルという 3 つの異なる設定レベルがあります。[Workspace settings](#workspace-settings) は Workspace 全体に適用されます。[セクション設定](#section-settings) はセクション内のすべてのパネルに適用されます。[パネル設定](#panel-settings) は個々のパネルに適用されます。

<div id="workspace-settings">
  ## Workspace settings
</div>

Workspace settings は、すべてのセクションと、その中にあるすべてのパネルに適用されます。編集できる Workspace settings には、[Workspace layout](#workspace-layout-options) と [折れ線グラフ](#line-plots-options) の 2 種類があります。**Workspace layout** はワークスペースの構造を決定し、**折れ線グラフ** の設定はワークスペース内の折れ線グラフのデフォルト設定を制御します。

このワークスペース全体の構造に適用される設定を編集するには、次の手順に従います。

1. プロジェクトのワークスペースにアクセスします。
2. **New report** ボタンの横にある歯車アイコンをクリックして、Workspace settings を表示します。
3. **Workspace layout** を選択してワークスペースの Layout を変更するか、**折れ線グラフ** を選択してワークスペース内の折れ線グラフのデフォルト設定を構成します。

<Frame>
  <img src="/images/app_ui/workspace_settings.png" alt="Workspace settings の歯車アイコン" />
</Frame>

<Note>
  ワークスペースをカスタマイズしたら、*Workspace templates* を使用して同じ設定の新しいワークスペースをすばやく作成できます。[Workspace templates](/ja/models/track/workspaces/#workspace-templates) を参照してください。
</Note>

<div id="workspace-layout-options">
  ### Workspace layout オプション
</div>

Workspace の全体構造を定義するには、Workspace layout を設定します。これには、セクションの分け方やパネルの構成が含まれます。

<Frame>
  <img src="/images/app_ui/workspace_layout_settings.png" alt="Workspace layout オプション" />
</Frame>

Workspace layout オプションページでは、Workspace でパネルを自動生成するか手動で生成するかを確認できます。Workspace のパネル生成モードを調整するには、[Panels](/ja/models/app/features/panels/)を参照してください。

この表では、各 Workspace layout オプションについて説明します。

| Workspace setting      | Description                                                                   |
| ---------------------- | ----------------------------------------------------------------------------- |
| **検索時に空のセクションを非表示にする** | パネルを検索するときに、パネルを含まないセクションを非表示にします。                                            |
| **パネルをアルファベット順に並べ替える** | Workspace 内のパネルをアルファベット順に並べ替えます。                                              |
| **セクションの構成**           | 既存のすべてのセクションとパネルを削除し、新しいセクション名で再作成します。再作成されたセクションは、最初または最後のプレフィックスでグループ化されます。 |

<Note>
  W&amp;B では、最後のプレフィックスでグループ化するよりも、最初のプレフィックスでセクションをグループ化することを推奨しています。最初のプレフィックスでグループ化すると、セクション数を減らせるため、パフォーマンスの向上につながる場合があります。
</Note>

<div id="line-plots-options">
  ### 折れ線グラフのオプション
</div>

**折れ線グラフ** のWorkspace設定を変更すると、Workspace 内の折れ線グラフに対してグローバルなデフォルトとカスタムルールを設定できます。

<Frame>
  <img src="/images/app_ui/workspace_settings_line_plots.png" alt="折れ線グラフの設定" />
</Frame>

**折れ線グラフ** 設定では、主に **Data** と **Display preferences** の 2 つを編集できます。**Data** タブには、次の設定があります。

| 折れ線グラフの設定           | 説明                                                                                                              |
| ------------------- | --------------------------------------------------------------------------------------------------------------- |
| **X 軸**             | 折れ線グラフの x-axis のスケールです。x-axis はデフォルトで **Step** に設定されています。x-axis オプションの一覧は次の表を参照してください。                          |
| **範囲**              | x-axis に表示する最小値と最大値の設定です。                                                                                       |
| **スムージング**          | 折れ線グラフのスムージングを変更します。スムージングの詳細については、[折れ線グラフのスムージング](/ja/models/app/features/panels/line-plot/smoothing/) を参照してください。 |
| **外れ値**             | デフォルトのプロットの最小スケールと最大スケールから外れ値を除外するように再スケーリングします。                                                                |
| **ポイント集約 method**   | Data Visualization の精度とパフォーマンスを向上させます。詳細は [ポイント集約](/ja/models/app/features/panels/line-plot/sampling/) を参照してください。  |
| **run またはグループの最大数** | 折れ線グラフに表示する run またはグループの数を制限します。                                                                                |

**Step** 以外にも、x-axis には次のオプションがあります。

| X 軸オプション                    | 説明                                                                                                  |
| --------------------------- | --------------------------------------------------------------------------------------------------- |
| **Relative Time (Wall)**    | プロセスの開始からの Timestamp です。たとえば、run を開始し、その run を翌日に Resume したとします。その後何かをログすると、記録されるポイントは 24 時間になります。  |
| **Relative Time (Process)** | 実行中のプロセス内の Timestamp です。たとえば、run を開始して 10 秒間継続させたとします。翌日にその run を Resume すると、そのポイントは 10 秒として記録されます。 |
| **Wall Time**               | グラフ上の最初の run の開始から経過した分数です。                                                                         |
| **Step**                    | `wandb.Run.log()` を call するたびに増加します。                                                                |

<Note>
  個別の折れ線グラフを編集する方法については、折れ線グラフ の [折れ線パネル設定を編集する](/ja/models/app/features/panels/line-plot/#edit-line-panel-settings) を参照してください。
</Note>

**Display preferences** タブでは、次の設定を切り替えられます。

| 表示設定                                            | 説明                                    |
| ----------------------------------------------- | ------------------------------------- |
| **すべてのパネルから凡例を削除**                              | パネルの凡例を削除します                          |
| **ツールチップに色付きの run 名を表示**                        | ツールチップ内で run を色付きのテキストとして表示します        |
| **companion chart のツールチップにはハイライトされた run のみを表示** | chart のツールチップには、ハイライトされた run のみを表示します |
| **ツールチップに表示する run 数**                           | ツールチップに run 数を表示します                   |
| **プライマリ chart のツールチップに完全な run 名を表示**            | chart のツールチップに run の完全な名前を表示します       |

<div id="section-settings">
  ## セクション設定
</div>

セクション設定は、そのセクション内のすべてのパネルに適用されます。Workspace のセクションでは、パネルの並べ替えや再配置、セクション名の変更を行えます。

セクション設定を変更するには、セクションの右上にある三つの点 (**...**) を選択します。

<Frame>
  <img src="/images/app_ui/section_settings.png" alt="セクション設定メニュー" />
</Frame>

ドロップダウンから、セクション全体に適用される以下の設定を編集できます。

| セクション設定            | 説明                                    |
| ------------------ | ------------------------------------- |
| **セクション名を変更**      | セクションの名前を変更します                        |
| **パネルを A-Z で並べ替え** | セクション内のパネルをアルファベット順に並べ替えます            |
| **パネルを再配置**        | セクション内のパネルを選択してドラッグし、パネルの順序を手動で並べ替えます |

次のアニメーションは、セクション内でパネルを再配置する方法を示しています。

<Frame>
  <img src="/images/app_ui/rearrange_panels.gif" alt="パネルの再配置" />
</Frame>

<Note>
  上の表で説明した設定に加えて、**Add section below**、**Add section above**、**Delete section**、**Add section to report** など、Workspace でのセクションの表示に関する設定も編集できます。
</Note>

<div id="panel-settings">
  ## パネル設定
</div>

個々のパネル設定をカスタマイズすると、同じプロット上で複数の線を比較したり、カスタム軸を設定したり、ラベル名を変更したりできます。パネル設定を編集するには、次の手順に従います。

1. 編集するパネルにマウスカーソルを合わせます。
2. 表示された鉛筆アイコンを選択します。

<Frame>
  <img src="/images/app_ui/panel_settings.png" alt="パネル編集アイコン" />
</Frame>

3. 表示されるモーダルで、パネルのデータや表示設定などを編集できます。

<Frame>
  <img src="/images/app_ui/panel_settings_modal.png" alt="パネル設定モーダル" />
</Frame>

パネルに適用できる設定の一覧については、[折れ線パネル設定を編集する](/ja/models/app/features/panels/line-plot/#edit-line-panel-settings)を参照してください。
