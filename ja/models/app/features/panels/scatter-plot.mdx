このページでは、W&B での散布図の使い方を説明します。

<div id="use-case">
  ## ユースケース
</div>

散布図を使用して複数のrunを比較し、実験のパフォーマンスを可視化します:

- 最小値、最大値、平均値の線をプロットする。
- ツールチップに表示するメタデータをカスタマイズする。
- 点の色を設定する。
- 軸の範囲を調整する。
- 軸を対数スケールにする。

<div id="example">
  ## 例
</div>

次の例は、数週間にわたる実験で得られた、さまざまなモデルの検証精度を表示した散布図です。ツールチップには、バッチサイズ、ドロップアウト、および軸の値が含まれます。さらに、検証精度の移動平均を示す線も表示されます。

<div id="create-a-scatter-plot">
  ## 散布図を作成する
</div>

W&B の UI で散布図を作成するには、次の手順に従います。

1. **Workspaces** タブを開きます。
2. **Charts** パネルで、アクションメニュー `...` をクリックします。
3. ポップアップメニューから **Add panels** を選択します。
4. **Add panels** メニューで **Scatter plot** を選択します。
5. 表示したいデータがプロットされるように `x` 軸と `y` 軸を設定します。必要に応じて、軸の最大値と最小値の範囲を設定したり、`z` 軸を追加したりします。
6. **Apply** をクリックして散布図を作成します。
7. **Charts** パネルで新しい散布図を確認します。
