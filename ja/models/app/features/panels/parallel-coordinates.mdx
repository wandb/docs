平行座標チャートは、多数のハイパーパラメータとモデルのメトリクスとの関係を、一目で把握できるように要約して表示します。

<Frame>
  <img src="/images/app_ui/parallel_coordinates.gif" alt="Parallel coordinates plot" />
</Frame>

* **Axes**: [`wandb.Run.config`](/ja/models/tables/evaluate-models) に含まれるさまざまなハイパーパラメータと、[`wandb.Run.log()`](/ja/models/tables/evaluate-models) からのメトリクス。
* **Lines**: 各線は 1 つの run を表します。線にマウスオーバーすると、その run の詳細を含むツールチップが表示されます。現在のフィルタに一致するすべての線が表示されますが、目のアイコンをオフにすると、線はグレー表示になります。

<div id="create-a-parallel-coordinates-panel">
  ## 並列座標パネルを作成する
</div>

1. ワークスペースのランディングページを開きます
2. **Add Panels** をクリックします
3. **Parallel coordinates** を選択します

<div id="panel-settings">
  ## パネル設定
</div>

パネルを設定するには、パネル右上の編集ボタンをクリックします。

* **ツールチップ**: ホバーすると、各runに関する情報を表示する凡例が現れます
* **タイトル**: 軸タイトルを編集して、より読みやすくします
* **グラデーション**: 好きな色の範囲になるようにグラデーションをカスタマイズします
* **対数スケール**: 各軸は個別に対数スケールで表示するよう設定できます
* **軸反転**: 軸の向きを切り替えます。精度と損失の両方を列として持つ場合に便利です
