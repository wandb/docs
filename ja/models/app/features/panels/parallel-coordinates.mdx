平行座標チャートでは、多数のハイパーパラメーターとモデルのメトリクスの関係をひと目で把握できます。

<Frame>
  <img src="/images/app_ui/parallel_coordinates.gif" alt="平行座標プロット" />
</Frame>

* **Axes**: [`wandb.Run.config`](/ja/models/tables/evaluate-models) のさまざまなハイパーパラメーターと、[`wandb.Run.log()`](/ja/models/tables/evaluate-models) のメトリクス。
* **Lines**: 各ラインは1つの run を表します。ラインにマウスオーバーすると、run の詳細を示すツールチップが表示されます。現在のフィルターに一致するすべてのラインが表示されますが、目のアイコンをオフにすると、ラインはグレーアウトされます。

<div id="create-a-parallel-coordinates-panel">
  ## 平行座標 パネルを作成する
</div>

1. Workspace のランディングページにアクセスします
2. **Add Panels** をクリックします
3. **平行座標** を選択します

<div id="panel-settings">
  ## パネル設定
</div>

パネルを設定するには、パネルの右上にある編集ボタンをクリックします。

* **ツールチップ**: ホバーすると、各 run の情報を示す凡例が表示されます
* **タイトル**: 軸タイトルを編集して、より読みやすくできます
* **グラデーション**: 好みの色の範囲にグラデーションをカスタマイズできます
* **対数スケール**: 各軸はそれぞれ個別に対数スケールで表示するよう設定できます
* **軸の反転**: 軸の向きを切り替えます。精度と損失の両方を列として使用している場合に便利です
