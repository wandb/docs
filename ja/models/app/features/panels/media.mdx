メディアパネルは、3D オブジェクト、オーディオ、画像、動画、ポイントクラウドなど、[メディアオブジェクトのログされたキー](/ja/models/track/log/media/)を可視化します。このページでは、Workspace でメディアパネルを追加・管理する方法を説明します。

<div id="add-a-media-panel">
  ## メディアパネルを追加する
</div>

デフォルト設定でログされたキーのメディアパネルを追加するには、Quick Add を使用します。メディアパネルはグローバルに追加することも、特定のセクションに追加することもできます。

1. **グローバル**: パネル検索フィールド付近のコントロールバーにある **Add panels** をクリックします。
2. **セクション**: セクションのアクション `...` メニューをクリックし、**Add panels** をクリックします。
3. 利用可能なパネルのリストから、パネルのキーを見つけ、**Add** をクリックします。追加するメディアパネルごとにこの step を繰り返し、完了したら右上の **X** をクリックして **Quick Add** リストを閉じます。
4. 必要に応じて、[パネルを設定します](#configure-a-media-panel)。

メディアパネルはグローバルに追加することも、特定のセクションに追加することもできます。

1. **グローバル**: パネル検索フィールド付近のコントロールバーにある **Add panels** をクリックします。
2. **セクション**: セクションのアクション `...` メニューをクリックし、**Add panels** をクリックします。
3. **Media** セクションをクリックして展開します。
4. パネルで可視化するメディアのタイプ (3D objects、画像、video、またはオーディオ) を選択します。パネルの設定画面が表示されます。パネルを設定し、**Apply** をクリックします。[メディアパネルを設定する](#configure-a-media-panel)を参照してください。

<div id="configure-a-media-panel">
  ## メディアパネルを設定する
</div>

すべてのメディアタイプのパネルには、同じオプションがあります。

メディアパネルを手動で追加すると、メディアのタイプを選択した後に設定ページが開きます。既存のパネルの設定を更新するには、パネルにカーソルを合わせ、右上に表示されるギアアイコンをクリックします。このセクションでは、各タブで利用できる設定について説明します。

<div id="overlays">
  ### オーバーレイ
</div>

このタブは、セグメンテーションマスクまたはバウンディングボックスと共にログされた画像やポイントクラウドに対して表示されます。

* 名前でオーバーレイを検索およびフィルタリングします。
* オーバーレイのカラーをカスタマイズします。

<div id="sync">
  ### Sync
</div>

このタブは Workspace とセクションの設定に表示されます。

* **Sync slider by key**: セクション内の動画の step スライダーを同期して動かすかどうかを設定します。
* **Autoplay videos**: ページ読み込み時に動画を自動再生するかどうかを設定します。
* **Loop videos**: セクション内の動画を自動的に再起動して、停止するまで連続再生するかどうかを設定します。セクションレベルではカスタマイズできません。Workspace に動画メディアパネルがある場合にのみ表示されます。

<div id="display">
  ### Display
</div>

パネルの全体的な外観と動作をカスタマイズします。

* パネルのタイトルを設定します。
* 可視化するメディアキーを選択します。
* パネルのスライダーと再生の動作をカスタマイズします。
  * スライダーキーを設定します。デフォルトは **Step** です。
  * **Stride length** に、スライダーを1回クリックするたびに進む step 数を設定します。
  * **Snap to existing step** のオン/オフを切り替えます。オンにすると、ステッパーは **Stride length** 後の次の既存の step に進みます。オフにすると、既存の step と一致しない場合でも **Stride length** 分だけ進みます。
* **Images**: スムージングのオン/オフを切り替えます。
* **3d objects**: 背景色とポイントの色を設定します。

<div id="layout">
  ### Layout
</div>

パネルの個々のアイテムの表示をカスタマイズします。

* 必要に応じて、パネルに含める **Max runs to include** を制限します。
* 必要に応じて、**Media display limit** を指定して、run ごとに含めるメディアアイテムの数を制限します。
* **Panel mode** を設定します：
  * **Gallery** (デフォルト) ：列数と、グリッドに run (デフォルト) 、step、またはインデックスごとに1つのアイテムを表示するかどうかを指定します。
  * **Grid**：列数、x 軸 (デフォルトは **Log Step**) 、および y 軸 (デフォルトは **Run**) を指定します。
  * **Compare**：最大4つのメディアアイテムを並べて比較し、必要に応じて step またはインデックスでファンアウトします。詳細については [Compare mode](#compare-mode) を参照してください。
* **Use original size**：オフ (デフォルト) の場合、パネル内の画像と動画は同じサイズに見えるようにスケーリングされます。**Compare** モードでは効果がありません。
* **Fit to available space**：オン (デフォルト) の場合、パネル内の画像と動画はパネル内でできるだけ多くのスペースを占めるようにスケーリングされます。**Gallery** モードでは、各画像がパネルの幅いっぱいに表示されます。**Grid** モードでは、各画像がグリッドの幅いっぱいに表示されます。**Compare** モードでは効果がありません。
* **Right-handed system**：オンの場合、ポイントクラウドはデフォルトの左手系ではなく、右手系を使用してポイントをプロットします。

<div id="all-media-panels-in-a-section">
  ### セクション内のすべてのメディアパネル
</div>

セクション内のすべてのメディアパネルのデフォルト設定をカスタマイズし、メディアパネルの Workspace 設定を上書きするには、次の手順を実行します。

1. セクションのギアアイコンをクリックして設定を開きます。
2. **Media settings** をクリックします。
3. 表示された drawer 内で、**Display**、**Layout**、または **Sync** タブをクリックして、セクションのデフォルトのメディア設定を構成します。画像、動画、オーディオ、3D オブジェクトの設定を構成できます。表示される設定は、セクションの現在のメディアパネルによって異なります。

**Display** または **Layout** メディア設定の特定の設定の詳細については、[メディアパネルを設定する](#configure-a-media-panel) を参照してください。**Sync** タブは、セクションレベルまたは Workspace レベルでのみ使用可能で、個々のメディアパネルには使用できません。

**Step slider syncing** がオンになっている場合、同じ step スライダーを持つセクションのメディアパネルは同期された状態に保たれます。Step slider syncing をオンにするには、次の手順を実行します。

1. **Sync** タブをクリックします。
   1. **Sync slider by key (Step)** をオンにします。

<div id="all-media-panels-in-a-workspace">
  ### Workspace 内のすべてのメディアパネル
</div>

Workspace 内のすべてのメディアパネルのデフォルト設定をカスタマイズするには、次の手順に従います。

1. **Settings** というラベルの付いた歯車アイコンをクリックして、Workspace の設定を開きます。
2. **Media settings** をクリックします。
3. 表示された drawer 内で、**Display** タブまたは **Layout** タブをクリックして、Workspace のデフォルトのメディア設定を行います。画像、動画、オーディオ、3D オブジェクトの設定を行うことができます。表示される設定は、Workspace の現在のメディアパネルによって異なります。

**Sync** タブを除き、各設定の詳細については[メディアパネルを設定する](#configure-a-media-panel)を参照してください。**Sync** タブは、セクションまたは Workspace レベルでのみ利用可能で、個々のメディアパネルには適用されません。

**Step slider syncing** がオンになっている場合、セクション内で同じ step スライダーを持つメディアパネルが同期された状態に保たれます。Step slider syncing をオンにするには、次の手順に従います。

1. **Sync** タブをクリックします。
2. **Sync slider by key (Step)** をオンにします。

各設定の詳細については、[メディアパネルを設定する](#configure-a-media-panel)を参照してください。

<div id="compare-mode">
  ## Compare mode
</div>

Compare mode を使用すると、任意のRuns、step、またはインデックスの組み合わせから2〜4枚の画像または動画を選択し、単一のグリッドで表示できます。ファイルをダウンロードしたりタブを切り替えたりすることなく、メディアを比較できます。

Compare mode の用途:

* 新しいチェックポイントまたはrunが、ベースラインや以前のチェックポイントよりも優れた出力を生成することを確認する。
* 現在のrunを、同じstepまたはインデックスの既知の正常なチェックポイントと比較して、リグレッションを検出する。
* 参照メディアと生成メディアを並べて表示することで、モデルの出力をグラウンドトゥルースと照合して検証する。
* 生成された画像や動画が、トレーニングのstep間、または異なるチェックポイントやrun間でどのように変化するかを確認する。

Compare mode は、拡散モデルや生成モデルなどの画像・動画生成ワークフロー、マルチモーダル評価、ビジョン実験、およびメディアをログしてruns、step、またはインデックス間で比較する必要があるその他のワークフローに役立ちます。

Compare mode を使用するには:

1. Workspace でメディアパネルを開きます (画像または動画用) 。
2. **Panel mode** を **Compare** に設定します。
3. 列数とファンアウトモードを指定して、グリッドのサイズと密度を設定します。最適なグリッドのレイアウトは、Runsがメディアをどのようにログするかによって異なります。
   * **Number of columns**: 1行に表示するアイテム数 (`1`〜`4`) 。
   * **Fan out**: グリッドの行ごとに表示するメディアの粒度を制御します。最適な設定は、runごとのstepあたりのメディアアイテム数と、簡易確認が必要か詳細な分析が必要かによって異なります。
     * **None** (デフォルト) : ファンアウトなし。各グリッド列はrunを表し、各グリッド行はログされた画像または動画を表します。
       <Frame>![Screenshot showing a media panel in compare mode with two runs and no fan out](/images/models/compare-mode-fan-out-none.png)</Frame>
     * **Step**: stepの軸に沿ってファンアウトします。各グリッド列はrunを表し、各グリッド行はrun:stepの組み合わせで最初にログされた画像または動画を表示します。
       <Frame>![Screenshot showing a media panel in compare mode with two runs and fan out by step](/images/models/compare-mode-fan-out-by-step.png)</Frame>
     * **Index**: インデックスの軸に沿ってファンアウトします。stepごとに複数のアイテムをログする場合 (例: `run.log({"images": [img0, img1, img2, img3]})`) に便利です。各行のセルは、指定されたrun:step:インデックスの組み合わせでログされた画像または動画を表示します。
       <Frame>![Screenshot showing a media panel in compare mode with two runs and fan out by index](/images/models/compare-mode-fan-out-by-index.png)</Frame>
4. 特定の変数が単一のタイル (**Individual**) に影響するか、すべてのタイル (**Linked**) に影響するかを設定します。
   * **Media key**: デフォルトは **Linked**。
   * **Run**: デフォルトは **Individual**。
   * **Step**: デフォルトは **Linked**。
   * **Index**: デフォルトは **Linked**。

選択内容は保存されるため、パネルを再設定することなく、後で同じ比較に戻ることができます。

<div id="interact-with-a-media-panel">
  ## メディアパネルを操作する
</div>

* メディアパネルを設定するには、パネルにカーソルを合わせ、上部のギアアイコンをクリックします。
* メディアパネルの step スライダーを移動するには、**CMD + 左右矢印キー** (macOS) または **Ctrl + 左右矢印キー** (Windows / Linux) を使用します。セクションまたは Workspace で **Sync slider by key** が有効になっている場合、1つのメディアパネルで step スライダーを移動すると、同じ step スライダーキーを持つ他のメディアパネルの step スライダーも連動して移動します。
* メディアパネル上部のステッパーを使用して、メディアの run をステップ実行します。step スライダーを移動するには、UI コントロールを使用します。
* メディアパネルをクリックすると、全画面モードで表示されます。全画面モードを終了するには、パネル上部の矢印ボタンをクリックします。
* 全画面モードを終了せずにセクションのパネルを移動するには、パネル下部の **Previous** ボタンと **Next** ボタン、または左右矢印キーを使用します。
* 全画面モードで画像を表示しているとき、右上のズームコントロールをクリックしてズームイン・ズームアウトできます。デフォルトでは、画像はフィットするようにズームされます。ズームインした画像をパンするには、クリックしてドラッグします。キーボードショートカットを使用して、ズームイン、ズームアウト、100%ズームへのリセット、またはフィットズームを行うこともできます。[キーボードショートカット](/ja/models/app/keyboard-shortcuts#media-panels)を参照してください。
* セグメンテーションマスクと共にログされた画像の場合、外観をカスタマイズしたり、各マスクのオン・オフを切り替えたりできます。パネルにカーソルを合わせ、下部のギアアイコンをクリックします。
* バウンディングボックスと共にログされた画像またはポイントクラウドの場合、外観をカスタマイズしたり、各ボックスのオン・オフを切り替えたりできます。パネルにカーソルを合わせ、下部のギアアイコンをクリックします。
* メディアコントロールを使用して、動画の再生、一時停止、または停止を行います。**Sync video playback** が有効になっている場合、セクション内のすべての動画が同期して再生されます。**Loop videos** が有効になっている場合、セクション内の動画は自動的に再起動し、停止するまで継続して再生されます。
