---
title: メディアパネル
---

メディアパネルは、3D オブジェクト、音声、画像、動画、ポイントクラウドなどの[メディアオブジェクトに対してログされたキー](/ja/models/track/log/media/)を可視化します。このページでは、ワークスペースでメディアパネルを追加して管理する方法を説明します。

<div id="add-a-media-panel">
  ## メディアパネルを追加する
</div>

ログ済みのキーに対してデフォルト設定でメディアパネルを追加するには、Quick Add を使用します。メディアパネルは、グローバルまたは特定のセクションに追加できます。

1. **Global**: パネル検索フィールド付近のコントロールバーで **Add panels** をクリックします。
2. **Section**: セクションのアクション `...` メニューをクリックし、**Add panels** をクリックします。
3. 利用可能なパネルの一覧から対象パネルのキーを見つけて **Add** をクリックします。追加したい各メディアパネルについてこの手順を繰り返し、完了したら右上の **X** をクリックして **Quick Add** リストを閉じます。
4. 必要に応じて、[パネルを設定](#configure-a-media-panel)します。

メディアパネルは、グローバルまたは特定のセクションに追加できます。

1. **Global**: パネル検索フィールド付近のコントロールバーで **Add panels** をクリックします。
2. **Section**: セクションのアクション `...` メニューをクリックし、**Add panels** をクリックします。
3. **Media** セクションをクリックして展開します。
4. パネルで可視化するメディアの種類（3D オブジェクト、画像、動画、音声）を選択します。パネルの設定画面が表示されます。パネルを設定し、**Apply** をクリックします。[メディアパネルの設定](#configure-a-media-panel)を参照してください。

<div id="configure-a-media-panel">
  ## メディアパネルを設定する
</div>

すべてのメディアタイプのパネルには同じオプションが用意されています。

メディアパネルを手動で追加すると、メディアの種類を選択した後に、その設定ページが開きます。既存のパネルの設定を更新するには、パネルにカーソルを合わせ、右上に表示される歯車アイコンをクリックします。本セクションでは、各タブで利用可能な設定について説明します。

<div id="overlays">
  ### オーバーレイ
</div>

このタブは、セグメンテーションマスクやバウンディングボックスを含む画像およびポイントクラウドがログされている場合に表示されます。

* 名前でオーバーレイを検索・フィルタリングします。
* オーバーレイの色をカスタマイズします。

<div id="sync">
  ### 同期
</div>

このタブは、ワークスペースとセクションの設定画面に表示されます。

* **キーでスライダーを同期**: セクション内の動画のステップ用スライダーを同期して動かすかどうかを設定します。
* **動画を自動再生**: ページ読み込み時に動画を自動的に再生するかどうかを設定します。
* **動画をループ再生**: セクション内の動画を自動的に最初から再生し直し、停止するまで連続再生するかどうかを設定します。セクション単位ではカスタマイズできません。ワークスペースに動画メディアパネルがある場合にのみ表示されます。

<div id="display">
  ### 表示
</div>

パネル全体の見た目と動作をカスタマイズします。

* パネルのタイトルを設定します。
* 可視化するメディアキーを選択します。
* パネルのスライダーと再生動作をカスタマイズします。
  * 既定値が **Step** のスライダーキーを設定します。
  * スライダーを 1 回クリックするごとに進めるステップ数として **Stride length** を設定します。
  * **Snap to existing step** をオンまたはオフにします。オンの場合、**Stride length** ぶん進んだあと、次に存在するステップにスナップします。オフの場合、既存のステップと一致しなくても **Stride length** ぶんだけ進みます。
* **Images**: スムージングをオンまたはオフにします。
* **3d objects**: 背景色と点の色を設定します。

<div id="layout">
  ### レイアウト
</div>

パネル内の各アイテムの表示方法をカスタマイズします。

* **Grid mode** のオン／オフを切り替えます。
  * オンの場合、各アイテムに対して重ねてプロットするカスタムの X 軸と Y 軸を選択できます。1 行に複数のアイテムが表示され、表示する行数の上限を設定できます。
  * オフの場合、パネルのコンテンツに使用する列数をカスタマイズでき、パネルのコンテンツ自体も構成できます（デフォルトは **Run**）。
* パネルに含める **Max runs to include**（含める run の最大数）を必要に応じて制限します。
* 1 run あたりに含めるメディアアイテム数を制限するために、**Media display limit** を必要に応じて指定します。
* **Images and videos**: フルサイズのメディアを表示するかどうかを切り替えます。
* **Images**: **Fit media** がオンの場合、パネルのサイズに合わせてパネル内のメディアをリサイズします。
* **Point clouds**: デフォルトの左手系ではなく、必要に応じて右手系の座標系を有効にしてポイントをプロットします。

<div id="all-media-panels-in-a-section">
  ### セクション内のすべてのメディアパネル
</div>

セクション内のすべてのメディアパネルのデフォルト設定をカスタマイズして、メディアパネルに対するワークスペースの設定を上書きするには、以下を実行します。

1. セクションの歯車アイコンをクリックして設定を開きます。
2. **Media settings** をクリックします。
3. 表示されるドロワー内で、**Display**、**Layout**、または **Sync** タブをクリックして、そのセクションのデフォルトのメディア設定を構成します。画像、動画、音声、3D オブジェクトに対する設定を行えます。表示される設定は、そのセクションの現在のメディアパネルによって異なります。

**Display** または **Layout** の各メディア設定の詳細については、[Configure a media panel](#configure-a-media-panel) を参照してください。**Sync** タブは、個々のメディアパネルではなく、セクションまたは Workspace レベルでのみ利用できます。

**Step slider syncing** をオンにすると、同じステップスライダーを持つセクション内のメディアパネル同士が同期された状態に保たれます。ステップスライダーの同期を有効にするには、以下を実行します。

1. **Sync** タブをクリックします。
   1. **Sync slider by key (Step)** をオンにします。

<div id="all-media-panels-in-a-workspace">
  ### ワークスペース内のすべてのメディアパネル
</div>

ワークスペース内のすべてのメディアパネルに対するデフォルト設定をカスタマイズするには、次の手順を実行します。

1. 歯車アイコンと **Settings** ラベルが付いたワークスペースの設定をクリックします。
2. **Media settings** をクリックします。
3. 表示されるドロワー内で **Display** または **Layout** タブをクリックし、ワークスペースのデフォルトのメディア設定を行います。画像、動画、音声、および 3D オブジェクトに対する設定を行うことができます。表示される設定は、ワークスペース内の現在のメディアパネルによって異なります。

**Sync** タブを除き、各設定の詳細については [Configure a media panel](#configure-a-media-panel) を参照してください。**Sync** タブは、セクションまたはワークスペースレベルでのみ使用でき、個々のメディアパネルでは使用できません。

**Step slider syncing** をオンにすると、同じステップスライダーを持つセクション内のメディアパネルが同期されます。ステップスライダーの同期を有効にするには、次の手順を実行します。

1. **Sync** タブをクリックします。
2. **Sync slider by key (Step)** をオンにします。

各設定の詳細については [Configure a media panel](#configure-a-media-panel) を参照してください。

<div id="interact-with-a-media-panel">
  ## メディアパネルを操作する
</div>

* メディアパネルをクリックしてフルスクリーンモードで表示します。パネル上部の矢印ボタンをクリックするとフルスクリーンモードを終了します。
* フルスクリーンモードを終了せずにセクション内のパネル間を移動するには、パネル下の **Previous** と **Next** ボタン、または左右の矢印キーを使用します。
* メディアパネルのステップスライダーを動かすには、**CMD + 左右の矢印キー**（macOS）または **Ctrl + 左右の矢印キー**（Windows / Linux）を使用します。セクションまたはワークスペースで **Sync slider by key** がオンになっている場合、あるメディアパネルのステップスライダーを動かすと、同じステップスライダーキーを持つ他のメディアパネルのステップスライダーも連動して動きます。
* メディアパネル上部のステッパーを使って、メディアの run を切り替えます。ステップスライダーを動かすには、UI コントロールを使用します。
* メディアコントロールを使用して、動画の再生、一時停止、停止を行います。**Sync video playback** がオンになっている場合、セクション内のすべての動画が同期して再生されます。**Loop videos** がオンになっている場合、セクション内の動画は自動的に最初から再生され、停止するまで連続再生されます。
* メディアパネルを設定するには、その上にマウスカーソルを合わせ、上部の歯車アイコンをクリックします。
* セグメンテーションマスク付きでログされた画像では、マスクの見た目をカスタマイズしたり、個別にオン・オフを切り替えたりできます。パネル上にマウスカーソルを合わせ、下部の歯車アイコンをクリックします。
* バウンディングボックス付きでログされた画像やポイントクラウドでは、ボックスの見た目をカスタマイズしたり、個別にオン・オフを切り替えたりできます。パネル上にマウスカーソルを合わせ、下部の歯車アイコンをクリックします。
