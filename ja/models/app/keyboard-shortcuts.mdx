W&amp;B は、Experiments、Workspace、データをより効率的に移動・操作できるよう、キーボードショートカットをサポートしています。このリファレンスガイドでは、W&amp;B App と W&amp;B LEET (Lightweight Experiment Exploration Tool) ターミナル UI のキーボードショートカットを紹介します。

<Tabs>
  <Tab title="App">
    これらのキーボードショートカットは W&amp;B App で使用できます。

    ## Workspace 管理

    | Shortcut                                             | Description                           |
    | ---------------------------------------------------- | ------------------------------------- |
    | **Cmd+Z** (macOS) / **Ctrl+Z** (Windows/Linux)       | Workspace やパネルの変更など、UI で行った変更を元に戻します。 |
    | **Cmd+Shift+Z** (macOS) / **Ctrl+Y** (Windows/Linux) | Workspace で前に元に戻した変更をやり直します。          |

    ## ナビゲーション

    | Shortcut                                       | Description                                                                    |
    | ---------------------------------------------- | ------------------------------------------------------------------------------ |
    | **Tab**                                        | インタラクティブな要素間を移動します。                                                            |
    | **Cmd+J** (macOS) / **Ctrl+J** (Windows/Linux) | プロジェクトのサイドバーで Workspaces タブと Runs タブを切り替えます。                                   |
    | **Cmd+.** (macOS) / **Ctrl+.** (Windows/Linux) | 画面スペースを確保するために、Runs セレクターのサイドバーを最小化または復元します。                                   |
    | **Cmd+K** (macOS) / **Ctrl+K** (Windows/Linux) | クイック検索ダイアログを開き、プロジェクト、Runs、そのほかのリソースを検索します。                                    |
    | **Esc**                                        | W&amp;B App 全体で、全画面パネルビューを終了したり、設定ドロワーやクイック検索ダイアログ、エディター、そのほかのオーバーレイを閉じたりできます。 |

    ## パネルナビゲーション

    | Shortcut                                                                                                 | Description                                |
    | -------------------------------------------------------------------------------------------------------- | ------------------------------------------ |
    | **Left Arrow / Right Arrow**                                                                             | 全画面モードで、セクション内のパネルを順に切り替えます。               |
    | **Esc**                                                                                                  | 全画面パネルビューを終了して Workspace に戻ります。            |
    | **Cmd+Left Arrow / Cmd+Right Arrow** (macOS)<br />**Ctrl+Left Arrow / Ctrl+Right Arrow** (Windows/Linux) | 全画面モードでメディアパネルを表示しているときに、step スライダーを移動します。 |

    ## メディアパネル

    | Shortcut                                                  | Description                                                              |
    | --------------------------------------------------------- | ------------------------------------------------------------------------ |
    | **Cmd + +/-** (macOS)<br />**Ctrl + +/-** (Windows/Linux) | 全画面で画像を表示しているときに、ズームインまたはズームアウトします。<br />ズームイン時は、クリックしてドラッグすると画像を移動できます。 |
    | **Cmd + 0** (macOS)<br />**Ctrl + 0** (Windows/Linux)     | 全画面で画像を表示しているときに、ズームを 100% にリセットします。                                     |
    | **Shift + L**                                             | 全画面で画像を表示しているときに、画面に収まるようにズームします。                                        |

    ## Reports

    | Shortcut               | Description                                          |
    | ---------------------- | ---------------------------------------------------- |
    | **Delete / Backspace** | 選択したパネルグリッドをレポートから削除します。                             |
    | **Enter**              | レポートで &quot;/mark&quot; と入力した後に Markdown ブロックを挿入します。 |
    | **Esc**                | レポートエディターを終了します。                                     |
    | **Tab**                | レポート内のインタラクティブな要素間を移動します。                            |

    ## メモ

    * ほとんどのキーボードショートカットでは、macOS では **Cmd**、Windows/Linux では **Ctrl** を使用します。
    * W&amp;B App では、一部のブラウザー既定のショートカットに対してカスタム処理を実装しています。
    * 一部のショートカットはコンテキスト依存で、アプリケーション内の特定の領域でのみ機能します。
