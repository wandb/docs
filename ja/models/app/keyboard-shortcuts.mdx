W&amp;B は、Experiments、Workspace、データをより効率的に操作・ナビゲートするためのキーボードショートカットをサポートしています。このリファレンスガイドでは、W&amp;B App および W&amp;B LEET (Lightweight Experiment Exploration Tool) ターミナル UI のキーボードショートカットについて説明します。

<Tabs>
  <Tab title="App">
    これらのキーボードショートカットは W&amp;B App で使用できます。

    ## Workspace の管理

    | ショートカット                                              | 説明                                     |
    | ---------------------------------------------------- | -------------------------------------- |
    | **Cmd+Z** (macOS) / **Ctrl+Z** (Windows/Linux)       | Workspace やパネルへの変更など、UI で行った変更を元に戻します。 |
    | **Cmd+Shift+Z** (macOS) / **Ctrl+Y** (Windows/Linux) | Workspace で以前に元に戻した変更をやり直します。          |

    ## ナビゲーション

    | ショートカット                                        | 説明                                                                                               |
    | ---------------------------------------------- | ------------------------------------------------------------------------------------------------ |
    | **Tab**                                        | インタラクティブな要素間を移動します。                                                                              |
    | **Cmd+J** (macOS) / **Ctrl+J** (Windows/Linux) | プロジェクトのサイドバーで Workspace タブと Runs タブを切り替えます。                                                      |
    | **Cmd+.** (macOS) / **Ctrl+.** (Windows/Linux) | Runs セレクターのサイドバーを最小化または復元して、画面スペースを確保します。                                                        |
    | **Cmd+K** (macOS) / **Ctrl+K** (Windows/Linux) | クイック検索ダイアログを開き、プロジェクト、run、その他のリソースを横断して検索します。                                                    |
    | **Esc**                                        | W&amp;B App 全体で、フルスクリーンのパネルビューを終了したり、設定ドロワーを閉じたり、クイック検索ダイアログを閉じたり、エディターを閉じたり、その他のオーバーレイを閉じたりします。 |

    ## パネルのナビゲーション

    | ショートカット                                                                    | 説明                                             |
    | -------------------------------------------------------------------------- | ---------------------------------------------- |
    | **左矢印 / 右矢印**                                                              | フルスクリーンモード時に、セクション内のパネルを順に移動します。               |
    | **Esc**                                                                    | フルスクリーンのパネルビューを終了し、Workspace に戻ります。            |
    | **Cmd+左矢印 / Cmd+右矢印** (macOS)<br />**Ctrl+左矢印 / Ctrl+右矢印** (Windows/Linux) | メディアパネルをフルスクリーンモードで表示しているときに、step スライダーを移動します。 |

    ## メディアパネル

    | ショートカット                                                   | 説明                                                                             |
    | --------------------------------------------------------- | ------------------------------------------------------------------------------ |
    | **Cmd + +/-** (macOS)<br />**Ctrl + +/-** (Windows/Linux) | フルスクリーンで画像を表示しているときに、ズームインまたはズームアウトします。<br />ズームイン時は、クリックしてドラッグすることで画像をパンできます。 |
    | **Cmd + 0** (macOS)<br />**Ctrl + 0** (Windows/Linux)     | フルスクリーンで画像を表示しているときに、ズームを 100% にリセットします。                                       |
    | **Shift + L**                                             | フルスクリーンで画像を表示しているときに、画面に合わせてズームします。                                            |

    ## Reports

    | ショートカット                | 説明                                       |
    | ---------------------- | ---------------------------------------- |
    | **Delete / Backspace** | レポートから選択したパネルグリッドを削除します。                 |
    | **Enter**              | レポートで「/mark」と入力した後に Markdown ブロックを挿入します。 |
    | **Esc**                | レポートエディターを終了します。                         |
    | **Tab**                | レポート内のインタラクティブな要素間を移動します。                |

    ## メモ

    * ほとんどのキーボードショートカットは、macOS では **Cmd**、Windows/Linux では **Ctrl** を使用します。
    * W&amp;B App は、一部のブラウザーデフォルトショートカットに対してカスタム処理を実装しています。
    * 一部のショートカットはコンテキスト依存であり、アプリケーションの特定の領域でのみ機能します。
