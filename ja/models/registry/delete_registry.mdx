---
title: レジストリの削除
description: "Python SDK または W&B App UI を使用して、Team admin または Registry admin として W&B Registry を削除します。"
---

このページでは、Team admin または Registry admin がレジストリを削除する方法を説明します。

* Team admin は、組織内の任意のレジストリを削除できます。
* Registry admin は、自分で作成したレジストリを削除できます。

レジストリを削除すると、そのレジストリに属するコレクションも削除されますが、レジストリにリンクされている Artifacts は削除されません。このようなアーティファクトは、ログされた元のプロジェクトにそのまま残ります。

<Tabs>
  <Tab title="Python SDK">
    `wandb` API の `delete()` method を使用して、レジストリをプログラムで削除できます。次の例では、以下を行います。

    1. 削除するレジストリを `api.registry()` で取得します。
    2. 返されたレジストリ object に対して `delete()` method を呼び出し、レジストリを削除します。

    ```python
    import wandb

    # W&B API を初期化
    api = wandb.Api()

    # 削除するレジストリを取得
    fetched_registry = api.registry("<registry_name>")

    # レジストリを削除
    fetched_registry.delete()
    ```
  </Tab>

  <Tab title="W&B App">
    1. https://wandb.ai/registry/ にアクセスして、W&amp;B Registry を開きます。
    2. 削除するレジストリを選択します。
    3. 右上の歯車アイコンをクリックして、レジストリの設定を表示します。
    4. レジストリを削除するには、設定ページの右上にあるゴミ箱アイコンをクリックします。
    5. 表示されるモーダルでレジストリ名を入力して削除対象を確認し、**Delete** をクリックします。
  </Tab>
</Tabs>
