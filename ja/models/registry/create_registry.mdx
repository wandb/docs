Registry では、使用できるアーティファクトタイプに対する柔軟性と制御が提供され、Registry の公開範囲を制限することもできます。

<div id="create-a-registry">
  ## Create a registry
</div>

W&amp;B Registry UI を使用するか、W&amp;B Python SDK を使用してプログラム的に Registry を作成します。

<Tabs>
  <Tab title=" W&B Registry UI">
    1. W&amp;B Registry の https://wandb.ai/registry/ にアクセスします。
    2. **Create registry** ボタンをクリックします。
    3. **Name** フィールドに Registry の名前を入力します。
    4. 必要に応じて、Registry に関する説明を入力します。
    5. **Registry visibility** ドロップダウンから、Registry を閲覧できる対象を選択します。Registry の公開範囲オプションの詳細については、[Registry visibility types](./configure_registry#registry-visibility-types) を参照してください。
    6. **Accepted artifacts type** ドロップダウンから **All types** または **Specify types** を選択します。
    7. (**Specify types** を選択した場合) Registry が受け入れる 1 つ以上のアーティファクト タイプを追加します。
    8. **Create registry** ボタンをクリックします。
  </Tab>

  <Tab title="Python SDK">
    [`wandb.Api().create_registry()`](/ja/models/ref/python/#method-apicreate_registry) メソッドを使用して、プログラム的に Registry を作成します。`name` と `visibility` パラメーターには、それぞれ Registry の名前と [公開範囲](#visibility-types) を指定します。

    以下のコードブロックをコピー＆ペーストし、`<>` で囲まれた値を自分の値に置き換えてください。

    ```python
    import wandb

    registry = wandb.Api().create_registry(
        name="<registry_name>",
        visibility="< 'restricted' | 'organization' >",
    )
    ```

    Registry を作成する際に指定できるパラメーターの全一覧については、[`wandb.Api().create_registry()`](/ja/models/ref/python/#method-apicreate_registry) メソッドのリファレンスを参照してください。
  </Tab>
</Tabs>

<Note>
  アーティファクト タイプは、Registry の設定に保存された後は Registry から削除できません。
</Note>

例えば、次の画像はユーザーが作成しようとしている `Fine_Tuned_Models` という名前の Registry を示しています。この Registry は **Restricted** に設定されており、手動で追加されたメンバーのみがアクセスできます。

<div id="visibility-types">
  ## 公開範囲の種類
</div>

Registry の *公開範囲* は、その Registry に誰が 액세스できるかを決定します。Registry の公開範囲を制限することで、指定したメンバーのみがその Registry に 액세スできることを確保できます。

Registry の公開範囲には、次の 2 種類のオプションがあります：

| Visibility   | Description                        |
| ------------ | ---------------------------------- |
| Restricted   | 招待された組織メンバーのみが Registry に 액세스できます。 |
| Organization | 組織内の全員が Registry に 액세スできます。        |

チーム管理者または Registry 管理者が、Registry の公開範囲を設定できます。

公開範囲を Restricted に設定して Registry を作成したユーザーは、その Registry の Registry 管理者として自動的に追加されます。

<div id="configure-the-visibility-of-a-registry">
  ## Registry の公開範囲を設定する
</div>

チーム管理者またはRegistry 管理者は、Registry の作成時または作成後に、Registry の公開範囲を부여することができます。

既存の Registry の公開範囲を制限するには:

1. https://wandb.ai/registry/ の W&amp;B Registry にアクセスします。
2. Registry を選択します。
3. 右上隅の歯車アイコンをクリックします。
4. **Registry visibility** ドロップダウンから、目的の Registry の公開範囲を選択します。
5. **Restricted visibility** を選択した場合:
   1. この Registry への 액세스 を許可する組織のメンバーを追加します。**Registry members and roles** セクションまでスクロールし、**Add member** ボタンをクリックします。
   2. **Member** フィールドに、追加するメンバーのメールアドレスまたはユーザー名を追加します。
   3. **Add new member** をクリックします。
