title: リネージグラフと監査履歴
---

リネージグラフを使用して、リンクされたアーティファクトの履歴を可視化します。コレクションの履歴を監査して、そのコレクション内のアーティファクトに対して行われた変更を追跡します。

<div id="lineage-graphs">
  ## リネージグラフ
</div>

W&amp;B Registry 内のコレクションでは、ML 実験で使用されるアーティファクトの履歴を表示できます。この履歴は *リネージグラフ* と呼ばれます。

リネージグラフでは、次の内容が表示されます：

* [run の入力として](/ja/models/artifacts/explore-and-traverse-an-artifact-graph#track-the-input-of-a-run)使用されたアーティファクト
* [run の出力として](/ja/models/artifacts/explore-and-traverse-an-artifact-graph#track-the-output-of-a-run)作成されたアーティファクト

言い換えると、リネージグラフは run の入力と出力を表示します。

たとえば、次の画像は、ML 実験全体で作成および使用されるアーティファクトの典型的なリネージグラフを示しています：

<Frame>
  <img src="/images/registry/registry_lineage.png" alt="Registry リネージ" />
</Frame>

左から右にかけて、画像は次の内容を示しています：

1. 複数の run が `split_zoo_dataset:v4` アーティファクトをログします。
2. &quot;rural-feather-20&quot; run は、学習のために `split_zoo_dataset:v4` アーティファクトを使用します。
3. &quot;rural-feather-20&quot; run の出力は、`zoo-ylbchv20:v0` という名前のモデルアーティファクトです。
4. &quot;northern-lake-21&quot; という run が、モデル評価のためにモデルアーティファクト `zoo-ylbchv20:v0` を使用します。

コレクション内のアーティファクトのリネージグラフを表示するには、次の手順を実行します：

1. W&amp;B Registry に移動します。
2. 対象のアーティファクトを含むコレクションを選択します。
3. ドロップダウンから、リネージグラフを表示したいアーティファクトバージョンを選択します。
4. **Lineage** タブを選択します。
5. ノードを選択して、run またはアーティファクトに関する詳細情報を表示します。

<Info>
  W&amp;B Python SDK を使用して run の入力と出力を追跡する方法については、[リネージグラフのトラッキングを有効化](/ja/models/artifacts/explore-and-traverse-an-artifact-graph#enable-lineage-graph-tracking)を参照してください。
</Info>

</Info>

<div id="audit-a-collections-history">
  ## コレクションの履歴を監査する
</div>

組織のメンバーがそのコレクションに対して実行した操作を確認できます。次の内容を確認できます:

* エイリアスがアーティファクトバージョンに追加されたか、削除されたか。
* アーティファクトバージョンがコレクションに追加されたか、削除されたか。

どちらの操作についても、その操作を実行したユーザーと、操作が行われた日付を確認できます。

コレクションの操作履歴を表示するには:

1. W&amp;B Registry に移動します。
2. 操作履歴を表示したいコレクションを選択します。
3. コレクション名の横にあるドロップダウンメニューを選択します。
4. **Action History** を選択します。
