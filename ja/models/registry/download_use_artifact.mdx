---
title: レジストリからアーティファクトをダウンロードする
description: "パスを組み立てて Python SDK を使用し、W&B Registry のコレクションにリンクされたアーティファクトをダウンロードします。"
---

レジストリにリンクされたアーティファクトをダウンロードするには、W&amp;B Python SDK を使用します。アーティファクトをダウンロードして使用するには、レジストリの名、コレクションの名、およびダウンロードしたいアーティファクト バージョンのエイリアスまたはインデックスを把握している必要があります。

アーティファクトのプロパティがわかれば、[リンクされたアーティファクトへのパスを組み立て](#construct-path-to-linked-artifact)、アーティファクトをダウンロードできます。あるいは、W&amp;B App UI から[事前生成されたコードスニペットをコピー＆ペースト](#copy-and-paste-pre-generated-code-snippet)して、レジストリにリンクされたアーティファクトをダウンロードすることもできます。

<div id="construct-path-to-linked-artifact">
  ## リンクされたアーティファクト へのパスを構築する
</div>

Registry にリンクされたアーティファクト をダウンロードするには、そのアーティファクト へのパスを把握しておく必要があります。パスは、Registry 名、コレクション 名、およびアクセスしたいアーティファクト バージョンのエイリアス またはインデックスで構成されます。

アーティファクト バージョンの Registry、コレクション、エイリアス またはインデックスがわかれば、次の文字列テンプレートを使ってリンクされたアーティファクト へのパスを構築できます。

波かっこ `{}` 内の値を、アクセスしたいアーティファクト バージョンの Registry 名、コレクション 名、および alias またはインデックスに置き換えてください。

リンクされたアーティファクト のパスがわかったら、`wandb.Run.use_artifact()` method を使用してアーティファクト にアクセスし、その内容をダウンロードします。次のコードスニペットは、W&amp;B Registry にリンクされた アーティファクト の使い方とダウンロード方法を示しています。`<>` 内の値は必ずご自身のものに置き換えてください。

`wandb.Run.use_artifact()` method は、[run](/ja/models/runs) を作成すると同時に、ダウンロードしたアーティファクト をその run の入力としてマークします。
アーティファクト を run の入力としてマークすると、W&amp;B はそのアーティファクト のリネージをトラッキングできるようになります。

run を作成したくない場合は、`wandb.Api()` オブジェクトを使用してアーティファクト にアクセスできます:

```python
import wandb

REGISTRY = "<registry_name>"
COLLECTION = "<collection_name>"
VERSION = "<version>"

api = wandb.Api()
artifact_name = f"wandb-registry-{REGISTRY}/{COLLECTION}:{VERSION}"
artifact = api.artifact(name = artifact_name)
```

<details>
  <summary>例: W&amp;B Registry にリンクされた アーティファクト を使用してダウンロードする</summary>

  次のコード例は、**Fine-tuned Models** Registry 内の `phi3-finetuned` という コレクション にリンクされた アーティファクト を、ユーザーがダウンロードする方法を示しています。アーティファクト バージョンの alias は `production` に設定されています。
</details>

パラメーターと戻り値の型については、API Reference の [`wandb.Run.use_artifact()`](/ja/models/ref/python/experiments/run#use_artifact) と [`Artifact.download()`](/ja/models/ref/python/experiments/artifact#download) を参照してください。

<Note>
  **複数の組織に属する personal entity を持つ Users**

  複数の組織に属し、personal entity を持つ Users が Registry にリンクされたアーティファクト にアクセスする場合は、組織名を指定するか、チーム entity を使用する必要があります。
</Note>

<div id="copy-and-paste-pre-generated-code-snippet">
  ## 事前に生成されたコードスニペットをコピー＆ペーストする
</div>

W&amp;B では、レジストリに関連付けられた artifact をダウンロードするためのコードスニペットが生成されます。このコードスニペットは、Python スクリプト、ノートブック、またはターミナルにコピー＆ペーストして使用できます。

1. W&amp;B Registry にアクセスします。
2. artifact が含まれているレジストリの名を選択します。
3. コレクションの名を選択します。
4. artifact バージョンの一覧から、アクセスするバージョンを選択します。
5. **Usage** タブを選択します。
6. **Usage API** セクションに表示されているコードスニペットをコピーします。
7. コードスニペットを Python スクリプト、ノートブック、またはターミナルに貼り付けます。
