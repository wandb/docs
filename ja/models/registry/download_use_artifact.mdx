title: Registry からアーティファクトをダウンロードする
description: "パスを構築し、Python SDK を使用して W&B Registry のコレクションにリンクされたアーティファクトをダウンロードします。"
---

W&amp;B Python SDK を使用して、Registry にリンクされたアーティファクトをダウンロードします。アーティファクトをダウンロードして使用するには、Registry の名前、コレクションの名前、およびダウンロードするアーティファクト バージョンの 별칭 またはインデックスを把握している必要があります。

アーティファクトのプロパティがわかれば、[リンクされたアーティファクトへのパスを構築](#construct-path-to-linked-artifact)してアーティファクトをダウンロードできます。あるいは、W&amp;B App UI から [事前生成された코드 스니펫をコピー＆ペースト](#copy-and-paste-pre-generated-code-snippet)して、Registry にリンクされたアーティファクトをダウンロードすることもできます。

<div id="construct-path-to-linked-artifact">
  ## リンクされたアーティファクトへのパスを作成する
</div>

Registry にリンクされたアーティファクトをダウンロードするには、そのリンク先アーティファクトのパスを把握しておく必要があります。パスは、Registry 名、コレクション名、およびアクセスしたいアーティファクトバージョンの 별칭 またはインデックスで構成されます。

Registry、コレクション、およびアーティファクトバージョンの 별칭 またはインデックスが分かったら、次の文字列テンプレートを使用してリンクされたアーティファクトへのパスを作成できます。

```python
# バージョンインデックスを指定したアーティファクト名
f"wandb-registry-{REGISTRY}/{COLLECTION}:v{INDEX}"

# エイリアスを指定したアーティファクト名
f"wandb-registry-{REGISTRY}/{COLLECTION}:{ALIAS}"
```

中かっこ `{}` 内の値を、액세스 したいアーティファクト バージョンの Registry の名、コレクション、および 별칭 またはインデックスに置き換えてください。

リンクされたアーティファクトのパスを取得したら、`wandb.Run.use_artifact()` method を使用してアーティファクトに 액세스 し、その内容をダウンロードします。次の 코드 스니펫 は、W&amp;B Registry にリンクされたアーティファクトの使い方とダウンロード方法を示しています。`<>` 内の値はご自身のものに置き換えてください。

```python
import wandb

REGISTRY = '<registry_name>'
COLLECTION = '<collection_name>'
ALIAS = '<artifact_alias>'

with wandb.init(entity = '<team_name>', project = '<project_name>')  as run:
    artifact_name = f"wandb-registry-{REGISTRY}/{COLLECTION}:{ALIAS}"
    # artifact_name = '<artifact_name>' # W&B Registry UIで指定されたフル名前をコピー＆ペースト
    fetched_artifact = run.use_artifact(artifact_or_name = artifact_name)  
    download_path = fetched_artifact.download()  
```

`wandb.Run.use_artifact()` method は、[run](/ja/models/runs) を作成すると同時に、ダウンロードしたアーティファクトをその run への入力としてマークします。
アーティファクトを run への入力としてマークすると、W&amp;B はそのアーティファクトのリネージをトラッキングできます。

run を作成したくない場合は、`wandb.Api()` object を使用してアーティファクトに 액세スできます。

```python
import wandb

REGISTRY = "<registry_name>"
COLLECTION = "<collection_name>"
VERSION = "<version>"

api = wandb.Api()
artifact_name = f"wandb-registry-{REGISTRY}/{COLLECTION}:{VERSION}"
artifact = api.artifact(name = artifact_name)
```

<details>
  <summary>例: W&amp;B Registry にリンクされたアーティファクトを使用してダウンロードする</summary>

  次のコード例は、ユーザーが **Fine-tuned Models** Registry 内の `phi3-finetuned` というコレクションにリンクされたアーティファクトをダウンロードする方法を示しています。アーティファクトバージョンの별칭は `production` に設定されています。

  ```python
  import wandb

  TEAM_ENTITY = "product-team-applications"
  PROJECT_NAME = "user-stories"

  REGISTRY = "Fine-tuned Models"
  COLLECTION = "phi3-finetuned"
  ALIAS = 'production'

  # 指定したチームとプロジェクト内で run を初期化します
  with wandb.init(entity=TEAM_ENTITY, project = PROJECT_NAME) as run:

      artifact_name = f"wandb-registry-{REGISTRY}/{COLLECTION}:{ALIAS}"

      # アーティファクトに액세스し、リネージ追跡のために run への入力としてマークします
      fetched_artifact = run.use_artifact(artifact_or_name = artifact_name)  

      # アーティファクトをダウンロードします。ダウンロードされた内容へのパスを返します
      downloaded_path = fetched_artifact.download()  
  ```
</details>

パラメーターと戻り値の型については、API Reference の [`wandb.Run.use_artifact()`](/ja/models/ref/python/experiments/run#use_artifact) と [`Artifact.download()`](/ja/models/ref/python/experiments/artifact#download) を参照してください。

<Note>
  **複数の組織に所属する個人 entity を持つ Users**

  複数の組織に所属する個人 entity を持つ Users は、Registry にリンクされたアーティファクトに액세스するときに、組織名を指定するか、チーム entity を使用する必要もあります。

  ```python
  import wandb

  REGISTRY = "<registry_name>"
  COLLECTION = "<collection_name>"
  VERSION = "<version>"

  # API を初期化する際は、チーム entity を使用していることを確認します
  api = wandb.Api(overrides={"entity": "<team-entity>"})
  artifact_name = f"wandb-registry-{REGISTRY}/{COLLECTION}:{VERSION}"
  artifact = api.artifact(name = artifact_name)

  # パスを構築します
  api = wandb.Api()
  artifact_name = f"wandb-registry-{REGISTRY}/{COLLECTION}:{VERSION}"
  artifact = api.artifact(name = artifact_name)
  ```
</Note>

<div id="copy-and-paste-pre-generated-code-snippet">
  ## 事前生成されたコード 스니펫をコピー＆ペーストする
</div>

W&amp;B は、Registry にリンクされたアーティファクトをダウンロードするために、Python スクリプト、ノートブック、またはターミナルにコピー＆ペーストできるコード 스니펫を作成します。

1. W&amp;B Registry にアクセスします。
2. アーティファクトを含む Registry 名を選択します。
3. コレクション名を選択します。
4. アーティファクト バージョンの一覧から、액세스したいバージョンを選択します。
5. **Usage** タブを選択します。
6. **Usage API** セクションに表示されているコード 스니펫をコピーします。
7. コピーしたコード 스니펫を Python スクリプト、ノートブック、またはターミナルにペーストします。
