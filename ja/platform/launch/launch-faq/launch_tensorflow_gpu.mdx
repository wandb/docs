---
title: W&B Launch を TensorFlow の GPU 環境で動作させるにはどうすればよいですか？
---

GPU を利用する TensorFlow ジョブでは、コンテナのビルドに使用するカスタムベースイメージを指定してください。これにより、run 実行時に GPU を適切に活用できます。リソース設定内の `builder.accelerator.base_image` キーの下に image タグを追加してください。例えば、次のように指定します。

```json
{
    "gpus": "all",
    "builder": {
        "accelerator": {
            "base_image": "tensorflow/tensorflow:latest-gpu"
        }
    }
}
```

W&amp;B 0.15.6 より前のバージョンでは、`base_image` の親キーには `accelerator` ではなく `cuda` を指定してください。
