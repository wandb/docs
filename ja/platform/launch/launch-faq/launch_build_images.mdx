---
title: W&B Launch はどのようにイメージをビルドしますか？
---

イメージをビルドする手順は、ジョブのソースと、リソース設定で指定されたアクセラレーターのベースイメージに依存します。

<Note>
  キューを設定するかジョブを送信するときは、キューまたはジョブのリソース設定にベースとなるアクセラレーターイメージを含めてください:

  ```json
  {
      "builder": {
          "accelerator": {
              "base_image": "image-name"
          }
      }
  }
  ```
</Note>

ビルドプロセスには、ジョブタイプと指定されたアクセラレーターのベースイメージに基づいて、次の処理が含まれます。

| | apt を使用して Python をインストール | Python パッケージをインストール | ユーザーと作業ディレクトリ (workdir) を作成 | コードをイメージ内にコピー | エントリポイントを設定 | |
