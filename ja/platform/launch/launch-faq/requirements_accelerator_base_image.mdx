---
title: アクセラレータ用のベースイメージにはどのような要件がありますか？
---

アクセラレータを利用するジョブでは、必要なアクセラレータコンポーネントを含んだベースイメージを用意してください。アクセラレータ用イメージについて、次の要件を満たしていることを確認してください。

* Debian との互換性 (Launch の Dockerfile は `apt-get` を使って Python をインストールします) 
* サポートされる CPU および GPU のハードウェア命令セット (想定する GPU と CUDA バージョンの互換性を確認してください) 
* 指定したアクセラレータバージョンと、機械学習アルゴリズムで使用するパッケージとの互換性
* ハードウェア互換性のために追加手順が必要なパッケージのインストール
