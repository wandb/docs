---
title: アクセラレータのベースイメージにはどのような要件がありますか？
---

アクセラレータを使用するジョブでは、必要なアクセラレータコンポーネントを含むベースイメージを用意してください。アクセラレータイメージが次の要件を満たしていることを確認してください。

* Debian との互換性があること (Launch の Dockerfile は Python のインストールに `apt-get` を使用します) 
* サポート対象の CPU および GPU のハードウェア命令セットに対応していること (使用予定の GPU と CUDA バージョンの互換性を確認してください) 
* 使用するアクセラレータのバージョンと、機械学習アルゴリズムで使用するパッケージとの互換性があること
* ハードウェア互換性のために追加の手順が必要なパッケージがインストールされていること
