---
title: アクセラレータのベースイメージにはどのような要件がありますか？
---

アクセラレータを使用する W&amp;B Launch ジョブでは、必要なアクセラレータコンポーネントを含むベースイメージを用意してください。これにより、ジョブの実行に必要な GPU ドライバー、ランタイムライブラリ、その他のハードウェア固有の依存関係を確実に含めることができます。アクセラレータイメージは、次の要件を満たす必要があります。

* Debian との互換性 (Launch Dockerfile は Python のインストールに `apt-get` を使用します) 。
* サポートされる CPU および GPU のハードウェア命令セット (対象の GPU と CUDA バージョンの互換性を確認してください) 。
* 使用するアクセラレータのバージョンと、機械学習アルゴリズム内のパッケージとの互換性。
* ハードウェア互換性のために追加の手順が必要なパッケージのインストール。
