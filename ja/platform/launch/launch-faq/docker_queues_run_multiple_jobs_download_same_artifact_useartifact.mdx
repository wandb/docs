---
title: Docker キュー内の複数のジョブが同じ artifact をダウンロードする場合、キャッシュは
  使用されますか、それとも run のたびに再ダウンロードされますか？
keywords: ["artifact キャッシュ", "Docker キャッシュ", "artifact のダウンロード", "artifact のマウント"]
---

Launch ジョブはキャッシュを共有しません。各 launch ジョブは独立して実行されます。キュー設定の Docker 引数を使用して、キューまたは agent が共有キャッシュをマウントするように設定してください。

さらに、特定のユースケースでは、W&amp;B Artifacts のキャッシュを永続ボリュームとしてマウントすることもできます。
