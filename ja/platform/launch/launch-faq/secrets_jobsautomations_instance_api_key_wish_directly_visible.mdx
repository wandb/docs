---
title: ジョブやオートメーションにシークレットを指定できますか？たとえば、ユーザーには直接見えないようにしておきたい APIキーなどを指定できますか？
---

はい。次の手順に従ってください。

1. 次のコマンドを使って、run 用に指定された名前空間内に Kubernetes のシークレットを作成します:  
   `kubectl create secret -n <namespace> generic <secret_name> <secret_value>`

2. シークレットを作成したら、run の開始時にシークレットがインジェクトされるようにキューを設定します。シークレットを閲覧できるのはクラスター管理者のみであり、エンドユーザーからは参照できません。
