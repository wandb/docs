指定された run キューをポーリングし、キュー内の run を wandb Launch で起動する Launch エージェントクラスです。

```python
LaunchAgent(
    api: Api,
    config: Dict[str, Any]
)
```

| 引数       | 説明                             |
| :------- | :----------------------------- |
| `api`    | バックエンドへのリクエストに使用する API オブジェクト。 |
| `config` | エージェント用の設定を格納した辞書。             |

| 属性                       | 説明                            |
| :----------------------- | :---------------------------- |
| `num_running_jobs`       | スケジューラを除いたジョブ数を返します。          |
| `num_running_schedulers` | スケジューラの数のみを返します。              |
| `thread_ids`             | エージェントで実行中のスレッド ID のリストを返します。 |
