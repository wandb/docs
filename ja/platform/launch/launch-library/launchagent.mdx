指定された run キューをポーリングし、wandb Launch 用の run を起動する Launch エージェント クラスです。

```python
LaunchAgent(
    api: Api,
    config: Dict[str, Any]
)
```

| 引数       | 説明                                   |
| :------- | :----------------------------------- |
| `api`    | バックエンドにリクエストを送信するために使用する API オブジェクト。 |
| `config` | エージェント用の設定を表す辞書。                     |

| 属性                       | 説明                                 |
| :----------------------- | :--------------------------------- |
| `num_running_jobs`       | スケジューラー以外のジョブ数を返します。               |
| `num_running_schedulers` | スケジューラーの数のみを返します。                  |
| `thread_ids`             | エージェントで実行中のスレッド ID を表すキーのリストを返します。 |
