---

個人のユーザーアカウントでは、次のことができます。

* [プロフィールを編集](/ja/platform/app/settings-page/user-settings#profile)して、プロフィール画像、表示名、所在地、自己紹介、アカウントに関連付けられたメールアドレスを変更できます。
* [APIキーを設定](/ja/platform/app/settings-page/user-settings#api-keys)できます。
* run の[アラートを設定](/ja/platform/app/settings-page/user-settings#alerts)できます。
* [GitHubリポジトリをリンク](/ja/platform/app/settings-page/user-settings#personal-github-integration)できます。
* [アカウントを削除](/ja/platform/app/settings-page/user-settings#delete-your-account)できます。

詳細は[ユーザー設定](/ja/platform/app/settings-page/user-settings/)を参照してください。

チーム設定ページでは、次のことができます。

* チームの[メンバーを招待または削除](/ja/platform/app/settings-page/teams#members)できます。
* チームの run の[アラートを管理](/ja/platform/app/settings-page/teams#alerts)できます。
* [プライバシー設定を変更](/ja/platform/app/settings-page/teams#privacy)できます。
* [チームのストレージ使用量を表示・管理](/ja/platform/app/settings-page/teams#usage)できます。

チーム設定の詳細は、[Team settings](/ja/platform/app/settings-page/teams/)を参照してください。
