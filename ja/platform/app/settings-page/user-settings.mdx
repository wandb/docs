ユーザー プロフィールページにアクセスし、画面右上のユーザーアイコンをクリックします。表示されるドロップダウン メニューから **Settings** を選択します。

<div id="profile">
  ## プロフィール
</div>

**プロフィール** セクションでは、アカウント名や所属機関を管理・変更できます。必要に応じて、自己紹介、所在地、個人または所属機関のウェブサイトへのリンクを追加したり、プロフィール画像をアップロードしたりすることもできます。

<div id="edit-your-intro">
  ## 自己紹介を編集する
</div>

自己紹介を編集するには、プロフィール上部の **Edit** をクリックします。表示される WYSIWYG エディターでは Markdown を使用できます。

1. 行を編集するには、その行をクリックします。手早く操作するには、`/` を入力して一覧から Markdown を選択します。
2. 項目のドラッグハンドルをドラッグして移動します。
3. ブロックを削除するには、ドラッグハンドルをクリックしてから **Delete** をクリックします。
4. 変更を保存するには、**Save** をクリックします。

<div id="add-social-badges">
  ### ソーシャルバッジを追加する
</div>

X 上の `@weights_biases` アカウントのフォローバッジを追加するには、バッジ画像を指す HTML の `<img>` タグを含む Markdown 形式のリンクを追加できます。

```markdown
[![X: @weights_biases](https://img.shields.io/twitter/follow/weights_biases?style=social)](https://x.com/intent/follow?screen_name=weights_biases)
```

`<img>` タグでは、`width`、`height`、またはその両方を指定できます。どちらか一方のみを指定した場合、画像の縦横比は維持されます。

<div id="default-team">
  ## デフォルトチーム
</div>

複数のチームに所属している場合、**Default team** セクションで、run または Weave trace でチームが指定されていないときに使用されるデフォルトのチームを設定できます。1 つのチームにしか所属していない場合、そのチームがデフォルトとなり、このセクションは表示されません。

該当するタブを選択します。

<Tabs>
  <Tab title="Multi-tenant Cloud">
    **Default location to create new projects in** の横にあるドロップダウンをクリックし、デフォルトのチームを選択します。
  </Tab>

  <Tab title="Dedicated Cloud / Self-Managed">
    1. **Default location to create new projects in** の横にあるドロップダウンをクリックし、デフォルトのチームまたは自分の personal entity を選択します。
    2. (**任意**) 管理者が **Account** &gt; **Settings** &gt; **Privacy** で public projects を有効にしている場合は、新しいプロジェクトのデフォルトの公開範囲を設定します。**Default project privacy in your personal account** の横のボタンをクリックし、**Private** (デフォルト) または **Public** を選択します。
    3. (**任意**) 管理者が **Account** &gt; **Settings** &gt; **Privacy** で [default saving and diffing code](/ja/models/app/features/panels/code/) を有効にしている場合、run で有効化するには **Enable code saving in your personal account** をクリックします。
  </Tab>
</Tabs>

<Note>
  スクリプトを自動実行環境で実行する際にデフォルトのチームを指定するには、`WANDB_ENTITY` [environment variable](https://docs.wandb.ai/models/track/environment-variables) を使用してデフォルトのロケーションを指定できます。
</Note>

<div id="teams">
  ## Teams
</div>

**Teams** セクションには、所属しているすべてのチームが一覧表示されます。

1. チーム名をクリックすると、チームページに移動します。
2. 追加のチームに参加する権限がある場合は、**We found teams for you to join** の横にある **View teams** をクリックします。
3. 必要に応じて、**Hide teams in public profile** をオンにします。

<Note>
  チームを作成または管理するには、[Manage teams](/ja/platform/app/settings-page/teams/) を参照してください。
</Note>

<div id="api-keys">
  ## APIキー
</div>

**APIキー** セクションでは、W&amp;B サービスでの認証に使用する個人用APIキーを管理できます。

<div id="view-your-api-keys">
  ### APIキーを表示する
</div>

APIキーの表には次の情報が表示されます。

* **Key ID**: 各 APIキーの先頭部分で、識別に使用されます
* **Name**: キーの作成時に指定した説明用の名
* **Created**: キーが作成された日時
* **Last used**: 最後に使用されたタイムスタンプ

<Note>
  この表にはセキュリティ上、Key ID (キーの先頭部分) のみが表示されます。完全な secret APIキーは、作成時に一度だけ表示されます。
</Note>

キー名または ID の一部を入力して、APIキーの一覧を検索・絞り込みします。

<div id="delete-an-api-key">
  ### APIキーを削除する
</div>

APIキーを削除してアクセスを取り消すには、次の手順を実行します。

1. APIキー テーブルで、削除したいキーを見つけます。
2. キーの横にある削除ボタンをクリックします。
3. 削除を確認します。

<Warning>
  APIキーを削除すると、そのキーを使用しているすべてのスクリプトやサービスへのアクセスは即座に取り消されます。古いキーを削除する前に、すべてのシステムが新しいキーを使用するよう更新されていることを必ず確認してください。
</Warning>

<div id="beta-features">
  ## ベータ機能
</div>

**Beta Features** セクションでは、開発中の新しいプロダクトの先行プレビューや追加機能を必要に応じて有効にできます。有効にするベータ機能の横にあるトグルスイッチをオンにしてください。

<div id="alerts">
  ## アラート
</div>

run がクラッシュしたときや完了したとき、または [wandb.Run.alert()](/ja/models/runs/alert/) でカスタムアラートを設定したときに通知を受け取れます。通知はメールまたは Slack で受信できます。アラートを受け取りたいイベントタイプの横にあるスイッチをオンにします。

* **Runs finished**: Weights &amp; Biases の run が正常に完了したときの通知。
* **Run crashed**: run が正常に完了せず失敗した場合の通知。

アラートの設定と管理方法の詳細は、[wandb.Run.alert() でアラートを送信する](/ja/models/runs/alert/) を参照してください。

<div id="personal-github-integration">
  ## 個人 GitHub インテグレーション
</div>

個人の GitHub アカウントを接続します。GitHub アカウントを接続するには、次の手順を実行します。

1. **Connect Github** ボタンを選択します。OAuth 認可ページにリダイレクトされます。
2. **Organization access** セクションで、アクセスを許可する組織を選択します。
3. **Authorize** **wandb** を選択します。

<div id="delete-your-account">
  ## アカウントを削除する
</div>

アカウントを削除するには、**Delete Account** ボタンをクリックします。

<Warning>
  アカウントの削除は元に戻すことができません。
</Warning>

<div id="storage">
  ## ストレージ
</div>

**Storage** セクションには、アカウントが Weights &amp; Biases サーバー上で使用している合計ストレージ容量が表示されます。デフォルトのストレージプランは 100GB です。ストレージと料金の詳細については、[Pricing](https://wandb.ai/site/pricing) ページを参照してください。
