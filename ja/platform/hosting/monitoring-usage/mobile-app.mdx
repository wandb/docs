iOS 向け W&amp;B Mobile App を使えば、どこにいても W&amp;B Models プロジェクトとのつながりを保てます。iPhone や iPad から、トレーニングをリアルタイムでトラッキングし、折れ線グラフを表示して、プロジェクトの履歴を探索し、チームの進捗を追えます。

<Info>
  The mobile app is only available for [Multi-tenant Cloud](/ja/platform/hosting/hosting-options/multi_tenant_cloud) accounts. It is not available for [Dedicated Cloud](/ja/platform/hosting/hosting-options/dedicated-cloud) or [Self-Managed](/ja/platform/hosting/hosting-options/self-managed) deployments.
</Info>

<div id="features">
  ## 機能
</div>

* **トレーニング Runs を追跡する**: Run のステータス、メトリクス、折れ線グラフをリアルタイムで確認できます。
* **プロジェクトの履歴を探索する**: W&amp;B のプロジェクト全体を閲覧、検索できます。**プロジェクト パネル** のメトリクス チャートには、最新の Run だけでなく、そのプロジェクト内の **すべての Runs** のデータが含まれます。
* **チームを切り替える**: **チーム ピッカー** を使用してチームを切り替えます。このピッカーでは、すばやく移動できるように更新されたレイアウトを採用しています。
* **常に最新情報を把握する**: 通知を使うと、ノート PC を開かなくても実験の更新を受け取れます。不要になった **アラート** は、**Notifications** タブでスワイプして削除できます。

<div id="set-up-notifications">
  ## 通知を設定する
</div>

通知は、次の 2 つの方法で設定できます。

**メトリクスしきい値アラート**

1. Run に移動します。
2. グラフをタップして全画面表示にします。
3. 右上のベルアイコンをタップします。
4. 今後の Runs が指定したメトリクスしきい値を超えたときに通知されるよう設定します。

**Run failure alerts**

1. プロジェクトに移動します。
2. 右上の 3 点メニューをタップします。
3. **Run failed alert** を選択すると、そのプロジェクトで今後発生する Run の失敗に対する通知を受け取れます。

**通知管理** 画面では、両方の種類の設定が 1 か所に一覧表示され、各セクションで制御される内容を説明する短い説明も確認できます。
