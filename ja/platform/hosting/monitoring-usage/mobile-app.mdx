iOS 向け W&amp;B Mobile App を使うと、どこにいても W&amp;B Models プロジェクトを確認できます。iPhone や iPad から、トレーニング run をリアルタイムでトラッキングし、折れ線グラフを表示して、プロジェクトの履歴を確認し、チームの進捗を把握できます。

<Info>
  このモバイルアプリは、[Multi-tenant Cloud](/ja/platform/hosting/hosting-options/multi_tenant_cloud) アカウントでのみ利用できます。[Dedicated Cloud](/ja/platform/hosting/hosting-options/dedicated-cloud) または [Self-Managed](/ja/platform/hosting/hosting-options/self-managed) のデプロイでは利用できません。
</Info>

<div id="features">
  ## 機能
</div>

* **トレーニング run を追跡する**: run のステータス、メトリクス、折れ線グラフをリアルタイムで確認できます。
* **プロジェクトの履歴を確認する**: W&amp;B のプロジェクト全体を閲覧して検索できます。**プロジェクト パネル** のメトリクス チャートには、最新の run だけでなく、プロジェクト内の **すべての run** のデータが含まれます。
* **チームを切り替える**: **チーム ピッカー** を使用してチームを切り替えます。ピッカーは、すばやく移動できるよう更新されたレイアウトになっています。
* **常に最新情報を把握する**: 通知を使用すると、ノート PC を開かなくても実験の更新を受け取れます。不要になった **アラート** は、**通知** タブでスワイプして削除します.

<div id="set-up-notifications">
  ## 通知を設定する
</div>

通知は次の 2 つの方法で設定できます。

**メトリクスしきい値アラート**

1. run に移動します。
2. グラフをタップして全画面表示に切り替えます。
3. 右上のベル アイコンをタップします。
4. 今後の run が指定したメトリクスのしきい値を超えたときに通知されるよう設定します。

**run 失敗アラート**

1. プロジェクトに移動します。
2. 右上の 3 点メニューをタップします。
3. **Run failed alert** を選択すると、そのプロジェクトで今後 run が失敗したときの通知を受け取れます。

**通知を管理** 画面では、これら 2 種類の設定が 1 か所に一覧表示され、各セクションで制御する内容を説明する短い説明も表示されます。
