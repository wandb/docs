iOS 向け W&amp;B Mobile App を使うと、どこにいても W&amp;B Models プロジェクトを確認できます。iPhone や iPad から、トレーニング run をリアルタイムでトラッキングし、折れ線グラフを表示して、プロジェクトの履歴を確認し、チームの進捗を把握できます。

<Info>
  このモバイルアプリは、[Multi-tenant Cloud](/ja/platform/hosting/hosting-options/multi_tenant_cloud) アカウントでのみ利用できます。[専用クラウド](/ja/platform/hosting/hosting-options/dedicated-cloud) または [セルフマネージド](/ja/platform/hosting/hosting-options/self-managed) のデプロイでは利用できません。
</Info>

<div id="features">
  ## 機能
</div>

* **トレーニング run をトラッキングする**: run のステータス、メトリクス、折れ線グラフをリアルタイムで確認できます。
* **プロジェクトの履歴を確認する**: W&amp;B のプロジェクト全体を閲覧して検索できます。**プロジェクト パネル** のメトリクス チャートには、最新の run だけでなく、プロジェクト内の **すべての run** のデータが含まれます。
* **チームを切り替える**: **チーム ピッカー** を使用してチームを切り替えます。ピッカーは、すばやく移動できるよう更新された Layout になっています。
* **常に最新情報を把握する**: 通知を使用すると、ノート PC を開かなくても Experiments の更新を受け取れます。不要になった**アラート**は、**通知**タブでスワイプして削除します。

<div id="set-up-notifications">
  ## 通知を設定する
</div>

通知は2つの方法で設定できます。

**メトリクスしきい値アラート**

1. runにアクセスします。
2. グラフをタップしてフルスクリーン表示に切り替えます。
3. 右上のベルアイコンをタップします。
4. 今後のrunが指定したメトリクスのしきい値を超えたときに通知が送信されるよう設定します。

**run失敗アラート**

1. プロジェクトにアクセスします。
2. 右上の三点メニューをタップします。
3. **Run failed alert** を選択すると、そのプロジェクトで今後runが失敗した際に通知を受け取れます。

**通知管理**画面では、両方の種類の設定を一か所で確認でき、各セクションが何を制御するかを説明する簡単な説明が表示されます。
