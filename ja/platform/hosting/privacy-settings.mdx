---
title: プライバシー設定を設定する
description: "W&B で、チームの可視性、プロジェクトへのアクセス、コード保存のデフォルトに関するプライバシー設定を設定し、適用します。"
---

組織管理者とチーム管理者は、それぞれ組織スコープとチームスコープで一連のプライバシー設定を設定できます。組織スコープで設定した場合、組織管理者はその組織内のすべてのチームにそれらの設定を適用できます。

<Note>
  チームのワークフローに予期しない変更が生じるのを防ぐため、W&amp;B では、組織管理者がプライバシー設定を適用する場合は、事前にその内容を組織内のすべてのチーム管理者とユーザーに周知してから行うことを推奨しています。
</Note>

<div id="enforce-privacy-settings-for-all-teams">
  ## すべてのチームにプライバシー設定を適用する
</div>

組織管理者は、アカウントまたは組織のダッシュボードにある **Settings** タブの **Privacy** セクションから、組織内のすべてのチームにプライバシー設定を適用できます。組織管理者が設定を適用すると、チーム管理者は各チームでその設定を変更できなくなります。

* **チームの可視性制限を適用する** - 組織全体で、メンバー以外にはすべてのチームを非表示にします
* **今後のプロジェクトのプライバシーを適用する** - すべてのチームで新規作成されるすべてのプロジェクトを、非公開または[restricted](./iam/access-management/restricted-projects)にすることを必須にします
* **招待制御を適用する** - 管理者以外のメンバーがチームにメンバーを招待できないようにします
* **レポート共有制御を適用する** - 非公開プロジェクト内のレポートの公開共有を無効にし、既存のマジックリンクも無効化します
* **チームへの自動参加制限を適用する** - 組織のメールドメインと一致するユーザーが、自動的にチームへ参加できないように制限します
  <Note>この設定は、[Multi-tenant Cloud](./hosting-options/multi_tenant_cloud)デプロイでのみ利用できます。</Note>
* **デフォルトのコード保存制限を適用する** - すべてのチームで、デフォルトでコード保存を無効にします

組織のプライバシー設定を開くには、次の手順に従います。

1. 組織管理者としてサインインします。
2. Multi-tenant Cloud では、`https://wandb.ai/account-settings/<organization>/settings` にアクセスします。`<organization>` はご自身の組織名に置き換えてください。専用クラウドまたはセルフマネージドのデプロイでは、インスタンスの組織ダッシュボードを開きます。環境固有のURLについては、[Add and manage users](/ja/platform/hosting/iam/access-management/manage-organization#add-and-manage-users)を参照してください。
3. **Privacy** セクションを開き、**Enforce default code saving restrictions** と、必要なその他の組織全体の制御を設定します。

<div id="configure-privacy-settings-for-a-team">
  ## チームのプライバシー設定を設定する
</div>

チーム管理者は、チームの **Settings** タブ内にある **Privacy** セクションから、各チームのプライバシー設定を設定できます。各設定は、組織スコープで強制されていない限り変更できます。

* **このチームをすべての非メンバーから非表示にする** - メンバーではないユーザーにはこのチームが表示されないようにします
* **今後このチームで作成されるすべてのプロジェクトを非公開にする** - チームで新規作成されるすべてのプロジェクトを非公開にします (公開共有はできません)
* **すべてのチームメンバーが他のメンバーを招待できるようにする** - 管理者だけでなく、すべてのチームメンバーが新しいメンバーを招待できるようにします
* **非公開プロジェクト内のレポートの公開共有を無効にする** - レポートの公開共有を無効にし、既存のマジックリンクも無効化します
* **組織のメールドメインが一致するユーザーにこのチームへの参加を許可する** - 同じメールドメインを持つユーザーがこのチームに自動的に参加できるようにします
  <Note>この設定は、[Multi-tenant Cloud](./hosting-options/multi_tenant_cloud) デプロイでのみ使用できます。</Note>
* **デフォルトでコード保存を有効にする** - チーム内のすべての run でコードを自動保存します

チームのプライバシー設定を開くには:

1. `https://wandb.ai/<team>` にアクセスします。`<team>` は自分のチーム名に置き換えてください。
2. 左側の navigation で **Team settings** を選択します。
3. **Privacy** セクションを開き、**デフォルトでコード保存を有効にする** を設定します。
