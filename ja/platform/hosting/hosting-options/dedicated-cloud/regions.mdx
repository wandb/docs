title: Dedicated Cloud でサポートされるリージョン
description: "W&B Dedicated Cloud インスタンスで利用可能な、サポートされる AWS、Google Cloud、Azure の全リージョンを表示します。"
---

AWS、Google Cloud、Azure は、世界各地の複数のリージョンでクラウドコンピューティングサービスを提供しています。グローバルリージョンを利用することで、データ所在地とコンプライアンス、レイテンシ、コスト効率などに関する要件を満たしやすくなります。W&amp;B は、Dedicated Cloud で利用可能なグローバルリージョンの多くをサポートしています。

<Note>
  ご希望の AWS、Google Cloud、または Azure のリージョンが一覧にない場合は、W&amp;B Support にお問い合わせください。W&amp;B では、該当リージョンで Dedicated Cloud に必要なすべてのサービスが利用可能かを確認し、評価結果に応じてサポートを優先できます。
</Note>
