title: サポートされる Dedicated Cloud リージョン
description: "W&B Dedicated Cloud のデプロイで利用可能な、サポートされる AWS、Google Cloud、および Azure リージョンをすべて表示します。"
---

AWS、Google Cloud、および Azure は、世界各地の複数のロケーションでクラウドコンピューティングサービスを提供しています。グローバルリージョンを活用することで、データ所在地とコンプライアンス、レイテンシ、コスト効率などに関する要件を満たすことができます。W&amp;B は、Dedicated Cloud 向けに利用可能な多くのグローバルリージョンをサポートしています。

<Note>
  希望する AWS、Google Cloud、または Azure リージョンが一覧に含まれていない場合は、W&amp;B Support までお問い合わせください。W&amp;B は、対象リージョンに Dedicated Cloud が必要とするすべてのサービスが揃っているかを検証し、評価結果に応じてサポート対応の優先度を調整できます。
</Note>
