---
title: サポートされている Dedicated Cloud リージョン
---

AWS、Google Cloud、および Azure は、世界各地の複数のロケーションでクラウドコンピューティングサービスを提供しています。グローバル リージョンを利用することで、データ所在地とコンプライアンス、レイテンシー、コスト効率などに関する要件を満たすことができます。W&B は、Dedicated Cloud 向けに利用可能な多くのグローバル リージョンをサポートしています。

<Note>
希望する AWS、Google Cloud、または Azure リージョンが一覧に含まれていない場合は、W&B Support までお問い合わせください。W&B は、対象リージョンに Dedicated Cloud が必要とするすべてのサービスが揃っているかを検証し、評価結果に応じてサポート対応の優先度を調整できます。
</Note>
