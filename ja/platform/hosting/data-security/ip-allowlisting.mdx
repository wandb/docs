---
title: 専用クラウドの IP 許可リストを設定する
description: "IP 許可リストを使用して、W&B Dedicated Cloud インスタンスへの 액세스 を承認済み IP アドレスに制限します。"
---

[専用クラウド](/ja/platform/hosting/hosting-options/dedicated-cloud) インスタンスへの 액세스 は、承認済み IP アドレスのリストのみに制限できます。これは、AI ワークロードから W&amp;B API への 액세ス と、ユーザーのブラウザーから W&amp;B App UI への 액세스 の両方に適用されます。Dedicated Cloud インスタンスに IP 許可リストを設定すると、W&amp;B は承認されていない他の location からのあらゆる Request を拒否します。Dedicated Cloud インスタンスの IP 許可リストを設定するには、your W&amp;B team にお問い合わせください。

IP 許可リストは、AWS、Google Cloud、Azure 上の Dedicated Cloud instances で 사용 가능합니다。

IP 許可リストは、[安全なプライベート接続](./private-connectivity) と併用できます。IP 許可リストを安全なプライベート接続と併用する場合、W&amp;B は、可能であれば AI ワークロードからのすべての トラフィック と、ユーザーのブラウザーからの大半の トラフィック には安全なプライベート接続を使用し、特権 location からのインスタンス管理には IP 許可リストを使用することを推奨します。

<Warning>
  W&amp;B は、個別の `/32` IP アドレスではなく、企業または業務用の外向きゲートウェイに割り当てられた [CIDR blocks](https://en.wikipedia.org/wiki/Classless_Inter-Domain_Routing) を使用することを強く推奨します。個別の IP アドレスの使用はスケーラブルではなく、クラウドごとに厳しい 制限 があります。
</Warning>
