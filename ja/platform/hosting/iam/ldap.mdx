---
title: LDAP で SSO を設定する
---

W&amp;B Server の LDAP サーバーを使って認証情報の認証を行います。以下のガイドでは、W&amp;B Server の設定方法について説明します。必須項目と任意項目の設定に加え、system settings UI から LDAP 接続を設定する手順についても扱います。また、アドレス、ベース識別名、属性など、LDAP 設定のさまざまな入力項目に関する情報も提供します。これらの属性は、W&amp;B App UI または環境変数で指定できます。匿名バインド、または管理者 DN とパスワードを使用したバインドのいずれかを選択して設定できます。

{/* <Note>
  W&B Team Admin として、匿名バインドか、管理者 DN とパスワードを使用したバインドのいずれかを設定できます。
  </Note> */}

<Note>
  LDAP 認証を有効化して設定できるのは W&amp;B Admin ロールのみです。
</Note>


<div id="configure-ldap-connection">
  ## LDAP 接続を設定する
</div>

<Tabs>
<Tab title="W&B App">
1. W&B App に移動します。 
2. 右上のプロフィールアイコンを選択し、ドロップダウンから **System Settings** を選択します。 
3. **Configure LDAP Client** をオンにします。
4. フォームに必要な情報を入力します。各入力項目の詳細については **Configuring Parameters** セクションを参照してください。
5. 設定をテストするには **Update Settings** をクリックします。これにより W&B サーバーとのテストクライアント／接続が確立されます。
6. 接続が検証されたら、**Enable LDAP Authentication** をオンにし、**Update Settings** ボタンをクリックします。
</Tab>
<Tab title="環境変数">
次の環境変数を使用して LDAP 接続を設定します:

| 環境変数                      | 必須     | 例                              |
| ----------------------------- | -------- | ------------------------------- |
| `LOCAL_LDAP_ADDRESS`          | はい     | `ldaps://ldap.example.com:636`  |
| `LOCAL_LDAP_BASE_DN`          | はい     | `email=mail,group=gidNumber`    |
| `LOCAL_LDAP_BIND_DN`          | いいえ   | `cn=admin`, `dc=example,dc=org` |
| `LOCAL_LDAP_BIND_PW`          | いいえ   |                                 |
| `LOCAL_LDAP_ATTRIBUTES`       | はい     | `email=mail`, `group=gidNumber` |
| `LOCAL_LDAP_TLS_ENABLE`       | いいえ   |                                 |
| `LOCAL_LDAP_GROUP_ALLOW_LIST` | いいえ   |                                 |
| `LOCAL_LDAP_LOGIN`            | いいえ   |                                 |

各環境変数の定義については、[設定パラメーター](#configuration-parameters) セクションを参照してください。わかりやすくするために、定義で使用している名称から環境変数プレフィックス `LOCAL_LDAP` は省略されています。
</Tab>
</Tabs>

<div id="configuration-parameters">
  ## 設定パラメーター
</div>

次の表は、必須および任意の LDAP 設定を一覧し、その説明を示します。

| Environment variable | Definition              | Required |
| -------------------- | ----------------------- | -------- |
| `ADDRESS`            | VPC 内で W&B Server をホストしている LDAP サーバーのアドレスです。      | Yes      |
| `BASE_DN`            | 検索を開始するルートパスであり、このディレクトリに対してあらゆるクエリを行うために必要です。             | Yes      |
| `BIND_DN`            | LDAP サーバーに登録されている管理者ユーザーのパスです。LDAP サーバーが未認証バインドをサポートしない場合に必要です。指定された場合、W&B Server はこのユーザーとして LDAP サーバーに接続します。指定されていない場合、W&B Server は匿名バインドで接続します。 | No       |
| `BIND_PW`            | 管理者ユーザーのパスワードで、バインドを認証するために使用されます。空のままにした場合、W&B Server は匿名バインドで接続します。   | No       |
| `ATTRIBUTES`         | メールアドレスおよびグループ ID の属性名をカンマ区切りの文字列で指定します。    | Yes      |
| `TLS_ENABLE`         | TLS を有効にします。                | No       |
| `GROUP_ALLOW_LIST`   | グループ allowlist です。           | No       |
| `LOGIN`              | W&B Server に LDAP を使用して認証を行うよう指示します。`True` または `False` のいずれかを設定します。LDAP 設定をテストする場合は、任意でこれを `False` に設定します。LDAP 認証を開始するには、`True` に設定します。 | No       |
