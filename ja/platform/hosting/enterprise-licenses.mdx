エンタープライズライセンスは、セキュリティ、コンプライアンス、または管理面で強化された機能を必要とする組織向けに設計された W&amp;B の機能を有効化します。[エンタープライズ機能](#enterprise-features)の一覧を参照してください。

* **セキュリティ**: 認証、暗号化、アクセス制御の強化。
* **コンプライアンス**: 監査ログ、HIPAA 準拠オプション、データガバナンスの制御機能。
* **管理**: SCIM プロビジョニング、カスタムロール、自動化機能などのユーザー管理機能。

このページでは、エンタープライズライセンスについて、そのライセンスで有効になる機能、取得方法、およびご利用のデプロイメント環境向けの設定方法を包括的に説明します。

<div id="deployment-options">
  ## デプロイメントオプション
</div>

Enterprise ライセンスは、W&amp;B Dedicated Cloud と Self-Managed デプロイメントで利用できます。

* [W&amp;B Dedicated Cloud](/ja/platform/hosting/hosting-options/dedicated_cloud) は、W&amp;B が管理し Google Cloud 上にデプロイするシングルテナントのインフラストラクチャを提供します。Dedicated Cloud デプロイメントには自動的に Enterprise ライセンスが含まれます。追加の設定は不要で、プロビジョニング後すぐにすべての Enterprise 機能を利用できます。
* [W&amp;B Self-Managed](/ja/platform/hosting/hosting-options/self-managed) は、自社のインフラストラクチャ上にデプロイする方式で、デプロイメントおよびデータを完全に制御できます。エアギャップ環境でのデプロイメントもサポートしています。Enterprise ライセンスがない場合、[Enterprise 機能](#enterprise-features)は利用できません。

<div id="obtain-an-enterprise-license">
  ## Enterprise ライセンスを取得する
</div>

このセクションでは、Enterprise ライセンスの取得方法について説明します。利用ケースに応じてタブを選択して、次のステップに進んでください。

<Tabs>
  <Tab title="Dedicated Cloud">
    W&amp;B Dedicated Cloud には Enterprise ライセンスが含まれているため、追加の操作は不要です。[Enterprise トライアルを開始](https://wandb.ai/site/for-enterprise/dedicated-saas-trial/)するか、要件について相談するには [Sales](https://wandb.ai/site/contact-sales) に問い合わせてください。
  </Tab>

  <Tab title="Self-Managed">
    Enterprise ライセンスのリクエスト方法:

    * **評価**: [トライアルライセンスのリクエスト](https://wandb.ai/site/for-enterprise/self-hosted-trial) を送信するか、要件について相談するには [Sales](mailto:sales@wandb.ai) に問い合わせてください。
    * **本番**: デプロイ要件と規模について相談し、ライセンス情報を受け取るには [Sales](https://wandb.ai/site/contact-sales) に問い合わせてください。

    <Warning>ライセンスキーは安全に保管し、機密性の高い認証情報として取り扱ってください。</Warning>
  </Tab>
</Tabs>

<div id="configure-an-enterprise-license">
  ## Enterprise ライセンスを設定する
</div>

<Info>このセクションは W&amp;B Self-Managed の場合にのみ該当します。</Info>

[Enterprise ライセンスを取得](#obtain-an-enterprise-license)したら、プログラムから設定するか、デプロイ後に W&amp;B App から設定できます。手順はタブを選択して確認してください。

<Tabs>
  <Tab title="Helm">
    1. `values.yaml` ファイルを更新します:
       ```yaml
       wandb:
         license: "YOUR_LICENSE_KEY_HERE"
       ```

    2. Helm リリースをアップグレードします:
       ```bash
       helm upgrade wandb wandb/wandb \
         --namespace wandb \
         --reuse-values \
         --set license=$LICENSE_KEY
       ```
  </Tab>

  <Tab title="Terraform">
    1. Terraform 変数を更新します:
       ```hcl
       variable "license" {
         default = "YOUR_LICENSE_KEY_HERE"
       }
       ```

    2. Terraform 設定を適用します:
       ```bash
       terraform apply -var="license=$LICENSE_KEY"
       ```
  </Tab>

  <Tab title="W&B App">
    環境変数 の設定を行っていない既存のデプロイメントの場合:

    1. 管理者ユーザーとしてログインします
    2. **System Settings** に移動します
    3. **License** セクションを探します
    4. 新しいライセンスキーを入力します
    5. 変更を保存します
  </Tab>

  <Tab title="Docker">
    <Warning>Docker 上への W&amp;B のデプロイは非推奨です。W&amp;B は、Helm または Terraform で管理される [Kubernetes Operator]() デプロイメントへの移行を強く推奨します。詳細については [リファレンスアーキテクチャ](/ja/platform/hosting/self-managed/ref-arch.mdx) を参照するか、[サポート](mailto:support@wandb.ai) にお問い合わせください。</Warning>
    ライセンスキーを環境変数として設定します:

    ```bash
    docker run -d \
      -e LICENSE="YOUR_LICENSE_KEY_HERE" \
      -e OTHER_ENV_VARS \
      wandb/local
    ```
  </Tab>
</Tabs>

次に、[Enterprise ライセンスを検証する](#verify-or-update-an-enterprise-license) に進みます。

<div id="verify-and-manage-an-enterprise-license">
  ## Enterprise ライセンスの確認と管理
</div>

インスタンスまたは組織の管理者として、次の手順に従って Enterprise ライセンスを確認および管理します。

1. W&amp;B App で右上のユーザー プロフィールアイコンをクリックし、**System Console** をクリックします。
2. **License** タブを開きます。ダッシュボードには、そのライセンスの利用権、期限、利用状況の推移が要約表示されます。エラーが表示される場合は、[Common issues](#common-issues) を参照してください。
3. ライセンスおよびその利用権の詳細情報を確認するには、**View license online** をクリックします。

   このページから、新しいトライアルライセンスまたは本番ライセンスを申請することもできます。ページ上部の **New license order** をクリックし、フォームを送信します。
4. 期限切れのライセンスを更新するか、ライセンスキーを更新するには、**Update License** をクリックします。
   <Tip>
     サービスを中断なくご利用いただけるよう、W&amp;B から次のタイミングでご連絡します:

     * ライセンス有効期限の 30 日前
     * ライセンスの有効期限切れ時
   </Tip>

<div id="common-issues">
  ## よくある問題
</div>

<div id="license-not-recognized">
  ### ライセンスが認識されない
</div>

* ライセンスキーの形式が正しいか（余分なスペースがないか）確認する
* ライセンスが有効期限切れになっていないことを確認する
* ライセンスが正しい設定箇所に設定されていることを確認する

<div id="features-not-available-after-setting-license">
  ### ライセンス設定後に利用できない機能がある
</div>

* ライセンスを設定した後に W&amp;B サービスを再起動する
* 使用しようとしている特定の機能がライセンスに含まれているか確認する
* システムログを確認し、ライセンス検証エラーが出力されていないか確認する

<div id="license-expiration-warnings">
  ### ライセンス有効期限の警告
</div>

* System Settings ページを確認して、有効期限に関する通知がないか確認する
* 監視システムでライセンス有効期限のアラートを設定する
* アカウントチームの連絡先情報を最新の状態に保つ

<div id="get-support">
  ### サポートを受ける
</div>

Enterprise ライセンスに関するサポートが必要な場合は、ご担当のアカウントチーム、[Sales](mailto:sales@wandb.ai)、または [Support](mailto:support@wandb.ai) までお問い合わせください。
