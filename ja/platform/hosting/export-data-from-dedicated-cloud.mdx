description: "W&B SDK API を使用して、W&B Dedicated Cloud インスタンスから Runs、メトリクス、Artifacts、Reports をエクスポートします。"
title: Dedicated Cloud からデータをエクスポートする
---

Dedicated Cloud インスタンスで管理されているすべてのデータをエクスポートする場合は、[Import and Export API](/ja/models/ref/python/public-api/) から W&amp;B SDK API を使用して、Runs、メトリクス、Artifacts などを取得できます。次の表に、主なエクスポートのユースケースを示します。

| 目的                      | ドキュメント                                                                                                                                                                                                        |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| プロジェクトのメタデータをエクスポートする   | [Projects API](/ja/models/ref/python/public-api/projects)                                                                                                                                                        |
| プロジェクト内の Runs をエクスポートする | [Runs API](/ja/models/ref/python/public-api/runs)                                                                                                                                                                |
| Reports をエクスポートする       | [Report and Workspace API](/ja/models/reports/clone-and-export-reports/)                                                                                                                                         |
| Artifacts をエクスポートする     | [Explore artifact graphs](/ja/models/artifacts/explore-and-traverse-an-artifact-graph/), [Download and use artifacts](/ja/models/artifacts/download-and-use-an-artifact/#download-and-use-an-artifact-stored-on-wb) |

Dedicated Cloud に保存された Artifacts を [Secure Storage Connector](/ja/platform/app/settings-page/teams/#secure-storage-connector) で管理している場合は、W&amp;B SDK API を使用して Artifacts をエクスポートする必要がないことがあります。

<Note>
  W&amp;B SDK API を使用してすべてのデータをエクスポートすると、Runs や Artifacts などの数が多い場合は時間がかかることがあります。W&amp;B では、Dedicated Cloud インスタンスに過度な負荷をかけないよう、適切なサイズのバッチに分けてエクスポート処理を実行することを推奨しています。
</Note>
