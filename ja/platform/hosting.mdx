W&amp;B は、組織のインフラストラクチャーおよびセキュリティ要件に応じて、さまざまな方法でデプロイできます。このページでは、管理者やプラットフォームの意思決定者が利用可能なデプロイメントオプションを比較し、セキュリティ、スケール、運用上のニーズに最も適したものを選択できるようにします。

<Tip title="はじめに">
  すぐに利用を開始するには、W&amp;B は [Multi-tenant Cloud](https://wandb.ai/home) を推奨します。

  一部の機能は [Enterprise](https://wandb.ai/site/pricing/) ライセンスが必要です。これらの機能を試すには、[Enterprise trial](https://wandb.ai/site/enterprise-trial/) にサインアップして、デプロイメントタイプを選択してください。
</Tip>

<CardGroup cols={1}>
  <a id="wb-multi-tenant-cloud" aria-label="W&B Multi-tenant Cloud" />

  <Card title="W&B Multi-tenant Cloud">
    W&amp;B のクラウドインフラストラクチャー上にデプロイされる完全マネージドサービスです。必要な規模で W&amp;B プロダクトにアクセスでき、コスト効率に優れた料金オプションを利用できるほか、最新の機能を継続的に利用できます。プライベートデプロイメントレベルのセキュリティが不要で、セルフサービスでのオンボーディングが重要であり、コスト効率を重視する場合、W&amp;B はプロダクトの試用や本番 AI ワークフローの運用に Multi-tenant Cloud を推奨します。

    詳細は、[W&amp;B Multi-tenant Cloud](/ja/platform/hosting/hosting-options/multi_tenant_cloud) を参照してください。
  </Card>

  <a id="wb-dedicated-cloud" aria-label="W&B 専用クラウド" />

  <Card title="W&B 専用クラウド">
    W&amp;B のクラウド内で、専用かつ分離されたインフラストラクチャー上に構築される完全マネージドサービスです。データレジデンシーを含む厳格なガバナンス管理への準拠が必要な場合、高度なセキュリティ機能が求められる場合、またはセキュリティ、スケール、パフォーマンスの要件を満たすインフラストラクチャーを自社で構築・管理せずに AI の運用コストを最適化したい場合は、このオプションを選択してください。

    詳細は、[W&amp;B 専用クラウド](/ja/platform/hosting/hosting-options/dedicated-cloud) を参照してください。
  </Card>

  <a id="wb-Self-Managed" aria-label="W&B セルフマネージド" />

  <Card title="W&B セルフマネージド">
    お客様が管理する独自のインフラストラクチャー上に W&amp;B Server をデプロイして運用します。W&amp;B Server は、W&amp;B Platform と、それによってサポートされる W&amp;B プロダクトを実行するための自己完結型のパッケージ製品です。既存のインフラストラクチャーがすべてオンプレミスにある場合や、W&amp;B 専用クラウド では満たせない厳格な規制要件がある場合、W&amp;B はこのオプションを推奨します。このオプションでは、W&amp;B Server を支えるために必要なインフラストラクチャーのプロビジョニング、継続的な保守、アップグレードについて、すべてお客様が責任を負います。

    詳細は、[W&amp;B セルフマネージド](/ja/platform/hosting/hosting-options/self-managed) を参照してください。
  </Card>
</CardGroup>
