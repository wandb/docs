<div id="product-comparison">
  ## プロダクト比較
</div>

Weights &amp; Biases へようこそ！当社のプロダクトを使い始める前に、どのプロダクトがご利用のユースケースに適しているかを見極めることが重要です。

| Product                                   | Best For             | Key Features                                |
| ----------------------------------------- | -------------------- | ------------------------------------------- |
| **[W&amp;B Models](#w%26b-models)**       | MLモデルをゼロからトレーニングする場合 | 実験管理、ハイパーパラメーター最適化、モデル Registry、可視化         |
| **[W&amp;B Weave](#w%26b-weave)**         | LLMアプリケーションの構築       | トレース、プロンプト管理、評価、本番AIアプリのコスト追跡               |
| **[W&amp;B Inference](#w%26b-inference)** | 事前トレーニング済みモデルの使用     | ホスト型のオープンソースモデル、API 액세스、テスト用のモデル playground |
| **[W&amp;B Training](#w%26b-training)**   | モデルのファインチューニング       | 強化学習を用いた LoRA とカスタムモデル適応の作成とデプロイ            |
