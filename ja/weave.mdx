W&amp;B Weave は、LLM アプリケーションを追跡・評価・改善するための観測性および評価プラットフォームです。Weave を使うと、次のことができます:

* LLM アプリケーションを[観測してデバッグする](/ja/weave/quickstart)
* LLM ジャッジとカスタムスコアラーを使って、アプリケーションの応答を[評価する](/ja/weave/tutorial-eval)

<div id="install-weave">
  ## Weave をインストールする
</div>

W&amp;B の Weave は、Python と TypeScript 向けのライブラリを提供します。Weave ライブラリをインストールするには、次のコマンドを実行します：

<Tabs>
  <Tab title="Python">
    ```bash
    pip install weave
    ```
  </Tab>

  <Tab title="TypeScript">
    ```bash
    pnpm install weave
    ```
  </Tab>
</Tabs>

Weave ライブラリを使い始めるには、[Weights &amp; Biases (W&amp;B) アカウント](https://wandb.ai)と、[User Settings での APIキー](https://wandb.ai/settings) を作成します。APIキーを使うことで、W&amp;B アカウントに対して認証を行い、データの送信を開始できます。
