W&amp;B Weave は、LLM アプリケーションのトラッキング、評価、改善を支援するオブザーバビリティおよび評価プラットフォームです。Weave を使うと、次のことができます。

* LLM アプリケーションを [観測およびデバッグ](/ja/weave/quickstart) する
* LLM ジャッジモデルやカスタム スコアラーを使って、アプリケーションの応答を [評価](/ja/weave/tutorial-eval) する

<div id="install-weave">
  ## Weave をインストールする
</div>

W&amp;B Weave は、Python と TypeScript のライブラリを提供します。Weave ライブラリをインストールするには、以下のコマンドを実行してください：

<Tabs>
  <Tab title="Python">
    ```bash
    pip install weave
    ```
  </Tab>

  <Tab title="TypeScript">
    ```bash
    pnpm install weave
    ```
  </Tab>
</Tabs>

Weave ライブラリを使用するには、[Weights &amp; Biases (W&amp;B) アカウント](https://wandb.ai) を作成し、[User Settings で API キー](https://wandb.ai/settings) を生成してください。API キーにより、W&amp;B アカウントに対して認証を行い、データを送信できるようになります。
