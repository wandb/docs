W&amp;B Weave は、LLM アプリケーションのトラッキング、評価、改善を支援するオブザーバビリティおよび評価プラットフォームです。Weave を使うと、次のことができます。

* LLM アプリケーションを [監視およびデバッグ](/ja/weave/quickstart) する
* LLM ジャッジモデルやカスタムスコアラーを使って、アプリケーションの応答を [評価](/ja/weave/tutorial-eval) する

<div id="install-weave">
  ## Weave のインストール
</div>

W&amp;B Weave は Python と TypeScript のライブラリを提供しています。Weave ライブラリをインストールするには、次のコマンドを実行します。

<Tabs>
  <Tab title="Python">
    ```bash
    pip install weave
    ```
  </Tab>

  <Tab title="TypeScript">
    ```bash
    pnpm install weave
    ```
  </Tab>
</Tabs>

Weave ライブラリの使用を開始するには、[Weights &amp; Biases (W&amp;B) アカウント](https://wandb.ai) と [User Settings での APIキー](https://wandb.ai/settings) を作成します。APIキー を使って W&amp;B アカウントに認証し、データの送信を開始できます。
