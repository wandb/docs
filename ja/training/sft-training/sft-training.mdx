---
title: Serverless SFTの使い方
description: "OpenPipe の ART フレームワークと W&B Training API を使用して、教師あり学習向けにモデルをファインチューニングします。"
---

Serverless SFT は、[OpenPipe の ART フレームワーク](https://art.openpipe.ai/getting-started/about) と [W&amp;B Training API](/ja/training/api-reference) を通じてサポートされています。

Serverless SFT を使い始めるには、まず W&amp;B ツールを使用するための[前提条件](/ja/training/prerequisites)を満たし、その後 ART の [Serverless SFT ドキュメント](https://art.openpipe.ai/fundamentals/sft-training)を確認してください。

* Serverless SFT の API エンドポイントについては、[W&amp;B Training API リファレンス](/ja/training/api-reference)を参照してください。
