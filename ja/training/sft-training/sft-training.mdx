---
title: Serverless SFT の使用方法
description: "OpenPipe の ART フレームワークと W&B Training API を使用して、Serverless SFT で教師あり学習向けにモデルをファインチューニングします。"
---

Serverless SFT は、[OpenPipe の ART フレームワーク](https://art.openpipe.ai/getting-started/about) と [W&amp;B Training API](/ja/training/api-reference) を通じて利用できます。

Serverless SFT の利用を開始するには、まず W&amp;B ツールを使用するための [前提条件](/ja/training/prerequisites) を満たし、その後 ART の [Serverless SFT ドキュメント](https://art.openpipe.ai/fundamentals/sft-training) を参照してください。

* Serverless SFT の API エンドポイントについては、[W&amp;B Training API リファレンス](/ja/training/api-reference) を参照してください。
