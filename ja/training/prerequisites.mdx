OpenPipe ART フレームワークまたは API 経由で W&amp;B Training 機能を使用する前に、以下の手順を完了してください。

<Tip>
  開始する前に、コストや制限を理解するために [利用情報と制限](/ja/training/serverless-rl/usage-limits) を確認してください。
</Tip>

<div id="sign-up-and-create-an-api-key">
  ## サインアップして API キーを作成する
</div>

マシンを W&amp;B で認証するには、まず API キーを発行する必要があります。

<ApiKeyCreate />

<div id="create-a-project-in-wb">
  ## W&amp;B でプロジェクトを作成する
</div>

W&amp;B アカウント内にプロジェクトを作成して、使用状況を追跡し、トレーニング メトリクスを記録し、学習済みモデルを保存します。詳細については、[Projects guide](https://docs.wandb.ai/models/track/project-page) を参照してください。

<div id="next-steps">
  ## 次のステップ
</div>

前提条件を完了したら、次のステップに進みます。

* 利用可能なエンドポイントについて知るには、[API リファレンス](/ja/training/api-reference) を確認してください
* [ART クイックスタート](https://art.openpipe.ai/getting-started/quick-start) を試してください
