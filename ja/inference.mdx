W&amp;B Inference を使うと、W&amp;B Weave と OpenAI 互換 API 経由で主要なオープンソースの基盤モデルにアクセスできます。

* Inference を使うと、ホスティングプロバイダーへの登録やモデルのセルフホスティングをせずに、AI アプリケーションやエージェントを構築できます。
* Weave を使うと、W&amp;B Inference を利用したアプリケーションをトレース、評価、監視、改善できます。

<div id="try-out-inference-in-the-ui">
  ## UI で Inference を試す
</div>

Web インターフェイスの詳細については、[UI Guide](/ja/inference/ui-guide/) を参照してください。

<div id="use-inference-through-the-api">
  ## API 経由で Inference を使用する
</div>

この Python の例では、Inference を使用して LLM への chat completion リクエストを送信します。

```python lines
import openai

client = openai.OpenAI(
    # カスタムベースURLはW&B Inferenceを指しています
    base_url='https://api.inference.wandb.ai/v1',

    # https://wandb.ai/settings でAPIキーを作成してください
    api_key="<your-api-key>",

    # オプション: 使用状況トラッキング用のチームとproject
    project="<your-team>/<your-project>",
)

response = client.chat.completions.create(
    model="meta-llama/Llama-3.1-8B-Instruct",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Tell me a joke."}
    ],
)

print(response.choices[0].message.content)
```
