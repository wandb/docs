---
title: Serverless SFT の使い方
description: "OpenPipe の ART フレームワークと Serverless RL API を使用して、教師あり学習向けにモデルをファインチューニングします。"
---

Serverless SFT は、[OpenPipe の ART フレームワーク](https://art.openpipe.ai/getting-started/about) と [Serverless RL API](/ja/serverless-rl/api-reference) を通じて利用できます。

Serverless SFT の利用を開始するには、まず W&amp;B ツールを使用するための[前提条件](/ja/serverless-rl/prerequisites)を満たしたうえで、ART の [Serverless SFT ドキュメント](https://art.openpipe.ai/fundamentals/sft-training)をご確認ください。

* Serverless SFT の API エンドポイントについては、[Serverless RL API リファレンス](/ja/serverless-rl/api-reference)を参照してください。
