<Note>
  Serverless RL API は、サーバーレス強化学習 (RL) や教師ありファインチューニング (SFT) などのトレーニングジョブを管理し、操作するためのエンドポイントを提供します。この API は、chat completions で OpenAI 互換です。
</Note>

<div id="authentication">
  ## 認証
</div>

すべての API リクエストでは、W&amp;B の APIキーを使用した認証が必要です。[wandb.ai/settings](https://wandb.ai/settings) で APIキーを作成してください。

`Authorization` ヘッダーに APIキーを含めます。

```
Authorization: Bearer YOUR_API_KEY
```

<div id="base-url">
  ## ベース URL
</div>

```
https://api.training.wandb.ai/v1
```

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="chat-completions">
  ### chat-completions
</div>

* **[POST /v1/chat/completions](/ja/serverless-rl/api-reference/chat-completions/create-chat-completion)** - Chat Completion を作成します
* **[POST /v1/chat/completions/](/ja/serverless-rl/api-reference/chat-completions/create-chat-completion)** - Chat Completion を作成します

<div id="models">
  ### models
</div>

* **[POST /v1/preview/models](/ja/serverless-rl/api-reference/models/create-model)** - モデルを作成
* **[DELETE /v1/preview/models/{model_id}](/ja/serverless-rl/api-reference/models/delete-model)** - モデルを削除
* **[DELETE /v1/preview/models/{model_id}/checkpoints](/ja/serverless-rl/api-reference/models/delete-model-checkpoints)** - モデル チェックポイントを削除
* **[GET /v1/preview/models/{model_id}/checkpoints](/ja/serverless-rl/api-reference/models/list-model-checkpoints)** - モデル チェックポイントを一覧表示
* **[POST /v1/preview/models/{model_id}/log](/ja/serverless-rl/api-reference/models/log)** - ログする

<div id="training-jobs">
  ### training-jobs
</div>

* **[POST /v1/preview/sft-training-jobs](/ja/serverless-rl/api-reference/training-jobs/create-sft-training-job)** - SFTトレーニングジョブを作成
* **[POST /v1/preview/training-jobs](/ja/serverless-rl/api-reference/training-jobs/create-rl-training-job)** - RLトレーニングジョブを作成
* **[GET /v1/preview/training-jobs/{training_job_id}](/ja/serverless-rl/api-reference/training-jobs/get-training-job)** - トレーニングジョブを取得
* **[GET /v1/preview/training-jobs/{training_job_id}/events](/ja/serverless-rl/api-reference/training-jobs/get-training-job-events)** - トレーニングジョブのイベントを取得

<div id="health">
  ### ヘルス

* **[GET /v1/health](/ja/serverless-rl/api-reference/health/health-check)** - ヘルスチェック
* **[GET /v1/system-check](/ja/serverless-rl/api-reference/health/system-check)** - システムチェック
