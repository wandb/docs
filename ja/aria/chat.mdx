---
title: "ARIA とチャットする"
---

このページでは、新しいチャットを開始する方法、チャット履歴を表示する方法、過去の会話を削除する方法、チャットウィンドウを切り離す方法、プロンプト内で特定の run を参照する方法、会話に画像を追加する方法、ARIA の応答にフィードバックを送信する方法について説明します。

<div id="start-a-new-chat">
  ## 新しいチャットを開始する
</div>

新しいチャットを開始するには、ARIA を開き、チャットウィンドウの右上にある鉛筆アイコンをクリックします。

複数のチャットを同時並行で進めることができます。チャットウィンドウを閉じていても、チャットに返信があると ARIA から通知されます。

新しいチャットを開始しても、以前の会話は削除されません。[以前の会話を表示する](/ja/aria/chat#view-chat-history)には、チャットウィンドウの左上にある 3 本線をクリックしてチャット履歴を開きます。

<div id="view-chat-history">
  ## チャット履歴を表示する
</div>

以前の会話を表示するには、次の手順に従います。

1. ARIA を開きます。
2. チャットウィンドウの左上にある 3 本線をクリックして、チャット履歴を開きます。
3. 会話を選択します。

W&amp;B では、ARIA との会話は project ごと、および日付ごとに整理されます。チャット履歴から選択すると、以前の会話を再開できます。また、会話の横にあるごみ箱アイコンをクリックして、[会話を削除する](/ja/aria/chat#delete-previous-conversations)こともできます。

<div id="delete-previous-conversations">
  ## 以前の会話を削除する
</div>

以前の会話を削除するには、次の手順に従ってください。

1. ARIA を開きます。
2. チャットウィンドウの左上にある 3 本線をクリックして、チャット履歴を開きます。
3. 削除したい会話の横にあるごみ箱アイコンをクリックします。

<div id="undock-chat-window">
  ## チャットウィンドウを切り離す
</div>

チャットウィンドウを別ウィンドウとして切り離すことができます。これにより、エージェントの応答を project データと並べて表示できます。

チャットを切り離すには、チャットウィンドウの右上にある閉じる (**X**) ボタンの横の切り離しアイコンをクリックします。

<div id="add-context-to-a-conversation">
  ## 会話にコンテキストを追加する
</div>

`@` 記号を使用して、プロンプト内で特定の run を指定できます。たとえば、プロンプトにその run 名を含めることで、ARIA に特定の run を分析するよう依頼できます。

```text title="User prompt"
Analyze the run @run-20230615_123456-abcdefg and summarize the results
```

<div id="add-an-image-to-a-conversation">
  ## 会話に画像を追加する
</div>

チャットウィンドウの左下にあるプラスアイコン (`+`) をクリックすると、会話に画像を追加できます。その後、ARIA に画像の分析や説明を依頼できます。

<div id="provide-feedback">
  ## フィードバックを送信する
</div>

ARIA の応答についてフィードバックを送信するには、応答の末尾にあるサムズアップまたはサムズダウンのアイコンをクリックします。お寄せいただいたフィードバックは、W&amp;B による ARIA のパフォーマンスと精度の向上に役立ちます。

W&amp;B における ARIA のデータ利用の取り扱いについては、[ガバナンスとセキュリティ](/ja/aria/governance)を参照してください。
