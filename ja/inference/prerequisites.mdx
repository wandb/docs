API または UI を通じて W&amp;B Inference サービスを使用する前に、これらの step を完了してください。

<Tip>
  開始する前に、コストや制限を把握するために [使用状況情報と制限](/ja/inference/usage-limits/) を確認してください。
</Tip>

<div id="set-up-your-wb-account-and-project">
  ## W&amp;B アカウントとプロジェクトを設定する
</div>

W&amp;B Inference にアクセスするには、次の項目が必要です。

1. **W&amp;B アカウント**
   [W&amp;B](https://app.wandb.ai/login?signup=true) でサインアップしてください

2. **W&amp;B API キー**

   <ApiKeyCreate />

3. **W&amp;B プロジェクト**
   利用状況を追跡するために、W&amp;B アカウント内にプロジェクトを作成してください

<div id="set-up-your-environment-python">
  ## 環境を設定する (Python)
</div>

Python で Inference API を使用するには、以下の手順も実行してください。

1. 上記の一般要件を完了してください

2. 必要なライブラリをインストールしてください:

   ```bash
   pip install openai weave
   ```

<Note>
  **注意**

  `weave` ライブラリはオプションですが、推奨されます。LLM アプリケーションをトレースできるためです。詳細は [Weave クイックスタート](/ja/models/quickstart/) をご覧ください。

  W&amp;B Inference を Weave で使用したコード サンプルについては、[使用例](/ja/inference/examples/) を参照してください。
</Note>

<div id="next-steps">
  ## 次のステップ
</div>

前提条件を完了したら:

* 利用可能なエンドポイントについて学ぶために、[API リファレンス](/ja/inference/api-reference/)を確認してください
* サービスの動作を確認するために、[使用例](/ja/inference/examples/)を試してください
* Web インターフェース経由でモデルにアクセスするために、[UI ガイド](/ja/inference/ui-guide/)を使用してください
