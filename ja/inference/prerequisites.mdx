API または UI を通じて W&amp;B Inference サービスを使用する前に、これらの手順を完了しておいてください。

<Tip>
  開始する前に、コストや制限事項を把握するために [使用状況情報と制限](/ja/inference/usage-limits/) を確認してください。
</Tip>

<div id="set-up-your-wb-account-and-project">
  ## W&amp;B アカウントとプロジェクトを設定する
</div>

W&amp;B Inference にアクセスするには、次の項目が必要です。

1. **W&amp;B アカウント**\
   [W&amp;B](https://app.wandb.ai/login?signup=true) でサインアップしてください

2. **W&amp;B APIキー**

   <ApiKeyCreate />

3. **W&amp;B プロジェクト**\
   利用状況を追跡するために、W&amp;B アカウント内にプロジェクトを作成してください

<div id="set-up-your-environment-python">
  ## 環境をセットアップする (Python)
</div>

Python で Inference API を使用するには、次のことも行う必要があります。

1. 上記の一般要件を満たします

2. 必要なライブラリをインストールします:

   ```bash
   pip install openai weave
   ```

<Note>
  **注意**

  `weave` ライブラリは必須ではありませんが、LLM アプリケーションのトレースに役立つため、利用を推奨します。詳しくは [Weave クイックスタート](/ja/models/quickstart/) を参照してください。

  Weave と組み合わせて W&amp;B Inference を使用するコードサンプルについては、[使用例](/ja/inference/examples/) を参照してください。
</Note>
