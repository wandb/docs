Web UIでW&amp;B Inferenceサービスを使用する方法を説明します。UIを使用する前に、[前提条件](/ja/inference/prerequisites/)を完了してください。

<div id="access-the-inference-service">
  ## Inference サービスにアクセスする
</div>

Inference サービスには、次の 3 つの場所からアクセスできます。

<div id="direct-link">
  ### 直接リンク
</div>

[https://wandb.ai/inference](https://wandb.ai/inference) にアクセスしてください。

<div id="from-the-inference-tab">
  ### Inference タブから
</div>

1. [https://wandb.ai/](https://wandb.ai/) で W&amp;B アカウントにアクセスします
2. 左サイドバーから **Inference** を選択します
3. 利用可能なモデルとその情報が表示されるページが開きます

<Frame>
  <img src="/images/inference/inference-playground-single.png" alt="Playground で Inference モデルを使用する" />
</Frame>

<div id="from-the-playground-tab">
  ### Playground タブから
</div>

1. 左サイドバーで **Playground** を選択します。Playground のチャット UI が表示されます
2. LLM のドロップダウンリストで **W&amp;B Inference** にカーソルを合わせます。右側に利用可能なモデルのドロップダウンが表示されます
3. モデルのドロップダウンでは、次の操作ができます:
   * 任意のモデル名をクリックして、[そのモデルを Playground で試す](#try-a-model-in-the-playground)
   * [複数のモデルを比較する](#compare-multiple-models)

<Frame>
  <img src="/images/inference/inference-playground.png" alt="Playground の Inference モデルのドロップダウン" />
</Frame>

<div id="try-a-model-in-the-playground">
  ## Playgroundでモデルを試す
</div>

[モデルを選択](#access-the-inference-service)したら、Playgroundでテストできます。実行できる操作は次のとおりです。

* [Playgroundの設定とモデルのパラメーターをカスタマイズする](/ja/weave/guides/tools/playground#customize-playground-settings)
* [メッセージを追加、再試行、編集、削除する](/ja/weave/guides/tools/playground#message-controls)
* [カスタム設定を適用したモデルを保存して再利用する](/ja/weave/guides/tools/playground#saved-models)
* [複数のモデルを比較する](#compare-multiple-models)

<div id="compare-multiple-models">
  ## 複数のモデルを比較する
</div>

Playground では、Inference のモデルを並べて比較できます。Compare ビューには 2 つの場所からアクセスできます。

### Inferenceタブから

1. 左サイドバーから **Inference** を選択します。利用可能なモデルのページが表示されます
2. モデルカード上の任意の場所 (モデル名を除く) をクリックして選択します。カードの枠線が青色に変わります
3. 比較したい各モデルについて、この操作を繰り返します
4. 選択したいずれかのカードで **Playground で N 個のモデルを比較** をクリックします。`N` には選択したモデル数が表示されます
5. 比較ビューが開きます

これで、モデルを比較したり、[Playground でモデルを試す](#try-a-model-in-the-playground) のすべての機能を利用したりできます。

<Frame>
  <img src="/images/inference/inference-playground-compare.png" alt="Playground で比較する複数のモデルを選択" />
</Frame>

<div id="from-the-playground-tab">
  ### Playground タブから
</div>

1. 左サイドバーから **Playground** を選択します。Playground のチャット UI が表示されます
2. LLM のドロップダウンリストで **W&amp;B Inference** にカーソルを合わせます。右側にモデルのドロップダウンが表示されます
3. ドロップダウンから **Compare** を選択します。**Inference** タブが表示されます
4. モデルカード上の任意の場所 (モデル名を除く) をクリックして選択します。カードの枠線が青色になります
5. 比較したい各モデルに対してこれを繰り返します
6. 選択したいずれかのカードで **Playground で N 個のモデルを比較** をクリックします。比較ビューが開きます

これで、モデルを比較し、[Playground でモデルを試す](#try-a-model-in-the-playground) のすべての機能を利用できます。

<div id="view-billing-and-usage-information">
  ## 請求と利用状況の情報を確認する
</div>

組織管理者は、W&amp;B UI でクレジット残高、利用履歴、今後の請求額を確認できます。

1. UI の W&amp;B **Billing** ページにアクセスする
2. 右下にある Inference の請求情報カードを検索する
3. ここから次の操作を行えます:
   * **View usage** をクリックして、一定期間の利用状況を確認する
   * 今後の Inference の請求額を確認する (有料プランの場合)

<Tip>
  モデルごとの料金の詳細は、[Inference pricing page](https://wandb.ai/site/pricing/inference)を参照してください。
</Tip>
