title: "W&B Inference を使用する Cline"
description: >
  W&B Inference を使用するように Cline コーディングエージェントを設定する方法を学びます。
---

[Cline](https://cline.bot) は AI 搭載のコーディングアシスタントです。このチュートリアルでは、W&amp;B Inference が提供するモデルを使用するように Cline を設定する方法を説明します。

Cline エージェントは、コマンドラインツールとして、または多くの IDE とのインテグレーションとして使用可能です。このページでは、Cline CLI を使用する設定と、Visual Studio Code 拡張機能としての Cline の設定を示しますが、他の IDE での設定も同様です。

<div id="prerequisites">
  ## 前提条件
</div>

[W&amp;B APIキー](../prerequisites#set-up-your-wb-account-and-project)が必要です。

<div id="set-up-cline-in-the-command-line">
  ## コマンドラインで Cline を設定する
</div>

[Cline CLI](https://docs.cline.bot/cline-cli/installation) をインストールします。

```
npm install -g cline
```

トラブルシューティングについては、[Cline のインストール手順](https://docs.cline.bot/cline-cli/installation)を参照してください。

これらの手順は、Cline CLI バージョン `2.5.1` でテスト済みです。インストールされているバージョンを確認するには、`cline version` を実行してください。

次のコマンドを実行し、`<your_api_key>` を W&amp;B APIキーに置き換えてください。

```
cline auth -k <your_api_key> -p openai -b https://api.inference.wandb.ai/v1 -m moonshotai/Kimi-K2.5
```

これにより、Cline は W&amp;B Inference の OpenAI 互換エンドポイントと Kimi K2.5 モデルを使用するように設定されます。モデルの提供状況と料金については、[モデルカタログ](https://wandb.ai/inference) を参照してください。

すべてが正しく動作していることを確認するため、簡単なテストを実行してください:

```
cline "What is 2 + 2?"
```

Cline が応答を返したら、インストールと認証は完了です。

Cline CLI の使い方の詳細については、[Cline クイックスタート](https://docs.cline.bot/cline-cli/installation#quick-start) を参照してください.


<div id="set-up-cline-in-visual-studio-code">
  ## Visual Studio Code で Cline を設定する
</div>

Cline は Visual Studio Code の拡張機能としてインストールすることもできます。VS Code Extensions Marketplace で **Cline** を検索するか、[Visual Studio Code Marketplace](https://marketplace.visualstudio.com/items?itemName=saoudrizwan.claude-dev) からインストールします。

`Install` ボタンをクリックします。

<Frame>![VS Code extension installation](/images/inference/inference-tutorial-cline-install.png)</Frame>

アクティビティバーに追加された Cline アイコンをクリックして、Cline サイドバーを開きます。アクティビティバーに Cline アイコンが表示されていない場合は、ドロップダウンをクリックして他のオプションを表示します。

<img src="/images/inference/inference-tutorial-cline-activity-bar.png" alt="VS Code extension installation" style={{width: "60px"}} />

&quot;Bring my own API key&quot; を選択し、**Continue** ボタンをクリックします。

<img src="/images/inference/inference-tutorial-cline-byok.png" alt="Select Bring Your Own Key" style={{width: "300px"}} />

以下の値を指定し、`<your_api_key>` を自分の W&amp;B APIキーに置き換えます。

| Setting                   | Value                             |
| ------------------------- | --------------------------------- |
| API Provider              | OpenAI Compatible                 |
| Base URL                  | https://api.inference.wandb.ai/v1 |
| OpenAI Compatible API Key | `<your_api_key>`                  |
| Model ID                  | moonshotai/Kimi-K2.5              |

料金は Model Configuration を展開して設定できます。モデルの利用可能性および価格については、[モデルカタログ](https://wandb.ai/inference) を参照してください。

`Continue` ボタンをクリックします。

&quot;Write a Python program to compute the first 10 Fibonacci Numbers&quot; のようなプロンプトを送信してみてください。

<img src="/images/inference/inference-tutorial-cline-act.png" alt="Submit your coding task to the agent" style={{width: "300px"}} />

Cline の詳しい使い方については、[Cline Documentation](https://docs.cline.bot) を参照してください。
