---
title: "W&B Inference で Cline を使う"
description: >
  W&B Inference を使用するように Cline コーディングエージェントを設定する方法を説明します。
---

[Cline](https://cline.bot) は AI 搭載のコーディングアシスタントです。このチュートリアルでは、W&amp;B Inference が提供するモデルを使用するように Cline を設定する方法を紹介します。

Cline エージェントは、コマンドラインツールとして、または多くの IDE とのインテグレーションとして利用できます。このページでは、Cline CLI での設定と、Visual Studio Code 拡張機能として使用する Cline の設定を紹介しますが、他の IDE での設定もほぼ同様です。

<div id="prerequisites">
  ## 前提条件
</div>

ご自身の[W&amp;B APIキー](../prerequisites#set-up-your-wb-account-and-project)が必要です。

<div id="set-up-cline-in-the-command-line">
  ## コマンドラインで Cline を設定する
</div>

[Cline CLI](https://docs.cline.bot/cline-cli/installation) をインストールします。

```
npm install -g cline
```

トラブルシューティングについては、[Cline のインストール手順](https://docs.cline.bot/cline-cli/installation)を参照してください。

これらの手順は Cline CLI バージョン `2.5.1` で検証されています。`cline version` を実行すると、インストールされているバージョンを確認できます。

次のコマンドを実行し、`<your_api_key>` を W&amp;B の APIキーに置き換えてください。

```
cline auth -k <your_api_key> -p openai -b https://api.inference.wandb.ai/v1 -m moonshotai/Kimi-K2.5
```

これで、Cline が W&amp;B Inference の OpenAI 互換エンドポイントと Kimi K2.5 モデルを使用するように設定されます。利用可能なモデルと料金については、[モデルカタログ](https://wandb.ai/inference)を参照してください。

すべてが正しく動作していることを確認するため、簡単なテストを実行します：

```
cline "What is 2 + 2?"
```

Cline から応答が返ってきたら、インストールと認証は完了です。

Cline CLI の使い方について詳しくは、[Cline クイックスタート](https://docs.cline.bot/cline-cli/installation#quick-start)を参照してください。


<div id="set-up-cline-in-visual-studio-code">
  ## Visual Studio Code で Cline を設定する
</div>

Cline は、Visual Studio Code の拡張機能としてインストールすることもできます。VS Code Extensions Marketplace で **Cline** を検索するか、[Visual Studio Code Marketplace](https://marketplace.visualstudio.com/items?itemName=saoudrizwan.claude-dev) からインストールしてください。

`Install` ボタンをクリックします。

<Frame>![VS Code 拡張機能のインストール](/images/inference/inference-tutorial-cline-install.png)</Frame>

新しく追加された Activity Bar の Cline アイコンをクリックして、Cline サイドバーを開きます。Activity Bar に Cline アイコンが表示されていない場合は、ドロップダウンをクリックしてその他のオプションを表示します。

<img src="/images/inference/inference-tutorial-cline-activity-bar.png" alt="VS Code 拡張機能のインストール" style={{width: "60px"}} />

&quot;Bring my own API key&quot; を選択し、**Continue** ボタンをクリックします。

<img src="/images/inference/inference-tutorial-cline-byok.png" alt="Bring Your Own Key を選択" style={{width: "300px"}} />

次の値を指定します。`<your_api_key>` はご自身の W&amp;B APIキーに置き換えてください。

| 設定                      | 値                                 |
| ----------------------- | --------------------------------- |
| APIプロバイダー               | OpenAI Compatible                 |
| Base URL                | https://api.inference.wandb.ai/v1 |
| OpenAI Compatible APIキー | `<your_api_key>`                  |
| モデル ID                  | moonshotai/Kimi-K2.5              |

Model Configuration を展開して料金を指定します。利用可能なモデルと料金については、[モデルカタログ](https://wandb.ai/inference)を参照してください。

`Continue` ボタンをクリックします。

「最初の 10 個のフィボナッチ数を計算する Python プログラムを書いて」のようなプロンプトを送信してみてください。

<img src="/images/inference/inference-tutorial-cline-act.png" alt="コーディングタスクをエージェントに送信する" style={{width: "300px"}} />

Cline の使用方法の詳細については、[Cline Documentation](https://docs.cline.bot)を参照してください。
