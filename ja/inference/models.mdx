<div id="using-model-ids">
  ## モデル ID の使用
</div>

API を使用する場合は、上の表に記載されている `Model ID` を使ってモデルを指定します。例えば、次のように指定します。

```python
response = client.chat.completions.create(
    model="meta-llama/Llama-3.1-8B-Instruct",
    messages=[...]
)
```
