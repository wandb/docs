<div id="using-model-ids">
  ## モデル ID の使用
</div>

APIを使用する場合は、上の表にある `Model ID` を使ってモデルを指定します。たとえば:

```python
response = client.chat.completions.create(
    model="meta-llama/Llama-3.1-8B-Instruct",
    messages=[...]
)
```
